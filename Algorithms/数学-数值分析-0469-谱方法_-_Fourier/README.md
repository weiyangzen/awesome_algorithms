## R01

Fourier 谱方法的核心思想是：在周期区间上用全局三角基函数表示解，把微分算子变成频域中的代数运算。

本条目给出一个最小可运行 MVP：
- 目标问题：一维周期 Poisson 方程 `-u''(x)=f(x)`；
- 数值方法：`FFT -> 频域逐模态求解 -> iFFT`；
- 约束处理：显式处理零频模态，固定 `mean(u)=0`；
- 验证方式：多分辨率误差表 + 方程残差检查。

## R02

连续问题定义：

- 区间：`x in [0, 2pi)`，周期边界；
- 方程：`-u''(x)=f(x)`；
- 周期条件：`u(0)=u(2pi)`，`u'(0)=u'(2pi)`；
- 规范条件（gauge）：`mean(u)=0`。

对周期 Poisson 方程，必须满足可解性条件 `mean(f)=0`。在实现里会先对离散 `f` 做去均值处理以消除浮点误差引入的零模漂移。

## R03

Fourier 展开设为：

`u(x) = sum_k u_hat_k * exp(i k x)`，
`f(x) = sum_k f_hat_k * exp(i k x)`。

代入 `-u''=f` 后，对每个模态有：

`k^2 * u_hat_k = f_hat_k`。

因此：
- `k != 0` 时：`u_hat_k = f_hat_k / k^2`；
- `k = 0` 时：方程不给出唯一值，需通过规范条件指定，这里取 `u_hat_0 = 0`（即 `mean(u)=0`）。

## R04

离散化与 FFT 对应关系：

- 网格：`x_j = j*dx`，`j=0..N-1`，`dx=2pi/N`；
- 使用 `numpy.fft.fft` / `numpy.fft.ifft` 在物理域与频域间变换；
- 角波数通过 `k = 2pi * fftfreq(N, d=dx)` 构建；
- 频域解完成后做 `ifft` 并取实部得到数值解。

该实现是典型 Fourier-Galerkin 的离散版本，微分算子在频域精确对角化。

## R05

零模态与可解性处理：

- 连续理论要求 `integral(f dx)=0`；
- 离散场景中即使解析上满足，采样+浮点也可能出现 `mean(f)!=0` 的极小偏差；
- `demo.py` 先执行 `f_zero_mean = f - mean(f)`；
- 再设 `u_hat[0]=0`，固定解的常数偏移。

这样可避免除零与不可解模式污染。

## R06

精度特性：

- 对光滑周期函数，Fourier 谱方法通常表现为“谱收敛”（误差随 `N` 近似指数下降）；
- 对有限正则或含间断函数，收敛退化并可能出现 Gibbs 现象；
- 本 MVP 选用解析光滑函数构造 `u` 与 `f`，用于展示谱方法在光滑问题上的高精度优势。

## R07

本实现的数值稳定性说明：

- 问题是线性椭圆方程，不含时间推进，因此没有 CFL 约束；
- 条件数由 `1/k^2` 放大低频分量决定，`k=0` 已单独处理；
- 对当前规模（`N<=24`）和双精度浮点，误差通常由截断误差转向舍入误差主导。

## R08

复杂度：

- 一次 FFT 与一次 iFFT 各 `O(N log N)`；
- 频域逐模态求解是 `O(N)`；
- 总时间复杂度 `O(N log N)`，空间复杂度 `O(N)`。

相较直接稠密线性求解 `O(N^3)`，谱法在周期问题上有明显效率优势。

## R09

算法伪代码：

```text
输入: 网格点数 N, 周期长度 L=2pi, 离散右端 f_j
1) 构建网格与步长 dx=L/N
2) f <- f - mean(f)           # 可解性投影
3) k <- 2pi * fftfreq(N, d=dx)
4) f_hat <- FFT(f)
5) 对每个模态:
   - 若 |k|>eps: u_hat = f_hat / k^2
   - 否则:       u_hat = 0      # mean(u)=0
6) u <- Re(iFFT(u_hat))
7) 输出 u，并可计算残差 r=(-u'')-f
```

## R10

`demo.py` 函数分工：

- `build_periodic_grid`：构建周期网格与 `dx`；
- `exact_solution_raw`：解析解模板（未减均值）；
- `forcing_from_exact`：解析构造右端 `f=-u''`；
- `fourier_wavenumbers`：生成角波数数组；
- `solve_poisson_fourier`：核心 Fourier 谱求解器；
- `poisson_residual`：在物理空间计算方程残差；
- `run_case`：单分辨率误差与残差统计；
- `effective_order`：估计相邻分辨率有效阶；
- `main`：批量实验、打印表格、执行最小断言。

## R12

输出字段说明：

- `nx`：空间网格点数；
- `L2_error`：数值解与解析解的离散 `L2` 误差；
- `Linf_error`：最大绝对误差；
- `residual_L2`：方程残差 `(-u''-f)` 的离散 `L2`；
- `residual_Linf`：方程残差最大范数；
- `p`：由相邻网格误差估计的有效阶（`err ~ N^{-p}`）。

## R13

最小验证集：

1. `N={8,12,16,20,24}` 多分辨率误差实验，观察 `L2_error` 快速下降；
2. 每组都计算残差范数，验证方程在离散意义下被满足；
3. 检查最终网格误差相对粗网格显著下降（脚本内置阈值断言）；

## R14

关键参数建议：

- `resolutions`：可先用 `[8,12,16,20,24]`，若想观察舍入误差平台可继续加大 `N`；
- `length`：默认 `2pi`，若改区间需同步检查波数定义；
- `eps`（零模判定阈值）：当前使用 `1e-14`，双精度下足够。

## R15

常见实现错误：

- 直接对 `k=0` 做除法导致 NaN/Inf；
- 忘记对 `f` 去均值，导致不可解分量混入；
- 波数构造误把 `fftfreq` 当角波数，漏乘 `2pi`；
- 把 `-u''=f` 的符号写反，导致得到 `u_hat=-f_hat/k^2`；
- 比较解析解时忽略常数偏移（未对齐 `mean(u)=0` 规范）。

## R16

与常见方法对比：

- 相比二阶中心差分：谱法在光滑周期问题上精度更高、达到同精度所需网格更少；
- 相比有限元通用边界处理：谱法在规则周期几何上实现更简洁；
- 相比黑盒 PDE 求解器：当前实现透明展示了“变换-逐模态-逆变换”的算法核心。

## R17

可扩展方向：

- 扩展到二维/三维周期 Poisson（张量积 FFT）；
- 扩展到时间依赖 PDE（如热方程、KdV、NLS）的 Fourier 伪谱离散；
- 在非线性问题中加入反混叠策略（如 `2/3` 截断）；
- 增加与有限差分/有限元的精度-耗时对比实验。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 设定分辨率列表 `resolutions`，逐个调用 `run_case`。  
2. `run_case` 构建周期网格，生成解析解 `u_exact` 并做 `mean(u_exact)=0` 对齐。  
3. 同一网格上用 `forcing_from_exact` 构造右端 `f=-u''`。  
4. `solve_poisson_fourier` 先对 `f` 去均值，再调用 `fft` 得到 `f_hat`。  
5. 通过 `fourier_wavenumbers` 生成角波数 `k`，对每个非零模态计算 `u_hat=f_hat/k^2`，零模置 0。  
6. 对 `u_hat` 做 `ifft` 回到物理空间得到 `u_num`。  
7. `run_case` 计算误差 `u_num-u_exact`，并在 `poisson_residual` 中按 `(-u'')-f` 计算残差范数。  
8. `main` 汇总打印误差/残差表与相邻分辨率有效阶，并执行收敛与残差阈值断言。
