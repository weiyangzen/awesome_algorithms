## R01

高斯定律给出“闭合曲面电通量”和“包围电荷量”之间的直接关系。

- 积分形式：
  - `∮_S E·dA = Q_enclosed / epsilon_0`
- 微分形式（结合散度定理）：
  - `∇·E = rho / epsilon_0`

本目录 MVP 输入/输出：

- 输入：点电荷 `q`、高斯面几何参数（球半径或立方体边长）、Monte Carlo 采样数。
- 输出：数值电通量 `flux`、理论值 `q/epsilon_0`、相对误差、以及若干一致性验证指标。

## R02

历史背景（简要）：

- Carl Friedrich Gauss 在 19 世纪提出并系统化该定律（相关工作形成于 1830 年代）。
- 高斯定律是 Maxwell 方程组中静电部分的核心之一。
- 它把“局部场分布”转换为“全局积分约束”，极大简化了对称问题分析。

## R03

问题价值与工程动机：

- 直接计算 `E(x)` 在复杂几何上通常困难，但高斯定律允许先算“总通量”。
- 在电容器、绝缘结构、电荷封装、电场屏蔽等场景中，判断包围净电荷是第一步。
- 对称结构（球/柱/平面）常可由高斯定律得到闭式场解，减少数值求解成本。

## R04

本实现采用 Monte Carlo 曲面积分验证高斯定律，复杂度如下：

- 球面一次通量估计：`O(N)`，其中 `N` 为球面采样点数。
- 立方体一次通量估计：`O(6N_f)`，其中 `N_f` 为每个面的采样点数。
- 批量半径实验（`m` 个半径）：`O(mN)`。
- 线性回归验证（`k` 个半径点）：`O(k)`（该规模下矩阵极小，可视为常数级）。

## R05

空间复杂度：

- 球面采样需保存 `N x 3` 点与向量，空间 `O(N)`。
- 立方体每次单面采样 `N_f x 3`，循环复用数组，峰值 `O(N_f)`。
- 汇总表格 `pandas.DataFrame` 为 `O(m)`，远小于采样数组。

## R07

高斯定律在静电学中的算法意义：

- 提供从“体积分布”到“边界通量”的约束桥梁。
- 使很多场计算从 PDE 问题退化为代数关系（先求 `Q_enclosed`）。
- 是 Poisson/Laplace 方程、有限元场解和电容提取的理论基线。

## R08

前置知识与依赖：

- 物理：库仑场、通量定义、体电荷密度 `rho`。
- 数学：向量点积、曲面积分、散度定理。
- 编程依赖（本 MVP）：
  - `numpy`：向量化场计算与随机采样。
  - `scipy.constants`：`epsilon_0` 常数。
  - `pandas`：结果表格化展示。
  - `scikit-learn`：`LinearRegression` 做 `E-r` 线性拟合。
  - `torch`：与 NumPy 路径交叉校验球面积分。

## R09

适用前提与边界：

- 适用于静电或准静态条件。
- 公式对任意闭合曲面成立，但“快速求 `E`”通常需几何对称性。
- 若曲面穿过点电荷奇点，数值积分会失效；本实现显式禁止采样点与电荷重合。
- 非真空介质可改写为 `Q_enclosed/epsilon`；当前 demo 默认真空 `epsilon_0`。

## R10

正确性依据（两层）：

1. 物理层：高斯定律本身来自 Maxwell 方程与大量实验一致性。
2. 数值层：
   - 球面中心点电荷时，`E·n` 常数，数值积分应与 `q/epsilon_0` 一致。
   - 更换为立方体后通量仍应保持不变（几何无关、仅与净包围电荷有关）。
   - 电荷置于高斯面外，净通量应为 0（数值上接近 0）。

## R11

数值稳定性与误差特征：

- Monte Carlo 估计误差标准量级 `O(1/sqrt(N))`。
- 距离电荷很近时 `|E| ~ 1/r^2`，会放大局部采样方差。
- 本实现通过以下方式减小风险：
  - 固定随机种子，保证复现。
  - 输出 `stderr` 监控采样方差。
  - 对非法几何（`r<=0`、采样点与电荷重合）做显式异常。

## R12

真实执行成本（超越渐近符号）：

- 核心耗时在大规模向量运算（`N x 3` 张量/数组）与随机数生成。
- Pandas 与回归开销相对很小，主要用于可解释性验证。
- 在当前默认参数（十万级采样）下，脚本为秒级执行。

## R13

概率保证/误差界（本实现是随机采样积分）：

- 球面积分估计器是无偏的（均匀采样前提下）。
- 标准误差估计：`stderr = area * std(E·n) / sqrt(N)`。
- 在近似正态条件下，可给出约 `95%` 区间：
  - `flux ± 1.96 * stderr`。
- 因此这不是“确定性零误差算法”，而是“可随样本数收敛的随机估计”。

## R14

鲁棒性与失效模式：

- 失效 1：采样点落在点电荷位置，`1/r^2` 奇异导致发散。
- 失效 2：采样数太少，方差大，通量估计不稳定。
- 失效 3：单位混乱（cm/m 混用）导致数量级错误。
- 失效 4：把“高斯面外电荷”误计入 `Q_enclosed`。

缓解措施（已在 `demo.py` 中）：

- 参数合法性检查。
- 输出误差与 `stderr`。
- 采用多场景交叉验证（球、立方体、外部电荷、线性拟合）。

## R15

工程实现要点：

- `electric_field_point_charge` 统一封装场计算，避免公式重复。
- 球面积分使用“单位球方向 + 固定半径缩放”，实现均匀法向采样。
- 立方体积分按 6 个面独立采样并累加，结构清晰且易扩展。
- 使用 PyTorch 复写球面积分主路径，做 NumPy/PyTorch 双实现一致性校验。
- 用 `pandas` 输出对比表，便于批量审阅和自动化比对。

## R18

源码级流程拆解（`demo.py`，8 步，避免把三方库当黑箱）：

1. `main` 设定电荷、高斯面参数和随机种子，先计算理论通量 `q/epsilon_0` 作为对照基线。
2. `gauss_flux_sphere_mc` 调用 `sample_unit_sphere` 生成均匀方向，再构造球面采样点 `x = center + r*n`。
3. `electric_field_point_charge` 对每个采样点显式计算 `E = k*q*r_vec/|r_vec|^3`，并检查奇异点。
4. 通过 `integrand = E·n` 与 `flux = area * mean(integrand)` 形成 Monte Carlo 曲面积分，同时计算 `stderr`。
5. `gauss_flux_sphere_torch` 使用同一批方向样本，把第 3-4 步改写成 PyTorch 张量广播和归约；最终与 NumPy 结果逐项比较，验证实现一致性。
6. `gauss_flux_cube_mc` 按六个面分别采样 `(u,v)`，对每面计算 `E·n` 平均并乘面元面积，再求和得到立方体总通量，用来验证“与几何形状无关”。
7. `linear_fit_inside_uniform_sphere` 构造 `E(r)=rho*r/(3*epsilon_0)` 数据，调用 `LinearRegression.fit`；在 sklearn 内部会先预处理特征/目标，再通过最小二乘线性代数例程求解系数，返回斜率、截距与 `R^2`。
8. `main` 将各实验结果汇总为 `pandas.DataFrame` 并打印，形成“理论值-数值值-误差”闭环，确保脚本一次执行即可完成可审计验证。
