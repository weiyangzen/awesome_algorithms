## R01

阿哈罗诺夫-玻姆效应（AB 效应）说明：即使电子路径上局域磁场 `B=0`，只要包围了非零磁通 `Phi`，波函数依然会获得可观测相位。

核心相位关系：

`Delta_phi = (q / hbar) * ∮ A·dl = 2*pi*(Phi/Phi0)`（符号取决于电荷与路径方向）

其中 `Phi0 = h/|q|` 是磁通量子。该效应直接证明了规范势 `A` 的物理可观测性。

## R02

本条目目标是给出一个最小可运行数值演示：

- 用双路径干涉模型构造屏幕强度 `I(x, Phi)`；
- 显式注入 AB 相位 `Delta_phi`；
- 自动验证三件事：
  - 磁通周期性：`I(Phi + Phi0) = I(Phi)`；
  - 相位与磁通线性关系；
  - 条纹平移量与 `Delta_phi` 成正比。

## R03

MVP 使用 Fraunhofer 近似下的一维屏幕模型：

`I(x, Phi) = 1 + V * cos(beta*x + Delta_phi)`

- `V` 是可见度（`0~1`）；
- `beta = k*d/L`，`k=2*pi/lambda`；
- `d` 是双缝间距，`L` 是屏距，`lambda` 是德布罗意波长。

该模型省略了复杂包络函数与实验噪声，保留 AB 相位的主导可观测量。

## R04

实现用到的关键公式：

1. 磁通量子：`Phi0 = h/|q|`
2. AB 相位：`Delta_phi = s * 2*pi*(Phi/Phi0)`，`s` 为方向约定符号
3. 干涉条纹：`I(x, Phi) = 1 + V*cos(beta*x + Delta_phi)`
4. 条纹周期：`Lambda_x = 2*pi/beta = lambda*L/d`
5. 条纹位移：`x_shift = -Delta_phi/beta`

## R05

复杂度分析（`N` 为屏幕采样点数，`M` 为磁通采样点数）：

- 生成单个磁通下条纹：`O(N)`；
- 扫描全部磁通：`O(MN)`；
- 频率投影估计相位：每个磁通 `O(N)`，总 `O(MN)`。

总时间复杂度 `O(MN)`，空间复杂度 `O(N + M)`。

## R10

正确性直觉：

1. AB 效应本质是回路相位，理应按 `Phi0` 周期重复；
2. 在双缝远场中，相位只会把余弦条纹平移而不改周期；
3. 相位项线性加到 `beta*x`，故 `Delta_phi` 对 `Phi` 应呈线性；
4. 条纹平移 `x_shift = -Delta_phi/beta` 直接来自极大值条件；
5. 数值中若三者同时满足，说明模型实现链条自洽。

## R11

数值稳定策略：

- 使用高分辨率网格（默认 `4001` 点）保证频率投影稳定；
- 相位误差使用环形差值（wrap 到 `[-pi, pi]`），避免 `2*pi` 分支跳跃伪误差；
- 对可见度、采样点数、几何参数设置显式输入校验；
- 周期性检查采用完整条纹 RMSE，而不是单点比较。

## R12

关键参数与调参建议：

- `wavelength_m`：越短则 `beta` 越大，条纹周期更小；
- `slit_separation_m`：越大条纹越密，采样需更细；
- `screen_distance_m`：越大条纹越稀疏；
- `visibility`：接近 1 时相位估计更稳；
- `flux_scan`：建议覆盖负、零、正磁通，便于检验线性与对称性。

## R14

常见失效模式：

1. 屏幕采样过稀，导致相位投影混叠；
2. `visibility` 太低时，条纹对比度不足，相位估计噪声放大；
3. 使用不恰当的相位差定义，误把 `2*pi` 分支当作真实误差；
4. 参数单位混乱（`pm/um/m` 混用）导致周期量级错误。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `ABConfig` 定义几何与采样参数（波长、双缝距、屏距、磁通扫描范围等）。
2. `magnetic_flux_quantum` 计算 `Phi0=h/|q|`，作为磁通归一化基准。
3. `fringe_wave_number` 由 `k*d/L` 得到条纹相位系数 `beta`。
4. `ab_phase_shift` 按 `Delta_phi = s*2*pi*(Phi/Phi0)` 计算 AB 相位。
5. `interference_intensity` 在屏幕网格上生成 `I(x,Phi)=1+V*cos(beta*x+Delta_phi)`。
6. `estimate_phase_shift` 对条纹做单频复投影，反推出数值相位偏移。
7. `run_ab_effect_mvp` 扫描多个磁通，生成 `pandas` 报表并计算周期 RMSE、线性拟合斜率与 `R^2`。
8. `run_checks` 对周期性、相位误差、斜率误差、位移误差做断言，`main` 打印结果并给出 `All checks passed.`。
