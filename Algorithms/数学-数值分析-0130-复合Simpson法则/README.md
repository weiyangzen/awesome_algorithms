# 复合Simpson法则

## R01

复合 Simpson 法则（Composite Simpson's Rule）用于近似定积分
`I = \int_a^b f(x)\,dx`。

它把区间 `[a,b]` 等分为 `n` 个小区间（`n` 必须为偶数），在每两个小区间上用一个二次插值抛物线近似积分，再把各段结果求和。

标准公式（`x_i = a + i h`, `h=(b-a)/n`）:

`S_n = (h/3) * [ f(x_0) + f(x_n) + 4*sum_{i=1,3,...,n-1} f(x_i) + 2*sum_{i=2,4,...,n-2} f(x_i) ]`

## R02

- 历史来源: 归属于 Thomas Simpson（18 世纪）相关数值求积工作。
- 方法定位: Newton-Cotes 封闭型求积公式中的经典三点公式扩展。
- 现代地位: 在工程科学中是最常用的一维确定性数值积分基线方法之一。

## R03

很多积分没有解析原函数，或者解析表达式虽存在但计算代价高。复合 Simpson 的价值在于:

- 相比梯形法，通常在同样网格密度下精度更高。
- 相比高阶全局插值，局部二次拟合更稳定、实现更简单。
- 对平滑函数可达到四阶收敛（全局误差 `O(h^4)`），在中等精度需求下性价比高。

## R04

时间复杂度（设分段数为 `n`，且 `n` 为偶数）:

- 采样 `n+1` 个点并计算函数值: `O(n)`。
- 奇偶索引加权求和: `O(n)`。
- 总体: `O(n)`。

若函数值计算本身很昂贵，则真实主成本会由 `f(x_i)` 的评估开销主导。

## R05

空间复杂度:

- 流式累加实现: 额外 `O(1)`。
- 向量化实现（一次存储网格和函数值）: 额外 `O(n)`。

本目录 MVP 使用 NumPy 向量化，优先可读性与演示稳定性，空间为 `O(n)`。

## R06

手算示例（单次 Simpson，`n=2`）:

积分 `\int_0^1 x^2 dx`，取 `h=0.5`，节点 `x_0=0, x_1=0.5, x_2=1`。

- `f(x_0)=0`
- `f(x_1)=0.25`
- `f(x_2)=1`

`S_2 = (h/3) * [f(x_0)+4f(x_1)+f(x_2)]`
`    = (0.5/3) * (0 + 1 + 1)`
`    = 1/3`

与真值 `\int_0^1 x^2 dx = 1/3` 一致（Simpson 对三次及以下多项式精确）。

## R07

该方法的重要性:

- 作为“高精度且低实现复杂度”的通用一维积分工具。
- 是自适应 Simpson、Romberg 外推等更复杂方法的基础组件。
- 在物理仿真、统计期望近似、控制与信号处理中广泛使用。

## R08

核心理论基础:

- 插值思想: 在每两个小区间上用二次拉格朗日插值逼近 `f(x)`。
- 分片求积: 把全区间误差分散到局部小段并叠加。
- 误差项（`f \in C^4[a,b]`）:
  `E = I - S_n = -((b-a)/180) * h^4 * f^{(4)}(\xi), \xi\in(a,b)`。
- 因此有误差上界:
  `|E| <= ((b-a)/180) * h^4 * max_{x\in[a,b]} |f^{(4)}(x)|`。

## R09

适用条件与边界:

- `n` 必须为正偶数。
- 函数在区间上应足够平滑（至少近似满足四阶导有界）才能体现四阶收敛。
- 对尖点、间断、剧烈振荡函数，误差可能明显恶化。
- 区间可正向或反向；实现上应统一处理 `a>b` 的符号。

## R10

正确性依据（工程可用版本）:

1. 对每个长度 `2h` 的局部区间，三点二次插值积分公式成立。
2. 复合公式是对所有局部面板结果的线性求和。
3. 线性组合与积分算子线性一致，得到全局近似值。
4. 当 `f` 为三次及以下多项式时局部公式精确，复合后仍精确。
5. 对一般光滑函数，泰勒展开给出全局误差 `O(h^4)`。

## R11

数值稳定性与误差传播:

- 截断误差: 主项为 `O(h^4)`，由公式离散化带来。
- 舍入误差: 大量浮点求和会引入 `O(\epsilon)` 量级累积误差。
- 误差折中: `n` 增大可降低截断误差，但会增加函数评估次数与舍入噪声。
- 实践建议:
  - 对平滑函数先用几何级加密（如 `n=20,40,80...`）观察收敛比。
  - 对强振荡函数优先缩小步长并结合分段策略。

## R12

实际性能关注点:

- CPU 侧，函数求值通常比“加权求和”更耗时。
- NumPy 向量化能显著降低 Python 循环开销。
- 若函数非常昂贵，可缓存重复使用的采样值。
- 单次一维积分并行价值有限；多区间或多函数批量积分更适合并行。

## R13

可验证保证:

- 确定性误差界（`f^{(4)}` 有界）:
  `|I-S_n| <= ((b-a)/180) h^4 M4`，其中 `M4 = max |f^{(4)}(x)|`。
- 收敛阶保证:
  当步长减半（`h -> h/2`）时，主误差理论上约缩小 `2^4 = 16` 倍。
- 本目录 `demo.py` 会用 `exp(x)` 示例打印并验证这一趋势。

## R14

鲁棒性与常见失效模式:

- `n` 为奇数: 公式不成立，应立即报错。
- 非有限端点（NaN/Inf）: 应拒绝输入。
- 函数返回形状不匹配: 需检查并报错，防止静默广播错误。
- 非光滑函数: 即使代码正确，结果也可能不满足预期精度。

## R18

源码级算法拆解（对应 `demo.py` 的 `composite_simpson`，未调用黑箱积分器）:

1. 校验参数: `n` 必须是正偶数，`a,b` 必须是有限实数。
2. 处理方向: 若 `b<a` 则交换端点，并记录符号 `sign=-1`。
3. 计算步长 `h=(b-a)/n`，生成 `n+1` 个等距节点 `x_i`。
4. 向量化评估函数值 `y_i=f(x_i)`，并检查 `y` 与 `x` 形状一致。
5. 用切片分别求和:
   - 奇索引 `sum_odd = y[1:n:2]`
   - 偶索引 `sum_even = y[2:n:2]`
6. 按复合 Simpson 权重组合:
   `S=(h/3)*(y_0+y_n+4*sum_odd+2*sum_even)`。
7. 恢复区间方向符号并返回 `sign*S` 作为积分近似结果。

这个流程与数学公式一一对应，便于审计、测试和扩展到自适应版本。
