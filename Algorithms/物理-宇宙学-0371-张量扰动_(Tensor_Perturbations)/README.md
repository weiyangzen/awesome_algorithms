## R01

张量扰动是 FRW 背景上的横向无迹度规扰动，可理解为宇宙学引力波模。对每个 Fourier 模 `h_k(eta)`（`eta` 为共形时），无各向异性应力源时满足：

`h_k'' + 2 (a'/a) h_k' + k^2 h_k = 0`

其中 `a(eta)` 是尺度因子，`a'/a` 给出“宇宙膨胀阻尼项”，`k` 是共动波数。

## R02

它在宇宙学中的核心意义：

- 连接暴涨初始涨落与后期引力波谱观测（例如 CMB B 模、随机引力波背景）；
- 作为线性扰动理论里最干净的自由度之一（无标量/矢量规范复杂性）；
- 是研究“模式何时入视界、入视界后如何红移衰减”的标准教学对象。

## R03

本 MVP 目标是做一个最小但完整的“方程 -> 数值 -> 可验证”闭环：

1. 在一个解析可写的辐射-物质过渡背景上求解 `h_k`；
2. 同时计算多个 `k` 的末态传递系数 `|h_final/h_init|`；
3. 自动验证三条物理趋势：
   - 低 `k` 模在有限时间窗内可保持超视界近冻结；
   - 高 `k` 模入视界更早且阻尼更强；
   - 深入视界的高 `k` 模存在近守恒的绝热不变量代理量。

## R04

`demo.py` 采用的背景与方程：

- 背景插值（辐射 `a~eta` 到物质 `a~eta^2`）：
  `a(eta) = a_eq * [(eta/eta_eq)^2 + 2(eta/eta_eq)]`
- 导数：
  `a'(eta) = a_eq * [2(eta/eta_eq) + 2] / eta_eq`
- 视界判据（共形写法）：
  `k = a'/a`
- 张量扰动 ODE：
  `h'' + 2(a'/a)h' + k^2 h = 0`
- 数值初值：`h(eta_min)=1, h'(eta_min)=0`。

## R05

复杂度（设 `M` 个 `k` 模式、每个模式约 `N` 个采样点）：

- 每个模式一次 ODE 积分近似 `O(N)`（RK45 自适应步长下是同阶量级）；
- 总时间复杂度约 `O(MN)`；
- 存储所有时序的空间复杂度约 `O(MN)`；
- 汇总表与断言统计为 `O(M)`。

## R06

脚本的输出内容：

- 一个按 `k` 排序的 pandas 表，包含：
  - `eta_entry`（估计入视界时刻；若未入则 `not-entered`）
  - `h_initial`、`h_final`
  - `transfer_abs = |h_final/h_initial|`
  - `omega_gw_proxy`（简化能量密度代理）
- 三个聚合诊断：
  - 低 `k` 平均传递系数
  - 高 `k` 平均传递系数
  - `k=10` 在晚期的绝热不变量变异系数（CV）
- 若断言全通过，打印 `All checks passed.`。

## R09

## R10

正确性直觉：

1. 当 `k << a'/a`（超视界），`k^2 h` 项弱，`h_k` 近冻结；
2. 当 `k >> a'/a`（入视界后），方程呈阻尼振子，振幅随膨胀衰减；
3. 更大 `k` 更早满足 `k=a'/a`，因此更早入视界、经历更长阻尼阶段；
4. 深入视界下，`a^2(h'^2+k^2 h^2)/(2k)` 近似绝热守恒，可作数值一致性检查。

## R11

数值稳定策略：

- 从 `eta=0.05` 起步，避开 `eta->0` 的摩擦项奇异放大；
- 用 `solve_ivp(..., method="RK45", rtol=1e-8, atol=1e-10)` 控制精度；
- 视界入点用网格插值而非单点阈值，减少离散跳变误差；
- 晚期不变量检查仅在 `eta>=10` 执行，避免过渡期干扰。

## R12

关键参数与影响：

- `eta_eq`：辐射/物质过渡时标；
- `eta_grid=[0.05, 50]`：决定可观测到的入视界窗口；
- `k_values=[0.02, 0.05, 0.1, 0.3, 1, 3, 10]`：覆盖“未入视界 -> 深入视界”；
- 初值 `h=1, h'=0`：作为统一归一化，便于跨 `k` 比较；
- 断言阈值：
  - `k=0.02` 传递系数 `>0.85`
  - 高 `k` 平均传递系数 `< 0.05 * 低k均值`
  - `k=10` 不变量 CV `<0.02`。

## R14

常见失效模式：

1. 把 `a'/a` 写错成 `a/a'`，会完全颠倒阻尼行为；
2. `eta` 起点太靠近 0 导致刚性增强、积分噪声放大；
3. 只看单个 `k` 的某个相位点，误判阻尼趋势；
4. 用过短时间窗，低 `k` 可能看不到是否真正未入视界。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `scale_factor_and_derivative` 计算解析背景 `a(eta), a'(eta)`，把“宇宙学背景”与“扰动方程”显式解耦。  
2. `tensor_mode_rhs` 把二阶方程重写为一阶系统 `y=[h,h']`，返回 `[h', -2(a'/a)h' - k^2 h]`。  
3. `evolve_tensor_mode` 调用 `scipy.integrate.solve_ivp(RK45)`，在给定 `eta_grid` 上获得每个 `k` 的整条轨迹。  
4. `estimate_horizon_entry_eta` 先算 `a'/a` 网格，再对 `k >= a'/a` 的首个跨越点做线性插值，得到 `eta_entry`。  
5. 主循环对每个 `k` 计算 `h_final`、`transfer_abs`、`omega_gw_proxy`，累积到 `rows`，最后构造 `pandas.DataFrame`。  
6. `adiabatic_invariant_proxy` 计算 `I_k = a^2(h'^2+k^2 h^2)/(2k)`，用于检查高 `k` 深入视界后的近守恒性。  
7. `run_mvp` 执行四个自动断言：超视界冻结、高低 `k` 阻尼分离、入视界顺序、不变量 CV 阈值。  
8. `main` 负责把 `eta_entry` 做可读格式化并打印汇总与指标，全部通过后输出 `All checks passed.`。  

关于第三方库调用的可追踪性：本实现未把问题留给“单函数黑盒”。`solve_ivp` 只承担通用 ODE 步进；具体物理 RHS、入视界判据、传递系数统计、不变量构造和验证逻辑都在源码中显式展开。
