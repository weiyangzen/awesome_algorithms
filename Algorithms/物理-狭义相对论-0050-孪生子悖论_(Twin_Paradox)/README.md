## R01

孪生子悖论讨论“谁经历了更多固有时（proper time）”。
在狭义相对论中，真正可比较的是两条世界线之间的固有时长度，而不是“谁看谁慢”这种局部叙述。
往返旅行者在中途换向，世界线不是单一惯性系直线，因此其累计固有时通常小于地球静止孪生子。

## R02

本条目 MVP 要解决的最小问题：

- 构造一个对称往返行程：去程速度 `+beta`、回程速度 `-beta`，每段地球系时间为 `T_leg`；
- 用离散积分计算旅行者固有时 `tau_traveler`；
- 与解析解 `tau_ref = 2*T_leg*sqrt(1-beta^2)` 对比；
- 额外展示掉头瞬间的“同时性跳跃”，解释悖论为何不矛盾。

## R03

本实现使用自然单位 `c=1`（光年/年），核心关系：

- 洛伦兹因子：`gamma = 1/sqrt(1-beta^2)`；
- 固有时元：`d tau = dt * sqrt(1-beta(t)^2)`；
- 地球孪生子静止于 `x=0`，其固有时等于坐标时：`tau_earth = T_total`；
- 对称往返解析解：`tau_traveler = 2*T_leg/gamma = 2*T_leg*sqrt(1-beta^2)`。

同时性跳跃（在掉头事件 `t_turn, x_turn` 处）：

- 去程惯性系同时地球时刻：`t_sim_out = t_turn - beta*x_turn`；
- 回程惯性系同时地球时刻：`t_sim_in = t_turn + beta*x_turn`；
- 跳跃量：`Delta t_sim = 2*beta*x_turn`。

## R04

`demo.py` 的算法流程：

1. 建立均匀时间网格 `t in [0, 2*T_leg]`；
2. 依据分段定义生成速度序列 `beta(t)`；
3. 数值积分得到位置 `x(t)`，检查能否回到 `x=0`；
4. 数值积分 `d tau = dt*sqrt(1-beta^2)` 得到 `tau(t)`；
5. 计算解析固有时 `tau_ref` 并比较误差；
6. 计算掉头事件的同时性跳跃；
7. 打印年龄差、误差、跳跃量；
8. 用 `assert` 做非交互验证。

## R05

设时间离散点数为 `n`：

- 位置积分：`O(n)`；
- 固有时积分：`O(n)`；

总时间复杂度 `O(n)`，空间复杂度 `O(n)`。

## R10

正确性直觉：

1. 固有时是世界线长度，非“相互观测快慢”的简单对称；
2. 地球孪生子在同一惯性系保持静止，其世界线在该边界条件下固有时最大；
3. 旅行者换向导致其 simultaneity 切片改变，出现地球年龄的坐标跳跃；
4. 数值积分应逼近解析结果，且地球年龄应严格更大。

## R11

数值策略：

- 使用均匀时间网格，避免非均匀步长带来的积分解释歧义；
- 速度仅要求 `|beta|<1`，并做显式检查；
- 采用足够细的 `n_steps`，使 `tau_num` 与 `tau_ref` 误差落在毫年量级；
- 通过回到原点误差、固有时误差、符号性断言三重约束保证稳定性。

## R12

关键参数：

- `beta`：旅行速度（相对光速）；
- `duration_each_leg`：单程地球系时间；
- `n_steps`：离散采样数；
- `segments`：分段行程定义。

调参建议：

- 提高 `beta` 会显著增大年龄差；
- 减小 `n_steps` 会增大离散误差；
- 不对称行程可通过修改 `segments` 直接扩展。

## R14

常见失效模式：

1. `|beta|>=1` 导致根号项无定义；
2. 时间网格非均匀导致积分结果不可直接对比；
3. 行程总时长与分段时长不一致，造成“伪回程”；
4. 样本过粗导致数值误差掩盖解析关系。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 设定 `beta=0.8`、`duration_each_leg=10` 年、`n_steps=20001`，并创建对称两段 `Segment`。  
2. `build_piecewise_velocity` 在均匀网格上把分段定义展开成 `beta(t)` 序列，并校验总时长一致性。  
3. `integrate_position` 对 `dx = beta*dt` 做离散累积，得到 `x(t)` 并用于重逢位置检查。  
4. `integrate_proper_time` 对 `d tau = dt*sqrt(1-beta^2)` 做离散累积，得到旅行者固有时曲线。  
5. `analytic_traveler_proper_time` 计算闭式参考值 `2*T_leg*sqrt(1-beta^2)`。  
6. `lorentz_gamma` 计算 `gamma`，用于输出和物理解释。  
7. `simultaneity_jump_at_turnaround` 计算 `t_sim_out`、`t_sim_in` 及跳跃 `Delta t_sim`，展示换向导致的 simultaneity 切片变化。  
8. `main` 汇总打印指标并执行四条断言（回程位置、固有时误差、年龄不等、跳跃正值），构成一次非交互可复验流程。  

这里没有把第三方库当黑箱：`numpy` 只用于数组和向量化计算，世界线离散、固有时积分、同时性跳跃公式均在源码中显式实现。
