# 量子隐形传态 (Quantum Teleportation)

- UID: `PHYS-0233`
- 学科: `物理`
- 分类: `量子信息`
- 源序号: `234`
- 目标目录: `Algorithms/物理-量子信息-0234-量子隐形传态_(Quantum_Teleportation)`

## R01

量子隐形传态（Quantum Teleportation）是把未知单量子比特态
`|psi> = alpha|0> + beta|1>`
从 Alice 传到 Bob 的协议。关键点是：

- 不直接传输量子态载体本身；
- 需要预共享纠缠（Bell 对）；
- 只发送 2 比特经典信息；
- 原始量子态在 Alice 侧测量后不再保留（符合不可克隆定理）。

本条目给出一个“可运行、可校验、无黑盒量子 SDK”的最小数值实现。

## R02

目标是构建一个三量子比特状态向量 MVP，完整覆盖协议闭环：

1. 准备未知态 `q0`；
2. 在 `q1-q2` 上制备 Bell 态；
3. Alice 做 Bell 基测量（`CNOT(q0->q1)` + `H(q0)` + 测量）；
4. Bob 根据 `(m0, m1)` 执行 `Z^m0 X^m1`；
5. 用保真度验证 Bob 最终态是否恢复为输入 `|psi>`。

## R03

`demo.py` 的建模方式：

- 使用 `numpy` 维护 3-qubit 纯态向量（维度 `2^3 = 8`）；
- 量子位顺序约定为 `q0 q1 q2`（`q0` 为最高位）；
- 单比特门通过 Kronecker 积扩展到全系统；
- `CNOT` 通过基矢索引置换显式实现；
- 测量通过对 `(m0,m1)` 分支做条件归一化得到 Bob 条件态。

这使每一步物理含义都能在源码中直接追踪。

## R04

核心公式如下：

1. 未知态参数化：
`|psi(theta,phi)> = cos(theta/2)|0> + exp(i*phi)sin(theta/2)|1>`

2. Bell 对：
`|Phi+> = (|00> + |11>) / sqrt(2)`

3. 传态测量后 Bob 条件态（标准结论）：
`|psi_B(m0,m1)> = X^m1 Z^m0 |psi>`

4. 校正操作：
`|psi_out> = Z^m0 X^m1 |psi_B(m0,m1)> = |psi>`（相差全局相位等价）

5. 保真度：
`F = |<psi_ref|psi_candidate>|^2`

## R05

复杂度（推广到 `n` qubit 表示更有意义）：

- 单比特门扩展并作用：`O(2^n)`（实现上构造 `2^n x 2^n` 线性算子）；
- CNOT（基矢重排）：`O(2^n)`；
- 固定 3 qubit、4 个测量分支：常数规模；
- 空间复杂度：`O(2^n)`。

在本 MVP 中 `n=3`，所以运行时间极小，适合教学和回归测试。

## R06

脚本默认执行两层验证：

- **确定性测试**：对 5 个代表态（`|0>`, `|1>`, `|+>`, `|->`, complex）枚举全部 4 个测量分支；
- **随机测试**：固定随机种子，从 Bloch 球采样 128 个随机态并逐分支验证。

输出包括：

- 每个分支的概率；
- 校正前/后的保真度；
- 诊断指标（概率和误差、均匀性误差、最大保真损失）；
- 全部断言通过后输出 `All checks passed.`。

## R07

优点：

- 无黑箱量子框架，门操作和测量逻辑完全显式；
- 结果可复现（固定参数与随机种子）；
- 既验证“能传”，也验证“概率分布和物理一致性”。

局限：

- 仅做无噪声纯态传态，不含退相干与门误差；
- 不涉及密度矩阵、混态与纠错码；
- 未包含实验控制脉冲层或硬件拓扑约束。

## R09

## R10

正确性直觉：

1. `q1-q2` 的 Bell 对提供“量子相关性通道”；
2. Alice 的 Bell 基测量把未知态信息转移到 Bob 的条件态中；
3. 两个经典比特 `(m0,m1)` 指明了 Bob 条件态相对目标态差了哪组 Pauli；
4. Bob 施加 `Z^m0 X^m1` 后，分支依赖被消去，最终态回到原态；
5. 若对所有分支与随机态都满足 `F≈1`，说明实现链路自洽。

## R11

数值稳定与工程细节：

- 全部使用 `complex128`；
- 关键步骤做归一化并检查范数；
- 用 `EPS=1e-12` 过滤近零概率分支异常；
- 断言阈值采用 `1e-11 ~ 1e-12`，保证严格但可通过浮点误差。

此外，测量概率总和与 1 的偏差单独监控，防止隐式归一化错误。

## R12

关键参数：

- `state_specs`：固定测试态集合（覆盖实态、相位态与复振幅态）；
- `n_random_states`：随机态压力测试数量；
- `random_seed`：保证复现实验；
- 断言阈值：概率均匀性、总概率守恒、保真度下界。

调参建议：

- 做教学演示可保留默认参数；
- 做更强压力测试可增大 `n_random_states` 到 `1e3` 以上；
- 如引入噪声扩展，需要相应放宽保真度阈值并改为统计检验。

## R13

该任务不是近似优化问题，因此无“近似比”。

可给出的工程保证：

- **确定性保证**：默认 5 个代表态的 4 个分支均达到 `fidelity_after ≈ 1`；
- **统计保证**：默认 128 个随机态全部通过高保真断言；
- **概率一致性保证**：每个输入态分支概率和为 1，且理想情形下各分支为 `1/4`。

## R14

常见失效模式：

1. 量子位索引约定混乱（大小端或位序写反）导致校正映射错误；
2. CNOT 控制位/目标位写反，Bell 态或 Bell 测量失真；
3. 条件态提取索引错误，导致概率不守恒；
4. 漏掉归一化，保真度出现系统性偏差；
5. 错把全局相位差当成失败（应以保真度判断）。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `TeleportationConfig` 定义固定测试态、随机测试规模和随机种子。  
2. `bloch_state` 按 `(theta, phi)` 构造未知输入态 `|psi>`。  
3. `build_pre_measurement_state` 依次执行 Bell 对制备与 Alice 的 Bell 测量前门序列，得到 3-qubit 总态。  
4. `conditional_bob_state` 按 `(m0,m1)` 从总态中抽取 Bob 条件态并计算该分支概率。  
5. `apply_bob_correction` 根据经典比特执行 `Z^m0 X^m1` 校正。  
6. `state_fidelity` 计算校正前/后的纯态保真度，量化恢复质量。  
7. `run_mvp` 汇总确定性分支表，并对随机态做批量鲁棒性检查，输出 `pandas` 报告与诊断字典。  
8. `run_checks` 对概率守恒、分支均匀性、保真度阈值做断言，`main` 打印结果并给出 `All checks passed.`。  

说明：`numpy/pandas` 仅用于线性代数与结果表格化；核心传态逻辑（门操作、测量分支、Pauli 校正、保真度校验）都在源码中显式实现，不依赖黑盒量子算法调用。
