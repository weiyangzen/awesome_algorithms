## R01

光学谐振腔（Optical Cavity）是由反射元件（通常两面镜子）构成的闭合光学反馈结构。  
只有满足相位闭合条件的纵模频率才会在腔内被增强，形成离散共振谱。

对最常见的两镜法布里-珀罗（Fabry-Perot）腔，可抓住三条主线：

- 几何稳定性：光线往返后是否束缚在腔内（`0 < g1*g2 < 1`）；
- 模场自洽性：高斯光束复参数 `q` 在一次往返后保持不变；
- 频谱选择性：自由光谱区 `FSR`、精细度 `F` 与线宽 `Delta nu`。

## R02

该算法在激光物理与精密测量中都属于基础模块：

- 激光器设计：决定是否能稳定支持 TEM00 等横模；
- 频率参考：窄线宽腔用于激光稳频、腔增强光谱；
- 光学系统耦合：模式匹配需要知道腰斑位置/大小与镜面光斑。

本条目不是做“大型光学仿真平台”，而是提供一个可运行、可验证、可追溯的最小实现。

## R03

`demo.py` 的 MVP 范围：

- 用 ABCD 矩阵计算两镜腔一轮往返矩阵；
- 用 `g` 参数判定稳定区；
- 解自洽方程 `q = (Aq+B)/(Cq+D)`，得到高斯模参数；
- 计算 `FSR`、近似精细度、理论线宽；
- 计算一段传输谱并做 FWHM 数值估计；
- 同时给出一个稳定腔和一个不稳定腔，脚本内置断言验证。

## R04

核心公式（两镜腔，腔长 `L`，镜曲率半径 `R1,R2`）：

1. `g` 参数  
   `g1 = 1 - L/R1`, `g2 = 1 - L/R2`（平面镜 `R=inf -> g=1`）  
   稳定条件：`0 < g1*g2 < 1`。

2. ABCD 传输  
   传播矩阵：`P(L) = [[1, L], [0, 1]]`  
   球面镜反射矩阵：`M(R) = [[1, 0], [-2/R, 1]]`  
   一次往返：`M_rt = M1 * P(L) * M2 * P(L) = [[A,B],[C,D]]`。

3. 高斯模自洽  
   `q = (Aq+B)/(Cq+D)`  
   等价二次方程：`C q^2 + (D-A) q - B = 0`，选 `Im(q) > 0` 的根。

4. 光斑与腰斑  
   `1/q = 1/R_wavefront - i * lambda/(pi*w^2)`  
   `w = sqrt(-lambda/(pi*Im(1/q)))`  
   若 `q = z + i z_R`，则 `w0 = sqrt(lambda*z_R/pi)`，腰斑位置 `z0 = -Re(q)`（相对镜1参考面）。

5. 频谱量  
   `FSR = c/(2L)`  
   `F ~= pi*(R1*R2)^(1/4) / (1 - sqrt(R1*R2))`（高反射近似）  
   `Delta nu ~= FSR/F`。

## R05

设传输谱采样点数为 `N`：

- 稳定性判定、`q` 求解、派生参数计算：`O(1)`；

- 总体时间复杂度：`O(N)`，空间复杂度：`O(N)`（保存频率与透过率数组）。

## R07

优点：

- 把“几何稳定 + 高斯模 + 频谱线宽”放在同一最小脚本里闭环；
- 不依赖黑盒光学仿真器，计算链路透明；

## R10

正确性直觉：

1. `g1*g2` 落在 `(0,1)` 时，光线在往返中不会指数发散。  
2. 稳定腔的往返 ABCD 必然允许 `Im(q)>0` 的自洽根，对应有限光斑。  
3. 反射率越高，腔内光子寿命越长，表现为更高精细度和更窄线宽。  
4. 频谱上单峰半高全宽（FWHM）应接近 `FSR/F`，这是 demo 的关键数值闭环检验。

## R11

数值与工程细节：

- 平面镜通过 `R=inf` 特判，避免除零；
- `q` 根选择时强制 `Im(q)>0`，过滤非物理解；
- FWHM 用线性插值估计过半点，减少采样栅格误差；
- 对断言使用相对误差阈值（而非苛求机器精度）。

## R12

关键参数（`CavityConfig`）：

- `length_m`：腔长，直接决定 `FSR = c/(2L)`；
- `wavelength_m`：工作波长，决定 `q` 到光斑半径映射；
- `mirror1_radius_m` / `mirror2_radius_m`：镜曲率，决定稳定区与模大小；
- `reflectivity1` / `reflectivity2`：镜反射率，决定精细度与线宽；
- `spectrum_points` / `spectrum_span_fsr`：谱采样密度与跨度，影响 FWHM 估计精度。

调参建议：若线宽估计抖动，优先增大 `spectrum_points`。

## R14

常见失效模式：

1. 把镜面反射矩阵中的 `-2/R` 号或系数写错，导致稳定性判断偏差；
2. `q` 根选错（取了 `Im(q)<0`），得到虚假光斑；
3. 反射率接近 1 且采样点过少，FWHM 无法可靠定位；
4. 单位混用（米、毫米、纳米），使 `FSR`/光斑数量级异常；
5. 用不稳定腔参数却强行解释为有限束腰。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `main` 构造两个配置：一个稳定两镜腔、一个不稳定两镜腔，并逐个调用 `analyze_cavity`。  
2. `roundtrip_abcd` 通过镜面矩阵与传播矩阵相乘，得到一次往返 `[[A,B],[C,D]]`。  
3. `compute_g_parameters` 计算 `g1,g2,g_product`，并结合 `trace_half=(A+D)/2` 判断几何稳定性。  
4. 对稳定腔，`solve_self_consistent_q` 解二次方程 `C q^2 + (D-A) q - B = 0`，选 `Im(q)>0` 根。  
5. `gaussian_beam_metrics_from_q` 把 `q` 转成可观测量：镜面光斑、腰斑半径、腰斑位置。  
6. `transmission_spectrum` 按 Airy 公式生成一段频率扫描的透过率曲线，并由 `estimate_fwhm` 取半高全宽。  
7. `analyze_cavity` 汇总 `FSR`、精细度、理论线宽与数值 FWHM，打包为 `CavitySummary`。  
8. `main` 用 `pandas` 打印结果表并执行断言：稳定/不稳定分类正确，稳定腔参数有限，且数值线宽与理论线宽相符。  

第三方库说明：`numpy` 仅用于矩阵和向量数值运算，`pandas` 仅用于结果表格展示；核心算法（ABCD 链、`q` 求解、谱线与 FWHM）均在源码中显式实现，没有调用黑盒光学求解器。
