## R01

核技巧（Kernel Trick）解决的问题是：
当数据在原始特征空间中线性不可分时，如何在不显式构造高维特征的前提下，仍然使用线性模型的优化框架得到非线性决策边界。

本目录 MVP 用一个可复现实验展示三件事：

- 线性模型在非线性数据上的性能上限；
- 核化模型（RBF SVM）如何显著提升分类效果；
- 自实现的对偶核岭分类如何直接体现“只用核函数，不显式映射”的计算流程。

## R02

核技巧的核心替换是把内积
`<phi(x_i), phi(x_j)>`
替换为核函数
`k(x_i, x_j)`。

对偶形式中，预测函数写为：
`f(x) = sum_i alpha_i * k(x_i, x)`。

因此只要能计算样本间核值，就能在隐式高维空间完成学习和预测，而不需要显式写出 `phi(x)`。

## R03

`demo.py` 使用 `make_circles` 构造经典线性不可分二分类数据：

- 总样本数：`1200`
- 同心圆系数：`factor=0.45`
- 噪声：`0.08`
- 划分：`70% train / 30% test`（分层抽样）
- 标准化：`StandardScaler` 仅在训练集拟合

这类数据能稳定触发“线性边界不够用”的现象，适合演示核技巧价值。

## R04

本 MVP 使用两类核：

- 多项式核：`k(x,z)=(x·z+1)^2`（用于验证显式映射等价性）
- RBF 高斯核：`k(x,z)=exp(-gamma ||x-z||^2)`（用于实际分类）

其中 RBF 核对应无限维特征空间的隐式映射，通常对非线性边界更灵活。

## R05

整体流程：

1. 生成并标准化同心圆数据；
2. 用二次多项式显式特征映射验证核矩阵等价；
3. 训练线性 SVM 作为 baseline；
4. 训练 RBF SVM 作为核化模型；
5. 自实现对偶核岭分类（RBF 核）；
6. 汇总三个模型测试准确率并执行质量断言。

## R06

脚本中的 `poly2_explicit_feature_map` 给出了二维输入下
`(x·z+1)^2`
对应的一组显式特征：
`[1, sqrt(2)x1, sqrt(2)x2, x1^2, sqrt(2)x1x2, x2^2]`。

随后比较：

- `K_explicit = Phi Phi^T`
- `K_poly = polynomial_kernel(X, X)`

并断言二者最大绝对误差小于 `1e-10`，直接证明“显式映射内积 = 核函数值”。

## R07

基线模型与核化模型都用 `sklearn.svm.SVC`：

- 线性基线：`kernel="linear", C=1.0`
- RBF 核模型：`kernel="rbf", C=2.0, gamma=2.0`

这两个设置唯一关键差异是核类型，便于把性能增益归因到核技巧本身。

## R08

为避免把第三方库当黑箱，`demo.py` 还实现了最小对偶核岭分类：

- 训练解：`alpha = (K + lambda I)^(-1) y`，其中 `y in {-1,+1}`；
- 预测分数：`f(x)=sum_i alpha_i k(x_i,x)`；
- 分类：`f(x) >= 0` 判为正类。

这个实现只依赖 `numpy` 线性代数，完整暴露了核矩阵构造与对偶推断细节。

## R09

复杂度分析（训练集规模 `n`）：

- 核矩阵构建：`O(n^2 d)`（`d` 为原始特征维度）
- 核岭线性求解：`O(n^3)`（稠密求解）
- 单样本预测：`O(n)`（需对所有训练样本求核）

空间复杂度主要由核矩阵决定，为 `O(n^2)`。这也是核方法在大样本下的主要瓶颈。

## R10

脚本输出：

- 数据规模（train/test）；
- 多项式显式映射与核矩阵误差；
- 三个模型的 `test_accuracy` 排名表：
  - `Linear SVM`
  - `RBF SVM`
  - `Custom Dual Kernel Ridge`

## R12

`demo.py` 主要函数分工：

- `polynomial_kernel` / `rbf_kernel`：核函数计算；
- `poly2_explicit_feature_map`：二次多项式显式特征映射；
- `kernel_ridge_fit`：构造核矩阵并求解对偶系数 `alpha`；
- `kernel_ridge_decision` / `kernel_ridge_predict`：对偶预测；
- `main`：组织数据、训练、评估、汇总与断言。

## R13

质量门禁（断言）包含：

1. 显式映射与多项式核矩阵误差 `< 1e-10`；
2. `RBF SVM` 测试准确率至少比线性 SVM 高 `0.10`；
3. `RBF SVM` 与自实现核岭分类准确率均 `> 0.90`；
4. 二者准确率差值 `< 0.08`，避免“自实现严重失真”。

这些门禁保证“能跑”之外还有“结果可信、教学结论成立”。

## R14

常见失败模式与处理：

- 失败 1：`gamma` 过大导致过拟合。  
  处理：减小 `gamma` 或增大正则项。

- 失败 2：`gamma` 过小导致欠拟合。  
  处理：增大 `gamma`，让局部非线性能力增强。

- 失败 3：未标准化导致距离尺度畸变。  
  处理：保持 `StandardScaler` 训练集拟合策略。

- 失败 4：核矩阵近奇异导致数值不稳。  
  处理：增大 `reg`（`lambda`）稳定求解。

## R15

与相关方法的关系：

- 与线性模型：核技巧把线性模型扩展到非线性边界；
- 与显式特征工程：核技巧避免手工构造高维特征，直接算核值；
- 与神经网络：两者都能学非线性，核方法在小中样本和可解释对偶形式中常更直接。

## R16

可扩展方向：

- 用 `GridSearchCV` 搜索 `C/gamma/reg`；
- 增加多项式核、Sigmoid 核的并行比较；
- 用 Nyström 或随机特征近似降低 `O(n^2)` 存储成本；
- 在多分类数据上扩展 one-vs-rest/one-vs-one 实验。

## R17

本 MVP 的边界与假设：

- 任务是二维 toy 数据二分类，主要用于机制演示；
- 不追求大规模数据下的工程最优实现；
- 自实现的是核岭分类，不等同于完整软间隔 SVM 的优化过程；
- 结论聚焦“核化后可分性提升”，而非所有数据集都必然显著提升。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 调用 `make_circles` 生成非线性数据，并用 `train_test_split + StandardScaler` 完成训练/测试数据准备。  
2. 取训练子集 `x_probe`，通过 `poly2_explicit_feature_map` 构造显式特征 `Phi`，计算 `K_explicit = Phi Phi^T`。  
3. 同时调用 `polynomial_kernel(x_probe, x_probe)` 得到 `K_poly`，并计算两者最大误差，验证核技巧恒等式。  
4. 训练 `linear_svm = SVC(kernel="linear")`，在测试集得到线性基线准确率。  
5. 训练 `rbf_svm = SVC(kernel="rbf")`，在同一测试集得到核化 SVM 准确率。  
6. 将标签转成 `{-1,+1}`，在 `kernel_ridge_fit` 中先算 `K_train`，再求解 `(K + lambda I) alpha = y` 得到对偶系数 `alpha`。  
7. 在 `kernel_ridge_predict` 中计算 `K(test, train) @ alpha` 并做符号判别，得到自实现核分类预测。  
8. 用 `pandas` 汇总三种模型结果，执行断言门禁（精度差、阈值、等价性）并输出最终结论。

以上 8 步完整呈现了核技巧的工程闭环：
从核函数定义、核矩阵构造、对偶求解到预测与验证，全程无需显式构造高维特征。
