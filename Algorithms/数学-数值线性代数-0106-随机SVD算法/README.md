# 随机SVD算法

- UID: `MATH-0106`
- 学科: `数学`
- 分类: `数值线性代数`

- 目标目录: `Algorithms/数学-数值线性代数-0106-随机SVD算法`

## R01

本条目实现随机 SVD（Randomized SVD）的最小可运行版本（MVP），用于在低秩近似场景下替代完整 SVD。

核心思想是：
- 先用随机投影快速抓取矩阵主子空间；
- 再在小矩阵上做精确 SVD；
- 最后映射回原空间得到近似奇异三元组。

实现目标是“流程透明、可复现、可运行”，而不是封装成大型框架。

## R02

问题定义（截断分解版）：
- 输入：
  - 矩阵 `A ∈ R^(m×n)`；
  - 目标秩 `k`；
  - 超采样参数 `p`；
  - 幂迭代次数 `q`。
- 输出：
  - `U_k, S_k, V_k^T`，满足 `A ≈ U_k diag(S_k) V_k^T`；
  - 误差评估（与精确截断 SVD 对比）。

前提：`1 <= k <= min(m, n)`，且输入矩阵元素为有限实数。

## R03

方法原理（范围查找 + 小矩阵分解）：
- 随机范围查找：构造高斯随机矩阵 `Ω`，计算 `Y = AΩ`，使 `Y` 的列空间逼近 `A` 的主子空间；
- 幂迭代增强：做 `Y <- A(A^T Y)` 若干次，提升谱间隔较小场景下的近似质量；
- 正交化：对 `Y` 做 QR，得正交基 `Q`；
- 降维分解：对 `B = Q^T A` 做标准 SVD，最后 `U = Q U_B`。

这样把大矩阵分解问题转成了更小维度的精确分解问题。

## R04

算法总览：
1. 校验输入矩阵维度、有限性与参数范围；
2. 设草图维度 `l = min(k + p, min(m,n))`；
3. 采样 `Ω ~ N(0,1)^(n×l)`，计算 `Y = AΩ`；
4. 执行 `q` 轮幂迭代 `Y = A(A^T Y)`；
5. 对 `Y` 做约化 QR，得到 `Q`；
6. 形成小矩阵 `B = Q^T A`；
7. 对 `B` 做精确 SVD：`B = U_B S V^T`；
8. 回升到原空间：`U = Q U_B`，截断前 `k` 个奇异分量。

## R05

核心数据结构：
- `RandomizedSVDResult`（`dataclass`）：
  - `U, S, Vt`：截断后的分解结果；
  - `m, n`：原矩阵形状；
  - `rank`：目标秩 `k`；
  - `target_dim`：草图维度 `l`；
  - `n_iter`：幂迭代次数 `q`。

该结构让结果与关键超参数绑定，便于复现实验。

## R06

正确性要点：
- `Q` 的列空间近似覆盖 `A` 的主右奇异子空间映射后的像空间；
- `B = Q^T A` 在低维子空间内保留了 `A` 的主要谱信息；
- 对 `B` 做精确 SVD 再左乘 `Q`，可得到 `A` 的低秩近似奇异向量；
- 当 `p` 合理且 `q` 适当增加时，近似质量会更稳定。

本实现不改变随机 SVD 的标准数学流程，仅做必要工程化封装。

## R07

复杂度分析（`A` 为 `m×n`，`l=k+p`）：
- 时间复杂度（主项）：
  - 随机投影与幂迭代约为 `O((q+1) m n l)`；
  - QR 约为 `O(m l^2)`；
  - 小矩阵 SVD 约为 `O(l^2 n)`。
- 空间复杂度：
  - 主要为 `Y/Q`、`B` 和结果矩阵，约 `O((m+n)l)`。

当 `l << min(m,n)` 时，通常比完整 SVD 更省时。

## R08

边界与异常处理：
- 非二维输入：抛出 `ValueError`；
- 空矩阵或含 `NaN/Inf`：抛出 `ValueError`；
- `rank` 越界：抛出 `ValueError`；
- `oversample < 0` 或 `n_iter < 0`：抛出 `ValueError`；
- 参数非整数（含布尔值）：抛出 `TypeError`。

这些检查都在算法入口处执行，避免静默失败。

## R10

`demo.py` 函数职责：

- `randomized_svd`：随机 SVD 主过程；
- `reconstruct`：由 `U,S,Vt` 重构近似矩阵；
- `relative_fro_error`：计算相对 Frobenius 误差；

## R12

输出解读：
- `relative Fro error (rand)`：随机 SVD 重构误差；
- `relative Fro error (best)`：精确截断 SVD 的最优误差（理论下界基准）；
- `error ratio rand/best`：随机解相对最优解的误差倍率，越接近 `1` 越好；
- `top-k singular rel error`：前 `k` 个奇异值相对误差；
- `captured energy ratio`：捕获能量比例 `sum(S_k^2)/sum(S_all^2)`。

## R13

最小测试集建议：

## R14

可调参数：
- `rank`：目标秩，控制压缩率与精度；
- `oversample`：超采样维度，常见取值 `5~20`；
- `n_iter`：幂迭代次数，常见取值 `0~3`；
- `seed`：随机种子，控制可复现性；
- `build_demo_matrix` 中 `latent_rank/noise_std`：控制任务难度。

工程实践里可先固定 `k`，再调 `p,q` 平衡速度与误差。

## R15

方法对比：
- 对比完整 SVD：
  - 完整 SVD 精度最优但成本高；
  - 随机 SVD 在低秩目标下速度更优、精度可接受。
- 对比幂法/子空间迭代：
  - 随机 SVD 初始化更直接，通常实现更简洁；
  - 在奇异值间隔小场景下需要幂迭代增强。
- 对比黑盒库调用：
  - 黑盒调用短，但不利于理解内部流程；
  - 本条目保留每个步骤以便教学与审计。

## R16

## R18

源码级算法流（对应 `demo.py`，8 步）：
1. `main` 先调用 `build_demo_matrix` 构造近低秩矩阵 `A`，设置 `k,p,q,seed`。  
2. `main` 调用 `randomized_svd(A, rank=k, oversample=p, n_iter=q, seed=seed)` 进入核心流程。  
3. `randomized_svd` 用 `_as_finite_2d_array` 做输入校验，并计算草图维度 `l=min(k+p,min(m,n))`。  
4. 生成高斯随机矩阵 `Ω`，执行随机投影 `Y = AΩ`。  
5. 循环执行幂迭代 `Y = A(A^T Y)` 共 `q` 轮，提升主子空间分离度。  
6. 对 `Y` 做约化 QR 得到 `Q`，再形成小矩阵 `B = Q^T A`。  
7. 对 `B` 做精确 SVD 得 `U_B,S,V^T`，并回升 `U = Q U_B`，截断前 `k` 项返回 `RandomizedSVDResult`。  
8. `main` 用 `reconstruct` 重构近似矩阵，并与精确截断 SVD 比较误差后打印指标。
