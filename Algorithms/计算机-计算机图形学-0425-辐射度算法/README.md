## R01

辐射度（Radiosity）是面向漫反射（Lambertian）场景的全局光照算法。它把场景离散成若干面片，求每个面片单位面积向外辐射的总能量 `B`，从而把“多次反弹”的间接光照显式纳入计算。

核心思想不是追踪单条光线，而是求解一个能量平衡方程组。

## R02

本条目 MVP 的问题定义：

1. 输入：一组面片（中心点、法向、面积、反射率 `rho`、自发光 `E`）；
2. 输入：面片间形状因子矩阵 `F`（由几何近似计算得到）；
3. 输出：每个面片 RGB 辐射度 `B`；
4. 输出：迭代解与直接线性解的一致性误差、收敛统计与稳定性指标。

## R03

离散辐射度方程（逐面片、逐颜色通道）可写为：

- `B = E + R * (F @ B)`
- 其中 `R` 是按面片反射率组成的对角算子（在代码中对 RGB 做逐元素乘法）。

等价线性系统（每个颜色通道单独求解）：

- `(I - diag(rho) * F) * B = E`

这就是 `demo.py` 中“迭代法 + 直接法”的共同目标方程。

## R04

MVP 采用的形状因子近似（无可见性遮挡项）：

- `F_ij ≈ (A_j * cos(theta_i) * cos(theta_j)) / (pi * r^2)`，当 `cos(theta_i)>0` 且 `cos(theta_j)>0` 时有效；
- `F_ii = 0`；
- 对每行做上限缩放，使 `sum_j F_ij <= 0.92`，留出能量泄露给“未建模环境”，也保证迭代收敛裕度。

这是教学/验证导向的近似，不是工业渲染级可见性积分。

## R05

`demo.py` 的核心数据结构：

- `Patch`：面片参数（几何 + 材质 + 发光）；
- `F (ndarray[n,n])`：面片到面片的能量耦合；
- `rho_rgb (ndarray[n,3])` / `emit_rgb (ndarray[n,3])`：反射率与自发光；
- `detail_df`：逐面片输出（`B_iter`、`B_direct`、误差、行和）；
- `summary_df`：全局统计（迭代次数、谱半径、残差等）。

## R06

正确性直觉：

1. 每个面片“出射能量 = 自发光 + 接收能量×反射率”；
2. 接收能量由其他面片辐射度经 `F` 线性叠加；
3. 因此整个场景构成一个耦合线性系统；
4. 当迭代算子谱半径 `< 1` 时，固定点迭代会收敛到唯一物理解。

`demo.py` 显式检查谱半径与固定点残差来支撑该直觉。

## R07

设面片数为 `n`，迭代轮数为 `k`：

1. 构建 `F`：`O(n^2)` 时间，`O(n^2)` 空间；
2. 每轮迭代 `F @ B`：`O(n^2)`；总迭代 `O(k n^2)`；
3. 直接解（每通道一次稠密线性求解）：`O(n^3)`，RGB 共常数倍。

MVP 中 `n=6`，规模很小，重点是流程正确性而非性能。

## R08

数值与稳定性处理：

1. 使用 `EPS=1e-12` 防止零长度向量和除零；
2. 形状因子行和限制到 `<1`，降低发散风险；
3. 检查迭代算子最大谱半径 `spectral_radius_max < 1`；
4. 通过 `||B - (E + RFB)||_inf` 验证固定点残差；
5. 迭代解与直接解做 `max_abs_err` 对照，作为实现自检。

## R09

MVP 取舍：

1. 只做漫反射辐射度，不处理镜面高光与 BRDF 复杂项；
2. 用几何闭式近似代替半立方体/蒙特卡洛积分；
3. 不实现遮挡可见性（无遮挡假设）；
4. 但保留“方程求解 + 校验链路”的完整主干。

这样可以在小代码量下真实体现辐射度算法核心。

## R10

主要函数职责：

1. `build_scene`：构造 6 个面片（顶灯 + 地面 + 四面墙）；
2. `build_form_factor_matrix`：按几何关系生成 `F` 并做行归一上限控制；
3. `iterate_radiosity`：Jacobi 固定点迭代；
4. `solve_radiosity_direct`：每个颜色通道解线性系统；
5. `spectral_radius_max`：估计收敛性指标；
6. `run_experiment`：汇总表格并执行断言门禁；
7. `main`：打印矩阵、逐面片结果、汇总结果。

## R12

输出解释：

1. `Form-factor matrix F`：面片间能量耦合矩阵（四舍五入显示）；
2. `Per-patch radiosity`：
- `emit_*`：初始发光；
- `B_iter_*`：迭代法收敛结果；
- `max_abs_err`：与直接解每面片最大通道误差；
- `row_sum_F`：该面片能量外送比例上限。
3. `Summary`：
- `iterations`、`final_delta`：迭代收敛信息；
- `spectral_radius_max`：收敛性关键指标；
- `fixed_point_residual_inf`：方程残差；
- `non_emissive_patch_max_bounce`：非发光面片获得的最大间接光。

## R13

内置校验门禁：

1. `max(row_sum_F) < 0.999`；
2. `spectral_radius_max < 1.0`；
3. 迭代解与直接解误差 `<= 5e-7`；
4. 固定点残差 `<= 1e-8`；
5. 所有辐射度非负；
6. 至少一个非发光面片获得正的反弹能量。

全部通过后输出 `All checks passed.`。

## R14

实验配置（固定）：

1. 场景：6 面片（`ceiling_light`、`floor`、`left_wall`、`right_wall`、`back_wall`、`front_wall`）；
2. 顶灯发光：`[24,22,18]`；
3. 其余面片发光为 0，反射率按 RGB 给定；
4. 形状因子行和上限：`0.92`；
5. 迭代参数：`max_iters=4000`，`tol=1e-10`。

该配置可稳定展示“直接光 + 多次漫反射”效应。

## R15

与相近算法对比：

1. 与光线追踪：辐射度擅长漫反射全局能量平衡，不擅长镜面路径；
2. 与路径追踪：路径追踪更通用，但有随机噪声；辐射度在漫反射条件下更平滑确定；
3. 与有限元表述：辐射度本质就是一个离散能量传输线性系统。

## R16

适用范围与限制：

1. 适用：静态场景、漫反射主导、离线预计算；
2. 限制：
- 面片划分粗时会有块状伪影；
- 未建模可见性会高估能量交换；
- 对镜面/折射材质不直接适用。

## R17

可扩展方向：

1. 加入可见性项（射线遮挡测试）；
2. 用半立方体法或 Monte Carlo 估计更可靠的 `F`；
3. 自适应细分高梯度面片；
4. 使用稀疏矩阵与分块迭代提升规模；
5. 与路径追踪混合（漫反射用辐射度，镜面用路径方法）。

## R18

`demo.py` 的源码级算法流程（8 步，非黑盒）：

1. `main()` 调用 `build_scene()` 构造 6 个面片，明确每个面片的中心、法向、面积、反射率与发光。  
2. `run_experiment()` 调用 `build_form_factor_matrix()`，逐对面片计算 `r`、`cos(theta_i)`、`cos(theta_j)` 并据公式生成 `F_ij`。  
3. 在 `build_form_factor_matrix()` 内把 `F_ii` 置零，并对行和超限行进行缩放，确保能量传输矩阵满足收敛性需求。  
4. `iterate_radiosity()` 从 `B0=E` 开始，按 `B_{k+1}=E + rho*(F@B_k)` 迭代，记录每轮最大改变量直到小于容差。  
5. `solve_radiosity_direct()` 对 RGB 三个通道分别构造 `(I - diag(rho_c)F)` 并用 `scipy.linalg.solve` 求线性系统解。  
6. `run_experiment()` 比较迭代解与直接解，计算 `max_abs_err`、固定点残差和谱半径，并执行门禁断言。  
7. 结果被整理成 `detail_df`（逐面片）和 `summary_df`（全局指标），同时输出 `ff_df` 便于检查形状因子矩阵。  
8. `main()` 打印矩阵、逐面片辐射度和汇总统计，全部断言通过后输出 `All checks passed.`。
