# 电磁辐射的相对论变换 (Relativistic Transformation of EM Fields)

- UID: `PHYS-0183`
- 学科: `物理`
- 分类: `相对论电动力学`
- 源序号: `184`
- 目标目录: `Algorithms/物理-相对论电动力学-0184-电磁辐射的相对论变换_(Relativistic_Transformation_of_EM_Fields)`

## R01

电磁辐射的相对论变换关注：当观测者从惯性系 `S` 切换到以速度 `v` 运动的惯性系 `S'` 时，电场 `E` 与磁场 `B` 如何按洛伦兹变换重组。

在 SI 制下，常用矢量形式为：

- `E' = gamma * (E + v x B) - ((gamma - 1)/|v|^2) * v * (v·E)`
- `B' = gamma * (B - (v x E)/c^2) - ((gamma - 1)/|v|^2) * v * (v·B)`

其中 `gamma = 1/sqrt(1-|v|^2/c^2)`，`c` 为真空光速。

## R02

该变换是狭义相对论电动力学的基础接口，直接关系到：

- 不同参考系下辐射场强度与方向的观测差异（多普勒与像差背景）；
- 场不变量 `E^2-c^2B^2` 与 `E·B` 的一致性；
- 粒子加速器、天体喷流、电磁诊断等场景中的跨系量测换算。

## R03

本条目 MVP 的任务：

- 输入：一组 `E, B` 与参考系相对速度 `v`（可单样本或批量）；
- 输出：变换后的 `E', B'`，以及关键不变量检验结果；
- 验证：对“平面电磁波”“纯电场”“随机场批量”给出可复核数值例子。

目标是构建最小可运行、公式透明、含自检断言的实现，而非完整协变张量框架。

## R04

MVP 采用显式公式与向量化计算：

1. 计算 `beta^2 = |v|^2/c^2` 与 `gamma = 1/sqrt(1-beta^2)`；
2. 计算 `v x B`、`v x E`、`v·E`、`v·B`；
3. 代入洛伦兹场变换公式得到 `E'` 与 `B'`；
4. 计算不变量 `I1 = E^2 - c^2 B^2`，`I2 = E·B`；
5. 比较变换前后 `I1, I2` 误差并断言在数值容忍度内。

## R05

设批量样本数为 `N`：

- 每个样本只涉及常数次 3 维点乘/叉乘，时间复杂度 `O(N)`；
- 额外空间主要是输入输出数组，空间复杂度 `O(N)`；
- 单样本模式可视作 `O(1)`。

## R06

`demo.py` 内置 4 个非交互场景：

- 场景 A：`N=256` 随机场 + 随机亚光速 boost，检查不变量守恒；
- 场景 B：真空平面波沿传播方向 boost，验证幅度缩放与 `|E'|=c|B'|`；
- 场景 C：纯电场在运动系诱导磁场，核对解析表达；
- 场景 D：先做一次 boost 再用 `-v` 反变换，验证可逆性。

## R07

优点：

- 公式直接落地为源码，便于教学和审计；
- 支持批量向量化，适合快速参数扫描；
- 断言覆盖不变量、平面波约束和逆变换一致性。

局限：

- 只处理经典连续场，不含量子电动力学效应；
- 不涉及场源方程（如麦克斯韦方程组求解）；
- 未引入时空事件坐标变换，仅做同一点的场量变换。

- 三维向量点乘/叉乘；
- 电磁场不变量含义。

运行依赖：

## R09

适用：

- 相对论电磁学入门验证；
- 代码中快速验证场变换和不变量是否实现正确；
- 作为更复杂协变电磁模块的单元测试基线。

不适用：

- 需要解含边界条件的全波电磁传播问题；
- 需要广义相对论曲率时空效应；
- 需要量子辐射修正或强场 QED 效应。

## R10

正确性直觉来自三点：

1. 使用的是标准狭义相对论场变换闭式公式；
2. 两个洛伦兹标量不变量 `I1, I2` 理论上应在任意惯性系保持不变；
3. 真空辐射特征 `E·B=0` 与 `E^2-c^2B^2=0` 在变换后仍应成立。

当代码满足上述约束并通过断言，说明实现与理论一致。

## R11

数值稳定策略：

- 显式检查 `|v| < c`，避免 `gamma` 出现奇异；
- 对 `|v|=0` 的情况将 `(gamma-1)/|v|^2` 系数置零，避免 `0/0`；
- 不变量误差用相对误差 `|delta|/max(1,|ref|)` 衡量，避免小量分母放大噪声。

注意：当 `|v|` 极接近 `c` 时，`gamma` 很大，浮点误差会放大，工程上应限制 `beta` 上界。

## R12

关键参数与影响：

- `v/c`：越接近 1，场分量混合越强；
- `v` 与 `E,B` 的夹角：决定“平行分量保持、垂直分量混合”的程度；
- 场量级：影响绝对误差规模，但不改变无量纲守恒关系；
- 批量规模 `N`：只影响计算吞吐，不改变单样本公式。

## R13

本问题不是近似优化算法，因此无“近似比/随机成功率”叙述。

在模型与数值精度条件下可给出的保证是：

- 对每个样本，`E', B'` 按解析洛伦兹公式计算；
- 断言保证不变量在设定容差内守恒；
- 断言保证特例（平面波、逆变换）符合物理关系。

## R14

常见失效模式：

1. 单位混用（把 `B` 的单位误当为与 `E` 同量纲）；
2. 使用超光速或接近超光速 `v` 导致数值崩溃；
3. 叉乘顺序写反导致符号错误；
4. 将“场变换”误用于“场源演化”问题。

建议先检查单位和符号，再检查速度范围。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `_as_batch3` 把输入 `E,B` 统一成 `(n,3)`，同时保留单样本标记。  
2. `_broadcast_velocity` 把 `v` 统一广播到 `(n,3)`，并保证批量对齐。  
3. `lorentz_transform_fields` 计算 `beta^2`、`gamma`，并检查 `|v|<c`。  
4. 在同一函数内计算 `v x B`、`v x E`、`v·E`、`v·B`，代入显式公式得到 `E',B'`。  
5. `field_invariants` 计算 `I1=E^2-c^2B^2` 与 `I2=E·B`，用于理论约束校验。  
6. `run_random_invariance_demo` 批量生成随机场与随机速度，比较变换前后不变量误差并断言。  
7. `run_plane_wave_demo` 与 `run_pure_electric_demo` 分别验证“辐射场特性保持”和“运动诱导磁场”两类经典特例。  
8. `run_inverse_transform_demo` 用 `v` 与 `-v` 组合验证可逆性，`main` 串联所有场景并输出结果。  

这里没有把第三方库当黑箱算法：`numpy` 仅负责线性代数与向量化，核心相对论变换逻辑全部在源码中显式实现。
