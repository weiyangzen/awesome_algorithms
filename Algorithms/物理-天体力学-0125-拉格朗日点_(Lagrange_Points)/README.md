# 拉格朗日点 (Lagrange Points)

- UID: `PHYS-0125`
- 学科: `物理`
- 分类: `天体力学`
- 源序号: `125`
- 目标目录: `Algorithms/物理-天体力学-0125-拉格朗日点_(Lagrange_Points)`

## R01

拉格朗日点是圆限制性三体问题（CR3BP）中的平衡点：在随两主星一起旋转的参考系中，小质量粒子若位于这些点，可在离心力与两主星引力的共同作用下保持相对静止。

本 MVP 完成三件事：
- 计算 Earth-Moon 质量参数下的 `L1~L5` 归一化坐标；
- 计算每个点的 Jacobi 常数 `C=2*Omega`（静止速度假设）；
- 用平面线性化雅可比矩阵特征值判断线性稳定性。

## R02

问题定义（归一化 CR3BP）：
- 两主星总质量归一，距离归一，角速度归一；
- 质量参数 `mu = m2 / (m1 + m2)`，其中 `0 < mu < 0.5`；
- 主星坐标：`m1` 在 `(-mu, 0)`，`m2` 在 `(1-mu, 0)`。

目标：在旋转坐标系中求平衡点 `(x, y)`，满足 `dOmega/dx=0` 与 `dOmega/dy=0`。

## R03

为何选这个实现路径：
- 拉格朗日点属于天体力学最基础、最可验证的算法问题之一；
- `L4/L5` 有闭式解，而 `L1/L2/L3` 需要数值根求解，适合做“解析+数值”结合的最小可运行示例；
- 线性稳定性可通过 4x4 矩阵特征值直接审计，不依赖黑盒仿真器。

## R04

核心模型公式：

1. 有效势函数
`Omega(x,y)= (1-mu)/r1 + mu/r2 + 0.5*(x^2+y^2)`

其中：
`r1 = sqrt((x+mu)^2 + y^2)`，`r2 = sqrt((x-1+mu)^2 + y^2)`。

2. 共线点（一维）方程（`y=0`）：
`f(x)=x - (1-mu)*(x+mu)/|x+mu|^3 - mu*(x-1+mu)/|x-1+mu|^3 = 0`

3. 三角点闭式坐标：
`L4=(0.5-mu, +sqrt(3)/2)`，`L5=(0.5-mu, -sqrt(3)/2)`

4. Jacobi 常数（静止）
`C = 2*Omega(x,y)`。

## R05

算法流程：
1. 用 `m1,m2` 计算 `mu`，并验证 `0 < mu < 0.5`。
2. 对 `L1/L2/L3` 分别构造几何上可行的区间并求 `f(x)=0`。
3. 对 `L4/L5` 直接使用闭式公式。
4. 对每个拉格朗日点计算 `C=2*Omega`。
5. 计算 `Omega_xx, Omega_yy, Omega_xy`，组装 4x4 平面线性化矩阵。
6. 用特征值实部最大值判定线性稳定/不稳定。
7. 汇总为表格输出，便于人工核查。

## R06

正确性依据：
- 平衡点定义直接来自旋转系运动方程在 `vx=vy=ax=ay=0` 时的条件；
- `L1/L2/L3` 的分区求根利用了奇点分隔和符号变化，避免跨越不可导点；
- `L4/L5` 闭式解源自等边几何关系；
- 线性稳定性由线性化系统特征值决定：若存在正实部特征值，则局部指数发散。

## R07

复杂度：
- 共线点求根：每个点 `O(k)` 次函数评估，`k` 为 Brent 迭代次数（通常几十步内）；
- 单次函数评估常数复杂度，因此 `L1~L3` 总体约 `O(k)`；
- 稳定性分析：每点一次 `4x4` 特征值分解，常数复杂度；
- 总体时间/空间都非常小，适合作为教学级 MVP。

## R08

边界与异常处理：
- `m1,m2` 必须为有限正数；
- `mu` 必须位于 `(0,0.5)`；
- Brent 求根前严格检查区间端点符号相反；
- 若根求解不收敛，抛出明确异常；
- 对非有限函数值（极端参数）提前报错，避免静默错误。

## R09

MVP 取舍：
- 使用 `numpy + scipy + pandas`，避免构建重型仿真框架；
- 不做长时间轨道积分，仅做平衡点和局部稳定性判别；
- 输出以可读表格为主，不做图形 UI；
- 采用 Earth-Moon 示例保证结果直观可解释。

## R10

`demo.py` 函数职责：
- `mass_ratio`：由主星质量得到 `mu`。
- `collinear_equation`：定义共线点方程 `f(x)`。
- `solve_brent_root`：带区间合法性检查的 Brent 根求解。
- `effective_potential` / `jacobi_constant_at_rest`：计算有效势和 Jacobi 常数。
- `second_derivatives_omega`：计算二阶偏导。
- `planar_linearization_eigenvalues`：生成线性化矩阵并求特征值。
- `classify_stability`：按最大实部分类型。
- `solve_lagrange_points`：统一计算 `L1~L5` 全部结果。
- `to_table`：整理为 `pandas` 表格。
- `main`：执行示例并打印输出。

## R12

输出字段说明：
- `point`：拉格朗日点编号（L1-L5）。
- `x, y`：归一化旋转系坐标。
- `Jacobi_C`：该平衡点静止时 Jacobi 常数。
- `max_real_eig`：线性化矩阵特征值实部最大值。
- `stability`：`linearly stable` 或 `linearly unstable`。
- `root_iters`：Brent 迭代步数（L4/L5 为闭式解，记 0）。

## R13

最小验证项：
- 数值上 L4/L5 的 `x` 坐标应都接近 `0.5-mu`，`y` 互为正负；
- 对 Earth-Moon `mu≈0.01215`，L4/L5 应为线性稳定，L1/L2/L3 不稳定；
- 共线点应满足排序：`L3 < -mu < L1 < 1-mu < L2`；
- `root_iters` 应为有限正整数（L1-L3）。

## R14

关键参数建议：
- `bracket_eps`：避免贴近奇点，默认 `1e-6`；
- `root_xtol/root_rtol`：根求解精度与稳定性的折中；
- `m1,m2`：可替换为 Sun-Earth、Earth-Moon 等系统；
- 稳定性阈值 `tol`：默认 `1e-9`，用于识别数值噪声下的“近零实部”。

## R15

与常见替代方案对比：
- 对比纯解析：仅 `L4/L5` 可直接闭式，`L1/L2/L3` 仍需数值法；
- 对比牛顿法：牛顿法对初值更敏感，Brent 法在有符号区间时更稳健；
- 对比全轨道积分：轨道积分可研究非线性全局行为，但实现和计算成本更高，本任务先聚焦平衡点计算。

## R16

应用场景：
- 航天任务初步选址（如日地 L1、L2 任务；地月 L2 中继概念）；
- 课程教学中的三体平衡点数值实验；
- 任务早期参数扫掠（不同质量比下平衡点与稳定性变化）。

## R18

`demo.py` 的源码级算法流（8 步）：

1. `main` 构造 `CR3BPConfig(m1,m2,eps,tol)`，并调用 `validate_config` 做参数合法性检查。
2. `mass_ratio` 计算 `mu`，随后 `solve_lagrange_points` 定义共线方程 `f(x)`。
3. 对 `L1/L2/L3` 分别调用 `solve_brent_root`，每次先检查端点 `f(a), f(b)` 是否异号。
4. 进入 SciPy `root_scalar(..., method="brentq")`：内部维持一个“始终夹住根”的区间 `[a,b]`。
5. Brent 核心迭代会优先尝试逆二次插值/割线步；若候选点不安全或不收敛，则退回二分步，保证区间继续收缩。
6. 当满足 `xtol/rtol` 终止条件后返回根与迭代次数，得到 `L1/L2/L3` 的 `x` 坐标。
7. `L4/L5` 直接用闭式 `(0.5-mu, ±sqrt(3)/2)`；随后对每个点计算 `Jacobi_C` 与线性化矩阵特征值。
8. `classify_stability` 读取最大实部并标记稳定性，`to_table` 汇总结果，`main` 统一打印可审计表格与 `mu_crit` 对照结论。
