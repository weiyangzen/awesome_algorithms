## R01

本条目实现“辛几何算法”的最小可运行版本（MVP）：
- 以一维哈密顿系统（简谐振子）为载体；
- 实现并对比 `Symplectic Euler`、`Velocity Verlet` 与非辛基线 `Explicit Euler`；
- 用长期能量行为与一步映射雅可比行列式来验证“辛性”在数值上的效果。

重点不是做大而全的 ODE 框架，而是把辛积分方法的核心机制跑通并可解释。

## R02

问题定义（哈密顿系统数值积分）：
- 哈密顿量：`H(q,p) = 0.5 * (p^2 + omega^2 * q^2)`；
- 正则方程：
  - `dq/dt = dH/dp = p`
  - `dp/dt = -dH/dq = -omega^2 * q`
- 给定初值 `(q0, p0)`、步长 `h`、步数 `N`，求离散轨迹 `{(q_n, p_n)}_{n=0..N}`。

输出不仅包括轨迹，还包括：
- 每一步能量 `H_n`；
- 相对解析解的状态误差；
- 单步映射 `Phi_h` 的数值雅可比行列式 `det(dPhi_h/d(q,p))`。

## R03

辛积分的关键思想：
- 连续系统保持辛二形式 `dq ^ dp` 不变；
- 理想离散映射也应尽量保持该几何结构（常见判据是 `det(J) = 1`，在 1 自由度时对应面积保持）；
- 辛方法通常不保证每一步能量严格不变，但会在长时间上把能量误差控制在有界振荡范围内。

本实现展示两类典型辛方法：
- `Symplectic Euler`（一阶，kick-drift）；
- `Velocity Verlet`（二阶，时间可逆，常用于分子动力学）。

## R04

算法总览：
1. 固定模型参数 `omega, q0, p0, h, periods`，换算总步数 `N`；
2. 构造解析解作为误差参考；
3. 分别运行三种步进器（显式欧拉 / 辛欧拉 / Verlet）；
4. 对每种方法计算能量漂移指标与状态误差指标；
5. 用中心差分估计每个单步映射的雅可比矩阵并求行列式；
6. 输出统一对比表并给出结论解释。

## R05

核心数据结构：
- `SimulationResult`（`dataclass`）：
  - `method`: 方法名；
  - `time`: 时间网格；
  - `q, p`: 离散轨迹；
  - `energy`: 每步哈密顿量。
- 指标字典（`dict[str, float | str]`）：
  - `final_energy_drift`：最终能量漂移；
  - `max_abs_energy_drift`：全过程最大绝对能量漂移；
  - `rms_state_error`：状态误差均方根；
  - `final_state_error`：终点状态误差；
  - `jacobian_det`：单步映射雅可比行列式。

## R06

正确性与可信性要点：
- 方程层面：实现严格对应哈密顿正则方程（简谐振子可解析对照）；
- 算法层面：
  - 显式欧拉用于非辛对照；
  - 辛欧拉与 Verlet 使用标准更新公式；
- 验证层面：
  - 以解析解衡量离散状态误差；
  - 用 `det(J_step)` 检查单步映射的面积保持特性；
  - 用长期能量漂移检验辛方法“有界振荡”特征。

## R07

复杂度分析（每个方法一次仿真）：
- 时间复杂度：`O(N)`（每步常数次算术操作）；
- 空间复杂度：`O(N)`（保存完整轨迹与能量）；
- 雅可比行列式估计：每个方法额外常数次单步调用，代价 `O(1)`。

因此总成本线性随步数增长，适合作为教学和原型验证。

## R08

边界与异常处理：
- `h <= 0`：抛出 `ValueError`；
- `steps < 1`：抛出 `ValueError`；
- 积分中若状态出现非有限值（`nan/inf`）：抛出 `RuntimeError`；
- 雅可比差分参数 `eps <= 0`：抛出 `ValueError`。

这保证了脚本在异常数值条件下会明确失败而非静默污染结果。

## R09

MVP 取舍说明：
- 仅使用 `numpy` + Python 标准库，避免引入重框架；
- 只选 1D 简谐振子，不扩展到多体系统或约束系统；
- 选择三种方法做最小闭环：
  - 非辛基线（显式欧拉）；
  - 一阶辛方法（辛欧拉）；
  - 二阶辛方法（Velocity Verlet）。

该取舍能以最小代码量展示辛算法的核心收益。

## R10

`demo.py` 函数职责：
- `hamiltonian`：计算哈密顿量；
- `exact_solution`：给出解析解；
- `explicit_euler_step` / `symplectic_euler_step` / `velocity_verlet_step`：三个单步映射；
- `integrate`：统一步进循环并返回轨迹；
- `step_jacobian_determinant`：中心差分估计单步雅可比行列式；
- `summarize_result`：计算误差与能量统计指标；
- `print_metrics_table`：打印对比表；
- `main`：组装参数、运行实验、输出解释。

## R12

输出解读：
- `final dH`：终点能量相对初始能量的偏差；
- `max |dH|`：全程最大绝对能量漂移；
- `RMS state err`：相对解析解的状态误差均方根；
- `final state err`：终点状态误差；
- `det(J_step)`：单步映射雅可比行列式（1 自由度时可看作面积保持指标）。

典型现象：
- 显式欧拉 `det(J_step) > 1`，能量漂移快速累计；
- 辛欧拉/Verlet `det(J_step) ~= 1`，长期能量漂移明显受控。

## R13

建议最小测试集：
- 默认参数回归：`omega=1, q0=1, p0=0, h=0.1, periods=80`；
- 步长敏感性：对比 `h=0.2/0.1/0.05`，检查误差收敛趋势；
- 长时稳定性：固定 `h` 提高 `periods`，观察显式欧拉与辛方法分化；
- 异常分支：`h<=0`、`steps<1`、`eps<=0` 触发预期异常。

## R14

可调参数：
- `omega`：系统频率；
- `q0, p0`：初始状态；
- `h`：时间步长；
- `periods`：模拟时长（按周期数）；
- `eps`：雅可比中心差分扰动量。

工程上通常先定物理时间窗，再按精度需求选 `h`，最后用 `det(J_step)` + 能量统计联合检查离散质量。

## R15

方法对比总结：
- `Explicit Euler`：
  - 优点：实现最简单；
  - 缺点：非辛，长期能量漂移严重。
- `Symplectic Euler`：
  - 优点：辛结构保持，代码仍然非常轻量；
  - 缺点：一阶精度，局部截断误差较 Verlet 大。
- `Velocity Verlet`：
  - 优点：辛 + 二阶精度 + 时间可逆，长期行为通常最好；
  - 缺点：每步计算稍多于辛欧拉。

## R16

应用场景：
- 哈密顿系统长期积分（轨道力学、分子动力学、等离子体模拟）；
- 需要保持几何结构而不只追求局部误差最小的场景；
- 数值分析教学中演示“非辛方法与辛方法的长期差异”。

## R18

源码级算法流（对应 `demo.py`，8 步）：
1. `main` 设定 `omega/q0/p0/h/periods`，并据此计算总步数 `steps`。  
2. `main` 先调用 `exact_solution` 生成同一时间网格上的解析参考轨迹。  
3. 对每个步进器（显式欧拉、辛欧拉、Verlet），调用 `integrate` 执行 `steps` 次离散推进并得到 `q,p,energy`。  
4. `integrate` 内部逐步调用对应 `*_step` 函数，按各自公式更新 `(q_n,p_n)->(q_{n+1},p_{n+1})`，并做非有限数检查。  
5. `summarize_result` 对每种方法计算 `final_energy_drift`、`max_abs_energy_drift`、`RMS/final state error`。  
6. `step_jacobian_determinant` 对单步映射做中心差分，构造 2x2 雅可比矩阵并计算 `det(J_step)`。  
7. `print_metrics_table` 把三种方法指标汇总成同一张可读表格，便于横向比较。  
8. `main` 最后打印解释：辛方法 `det(J_step)` 近似 1 且长期能量误差受控，非辛显式欧拉漂移显著。
