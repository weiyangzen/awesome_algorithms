## R01

球谐函数 `Y_l^m(θ, φ)` 是定义在单位球面上的一组正交归一基函数，常用于把角向依赖量展开为谱系数。标准复球谐定义为：

`Y_l^m(θ,φ)=sqrt((2l+1)/(4π) * (l-m)!/(l+m)!) * P_l^m(cosθ) * exp(i m φ)`

其中 `l>=0`，`-l<=m<=l`，`P_l^m` 为缔合勒让德函数。

## R02

在数学物理中，球谐函数是球坐标分离变量法的核心对象，典型出现在：

- 拉普拉斯方程与泊松方程（角向本征函数）；
- 量子力学角动量本征态（氢原子角向波函数）；
- 散射、天体物理、地球物理中的球面场展开。

它把“几何角向变化”转成“离散谱系数”，便于数值计算与物理解释。

## R04

`demo.py` 用到的关键数学关系：

1. 归一化系数：`N_lm = sqrt((2l+1)/(4π) * (l-m)!/(l+m)!)`。
2. 正阶球谐：`Y_l^m = N_lm * P_l^m(cosθ) * exp(i m φ)`（`m>=0`）。
3. 负阶对称：`Y_l^{-m} = (-1)^m * conjugate(Y_l^m)`（`m>0`）。
4. 内积定义：`<f,g> = ∫_Ω f(θ,φ) * conjugate(g(θ,φ)) dΩ`。
5. 正交归一：`<Y_l^m, Y_{l'}^{m'}> = δ_ll' δ_mm'`。
6. 谱投影：`c_lm = <f, Y_l^m>`，重建 `f_hat = Σ c_lm Y_l^m`。

## R05

记球面网格点数 `N = n_theta * n_phi`，最大阶 `L`，基函数总数 `K=(L+1)^2`。

- 生成全部基函数：`O(KN)`；
- 计算 Gram 型正交性指标（两两内积）：`O(K^2 N)`；
- 谱投影与重建：`O(KN)`；
- 排序并输出主要系数：`O(K log K)`。

整体由 `O(K^2 N)` 主导（因为做了显式正交性检查），空间复杂度约 `O(KN)`。

## R06

`demo.py` 提供四个可验证环节：

- **A. 公式实现一致性**：手写 `Y_l^m` 与 `scipy.special.sph_harm_y` 逐点比较。
- **B. 正交归一检查**：在球面离散积分下统计最大对角偏差与最大非对角项。

## R10

正确性直觉：

1. 球谐函数定义可由 `P_l^m(cosθ)` 与 `exp(i m φ)` 分离得到；
2. 负阶项可由共轭对称关系生成，不需要重复推导；
3. 若基函数近似正交，则投影系数可稳定恢复稀疏真系数；
4. 重建误差小意味着“离散积分 + 基函数实现”整体链路自洽。

## R11

数值精度与稳定性策略：

- 使用较密集网格（`n_theta=96, n_phi=192`）降低积分离散误差；
- 明确 `theta∈[0,π]`、`phi∈[0,2π)` 约定，避免角定义混淆；
- 采用复数双精度 `complex128` 进行累计；
- 通过对角/非对角指标分别监控归一偏差和串扰泄漏。

## R12

关键参数与作用：

- `l, m`：球谐阶数与阶次，决定角向振荡结构；
- `l_max`：参与展开的最高阶，控制基函数数量 `K=(L+1)^2`；
- `n_theta, n_phi`：积分网格分辨率，影响正交与系数精度；
- `true_coeffs`：合成信号的真谱系数，用于恢复误差基准。

调参建议：

- 若正交误差偏大，优先提高 `n_theta/n_phi`；
- 若要表示更尖锐角向结构，增大 `l_max`；
- 高阶时应考虑更高质量求积或数值稳定技巧。

## R14

常见失效模式：

1. 把 `theta`/`phi` 角定义交换，导致结果相位和形状错误；
2. 忽略负阶 `m<0` 的共轭关系，导致谱不完整；
3. 网格过粗时，正交性和投影误差显著增大；
4. 阶数升高后仍沿用低分辨率网格，出现虚假泄漏系数。

## R18

`demo.py` 源码级算法流程（9 步）：

1. `build_sphere_grid` 构造 `theta-phi` 规则网格，并计算 `dΩ=sinθ dθ dφ` 离散权重。  
2. `normalization_factor` 根据 `(l,m)` 计算球谐归一化系数。  
3. `spherical_harmonic_manual` 调用 `scipy.special.lpmv` 生成 `P_l^m(cosθ)`，再乘以 `exp(i m φ)` 和归一化系数得到 `Y_l^m`。对 `m<0` 显式使用共轭对称关系而非黑盒调用。  
4. `compare_with_scipy` 对若干 `(l,m)` 点位比较手写结果与 `scipy.special.sph_harm_y`，输出逐项最大绝对误差。  
5. 在 `main` 中构造 `l<=4` 的完整基集合 `basis={(l,m):Y_l^m}`。  
6. `orthogonality_metrics` 通过加权内积 `sum(f*conj(g)*w)` 计算 Gram 型指标，提取最大对角偏差和最大非对角泄漏。  
7. `reconstruct_signal` 先用少量真系数 `true_coeffs` 合成信号，再用 `project_coefficients` 执行 `c_lm=<f,Y_l^m>` 进行谱恢复。  
8. 对恢复结果计算 `true_coef_error`、`leakage` 与复数 RMSE；并分别用 `numpy`、`scikit-learn`、`torch` 交叉计算误差一致性。  
9. `main` 打印数据表与关键指标，执行阈值断言，确认该 MVP 在当前网格与参数下通过。  

说明：虽然 SciPy 提供现成球谐 API，本实现没有把其当黑盒。源码中明确展开了“勒让德函数 -> 归一化 -> 相位因子 -> 内积投影 -> 重建校验”的可追踪流程。
