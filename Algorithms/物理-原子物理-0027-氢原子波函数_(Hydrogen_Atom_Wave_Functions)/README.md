## R01

氢原子波函数是中心势量子系统的标准解，记为 `psi_{n,l,m}(r,theta,phi)`，可分离为：

`psi_{n,l,m}(r,theta,phi) = R_{n,l}(r) * Y_l^m(theta,phi)`

其中 `n,l,m` 是主量子数、角量子数、磁量子数。它直接决定电子在空间中的概率分布 `|psi|^2`。

## R02

该模型是量子力学教学与原子光谱计算的基础样例，意义在于：

- 将薛定谔方程解析解具体化为可计算函数；
- 解释离散能级 `E_n ~ -1/n^2` 的来源；
- 作为更复杂多电子原子、量子化学基函数的起点；
- 提供“归一化、正交性、期望值”三类可数值验证基准。

## R03

本条目 MVP 输入输出约定（无交互）：

- 输入（脚本内部固定）：
1. 一组量子数状态，如 `(1,0,0)`、`(2,1,1)`、`(2,0,0)`；
2. 三维球坐标积分网格 `r/theta/phi`；
3. 物理常数（玻尔半径采用原子单位 `a0=1`）。
- 输出：
1. 波函数归一化积分误差；
2. 两个态的重叠积分（正交性检查）；
3. `1s` 态的 `<r>` 数值与解析对比；
4. `n=1..4` 的能级表；
5. `All checks passed.` 自动断言结果。

## R04

MVP 使用的核心公式：

1. 径向函数：
`R_{n,l}(r)=sqrt((2/(n a0))^3 * (n-l-1)!/(2n*(n+l)!)) * exp(-rho/2) * rho^l * L_{n-l-1}^{2l+1}(rho)`
其中 `rho=2r/(na0)`。

2. 角向函数（球谐）：
`Y_l^m(theta,phi)=N_{l,m} P_l^m(cos theta) exp(i m phi)`。

3. 概率密度：`p=|psi|^2`。

4. 归一化条件：
`int |psi|^2 r^2 sin(theta) dr dtheta dphi = 1`。

5. 氢原子能级：`E_n = -13.605693122994 / n^2 (eV)`。

## R05

令网格大小为 `nr * ntheta * nphi = N`：

- 单次波函数采样复杂度 `O(N)`；
- 单次三重积分复杂度 `O(N)`；
- 若做 `k` 个态对比，总成本约 `O(kN)`；
- 主要内存占用是三维张量，空间复杂度 `O(N)`。

## R06

`demo.py` 实现三个最小闭环：

- **闭环 A（归一化）**：数值积分验证 `|psi|^2` 是否接近 1；
- **闭环 B（正交性）**：计算 `<1s|2s>` 的复重叠积分，检查是否接近 0；
- **闭环 C（期望值与能级）**：验证 `1s` 态 `<r>≈1.5a0`，并输出 `n=1..4` 理论能级。

## R07

优点：

- 公式到代码一一对应，可审计；
- 不依赖“整包求解器”直接返回结果；
- 同时覆盖函数值、积分与物理量验证。

局限：

- 仅讨论单电子氢原子，不含精细结构/自旋轨道耦合；
- 数值积分精度受网格分辨率和 `r_max` 截断影响；
- 未实现时间演化与外场微扰。

## R08

前置知识与环境：

- 量子数约束：`n>=1, 0<=l<n, -l<=m<=l`；
- 球坐标体积元 `r^2 sin(theta)`；
- Python `>=3.10`；
- 依赖：`numpy`, `scipy`。

## R09

适用场景：

- 原子物理/量子力学课程演示；
- 检查波函数归一化和正交性数值流程；
- 给后续多电子基函数、变分法提供基线。

不适用场景：

- 需要高精度原子光谱（相对论修正、QED 修正）；
- 多电子体系（如 He, Li）直接求解；
- 强场驱动下的时间依赖薛定谔方程问题。

## R10

正确性直觉：

1. 波函数模平方积分必须是 1（概率守恒）；
2. 不同本征态在完备内积空间应正交；
3. `1s` 态平均半径应接近解析值 `3a0/2`；
4. 能级只由 `n` 决定并按 `1/n^2` 递减。

## R11

数值稳定策略：

- 使用显式量子数合法性检查，避免非法输入；
- `r` 从极小正数开始，规避 `r=0` 处数值分支问题；
- 角向采用 `lpmv` + 归一化常数，负 `m` 用共轭关系构造；
- 三重积分统一用 `np.trapezoid`，避免手写离散权重错误。

## R12

关键参数：

- `r_max`：径向截断边界，过小会低估归一化积分；
- `nr/ntheta/nphi`：网格密度，决定积分误差和运行时间；
- `tol_norm/tol_overlap/tol_expectation`：自动断言阈值；
- `a0`：长度尺度（本 MVP 采用原子单位 `a0=1`）。

调参建议：

- 归一化偏差大时先增大 `r_max`；
- 再提升 `nr` 与角向网格；
- 精度要求提高时可改为 Gauss-Legendre 角积分与自适应径向积分。

## R14

常见失效模式：

1. 量子数不合法（如 `l>=n` 或 `|m|>l`）；
2. `r_max` 太小导致尾部概率截断严重；
3. 网格太稀导致正交性/期望值误差偏大；
4. 误把库函数当黑盒而不校验归一化常数。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `validate_quantum_numbers` 检查 `(n,l,m)` 是否满足氢原子本征态约束。  
2. `radial_wavefunction` 用指数项、幂函数和 `eval_genlaguerre` 显式构造 `R_{n,l}(r)`。  
3. `spherical_harmonic` 用 `lpmv`、归一化因子和 `exp(i m phi)` 构造 `Y_l^m`，负 `m` 由共轭关系得到。  
4. `hydrogen_wavefunction_on_grid` 通过广播把径向与角向组合为三维网格波函数 `psi`。  
5. `integrate_spherical` 对 `r/theta/phi` 三轴做 `np.trapezoid`，实现球坐标体积积分。  
6. `check_normalization` 与 `check_overlap` 分别计算 `int|psi|^2 dV` 和 `<psi_a|psi_b>`。  
7. `expectation_r` 计算 `<r>` 并与 `analytic_expectation_r`（解析式）比较误差。  
8. `main` 汇总打印指标、能级表，并用 `assert` 执行阈值判定，最终输出 `All checks passed.`。  

说明：`scipy.special` 仅提供关联拉盖尔与关联勒让德多项式基元；量子数校验、归一化常数、波函数组装与积分验证逻辑都在源码中显式展开。
