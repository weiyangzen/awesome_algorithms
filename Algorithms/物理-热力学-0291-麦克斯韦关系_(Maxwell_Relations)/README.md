## R01

麦克斯韦关系（Maxwell relations）是热力学势的二阶混合偏导可交换性在物理中的直接结果。常用四条为：

1. 由 `U(S,V)`：
`(∂T/∂V)_S = -(∂P/∂S)_V`
2. 由 `H(S,P)`：
`(∂T/∂P)_S = (∂V/∂S)_P`
3. 由 `F(T,V)`：
`(∂S/∂V)_T = (∂P/∂T)_V`
4. 由 `G(T,P)`：
`(∂S/∂P)_T = -(∂V/∂T)_P`

其中 `U/H/F/G` 分别是内能、焓、亥姆霍兹自由能、吉布斯自由能。  
本条目给出一个最小可运行数值验证器：在规则网格上用中心差分同时计算左右两侧导数并比较残差。

## R02

工程价值主要有三点：

- 把“抽象热力学恒等式”落为可计算检查项，便于教学和单元测试；
- 为状态方程、自由能模型提供一致性校验（若违反麦克斯韦关系，模型通常不自洽）；
- 作为复杂仿真前的快速体检，尽早发现符号约定、变量定义、差分实现错误。

MVP 目标是透明、可审计，而不是构建完整工质数据库或高保真流程模拟。

## R03

`demo.py` 输入输出（脚本内置，无交互）：

- 输入：
1. 四个解析热力学势 `U(S,V), H(S,P), F(T,V), G(T,P)`；
2. 各变量网格（`S,V,T,P`）；
3. 中心差分步长 `h` 与通过阈值 `tolerance`。
- 输出：
1. 每条麦克斯韦关系在网格上的 `max_abs_residual`、`mean_abs_residual`；
2. `passed` 布尔判定；
3. 前 12 个采样点的 `lhs/rhs/residual` 详情；
4. 全部断言通过后打印 `All checks passed.`。

## R04

MVP 使用的核心物理关系与程序映射：

1. 基本微分式：
`dU = T dS - P dV`，`dH = T dS + V dP`，`dF = -S dT - P dV`，`dG = -S dT + V dP`。

2. 状态量定义：
`T=(∂U/∂S)_V`，`P=-(∂U/∂V)_S`，`V=(∂H/∂P)_S`，`S=-(∂F/∂T)_V` 等。

3. 麦克斯韦关系来源：
若势函数二阶导连续，则混合偏导可交换：
`∂²Φ/(∂x∂y) = ∂²Φ/(∂y∂x)`。

4. 数值离散：
`∂f/∂x ≈ [f(x+h,y)-f(x-h,y)]/(2h)`，  
`∂f/∂y ≈ [f(x,y+h)-f(x,y-h)]/(2h)`。

5. 判据：
`residual = lhs - rhs`，若 `max|residual| <= tolerance` 则判定该关系通过。

## R05

令每条关系在 `n x n` 网格上验证：

- 单条关系时间复杂度：`O(n^2)`；
- 四条总复杂度：`O(4n^2)`，即 `O(n^2)`；
- 空间复杂度：`O(n^2)`（保存明细表）。

默认 `sample_points=7`，总采样点 `4 * 49 = 196`，运行开销很小。

## R06

`demo.py` 的最小闭环：

- **闭环 A：导数算子**
  `partial_x/partial_y` 用中心差分近似偏导。
- **闭环 B：关系构造**
  对四个势分别构造 `lhs` 和 `rhs` 导数表达。
- **闭环 C：一致性判定**
  `evaluate_relation` 汇总残差统计，`main` 断言全部通过。

## R10

正确性直觉：

1. 麦克斯韦关系本质上是“同一势函数的混合二阶导交换”；
2. 如果一阶状态量定义（如 `T, P, S, V`）来自同一势函数，那么交叉导数必然满足相应恒等式；
3. 数值上左右两侧不可能完全相等，但残差应随差分误差量级收敛到很小；
4. 因此检查 `max_abs_residual` 是否在阈值内，可作为实现正确性的可执行证据。

## R11

数值稳定策略：

- 对含 `log(v)`、`log(p)` 的势函数，采样区间选在严格正域（`v>0,p>0`）；
- 使用中心差分而非前向差分，截断误差阶更高；
- 用 `float64` 计算并设定统一阈值 `tolerance=1e-7`；
- 通过网格多点评估而非单点，降低“偶然对齐”风险。

## R12

关键参数：

- `h`：中心差分步长，控制离散误差与舍入误差平衡；
- `tolerance`：关系通过阈值；
- `sample_points`：每个变量维度的采样密度。

调参建议：

- 若残差偏大，先减小 `h` 到 `1e-7` 量级并观察变化趋势；
- 若数值噪声反而放大，可增大 `h` 到 `1e-5`；
- 若要更严格统计稳定性，可把 `sample_points` 提升到 `11` 或 `15`。

## R14

常见失效模式：

1. 把 `P` 的符号写错（例如忘记 `P = -(∂F/∂V)_T`）；
2. 在 `v` 或 `p` 非正区域采样导致 `log` 无定义；
3. 差分步长过小引发舍入误差放大，或过大引发截断误差；
4. 固定变量条件写错（例如把 `(∂T/∂V)_S` 误实现为 `(∂T/∂V)_P`）；
5. 只测单点并误判，实现缺陷未暴露。

## R18

`demo.py` 源码级算法流程（8 步）：

1. 定义 `MaxwellConfig(h, tolerance, sample_points)`，统一差分步长、阈值与采样密度。  
2. 定义 `partial_x` 与 `partial_y`，用中心差分计算二维标量场的一阶偏导。  
3. 写出四个热力学势 `internal_energy / enthalpy / helmholtz_free_energy / gibbs_free_energy` 以及由其导出的 `T,P,S,V` 一阶状态量函数。  
4. 在 `main` 中构建 `S,V,T,P` 网格，保证 `V,P` 严格为正以满足对数定义域。  
5. 为每条麦克斯韦关系构造 `lhs` 与 `rhs` 函数（例如 `lhs=(∂T/∂V)_S`，`rhs=-(∂P/∂S)_V`）。  
6. `evaluate_relation` 遍历二维网格点，逐点计算 `lhs`、`rhs`、`residual=lhs-rhs`，并记录详情表。  
7. 同时统计每条关系的 `max_abs_residual` 与 `mean_abs_residual`，生成汇总表并给出 `passed` 判定。  
8. `main` 打印汇总与样例点详情，随后断言全部 `passed`；若有任一关系不满足阈值则立即失败。  

补充说明：实现没有把第三方库当黑盒“直接返回麦克斯韦关系结论”，而是显式执行“差分求导 -> 左右项构造 -> 残差统计 -> 阈值判定”的完整链路。
