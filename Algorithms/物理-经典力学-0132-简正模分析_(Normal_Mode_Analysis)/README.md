# 简正模分析 (Normal Mode Analysis)

- UID: `PHYS-0132`
- 学科: `物理`
- 分类: `经典力学`
- 源序号: `132`
- 目标目录: `Algorithms/物理-经典力学-0132-简正模分析_(Normal_Mode_Analysis)`

## R01

简正模分析用于把“多个耦合自由度的振动系统”变换为“若干独立简谐振子”。

在经典力学中，典型对象是小振幅线性化后的质量-弹簧系统。原坐标下方程彼此耦合；切换到模态坐标后，每个模态独立演化，频率由系统固有特征决定。

## R02

本条目采用 `N` 自由度质量-弹簧链作为 MVP：
- 质量向量 `m = (m1, ..., mN)`。
- 弹簧向量 `k = (k0, k1, ..., kN)`，其中 `k0`/`kN` 分别连接左右固定墙，`k1..k_{N-1}` 连接相邻质量块。
- 小位移近似下，系统满足线性常系数二阶方程。

该模型足够小而完整，能覆盖“建模-求模态-投影-重构-验证”全链路。

## R03

线性化动力学写为：

`M x¨ + K x = 0`

其中：
- `M = diag(m1, ..., mN)` 为质量矩阵（对称正定）。
- `K` 为刚度矩阵（对称半正定/正定，取决于边界与约束）。
- `x(t)` 是物理坐标位移向量。

简正模分析目标是寻找一组基向量，使方程在该基下对角化。

## R04

设试探解 `x = phi * cos(omega t)`，代入得广义特征值问题：

`K phi = omega^2 M phi`

求得 `lambda_i = omega_i^2` 和特征向量 `phi_i` 后，可构成模态矩阵 `Phi = [phi_1 ... phi_N]`。若按质量归一化，则满足：

- `Phi^T M Phi = I`
- `Phi^T K Phi = diag(omega_1^2, ..., omega_N^2)`

从而系统在模态坐标中彻底解耦。

## R05

适用前提与假设：
- 小振幅（线性化成立）。
- 参数不随时间变化（`M, K` 常量）。
- 阻尼忽略或很小（本 MVP 使用无阻尼模型）。
- 质量均为正数；弹簧刚度非负，且整体约束避免不期望的奇异漂移。

若有强非线性、时变参数或大阻尼，应改用更一般的数值积分与模态更新方法。

## R06

本目录 `demo.py` 的输入输出约定：
- 输入：代码内置一组 4 自由度链参数、初始位移 `x0`、初始速度 `v0`、时间网格。
- 输出：
1. 固有频率表（`omega`、周期、残差）。
2. 模态正交性误差与对角化误差。
3. 轨迹重构误差、能量漂移。
4. 若全部满足阈值，输出 `Validation: PASS`。

脚本无交互输入，直接运行即可复现。

## R07

算法主流程（高层）：
1. 由 `m` 和 `k` 组装 `M, K`。
2. 质量缩放：构造 `A = M^{-1/2} K M^{-1/2}`。
3. 对 `A` 做对称特征分解，得到 `lambda, U`。
4. 还原物理模态 `Phi = M^{-1/2} U`，并进行质量归一化检查。
5. 将初值投影到模态坐标，得到每个模态的初始振幅和速度系数。
6. 使用解析式 `q_i(t)` 演化每个模态，再映射回物理坐标 `x(t)`。
7. 输出误差与守恒量检查结果。

## R08

复杂度（`N` 为自由度数，`T` 为时间采样点数）：
- 构建矩阵：`O(N)` 到 `O(N^2)`（本实现按稀疏结构填充，量级近似 `O(N)`）。
- 特征分解（密集实现）：`O(N^3)`。
- 模态时间演化：`O(N*T)`。
- 轨迹重构：矩阵乘法 `O(N^2*T)`。

在中小规模振动系统（教学/原型）下，该复杂度可接受且实现透明。

## R09

数值稳定性与误差控制：
- 将广义问题转为标准对称特征问题，可避免直接求 `M^{-1}K` 带来的非对称数值误差。
- 对负的微小特征值（浮点噪声）做 `clip(lambda, 0, +inf)`。
- 使用残差 `||Kphi - lambda Mphi||` 与正交性误差双重校验分解质量。
- 用总能量相对漂移评估长时间重构的数值可信度。

## R10

MVP 技术栈：

## R12

输出字段说明：
- `mode`: 模态编号（从 1 开始）。
- `omega(rad/s)`: 固有角频率。
- `period(s)`: 周期 `2*pi/omega`（零频模式记为 `inf`）。
- `eig_residual`: 广义特征方程残差。
- `max |offdiag(Phi^T M Phi)|`: 质量正交偏离。
- `max |offdiag(Phi^T K Phi)|`: 刚度对角化偏离。
- `max initial reconstruction error`: 对 `x(0), v(0)` 的回代误差。
- `relative energy drift`: 轨迹中能量最大相对漂移。

## R13

正确性验证（demo 内置断言）：
1. 特征方程残差应足够小。
2. `Phi^T M Phi` 的非对角项应接近 0。
3. `Phi^T K Phi` 的非对角项应接近 0。
4. `t=0` 时重构的位移与速度应接近初值。
5. 无阻尼系统总能量应近似守恒。

这五项同时通过时，认为简正模分析在该样例上实现正确。

## R14

当前实现局限：
- 仅处理线性无阻尼系统。
- 矩阵按密集方式处理，不适合超大规模有限元模型。
- 未处理模态近简并时的子空间旋转标定问题。
- 未包含实验噪声数据反演（例如仅由响应识别模态参数）。

## R15

可扩展方向：

- 引入外力项 `f(t)`，实现模态叠加下受迫振动响应。
- 加入参数扫描（质量或刚度扰动）做灵敏度分析。

## R16

实际应用场景：
- 机械结构振动（梁、板、机架）基础模态提取。
- 车辆/航天部件的固有频率避共振设计。
- 分子动力学中的小振动近似（normal modes）。
- 土木与地震工程中的结构动力特性评估。

## R17

方案对比：
- 直接时间积分（如显式/隐式积分器）可处理更一般系统，但不直接给出“固有模态结构”。
- 纯黑箱求特征值调用开发最快，但学习与审计价值低。
- 本方案保留数学结构：手工构造 `M,K`、显式完成质量缩放、投影与重构，兼顾可读性与可验证性。

因此它是教学和小型工程原型中“最小但完整”的实现路径。

## R18

`demo.py` 源码级流程拆解（9 步）：
1. `build_chain_matrices` 根据质量与弹簧拓扑显式组装 `M, K`。
2. `normal_mode_analysis` 计算 `M^{-1/2}`，把 `K phi = lambda M phi` 转为标准对称问题 `A u = lambda u`。
3. 调用 `numpy.linalg.eigh(A)` 获得升序特征值与正交基 `U`，并对极小负值做截断。
4. 回到物理坐标得到 `Phi = M^{-1/2}U`，再计算 `Phi^T M Phi` 与 `Phi^T K Phi` 检查归一化和解耦质量。
5. `modal_initial_conditions` 将 `x0, v0` 投影到模态空间，得到 `q(0)` 与 `qdot(0)`。
6. `modal_time_response` 对每个模态按解析式计算 `q_i(t)` 与 `qdot_i(t)`（含零频模式分支）。
7. 通过矩阵乘法 `x(t)=Phi q(t)`、`v(t)=Phi qdot(t)` 重构物理轨迹。
8. `total_energy` 在每个时刻计算 `0.5*(v^T M v + x^T K x)`，评估能量守恒漂移。
9. `main` 汇总频率表、残差、重构误差和能量漂移，并以阈值给出 `PASS/FAIL`。

即使核心特征分解调用了线性代数库，围绕它的建模、变换、验证与动力学重构流程都在源码层完整展开。
