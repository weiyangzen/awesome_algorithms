# 中心力场问题 (Central Force Problem)

## R01

中心力场指力始终沿径向，且大小仅依赖半径 `r=|x|`：

`F(x) = f(r) * x/r`

本 MVP 选用经典反平方引力模型（单位质量形式）：

`a(x) = -mu * x / r^3`

其中 `mu = GM` 是引力参数。该模型是开普勒两体问题的标准核心。

## R02

中心力场是经典力学中“从对称性到守恒律”的典型范例：

- 力矩为零 `x x F = 0`，直接导致角动量守恒；
- 反平方势 `U(r) = -mu/r` 进一步给出椭圆/抛物/双曲轨道分类；
- 有效势 `U_eff(r)` 将二维轨道问题化为一维径向问题，便于分析稳定性与转向点。

## R03

本条目 MVP 要解决的具体问题：

- 用源码验证“力总是径向指向中心”（不是只写定义）；
- 数值积分一条束缚轨道（椭圆）并输出关键守恒量漂移；
- 对比理论近心点/远心点与数值结果，形成可自动验证的最小闭环。

## R04

核心方程与关系：

1. 运动方程（单位质量）
   `x¨ = -mu * x / |x|^3`
2. 比机械能
   `eps = v^2/2 - mu/r`
3. 比角动量（二维）
   `h = x*vy - y*vx`
4. 有效势（固定 `h`）
   `U_eff(r) = h^2/(2r^2) - mu/r`
5. 径向能量形式
   `eps = r_dot^2/2 + U_eff(r)`
6. 椭圆轨道参数关系
   `r_peri = a(1-e)`, `r_apo = a(1+e)`。

## R05

设总积分步数为 `T`：

- Velocity Verlet 积分：`O(T)`；
- 每步计算加速度、能量、角动量均为常数时间；
- 存储完整轨迹时空间复杂度 `O(T)`，若仅做在线监控可降为 `O(1)`。

该复杂度适合教学验证与小规模参数实验。

## R06

`demo.py` 包含两段非交互演示：

- **Demo A（中心性检查）**：对若干采样点计算 `a(x)`，检查 `a` 与 `-x` 的夹角余弦是否接近 1；
- **Demo B（轨道积分）**：以给定 `a=1.5, e=0.4` 的近心初值做 Verlet 积分，输出：
  - 轨道元素回推值 `a_est/e_est`；
  - 能量与角动量相对漂移；
  - 近心点/远心点数值误差；
  - 径向能量恒等式残差。

## R10

正确性直觉：

1. 中心力定义要求 `a` 与 `-x` 共线，Demo A 直接数值验证这一几何约束；
2. 反平方势的解析轨道参数由初值唯一决定，`a_est/e_est` 可回推检验；
3. 守恒系统下能量与角动量应近似常数，漂移量可衡量积分可靠性；
4. 近心点/远心点是轨道几何最直观特征，能把“公式正确”连接到“轨迹形状正确”。

## R11

数值稳定策略：

- 使用辛结构友好的 Velocity Verlet，优于显式 Euler 的长期守恒表现；
- 统一 `float64`；
- 用适中固定步长 `dt=2e-3` 兼顾速度与精度；
- 用相对漂移阈值和几何误差阈值双重约束，避免只看单一指标。

## R12

关键参数：

- `mu`：中心引力参数；
- `a_true/e_true`：目标椭圆轨道几何参数；
- `dt`、`steps`：积分步长与总步数；
- `periods`：模拟多少个轨道周期用于统计转向点。

调参建议：

- 若守恒漂移偏大，优先减小 `dt`；
- 若转向点识别不足，增大 `periods` 或总步数；
- 高偏心轨道（`e -> 1`）应进一步减小步长。

## R14

常见失效模式：

1. 步长过大导致能量漂移增大，轨道逐步失真；
2. 半径过小接近奇点，`1/r^3` 放大误差并可能触发数值不稳定；
3. 初值与目标轨道参数不一致，导致“理论值比较”看似失败；
4. 仿真时长过短，无法稳定提取足够转向点。

## R17

`demo.py` 的 MVP 功能清单：

- `central_acceleration` 实现反平方中心力加速度；
- `estimate_orbit_elements` 从单个状态回推 `a/e/h/eps`；
- `velocity_verlet_integrate` 实现二维轨道固定步长积分；
- `find_turning_points` 提取近心点/远心点；
- 主流程输出并断言中心性、守恒漂移、转向点误差。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `central_acceleration` 按 `-mu*x/r^3` 计算每个位置的中心力加速度。  
2. `check_central_direction` 在多组采样点上计算 `cos(angle(a,-x))`，直接验证“力沿径向指向中心”。  
3. 由给定 `a_true/e_true` 构造近心初值 `(x0,v0)`，并用 `estimate_orbit_elements` 回推解析轨道参数。  
4. `velocity_verlet_integrate` 循环推进 `x,v`：先位置半步再速度校正，得到完整时间序列。  
5. 基于轨迹计算 `eps(t)` 与 `h(t)`，统计相对漂移上界作为守恒性指标。  
6. 构造 `U_eff(r)=h0^2/(2r^2)-mu/r`，验证 `eps = r_dot^2/2 + U_eff` 的径向能量恒等式残差。  
7. `find_turning_points` 从 `r(t)` 的离散极值中提取近心点/远心点，并与 `a(1-e), a(1+e)` 对照。  
8. `main` 汇总打印所有指标并执行阈值断言，保证 `uv run python demo.py` 一次性无交互通过。

整个实现没有把第三方库当黑盒：`numpy` 仅用于数组运算，动力学方程、积分器、守恒量与验证逻辑都在源码中逐步展开。
