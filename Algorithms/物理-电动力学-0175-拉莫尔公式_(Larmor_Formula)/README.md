# 拉莫尔公式 (Larmor Formula)

- UID: `PHYS-0174`
- 学科: `物理`
- 分类: `电动力学`
- 源序号: `175`
- 目标目录: `Algorithms/物理-电动力学-0175-拉莫尔公式_(Larmor_Formula)`

## R01

拉莫尔公式给出经典非相对论点电荷在加速运动时的瞬时辐射功率，是电磁辐射与带电粒子动力学之间最基础的能量桥梁之一。该条目实现一个最小可运行 MVP：对给定加速度时间序列计算瞬时辐射功率，并用简谐加速度场景验证数值平均功率与解析平均功率一致。

## R02

问题定义（非相对论）：
- 输入：电荷量 `q`（库仑），加速度 `a(t)`（米/秒²）或离散采样值。
- 输出：
  - 瞬时辐射功率 `P(t)`（瓦特）
  - 时间区间平均功率 `P_avg`
  - 与解析解对比误差（本 MVP 采用简谐加速度案例）

## R03

核心公式（SI 制）：

`P(t) = q^2 * a(t)^2 / (6 * pi * epsilon0 * c^3)`

其中：
- `epsilon0 = 8.8541878128e-12 F/m`（真空介电常数）
- `c = 2.99792458e8 m/s`（光速）

对 `a(t) = a0 cos(omega t)`，一个周期内解析平均功率：

`<P> = q^2 * a0^2 / (12 * pi * epsilon0 * c^3)`

## R04

物理适用条件：
- 粒子速度远小于光速（`v << c`）
- 点电荷近似成立
- 经典电动力学范围（不含量子辐射修正）

若速度接近光速，应改用 Liénard 公式而非拉莫尔公式。

## R05

MVP 采用的测试场景：
- 设定 `a(t)=a0*cos(omega*t)`
- 在一个周期 `T=2*pi/omega` 上离散采样
- 计算离散 `P(t)` 后积分得数值平均功率
- 与解析平均功率比较相对误差

该场景具有解析基准，便于快速验证实现正确性。

## R06

算法思路：
1. 生成一周期离散时间网格。
2. 构造简谐加速度序列 `a(t)`。
3. 使用拉莫尔公式逐点计算 `P(t)`。
4. 采用梯形积分计算 `P(t)` 在一周期上的积分，再除以周期得到 `P_avg_num`。
5. 计算解析平均 `P_avg_analytic`。
6. 报告两者相对误差并输出采样点。

## R07

复杂度：
- 时间复杂度：`O(N)`，`N` 为时间采样点数。
- 空间复杂度：`O(N)`，存储 `t`、`a(t)`、`P(t)`。

## R08

数值稳定性与精度：
- 公式本身仅包含乘法与平方，数值稳定。
- 平均功率误差主要来自时间积分离散误差，随采样点数增加而下降。
- 使用双精度浮点（Python `float` / NumPy `float64`）可满足此类演示精度需求。

## R09

参数选择（demo 默认）：
- `q = 1e-9 C`
- `f = 1e9 Hz`，`omega = 2*pi*f`
- `a0 = 5e13 m/s^2`
- 采样点数 `N = 5000`

该组参数会产生清晰可见的非零辐射功率量级，便于核对输出。

## R10

预期输出内容：
- 模型参数和常数
- 数值平均功率与解析平均功率
- 相对误差（通常应很小）
- 前若干个时间点的 `t, a(t), P(t)` 样本

## R11

正确性检查：
- 检查 `P(t) >= 0`（由 `a^2` 保证）
- 检查 `P_avg_num` 与 `P_avg_analytic` 的相对误差是否低于阈值（例如 `1e-3`）
- 当 `a0 = 0` 时应得到 `P(t)=0` 且平均功率为 `0`

## R12

边界与异常处理：
- `omega <= 0` 时不构成周期运动，demo 应拒绝。
- `samples < 3` 无法稳定积分，应拒绝。
- 极大参数可能导致浮点上溢，工程中应加量纲和范围检查。

## R13

与工程问题的连接：
- 加速电荷天线辐射的基础估算
- 束流/等离子体中粒子辐射损耗的量级判断
- 更复杂辐射模型（Liénard、同步辐射）前的基线模型

## R14

局限性：
- 非相对论近似，不包含 `gamma` 因子
- 不含辐射反作用（Abraham-Lorentz 力）
- 不含介质、边界条件和量子效应

## R15

可扩展方向：
- 升级到相对论 Liénard 公式
- 输入任意离散 `a(t)` 数据进行辐射功率后处理
- 增加频谱分析（FFT）研究辐射成分

## R16

最小实现的技术选择：
- 核心数值计算优先使用 NumPy；若环境无 NumPy，提供纯 Python 后备路径。
- 不依赖大型框架，避免把问题“黑盒化”。
- 将物理公式、积分、校验拆分为显式函数，便于审阅与复用。

## R17

运行方式：

```bash
uv run python demo.py
```

无需任何交互输入。运行后直接打印结果摘要与误差检查。

## R18

`demo.py` 的源代码级流程（8 步）：
1. 定义物理常数 `EPSILON_0` 与 `C_LIGHT`。
2. 定义 `larmor_power(q, a)`：按 `q^2 a^2 /(6*pi*epsilon0*c^3)` 返回瞬时功率。
3. 定义简谐加速度生成函数 `a(t)=a0*cos(omega*t)`。
4. 在 `simulate_one_period(...)` 中计算周期 `T=2*pi/omega`，并生成 `[0, T]` 采样网格。
5. 对每个采样点计算 `a(t)` 与 `P(t)`，得到离散时序。
6. 用梯形积分 `trapz(P, t)` 计算一周期辐射能量，再除以 `T` 得数值平均功率。
7. 计算解析平均功率 `q^2*a0^2/(12*pi*epsilon0*c^3)` 并给出相对误差。
8. 在 `main()` 中固定参数执行模拟，打印摘要、误差与前若干样本点，保证可复现实验。
