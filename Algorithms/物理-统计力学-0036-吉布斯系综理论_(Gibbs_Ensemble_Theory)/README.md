## R01

吉布斯系综理论把“宏观约束下的系统”转化为“微观状态概率分布”问题。

设微观状态 `i` 的能量为 `E_i`、粒子数为 `N_i`，则常见三类分布为：

1. 微正则系综（能量壳层内等概率）：`p_i = 1/Omega`。
2. 正则系综（固定 `T,N`）：`p_i = exp(-beta E_i) / Z`。
3. 巨正则系综（固定 `T,mu`）：`p_i = exp[-beta(E_i - mu N_i)] / Xi`。

其中 `beta = 1/(k_B T)`，`Z` 和 `Xi` 是配分函数（归一化常数）。

## R02

该理论由 J. Willard Gibbs 在 19 世纪末系统化，核心意义是：

- 用统一的概率框架连接热力学与微观力学；
- 把宏观可观测量写成期望值 `\langle A \rangle = \sum_i p_i A_i`；
- 提供从“状态枚举”到“采样算法”的直接工程路径。

在计算实践中，吉布斯系综是 Monte Carlo、分子模拟、统计推断中最常用的目标分布基础。

## R03

本目录 MVP 选择一个一维周期晶格气体（`L=6`）作为可完全枚举的演示系统。

- 微观状态：`n_i in {0,1}`，共 `2^L` 个。
- 能量函数：`E = epsilon * sum_i n_i + J * sum_i n_i n_{i+1}`（周期边界）。
- 粒子数：`N = sum_i n_i`。

`demo.py` 固定参数、无交互输入，输出：

1. 微正则/正则/巨正则三种分布对应的 `\langle E \rangle`、`\langle N \rangle`、熵；
2. 巨正则精确分布与 Metropolis 采样分布的 Top 状态对比；
3. 总变差距离 `TV` 和断言校验结果。

## R04

MVP 使用的核心公式：

1. 状态能量：
`E(s) = epsilon * N(s) + J * \sum_i n_i n_{i+1}`。

2. 正则分布（固定粒子数子空间内）：
`p(s|N=N0) = exp[-beta E(s)] / Z_{N0}`。

3. 巨正则分布（全状态空间）：
`p(s) = exp[-beta(E(s)-mu N(s))] / Xi`。

4. 可观测量期望：
`\langle A \rangle = \sum_s p(s) A(s)`。

5. Metropolis 接受率（单点翻转提议）：
`alpha = min(1, exp[-beta(Delta E - mu Delta N)])`。

## R05

设晶格长度为 `L`。

- 完全枚举复杂度：时间 `O(2^L * L)`，空间 `O(2^L * L)`。
- 一次 Metropolis 迭代（局部增量更新）：时间 `O(1)`。
- `S` 步采样总成本：`O(S)`。

本 MVP 取 `L=6`，因此枚举代价很小，可把“精确解”和“采样近似”并排验证。

## R06

`demo.py` 构造三个闭环：

1. **闭环 A（分布构造）**：从同一状态空间构造微正则、正则、巨正则分布，并验证概率归一化。
2. **闭环 B（采样一致性）**：用 Metropolis 采样巨正则分布，与精确枚举分布比较 `TV`。
3. **闭环 C（物理量一致性）**：比较精确与采样的 `\langle E \rangle`、`\langle N \rangle` 差异并断言。

## R07

优点：

- 同时覆盖三种经典系综，且都可追溯到同一能量模型；
- 既有解析枚举结果，也有随机采样验证，不停留在公式层面；
- 代码短小，函数职责清晰，适合教学与回归测试。

- 未实现 Wang-Landau、并行回火等高级采样器。

## R10

正确性直觉：

1. 吉布斯权重由 `exp(-beta * 有效能量)` 给出，归一化后即合法概率分布；
2. 正则分布必须严格限制在固定 `N` 子空间，子空间外概率应为 0；
3. Metropolis 通过接受率满足细致平衡，平稳分布是目标巨正则分布；
4. 当采样步数足够多时，经验频率应逼近精确分布（`TV` 下降）。

## R11

数值稳定策略：

- 权重归一化使用 `log-weight` 平移（减去最大值）避免指数上溢/下溢；
- 能量增量用局部差分 `Delta E`，避免每步全量重算带来的累积噪声和冗余；
- 概率校验和约束校验使用断言，第一时间暴露实现错误。

本例没有浮点微分方程积分，主要风险是指数计算和采样统计波动。

## R12

关键参数与影响：

- `L`：状态空间规模，控制枚举可行性；
- `beta`：温度倒数，越大越偏向低能态；
- `mu`：化学势，越大越偏向高粒子数态；
- `steps/burn_in/thin`：决定采样精度与运行时长；
- `epsilon/J`：单粒子与邻接相互作用的竞争强度。

调参建议：若 `TV` 偏大，优先增加 `steps` 与 `burn_in`，其次调整 `thin` 降低自相关。

## R13

- 近似比保证：N/A（非组合优化近似算法）。
- 随机成功率保证：本实现是可复现随机过程（固定 `seed`），输出并非“必然最优”，而是“统计收敛”型保证。

可验证保证：

1. 概率归一化必须成立；
2. 正则分布仅在固定 `N` 支持集上有质量；
3. 巨正则采样与精确分布的 `TV` 必须低于阈值；
4. `\langle E \rangle`、`\langle N \rangle` 采样误差必须在阈值内。

## R14

常见失效模式：

1. 状态编码/解码顺序不一致，导致“精确分布”和“采样计数”错位；
2. 忘记周期边界，邻接项计算错误；
3. 接受率写错号（`Delta E - mu Delta N`），导致平稳分布偏离目标；
4. 采样步数不足或 burn-in 太短，出现明显偏差。

脚本中的断言专门用于拦截这些问题。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `enumerate_states` 生成长度 `L` 的全部二进制占据态，并保证状态索引与二进制编码一致。  
2. `energies` 与 `particle_numbers` 对每个状态计算 `E(s)` 和 `N(s)`，得到离散状态空间的物理量表。  
3. `microcanonical_distribution` 在给定能量壳层上做均匀赋值；`canonical_distribution` 在固定 `N` 子空间按 `exp(-beta E)` 归一化；`grand_canonical_distribution` 在全空间按 `exp[-beta(E-muN)]` 归一化。  
4. `normalize_log_weights` 先对 `logw` 减最大值再指数化，避免权重数值溢出。  
5. `metropolis_grand_canonical` 执行单点翻转提议，用局部 `Delta E` 与 `Delta N` 计算接受率，经过 burn-in 与 thinning 后统计状态频率。  
6. `expectation` 与 `shannon_entropy` 计算各系综的 `\langle E \rangle`、`\langle N \rangle` 与熵，用于横向对比。  
7. `main` 汇总生成 `pandas` 表格，输出微正则壳层信息、正则支持集大小、巨正则 Top 状态概率对比和 `TV` 指标。  
8. `assert` 组执行自动验收：归一化、支持集约束、`TV` 阈值、以及 `\langle E \rangle/\langle N \rangle` 的精确-采样一致性，全部通过后打印 `All checks passed.`。  

说明：`numpy/pandas` 仅用于数组与表格运算；系综定义、权重构造、接受率、状态转移和验证逻辑均在源码中显式展开，不是黑箱调用。
