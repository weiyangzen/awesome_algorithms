# 朗道能级 (Landau Levels)

- UID: `PHYS-0078`
- 学科: `物理`
- 分类: `固体物理`
- 源序号: `78`
- 目标目录: `Algorithms/物理-固体物理-0078-朗道能级_(Landau_Levels)`

## R01

朗道能级（Landau Levels）描述的是带电粒子在均匀磁场中横向运动的量子化能谱。

在二维电子系统中，当外加垂直磁场 `B` 后，连续自由电子能谱会离散成等间距能级：

`E_n = hbar * omega_c * (n + 1/2),  n = 0,1,2,...`

其中 `omega_c = |q|B/m` 是回旋频率。该离散结构是量子霍尔效应、磁振荡等固体物理现象的重要基础。

## R02

本条目 MVP 的计算问题定义：

- 输入：
  - 物理参数：`hbar, m, |q|, B`；
  - 一维离散网格 `x in [x_min, x_max]` 与 `n_grid`；
  - 一组 `k_y` 采样值 `k_values`；
  - 目标低能级数量 `n_levels`。
- 输出：
  - 每个 `k_y` 下前 `n_levels` 个数值本征能；
  - 与解析朗道能级的误差统计；
  - 不同 `k_y` 之间的能级扩展（用于验证简并性）。

## R03

采用 Landau 规范 `A = (0, Bx, 0)` 后，二维哈密顿量可分离。固定 `k_y = k` 后得到一维约化问题：

`H_k = p_x^2/(2m) + (hbar*k - q*B*x)^2/(2m)`

这本质上是“中心位置随 `k` 平移”的谐振子，理论能谱与 `k` 无关，等于朗道能级。

MVP 用有限差分离散 `p_x^2`，将连续问题变成对称三对角矩阵本征值问题。

## R04

时间复杂度（设网格数 `N = n_grid`，`K = len(k_values)`，提取能级数 `L = n_levels`）：

- 每个 `k` 需要解一个 `N x N` 三对角本征问题；
- 使用 `scipy.linalg.eigh_tridiagonal(..., select="i")` 取前 `L` 个本征值，工程上复杂度近似 `O(N*L)` 到 `O(N*L + N)`；
- 总体约 `O(K*N*L)`（若按保守上界也可视作 `O(K*N^2)` 的数量级估计）。

本 MVP 规模小（`N=801, K=7, L=5`），运行在秒级以内。

## R05

空间复杂度：

- 网格与三对角系数：`O(N)`；
- 单个 `k` 的势能对角项：`O(N)`；
- 结果矩阵 `energies`：`O(K*L)`；
- 总体工作内存为 `O(N + K*L)`，远小于构造完整稠密矩阵 `O(N^2)`。

## R06

离散化微例（展示算法核心而非真实参数）：

设 `dx = 0.2, hbar = m = 1`，则动能离散系数：

- 主对角：`+1/dx^2 = 25`
- 次对角：`-1/(2dx^2) = -12.5`

若某个网格点 `x_i = 0.4`，且 `B=1, q=1, k=0.5`，势能项为：

`V_i = 0.5*(k - B*x_i)^2 = 0.5*(0.1)^2 = 0.005`

则该点哈密顿量主对角元素约 `25.005`。所有网格点组装后形成三对角矩阵，再取最低本征值即对应低朗道能级的数值近似。

## R07

朗道能级的重要性：

- 是二维电子气在磁场中量子化的基础结构；
- 直接关联量子霍尔效应平台与简并度；
- 在固体物理实验中影响磁输运、磁化率振荡（如 Shubnikov-de Haas/de Haas-van Alphen）；
- 也是理解拓扑能带与磁场耦合的入门基石。

## R08

理论要点（与本实现直接对应）：

1. 最小耦合 `p -> p - qA` 将磁场引入哈密顿量。  
2. 取 Landau 规范后 `y` 方向可用平面波分离，得到 `k_y` 参数化的一维问题。  
3. 一维有效势是平移谐振子势，频率固定为 `omega_c = |q|B/m`。  
4. 因频率固定、仅中心平移，理论本征能与 `k_y` 无关。  
5. 数值上，边界截断与网格离散会造成小误差和弱 `k` 依赖，可通过扩大区间与加密网格减小。

## R09

适用条件与局限：

- 适用于均匀磁场、单粒子近似、无杂质/无晶格势的基础模型；
- 适合教学、数值验证、构建更复杂模型前的基线；
- 不包含自旋塞曼劈裂、电子-电子相互作用、真实材料能带结构；
- 有限区间离散会引入边界效应，高能级误差通常大于低能级。

## R10

正确性框架（本目录采用）：

1. 用解析式 `E_n = hbar*omega_c*(n+1/2)` 作为基准。  
2. 对多个 `k_y` 独立求解约化哈密顿量最低本征值。  
3. 统计每个 `n` 的 `k` 向均值误差 `|E_num - E_analytic|`。  
4. 统计每个 `n` 在不同 `k` 下的能量展宽 `max-min`，检验简并性。  
5. 通过断言强制验收：误差、展宽和数值有限性都必须满足阈值。

## R11

误差来源与稳定性：

- 截断误差：二阶有限差分对 `d^2/dx^2` 的误差为 `O(dx^2)`；
- 边界误差：若振子中心或波函数尾部接近边界，能级会被抬升；
- 高能级误差：高 `n` 波函数更宽，受有限盒子影响更明显；
- 通过选取 `x in [-8, 8]` 且 `k in [-2.5, 2.5]`，保证低能级主要局域在区间内部。

## R12

性能与实现取舍：

- 不构造稠密矩阵，仅保存三对角向量，内存更省；
- `eigh_tridiagonal` 直接针对三对角结构，较通用稠密特征分解更快；
- 仅提取最低 `n_levels`，避免计算整个谱；
- MVP 重点是“算法透明 + 可复现”，不引入额外框架复杂度。

## R14

鲁棒性与常见失效模式：

- `n_grid` 太小：离散误差增大，低能级偏差明显；
- `x` 区间太窄：边界压缩波函数，破坏 `k` 简并；
- `k_values` 取值过大：振子中心移向边界，展宽增大；
- 参数非正（`m,B,|q|`）：模型物理上不成立，代码会抛出 `ValueError`。

## R15

`demo.py` 结构说明：

- `LandauConfig`：集中管理物理与离散参数，并校验合法性；
- `build_tridiagonal_kinetic`：构造动能二阶差分三对角项；
- `solve_reduced_hamiltonian`：组装 `H_k` 并求最低本征值；
- `analytic_landau_levels`：计算解析朗道能级；
- `main`：遍历 `k_values` 求谱、汇总误差、打印表格并断言验收。

## R16

相关算法与可扩展方向：

- 数值线性代数：对称三对角本征值算法；
- 量子数值方法：有限差分离散、边值问题本征求解；
- 可扩展到：
  - 加入塞曼项（自旋劈裂）；

  - 在晶格模型中加入 Peierls 相位，比较连续模型与 Hofstadter 谱。

## R18

源码级算法流程拆解（对应 `demo.py`，8 步）：

1. 读取并校验配置：`LandauConfig.validate()` 保证 `m,B,|q|,n_grid,n_levels` 等参数物理且数值上可解。  
2. 建立实空间网格：在 `[x_min, x_max]` 上生成 `x` 与步长 `dx`。  
3. 构造动能三对角：`build_tridiagonal_kinetic` 按二阶差分生成主对角和次对角。  
4. 对每个 `k_y` 生成势能：`V_k(x)= (hbar*k-qBx)^2/(2m)`，与动能对角相加得到 `H_k` 主对角。  
5. 求最低本征值：调用 `eigh_tridiagonal(..., select="i", select_range=(0,n_levels-1))`，提取低能谱。  
6. 汇总多 `k` 结果：形成 `energies[k_index, n]`，并计算 `k` 向均值与展宽。  
7. 对照解析解：由 `analytic_landau_levels` 计算 `E_n^analytic`，得到绝对/相对误差。  
8. 验收与输出：打印 `pandas` 表格，执行误差与简并性断言，全部通过后结束。

说明：虽然调用了 SciPy 的三对角特征值例程，但磁场哈密顿量离散、参数化扫描、误差评估与验收逻辑均在源码中显式展开，不是“黑箱一键求解”。
