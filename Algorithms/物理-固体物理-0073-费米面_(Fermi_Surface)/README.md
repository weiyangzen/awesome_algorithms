# 费米面 (Fermi Surface)

- UID: `PHYS-0073`
- 学科: `物理`
- 分类: `固体物理`
- 源序号: `73`
- 目标目录: `Algorithms/物理-固体物理-0073-费米面_(Fermi_Surface)`

## R01

费米面是金属或半金属在 `T=0` 时动量空间里满足 `epsilon(k)=mu` 的等能面（二维时为等能线）。

它把“电子占据态”与“可激发态”分开：
- `epsilon(k) < mu`：已占据；
- `epsilon(k) > mu`：未占据；
- `epsilon(k) = mu`：低能输运、热容、磁振荡等现象最敏感的边界。

## R02

本条目目标是提供一个最小、可运行、可审计的费米面计算闭环：
- 用二维正方晶格紧束缚色散定义能带；
- 用给定目标填充数反推化学势 `mu`；
- 从离散 `k` 网格上提取费米面点集；
- 计算费米速度并做局部线性色散拟合互校。

该 MVP 适合作为后续做 ARPES 拟合、输运近似或多带扩展前的基线。

## R03

`demo.py` 输入输出约定（无交互）：

- 输入（脚本内参数）：
1. 紧束缚参数 `t, t'`；
2. 目标填充 `n_target`；
3. 化学势求解的平滑逆温度 `beta`；
4. `k` 网格密度和局部拟合窗口。

- 输出：
1. 求得的 `mu`、平滑填充与 `T=0` 填充估计；
2. 费米面点数量、费米速度统计；
3. 局部线性色散斜率（`sklearn` / `scipy` / `torch`）；
4. 核心断言通过后输出 `All checks passed.`。

## R04

MVP 使用的核心模型与公式：

1. 二维正方晶格（含次近邻）色散：
`epsilon(kx,ky) = -2t(cos kx + cos ky) - 4t' cos kx cos ky`

2. 费米面定义：
`epsilon(kx,ky) = mu`

3. 有限 `beta` 下平滑填充（便于数值求根）：
`n(mu) = < 1/(exp(beta*(epsilon-mu)) + 1) >_BZ`

4. 费米速度：
`v_F(k) = |grad_k epsilon(k)|`

`grad_k epsilon = (
2t sin kx + 4t' sin kx cos ky,
2t sin ky + 4t' cos kx sin ky
)`

5. 局部线性色散近似（沿法向位移 `q`）：
`epsilon(k_F + q n_hat) - mu ≈ v_F q + O(q^2)`

## R05

设化学势求解网格为 `N x N`，费米面提取网格为 `M x M`：

- 计算能量网格：`O(N^2)`；
- `brentq` 求根每次需一次网格平均，总代价约 `O(I*N^2)`（`I` 为迭代步，通常很小）；
- 费米面边交检测：`O(M^2)`；
- 费米速度统计：`O(P)`（`P` 为费米面点数）。

空间复杂度主要由网格张量决定，约 `O(max(N^2, M^2))`。

## R06

最小闭环由四段组成：

- **闭环 A（化学势）**：给定目标填充，通过 `scipy.optimize.brentq` 反解 `mu`；
- **闭环 B（几何）**：在网格单元边上做线性插值，提取 `epsilon-mu=0` 的费米面点；
- **闭环 C（动力学量）**：在费米面点上计算 `|grad_k epsilon|` 得到 `v_F` 分布；
- **闭环 D（互校）**：对节点附近局部色散分别用 `sklearn`、`scipy`、`torch` 拟合同一斜率并比较一致性。

## R07

优点：
- 费米面定义、提取、速度计算和拟合验证全流程可见；
- 使用通用科学栈实现，便于迁移到更多晶格模型；
- 第三方库只负责数值求根/回归，不隐藏物理主逻辑。

局限：
- 单带二维模型，未包含自能修正与多轨道耦合；
- 费米面由规则网格插值得到，精度受网格密度限制；
- 未引入真实材料参数与实验误差模型。

## R10

正确性直觉：

1. 求解得到的 `mu` 应让平滑填充接近目标填充；
2. 提取出的费米面点应满足 `epsilon(k)-mu` 接近零；
3. 局部色散斜率应接近由解析梯度给出的 `v_F`；
4. 不同拟合器得到的斜率应彼此接近。

## R11

数值稳定策略：

- `fermi_dirac` 里对指数输入做截断，避免 `exp` 上溢；
- 用平滑填充函数求 `mu`，避免 `T=0` 阶梯函数导致根搜索不稳；
- 费米面点提取只在边跨符号时线性插值，避免不必要分支；
- 局部拟合窗口 `q_max` 取小值，保证线性项主导。

## R12

关键参数与调参建议：

- `target_filling`：决定费米能与费米面拓扑；
- `grid_n_mu`：越大，`mu` 反演越稳定；
- `grid_n_fs`：越大，费米面点集越密；
- `beta`：越大越接近 `T=0`，但过大可能导致数值变硬；
- `q_max/q_points`：控制局部线性拟合的“局部性 vs 抗噪性”。

建议先固定 `t,t'`，逐步提高网格密度看 `mu`、`v_F` 统计是否收敛。

## R13

可验证保证（本 MVP）：

- 确定性：固定参数与种子，输出可复现；
- 物理一致性：
1. `|n_smooth - n_target|` 在阈值内；
2. 费米面点数超过最小阈值；
3. 费米面点残差 `|epsilon-mu|` 均值足够小；
4. 三种回归斜率与解析 `v_F` 一致到设定容差。

说明：本问题不是近似算法竞赛任务，无“近似比”定义。

## R14

常见失效模式：

1. `k` 网格过粗导致费米面断裂或锯齿严重；
2. 填充目标过接近带边时，局部拟合点不足；
3. 直接用 `T=0` 阶梯函数求根导致化学势抖动；
4. 法向定义取反并不会改变 `|v_F|`，但会翻转拟合斜率符号，需统一比较方式。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `TBConfig` 定义紧束缚参数、目标填充、网格分辨率、拟合窗口和容差。  
2. `dispersion` 与 `grad_dispersion` 显式给出 `epsilon(k)` 及其梯度，避免把物理关系隐藏在外部库。  
3. `solve_mu_for_target_filling` 在布里渊区网格上构建平滑填充函数 `n(mu)`，用 `scipy.brentq` 求出化学势 `mu`。  
4. `extract_fermi_points` 遍历网格单元四条边，对 `epsilon-mu` 跨符号边做线性插值，得到费米面点集。  
5. `build_fermi_dataframe` 对每个费米面点计算残差与 `v_F=|grad epsilon|`，并整理为 `pandas` 表。  
6. `find_nodal_kf` 在 `kx=ky` 截线上求 `epsilon(k,k)=mu` 的根，定位一个局部费米点用于拟合。  
7. `fit_local_slope_sklearn`、`fit_local_slope_curve_fit`、`fit_local_slope_torch` 分别从同一局部数据估计线性色散斜率并互校。  
8. `main` 汇总打印关键统计并执行断言（填充、残差、点数、斜率一致性），全部通过后输出 `All checks passed.`。  

说明：第三方库只用于数值求根与回归求解器；费米面定义、点提取与物理校验全部在源码中显式展开，不是黑箱调用。
