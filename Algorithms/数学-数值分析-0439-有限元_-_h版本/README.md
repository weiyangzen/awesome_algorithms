## R01

h 版本有限元（h-FEM）指在固定单元阶次（本例为 1 次线性形函数）下，通过缩小网格尺寸 `h` 提高精度。它是有限元法最基础、最常用的收敛路径：不改“每个单元里的多项式次数”，只增加单元数量。

## R02

本题采用 1D Poisson 边值问题作为最小可运行对象：

- 强形式：`-u''(x)=f(x), x∈(0,1)`
- 边界条件：`u(0)=u(1)=0`
- 解析解选择：`u(x)=sin(pi x)`，故 `f(x)=pi^2 sin(pi x)`

输入：网格剖分数 `n_elem`。  
输出：节点数值解 `u_h` 与误差、收敛阶统计。

## R03

核心思想（h 版本）：

1. 把区间 `[0,1]` 划分为 `n_elem` 个小单元，`h=1/n_elem`。
2. 在每个单元上用线性基函数近似 `u_h`。
3. 通过弱形式构造全局线性方程 `A u = b`。
4. 逐步细化网格（`h` 变小）并观测误差衰减率。

当解足够光滑时，线性元通常表现为 `L2` 误差约 `O(h^2)`，`H1` 半范数误差约 `O(h)`。

## R04

弱形式推导（Dirichlet 边界）：

对任意测试函数 `v∈H_0^1(0,1)`，有

`∫_0^1 u'(x)v'(x) dx = ∫_0^1 f(x)v(x) dx`。

离散后在有限维子空间 `V_h ⊂ H_0^1` 中求 `u_h`，满足

`a(u_h,v_h)=l(v_h), ∀ v_h∈V_h`，

其中

- `a(w,v)=∫_0^1 w'v' dx`
- `l(v)=∫_0^1 f v dx`

## R05

线性单元离散公式：

- 每个单元 `e=[x_i,x_{i+1}]` 的局部刚度矩阵
  `K_e = (1/h_e) * [[1,-1],[-1,1]]`
- 局部载荷向量
  `F_e[j] = ∫_{x_i}^{x_{i+1}} f(x) N_j(x) dx`

MVP 中 `F_e` 使用 2 点 Gauss 积分近似计算，再装配到全局矩阵 `A` 和向量 `b`。

## R06

伪代码：

```text
input: n_elem
nodes = linspace(0,1,n_elem+1)
A = 0, b = 0

for each element e=[i,i+1]:
    h = nodes[i+1]-nodes[i]
    Ke = [[1,-1],[-1,1]] / h
    Fe = gauss2_integral(f(x) * [N1, N2])
    scatter Ke, Fe into global A, b

apply Dirichlet u(0)=u(1)=0:
    solve interior system A_ii * u_i = b_i
reconstruct full u with boundary zeros

return nodes, u
```

## R07

正确性直觉：

- 有限元是对弱形式的 Galerkin 投影，离散方程等价于“残差对离散测试空间正交”。
- 对 Poisson 问题，双线性型 `a(·,·)` 对称正定，离散系统在施加 Dirichlet 后可解且稳定。
- 网格细化让离散空间 `V_h` 更丰富，最佳逼近误差下降，数值解随 `h→0` 逼近真解。

## R08

复杂度（1D 线性元，朴素 dense 实现）：

- 装配：`O(n_elem)`（每单元常数规模计算）
- 求解：`O(n^3)`（`n=n_elem-1`，使用 `numpy.linalg.solve` 的 dense 代价）
- 存储：`O(n^2)`（dense 矩阵）

注：若改为稀疏三对角求解，时间和空间可降到近线性。

## R09

数值稳定性要点：

- 局部刚度含 `1/h`，网格极细时条件数会增大，误差受浮点舍入影响。
- 边界条件必须正确施加；若直接保留边界未知量且未约束，系统会奇异或错误。
- 载荷积分若过粗会引入额外积分误差，掩盖 h 收敛阶；本实现用 2 点 Gauss 以减少该影响。

## R10

常见失败模式：

- 单元索引装配错误（局部到全局映射错位）导致矩阵非对称或结果异常。
- 忘记去除/处理边界自由度，造成线性系统病态或解偏移。
- 仅看一个网格尺度就下结论，无法判断是否进入渐近收敛区。

## R13

输出解释：

- `L2_error`：按单元高斯积分计算的 `L2` 误差，理论上对线性元常见二阶收敛。
- `H1_semi_err`：按单元高斯积分计算的 `H1` 半范数误差，理论上常见一阶收敛。
- `L2_rate`/`H1_rate`：相邻网格层误差比得到的经验阶，用于判断是否达到预期渐近收敛区。

## R14

与其他 FEM 策略对比：

- h 版本：固定低阶基函数，靠网格加密提精度，实现简单、鲁棒。
- p 版本：固定网格、提高多项式阶次，对光滑解可能更高效，但实现与积分更复杂。
- hp 版本：同时调 `h` 与 `p`，最灵活但工程复杂度最高。

本任务聚焦 h 版本，是后续 p/hp 与自适应加密的基础。

## R15

工程实践注意事项：

- 生产规模问题建议使用稀疏矩阵存储与求解器（如 CG/多重网格预条件）。
- 对复杂几何和高维问题，应引入网格质量检查（畸变、最小角等）。
- 误差评估可加入后验误差估计器，驱动“局部”而非“全局均匀”h 细化。

## R18

`demo.py` 源码级算法流程（9 步）：

1. `main()` 定义网格序列 `[8,16,32,64,128]`，调用 `run_h_refinement_study()`。
2. `run_h_refinement_study()` 对每个 `n_elem` 调用 `solve_poisson_fem()`，并记录误差与经验阶。
3. `solve_poisson_fem()` 先调用 `assemble_system(n_elem)` 生成全局刚度矩阵 `A` 和载荷 `b`。
4. `assemble_system()` 在每个单元构造局部刚度 `K_e=(1/h)[[1,-1],[-1,1]]`。
5. `assemble_system()` 用 2 点 Gauss 在单元上计算局部载荷 `F_e`（含形函数 `N1,N2` 与雅可比 `h/2`）。
6. `assemble_system()` 将 `K_e, F_e` 按单元端点索引散装配到全局 `A, b`。
7. `solve_poisson_fem()` 施加齐次 Dirichlet：截取内部子系统 `A_ii u_i = b_i` 并用 `numpy.linalg.solve` 求解。
8. `compute_errors()` 在每个单元做高斯积分，与解析解 `sin(pi x)` 及其导数比较，返回 `L2_error` 与 `H1_semi_err`。
9. `main()` 打印每层网格的 `h`、误差和经验收敛阶，形成 h 版本 FEM 的最小验证闭环。
