## R01

相场方法（Phase-Field Method）是一类用于描述界面演化的连续场方法。

核心思想：
- 用一个连续序参量 `phi(x,t)` 表示材料相态，而不是显式追踪尖锐界面；
- `phi≈+1` 与 `phi≈-1` 表示两相，界面区域是平滑过渡带；
- 通过自由能泛函与梯度流方程驱动演化，实现相分离、晶体生长、界面运动等模拟。

本条目实现的 MVP 选用 Allen-Cahn 型相场方程。

## R02

本条目问题定义（离散计算版）：

- 输入：
  - 二维网格尺寸 `nx, ny`；
  - 空间步长 `dx`、时间步长 `dt`、总步数 `n_steps`；
  - 界面宽度参数 `epsilon`；
  - 初始相场 `phi0`（由圆形晶核 + 随机扰动构造）。
- 输出：
  - 终态相场 `phi`；
  - 若干采样时刻的自由能 `E(t)`；
  - 若干采样时刻的正相体积分数 `mean(phi>0)`。

## R03

MVP 的核心方程：

`phi_t = epsilon^2 * Lap(phi) - (phi^3 - phi) / epsilon^2`

对应自由能：

`F(phi)=∫ [ (epsilon^2/2)|grad(phi)|^2 + (phi^2-1)^2/(4*epsilon^2) ] dOmega`

算法策略：
- 空间离散：二维五点 Laplace（周期边界）；
- 时间推进：显式 Euler；
- 诊断：监控自由能是否整体下降，验证梯度流行为。

## R04

时间复杂度：

- 单步更新需要遍历 `nx*ny` 网格并做常数次邻域访问，复杂度 `O(nx*ny)`；
- 总步数为 `n_steps`，总时间复杂度 `O(nx*ny*n_steps)`；
- 记录日志（每 `report_every` 步）开销远小于主循环，不改变量级。

## R05

空间复杂度：

- 主场 `phi` 占 `O(nx*ny)`；
- RHS 与拉普拉斯中间量也为 `O(nx*ny)`；
- 日志数组仅 `O(n_steps/report_every)`。

整体空间复杂度为 `O(nx*ny)`。

## R06

一步离散更新微例（仅展示结构）：

若某网格点当前 `phi=0.2`，邻点离散 Laplace 近似为 `-0.5`，取 `epsilon=1.5`：

- 扩散项：`epsilon^2 * Lap = 2.25 * (-0.5) = -1.125`
- 反应项：`(phi^3-phi)/epsilon^2 = (0.008-0.2)/2.25 = -0.085333...`
- RHS：`-1.125 - (-0.085333...) = -1.039666...`

若 `dt=0.03`，则
`phi_new = phi + dt*RHS = 0.2 + 0.03*(-1.039666...) ≈ 0.16881`。

这说明局部状态由“界面平滑 + 势阱回拉”共同决定。

## R07

相场方法的重要性：

- 避免了显式界面追踪（如 front tracking）的拓扑重连困难；
- 可以自然处理合并、断裂、形核等复杂界面事件；
- 在材料科学、流体两相、图像分割、晶体生长中广泛使用；
- 数值实现上可直接落在常规 PDE 离散框架中。

## R08

理论基础（本 MVP 对应）：

1. 构造双稳态自由能密度 `(phi^2-1)^2/(4*epsilon^2)`，使 `phi=±1` 成为稳定相。  
2. 加入梯度惩罚 `(epsilon^2/2)|grad(phi)|^2` 控制界面宽度。  
3. 对自由能做变分得到化学势：  
   `deltaF/deltaphi = (phi^3-phi)/epsilon^2 - epsilon^2*Lap(phi)`。  
4. 采用梯度流 `phi_t = -deltaF/deltaphi`，得到 Allen-Cahn 演化方程。  
5. 连续情形下满足 `dF/dt <= 0`（能量耗散）。

## R09

适用条件与局限：

- 适合界面厚度可接受“弥散表示”的问题；
- 适合演化机制主要由局部自由能和梯度能控制的场景；
- 显式 Euler 对时间步长敏感，`dt` 过大会数值发散；
- Allen-Cahn 不守恒总相量（与 Cahn-Hilliard 不同），建模时要确认物理含义。

## R10

正确性框架（工程可验证版本）：

1. 离散算子实现与 PDE 右端一致：`RHS = eps^2 Lap - (phi^3-phi)/eps^2`。  
2. 每一步按 `phi^{n+1}=phi^n+dt*RHS(phi^n)` 显式推进。  
3. 在稳定步长下，离散轨迹应保留梯度流特征：自由能总体下降。  
4. `demo.py` 通过记录 `E(t)` 并断言其采样点单调不增来验证这一点。  
5. 若出现 `nan/inf` 或能量异常上升，直接报错，提示缩小 `dt`。

## R11

误差与稳定性：

- 时间离散为一阶精度（显式 Euler，`O(dt)`）；
- 空间离散为二阶中心差分 Laplace（`O(dx^2)`）；
- 非线性反应项会加重刚性，稳定步长通常需要更小；
- 本实现通过保守参数（`dt=0.03, epsilon=1.5, dx=1`）保证演示稳定。

## R12

性能视角：

- 主要成本来自每步的 `np.roll` 与逐点代数运算；
- 相比隐式法，显式法每步便宜、实现简单，但需要更多步数；
- 对教学和算法透明性，显式 MVP 更便于审计和二次扩展；
- 若要大规模计算，可进一步做向量化并行或半隐式谱方法。

## R13

本目录可验证保证：

- `demo.py` 固定随机种子，输出可复现；
- 输出采样时刻的 `energy` 与 `positive_fraction`，可直接检查演化趋势；
- 内置断言：
  - 总能量下降；
  - 采样能量单调不增；
  - 场变量全为有限值且幅值未爆炸；
- 正常运行结束会打印 `All checks passed.`。

## R14

鲁棒性与常见失效模式：

- 参数非法（负步长、过小网格、非有限值）会在配置校验阶段报错；
- `dt` 过大时可能出现数值爆炸，表现为 `nan/inf` 或能量上升；
- `epsilon` 太小会导致界面过薄，需要更细网格和更小时间步；
- 周期边界并不适合所有物理问题，真实边界条件需按任务改写。

## R15

实现设计（对应 `demo.py`）：

- `SimulationConfig`：集中管理并校验数值参数。  
- `build_initial_field`：构造圆形晶核初值。  
- `laplacian_periodic`：周期边界五点 Laplace。  
- `allen_cahn_rhs`：计算 PDE 右端。  
- `free_energy`：计算离散 Ginzburg-Landau 能量。  
- `run_simulation`：主循环推进并记录日志。  
- `print_report` + `run_checks`：输出诊断并做自检。  

实现完全展开了相场更新公式，没有调用黑箱 PDE 求解器。

## R16

相关算法链路：

- 同族：Cahn-Hilliard（守恒型相场）、相场晶体（PFC）。
- 对照：Level Set、VOF、Front Tracking（显式界面法）。
- 数值改进：半隐式 Fourier 谱法、能量稳定格式（convex splitting）。
- 多物理耦合：可与弹性场、流场、温度场联立。

## R18

源码级算法流程拆解（`run_simulation` 主流程，8 步）：

1. 读取并校验 `SimulationConfig`，提前拦截不合法参数。  
2. 调用 `build_initial_field` 生成 `phi0`：中心圆核为 `+1`，背景为 `-1`，叠加小噪声。  
3. 进入时间循环，每到采样步调用 `free_energy` 计算离散自由能并记录正相比例。  
4. 通过 `laplacian_periodic` 用 `np.roll` 构造五点离散 Laplace（周期边界）。  
5. 在 `allen_cahn_rhs` 中组装右端：`eps^2*Lap(phi) - (phi^3-phi)/eps^2`。  
6. 执行显式 Euler 更新：`phi <- phi + dt*RHS`。  
7. 每步检查 `phi` 是否全为有限值，若出现异常立即抛错终止。  
8. 循环结束后返回 `SimulationResult`，再由 `print_report/run_checks` 完成可视化诊断与自动验证。  

说明：本实现没有调用任何现成“相场黑盒”库函数，关键离散与更新细节都在源码中显式实现。
