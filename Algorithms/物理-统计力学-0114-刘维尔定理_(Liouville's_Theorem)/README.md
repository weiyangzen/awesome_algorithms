## R01

刘维尔定理说明：对哈密顿系统，**相空间流是不可压缩的**。若相空间坐标记为 `z=(q,p)`，概率密度 `rho(z,t)` 满足：

`drho/dt = partial_t rho + {rho, H} = 0`

等价地，相空间速度场 `F=(dot(q), dot(p))` 的散度为 0：

`nabla_z · F = sum_i [partial(dot(q_i))/partial(q_i) + partial(dot(p_i))/partial(p_i)] = 0`

这意味着随系统演化，微元相空间体积保持不变。

## R02

在统计力学中，刘维尔定理是“微观动力学 -> 系综演化”桥梁：

- 支撑微正则系综中“能壳上均匀测度”的合理性；
- 解释为何哈密顿演化不会把概率质量凭空压缩到低维集合；
- 为玻尔兹曼方程、BBGKY 层级与遍历假设提供基础背景。

## R03

本目录 MVP 用一个最小可验证实验展示定理核心：

- 系统：1D 谐振子 `H(q,p)=p^2/(2m)+k q^2/2`；
- 比较两种离散推进：
  - 辛欧拉（保持辛结构，应近似体积守恒）；
  - 显式欧拉（非辛，通常会人为膨胀/收缩体积）。

目标不是搭建通用分子动力学框架，而是用最短代码验证“哈密顿流不可压缩”与“离散方法会破坏/保持该性质”的差异。

## R04

本实现使用的关键方程：

1. 哈密顿方程：
   - `dot(q) = partial H / partial p = p/m`
   - `dot(p) = -partial H / partial q = -k q`
2. 相空间散度：
   - `partial(dot(q))/partial q + partial(dot(p))/partial p = 0 + 0 = 0`
3. 显式欧拉一步映射：
   - `q_{n+1}=q_n + dt * p_n/m`
   - `p_{n+1}=p_n - dt * k * q_n`
   - 雅可比行列式 `det = 1 + (dt^2 k / m) > 1`
4. 辛欧拉（kick-drift）一步映射：
   - `p_{n+1}=p_n - dt * k * q_n`
   - `q_{n+1}=q_n + dt * p_{n+1}/m`
   - 雅可比行列式 `det = 1`

因此，离散映射级别就能看出谁更符合刘维尔定理。

## R10

正确性直觉：

1. 哈密顿流满足泊松结构，理论上 `nabla·F=0`；
2. 体积保持可由映射雅可比 `det(J)=1` 判断；
3. 辛欧拉离散保留辛结构，通常守住体积性质；
4. 显式欧拉不保辛，`det(J) != 1`，长期演化会系统性扭曲相空间体积。

因此，若代码正确，应同时观察到“散度近零 + 辛法体积守恒 + 非辛法体积漂移”。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 固定 `m,k,dt,steps` 与随机种子，采样初始相空间云团 `[(q_i,p_i)]`。  
2. `hamiltonian_vector_field` 显式给出 `dq/dt=p/m`、`dp/dt=-kq`，不依赖外部黑盒求解器。  
3. `numerical_phase_divergence` 在一点上做中心差分，计算 `partial(dq/dt)/partial q + partial(dp/dt)/partial p`。  
4. `run_ensemble(..., method="symplectic")` 调用 `symplectic_euler_step` 推进全部样本。  
5. 同样地，`run_ensemble(..., method="explicit")` 调用 `explicit_euler_step` 做对照推进。  
6. 每个方法都用 `covariance_area_proxy = sqrt(det(cov))` 计算初末相空间“面积代理”，得到面积比。  
7. `one_step_jacobian_det` 给出离散映射的一步雅可比行列式，并计算理论累计缩放 `det(step)^steps`。  
8. `main` 打印散度与两种方法报告，并执行断言：散度近零、辛法面积比约 1、显式欧拉显著膨胀。  

第三方库使用说明：仅使用 `numpy` 进行数组、协方差与线性代数基础计算；刘维尔定理验证链路（方程、离散映射、雅可比、体积判据）全部在源码中逐步实现。
