# 克尔效应 (Kerr Effect)

## R01

克尔效应描述介质折射率随光强变化：
`n(I) = n0 + n2 * I`。

在脉冲光传播中，这会引入与时间相关的非线性相位：
`phi_nl(t) = k0 * n2 * I(t) * L`，其中 `k0=2*pi/lambda`。

当 `I(t)` 非常数时，`phi_nl(t)` 会产生瞬时频率偏移 `delta_omega(t)=-d(phi_nl)/dt`，形成自相位调制（SPM）与谱展宽。

## R02

克尔非线性是非线性光学中的基础三阶效应：
- 19 世纪后期通过电光实验被系统观察（液体 Kerr cell）
- 激光出现后，Kerr 非线性成为超快光学、光纤通信和频谱整形的核心机制
- 现代工程中通常把 `n2`、`B-integral`、SPM 谱展宽作为设计主指标

本条目以最小算法演示“强度 -> 折射率变化 -> 相位调制 -> 频谱变化”的完整链路。

## R03

MVP 解决的具体问题：
给定脉冲参数（波长、脉宽、能量、有效模场面积、介质长度、`n2`），数值计算：
- 峰值功率与峰值强度
- 非线性折射率增量 `Delta n_max`
- 峰值非线性相位 `phi_nl,max`（可视作简化 `B-integral`）
- SPM 导致的瞬时频率偏移与频谱 RMS 带宽变化

并通过能量缩放实验检查 `phi_nl,max` 对脉冲能量的近线性比例关系。

## R04

脚本中使用的核心方程：

- 高斯功率包络：
  `P(t)=P0 * exp(-4 ln2 * t^2 / tau_fwhm^2)`
- 峰值功率（由脉冲能量反推）：
  `P0 = E_pulse / (tau_fwhm * sqrt(pi)/(2*sqrt(ln2)))`
- 强度与折射率变化：
  `I(t)=P(t)/Aeff`，`Delta n(t)=n2*I(t)`
- 非线性相位：
  `phi_nl(t)=k0*n2*I(t)*L`
- 瞬时频率偏移：
  `delta_omega(t)=-d(phi_nl)/dt`
- 频谱宽度：
  对 `|FFT(E)|^2` 归一化后计算 RMS 带宽

## R05

`demo.py` 输出三块结果：
- `Core metrics`：峰值功率、峰值强度、`Delta n_max`、`phi_nl,max`、RMS 带宽展宽倍数
- `Energy scaling check`：不同脉冲能量下 `phi_nl,max` 的比例关系与单调性
- `Temporal sample`：若干时间点的 `P(t)`, `I(t)`, `phi_nl(t)`, `delta_omega(t)` 便于审计

末尾给出 `Validation: PASS/FAIL`。

## R06

实现策略：
- 用 `KerrConfig` 固化参数，避免位置参数错位
- 用向量化时间网格计算功率、强度、折射率增量和相位
- 用 `np.gradient` 计算相位时间导数得到瞬时频率偏移
- 用 `scipy.fft` 计算输入/输出场的频谱并提取 RMS 带宽
- 用 `pandas` 结构化打印审计表与能量扫描表

## R14

常见失效模式：
1. `Aeff` 过小或能量过大导致相位过强，薄介质近似失真
2. 时间窗太窄导致脉冲被截断，引入频谱伪展宽
3. 采样点太少导致频率分辨率不足
4. 把该模型误用到强色散/长距离传播场景

## R18

`demo.py` 源码级算法流（8 步，非黑箱）：
1. 读取 `KerrConfig` 并检查 `lambda, n2, L, Aeff, tau, E, N` 等参数均为正值。
2. 由脉冲能量和高斯积分闭式公式反推峰值功率 `P0`，建立时间网格 `t` 与功率包络 `P(t)`。
3. 计算 `I(t)=P(t)/Aeff`、`Delta n(t)=n2*I(t)`，得到强度诱导的折射率调制。
4. 计算 `phi_nl(t)=k0*n2*I(t)*L`，并用 `np.gradient` 得到 `delta_omega(t)`。
5. 构造输入场 `Ein=sqrt(P)` 与输出场 `Eout=Ein*exp(i*phi_nl)`（线性常相位不影响谱宽，故省略）。
6. 用 `scipy.fft` 计算输入/输出谱功率密度，并显式做归一化与 RMS 带宽统计。
7. 对多组能量因子重复步骤 2-6，形成 `phi_nl,max` 与谱宽对能量的扫描表。
8. 执行两项阈值检查（线性比例离散度与谱宽单调性），打印 `PASS/FAIL`。

第三方库分工可追踪：
- `numpy`：向量化物理量计算与梯度
- `scipy.fft`：离散傅里叶变换
- `pandas`：结果表格化

核心物理关系（`n(I)`、`phi_nl`、`delta_omega`）均在源码内显式实现。
