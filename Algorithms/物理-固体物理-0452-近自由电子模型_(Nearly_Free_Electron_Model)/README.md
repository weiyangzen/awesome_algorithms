# 近自由电子模型 (Nearly Free Electron Model)

- UID: `PHYS-0432`
- 学科: `物理`
- 分类: `固体物理`
- 源序号: `452`
- 目标目录: `Algorithms/物理-固体物理-0452-近自由电子模型_(Nearly_Free_Electron_Model)`

## R01

近自由电子模型（Nearly Free Electron Model, NFEM）用于描述电子在“弱周期势”中的能带形成机制。它可视为自由电子模型与紧束缚模型之间的桥梁：

- 电子主导行为接近自由粒子；
- 晶格周期势 `V(x)` 作为小扰动引入；
- 在布里渊区边界处，简并自由电子态被耦合并开隙。

本条目提供一个一维、可运行、可校验的最小实现。

## R02

本 MVP 的问题设定：

1. 一维晶格，晶格常数 `a`，倒格矢 `G=2π/a`；
2. 周期势取单一谐波 `V(x)=2V1 cos(Gx)`；
3. 使用截断平面波基 `|k+mG>`（`m=-M,...,M`）；
4. 在第一布里渊区 `k ∈ [-G/2, G/2]` 求能带。

目标：

- 数值求解前几条能带；
- 验证区边界能隙 `E_gap ≈ 2|V1|`（弱势极限）；
- 与 2×2 解析反交叉公式及二阶微扰结果交叉核验。

## R03

核心方程：

1. 自由电子能量（约化单位）
   `E0(q)=alpha*q^2`，其中 `alpha = hbar^2/(2m_e)`，脚本中记为 `kinetic_prefactor`。
2. 平面波基哈密顿量：
   `H_mn(k) = E0(k+mG) δ_mn + V1(δ_{m,n+1}+δ_{m,n-1})`
3. 区边界附近 2×2 子空间（`|k>`,`|k-G>`）解析本征值：
   `E± = (Ek + E_{k-G})/2 ± sqrt(((Ek-E_{k-G})/2)^2 + V1^2)`
4. 在 `k=G/2` 时，`E_gap = E+ - E- = 2|V1|`。

## R04

离散化与计算流程：

- `k` 方向采用均匀网格（默认 `n_k=241`）；
- 每个 `k` 上构造一个 `(2M+1) x (2M+1)` 实对称矩阵（默认 `M=3`）；
- 调用厄米本征值求解器得到能带；
- 以最低两条带计算直接带隙 `E2-E1`。

这一实现保留了 NFEM 的关键物理项，同时避免过大数值开销。

## R05

最小工具栈与职责：

- `numpy`：网格、矩阵与向量计算；
- `scipy.linalg.eigh`：厄米矩阵本征值；
- `pandas`：结构化结果表输出；
- `torch.linalg.eigvalsh`：独立后端交叉验证谱一致性；
- `scikit-learn`：线性回归拟合 `E_gap` 对 `V1` 的比例关系。

第三方库只提供基础数值能力；NFEM 的模型构建、比较路径、误差指标和断言逻辑都在源码中显式实现。

## R06

`demo.py` 的主要输出：

1. 配置参数；
2. 第一、二能带的采样表（含 `gap12`）；
3. 三组误差指标：
   - SciPy vs PyTorch 谱差；
   - 全模型 vs 2×2 近边界解析误差；
   - 全模型 vs 二阶微扰（远离区边界）误差；
4. `E_gap` 与 `2|V1|` 的一致性；
5. 多个 `V1` 下的带隙扫描与线性拟合结果。

## R07

正确性校验闭环：

1. 数值后端一致：`max|E_scipy - E_torch|` 应接近机器精度；
2. 局部理论一致：区边界附近与 2×2 解析模型误差应小；
3. 远离简并处，二阶微扰近似应与数值第一带接近；
4. 带隙应在区边界处最小，且满足 `E_gap ≈ 2|V1|`；
5. 弱势区间内 `E_gap-V1` 线性拟合斜率应接近 `2`。

脚本使用断言将这些物理预期转为自动化检查。

## R08

默认参数（`NFEMConfig`）：

- `lattice_constant = 1.0`
- `kinetic_prefactor = 1.0`
- `fourier_potential_v1 = 0.08`
- `plane_wave_cutoff = 3`
- `n_k = 241`
- `two_level_window_ratio = 0.22`
- `perturbation_exclusion_ratio = 0.20`

这组参数保证：势足够弱（近自由前提成立）、网格足够密、矩阵维数足够小，适合教学与验证。

## R09

复杂度分析：

设基维数 `B=2M+1`，`k` 点数 `N_k`。

- 构造单个哈密顿量：`O(B^2)`；
- 单个 `k` 的稠密本征分解：`O(B^3)`；
- 全布里渊区求带：`O(N_k * B^3)`；
- 势强度扫描若有 `N_v` 组：`O(N_v * N_k * B^3)`。

在默认 `B=7`、`N_k=241` 时，计算量很小，CPU 可快速完成。

## R12

物理含义总结：

- 若无周期势（`V1=0`），第一、二带在区边界可相交；
- 弱周期势引入布拉格散射，耦合简并态并导致反交叉；
- 反交叉大小由 `V_G`（本实现中 `V1`）控制，形成布里渊区边界能隙；
- 这正是“金属/绝缘行为与能带结构”联系的基础机制之一。

## R13

适用范围与局限：

适用：

- 周期势较弱、可用平面波低阶截断描述的场景；
- 教学演示：从自由电子到能隙形成的最小模型。

局限：

- 一维、单电子、单谐波势，未包含真实材料复杂势谱；
- 未纳入电子-电子相互作用、自旋轨道耦合、无序与温度效应；
- 不直接替代第一性原理能带计算。

## R14

常见错误与排查：

1. `G=2π/a` 写错导致区边界位置错误；
2. 平面波索引与矩阵下标映射错位，导致耦合项错误；
3. 在区边界附近误用微扰公式（分母接近 0）造成偏差；
4. 误把 2×2 局部模型当作全区精确解；
5. 忽略本征值排序导致能带错配。

## R16

与其他模型关系：

- 与自由电子模型：NFEM 在其基础上加入弱周期势；
- 与紧束缚模型：NFEM 是“弱势、离域”极限，TB 是“强势、局域”极限；
- 与 `k·p`：都在特定 `k` 附近建立有效低维哈密顿量；
- 与 DFT：NFEM 可作为理解 DFT 能带开隙来源的低阶解释框架。

## R18

`demo.py` 源码级算法流程（9 步）：

1. 在 `NFEMConfig` 中固定晶格常数、势强 `V1`、基截断 `M` 与 `k` 网格参数。  
2. `k_grid_first_bz()` 生成第一布里渊区离散 `k` 点；`plane_wave_indices()` 生成平面波索引 `m`。  
3. `build_hamiltonian_k()` 按 `H_mn(k)=E0(k+mG)δ_mn+V1(δ_{m,n+1}+δ_{m,n-1})` 构造截断哈密顿量。  
4. `solve_bands_scipy()` 对每个 `k` 调用 `scipy.linalg.eigh`，得到完整数值能带。  
5. `max_scipy_torch_eigen_diff()` 对同一矩阵使用 `torch.linalg.eigvalsh` 再求谱，计算后端差异。  
6. `compare_two_level_model()` 在 `+G/2` 邻域调用 `two_level_near_boundary()` 的 2×2 解析式，与前两条数值带比较。  
7. `compare_perturbation()` 在远离区边界区域调用 `perturbative_first_band()`，验证二阶微扰对第一带的近似。  
8. `first_direct_gap()` 提取最小直接带隙；`fit_gap_vs_potential()` 扫描多个 `V1` 并用 `LinearRegression` 拟合 `Egap~V1`。  
9. `main()` 汇总表格与指标，执行断言（谱一致性、局部解析一致性、微扰一致性、`Egap≈2|V1|`、线性斜率约 2），全部通过后打印 `All checks passed.`。

说明：第三方库未替代算法核心。模型构造、物理近似分区、误差定义、阈值校验与结论生成均由源码逐步展开，可直接审查每一环。
