# 超精细结构 (Hyperfine Structure)

- UID: `PHYS-0245`
- 学科: `物理`
- 分类: `原子物理`
- 源序号: `248`
- 目标目录: `Algorithms/物理-原子物理-0248-超精细结构_(Hyperfine_Structure)`

## R01

超精细结构描述的是电子总角动量 `J` 与核自旋 `I` 耦合后，对同一电子态进一步产生的精细能级分裂。零磁场下常用的有效哈密顿量可写为磁偶极项与电四极项两部分，对应超精细常数 `A` 与 `B`。

本条目的 MVP 聚焦“零场超精细能级”计算，不引入外磁场 Zeeman 项，因此目标是把 `I,J,A,B` 显式映射到允许的 `F` 多重态及其能量。

## R02

该算法条目的价值在于：

- 把抽象角动量耦合转成可执行、可检查的数值流程；
- 给出超精细分裂的最小工程接口（输入 `I,J,A,B`，输出 `F` 能级与间隔）；
- 能直接验证 Landé interval rule（`B=0` 时）；
- 为后续扩展到 Breit-Rabi 曲线、原子钟跃迁、谱线拟合提供基础模块。

## R03

`demo.py` 的输入输出约定（无交互）：

- 输入（脚本内固定）：
1. 一组氢基态示例参数：`I=1/2, J=1/2, A=1420.405751768 MHz, B=0`；
2. 一组含 `B` 项示例参数：`I=3/2, J=3/2, A=84.7185 MHz, B=12.4965 MHz`；
3. 一个 `B=0` 的合成案例用于严格验证 Landé 间隔律。

- 输出：
1. 每个案例的 `F` 能级表（`K, E_A, E_B, E_total`，含 MHz/GHz/J/eV）；
2. 相邻 `F` 间隔表；
3. 质心检查结果与最终 `All checks passed.`。

## R04

MVP 使用的核心公式：

1. 允许量子数：
`F = |I-J|, |I-J|+1, ..., I+J`。

2. Casimir 量：
`K = F(F+1) - I(I+1) - J(J+1)`。

3. 磁偶极项（MHz）：
`E_A(F) = (A/2) * K`。

4. 电四极项（MHz，需 `I>=1` 且 `J>=1`）：
`E_B(F) = B * [ (3/4)K(K+1) - I(I+1)J(J+1) ] / [2I(2I-1)J(2J-1)]`。

5. 总能量：
`E(F) = E_A(F) + E_B(F)`。

6. `B=0` 时 Landé 间隔律：
`E(F+1)-E(F)=A(F+1)`。

## R05

令 `N_F = I + J - |I-J| + 1` 为超精细子能级数：

- 枚举 `F` 值：`O(N_F)`；
- 每个 `F` 计算常数时间：`O(1)`；
- 建表与相邻间隔计算：`O(N_F)`；
- 空间复杂度：`O(N_F)`。

该模型规模很小，适合教学与自动回归测试。

## R06

`demo.py` 的最小闭环：

- **闭环 A（物理公式）**：由 `I,J,A,B` 与 Casimir 公式直接计算 `E(F)`；
- **闭环 B（工程输出）**：把每个 `F` 的能量转换为 MHz/GHz/J/eV 并表格化；
- **闭环 C（物理校验）**：验证氢基态两能级分裂、加权质心守恒、Landé 间隔律。

## R07

优点：

- 公式到代码一一对应，不依赖黑箱原子结构库；
- 同时覆盖 `A` 项与 `B` 项，兼容 `I=J=1/2` 与更高角动量情形；
- 内置确定性断言，便于 CI 与回归。

局限：

- 仅处理零磁场静态超精细结构；
- 未加入 Zeeman、Breit-Rabi 非线性混合与跃迁强度；
- 未覆盖碰撞展宽、辐射转移、QED 高阶修正。

- 依赖：`numpy`, `pandas`, `scipy`（仅用常数与单位换算，核心算法在源码显式实现）。

## R09

适用场景：

- 原子物理课程中的超精细能级计算演示；
- 快速构造某个 `(I,J)` 态的 `F` 能级与间隔表；
- 作为更复杂谱线模型的前置 sanity check。

不适用场景：

- 需要磁场下完整 Breit-Rabi 曲线与跃迁选择定则强度；
- 需要实验级频率拟合（MHz 以下误差控制）；
- 需要多电子相关与外场耦合的高精度建模。

## R10

正确性直觉：

1. 允许的 `F` 必须是从 `|I-J|` 到 `I+J` 的等步长序列；
2. `B=0` 时能级间隔应按 `A(F+1)` 线性递增；
3. 用简并度 `2F+1` 加权后的能量质心应为 0（相对重心定义）；
4. 对氢 `1S1/2`，应只得到 `F=0,1` 两条并且分裂等于 `A`。

## R11

数值稳定策略：

- 全部使用 `float64`；
- `E_B` 仅在 `I>=1` 且 `J>=1` 时计算，避免分母退化；
- 断言使用严格容差（绝对误差），确保回归可复现；
- 单位转换统一由 `h` 与 `electron_volt` 完成，避免手写换算常数误差。

## R12

关键参数：

- `I, J`：决定可用 `F` 子空间维度；
- `A_MHz`：磁偶极超精细常数，主导零场分裂尺度；
- `B_MHz`：电四极常数，仅在高角动量情形贡献；
- 校验阈值：质心误差、Landé 间隔误差、氢基态分裂误差。

调参建议：

- 若只想验证 Landé 间隔律，设 `B=0`；
- 若要观察 `B` 项导致的非等间隔，选 `I>=1, J>=1` 并设置非零 `B`；
- 若后续对接实验数据，可将案例参数改为对应谱项的已知常数。

- 近似比保证：N/A（非优化近似问题）。

## R14

常见失效模式：

1. `F` 枚举上界/下界写错，漏掉合法能级；
2. 忘记 `E_B` 的适用条件，导致分母出现 `0`；
3. 把 `A`、`B` 的 MHz 单位直接当作 Hz 使用；
4. 用未加权平均代替 `2F+1` 加权质心，造成错误“守恒”判断。

## R18

`demo.py` 源码级算法流程（8 步）：

1. 定义 `HyperfineCase` 数据结构，明确输入参数 `I,J,A_MHz,B_MHz` 的类型与语义。  
2. `allowed_f_values` 依据角动量耦合规则生成 `F` 序列，不把量子数关系交给外部库。  
3. `k_value` 计算 Casimir 量 `K`，为 `A/B` 两项提供统一中间变量。  
4. `hyperfine_energy_mhz` 显式实现 `E_A` 与 `E_B` 公式，并在 `I<1` 或 `J<1` 时关闭 `B` 项。  
5. `build_levels_table` 遍历全部 `F`，构造 `pandas` 表，同时完成 MHz -> GHz/J/eV 的单位换算。  
6. `build_intervals_table` 计算相邻 `F` 间隔；当 `B=0` 时附上 Landé 理论值用于逐项对照。  
7. `run_checks` 执行三组断言：氢基态两能级分裂、加权质心守恒、`B=0` 的 Landé 间隔律。  
8. `main` 按案例打印能级与间隔报告，触发断言并在通过后输出 `All checks passed.`。  

说明：`numpy/pandas/scipy` 在此仅承担数值容器和常数换算角色，超精细结构核心算法（`F` 枚举、`K` 计算、`A/B` 能量公式与校验逻辑）全部在源码中展开。
