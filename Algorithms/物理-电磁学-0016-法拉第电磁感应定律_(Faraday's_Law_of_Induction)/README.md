## R01

法拉第电磁感应定律描述了**磁通量变化如何产生感应电动势**。对 `N` 匝线圈：

`epsilon(t) = - d(lambda)/dt,  lambda(t) = N * Phi(t)`

其中 `lambda` 是磁链、`Phi` 是单匝磁通。负号体现楞次定律（感应效应总是反抗引起它的通量变化）。

## R02

在电磁学中，这条定律是发电机、变压器、感应加热、无线充电等技术的核心。它把“场的时间变化”与“电路可测量量（电压）”直接连接起来，是麦克斯韦方程组中旋度方程的电路层表达。

## R03

本条目 MVP 解决两个最小但完整的问题：

- 已知时变 `B(t)`，通过数值微分计算感应电动势 `epsilon(t)`；
- 将该 `epsilon(t)` 作为 RL 回路激励，验证 `L di/dt + R i = epsilon` 的数值响应与解析稳态是否一致。

目标是把“物理公式 -> 数值算法 -> 可运行验证”闭环打通。

## R04

本实现采用的核心方程：

1. 磁链定义：`lambda(t) = N * A * B_n(t)`（假设磁场在截面积 `A` 上均匀且与法向同向）。
2. 法拉第定律：`epsilon(t) = -d(lambda)/dt`。
3. 正弦磁场示例：`B(t) = B0 * sin(omega t + phase)`。
4. 对应解析电动势：`epsilon_ref(t) = -N A B0 omega cos(omega t + phase)`。
5. RL 回路动力学：`L di/dt + R i = epsilon(t)`。
6. 磁场能量密度（附加检查）：`u = B^2/(2*mu0)`。

## R05

设时间采样点数为 `n`：

- 磁链计算：`O(n)`；
- 有限差分求导：`O(n)`；
- Euler 积分 RL 方程：`O(n)`；
- 谐波最小二乘拟合（常数维度线性回归）：`O(n)`。

总体时间复杂度 `O(n)`，空间复杂度 `O(n)`（保留时序用于验证和报告）。

## R06

`demo.py` 提供两个主演示和一个附加指标：

- **Demo A（法拉第定律校验）**：数值 `epsilon_num` 与解析 `epsilon_ref` 对比，输出最大/平均绝对误差，并做楞次符号检查。
- **Demo B（RL 响应）**：用 `epsilon(t)` 驱动 RL 回路，比较数值稳态谐波系数与理论频域解。
- **Extra（能量密度）**：给定 `B` 样本，计算 `u=B^2/(2mu0)` 的均值与最大值。

## R10

正确性直觉：

1. 若 `B` 不变，则 `dPhi/dt=0`，应无感应电压；
2. `B` 增大时，`epsilon` 应产生反向效应（负号）；
3. 正弦 `B` 的导数仍为正弦/余弦，便于精确对照；
4. RL 稳态必须满足频域阻抗关系 `Z=R+j omega L`，数值仿真应逼近该结果。

## R11

数值稳定与精度策略：

- 导数采用中心差分，端点用单边差分；
- 使用均匀时间网格并显式检查 `dt` 一致性；
- RL 的 `dt` 选得足够小以压低 Euler 离散误差；
- 稳态比较只取后段样本，规避初始暂态影响。

## R12

关键参数及含义：

- `num_turns (N)`：线圈匝数，线性放大电动势；
- `area (A)`：有效截面积；
- `b0`、`omega`、`phase`：磁场激励幅值/角频率/相位；
- `resistance (R)`、`inductance (L)`：RL 回路参数；
- `time` 网格密度：决定导数与积分误差。

调参建议：

- 误差偏大时优先加密时间采样；
- 若频率提高，应同步减小 `dt`；
- 为保证稳定比较，仿真时长需覆盖多个时间常数 `L/R`。

## R14

常见失效模式：

1. 时间网格非均匀，导致差分与积分公式不成立；
2. `dt` 过大，Euler 积分相位/振幅偏差明显；
3. 把磁场方向变化或面积变化忽略时，物理建模与场景不匹配；
4. 用全部时段拟合稳态，混入暂态导致“理论不准”的假象。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `sinusoidal_b_field` 生成输入磁场 `B(t)`，作为全流程驱动信号。  
2. `magnetic_flux_linkage` 将 `B(t)` 映射成磁链 `lambda(t)=NAB(t)`。  
3. `finite_difference_derivative` 在均匀网格上计算 `d(lambda)/dt`（中心差分）。  
4. `induced_emf_from_flux_linkage` 按 `epsilon=-d(lambda)/dt` 得到感应电动势时序。  
5. `analytical_emf_for_sinusoidal_b` 计算解析参考解，用于验证数值微分精度。  
6. `simulate_rl_response` 用 Euler 离散推进 `L di/dt + R i = epsilon`，得到电流时序。  
7. `fit_first_harmonic` 分别拟合 `epsilon` 与 `i` 的一阶谐波系数，再用 2x2 线性方程得到 RL 理论稳态系数并比较误差。  
8. `main` 汇总打印指标并执行断言（EMF 误差、楞次符号、稳态幅值/系数误差、功率平衡残差），形成一次性非交互验证。  

该流程没有把第三方包当黑盒：`numpy` 只提供数组和线性代数基础，法拉第定律离散化、RL 更新、谐波比对逻辑均在源码中显式展开。
