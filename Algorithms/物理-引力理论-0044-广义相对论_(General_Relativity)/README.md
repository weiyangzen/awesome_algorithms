# 广义相对论 (General Relativity)

- UID: `PHYS-0044`
- 学科: `物理`
- 分类: `引力理论`
- 源序号: `44`
- 目标目录: `Algorithms/物理-引力理论-0044-广义相对论_(General_Relativity)`

## R01

广义相对论把引力解释为时空几何，而不是牛顿力。  
在弱场极限它回到牛顿引力，但在强引力和高速条件下会给出可观测偏差，例如光线偏折、引力红移、水星近日点进动等。

## R02

本条目选择一个最小、可运行、可核验的子问题：

- 背景时空：Schwarzschild（静态、球对称、无自旋）；
- 研究对象：无质量粒子（光）在赤道面的散射轨道；
- 任务：给定冲量参数 `b`，数值求广义相对论精确偏折角 `alpha_exact`，并与弱场近似比较。

## R03

MVP 目标不是构建完整 GR 求解框架，而是做一条透明的“方程 -> 数值 -> 验证”链路：

1. 从 Schwarzschild null geodesic 方程得到一维积分表达式；
2. 数值求每个 `b` 的最近接点与偏折角；
3. 与 `alpha_weak = 4M/b` 和二阶后牛顿展开做误差对照；
4. 输出阈值检查和 `Validation: PASS/FAIL`。

## R04

`demo.py` 使用几何单位 `G=c=1`，质量参数记为 `M`。对 `u = 1/r`，光线轨道满足：

- `F(u) = 1/b^2 - u^2 + 2 M u^3`
- 转折点 `u0` 满足 `F(u0)=0`（对应最近接距离 `r0 = 1/u0`）
- 偏折角：
  `alpha_exact = 2 * integral_0^{u0} du / sqrt(F(u)) - pi`

脚本通过变量变换 `u = u0 * (1 - t^2)` 把端点平方根奇异项规整到有限区间 `t in [0,1]` 上积分。

## R05

对照模型：

- 一阶弱场：`alpha_weak = 4M/b`
- 二阶后牛顿：`alpha_pn2 = 4M/b + (15*pi/4)*(M/b)^2`

输出表同时给出：

- `alpha_exact`、`alpha_weak`、`alpha_pn2`
- 相对误差 `|approx-exact|/exact`
- 最近接距离 `r0`

## R06

脚本运行后会输出三部分：

1. 参数摘要与冲量参数采样范围；
2. `pandas` 表格样例（前若干行）；
3. 验证指标：单调性、正值性、远场弱场误差、二阶近似改进率，以及最终 `PASS/FAIL`。

## R07

优点：

- 明确体现 GR 修正（相对牛顿/弱场近似）；
- 数学步骤可追踪，不依赖黑盒求解器直接给最终物理结论；
- 代码量小，便于教学和验证。

局限：

- 仅覆盖 Schwarzschild、静态球对称、非旋转情形；
- 未处理 Kerr 黑洞、等离子体传播、透镜成像反演等复杂问题；
- 当 `b` 过近临界捕获时，积分条件数显著恶化。

## R08

前置知识：

- 广义相对论中的测地线与 Schwarzschild 度规；
- 一维非线性方程求根（Brent）；
- 自适应数值积分与误差估计。

依赖环境：

- Python 3.10+
- `numpy`
- `scipy`
- `pandas`

## R09

记 `N` 为 `b` 采样数，`Q` 为单次 `quad` 的自适应评估成本。

- 转折点求根：`O(N)` 次标量根查找；
- 偏折角积分：`O(N*Q)`；
- 误差统计和表格构建：`O(N)`；
- 空间复杂度：`O(N)`。

## R10

数值稳定性策略：

1. 仅在 `b > b_crit = 3*sqrt(3)*M` 区间采样，保证散射根存在；
2. 转折点在固定物理区间 `(0, 1/(3M))` 内用 `brentq`；
3. 使用 `u=u0*(1-t^2)` 解除 `u=u0` 附近的平方根奇异行为；
4. 对 `t -> 0` 使用解析极限值，避免 `0/0` 数值不稳定。

## R11

默认参数（`demo.py`）：

- `mass = 1.0`
- `b_min = 6.0`
- `b_max = 40.0`
- `num_b = 28`
- `quad_epsabs = 1e-10`
- `quad_epsrel = 1e-10`

`b_crit ~= 5.196`，因此默认网格全部处于散射区。

## R12

内置正确性检查：

1. `alpha_exact(b)` 随 `b` 单调递减；
2. 全部 `alpha_exact > 0`；
3. 远场 (`b >= 25M`) 一阶弱场近似最大相对误差 `< 13%`；
4. 远场二阶 PN 近似最大相对误差 `< 1.7%`；
5. 远场二阶 PN 误差均值小于一阶弱场误差均值。

全部满足时输出 `Validation: PASS`。

## R13

保证边界：

- 这是“物理数值验证”任务，不是最优化问题，无近似比概念；
- 在固定参数和环境下，流程确定性可复现（无随机性）；
- 正确性主张限定在：Schwarzschild 光线偏折 + 指定参数区间。

## R14

常见失效模式：

1. 若 `b <= b_crit`，光线会进入捕获区，散射积分不再适用；
2. 若积分精度设得太松，近临界点误差会放大；
3. 若直接在 `u` 上硬积分而不做变换，端点奇异会降低鲁棒性；
4. 若把弱场公式用于小 `b`，会得到明显系统误差。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `critical_impact_parameter` 计算 `b_crit = 3*sqrt(3)*M`，作为散射可解性的硬边界。  
2. `radial_polynomial_u` 显式构造 `F(u)=1/b^2-u^2+2Mu^3`，这是从 Schwarzschild null geodesic 推导出的根方程。  
3. `find_turning_point_u` 在 `(0,1/(3M))` 上用 `brentq` 求 `F(u)=0`，得到最近接点 `u0`。  
4. `deflection_angle_exact_single` 用代换 `u=u0(1-t^2)` 把奇异积分转为 `t in [0,1]` 的规则积分，并用 `quad` 计算 `alpha_exact`。  
5. `weak_field_first_order` 与 `weak_field_second_order` 分别计算 `4M/b` 与 `4M/b + 15π/4*(M/b)^2` 两个解析近似。  
6. `build_result_table` 批量生成 `b, r0, alpha_exact, alpha_weak, alpha_pn2` 及对应相对误差表。  
7. `build_validation_report` 在远场子区间统计最大误差、均值误差和单调性，形成结构化检查结果。  
8. `main` 输出摘要、打印阈值检查并给出 `Validation: PASS/FAIL`，失败时以非零状态退出。  

第三方库未被当作黑盒结论器：`scipy` 只承担通用数值原语（求根/积分），GR 方程建模、变量变换、误差定义、验证标准都在源码中显式实现。
