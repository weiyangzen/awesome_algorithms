## R01

参量放大是二阶非线性介质（`chi(2)`）中的三波混频放大过程：强泵浦场把能量转移给信号与闲频。

在“泵浦不耗尽”近似下，`demo.py` 采用耦合振幅方程：
- `dA_s/dz = i * kappa * conj(A_i) * exp(i * Delta k * z)`
- `d(conj(A_i))/dz = -i * kappa * A_s * exp(-i * Delta k * z)`

其中 `kappa` 由泵浦强度决定，`Delta k` 为相位失配。核心目标是数值展示：
- 信号增益随泵浦增强而上升；
- 相位匹配（`Delta k ~= 0`）时增益最大；
- 闲频从零种子被放大生成。

## R02

参量放大（OPA/OPG）是超快激光和可调谐光源的关键机制：
- 典型应用包括飞秒放大链、红外可调谐源、量子光学相关光子对生成；
- 工程上常关注 `signal gain`、`idler conversion`、`phase-matching bandwidth`；
- 相较 Kerr 自相位调制，OPA 的能量来源是泵浦场并且对相位匹配高度敏感。

本条目做最小可运行 MVP，突出“耦合方程 -> 数值传播 -> 可审计指标”。

## R03

MVP 解决的具体问题：
给定泵浦/信号波长、晶体长度、泵浦强度、失配量与初始种子，计算并输出：
- 末端信号增益（线性与 dB）；
- 闲频转换效率；
- 沿传播方向的信号/闲频增长样本；
- 泵浦扫描与失配扫描表；
- 数值解与小信号解析增益公式的一致性误差。

## R04

脚本使用的核心关系：

1. 频率守恒（由波长反推闲频）：
`omega_p = omega_s + omega_i`。

2. 耦合系数与泵浦强度关系（简化标定）：
`kappa = kappa_ref * sqrt(I_p / I_ref)`。

3. 小信号解析增益（零闲频种子）：
- `g^2 = kappa^2 - (Delta k / 2)^2`
- 若 `g^2 > 0`：
  `G = 1 + (kappa^2 / g^2) * sinh^2(gL)`
- 若 `g^2 < 0`：
  `G = 1 + (kappa^2 / q^2) * sin^2(qL)`，`q = sqrt(-g^2)`

4. 不变量检查（泵浦不耗尽模型）：
`|A_s|^2 - |A_i|^2` 应沿 `z` 近似守恒（数值误差允许小漂移）。

## R05

`demo.py` 输出四块结果：
- `Core metrics`：`kappa`、`Delta k`、信号增益、闲频转换、守恒漂移、解析-数值误差；
- `Pump intensity scan`：泵浦强度变化对增益的影响；
- `Phase mismatch scan`：失配变化对增益的影响及与解析公式对照；
- `Propagation sample points`：若干 `z` 点上的信号/闲频功率归一化值。

末尾给出 `Validation: PASS/FAIL`。

## R06

实现策略：
- 用 `OPAConfig` 统一参数；
- 用显式 RK4（手写，不依赖黑箱求解器）推进复振幅；
- `numpy` 向量与复数计算负责主数值流程；
- `pandas` 负责把扫描结果和采样点结构化打印；
- 同时计算解析增益并对照数值结果，形成可审计闭环。

## R09

## R10

数值稳定性处理：
- 输入参数正值检查，避免非物理设置；
- 强制 `z_steps >= 200`，减少粗步长误差；
- 用 RK4（较 Euler 稳定）控制传播误差；
- 用不变量漂移量化数值可信度；
- 用解析增益对照进一步检查离散积分是否偏离。

- `I_ref = 1.0e12 W/m^2`
- `kappa_ref = 120 1/m`
- `Delta k = 0`
- `A_s(0)=1+0j`, `A_i(0)=0+0j`
- `z_steps = 4000`

## R12

内置验证规则：
1. 频率守恒闭合误差 `< 1e-15`；
2. 泵浦扫描中信号增益单调递增；
3. 失配扫描中最大增益出现在 `Delta k = 0` 采样点；
4. 不变量最大相对漂移 `< 5e-4`；
5. 基准点数值增益与解析增益相对误差 `< 2e-3`。

全部满足则 `PASS`。

## R14

常见失效模式：
1. `kappa` 设得过大或步数太低，积分误差累积导致守恒漂移；
2. 在强泵浦耗尽场景仍使用本模型，增益会被高估；
3. 将本模型直接外推到超宽带脉冲，忽略群速度失配与色散；
4. 输入波长组合不满足 `omega_p > omega_s`，导致无有效闲频。

## R18

`demo.py` 源码级算法流（8 步，非黑箱）：
1. 读取 `OPAConfig` 并检查波长、长度、强度、步数等参数有效性。
2. 由 `omega_p = omega_s + omega_i` 反推 `lambda_i`，计算频率闭合误差用于守恒检查。
3. 用 `kappa = kappa_ref * sqrt(I_p/I_ref)` 计算当前泵浦对应的耦合强度。
4. 建立状态向量 `[A_s, conj(A_i)]`，将耦合波方程写成显式复数 RHS。
5. 在 `z` 网格上执行手写 RK4，逐步推进到晶体末端，得到全程振幅轨迹。
6. 从轨迹计算 `|A_s|^2`、`|A_i|^2`、线性增益、dB 增益与不变量漂移。
7. 用解析增益公式计算理论值，并与数值增益求相对误差。
8. 执行泵浦扫描与失配扫描，做单调性/峰值/误差阈值检查并输出 `PASS/FAIL`。

第三方库分工可追踪：
- `numpy`：复数向量运算、RK4 状态更新、数组化指标计算
- `pandas`：扫描表与采样表打印
- `scipy.constants`：光速常数 `c`，用于频率-波长守恒计算
