# 镜像法 (Method of Images)

- UID: `PHYS-0154`
- 学科: `物理`
- 分类: `静电学`
- 源序号: `155`
- 目标目录: `Algorithms/物理-静电学-0155-镜像法_(Method_of_Images)`

## R01

镜像法用于把“含理想导体边界”的静电边值问题，转化为“自由空间中的等效电荷系统”。

本条目 MVP 聚焦经典场景：
- 点电荷 `q` 位于接地无限导体平面 `y=0` 上方 `(0,a)`；
- 通过在 `(0,-a)` 放置镜像电荷 `-q`，在 `y>0` 半空间重建真实解；
- 输出电势场、边界满足度、拉普拉斯残差、受力校验和感应总电荷校验。

## R02

问题定义（MVP）：
- 物理区域：`y >= 0`。
- 边界条件：导体接地，`V(x,0)=0`。
- 真实源：`q` 位于 `(0,a)`，其中 `a>0`。
- 镜像构造：在 `(0,-a)` 放置 `-q`。
- 目标：
  - 在网格上计算 `V(x,y)`；
  - 验证边界与方程性质；
  - 验证力学量与感应电荷守恒。

## R03

核心解析表达式：

- 电势（`y>=0`）：
`V(x,y)=\frac{1}{4\pi\epsilon_0}\left(\frac{q}{\sqrt{x^2+(y-a)^2}}-\frac{q}{\sqrt{x^2+(y+a)^2}}\right)`。
- 受力（真实电荷受导体作用，等效为镜像电荷作用）：
`F_y=-\frac{q^2}{16\pi\epsilon_0 a^2}`（方向指向导体）。
- 导体表面感应电荷密度（轴对称半径 `\rho`）：
`\sigma(\rho)=-\frac{q a}{2\pi(\rho^2+a^2)^{3/2}}`。

## R04

算法流程（数值层）：
1. 构建 `x` 与 `y` 均匀网格（`y` 从 0 开始）。
2. 用实电荷 + 镜像电荷公式直接计算二维截面势场 `V`。
3. 在 `y=0` 提取边界势，计算 `max|V(x,0)|`。
4. 在去除奇异点邻域后计算离散拉普拉斯 `Δ_h V`。
5. 统计原始残差 `||Δ_hV||_inf` 与归一化残差。
6. 用有限差分近似电场，估计受力并与闭式解对比。
7. 对 `\sigma(\rho)` 做数值积分估计总感应电荷，检查是否接近 `-q`。

## R05

核心数据结构：
- `numpy.ndarray`：
  - `x`, `y` 网格；
  - `potential` 势场；
  - 中间残差场与积分采样向量。
- `ImagesResult`（`dataclass`）：
  - 聚合全部诊断指标（边界误差、残差、受力误差、电荷误差）。
- `pandas.DataFrame`：
  - 终端展示多条 `y` 切线上的电势采样结果。

## R06

正确性闭环：
- 边界正确性：镜像构造应满足 `V(x,0)=0`。
- PDE 正确性：除真实电荷奇异点外，区域内应满足 `∇²V=0`。
- 力学校验：有限差分得到的 `F_y` 与解析 `-q^2/(16πϵ0a^2)` 一致。
- 电荷守恒校验：`∫σ dS` 应接近 `-q`。

这四类检查共同保证“不是只算出一个图”，而是物理关系被同时验证。

## R07

复杂度分析（网格规模 `nx × ny = N`，径向积分点数 `M`）：
- 势场计算：`O(N)`。
- 拉普拉斯残差计算：`O(N)`。
- 感应电荷积分：`O(M)`。
- 空间复杂度：`O(N + M)`。

默认参数下（约 12.9 万网格点）可在普通 CPU 上快速完成。

## R08

边界与异常处理：
- `q=0`：拒绝（退化为零场，失去演示意义）。
- `a<=0`：拒绝（几何无效）。
- 网格过小：拒绝（诊断不稳定）。
- `y_max<=a`：拒绝（无法覆盖有效上方区域）。
- 奇异点邻域：计算残差时显式屏蔽，避免把物理奇点误判为算法错误。

## R09

MVP 取舍：
- 只做“接地无限平面 + 单点电荷”，不扩展到球面/多导体复杂几何。
- 以解析镜像构造为主，不引入重型 PDE 求解框架。
- 保留必要数值诊断（残差、力、电荷）而不是只给单一公式。
- 目标是“小而诚实”：结构清晰、可运行、可验算。

## R10

`demo.py` 模块职责：
- `method_of_images_potential`：实电荷与镜像电荷叠加求势。
- `discrete_laplacian_residual`：计算屏蔽奇异点后的离散残差。
- `force_y_exact`：给出闭式受力。
- `force_y_finite_difference`：用数值微分估计受力做交叉验证。
- `induced_sigma` 与 `estimate_total_induced_charge`：感应电荷密度及总量积分。
- `solve_method_of_images_mvp`：主流程编排。
- `run_checks`：质量门槛断言。
- `build_profile_table`：输出切线采样表。

## R12

输出字段说明：
- `Boundary max |V(x,0)|`：接地边界满足度。
- `Laplace residual inf (raw)`：离散拉普拉斯残差原值。
- `Laplace residual inf (scaled)`：残差归一化后的无量纲指标。
- `Force Fy exact / finite-difference`：解析力与数值力。
- `Force relative error`：受力校验误差比例。
- `Induced total charge estimate`：积分得到的总感应电荷。
- `Induced charge relative error`：相对 `-q` 的误差比例。
- `Potential profile sample`：多条 `y` 切线上的电势采样。

## R13

建议最小测试集：
- 默认参数运行，检查全部断言通过。
- 网格加密（如 `nx=601, ny=481`），预期归一化残差下降。
- 改变 `a`（如 `0.03, 0.08`），验证受力比例 `~1/a^2`。
- 改变 `q`（如 `2e-9`），验证势场与受力缩放规律。
- 缩小积分截断半径，观察感应总电荷误差增大。

## R14

关键可调参数：
- `q`：点电荷大小。
- `a`：电荷到导体距离。
- `nx`, `ny`：势场离散分辨率。
- `x_limit`, `y_max`：观察窗口。
- `exclusion_radius`：残差计算时的奇异点屏蔽半径。
- `rho_max_factor`, `n_rho`：感应电荷积分区间与分辨率。

## R15

方法对比（本问题）：
- 镜像法：
  - 优点：解析构造直接、计算快、物理可解释性强。
  - 局限：只适用于特定可镜像几何。 
- 直接数值解 Poisson/Laplace（FDM/FEM/BEM）：
  - 优点：几何与边界更通用。
  - 代价：实现与算力成本更高。

本条目选择镜像法作为静电边值问题中的“高信噪比”入门样例。

## R16

应用场景：
- 点电荷-导体耦合的快速估算与教学演示。
- 探针-样品近场静电相互作用的初步模型。
- 电极附近电势分布的近似分析。
- 作为复杂数值解法（FEM/BEM）结果的 sanity check 基准。

## R17

可扩展方向：
- 从无限平面扩展到球导体/圆柱导体镜像问题。
- 多电荷 + 多镜像叠加及收敛性分析。
- 加入介质分层边界，转向“等效像电荷系数”模型。
- 与有限元结果做自动化对比回归测试。
- 扩展到能量、力矩、轨迹耦合仿真。

## R18

源码级算法流（对应 `demo.py`，8 步）：
1. `main` 调用 `solve_method_of_images_mvp`，设定 `q,a` 和网格参数。
2. `solve_method_of_images_mvp` 先经 `validate_inputs` 做物理与数值合法性检查。
3. `method_of_images_potential` 在每个网格点评估 `r_real` 与 `r_image`，按镜像公式得到 `V(x,y)`。
4. 从 `V` 的 `y=0` 行计算 `boundary_max_abs`，验证接地边界条件。
5. `discrete_laplacian_residual` 计算五点差分拉普拉斯，并屏蔽真实电荷邻域后统计原始/归一化残差。
6. `force_y_exact` 给出闭式力，`force_y_finite_difference` 通过中心差分计算数值力，得到相对误差。
7. `estimate_total_induced_charge` 生成 `σ(ρ)`，并用 `scipy.integrate.simpson` 积分 `σ·2πρ` 估计总感应电荷。
8. `run_checks` 统一执行阈值断言，`build_profile_table` 生成电势采样表并由 `main` 打印报告。

说明：第三方库仅承担基础数值计算（数组与积分），镜像构造、诊断指标和物理校验链路都在本源码中显式展开。
