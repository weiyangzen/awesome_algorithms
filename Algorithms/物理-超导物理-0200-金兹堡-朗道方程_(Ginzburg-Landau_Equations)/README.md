## R01

金兹堡-朗道（Ginzburg-Landau, GL）方程是描述超导序参量空间结构与磁场耦合的经典场论框架。  
完整静态 GL 理论通常写成两组耦合方程：
- 序参量方程（复数场 `psi`）；
- 超电流/矢势方程（`A` 与磁感应强度 `B`）。

本目录给出一个“最小但诚实”的数值 MVP：求解二维时间依赖 GL（TDGL）在 `A=0` 规范下的演化，观察序参量从随机初值向稳态超导纹理收敛。

## R02

MVP 目标：
1. 实现可运行的二维 TDGL 显式时间推进器；
2. 用空间非均匀 `alpha(x,y)` 模拟一个正常态缺陷区，展示序参量抑制与恢复；
3. 显式计算 GL 自由能并检查“总体下降”这一梯度流特征；
4. 输出结构化时间历史表与最终径向剖面，便于快速审阅数值行为。

## R03

本实现采用的方程（无量纲、`A=0`）：

`dpsi/dt = -(alpha*psi + beta*|psi|^2*psi - xi^2*Laplacian(psi))`

对应自由能泛函密度：

`f = alpha|psi|^2 + (beta/2)|psi|^4 + xi^2|grad psi|^2`

其中：
- `psi(x,y,t)`：复数超导序参量；
- `alpha`：局域二次项系数（`alpha<0` 倾向超导，`alpha>0` 倾向正常态）；
- `beta>0`：四次项稳定化系数；
- `xi`：相干长度尺度。

## R04

与完整 GL 方程相比，本 MVP 的简化假设：
- 固定 `A=0`，不解麦克斯韦方程，不包含磁场反馈；
- 不含噪声与随机热涨落项；
- 使用二维周期边界；
- 重点验证“序参量-自由能”演化链路，而不是实验级器件拟合。

该简化适合算法原型与教学，不等价于完整涡旋动力学或临界场计算。

## R05

离散化与求解策略：
1. 在 `ny x nx` 规则网格上存储复数 `psi`；
2. 用二阶有限差分（`np.roll`）构造周期边界拉普拉斯；
3. 用显式欧拉法推进 TDGL：`psi_{n+1} = psi_n + dt * RHS(psi_n)`；
4. 每隔 `record_every` 步记录统计量；
5. 用显式公式积分自由能密度，检查动力学是否合理。

算法核心全部写在 `demo.py`，没有调用第三方超导“黑盒求解器”。

## R06

`demo.py` 主要数据结构：
- `GLConfig`：参数容器（网格、时间步、物理系数、缺陷半径、随机种子等）；
- `alpha`：`shape=(ny,nx)` 的系数场，中心圆形区域取 `alpha_defect>0`；
- `psi`：`shape=(ny,nx)` 复数序参量场；
- `history`（`pandas.DataFrame`）：
  - `step`, `time`
  - `mean_abs_psi`
  - `mean_abs_psi_defect`
  - `mean_abs_psi_bulk`
  - `superconducting_fraction`
  - `free_energy`
- `profile`（`pandas.DataFrame`）：最终 `|psi|` 的径向平均分布。

## R07

## R10

正确性直觉：
1. TDGL 在该写法下是自由能的梯度流，系统应朝低能态演化；
2. `alpha_sc < 0` 的体相会推动 `|psi|` 长大并趋向稳定幅值；
3. 缺陷区 `alpha_defect > 0` 会抑制局部超导序参量；
4. 因而最终应观察到：体区 `|psi|` 较大、缺陷区 `|psi|` 较小，且总自由能较初态下降。

## R11

数值稳定性措施：
- 显式检查 `dt > 0`, `beta > 0`, `n_steps > 0`；
- 选取较保守时间步（默认 `dt=0.02`）避免显式扩散项爆炸；
- 使用周期边界，避免边界条件不一致导致的伪反射；
- 通过 `run_consistency_checks` 强制检查：
  - 总自由能是否下降；
  - 体区幅值是否恢复；
  - 缺陷区是否保持抑制。

## R14

常见失败模式：
1. `dt` 过大导致数值发散（`|psi|` 爆炸或 `nan`）；
2. `alpha_sc` 误设为非负导致体区不形成超导；
3. `beta<=0` 导致势能不稳定；
4. 忘记正确处理边界，导致非物理边缘效应主导结果。

本实现通过参数断言、周期拉普拉斯和结果检查来降低这些风险。

## R18

`demo.py` 源码级算法流程（8 步）：
1. `main()` 构造 `GLConfig`，调用 `run_tdgl()` 启动模拟。  
2. `build_alpha_map()` 生成空间系数场：体区 `alpha_sc<0`，中心圆形缺陷区 `alpha_defect>0`。  
3. 用固定随机种子生成复数噪声初值 `psi`，作为序参量初态。  
4. 每个时间步调用 `laplacian_periodic()` 计算 `Laplacian(psi)`，再按 TDGL 右端项组装 `rhs`。  
5. 通过显式欧拉更新 `psi <- psi + dt*rhs`，并在记录步计算 `mean_abs_psi`、体区/缺陷区均值与超导占比。  
6. `total_free_energy()` 显式积分 `free_energy_density()`，得到每个记录时刻的 GL 总自由能。  
7. 演化结束后，`radial_profile()` 对最终 `|psi|` 按半径分箱，输出径向平均剖面。  
8. `run_consistency_checks()` 检查“总能量下降 + 体区恢复 + 缺陷抑制”等关键趋势并打印结论。  

第三方库边界说明：
- `numpy` 仅用于数组运算与差分；
- `pandas` 仅用于表格化输出；
- GL 动力学离散、能量公式与检查逻辑都在源码内显式实现，不依赖领域黑盒库。
