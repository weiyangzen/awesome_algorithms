## R01

无网格方法（Meshfree Method）中的 RBF（Radial Basis Function，径向基函数）是一类用“样本点之间距离”来构造近似函数的方法。它不需要生成结构化网格或有限元网格，只需要散点坐标与对应函数值，就能进行插值或平滑拟合。

## R02

问题形式（以标量函数近似为例）：给定散点集合 `X={x_i}_{i=1}^N` 与观测值 `y_i=f(x_i)`（可含噪声），构造近似函数

`\hat f(x)=\sum_{j=1}^N w_j \phi(\|x-c_j\|)`。

其中 `c_j` 常取训练点本身，`phi` 是径向核函数（如高斯核、薄板样条核），`w_j` 为待求系数。

输入：训练点坐标、训练值、核参数（如 `epsilon`）、正则参数（如 `lambda`）。  
输出：可在任意查询点计算 `\hat f(x)` 的近似模型。

## R03

核心思想：

1. 把“局部几何关系”压缩为距离矩阵 `r_ij=||x_i-x_j||`。
2. 用径向核把距离映射成相似度 `K_ij=phi(r_ij)`。
3. 解线性系统 `(K+lambda I)w=y` 获得权重。
4. 预测时计算 `Phi(x_*,X)` 并线性组合得到 `\hat f(x_*)`。

由于只依赖距离，方法天然适合不规则采样点与无网格场景。

## R04

常用核函数示例：

- 高斯核：`phi(r)=exp(-(epsilon r)^2)`
- 多二次核：`phi(r)=sqrt(1+(epsilon r)^2)`
- 薄板样条：`phi(r)=r^2 log(r+1e-12)`

MVP 采用高斯核，并使用 Tikhonov 正则化：

`(K+lambda I)w=y`。

其中 `K` 为 `N x N` 核矩阵，`lambda>=0` 用于改善病态矩阵稳定性。

## R05

伪代码：

```text
input: X_train(Nxd), y_train(N), epsilon, lambda, X_query(Mxd)

# 训练
R <- pairwise_distance(X_train, X_train)
K <- exp(-(epsilon * R)^2)
A <- K + lambda * I
w <- solve(A, y_train)

# 预测
Rq <- pairwise_distance(X_query, X_train)
Phi <- exp(-(epsilon * Rq)^2)
y_pred <- Phi @ w

return y_pred
```

## R06

正确性直觉：

- 当 `lambda=0` 且 `K` 可逆时，训练点满足 `Kw=y`，因此在训练点上可精确插值。
- 预测值是“以训练点为中心的径向响应叠加”，每个中心对附近点贡献更大，远处贡献更小。
- 正则项 `lambda I` 相当于抑制过大权重，提升含噪数据下的泛化稳定性。

## R07

复杂度（朴素实现）：

- 构建训练核矩阵：`O(N^2 d)`
- 解线性系统：`O(N^3)`
- 单次批量预测（M 个点）：`O(MNd)` 计算距离 + `O(MN)` 线性组合
- 存储复杂度：`O(N^2)`（若显式保存训练核矩阵）

## R08

数值稳定性要点：

- `epsilon` 过大或过小都可能导致核矩阵条件数恶化。
- 含噪声数据建议使用 `lambda>0`（如 `1e-8` 到 `1e-3` 区间试验）。
- 距离平方可能出现极小负值（浮点误差），实现中可先 `max(value,0)` 再开方。

## R09

参数经验：

- `epsilon`：可按样本平均间距反比设置，或网格搜索。
- `lambda`：无噪插值可取接近 0；有噪回归建议小正数。
- 训练点数量 `N` 增加可提升表达能力，但 `O(N^3)` 求解成本会快速上升。

## R10

边界与失败模式：

- 训练点重复或过于接近，可能导致矩阵近奇异。
- 外推（查询点远离训练域）误差常明显增大。
- 高维问题会出现“距离集中”现象，需更谨慎调参或结合降维。

## R11

本目录 MVP 设计：

- 不调用黑盒 RBF 封装，直接按公式构建核矩阵并解线性方程。
- 使用 `numpy` 完成距离、线性代数和评估指标。
- 示例任务：1D 含噪函数拟合，并与线性插值基线对比。

## R13

结果解释：

- `RMSE (RBF)` 越低越好，代表整体拟合更准确。
- `MaxAbsErr (RBF)` 反映最差点误差，能揭示局部极端偏差。
- `RBF better than linear baseline` 为 `True` 表示该配置下 RBF 优于线性插值。

## R14

与常见方法对比：

- 对比多项式拟合：RBF 通常更局部、对复杂形状更灵活。
- 对比样条插值：RBF 更容易扩展到多维散点场景。
- 对比有限差分/有限元：RBF 无需构网格，前处理成本低，但大样本时线性代数成本高。

## R15

工程实现注意事项：

- 训练前建议做输入尺度归一化，避免某一维度主导距离。
- 大规模数据可采用中心点稀疏化、局部 RBF 或迭代求解器。
- 生产环境需监控外推占比与条件数，防止模型在域外失真。

## R16

可扩展方向：

- 支持多输出 `Y`（矩阵右端项）以处理向量场近似。
- 引入自适应 `epsilon` 或各向异性核（不同维度不同尺度）。
- 与 KD-Tree/近邻截断结合，形成局部稀疏 RBF，加速预测。

## R17

建议测试清单：

1. 无噪声插值：确认训练点误差接近机器精度。
2. 含噪回归：比较 `lambda=0` 与 `lambda>0` 的泛化差异。
3. 参数敏感性：扫描 `epsilon` 与 `lambda`，观察误差曲线。
4. 边界外推：在训练区间外采样，检查误差增长是否符合预期。

## R18

`demo.py` 中算法流程（源码级拆解，8 步）：

1. `main()` 生成训练散点 `x_train` 和目标值 `y_train`（含少量高斯噪声）。
2. `RBFRegressor.fit()` 把输入整理成二维数组，确保样本维度一致。
3. `fit()` 调用 `_pairwise_l2(X,X)`：通过 `||a-b||^2 = ||a||^2 + ||b||^2 - 2a·b` 向量化构造距离矩阵。
4. `fit()` 调用 `_kernel(R)` 计算高斯核矩阵 `K = exp(-(epsilon*R)^2)`。
5. `fit()` 组装 `A = K + lambda I`，用 `numpy.linalg.solve` 求解权重 `w`。
6. `predict()` 对查询点与训练中心再算一次距离矩阵 `Rq`，得到 `Phi = kernel(Rq)`。
7. `predict()` 返回 `Phi @ w`，即所有基函数响应的线性叠加值。
8. `main()` 计算 `RMSE`、`MaxAbsErr`，再与 `numpy.interp` 线性插值基线对比并打印。

这 8 步完整对应了 RBF 无网格方法从“散点 -> 核矩阵 -> 线性系统 -> 预测”的主链路。
