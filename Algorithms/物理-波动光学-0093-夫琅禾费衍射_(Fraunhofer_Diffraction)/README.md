# 夫琅禾费衍射 (Fraunhofer Diffraction)

- UID: `PHYS-0093`
- 学科: `物理`
- 分类: `波动光学`
- 源序号: `93`

## R01

夫琅禾费衍射描述的是“远场衍射”极限：
当观察屏距离孔径足够远时，屏上复振幅可看作孔径透过函数的傅里叶变换。

本条目采用最基础的一维单缝模型，展示“矩形孔径 -> `sinc^2` 强度分布”的核心规律。

## R02

本任务的最小可验证问题是：
给定单缝宽度 `a` 与波长 `lambda`，用数值 FFT 计算远场强度，并与解析公式
`I(theta) ~ sinc^2(a * sin(theta) / lambda)` 对比。

除曲线对比外，还验证第一暗纹位置 `sin(theta_1)=lambda/a` 是否被正确恢复。

## R03

MVP 建模边界：

- 仅做一维标量波模型（忽略偏振与矢量电磁效应）；
- 入射波取均匀平面波；
- 孔径是理想矩形单缝；
- 远场通过 FFT 映射到角域，不做近场 Fresnel 积分；
- 验证集中在中心角域与第一暗纹位置。

## R04

`demo.py` 使用的核心方程：

- 远场复振幅：
  `E(f) = FT{A(x)}`
- 角度映射：
  `f = sin(theta)/lambda`
- 数值强度：
  `I_fft = |E|^2`
- 单缝解析强度：
  `I_theory ~ sinc^2(a*f)`（`numpy.sinc(z)=sin(pi z)/(pi z)`）
- 第一暗纹理论位置：
  `sin(theta_1)=lambda/a`

## R05

脚本输出三部分：

- 参数回显（`lambda`, `a`, 采样点数、孔径平面窗口）；
- 两项验证指标：中心区 RMSE、第一暗纹相对误差；
- 一小段剖面表（中心和第一暗纹附近）用于人工审计。

最终输出 `Validation: PASS/FAIL`。

## R06

实现策略：

- `FraunhoferConfig` 统一管理物理和数值参数；
- `build_aperture` 构造矩形单缝离散透过函数；
- `far_field_fft` 计算数值远场并同步生成解析 `sinc^2` 参考；
- `find_peaks(-I)` 定位局部最小值来估计第一暗纹；
- `pandas` 仅负责结果整理与预览打印。

## R07

优点：

- 直接体现“孔径傅里叶变换”这一夫琅禾费衍射主机制；
- 既有解析基准，也有数值实现，便于一致性检查；
- 完全确定性，无交互输入，适合自动验证。

局限：

- 只覆盖单缝 1D 模型；
- 未包含探测器噪声、有限光谱宽度和像差；
- 未扩展到二维孔径（圆孔 Airy 图样等）。

## R09

设孔径采样点数为 `N`：

- FFT 主计算复杂度 `O(N log N)`；
- 指标计算与表格构建 `O(N)`；
- 总体时间复杂度 `O(N log N)`，空间复杂度 `O(N)`。

## R10

数值稳定性与可解释性处理：

- 用较大孔径平面窗口 `x_extent` 提升频率分辨率；
- 对强度统一做归一化，避免量纲与比例干扰；
- RMSE 仅在中心角域统计，减少边缘截断对判断的影响；
- 第一暗纹估计限定搜索窗口，避免选到高阶暗纹。

## R11

默认参数（见 `demo.py`）：

- `lambda = 532 nm`
- `a = 120 um`
- `x_extent = +/-8.0 mm`
- `N = 32768`

由此理论第一暗纹约为：
`sin(theta_1)=lambda/a ~= 4.433e-3`（约 `4.43 mrad` 小角度量级）。

## R12

内置正确性阈值：

1. 中心角域 RMSE `< 0.03`；
2. 第一暗纹相对误差 `< 5%`。

两项都通过才输出 `Validation: PASS`，否则进程非零退出。

## R13

保证说明：

- 本实现是物理一致性验证脚本，不是优化求解器；
- 给定同一参数时结果确定、可复现；
- “保证”体现在与经典解析关系的误差阈值，而非统计学习意义上的泛化保证。

## R14

常见失效模式：

1. `x_extent` 太小导致频率分辨率不足，暗纹定位误差增大；
2. `num_samples` 过低导致离散化误差明显；
3. 单位混用（nm/um/m）会造成数量级错误；
4. 暗纹搜索窗口设置不当会误识别到高阶极小值。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `FraunhoferConfig` 定义波长、缝宽、孔径平面窗口、采样数与验证阈值。  
2. `build_aperture` 在 `x` 轴上离散构造矩形单缝透过函数 `A(x)`。  
3. `far_field_fft` 对 `A(x)` 执行 `fftshift(fft(ifftshift(...)))` 得到远场复振幅，并映射到 `sin(theta)=lambda*f`。  
4. 同一函数内按 `I_fft=|E|^2` 得到数值强度，并用 `sinc^2(a*f)` 生成解析参考强度。  
5. `rmse_in_central_region` 在中心角域统计数值曲线与解析曲线的 RMSE。  
6. `estimate_first_minimum_sin_theta` 在第一暗纹附近对 `-I_fft` 调用 `find_peaks`，估计第一极小位置。  
7. `build_validation_report` 汇总 RMSE、理论第一暗纹、观测第一暗纹与相对误差。  
8. `main` 打印参数与剖面预览，执行阈值检查并输出 `Validation: PASS/FAIL`。

第三方库职责明确：

- `numpy`：离散网格、FFT、向量化运算；
- `scipy.signal.find_peaks`：极小值定位（通过对负强度找峰）；
- `pandas`：结构化展示结果。

物理建模主逻辑（孔径、频域映射、解析对照、阈值判定）全部在源码中显式实现，不是黑盒调用。
