## R01

F 检验（F-test）是一类基于 F 分布的假设检验方法，核心思想是比较两个“方差来源”的相对大小。最常见的两种形式是：

- 两样本方差齐性检验：比较 `sigma1^2` 与 `sigma2^2` 是否相等；
- 方差分析（ANOVA）中的总体均值检验：比较“组间方差”与“组内方差”。

当分子方差相对分母方差显著偏大时，F 统计量会偏离 1，从而触发拒绝原假设。

## R02

历史与定位（简要）：

- Ronald A. Fisher 在方差分析框架中系统化了 F 统计量；
- F 检验是经典频率学派推断工具，长期用于农业实验、工业质量控制和医学实验设计；
- 在现代实践里，它常作为 ANOVA 的核心检验量，也可用于线性模型整体显著性检验。

## R03

本条目 MVP 覆盖两个高频场景：

- 两独立样本方差比 F 检验（手写公式）；
- 单因素 ANOVA F 检验（手写平方和分解）。

输入：一维数值数组（两样本或多组样本）。  
输出：F 统计量、自由度、p 值、置信区间（方差比场景）、效应量与拒绝结论。

## R04

两样本方差比 F 检验定义（正态前提下）：

- 原假设：`H0: sigma1^2 = sigma2^2`
- 备择假设（双侧）：`H1: sigma1^2 != sigma2^2`
- 统计量：`F = s1^2 / s2^2`
- 自由度：`df1 = n1 - 1`, `df2 = n2 - 1`

在 `H0` 成立时，`F ~ F(df1, df2)`。

双侧 p 值可用 `2 * min(CDF(F), SF(F))` 计算（截断到 1）。

## R05

单因素 ANOVA F 检验定义：

- 原假设：`H0: mu1 = mu2 = ... = muk`
- 备择假设：至少一组均值不同
- 组间平方和：`SSB = sum_i n_i (xbar_i - xbar)^2`
- 组内平方和：`SSW = sum_i sum_j (x_ij - xbar_i)^2`
- 统计量：`F = (SSB/(k-1)) / (SSW/(N-k))`

在 `H0` 成立且各组满足方差齐性/正态误差近似时，`F ~ F(k-1, N-k)`。

## R06

复杂度：

- 方差比 F 检验：`O(n1+n2)` 时间，`O(1)` 额外空间；
- 单因素 ANOVA：`O(N)` 时间，`O(1)` 到 `O(k)` 额外空间（取决于中间统计量存储）。

MVP 核心是均值/方差/平方和聚合与分布函数查询，不涉及迭代优化。

## R07

简化手算示例（方差比检验）：

设样本方差 `s1^2=16`、`s2^2=9`，样本量 `n1=11`、`n2=13`。

1. `F = 16/9 = 1.7778`。  
2. 自由度 `df1=10`, `df2=12`。  
3. 在双侧检验中，先查 `F(10,12)` 的左右尾概率。  
4. 若双侧 p 值小于 `alpha=0.05`，则拒绝“方差相等”的原假设。  
5. 同时报告 `sigma1^2/sigma2^2` 的区间估计，避免只报 p 值。

## R08

适用前提：

- 方差比 F 检验通常要求两组样本独立且近似正态；
- ANOVA 通常要求各组独立、组内近似正态、方差齐性；
- 对严重偏态、厚尾和强离群点，F 检验会较敏感，应先做诊断。

## R09

标准检验流程：

1. 明确问题与 `H0/H1`（双侧还是单侧）；
2. 计算 F 统计量及自由度；
3. 由 F 分布计算 p 值；
4. 比较 `p` 与 `alpha` 得出拒绝/不拒绝结论；
5. 配套输出区间估计与效应量（如 `eta^2`），提升可解释性。

## R10

本 MVP 的正确性验证：

- 方差比 F 检验由手写公式直接计算统计量与 p 值；
- ANOVA 由手写平方和分解实现，并与 `scipy.stats.f_oneway` 对照；
- 运行时自动断言手写 ANOVA 与 SciPy 在数值误差范围内一致。

## R11

数值稳定性要点：

- 明确检查样本量 `n>=2`，避免自由度非正；
- 样本方差或组内均方接近 0 时主动报错，防止除零；
- 统一使用 `float64`，降低累计误差；
- 双侧 p 值用 `2*min(left,right)` 并截断到 `[0,1]`。

## R12

解释输出时的常见误区：

- “不拒绝 H0”不等于“证明 H0 为真”；
- p 值不是“原假设为真的概率”；
- ANOVA 显著仅表示“至少一组均值不同”，并不告诉哪一组不同；
- 方差齐性不成立时，经典 ANOVA 的结论可能偏乐观。

## R13

与其他方法关系：

- 方差齐性替代：Levene 检验、Brown-Forsythe（对非正态更稳健）；
- 多组均值替代：Welch ANOVA（放宽齐方差假设）；
- 非参数替代：Kruskal-Wallis（比较分布位置）；
- 线性模型中，回归整体显著性检验也可写成 F 统计量。

## R14

常见失败模式与防护：

- 失败：有极端异常值仍直接做 F 检验。  
  防护：先做箱线图/稳健统计审计，必要时改用稳健方法。
- 失败：ANOVA 显著后直接下结论“所有组都不同”。  
  防护：补做事后比较（如 Tukey HSD）并控制多重比较。
- 失败：忽略独立性（重复测量当独立样本）。  
  防护：改用重复测量 ANOVA 或混合模型。
- 失败：只汇报 p 值。  
  防护：统一报告 `F + df + p + 区间/效应量`。

## R15

本实现（`demo.py`）包含：

- `variance_f_test_manual`：两样本方差比 F 检验手写实现；
- `one_way_anova_f_test_manual`：单因素 ANOVA 手写实现；
- `_p_value_from_f`：按双侧/单侧规则计算 p 值；
- `compare_anova_with_scipy`：与 `scipy.stats.f_oneway` 对照并断言；
- `main`：生成可复现样本，执行两类检验并打印结果表。

## R16

工程实践建议：

- 先做数据质量检查（缺失、重复、离群点）；
- 固定随机种子，保留实验可复现性；

## R18

`demo.py` 源码级算法流程（8 步，非黑盒）：

1. `main` 固定随机种子，构造两组方差不同样本（用于方差比检验）和三组均值不同样本（用于 ANOVA）。  
2. `variance_f_test_manual` 计算两组样本方差 `s1^2,s2^2` 与统计量 `F=s1^2/s2^2`，并得到自由度 `df1,df2`。  
3. `_p_value_from_f` 按备择方向把 F 统计量映射到 p 值；双侧情形使用 `2*min(CDF,SF)`。  
4. `variance_f_test_manual` 再基于 F 分位数给出方差比 `sigma1^2/sigma2^2` 的置信区间，并形成拒绝决策。  
5. `one_way_anova_f_test_manual` 先聚合各组均值与总体均值，计算 `SSB`（组间）和 `SSW`（组内）。  
6. 同函数内继续计算 `MSB, MSW`，得到 `F = MSB/MSW` 与 `p = SF(F)`，同时输出效应量 `eta^2 = SSB/(SSB+SSW)`。  
7. `compare_anova_with_scipy` 调用 `scipy.stats.f_oneway`，将手写 ANOVA 的 `F` 与 `p` 逐项断言校验。  
8. `main` 将两类检验结果汇总为 `pandas.DataFrame` 打印，最后给出“手写实现与 SciPy 校验一致”的完成信号。

说明：SciPy 在本 MVP 中仅用于 F 分布函数和 ANOVA 对照校验；核心统计量与检验流程已在源码中手写展开。
