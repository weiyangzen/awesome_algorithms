# 磁场强度H (Magnetic Field Intensity H)

- UID: `PHYS-0160`
- 学科: `物理`
- 分类: `静磁学`
- 源序号: `161`
- 目标目录: `Algorithms/物理-静磁学-0161-磁场强度H_(Magnetic_Field_Intensity_H)`

## R01

磁场强度 `H` 的计算型定义可写成“由电流源生成、可按矢量叠加的场”：

- 输入:
  - 电流源集合（本 MVP 用无限长直导线，参数为 `(x0, y0, I)`）
  - 查询点集合 `P={(x_i,y_i)}`
  - 介质参数（若需要 `B`，再给 `mu`，真空取 `mu0`）
- 输出:
  - 每个查询点的 `H=(Hx, Hy)`（单位 A/m）
  - 可选 `B=mu*H`（单位 T）
- 核心机制:
  - 单根 `z` 向无限长直导线在平面点的解析式  
    `Hx = - I/(2*pi*rho^2) * (y-y0)`  
    `Hy =   I/(2*pi*rho^2) * (x-x0)`  
    其中 `rho^2=(x-x0)^2+(y-y0)^2`
  - 多电流源满足线性叠加: `H_total = sum(H_k)`
  - 通过 `oint H·dl = I_enclosed` 可做数值自检（安培环路定律）

## R02

关键时间线（静磁学语境）：

- `1820`：Oersted 发现电流与磁效应关联。
- `1820s`：Ampere 建立电流-磁场关系并形成环路定律框架。
- `1861-1865`：Maxwell 在场论体系中区分并系统化 `H` 与 `B`。
- `20` 世纪 SI 制建立后：`H` 单位统一为 `A/m`，工程建模中固定使用。

## R03

提出背景是“把电流源与介质响应分离建模”：

- 早期仅用磁感应强度 `B` 描述时，材料磁化效应与自由电流贡献混在一起，不利于工程求解与参数标定。
- 引入 `H` 后可把问题拆成：
  - 自由电流决定 `curl H = J_free`
  - 材料关系通过本构式连接到 `B`（线性各向同性时 `B=mu*H`）
- 该分离使磁路设计、线圈计算、屏蔽结构分析和多介质问题更可计算。

## R04

若有 `M` 根导线、`N` 个查询点：

- 直接叠加计算 `H` 的时间复杂度是 `O(MN)`。
- 若附带 `K` 个环路采样点评估 `oint H·dl`，则额外 `O(MK)`。
- 本条目 MVP 中 `M` 很小（示例为 2），但复杂度主项仍按 `MN` 计。

## R05

空间复杂度（额外工作内存）：

- 输出 `H` 需要 `O(N)` 存储。
- 每次处理一根导线时的临时向量 `dx,dy,rho2,coef` 也是 `O(N)`。
- 总体额外空间 `O(N)`；不随导线数 `M` 线性增长（导线逐根累加即可）。

## R06

可复现实例（与 `demo.py` 一致）：

- 两根无限长导线：
  - `wire1=(-0.03, 0), I1=+5 A`
  - `wire2=(+0.03, 0), I2=-3 A`
- 查询点：`y=0` 上 13 个点，`x in [-0.12, 0.12]`
- 以原点 `(0,0)` 为例：
  - 来自 `wire1` 的 `Hy ≈ +26.526 A/m`
  - 来自 `wire2` 的 `Hy ≈ +15.915 A/m`
  - 总和 `Hy ≈ +42.441 A/m`，方向沿 `+y`
- 安培环路验证：
  - 半径 `0.02 m` 圆环不包围导线，理论 `I_enclosed=0`
  - 半径 `0.06 m` 圆环包围两根导线，理论 `I_enclosed=2 A`
  - 数值积分 `oint H·dl` 与理论值应接近

## R07

算法意义：

- 理论上：`H` 是麦克斯韦方程组静磁极限中的核心场量，连接自由电流与旋度约束。
- 工程上：是线圈、母排、磁屏蔽与电机磁路初算的基础模块。
- 计算上：为更高阶数值方法（FEM/BEM/FMM）提供可验证的基准解析核。

## R08

算法路径直接依赖：

- 线性叠加原理（场的可加性）
- 右手定则（方向判定）
- 安培环路定律 `oint H·dl = I_enclosed`
- 无限长直导线解析核（可由 Biot-Savart/安培定律导出）
- 线性本构关系 `B=mu*H`（当需要从 `H` 转到 `B`）

## R09

适用前提与边界：

- 静磁或准静态近似（忽略位移电流耦合项）
- 导线模型为无限长直导体（本 MVP 的解析核）
- 介质按线性各向同性处理（若计算 `B`）
- 查询点不应落在理想导线轴线上（`rho=0` 奇异）
- 对有限长度导线、复杂铁磁非线性材料，本模型不直接适用

## R10

正确性依据：

- 解析核满足对单导线的已知闭式解。
- 多导线结果由麦克斯韦方程线性结构给出叠加正确性。
- 数值验证层面，采用独立条件 `oint H·dl` 对 `I_enclosed` 做守恒校验；若积分与包围电流一致，则实现与方向约定基本正确。

## R11

数值稳定性与误差传播：

- 主要敏感区在 `rho -> 0`：`|H| ~ 1/rho`，会放大坐标误差和舍入误差。
- 反向电流相消区域可能出现“差分消去”，相对误差上升。
- 环路积分离散误差来自角度采样数 `K`，`K` 太小会导致积分偏差。
- MVP 采用 `core_radius` 下限裁剪避免除零；这提升鲁棒性但引入近场偏差。

## R12

真实计算成本（非渐近）：

- 主耗时是向量化数组算子：乘加、除法、`numpy` 广播；常数项低，CPU 缓存友好。
- 对小规模 `M`（本例 `M=2`），`O(MN)` 的常数主导比复杂度更关键。
- 安培积分检查需要额外 `K` 点场评估；`K=4096` 时可在很短时间内完成。
- 无 I/O 瓶颈，无 GPU 必要性；典型场景 CPU 即可满足。

## R13

误差界与保证（本条目为确定性解析计算，不是随机算法）：

- 在模型假设成立且查询点远离奇异轴线时，单点 `H` 计算无统计误差，只有浮点舍入误差。
- 加入 `core_radius=a` 后，近场会产生模型化偏差；对 `rho >> a` 区域该偏差快速减弱。
- 环路积分数值误差可近似写成  
  `|E| <= C1/K^p + C2*eps_machine`，常见情形 `p` 在 2 左右或更高（光滑周期积分时收敛更快）。

## R14

鲁棒性与失效模式：

- 失效 1：查询点贴近导线轴线，结果极大且对噪声敏感。
- 失效 2：把有限导线误当无限导线，远场/端部误差显著。
- 失效 3：单位混用（mm 当 m）导致场强放大 `10^3` 量级。
- 失效 4：铁磁饱和、各向异性、磁滞等非线性介质下，`B=mu*H` 线性关系失真。
- 缓解：设置最小半径、进行单位检查、对复杂几何切换到 FEM/BEM。

## R15

工程实现要点：

- 数据结构：电流源用结构体（`x,y,current`），查询点用 `N x 2` 数组。
- 向量化：按导线循环、对点批量计算，避免 Python 逐点循环。
- 奇异处理：使用 `core_radius` 防止 `rho^2=0`。
- 自检：固定做一次安培环路校验，验证方向、符号和单位。
- 常见陷阱：右手定则符号写反、把 `B` 与 `H` 单位混淆、在导线位置取样。

前驱/后继与典型应用：

  - Oersted/Ampere 的电流致磁实验与定律
  - Biot-Savart 与安培环路分析
- 后继扩展：

## R18

源码追踪与 3-10 步拆解（本条目不使用第三方黑箱求解器，直接追踪 `demo.py`）：

1. 在 `InfiniteWire` 中定义电流源参数 `(x, y, current)`，统一输入语义与单位。
2. 准备查询点数组 `points_xy`，并初始化累计场 `h=zeros(N,2)`。
3. 对每根导线计算相对位移 `dx, dy` 与 `rho2=dx^2+dy^2`，再用 `core_radius` 做下限裁剪。
4. 用解析系数 `coef = I/(2*pi*rho2)` 计算该导线贡献：
   - `Hx += -coef*dy`
   - `Hy +=  coef*dx`
5. 对所有导线完成叠加，得到总场 `H_total`。
6. 按需转换 `B = mu0 * H_total` 并组织成表格输出。
7. 构造圆形安培环路采样点，计算每点切向分量 `H·t_hat`，离散求和得到 `oint H·dl`。
8. 计算同一环路的 `I_enclosed`，对比积分值与理论值，输出相对误差作为实现正确性证据。
