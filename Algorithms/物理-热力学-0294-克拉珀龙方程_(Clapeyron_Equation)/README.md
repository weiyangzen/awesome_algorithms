# 克拉珀龙方程 (Clapeyron Equation)

- UID: `PHYS-0291`

- 分类: `热力学`

## R01

克拉珀龙方程描述两相共存线（如液-汽平衡）上压强随温度的变化率：

`dP/dT = L / (T * Δv)`

其中：

- `L`：单位质量潜热（J/kg）；
- `Δv = v_g - v_l`：气相与液相比容差（m^3/kg）；
- `T`：绝对温度（K）。

该方程把“相变热力学量”（`L, Δv`）直接映射到“相图斜率”（`dP/dT`）。

## R02

本条目 MVP 目标是实现一个可运行、可解释、非黑盒的饱和压曲线构造器：

1. 以一个参考饱和点 `(T_ref, P_ref)` 为锚点；
2. 根据克拉珀龙方程数值积分得到 `P(T)`；
3. 同时给出克劳修斯-克拉珀龙近似曲线作对照；
4. 用反算 `L = T*Δv*(dP/dT)` 验证方程闭环一致性。

## R03

`demo.py` 采用的最小物理模型：

- 液相比容常数：`v_l = const`（弱可压缩近似）；
- 气相比容理想气体近似：`v_g = R_v*T/P`；
- 潜热线性模型：`L(T) = L_ref + a*(T - T_ref)`，并设置正值下限。

因此微分方程写成：

`dP/dT = L(T) / [T * (R_v*T/P - v_l)]`

数值方法使用手写四阶 Runge-Kutta（RK4）逐温度步推进。

## R04

与克劳修斯-克拉珀龙近似关系：

当 `v_l << v_g` 且 `L≈常数` 时，可化简为：

`d(ln P)/dT = L/(R_v*T^2)`

积分得到：

`ln(P/P_ref) = -L/R_v * (1/T - 1/T_ref)`

`demo.py` 同时输出：

- 完整克拉珀龙积分曲线 `P_clapeyron(T)`；
- 上述简化近似 `P_clausius(T)`；
- 二者相对误差，观察近似有效区间。

## R05

## R10

正确性直觉：

1. 共存线上 `P(T)` 必须满足克拉珀龙微分关系；
2. 给定锚点 `(T_ref,P_ref)` 后，沿温度轴积分可唯一生成局部曲线；
3. 若用积分结果再反算 `L = T*Δv*dP/dT`，应回到原模型 `L(T)`；
4. 若在近参考温区克劳修斯近似成立，则两条曲线偏差应较小。

`demo.py` 的断言正是对以上 2-4 点做自动校验。

## R11

数值稳定性策略：

- 强制 `T>0, P>0`，避免非物理输入；
- 对 `Δv = v_g-v_l` 设置下界检查，防止分母奇异；
- 采用 RK4 而非显式欧拉，降低步长敏感性；
- 温度网格设置为奇数点，保证精确包含参考锚点；
- 反算潜热误差使用“中位数+去边界点”评估，减弱数值微分边界噪声影响。

## R14

常见失败模式：

1. 误用单位（例如把 `kPa` 当 `Pa`）；
2. 温度用摄氏度直接代入方程；
3. 在超高压区仍套用理想气体 `v_g=RT/P`；
4. 温度步长过大导致积分误差放大；
5. 忽略 `Δv` 接近零时的数值病态。

本 MVP 用输入检查、RK4 与断言机制做基本防护。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `ClapeyronConfig` 定义参考点、温度网格、`R_v`、`v_l` 与潜热模型参数。  
2. `latent_heat_j_per_kg` 给出 `L(T)`，并用下界防止非物理负潜热。  
3. `clapeyron_rhs_pa_per_k` 计算 `dP/dT = L/[T*(v_g-v_l)]`，其中 `v_g=R_v*T/P`。  
4. `rk4_step_pressure` 用四次斜率组合完成单步 RK4，推进压强。  
5. `integrate_clapeyron_curve` 以 `(T_ref,P_ref)` 为锚点，分别向高温和低温方向积分，再拼接完整 `P(T)`。  
6. `clausius_clapeyron_pressure_pa` 在同温度网格上计算常潜热简化近似，用于对照。  
7. `run_mvp` 计算 `dP/dT`、反算潜热 `L_back = T*Δv*dP/dT` 与相对误差，汇总成 `DataFrame`。  
8. `main` 打印结果并执行断言：单调性、锚点一致性、近参考温区近似误差、潜热闭环误差。  

第三方库边界：`numpy/pandas` 只用于数组、梯度和表格；核心克拉珀龙积分流程由源码中的 `rhs + RK4 + 双向积分` 显式实现，不依赖黑盒 ODE 求解器。
