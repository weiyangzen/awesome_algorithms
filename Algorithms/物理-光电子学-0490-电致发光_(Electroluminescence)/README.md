# 电致发光 (Electroluminescence)

- UID: `PHYS-0467`
- 学科: `物理`
- 分类: `光电子学`

## R01

电致发光（EL）是指在电注入条件下，电子与空穴复合并发射光子的过程。对 LED 的最小建模可以写成“电学注入 -> 载流子复合 -> 光输出”三段：

- 电学：`I(V)` 由二极管方程决定；
- 复合：`A*n + B*n^2 + C*n^3 = J/(q*d)`；
- 光学：`P_opt = I * EQE * E_ph`。

这里 `A/B/C` 分别对应 SRH、辐射、Auger 复合通道。

## R02

该算法是光电子器件建模中的高性价比基线：

- 可解释 LED 的效率滚降（efficiency droop）；
- 可解释随偏置升高的热红移（峰值波长变长）；
- 适合做参数敏感性分析（`Rs`、`A/B/C`、提取效率、热阻）。

相比全 TCAD，成本更低、可读性更高，适合教学和早期方案筛选。

## R03

本 MVP 目标：

1. 用含串/并联电阻的 Shockley 方程计算 `I-V`；
2. 用 Newton 法求解 ABC 方程得到载流子浓度 `n`；
3. 计算 `IQE/EQE`、光功率 `P_opt`；
4. 结合 Varshni 带隙模型与高斯线型，生成可积分守恒的 EL 光谱；
5. 用断言自动检验“单调性、droop、红移、谱积分守恒”。

## R04

核心公式如下：

1. 二极管注入（隐式）：
   `I = I0 * (exp((V - I*Rs)/(n*Vt)) - 1) + (V - I*Rs)/Rsh`。
2. 电流密度：`J = I / area`。
3. ABC 复合平衡：
   `A*n + B*n^2 + C*n^3 = J/(q*d)`。
4. 内量子效率：
   `IQE = (B*n^2) / (A*n + B*n^2 + C*n^3)`。
5. 外量子效率：`EQE = eta_inj * eta_ext * IQE`。
6. Varshni 带隙：
   `Eg(T) = Eg0 - alpha*T^2/(T+beta)`。
7. 峰值波长近似：`lambda_peak[nm] = 1239.841984 / Eg[eV]`。
8. 光功率：`P_opt = I * EQE * Eg`（单位 W）。

## R05

设电压扫点数为 `N`，Newton 迭代上限为 `K`（常数级，几十步）：

- 二极管电流求解：`O(N*K)`；
- ABC 载流子求解：`O(N*K)`；
- 后处理（IQE/EQE/光谱摘要）：`O(N)`。

总体时间复杂度 `O(N*K)`，空间复杂度 `O(N)`。

## R06

`demo.py` 提供两类输出：

## R10

正确性直觉：

1. 偏压升高应使电流上升（至少不下降）；
2. `IQE` 在低注入受 `A` 项限制，中注入受益于 `B`，高注入受 `C` 项导致 droop；
3. 偏置升高会增加功耗与结温，`Eg` 下降导致波长红移；
4. 光谱是对 `P_opt` 的分布，积分应回到 `P_opt`。

## R11

数值稳定策略：

- 二极管方程用 Newton 迭代，并对指数输入做裁剪；
- 电压扫描采用“前一点电流”作为下一点初值，加速并稳定收敛；
- ABC 方程导数恒正（`A,B,C>0`）时根唯一，Newton 收敛性好；
- 对 `EQE` 做 `[0,1]` 限幅避免非物理值。

## R12

关键参数与影响方向：

- `series_resistance_ohm`：越大则高偏置电流受限越明显；
- `a_srh`：越大低注入损失越大；
- `b_rad`：越大辐射复合越强，峰值 IQE 提升；
- `c_auger`：越大高注入 droop 越明显；
- `thermal_resistance_k_per_w`：越大红移越明显；
- `extraction_efficiency`：线性影响 EQE 与 `P_opt`。

## R13

- 近似比保证：N/A（非组合优化问题）。
- 随机成功率保证：N/A（算法确定性）。

可验证保证（通过断言）：

- 电流随偏置非下降；
- `IQE` 峰值在内部偏置区间且存在明显 droop；
- 峰值波长出现正向红移；
- 光谱积分相对误差在阈值内。

## R14

常见失效模式：

1. 参数量纲不一致（尤其 `B/C` 系数单位）；
2. 过小 `Rsh` 导致漏电主导，掩盖正常注入行为；
3. `A/B/C` 组合不合理导致 IQE 全程单调而看不到 droop；
4. 光谱网格范围太窄导致积分无法守恒。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `solve_diode_current` 对每个偏置点求解隐式 Shockley 方程，得到 `I(V)`。  
2. 用 `J=I/area` 转成电流密度，并在 `solve_carrier_density_from_abc` 中对 `A*n+B*n^2+C*n^3=J/(q*d)` 做 Newton 迭代求根。  
3. `iqe_from_carrier_density` 把复合速率分解为 `A/B/C` 三通道并计算 `IQE`。  
4. `external_qe` 乘注入效率与提取效率得到 `EQE`。  
5. `varshni_bandgap_ev` 用结温修正带隙，`peak_wavelength_nm_from_bandgap` 映射到峰值波长。  
6. `optical_power_w` 由 `I、EQE、Eg` 计算总光功率。  
7. `gaussian_spectrum` 把总光功率分配到波长网格，并用数值积分归一化保证功率守恒。  
8. `run_validation` 执行物理一致性断言（单调、droop、红移、谱积分误差），`main` 打印报告并完成一次非交互可复现实验。  

第三方库仅用于数组与基础数值计算，核心电致发光流程（方程建立、迭代求解、效率计算、光谱归一化、校验）均在源码中显式实现。
