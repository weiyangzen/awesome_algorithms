## R01

贝肯斯坦-霍金熵是黑洞热力学的核心关系：黑洞熵与事件视界面积成正比，而不是与体积成正比。对 Schwarzschild 黑洞可写为：

`S_BH = k_B * c^3 * A / (4 * G * hbar)`

这条关系把几何量（面积）和统计物理量（熵）直接连接起来，是全息思想和量子引力研究的重要起点。

## R02

该条目在算法层面关注一个最小可运行链路：

1. 从质量 `M` 计算史瓦西半径 `r_s`；
2. 由 `r_s` 得到视界面积 `A`；
3. 由 `A` 计算 `S_BH`；
4. 用霍金温度 `T_H` 验证热力学一阶关系 `dS/dM = c^2/T_H`。

目标是让“物理公式 -> 数值验证 -> 可复现脚本”闭环可执行。

## R03

`demo.py` 的 MVP 解决三件事：

- 计算 `M -> (r_s, A, T_H, S_BH)` 的完整表格；
- 验证面积表达式与闭式质量平方表达式一致：`S ~ M^2`；
- 用有限差分验证 `dS/dM = c^2 / T_H`（黑洞第一定律在 Schwarzschild 情形下的质量形式）。

脚本无交互输入，直接运行即可输出指标并自动断言。

## R04

本实现使用的关键公式：

1. 史瓦西半径：`r_s = 2GM/c^2`
2. 视界面积：`A = 4pi r_s^2`
3. 贝肯斯坦-霍金熵：`S = k_B c^3 A / (4G hbar)`
4. 等价闭式：`S = 4pi k_B G M^2 / (hbar c)`
5. 霍金温度：`T_H = hbar c^3 / (8pi G M k_B)`
6. 信息比特数：`N_bits = S/(k_B ln2)`

## R05

设质量网格点数为 `n`：

- 基础物理量计算（`r_s, A, S, T`）：`O(n)`；
- 有限差分 `dS/dM`：`O(n)`；
- 对数线性拟合 `log(S)=a+p log(M)`：`O(n)`（特征维度常数）；

总体时间复杂度 `O(n)`，空间复杂度 `O(n)`。

## R06

`demo.py` 输出三类结果：

- **Summary Table**：给出 `M, r_s, A, T_H, S_BH, S/k_B, S_bits`；
- **公式一致性指标**：面积形式与闭式 `M^2` 形式的相对差；
- **热力学一致性指标**：`dS/dM` 数值导数与 `c^2/T_H` 的相对误差（均值和最大值）。

并执行断言，确保脚本可作为自动化校验的最小基线。

## R10

正确性直觉：

1. 质量越大，视界面积越大，熵应单调增大；
2. `S(M)` 为二次函数，log-log 斜率应接近 2；
3. 同一物理量的两种等价公式（面积式与闭式）应数值一致；
4. 若热力学关系成立，`dS/dM` 与 `c^2/T_H` 应匹配。

## R11

数值稳定与精度策略：

- 质量设下界 `min_mass_kg` 避免除零；
- 导数采用中心差分（端点单边差分）；
- 第一定律误差统计只取内部点，减弱边界离散误差干扰；
- 使用对数均匀质量网格提升跨数量级分辨率。

## R12

关键参数与作用：

- `min_mass_kg`：质量下界，防止奇异计算；
- `n_summary_points`：摘要表展示点数；

调参建议：

## R18

`demo.py` 源码级算法流程（8 步）：

1. `schwarzschild_radius_m` 把输入质量 `M` 映射到 `r_s=2GM/c^2`。  
2. `horizon_area_m2` 用 `A=4pi r_s^2` 计算视界面积。  
3. `bekenstein_hawking_entropy_j_per_k` 按 `S = k_B c^3 A / (4G hbar)` 计算熵。  
4. `entropy_closed_form_j_per_k` 独立实现等价闭式 `S ~ M^2`，与第 3 步做逐点对比，验证公式一致性。  
5. `hawking_temperature_kelvin` 计算 `T_H(M)`，为热力学一致性检查提供右端项。  
6. `finite_difference_derivative` 在非均匀质量网格上计算 `dS/dM`（中心差分 + 端点单边差分）。  
7. `validate_first_law` 对比 `dS/dM` 与 `c^2/T_H`，输出逐点相对误差并统计均值/最大值。  
8. `fit_loglog_power_law` 对 `S(M)` 做 `log-log` 线性拟合得到指数 `p`，`main` 汇总表格、误差指标并执行断言。  

第三方库没有被当作黑盒：`numpy/pandas/scipy.constants` 仅提供数组运算、表格展示和物理常数，核心物理关系、导数计算、一致性验证流程均在源码中显式实现。
