## R01

安培环路定律（静磁学形式）给出磁场环量与包围电流的关系：

`∮_C B · dl = μ0 * I_enclosed`

其中：

- `C` 是闭合积分回路（按右手定则定向）；
- `B` 是磁感应强度；
- `I_enclosed` 是被回路围住的代数电流（正负由电流方向与回路法向共同决定）；
- `μ0` 是真空磁导率。

在微分形式下可写为 `∇×B = μ0 J`（静磁场，忽略位移电流）。

## R02

安培定律是静磁学中“电流源 -> 磁场环量”的核心桥梁。它在以下场景中最常用：

- 长直导线磁场估算；
- 同轴结构、母线、环形器件中的快速量级分析；
- 作为麦克斯韦方程组在低频/稳恒电流条件下的工程近似。

它的价值在于：当问题具有对称性时，可绕开三维积分，直接通过闭合路径得到结果。

## R03

本条目 MVP 目标是把安培定律做成可运行、可验证的最小闭环：

- 以“无限长直导线”作为可解析基元，显式计算平面磁场；
- 对圆形/方形回路做数值线积分，验证 `∮B·dl = μ0 I_enclosed`；
- 通过多导线叠加与不同回路形状，验证“只看包围电流，不看具体路径细节”的性质。

## R04

实现采用的核心方程如下：

1. 单根无限长直导线（沿 `+z`）在平面点 `(x,y)` 的磁场：  
   `B = μ0 I /(2π r^2) * (-dy, dx)`，`dx = x-x0, dy = y-y0, r^2 = dx^2 + dy^2`。
2. 多导线叠加：  
   `B_total = Σ B_i`。
3. 连续环量离散化（中点求积）：  
   `∮ B·dl ≈ Σ_k B(p_k)·Δl_k`。
4. 圆形回路包围电流判定：  
   若导线到圆心距离 `< radius`，其电流计入 `I_enclosed`。

`demo.py` 完全显式实现上述步骤，不调用黑盒电磁求解器。

## R05

设：

- `K` 为导线数；
- `M` 为回路离散段数（圆环 `n_segments`，方环约 `4*n_per_side`）。

则：

- 单次环量计算时间复杂度：`O(K*M)`；
- 空间复杂度：`O(1)`（除常量大小临时向量外不保留大规模历史数据）。

对 MVP 规模（`K <= 4`, `M ~ 10^3`）而言，运行非常轻量。

## R06

`demo.py` 包含三个验证演示：

- **Demo A（单导线 + 多半径圆环）**：同一包围电流下，不同半径应得到同一环量 `μ0 I`。
- **Demo B（多导线 + 圆环）**：半径变化导致包围电流变化，环量应同步跟随 `μ0 ΣI_enclosed`。
- **Demo C（路径无关性）**：圆环与方环包围同一电流集合时，环量应一致（在离散误差内）。

每个演示均内置 `assert` 作为自动验收门槛。

## R10

正确性直觉：

1. 单导线解析场满足 `|B| ∝ 1/r`，圆环上 `B` 与切向 `dl` 同向；
2. 因而圆环积分中 `B*2πr` 的 `r` 抵消，得到常数 `μ0 I`；
3. 多导线时磁场可线性叠加，环量自然变成 `μ0` 乘代数电流和；
4. 对不穿过奇点（导线）的不同路径，环量只由拓扑包围关系决定。

## R11

数值稳定与精度策略：

- 线积分采用中点法，减少一阶截断误差；
- 对圆环/方环使用足够分段（千级）控制离散误差；
- 显式检查“积分路径不可穿导线奇点”；
- 误差判据用相对误差阈值，允许有限离散误差。

## R12

关键参数：

- `CurrentWire(x, y, current)`：导线几何与方向；
- `radius` / `half_side`：回路尺度；
- `n_segments` / `n_per_side`：积分离散密度；
- `center_xy`：回路中心（决定包围关系）。

调参建议：

- 误差偏大时先增大 `n_segments` 或 `n_per_side`；
- 回路离导线太近会放大局部梯度，需要更细分段；
- 避免导线落在回路边界附近（包围判定不稳定）。

## R14

常见失效模式：

1. 路径经过导线位置，导致奇点发散；
2. 分段太粗，路径积分误差偏大；
3. 电流方向符号约定混乱，造成 `I_enclosed` 正负错误；
4. 导线恰好压在边界附近，包围判定受浮点误差影响；
5. 忽略模型边界（把无限导线公式硬套到短导线场景）。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `CurrentWire` 定义导线位置与电流方向，作为全部物理输入。
2. `magnetic_field_from_wires` 对每根导线应用解析式 `μ0 I /(2πr^2)*(-dy,dx)` 并叠加得到 `B_total`。
3. `enclosed_current_circle` 用几何距离判定圆环是否包围某导线，求得代数和 `I_enclosed`。
4. `ampere_circulation_circle` 以中点法离散圆环，累加 `Σ B(p_k)·Δl_k` 近似 `∮B·dl`。
5. `ampere_circulation_square` 对四条边逐段积分，得到方形路径的环量。
6. `run_single_wire_radius_invariance_demo` 固定单导线电流，改变圆环半径，验证环量应保持 `μ0 I`。
7. `run_multiwire_enclosure_demo` 在多导线配置下改变半径，验证环量随 `μ0 ΣI_enclosed` 变化，并检查“零包围电流 -> 近零环量”。
8. `run_path_independence_demo` 比较“同包围电流”的圆环与方环结果，最后在 `main` 汇总打印并执行所有断言。

第三方库仅用于数组运算；安培定律离散步骤、包围判定和误差校验均由源码显式实现，没有把核心物理逻辑交给黑盒函数。
