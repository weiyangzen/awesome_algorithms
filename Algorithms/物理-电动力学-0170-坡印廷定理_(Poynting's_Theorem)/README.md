# 坡印廷定理 (Poynting's Theorem)

- UID: `PHYS-0169`
- 学科: `物理`
- 分类: `电动力学`

## R01

坡印廷定理给出电磁场的**局域能量守恒关系**。其微分形式为：

`∂u/∂t + ∇·S + J·E = 0`

其中：

- `u = (1/2)(epsilon0|E|^2 + |B|^2/mu0)` 是电磁能量密度；
- `S = (1/mu0)(E x B)` 是坡印廷矢量（能流密度）；
- `J·E` 是单位体积中电磁场对物质做功的功率密度。

## R02

该定理是麦克斯韦方程组的能量版本：

- `∇·S` 描述能量通过边界流出/流入；
- `∂u/∂t` 描述场能在局域的存储变化；
- `J·E` 描述场能转化为介质中热/机械等非场能。

因此它直接连接了“场量”与“可测功率”。

## R03

本条目 MVP 目标：

- 在真空一维平面波上离散计算 `u` 与 `S`；
- 数值验证局部守恒式 `∂u/∂t + ∂Sz/∂z ≈ 0`（`J=0`）；
- 数值验证积分守恒式 `d/dt∫u dz + [Sz(z2)-Sz(z1)] ≈ 0`；
- 检查平面波关系 `Sz = c u`。

这样能形成“物理定律 -> 离散算法 -> 自动断言”的最小闭环。

## R04

MVP 使用的场模型与公式：

1. 一维真空平面波（沿 `+z` 传播）
   `Ey(z,t) = E0 cos(kz - omega t + phi0)`
2. 磁场取
   `Bx(z,t) = -Ey(z,t)/c`
   （保证 `E x B` 指向 `+z`）
3. 能量密度
   `u = (1/2)(epsilon0 Ey^2 + Bx^2/mu0)`
4. 坡印廷分量
   `Sz = -(Ey*Bx)/mu0`
5. 真空理论关系
   `Sz = c u`
6. 局部守恒（`J=0`）
   `∂u/∂t + ∂Sz/∂z = 0`

## R05

设网格为 `nt x nz`：

- 场生成：`O(nt*nz)`；
- `u, Sz` 计算：`O(nt*nz)`；
- 两个方向有限差分：`O(nt*nz)`；
- 积分形式检查：`O(nt*nz)`。

总体时间复杂度 `O(nt*nz)`，空间复杂度 `O(nt*nz)`。

## R06

`demo.py` 输出三类核心指标：

- `plane_wave_identity_rel_err`：`Sz` 与 `c*u` 的相对误差；
- `local_rel_residual`：局部守恒残差的相对量级；
- `integral_rel_residual`：积分守恒残差的相对量级。

脚本使用 `assert` 自动判定阈值是否通过，便于回归验证。

## R10

正确性直觉：

1. 若 `u` 局部减少，那么该处应有净能流流出（`∇·S > 0`）；
2. 真空平面波中 `E` 与 `B` 同相，能流总向传播方向；
3. `Sz = c*u` 反映“能量以波速运输”；
4. 局部形式和积分形式都成立，说明离散实现没有仅在某一表达式上“凑巧对齐”。

## R11

数值策略与稳定性处理：

- 时间和空间均用均匀网格；
- `np.gradient(..., edge_order=2)` 提供二阶边界处理；
- 局部残差评价去掉边界点（`interior`）减少边界离散误差影响；
- 用相对残差而非绝对残差，降低量纲和尺度依赖。

## R12

关键参数：

- `e0`：电场峰值（V/m）；
- `wavelength`：波长（m）；
- `omega = c*k`：色散关系（真空）；
- `nt, nz`：网格密度，直接影响离散误差；
- `domain_z, domain_t`：空间/时间计算域范围。

调参建议：

- 若残差偏大，优先增大 `nt` 和 `nz`；
- 高频（小波长）下需更细空间网格；
- 保证域长度至少覆盖多个波长以避免偶然对消。

## R14

常见失效模式：

1. `Bx` 符号写反导致能流方向错误；
2. 把 `B^2/mu0` 写成 `mu0*B^2` 造成量纲错误；
3. 网格过粗导致差分误差掩盖守恒关系；
4. 边界点直接纳入统计，误把边界离散误差当成物理失配。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `build_plane_wave_fields` 在 `(t, z)` 网格上构造 `Ey(z,t)` 和 `Bx(z,t)`，并由 `omega=c*k` 固定色散关系。  
2. `compute_energy_density_and_flux` 根据定义式计算 `u` 与 `Sz`，不调用任何黑盒电磁库。  
3. `check_plane_wave_identity` 计算 `Sz` 与 `c*u` 的相对误差，先做代数一致性预检查。  
4. `compute_local_residual` 用二阶差分近似 `∂u/∂t` 和 `∂Sz/∂z`，构造局部残差场。  
5. `compute_local_residual` 同时返回 RMS/最大值与相对残差，用于量化局部守恒质量。  
6. `compute_integral_residual` 在选定空间子区间上离散积分 `U(t)=∫u dz`，再与边界通量差组合成积分残差。  
7. `run_demo` 汇总三类报告（代数关系、局部守恒、积分守恒），形成统一数值证据链。  
8. `main` 打印关键指标并执行阈值断言，保证 `uv run python demo.py` 非交互可自动验证。

说明：`numpy` 仅提供数组、梯度和积分等基础算子；坡印廷定理的离散化、残差定义、判据逻辑均在源码中显式展开。
