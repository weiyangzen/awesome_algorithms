# 傅里叶光学 (Fourier Optics)

## R01

傅里叶光学研究“光场在空间与频域之间如何映射”。

在标量近似和傍轴条件下，透镜可以把入射复振幅 `U(x)` 变换到频域 `U_f(f_x)`，因此许多成像、衍射与滤波问题都可转化为“傅里叶变换 + 频域操作 + 逆变换”的流程。

## R02

本条目最小目标：

1. 用离散 FFT 数值模拟单缝孔径的夫琅禾费衍射；
2. 将数值结果与解析 `sinc^2` 结果对比，给出误差；
3. 演示一个简化 4f 频域低通滤波过程，验证高频被压制、低频被保留。

## R03

MVP 的建模边界：

- 一维标量光场（`x` 方向），不做偏振矢量场建模；
- 夫琅禾费区（透镜傅里叶面）采用 FFT 近似积分；
- 孔径选取单缝矩形函数，便于与解析式核对；
- 4f 系统用“频域乘掩膜”表示空间滤波。

## R04

`demo.py` 使用的核心方程：

- 傅里叶面复振幅（连续形式）  
  `U_f(f_x) = ∫ U_0(x) * exp(-i 2π f_x x) dx`
- 离散近似  
  `U_f[k] ≈ FFT(U_0) * dx`
- 强度  
  `I(f_x) = |U_f(f_x)|^2`
- 单缝解析强度（宽度 `a`）  
  `I_ana(f_x) ∝ sinc^2(a f_x)`
- 透镜傅里叶面坐标映射  
  `x_sensor = λ f f_x`
- 4f 低通  
  `U_out = F^{-1}{F{U_in} * H(f_x)}`

## R05

脚本输出三类报告：

- `Fraunhofer validation`：`sinc^2` 误差和第一零点位置误差；
- `4f low-pass report`：低频幅值保持率和高频衰减比例；
- `Sample Fourier-plane profile`：频率/传感面坐标/数值强度/解析强度的样例表。

最后给出阈值检查和 `Validation: PASS/FAIL`。

## R06

## R09

设离散采样点为 `N`：

- 单次 FFT/IFTT 为 `O(N log N)`；
- 强度、解析对比、掩膜乘法均为 `O(N)`；
- 总体时间复杂度 `O(N log N)`，空间复杂度 `O(N)`。

## R10

数值稳定性处理：

- 使用 `fftshift/ifftshift` 保证零频在中心，避免坐标错位；
- 对强度做归一化，避免绝对量纲放大数值误差；
- 零点误差在预期频带窗口内搜索，降低离群点影响；
- 频率选为离散基频整数倍，减小谱泄漏对 4f 评估的污染。

## R11

默认参数（见 `demo.py`）：

- 波长 `λ = 532 nm`
- 透镜焦距 `f = 0.20 m`
- 单缝宽度 `a = 120 um`
- 采样窗口 `x ∈ [-5 mm, 5 mm)`，`N = 4096`
- 4f 测试频率：低频 `600 cycles/m`、高频 `4200 cycles/m`
- 低通截止 `1500 cycles/m`

## R12

内置正确性阈值：

1. 单缝数值-解析曲线中心带 RMSE `< 0.03`；
2. 第一零点相对误差 `< 8%`；
3. 高频分量保留率 `< 0.2`（即显著衰减）；
4. 低频分量保留率在 `[0.85, 1.15]`。

全部满足才输出 `Validation: PASS`。

## R13

保证类型说明：

- 这是物理一致性验证脚本，不是统计学习器；
- 在固定参数和同一运行环境下结果确定；
- “保证”指满足预设误差阈值，而非理论最优界。

## R14

常见失效模式：

1. 采样点过少导致零点定位跳变；
2. 忘记 `dx` 缩放导致频谱幅值量纲错误；
3. `fftshift` 顺序错误导致频率轴与谱错配；
4. 4f 截止频率设置不当，无法体现低/高频分离。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `make_grid` 在孔径面生成均匀采样网格，并计算 `dx`。  
2. `build_single_slit_aperture` 生成矩形单缝复振幅 `U0(x)`。  
3. `fourier_plane_field` 使用 `ifftshift -> FFT -> fftshift` 计算傅里叶面复场，并乘 `dx` 做离散积分尺度修正。  
4. `normalized_intensity` 将 `|U_f|^2` 归一化，得到数值衍射强度。  
5. `analytic_single_slit_intensity` 用 `sinc^2(a f_x)` 生成解析参考；`compute_first_zero_error` 在预期频带内估计第一零点偏差。  
6. `build_object_signal` 构造“低频 + 高频”一维对象，模拟待成像信号。  
7. `apply_4f_lowpass` 在频域乘低通掩膜 `H(f_x)`，再 `IFFT` 重建，`sinusoid_amplitude` 评估低/高频幅值变化。  
8. `main` 汇总 RMSE、零点误差、频率分量保持率并执行阈值判定，输出 `PASS/FAIL`。

第三方库职责拆解：

- `numpy`：光场数组、强度与频率坐标计算；
- `scipy.fft`：FFT/IFT 与频率轴生成；
- `pandas`：输出样例表格。

算法主体（孔径建模、误差定义、阈值判定、4f 掩膜逻辑）均在源码中显式实现，不是黑盒函数一键替代。
