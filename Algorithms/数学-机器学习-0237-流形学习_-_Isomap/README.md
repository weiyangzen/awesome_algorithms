## R01

Isomap（Isometric Mapping）是经典流形学习算法，目标是在低维空间中保留高维数据沿流形的“测地距离”结构，而不是仅保留欧氏直线距离。

设高维样本为 `X = {x_i}_{i=1}^n, x_i \in R^D`，Isomap 输出低维表示 `Y = {y_i}_{i=1}^n, y_i \in R^d (d << D)`，使得：
- 当两点在流形上“绕路”接近时，低维中也应接近；
- 当两点在流形上“绕路”很远时，低维中也应拉开。

这使它特别适合“卷曲但内在维度低”的数据，如 Swiss Roll。

## R02

输入与输出：
- 输入：
  - 样本矩阵 `X`，形状 `(n_samples, n_features)`；
  - 邻居数 `k`（或半径参数，本文 MVP 采用 kNN）；
  - 目标维度 `d`（通常 2 或 3）。
- 输出：
  - 低维嵌入矩阵 `Y`，形状 `(n_samples, d)`；
  - 过程产物（常用于诊断）：kNN 图、全对测地距离矩阵、MDS 特征值谱。

本目录 `demo.py` 会额外输出 `trustworthiness` 与 `residual_variance` 两个质量指标。

## R03

核心思想可分三段：
1. 局部建图：用欧氏距离构造邻接图，只保留局部邻边（保留流形局部几何）。
2. 全局展开：在邻接图上计算任意两点最短路径距离，近似流形测地距离。
3. 低维重建：把测地距离矩阵送入 Classical MDS，通过双中心化与特征分解得到低维坐标。

对比 PCA：
- PCA 假设全局线性子空间；
- Isomap 允许非线性流形，通过“图最短路”把曲面结构显式纳入。

## R04

关键数学形式：

1. kNN 图 `G=(V,E)`：
- 顶点 `V` 对应样本；
- 若 `x_j` 是 `x_i` 的 k 近邻之一，则连边 `(i,j)`，边权 `w_ij = ||x_i - x_j||_2`。

2. 测地距离近似：
- `D_G(i,j) = shortest_path_G(i,j)`，即图上的加权最短路长度。

3. Classical MDS：
- 构造平方距离矩阵 `D^2`；
- 双中心化 `B = -1/2 * H D^2 H`，`H = I - (1/n)11^T`；
- 对 `B` 做特征分解，取前 `d` 个正特征值 `lambda_1...lambda_d` 与对应特征向量 `v_1...v_d`；
- 嵌入 `Y = [sqrt(lambda_1)v_1, ..., sqrt(lambda_d)v_d]`。

## R05

`demo.py` 的算法流程：
1. 生成 Swiss Roll 三维样本（可控噪声，固定随机种子）。
2. 构建对称 kNN 加权图（先有向近邻，再对称化）。
3. 检查图连通性，不连通则尝试更大 `k`。
4. 对每个源点运行 Dijkstra，得到全对测地距离矩阵。
5. 对测地距离执行 Classical MDS，得到 2D 嵌入。
6. 计算 `trustworthiness` 与 `residual_variance`。
7. 打印嵌入预览表并执行断言，形成最小验证闭环。

## R06

复杂度（`n` 为样本数，`k` 为邻居数）：
- 建 kNN（本 MVP 用全对距离 + 排序）：`O(n^2 + n^2 log n)`，主导项可视作 `O(n^2 log n)`。
- 构图边数 `m ≈ O(nk)`。
- 全对最短路（重复 `n` 次 Dijkstra）：
  - 单次 `O(m log n)`，
  - 总计 `O(n m log n) ≈ O(n^2 k log n)`。
- MDS 特征分解（`n x n`）：`O(n^3)`，是大样本时主要瓶颈。

空间复杂度：
- 测地距离矩阵 `O(n^2)`；
- MDS 的 Gram 矩阵 `O(n^2)`。

## R07

优点：
- 能处理显著非线性流形（如卷曲面）。
- 局部邻接 + 全局最短路兼顾局部几何与全局拓扑。
- 输出可解释，MDS 特征值可辅助判断嵌入维度。

局限：
- 对 `k` 敏感：过小图不连通，过大又趋向线性方法。
- 对噪声与异常点较敏感（会污染邻接图和最短路）。
- `O(n^2)` 内存与 `O(n^3)` 分解限制了大规模场景。

## R09

适用场景：
- 低维流形嵌在高维空间的数据可视化；
- 需要保留“沿流形路径”关系，而非直线距离关系；
- 中小规模样本（几百到几千）分析原型。

不适用或需谨慎：
- 超大样本（内存/时间压力显著）；
- 数据拓扑复杂且噪声大，kNN 图不稳定；
- 流形假设不成立的数据（如高度离散、混合机制数据）。

## R10

正确性直觉（实现层面）：
1. kNN 图仅保留局部边，近似流形局部切空间邻接。
2. 图最短路在局部边拼接下逼近流形测地距离。
3. MDS 在“距离一致性”意义上恢复低维点坐标。
4. 若图断开，某些点对测地距离为 `inf`，算法前提被破坏，必须显式报错或调大 `k`。

因此本实现先做连通性检测，再进入最短路与 MDS。

## R11

数值稳定与鲁棒性要点：
- MDS 的 Gram 矩阵做 `(B + B^T)/2` 对称化，降低浮点不对称影响。
- 特征值只接受正值（阈值 `1e-12`），避免对负特征值开方。
- Dijkstra 距离初始化为 `inf`，并使用“过期堆元素跳过”写法。
- 若候选 `k` 下图不连通，自动尝试更大 `k`，避免静默输出坏结果。

## R12

关键超参数与经验：
- `k`（邻居数）：最关键。
  - 太小：图易断开；
  - 太大：跨流形“短路”风险增加，几何被拉直。
- `d`（嵌入维度）：
  - 可视化常用 2；
  - 可通过特征值衰减观察有效维度。
- 样本量 `n`：
  - 增大可改善流形采样密度；
  - 但会快速推高 `O(n^2)`/`O(n^3)` 成本。

本 MVP 用候选 `k = (8, 10, 12, 15, 20)` 自动选择首个连通图。

## R14

常见失效模式与防护：
- 失效：图不连通，测地距离含 `inf`。
  - 防护：连通性检查 + 增大 `k`。
- 失效：异常点形成“桥”，扭曲最短路。
  - 防护：预处理去噪/截断离群点。
- 失效：`k` 过大造成“捷径”边，破坏流形结构。
  - 防护：扫描多个 `k`，结合信任度指标选型。
- 失效：MDS 出现有效正特征值不足。
  - 防护：降低嵌入维度或检查数据/图构造是否合理。

## R15

工程实践建议：
- 先在已知流形（Swiss Roll/S-curve）上做 sanity check，再上真实数据。
- 固定随机种子，便于对比 `k` 或预处理策略。
- 记录 `k`、连通性、`trustworthiness`、`residual_variance` 四类关键信号。
- 对大样本优先考虑 Landmark-Isomap 或近似近邻，以降低资源占用。

## R16

相关方法与演化：
- LLE：强调局部线性重建权重保持。
- Laplacian Eigenmaps：基于图拉普拉斯谱嵌入。
- t-SNE / UMAP：更强可视化能力，但目标函数与几何解释不同。
- Landmark-Isomap / C-Isomap：用于扩展规模或处理复杂拓扑。

在“保持全局测地结构”这一目标下，Isomap 依旧是教学与基线中的代表算法。

## R18

源码级流程拆解（对应 `demo.py`，8 步）：
1. `main` 调用 `make_swiss_roll_numpy` 生成三维非线性流形样本 `X`，固定随机种子保证复现。
2. `fit_isomap` 依次尝试候选 `k`，每轮在 `build_knn_graph` 中先计算全对欧氏距离，再按行排序取前 `k` 邻边并做无向对称化。
3. `is_connected` 通过 DFS 检查图是否连通；若不连通直接抛 `ValueError`，并由 `fit_isomap` 继续尝试更大的 `k`。
4. 图连通后，`all_pairs_geodesic_distances` 对每个源点调用 `dijkstra_single_source`，累计得到 `n x n` 测地距离矩阵。
5. `classical_mds` 对测地距离做平方和双中心化，构造 Gram 矩阵 `B = -1/2 H D^2 H`，再进行对称特征分解。
6. `classical_mds` 取前 `d` 个正特征值和特征向量，按 `Y = V_d * sqrt(Lambda_d)` 生成低维嵌入。
7. `residual_variance` 与 `trustworthiness` 分别评估“全局距离保真”与“局部邻域保真”。
8. `main` 打印指标与嵌入预览表，并用阈值断言做最小正确性守护，最终给出可自动验证的结束信号。
