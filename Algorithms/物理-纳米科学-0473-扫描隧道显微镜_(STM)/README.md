# 扫描隧道显微镜 (STM)

## R01

扫描隧道显微镜（STM）利用量子隧穿效应在导电针尖与样品表面之间形成电流，从而在原子尺度成像与谱学测量。常见近似公式为：

- `I(x,y,z,V) ≈ C * exp(-2κz) * ∫ rho_s(x,y,E) dE`

其中 `z` 是针尖-样品距离，`κ` 与势垒高度相关，`rho_s` 是样品局域态密度（LDOS）。STM 的核心算法任务是在噪声和漂移下，从 `I-z` 或 `dI/dV` 数据中恢复表面结构与电子结构信息。

## R02

STM 由 Binnig 与 Rohrer 在 1980 年代初发展，推动了原子尺度表征革命。算法实践从早期“仅成像”逐步扩展到：

- 恒流模式拓扑重建；
- 恒高模式 `dI/dV` 谱图与态密度映射；
- 缺陷识别、晶格常数估计与电子态反演。

因此 STM 不只是显微成像设备，也是“物理模型 + 数值计算 + 机器学习”融合的典型场景。

## R03

本任务聚焦一个最小可运行问题：

1. 构造可解释的合成 LDOS 场 `rho(x,y,E)`；
2. 通过隧穿模型生成恒流拓扑图和多偏压 `dI/dV` 图；
3. 从拓扑图估计晶格常数，从 `I-z` 曲线估计 `κ`；
4. 用多偏压谱特征回归局域 LDOS，展示传统回归与小型神经网络对比。

这覆盖了 STM 算法链路中的“前向模拟 + 逆向估计 + 数据驱动补充”。

## R04

`demo.py` 使用的数学模型与近似：

- 空间模型：规则原子点阵（高斯峰叠加）+ 空位扰动 + 杂质态；
- 能量模型：`rho(x,y,E) = rho_spatial(x,y)*f(E) + defect(x,y)*g(E)`；
- 恒流拓扑：由 `I_set = C*exp(-2κz)*S(x,y,V)` 求解 `z(x,y)`，其中 `S = ∫ rho dE`；
- 恒高谱图：`dI/dV(x,y,V) ∝ exp(-2κz0)*rho(x,y,E=eV)`；
- `κ` 估计：对 `ln I` 与 `z` 做线性拟合，斜率约为 `-2κ`。

这是 Tersoff-Hamann 风格的简化实现，足以演示核心计算流程。

## R05

设网格为 `N x N`，能量采样点数 `M`，偏压个数 `B`，MLP 训练轮数 `E`：

- LDOS 立方构造：`O(N^2 * M)`；
- 偏压积分与拓扑求解：`O(N^2 * M)`；
- 峰检测与聚类（DBSCAN）：约 `O(P log P)` 到 `O(P^2)`（`P` 为候选峰数）；
- `I-z` 线性拟合：`O(T)`（`T` 为曲线采样点）；
- MLP 训练：`O(E * N^2 * B * H)`（`H` 是隐藏层规模）。

在本 MVP 默认参数下，CPU 可快速运行。

## R06

一个直观数值例子（与脚本模型一致）：

- `κ = 10.8 nm^-1`，`z = 0.55 nm`；
- 假设局部积分态密度 `S = 0.08`，比例常数 `C = 1e-6`；
- 则 `I ≈ C * exp(-2κz) * S`。

由于 `exp(-2κz)` 对 `z` 极敏感，针尖高度仅变化 `0.01 nm` 就会显著改变电流，这也是 STM 具备原子级纵向分辨率的原因。

## R07

优势：

- 对导电表面可实现原子级空间分辨；
- 通过 `dI/dV` 可近似读取局域电子态信息；
- 模型具备较强物理可解释性（`κ`、`rho`、`z`）。

局限：

- 强依赖导电性与洁净表面条件；
- 热漂移、振动、针尖态变化会造成系统误差；
- 简化模型（如本 MVP）无法覆盖多体效应、复杂针尖轨道贡献。

## R09

适用场景：

- STM 教学中的算法链路演示；
- 新成员入门“从物理模型到可运行代码”的最小范例；
- 缺陷/晶格参数估计方法的快速原型验证。

不适用或需谨慎：

- 真实实验数据的定量反演（需更严格仪器校准）；
- 强关联系统或超导隧穿谱的精确建模；
- 针尖态未知且显著变化的场景。

## R10

正确性检查要点：

1. `z(x,y)` 与积分态密度 `S(x,y)` 应呈单调对数关系；
2. `ln I-z` 拟合得到的 `κ` 应为正且接近设定值；
3. 拓扑图原子峰检测后应给出合理晶格尺度（接近 `a`）；
4. `dI/dV` 在不同偏压下统计量应有差异，反映能量依赖；
5. MLP 与 Ridge 误差应为有限值且量级合理。

## R11

数值稳定性策略：

- 对 `rho` 与电流下界做 `clip`，防止 `log(0)`；
- 在拓扑求解时对 `C*S` 设最小值，避免负无穷；
- 峰检测先平滑再阈值，降低噪声伪峰；
- `MLP` 训练前进行 `StandardScaler` 标准化，避免梯度尺度失衡。

## R12

调参建议（本实现对应）：

- `atom_sigma_nm` 决定原子峰锐度，过大将导致峰重叠；
- `quantile_threshold` 控制峰筛选严格度，过低会误检；
- DBSCAN 的 `eps` 决定近邻峰是否合并；
- MLP 轮数与学习率影响收敛，建议先固定随机种子比较。

## R13

理论性质与说明：

- `I-z` 对数线性关系来自指数势垒近似，在线性区间内可稳健估计 `κ`；
- `dI/dV ∝ rho(E)` 在低温、小调制、针尖态近似平坦时更成立；
- 本任务以确定性模拟为主，不涉及近似比或概率保证类算法结论；
- MLP 属非凸优化，仅保证经验收敛，不保证全局最优。

## R14

常见失效模式与防护：

- 失效：热漂移导致晶格常数估计偏大/偏小。  
  防护：加入漂移校正或参考标样。
- 失效：针尖双峰结构造成“鬼影原子”。  
  防护：通过多次扫描与针尖状态诊断剔除异常。
- 失效：仅用单偏压解释复杂能带。  
  防护：联合多偏压 `dI/dV` 与能量维分析。
- 失效：把简化前向模型直接用于真实数据定量。  
  防护：引入更完整传输模型和实验标定。

## R15

工程落地建议：

- 先在合成数据上验证“参数可恢复性”（如 `κ`、`a`）；
- 将前向模拟参数（噪声、偏压、网格）配置化，便于回归测试；
- 输出结构化指标表（pandas DataFrame）供自动验证脚本读取；
- 在进入真实数据前，建立最小质量门槛：峰检测稳定、`κ` 估计误差受控。

## R16

相关方法对比：

- 传统 STM 图像处理：滤波 + FFT + 峰检测，解释性强、实现快；
- 物理反演方法：从传输方程出发，精度高但实现复杂；
- 机器学习方法：在复杂噪声下可提升鲁棒性，但依赖数据分布；
- 本 MVP 选择“物理模拟 + 经典估计 + 小型神经网络”三段式，兼顾可解释性与扩展性。

## R17

`demo.py` 提供的可运行内容：

- 生成原子点阵 + 缺陷的三维 LDOS 合成数据；
- 计算恒流拓扑图、恒高多偏压 `dI/dV` 图；
- 使用峰检测+DBSCAN 估计晶格常数；

## R18

`demo.py` 的源码级算法流程（9 步）：

1. `make_grid` 构造二维扫描网格 `x,y` 与坐标张量 `xx,yy`。  
2. `make_surface_ldos_spatial` 叠加规则点阵高斯峰，并注入一个空位与一个杂质，得到空间强度底图。  
3. `build_ldos_cube` 将空间底图与能量趋势函数组合，外加局域共振项，得到 `rho(x,y,E)`。  
4. `integrate_to_bias` 对每个像素沿能量轴积分 `rho`，形成偏压相关积分态密度 `S(x,y,V)`。  
5. `simulate_topography` 用 `I_set = C*exp(-2κz)*S` 反解恒流模式高度图 `z(x,y)` 并叠加实验噪声。  
6. `simulate_didv_maps` 在固定针尖高度下取 `rho(E=eV)` 生成多偏压 `dI/dV` 图。  
7. `detect_atoms_and_lattice` 对高度图做高斯平滑、局域峰提取和 DBSCAN 合并，再用最近邻距离中位数估计晶格常数。  
8. `estimate_kappa_from_iz` 生成局域 `I-z` 曲线，对 `ln I` 与 `z` 进行 Ridge 线性拟合，按斜率计算 `κ`。  
9. `train_torch_ldos_regressor` 把多偏压谱作为特征，分别用 Ridge 与 PyTorch MLP 回归目标 LDOS，并在 `main` 中用 pandas 汇总输出指标。  

该流程没有把 STM 问题交给黑盒库“一键完成”，而是把物理建模、数值积分、参数估计、聚类和学习器训练分解为可追踪的源码步骤。
