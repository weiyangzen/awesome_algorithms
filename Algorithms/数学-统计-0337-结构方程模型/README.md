## R01

结构方程模型（Structural Equation Modeling, SEM）是把“潜变量测量模型”与“变量间路径关系模型”统一起来的统计框架。

它同时回答两类问题：

- 测量层：不可直接观测的潜变量（如能力、满意度）如何由多个观测指标反映；
- 结构层：这些潜变量之间如何相互影响（直接效应、间接效应）。

本目录的 MVP 使用线性高斯 SEM，并且显式写出协方差映射与目标函数，不依赖黑盒 SEM 库。

## R02

简化发展脉络：

- 路径分析（path analysis）先处理观测变量之间的线性因果路径；
- 因子分析（factor analysis）处理“潜变量 <- 指标”的测量关系；
- SEM 将两者统一，使“测量误差 + 结构关系”可在同一模型中估计。

工程上，SEM 常见于社会科学、心理测量、教育评估、市场研究与行为科学。

## R03

SEM 主要解决三类痛点：

- 直接把有噪声的观测变量当作真实构念，会导致回归系数偏差；
- 只做因子分析看不到潜变量之间的作用路径；
- 只做路径分析忽略测量误差，会夸大解释力度。

SEM 通过联合建模把“测量误差控制”和“路径估计”同时完成。

## R04

本实现采用两潜变量线性高斯 SEM：

1. 结构方程：
- `eta = beta * xi + zeta`
- `xi ~ N(0, var_xi)`，`zeta ~ N(0, var_zeta)`，且独立

2. 测量方程：
- `x1,x2,x3` 测量 `xi`（首个载荷固定为 1 做尺度识别）
- `y1,y2,y3` 测量 `eta`（首个载荷固定为 1）

3. 协方差形式：
- `Sigma(theta) = Lambda * Phi * Lambda^T + Theta`
- `Lambda` 为 6x2 载荷矩阵，`Phi` 为潜变量协方差，`Theta` 为对角测量误差方差

4. 估计目标（高斯协方差拟合）：
- `F_ML = log|Sigma| + tr(S Sigma^-1) - log|S| - p`
- 其中 `S` 是样本协方差，`p` 为观测变量维度。

## R05

设样本数 `n`，观测维度 `p`，迭代次数 `T`。

- 每次目标函数评估主要成本在 `p x p` 线性代数运算，约 `O(p^3)`；
- 总体优化成本约 `O(T * p^3)`；
- 在本 MVP 中 `p=6`，所以瓶颈主要在优化迭代次数，而非单次矩阵运算。

空间复杂度主要为协方差矩阵与中间线性代数缓存，约 `O(p^2)`。

## R06

本目录示例参数（`demo.py`）：

- 潜变量结构系数：`beta=0.72`
- 潜变量方差：`var_xi=1.15`, `var_zeta=0.55`
- 载荷：
  - `xi` 侧：`[1.0, 0.90, 0.75]`
  - `eta` 侧：`[1.0, 0.85, 0.95]`
- 测量误差方差（6 个指标）为中等噪声水平
- 样本量：`n=1800`

脚本先生成该真值数据，再拟合并输出“真值 vs 估计值”的参数恢复表。

## R07

优点：

- 同时处理潜变量测量与结构路径，解释力强；
- 可显式分离“结构效应”与“测量误差”；
- 在协方差框架下可直接做模型拟合优度诊断。

局限：

- 线性高斯假设可能过强；
- 识别条件敏感（固定载荷/方差策略必须明确）；
- 目标函数可能有局部最优，对初始化和样本规模有依赖。

## R08

建议前置知识：

- 多元正态与协方差矩阵；
- 因子分析、路径分析基础；
- 极大似然估计与数值优化；
- 矩阵求逆、行列式、迹运算。

运行环境：

- Python 3.9+
- `numpy`, `pandas`（若安装 `scipy` 会优先使用其 `L-BFGS-B` 优化器）

## R09

适用场景：

- 需要建模潜在构念（心理特质、满意度、风险偏好）；
- 既关心测量有效性，也关心构念间路径影响；
- 可以接受线性近似和较大样本统计推断。

不适用场景：

- 超小样本且参数很多（估计不稳定）；
- 强非线性、强离散、厚尾异常主导的数据；
- 仅需简单预测而不需要可解释路径/测量结构。

## R10

本 MVP 的正确性检查包括：

1. 拟合后 `Sigma_hat` 的正定性检查（`slogdet` 符号应为正）；
2. 目标值 `F_ML` 有限且可计算；
3. 结构系数 `beta_hat` 与真值接近（脚本中设容差断言）；
4. 样本协方差与模型协方差残差 RMS 应较小（断言 `rms_cov_resid < 0.20`）。

这些检查保证示例不是“能跑但无效”的伪实现。

## R11

数值稳定策略：

- 对方差类参数使用 `softplus` 变换，保证估计过程中始终为正；
- 目标函数中遇到非正定或线性求解失败时返回大惩罚值；
- 使用 `slogdet` 计算 `log|Sigma|`，比直接 `det` 更稳定；
- 数据在样本协方差前先中心化，避免均值偏移影响。

## R12

调参与建模建议：

- 样本量 `n`：参数数目固定时，`n` 越大恢复越稳；
- 初始化：复杂模型建议多起点优化，降低局部最优风险；
- 识别约束：常见是固定每个潜变量首个载荷为 1，或固定潜变量方差为 1；
- 若拟合差，可先检查测量模型，再逐步放开结构路径。

## R13

理论性质（对应本实现）：

- 在模型正确设定且样本足够大时，ML 估计具有一致性和渐近正态性；
- 本问题是连续参数优化问题，不存在“近似比”这类离散优化指标；
- 由于可能非凸，数值解通常是局部最优，需要通过多初始化增强鲁棒性。

## R14

常见失效模式与对策：

- 失效 1：模型不可识别（自由参数过多或约束不足）。
  对策：固定尺度、减少自由参数、检查 `df = p(p+1)/2 - t` 是否为正。

- 失效 2：Heywood 问题（误差方差逼近 0 或异常）。
  对策：加入合理约束、检查指标质量、提升样本量。

- 失效 3：局部最优或收敛慢。
  对策：多初值、分阶段拟合、调整优化器容差和迭代上限。

- 失效 4：模型设定错误导致拟合偏差。
  对策：检查残差结构、修改载荷或路径设定、做模型比较。

## R15

工程落地建议：

- 先做最小可识别模型，再逐步加路径与约束；
- 数据预处理要关注缺失、异常值和量纲问题；
- 将参数估计与拟合诊断（如 RMSEA、残差）一起输出；
- 对关键路径系数做敏感性分析，避免过度解释单次估计结果。

## R16

与相关方法关系：

- 因子分析：只关心测量层，不建模潜变量之间路径；
- 路径分析：只关心结构层，通常直接使用观测变量；
- 回归/PLS：偏预测导向，潜变量测量误差建模不如 SEM 完整。

一句话：SEM 的优势在于“测量模型 + 结构模型”的统一估计。

## R17

`demo.py` 功能清单：

- 合成一个两潜变量 SEM 数据集（6 个观测指标）；
- 基于样本协方差最小化 `F_ML` 估计参数；
- 输出拟合统计量（`F_ML`、近似卡方、RMSEA、协方差残差 RMS）；
- 打印参数恢复表（真值、估计值、绝对误差）；
- 执行断言检查，成功时输出 `All checks passed.`。

运行方式：

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `simulate_sem_data` 依据真值参数采样 `xi`、`zeta`，构造 `eta = beta*xi + zeta`，再生成 6 个观测指标。  
2. `unpack_raw_params` 把优化向量映射为可解释参数，并用 `softplus` 强制方差参数为正。  
3. `build_lambda` 与 `latent_covariance` 分别构造 `Lambda` 和 `Phi`，随后在 `implied_covariance_from_raw` 里得到 `Sigma(theta)`。  
4. `f_ml_objective` 计算 `F_ML = log|Sigma| + tr(S Sigma^-1) - log|S| - p`，若 `Sigma` 非正定或线性求解失败则返回大惩罚。  
5. `fit_sem_covariance_mle` 优先用 `scipy.optimize.minimize(L-BFGS-B)`，若环境无 `scipy` 则回退到源码内置的有限差分优化器，最小化 `F_ML`。  
6. `format_recovery_table` 组装参数恢复表，对比真值与估计值并计算绝对误差。  
7. `main` 计算 `chi-square` 近似量、`RMSEA` 近似量与协方差残差 RMS，输出模型拟合质量。  
8. `main` 执行质量门禁断言（`Sigma_hat` 正定、`beta` 恢复合理、残差不过大），全部通过后打印 `All checks passed.`。  

这 8 步完整覆盖了“数据生成 -> 参数化 -> 协方差映射 -> ML 拟合 -> 诊断校验”的 SEM 最小实现链路。
