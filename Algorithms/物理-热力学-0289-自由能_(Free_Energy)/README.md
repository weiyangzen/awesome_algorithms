# 自由能 (Free Energy)

## R01

自由能用于把“能量守恒”与“熵效应”合并成一个可比较的势函数。常用两种形式：

- Helmholtz 自由能：`F = U - TS`（定 `T,V,N` 下最小化）；
- Gibbs 自由能：`G = H - TS`（定 `T,P,N` 下最小化）。

本条目聚焦 Gibbs 自由能最小化，展示平衡态可由“自由能极小”直接求得。

## R02

在工程上，自由能最小化是一个“从状态方程走向决策”的接口：

- 给定 `T,P` 时，系统会趋向使 `G` 最小的组成/相态；
- 自由能曲线的梯度和曲率可直接判断平衡点与稳定性；
- 相比只看能量项，自由能同时包含混合熵贡献，更符合实际热力学行为。

MVP 目标：做一个可运行、可验证的最小算法，演示“数值最小化 = 解析平衡解”。

## R03

`demo.py` 输入输出（脚本内置参数，无交互）：

- 输入：
1. 二元理想混合物标准化学势 `mu_a0, mu_b0`（J/mol）；
2. 温度扫描数组 `T`（K）；
3. 组分 B 摩尔分数定义域 `x_B in (0,1)`。
- 输出：
1. 温度扫描表：`x_B` 数值最优解、解析解、误差、梯度、曲率、自由能对比；
2. 指定温度下的 `g(x_B)` 曲线样表；
3. 断言校验并打印 `All checks passed.`。

## R04

MVP 使用的核心关系：

1. 理想二元混合物摩尔 Gibbs 自由能：
`g(x_B) = (1-x_B)mu_A0 + x_B mu_B0 + RT[(1-x_B)ln(1-x_B) + x_B ln x_B]`。

2. 一阶导（平衡条件）：
`dg/dx_B = (mu_B0 - mu_A0) + RT ln(x_B/(1-x_B)) = 0`。

3. 解析平衡解：
`x_B* = 1 / (1 + exp((mu_B0 - mu_A0)/(RT)))`。

4. 二阶导（稳定性）：
`d2g/dx_B2 = RT(1/x_B + 1/(1-x_B)) > 0`，故内点极值是稳定极小值。

## R05

复杂度分析：

- 单温度点解析解：`O(1)`；
- 单温度点数值最小化：`O(K)`，`K` 为 `minimize_scalar` 迭代评估次数；
- 温度扫描 `M` 个点总复杂度：`O(MK)`；
- 空间复杂度：`O(M)`（主要是输出表格）。

## R06

`demo.py` 的最小算法闭环：

- 闭环 A（物理建模）：`gibbs_molar` 明确写出能量项 + 混合熵项；
- 闭环 B（平衡求解）：`numerical_equilibrium_fraction` 在 `(0,1)` 上做一维最小化；
- 闭环 C（理论对照）：`analytic_equilibrium_fraction` 给出闭式解并做误差对比；
- 闭环 D（稳定性校验）：检查 `dg/dx≈0`、`d2g/dx2>0`、`g_eq` 低于纯态基线。

## R10

正确性直觉：

1. 自由能曲线 `g(x_B)` 的最低点对应热力学平衡组成；
2. 解析一阶条件给出 `x_B*`，是理论基准；
3. 数值最小化若实现正确，应与解析解在容差内一致；
4. 在最优点附近应满足梯度接近 0 且曲率为正，保证是稳定极小值。

## R11

数值稳定性策略：

- 约束 `x_B` 在开区间 `(0,1)`，避免 `ln(0)` 奇异；
- 用 `z = clip((mu_B0-mu_A0)/(RT), [-700,700])` 防止指数溢出；
- 最小化边界设置为 `[1e-12, 1-1e-12]`；
- 用解析解、梯度、曲率三重校验降低“误收敛”风险。

## R12

关键参数：

- `temperature`（K）：温度，必须 `> 0`；
- `mu_a0, mu_b0`（J/mol）：两组分标准化学势；
- `x_B`：B 组分摩尔分数（`0 < x_B < 1`）；
- `xatol`：一维最小化停止精度。

调参建议：

- 若 `mu_b0 - mu_a0` 很大，低温下 `x_B*` 会更靠近 0；
- 温度升高时混合熵项增强，平衡组成通常更向中间移动；
- 做鲁棒性测试可扩大温度范围并观察 `abs_x_error` 与梯度残差。

## R14

常见失效模式：

1. 把温度用摄氏度输入导致比例项错误；
2. 允许 `x_B=0` 或 `1` 直接进入对数项造成数值崩溃；
3. 混淆“标准化学势差”符号，导致解析解方向反了；
4. 忽略单位一致性（`mu` 不是 J/mol）；
5. 只看优化器返回值，不检查梯度与曲率。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `BinaryFreeEnergyModel.validate` 检查 `T>0`，确保热力学表达式定义域有效。  
2. `gibbs_molar` 按“能量项 + 混合熵项”显式构造 `g(x_B)`，并限制 `x_B in (0,1)`。  
3. `gibbs_gradient` 写出 `dg/dx_B`，`gibbs_hessian` 写出 `d2g/dx_B2`，用于平衡与稳定性判定。  
4. `analytic_equilibrium_fraction` 由 `dg/dx_B=0` 给出闭式 `x_B*`，并做指数裁剪避免溢出。  
5. `numerical_equilibrium_fraction` 在有界区间上调用 `scipy.optimize.minimize_scalar` 求 `g(x_B)` 数值极小点。  
6. `build_equilibrium_scan` 对多个温度重复第 4-5 步，汇总解析/数值误差、梯度残差和曲率。  
7. `build_free_energy_profile` 在固定温度网格采样 `g(x_B)`，输出相对平衡点的 `delta_g` 轮廓。  
8. `main` 执行断言：`x` 误差小、梯度接近 0、曲率为正、平衡自由能优于纯态，最终输出通过信息。  

补充：`scipy` 只负责一维有界搜索器本身；自由能构造、导数、解析解、稳定性判据均在源码中显式展开，不是黑盒一键求解。
