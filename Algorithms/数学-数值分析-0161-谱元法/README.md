## R01

谱元法（Spectral Element Method, SEM）是把“谱方法的高阶基函数精度”和“有限元的分块装配能力”结合起来的一类高阶离散方法。它常用于流体力学、波动方程、椭圆型边值问题等场景，尤其适合追求高精度且解较光滑的问题。

本目录给出一个最小可运行 MVP：
- 问题：一维 Poisson 方程 `-u''=f`；
- 离散：多单元 + GLL（Gauss-Lobatto-Legendre）节点的拉格朗日基；
- 输出：残差、相对 `L2` 误差、最大绝对误差，并展示 `p` 加密与 `h` 加密趋势。

## R02

MVP 求解的连续问题：

- 方程：`-u''(x)=f(x), x in (0,1)`
- 边界：`u(0)=u(1)=0`
- 解析解（制造解）：`u*(x)=sin(pi x)`
- 对应右端：`f(x)=pi^2 sin(pi x)`

因为真解已知，脚本可直接给出离散解误差，便于验证算法实现是否正确。

## R03

谱元法在本问题中的建模思路：

1. 将区间 `[0,1]` 均匀划分为 `E` 个单元；
2. 每个单元映射到参考区间 `[-1,1]`；
3. 在参考区间上，用 `p+1` 个 GLL 节点构造 nodal Lagrange 基函数；
4. 单元刚度用高阶积分（此处用 GLL 配点/求积）计算；
5. 全局装配形成稀疏结构线性系统（MVP 用致密矩阵表达，逻辑等价）；
6. 施加 Dirichlet 边界后解内部自由度。

## R04

GLL 节点与权重是本实现的关键：

- 节点包含两端点 `-1, 1`，内部节点来自 `P_p'(x)=0`（`P_p` 为 Legendre 多项式）；
- 权重公式：`w_i = 2 / (p(p+1)[P_p(x_i)]^2)`；
- 代码中未直接调用一键黑盒节点生成器，而是通过：
  - `Legendre.basis(p).deriv().roots()` 求内部节点；
  - 再按上式计算权重。

这样可以把谱元法的核心数值对象（节点/权重）暴露出来，便于审查。

## R05

参考单元上的离散算子：

- 记 `D_{ij} = d l_j / d xi` 在节点 `xi_i` 处的值（`l_j` 是拉格朗日基）；
- `D` 通过重心权（barycentric weights）显式构造，不依赖外部黑盒；
- 参考刚度矩阵：`K_ref = D^T W D`，其中 `W=diag(w_i)`。

映射到物理单元后（单元雅可比 `J=h/2`）：
- `K_local = (1/J) K_ref`；
- `b_local = J * w * f(x_local)`。

## R06

全局装配规则（`C0` 连续）：

- 全局节点数：`N = E * p + 1`；
- 第 `e` 个单元第 `i` 个局部点的全局索引：`g = e*p + i`；
- 对每个单元执行：
  - `K[g_i, g_j] += K_local[i, j]`
  - `b[g_i] += b_local[i]`

该规则体现了谱元法“高阶单元 + 有限元式装配”的本质。

## R07

边界条件处理：

- 本例使用齐次 Dirichlet，故 `u_0=u_{N-1}=0`；
- 仅对内部自由度 `1..N-2` 求解线性系统；
- 求解后把边界值写回全局向量即可。

对应代码位于 `solve_sem_poisson`：
- 抽取 `K_ii` 与 `b_i`；
- `numpy.linalg.solve(K_ii, b_i)` 求内部解。

## R08

复杂度分析（MVP 实现口径）：

- 节点/导数矩阵构建：单元阶次为 `p` 时约 `O(p^2)`；
- 全局装配：约 `O(E * p^2)`；
- 线性求解：MVP 用致密求解器，复杂度约 `O(N^3)`；
- 内存：致密存储约 `O(N^2)`。

工程实现通常会把 `K` 换成稀疏格式并用迭代法，以更贴合谱元法的大规模应用。

## R09

数值特性与预期：

- 固定单元数时，提升多项式阶数（`p`-refinement）通常出现快速误差下降；
- 固定阶数时，加密网格（`h`-refinement）也会稳定降误差；
- 对光滑解，谱元法往往比低阶有限元更快达到高精度。

本条目用 `u*=sin(pi x)` 进行双重验证：
- 线性方程残差接近机器精度；
- 相对误差随 `p` 或 `E` 增大而下降。

## R10

`demo.py` 的默认实验组：

- Case A（`p` 加密）：固定 `E=4`，`p=1..6`；
- Case B（`h` 加密）：固定 `p=4`，`E=2,4,8,16`。

每组打印：
- `residual_l2`：内部自由度残差 `||Ku-b||_2`；
- `rel_l2_error`：按单元 GLL 积分估计的相对 `L2` 误差；
- `max_abs_error`：节点最大绝对误差。

## R11

代码结构与函数职责：

- `gll_nodes_weights`：构造 GLL 节点与权重；
- `differentiation_matrix`：构造参考单元导数矩阵 `D`；
- `reference_stiffness`：生成 `K_ref`；
- `assemble_global_system`：单元映射 + 全局装配；
- `sem_l2_relative_error`：基于 GLL 求积的相对 `L2` 误差；
- `solve_sem_poisson`：施加边界并求解；
- `main`：组织实验并打印结果。

## R13

输出解读建议：

- 先看 `residual_l2`：若接近 `1e-13 ~ 1e-15`，说明线性系统求解一致性正常；
- 再看 `rel_l2_error`：应随 `p` 或 `E` 增大而下降；
- 最后看 `max_abs_error`：用于观察节点点值误差上界。

若出现不降反升，通常意味着：
- 节点映射或局部装配系数写错；
- 边界条件处理不一致；
- 导数矩阵 `D` 对角项构造错误。

## R14

MVP 的取舍边界：

- 优点：
  - 实现短小；
  - 关键算子透明；
  - 可直接用于教学和算法流程验证。
- 限制：
  - 仅一维齐次 Dirichlet；
  - 全局矩阵用致密存储；
  - 未做并行与预条件优化。

这符合“最小但诚实”的交付目标。

## R15

常见实现坑：

- 把 `P_p` 的根误当作 GLL 内部节点（正确是 `P_p'` 的根）；
- 忘记物理映射因子（`K_local` 应乘 `1/J`，`b_local` 应乘 `J`）；
- 接口节点索引拼接错误（`E*p+1` 是核心）；
- 仅看方程残差，不看真解误差，导致错误难以及时发现。

## R16

与相近方法的关系：

- 与低阶有限元：
  - 相同装配思想；
  - SEM 单元内是高阶基函数，精度/自由度比更高。
- 与全局谱方法：
  - 都使用高阶多项式思想；
  - SEM 通过分块增强几何适应性和并行扩展性。
- 与有限差分：
  - SEM 在复杂几何与高精度需求下更有优势；
  - 但实现复杂度更高。

## R17

可扩展方向：

1. 改为稀疏矩阵 + 共轭梯度（或多重网格预条件）以适应更大规模；
2. 扩展到二维/三维张量积谱元；
3. 引入非均匀网格与曲线映射单元；
4. 支持非齐次边界与变系数算子；
5. 加入时间离散，扩展到对流扩散或波动问题。

## R18

`demo.py` 的源码级算法流程（8 步，非黑盒）如下：

1. `main` 设定 `(E,p)` 组合，分别做 `p` 加密与 `h` 加密实验。  
2. `solve_sem_poisson` 调用 `assemble_global_system`，为每组参数构建 `K,b,x`。  
3. `assemble_global_system` 内部先调用 `reference_stiffness`，后者通过 `gll_nodes_weights + differentiation_matrix` 显式得到 `K_ref`。  
4. `gll_nodes_weights` 先取 `P_p'` 的内部根，再用 `w_i = 2/(p(p+1)[P_p(x_i)]^2)` 计算 GLL 权重。  
5. 每个单元执行仿射映射 `x = x_left + J*(xi+1)`，并按 `K_local=(1/J)K_ref`、`b_local=J*w*f(x_local)` 形成局部离散。  
6. 依据 `g=e*p+i` 的索引规则把局部矩阵/向量累加到全局 `K,b`，形成连续谱元系统。  
7. 对内部自由度解 `K_ii u_i = b_i`（边界自由度固定为 0），再用 `K.dot(u)-b` 计算残差范数。  
8. `sem_l2_relative_error` 逐单元用 GLL 求积累计 `||u-u*||_2 / ||u*||_2`，并与 `max_abs_error` 一起输出，完成数值验证。
