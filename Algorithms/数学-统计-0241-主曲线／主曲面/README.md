# 主曲线/主曲面

- UID: `MATH-0241`
- 学科: `数学`
- 分类: `统计`
- 源序号: `241`
- 目标目录: `Algorithms/数学-统计-0241-主曲线／主曲面`

## R01

主曲线（Principal Curve）与主曲面（Principal Surface）是 PCA 的非线性推广：
- PCA 用线性子空间逼近数据；
- 主曲线/主曲面用“可弯曲”的 1D/2D 流形逼近数据。

本条目目标是给出可运行 MVP：
- 在 2D 数据上拟合主曲线；
- 在 3D 数据上拟合主曲面；
- 与线性 PCA 重构误差做对比，展示非线性收益。

## R02

问题定义：
- 输入：样本矩阵 `X in R^(nxd)`。
- 输出：
  - 主曲线情形：参数 `t_i` 与曲线映射 `f(t)`；
  - 主曲面情形：参数 `(u_i, v_i)` 与曲面映射 `g(u,v)`；
  - 以及重构点 `f(t_i)` 或 `g(u_i,v_i)`。

优化目标（近似）：最小化样本到流形的平方距离总和。

## R03

统计学定义（自洽性）要点：
- 主曲线满足 `f(t) = E[X | lambda_f(X)=t]`；
- 主曲面满足 `g(u,v) = E[X | lambda_g(X)=(u,v)]`；
- 其中 `lambda` 是“投影到流形最近点”的参数映射。

MVP 不直接解解析方程，而采用“参数-平滑-投影”迭代逼近该自洽条件。

## R04

主曲线算法（Hastie-Stuetzle 风格）流程：
1. 用 PCA 第一主成分分数初始化参数 `t`；
2. 固定 `t`，对每个坐标维度拟合一元平滑样条 `x_j ~ s_j(t)`；
3. 固定样条曲线，在参数网格上为每个样本找最近曲线点并更新 `t`；
4. 重复 2-3 直到参数变化小于阈值或达到迭代上限。

## R05

主曲面算法（二维潜变量）流程：
1. 用 PCA 前两主成分初始化潜变量 `UV=(u,v)`；
2. 固定 `UV`，对每个坐标维度拟合 `KernelRidge(RBF)` 回归 `x_j ~ m_j(u,v)`；
3. 在 `(u,v)` 网格上生成曲面点云；
4. 对每个样本分配最近网格点，得到新 `UV`；
5. 重复 2-4 直到收敛。

## R06

正确性直觉：
- “平滑步骤”近似了条件期望 `E[X|参数]`；
- “投影步骤”近似了 `lambda(X)` 最近点映射；
- 两步交替迭代本质上是“自洽流形”的固定点迭代；
- 收敛后可得到比线性 PCA 更贴合弯曲数据结构的重构。

## R07

复杂度（设样本数 `n`，曲线投影网格 `G`，曲面网格 `GxG`）：
- 主曲线每轮投影约 `O(n*G*d)`；
- 主曲面每轮最近点分配约 `O(n*G^2*d)`；
- Kernel Ridge 训练核心包含核矩阵运算，量级约 `O(n^3)`（小样本 MVP 可接受）。

因此本实现面向“教学与小规模验证”，非大规模工业 ANN 版本。

## R08

边界与异常处理：
- 输入必须是二维矩阵、样本数 `>=10`、且全为有限值；
- `max_iter`、网格大小设置下限，防止无意义参数；
- 样条自变量排序后做“严格递增修正”，避免重复参数导致拟合失败；
- 网格区间退化时注入最小跨度，避免除零/空区间。

## R09

MVP 工程取舍：
- 只用 `numpy + scipy + scikit-learn`，避免大框架；
- 不依赖“现成主曲线库”黑盒，核心迭代逻辑全部显式写出；
- 选可控合成数据（2D 弯曲曲线、3D 弯曲曲面）做最小闭环验证。

## R10

`demo.py` 主要函数职责：
- `make_curve_data` / `make_surface_data`：生成演示数据；
- `fit_principal_curve`：执行主曲线迭代；
- `fit_principal_surface`：执行主曲面迭代；
- `pca_reconstruction_mse`：给线性 PCA 做对照基线；
- `main`：运行两组实验并打印指标。

## R12

输出字段解读：
- `iterations` / `converged`：迭代轮数与是否触发收敛阈值；
- `principal_curve_mse`：主曲线重构误差（每样本平方距离均值）；
- `pca1_mse`：PCA 一维重构误差基线；
- `principal_surface_mse` 与 `pca2_mse`：主曲面与二维 PCA 对照；
- `improvement`：`PCA_MSE / Principal_MSE`，大于 1 表示非线性流形更优。

## R13

建议最小测试集：
- 功能测试：默认参数直接运行，确认无异常输出；
- 收敛测试：增大 `max_iter`，观察 `delta` 是否更快趋稳；
- 参数敏感性：调整 `smooth_factor`、`alpha`、`gamma`、网格大小；
- 退化测试：给近似线性数据，验证主曲线/面收益会下降。

## R14

关键可调参数：
- 主曲线：`smooth_factor`、`projection_grid_size`、`tol`、`max_iter`；
- 主曲面：`alpha`、`gamma`、`grid_size`、`tol`、`max_iter`；
- 数据规模：`n_samples` 可调，样本增大时注意曲面算法耗时上升。

## R15

与相关方法对比：
- 相比 PCA：主曲线/面可刻画弯曲结构，但计算更重；
- 相比 Isomap/LLE/t-SNE：这些偏“降维可视化”，不直接给可用重构映射；
- 相比自编码器：神经网络表达力更强，但实现和调参成本更高，MVP 不需要。

## R16

应用场景：
- 非线性轨迹压缩（时间轨迹、生物轨迹）；
- 低维流形近似（传感器数据、几何点云）；
- 作为更复杂流形学习方法的可解释基线。

## R17

可扩展方向：
- 用局部优化替代纯网格最近点投影，提高精度与速度；
- 加入鲁棒损失（Huber/分位损失）应对离群点；
- 曲面部分改为 Nyström/随机特征核近似以降低 `O(n^3)` 开销；
- 增加单元测试，自动检查收敛与误差上界。

## R18

`demo.py` 源码级算法流（8 步）：
1. `main` 生成 2D 与 3D 合成数据，并分别进入主曲线/主曲面流程。  
2. 主曲线初始化：`PCA(n_components=1)` 取线性分数 `t` 作为首轮参数。  
3. 主曲线平滑：`UnivariateSpline` 对每个坐标维拟合 `x_j = s_j(t)`（`s` 控制平滑强度）。  
4. 主曲线投影：在参数网格上计算曲线点，对每个样本找最近点并更新 `t`，直到 `delta < tol`。  
5. 主曲面初始化：`PCA(n_components=2)` 得到潜变量 `UV=(u,v)`。  
6. 主曲面平滑：对每个坐标维训练 `KernelRidge(RBF)`，其训练内部是“构造核矩阵 K + 求解 `(K + alpha I)w = y`”。  
7. 主曲面投影：在 `UV` 规则网格预测曲面点，对每个样本做最近点分配并更新 `UV`，迭代至收敛。  
8. 分别计算主曲线/主曲面 MSE，并与线性 `PCA(1D/2D)` 重构误差对照，输出改进倍数。
