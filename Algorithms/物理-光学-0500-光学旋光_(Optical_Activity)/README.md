## R01

光学旋光描述的是手性介质使线偏振光偏振面发生旋转的现象。常见实验关系式：

`theta = [alpha](lambda, T) * l * c + b`

- `theta`：测得旋光角（deg）
- `[alpha]`：比旋光度（与波长/温度相关）
- `l`：光程（dm）
- `c`：浓度（g/mL）
- `b`：仪器零点偏移（deg）

本条目聚焦“如何从多组实验数据反推出可解释的旋光模型参数”。

## R02

MVP 目标：

1. 用含噪模拟数据构造一批 `(lambda, l, c, theta)` 观测；
2. 拟合 Drude 型旋光色散模型：
   `[alpha](lambda) = A / (lambda^2 - lambda0^2)`；
3. 在拟合后输出 D 线（589 nm）比旋光度估计值；
4. 演示“已知 `theta` 反演未知样品浓度”的实用流程。

实现强调透明可追踪，不依赖黑盒求解器。

## R03

采用的一阶色散模型：

`theta_i = l_i * c_i * A/(lambda_i^2 - lambda0^2) + b + epsilon_i`

其中 `epsilon_i` 是测量噪声。模型结构特点：

- 对 `A, b` 线性；
- 对 `lambda0` 非线性；
- 因而可用“外层扫描 `lambda0` + 内层线性最小二乘”的分层求解。

## R04

理论与物理解释：

1. 在可见光区通常有 `lambda > lambda0`，故分母为正；
2. `A` 的符号决定旋光方向：`A>0` 常对应右旋（dextrorotatory），`A<0` 对应左旋；
3. `|[alpha]|` 随波长变化体现旋光色散；
4. 对工程测量而言，589 nm（钠 D 线）是常用参考点。

## R05

数值算法设计（非黑盒）：

- 外层：在给定区间内网格扫描 `lambda0`；
- 内层：固定 `lambda0` 后构造特征 `x_i=l_i*c_i/(lambda_i^2-lambda0^2)`；
- 线性回归：解 `theta_i ≈ A*x_i + b` 的最小二乘闭式解；
- 评价：以 SSE/RMSE 作为准则，选最优 `lambda0`；
- 导出：计算 `[alpha]_589` 与未知样品浓度反演结果。

## R06

`demo.py` 会输出：

- 合成数据前几行（便于人工核查量纲）；
- 拟合参数 `A, lambda0, b` 及 `RMSE, R^2`；
- `[alpha]_589` 的真值与估计对比；
- 未知样品浓度反演结果与误差；
- 3 条断言，自动检查“拟合是否达到基本可用精度”。

## R07

优点：

- 模型和求解都很小，便于教学与验证；
- 把非线性问题拆成“一维扫描 + 线性解”，稳定且可解释；
- 参数具备直接物理意义（旋光方向、色散强度、共振尺度）。

局限：

- 仅使用单项 Drude 近似；
- 未显式建模温度依赖；
- 噪声假设为独立同分布高斯，未处理系统漂移。

## R09

适用场景：

- 旋光实验课程中的参数反演示例；
- 快速评估材料是左旋还是右旋；
- 用于后续更复杂模型（多项色散、温度项）的基线初始化。

不适用场景：

- 需要高精度光谱反演（多峰共振）；
- 需要完整电磁微观响应建模；
- 含显著非高斯异常值却不做鲁棒回归的场景。

## R10

正确性直觉：

1. 若 `lambda0` 给定，问题退化为线性回归并可解析求解；
2. 对每个候选 `lambda0` 都能得到“该假设下最佳 `A,b`”；
3. 全局比较残差，残差最小的 `lambda0` 对应最一致的物理解释；
4. 参数回代后可直接检验“预测角度”和“浓度反演”是否自洽。

## R11

数值稳定性处理：

- `lambda0` 搜索上限设在最小测量波长下方，避免分母接近 0；
- 线性系统使用 `np.linalg.lstsq`，避免手写逆矩阵病态问题；
- 计算 `R^2` 时对总方差近零情况做保护；
- 通过固定随机种子保证示例可复现；
- 断言阈值设置为“物理上合理但不过度苛刻”。

## R12

关键参数（见 `OpticalActivityConfig`）：

- `lambda0_search_min/max/step`：外层扫描范围与精细度；
- `noise_std_deg`：测量噪声水平；
- `n_samples`：观测数量；
- `true_A/true_lambda0_nm/true_offset_deg`：生成演示数据的真值；
- `unknown_c_true`：未知样品真实浓度（用于验证反演误差）。

调参经验：

- 精度优先可减小 `step`；
- 速度优先可增大 `step`，但会降低 `lambda0` 分辨率。

## R13

保证说明：

- 近似比保证：N/A（连续参数估计问题，不是组合优化）。
- 求解保证：对每个候选 `lambda0`，最小二乘给出全局最优 `A,b`；整体仅对离散扫描网格保证“网格最优”。

工程侧保证：

- 无交互输入；
- 单文件可运行；
- 输出含误差指标与断言，便于自动验收。

## R14

常见失败模式：

1. 忽略零点偏移 `b`，导致浓度反演系统偏差；
2. 直接黑盒非线性拟合，参数互相关时难解释失败原因；
3. `lambda0` 扫描区间包含奇异点附近，数值爆炸；
4. 样本中 `l*c` 变化太小，导致回归可辨识性差；
5. 噪声过大但样本过少，出现“看似拟合、实则不稳”。

## R18

`demo.py` 的源码级流程（8 步）：

1. `generate_synthetic_dataset` 生成多组 `(lambda, l, c, theta)` 含噪观测。  
2. 在 `fit_optical_activity_model` 中构建 `lambda0` 候选网格。  
3. 固定某个 `lambda0`，用 `design_feature` 计算特征 `x=l*c/(lambda^2-lambda0^2)`。  
4. 调用 `solve_linear_least_squares` 解 `theta ≈ A*x+b`，得到该 `lambda0` 下最优 `A,b`。  
5. 计算该候选的 `SSE/RMSE`，并与当前最优解比较。  
6. 扫描结束后得到全局最优网格点 `(A_hat, lambda0_hat, b_hat)`，再计算 `R^2`。  
7. 用 `specific_rotation` 计算 589 nm 比旋光度，并据此用 `invert_concentration` 反演未知样品浓度。  
8. `main` 打印结果并执行断言（`lambda0` 误差、`[alpha]_589` 相对误差、浓度反演误差）。  

第三方库边界：`numpy/pandas` 仅用于数组运算与表格展示；参数搜索策略与回归流程均在源码中显式实现，不依赖专用黑盒光谱拟合包。
