## R01

声子（Phonon）是晶格振动的量子化激发。为保证条目“可运行、可验证”，本目录把抽象概念落成一个最小数值模型：

- 在一维单原子近邻弹簧链上显式计算色散关系 `omega(q)`；
- 用玻色分布计算热激发能量 `U(T)` 与热容 `C(T)`；
- 用可执行断言验证声学支的关键物理行为（小 `q` 线性、低温标度、高温极限）。

## R02

该条目的价值是把“声子概念”连接到可计算输出：

- 解释晶格热容随温度变化的来源；
- 提供热导分析中常用的中间量（色散、群速度、模态统计）；
- 为后续电-声子相互作用、德拜模型、输运模型提供可复用基础模块。

## R03

本 MVP 的边界与交付：

- 输入：`PhononConfig` 中的 `K, M, a, n_q, 温度网格`；
- 输出：
  1. 声子色散采样表（`q/pi/a`, `omega_THz`, `v_group`）；
  2. 热力学采样表（`U_th`, `C/k_B` 等）；
  3. 4 个物理 sanity checks 指标；
- 约束：不依赖黑盒声子库，核心流程在 `demo.py` 显式实现。

## R04

`demo.py` 的物理模型与公式（1D 单原子链，最近邻谐振）：

1. 色散关系：
`omega(q) = 2*sqrt(K/M)*|sin(qa/2)|`

2. 长波极限声速：
`v_s = a*sqrt(K/M)`，且 `omega(q) ~ v_s*|q|`（`q -> 0`）

3. 玻色占据数：
`n_B(omega,T) = 1 / (exp(hbar*omega/(k_B*T)) - 1)`

4. 每原子热激发能（按布里渊区模态平均）：
`U_th(T) = < hbar*omega_q * n_B(omega_q,T) >_q`

5. 每原子热容：
`C(T) = k_B * < x^2*exp(x)/(exp(x)-1)^2 >_q`，其中 `x = hbar*omega_q/(k_B*T)`

## R05

数值算法流程（实现级）：

1. 参数校验：保证 `K,M,a,T,n_q` 合法。
2. 构造第一布里渊区中点 `q` 网格（避免采样到严格 `q=0` 零频平移模）。
3. 计算 `omega(q)` 与解析群速度 `v_g = d omega / d q`。
4. 对温度网格广播计算 `x = hbar*omega/(k_B*T)`。
5. 显式计算 `U_th(T)` 与 `C(T)`（非黑盒积分器）。
6. 生成 `pandas` 输出表。
7. 执行 4 条 sanity checks 断言。

## R10

正确性验证逻辑（`demo.py` 断言）：

1. 小波矢线性色散：`max_rel_err < 2e-3`。
2. 低温标度（1D 声学支）：`C(T)/T` 在 `6K` 与 `12K` 近似常数，允许偏差 `< 0.12`。
3. 高温极限：`C/k_B -> 1`，在 `5000K` 要求偏差 `< 0.03`。
4. 热容正性：报告温区内 `C > 0`。

这四条对应“色散正确、统计正确、极限正确、数值健康”。

## R11

数值稳定性策略：

- `q` 网格采用中点采样，避免 `q=0` 处零频退化导致的公式病态；
- 指数计算使用 `clip`，抑制大 `x` 溢出；
- `x -> 0` 时热容核使用级数近似 `1 - x^2/12`，避免 `0/0`；
- 所有可变参数在入口做合法性检查，异常早失败。

## R14

常见失败模式与修复建议：

1. `K/M/a` 单位混乱（例如质量未用 SI kg）。
   - 修复：统一使用 SI，脚本内常量也采用 SI。
2. 温度网格含非正值。
   - 修复：保持 `T>0`，并依赖 `validate_config` 检查。
3. `n_q` 太小导致采样噪声，低温和小 `q` 校验失真。
   - 修复：增大 `n_q`（建议 `>=1024`）。
4. 直接用不稳定指数表达式，出现 `nan/inf`。
   - 修复：使用当前 `clip + expm1 + 小x级数` 实现。

## R18

`demo.py` 源码级算法流程（8 步，非黑盒）：

1. `main` 创建 `PhononConfig`，并调用 `validate_config` 进行参数合法性检查。  
2. `build_q_grid` 在第一布里渊区生成中点 `q` 网格。  
3. `phonon_dispersion_monatomic_1d` 计算 `omega(q)`，`group_velocity_monatomic_1d` 计算 `v_g(q)`。  
4. `build_thermo_table` 生成温度网格，并调用 `phonon_thermo_curves`。  
5. `phonon_thermo_curves` 构造二维无量纲数组 `x = hbar*omega/(k_B*T)`，显式求 `n_B`、`U_th(T)` 与 `C(T)`。  
6. `_heat_capacity_kernel` 在 `x->0` 用级数、在常规区用稳定重写公式，避免数值病态。  
7. `run_sanity_checks` 执行 4 条物理断言（小 `q` 线性、低温标度、高温极限、正性）。  
8. `main` 打印参数摘要、色散采样表、热力学采样表和校验指标。  

第三方库角色：`numpy` 负责数组与向量化计算，`pandas` 负责表格展示；声子色散、统计核、热容与断言流程均在源码显式展开，没有把核心算法委托给黑盒库。
