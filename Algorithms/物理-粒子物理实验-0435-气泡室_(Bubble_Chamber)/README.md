# 气泡室 (Bubble Chamber)

- UID: `PHYS-0415`
- 学科: `物理`
- 分类: `粒子物理实验`
- 源序号: `435`
- 目标目录: `Algorithms/物理-粒子物理实验-0435-气泡室_(Bubble_Chamber)`

## R01

气泡室算法的目标是把“照片里的气泡轨迹”还原成可计算的粒子运动参数。  
本目录 MVP 采用一个透明链路：
- 输入：单条轨迹的二维点集 `(x_i, y_i)`（视作已分割）与单位长度气泡密度观测值
- 步骤1：用最小二乘圆拟合恢复曲率半径 `r`
- 步骤2：依据点序方向估计旋转方向，再结合磁场方向推断电荷符号 `q`
- 步骤3：由 `p_T = 0.3 * |q| * B * r`（GeV/c, T, m）反演横向动量
- 步骤4：以 `(p_T, 气泡密度)` 作为特征做粒子粗分类（电子/π介子/质子等）
- 输出：`r, q, p_T, 拟合残差, 粒子类别`

## R02

历史节点可压缩为三段：
- 1952 年：Donald Glaser 提出气泡室并实现可视化带电粒子轨迹
- 1950s-1970s：气泡室成为高能实验主力装置，发现大量共振态与新粒子
- 后续：逐步被多丝室、漂移室、硅径迹探测器替代，但“曲率重建动量”的核心算法思想延续至今

因此该条目的算法起点可锚定在 `1952` 年。

## R03

提出背景：实验上需要从照片证据恢复动力学量，而不是只做定性观察。  
在均匀磁场中，带电粒子轨迹近似为圆弧（二维投影）或螺旋线（三维）。如果不做曲率拟合：
- 无法从图像定量反演动量
- 无法通过旋向判断电荷符号
- 无法把气泡疏密（与 `dE/dx` 相关）用于粒子鉴别

该算法正是把“实验影像 -> 可测物理量”的关键桥梁。

## R04

设事件中有 `K` 条轨迹，每条有 `N` 个采样点，非线性圆拟合迭代 `I` 次：
- 圆拟合时间复杂度：`O(K * N * I)`（代数初值 + 迭代优化）
- 电荷与动量反演：`O(K * N)`（角度展开与一阶统计）
- KNN 分类预测：训练后单样本约 `O(M * d)`（`M` 为训练样本数，`d=2`）

对教学级/小样本 MVP，主要耗时在拟合和距离计算常数项，不在渐近项本身。

## R05

空间复杂度：
- 轨迹点存储：`O(K * N)`
- 逐轨中间量（半径、残差、动量、电荷等）：`O(K)`
- KNN 训练集：`O(M * d)`

总空间一般由“轨迹点阵列 + 训练特征矩阵”主导。

## R06

示例（脚本内置）：
- 先合成 6 条不同粒子轨迹（电子、π介子、质子混合），给定真实 `p` 与 `q`
- 对每条轨迹加测量噪声，模拟气泡室读取误差
- 用圆拟合得到 `r_est`，再算 `pT_est = 0.3 * B * r_est`
- 用旋向判定 `q_est`，并用 `(pT_est, bubble_density)` 做粒子识别

输出表格会并列展示 `truth vs estimate`，便于快速检查误差和分类效果。

## R07

算法意义：
- 物理意义：把洛伦兹力定律转成可执行的反问题求解
- 实验意义：实现从“照片证据”到“动量、电荷、粒子种类”的可复现流程
- 方法意义：展示几何拟合、统计分类与物理先验如何组合成完整重建链

## R08

核心理论与数值组件：
- 洛伦兹力与曲率关系：`r = p_T / (0.3|q|B)`
- 气泡密度近似：低能区可用 `1/beta^2` 形态描述 `dE/dx` 趋势
- 代数圆拟合（线性最小二乘）作为初值
- SciPy `optimize.least_squares` 做非线性半径残差最小化
- scikit-learn `KNeighborsClassifier` 做二维特征粒子鉴别（非深度学习黑箱）

## R09

适用前提与边界：
- 默认二维投影重建（忽略纵向螺旋节距）
- 轨迹点默认已分割为“单条轨迹”，不含复杂多轨交叠分割过程
- 假设磁场近似均匀，且粒子荷电量绝对值取 `|q|=1e`（教学简化）
- 气泡密度模型用于粗分类，不替代精细 Bethe-Bloch 标定曲线

## R10

正确性来源可拆为三层：
- 几何正确性：圆拟合残差最小化给出与观测点最一致的曲率半径
- 物理正确性：`r -> p_T` 的映射来自标准磁谱仪关系式
- 分类正确性：KNN 按“特征空间近邻”投票，决策依据可直接检查，不是不可解释模型

即使分类有误差，`r` 与 `p_T` 重建链路仍可独立验证。

## R11

数值稳定性注意点：
- 圆弧太短（角跨度小）会导致圆参数病态，半径估计方差上升
- 半径很大时（高动量），曲率接近 0，微小点噪声会显著放大到 `p_T` 误差
- 旋向判断依赖点序，若点顺序错乱会直接翻转电荷符号
- 分类阶段需做特征缩放，避免量纲差异导致距离度量失真

## R12

真实成本（超越大 O）：
- 单条轨迹几十个点时，拟合通常毫秒级完成
- 训练 KNN 的成本主要是样本存储，训练阶段近似“零参数拟合”
- 预测时距离计算是主耗时，但二维特征下常数非常小
- 因此本 MVP 可在普通 CPU 上快速批量运行，无需 GPU

## R13

误差来源与可解释界限：
- 统计误差：点坐标噪声、气泡计数噪声
- 系统误差：磁场不均匀、投影几何简化、未建模能量损失
- 算法误差：短弧病态拟合、KNN 的局部样本偏差

本实现更适合“机制演示与量级重建”，不宣称达到现代实验级精度。

## R14

鲁棒性与失效模式：
- 若轨迹点数过少（如 `< 6`），圆拟合可能不稳定
- 若轨迹近直线，电荷符号与半径都会变得不可靠
- 若气泡密度超出训练分布，KNN 会产生外推偏差
- 若存在多轨交叉但未正确分割，单轨模型会系统失真

脚本通过固定随机种子和残差输出，便于复现实验与定位问题。

## R15

工程实现要点：
- 用 `dataclass` 固定粒子参数，减少硬编码与参数错位
- 采用“代数初值 + 非线性修正”两阶段拟合，兼顾速度与精度
- 统一以米、特斯拉、GeV/c 计算，避免单位混乱
- 输出同时保留真值与估计值，直接观测误差传播链路

## R16

前驱/后继关系：
- 前驱：洛伦兹力学、最小二乘几何拟合、Bethe-Bloch 电离损失模型
- 同层：线圈磁谱仪轨迹重建、火花室/漂移室的曲率拟合
- 后继：卡尔曼滤波全局轨迹重建、三维螺旋拟合、端到端图神经网络重建
- 应用：历史数据再分析、教学实验、重建流程原型验证

## R18

源码级算法流（非黑箱，9 步）：
1. 定义粒子库（质量与电荷）并设定磁场 `B`、噪声水平、随机种子。  
2. 合成轨迹：按 `r = p/(0.3|q|B)` 生成圆弧点列，再叠加高斯测量噪声。  
3. 由 `beta = p/sqrt(p^2+m^2)` 计算理论气泡密度，并加入观测噪声得到 `bubble_density_obs`。  
4. 对每条轨迹做代数圆拟合：解线性最小二乘获得 `(xc, yc, r)` 初值。  
5. 调用 `scipy.optimize.least_squares` 最小化几何残差 `sqrt((x-xc)^2+(y-yc)^2)-r`，输出精化半径和 `fit_rms`。  
6. 对拟合中心角度序列做 `unwrap`，用平均角增量符号判定旋向，再结合 `B` 方向推得 `q_est`。  
7. 用 `pT_est = 0.3 * |q_est| * B * r_est` 反演动量，并和真值对比给出相对误差。  
8. 用大量合成样本训练 KNN：其内部流程是“标准化特征 -> 计算到全部训练点距离 -> 取前 k 个近邻 -> 按距离加权投票输出类别”。  
9. 用 `(pT_est, bubble_density_obs)` 做轨迹级粒子分类，汇总成 pandas 表格输出全链路重建结果。  

第 8 步说明：虽然代码调用了 `KNeighborsClassifier`，但决策机制可完全拆解为“距离排序 + 投票”，并非不可解释黑盒。
