# 数值微分 - 有限差分

- UID: `MATH-0141`
- 学科: `数学`
- 分类: `数值分析`
- 源序号: `141`
- 目标目录: `Algorithms/数学-数值分析-0141-数值微分_-_有限差分`

## R01

本条目实现“数值微分 - 有限差分”的最小可运行 MVP，覆盖一阶导数的三种经典差分格式：
- 前向差分（Forward Difference）
- 后向差分（Backward Difference）
- 中心差分（Central Difference）

目标是把“离散公式、误差阶、代码执行流”完整打通，并通过可复现实验展示收敛行为。

## R02

问题定义：
- 输入：一维均匀网格采样点 `x_i` 与函数值 `f_i = f(x_i)`
- 输出：对 `f'(x)` 的离散近似
- 约束：本 MVP 假设 `x` 等间距（步长 `h` 常数）

测试函数取 `f(x)=sin(x)`，解析导数为 `f'(x)=cos(x)`，便于直接计算误差。

## R03

一阶导数有限差分公式（均匀网格）：
- 前向差分：`f'(x_i) ≈ (f_{i+1} - f_i) / h`，截断误差 `O(h)`
- 后向差分：`f'(x_i) ≈ (f_i - f_{i-1}) / h`，截断误差 `O(h)`
- 中心差分：`f'(x_i) ≈ (f_{i+1} - f_{i-1}) / (2h)`，截断误差 `O(h^2)`

因此在足够光滑函数上，中心差分通常比前/后向差分更精确。

## R04

算法流程：
1. 校验输入 `x, y`：同长度、有限值、严格递增、且为均匀网格。
2. 计算并返回步长 `h`。
3. 用向量化切片计算前向差分和后向差分。
4. 用向量化切片计算中心差分（仅内部点）。
5. 与解析导数在对应网格位置比较，得到最大绝对误差。
6. 在多组分辨率上重复，观察误差随 `h` 缩小的变化。
7. 用相邻两组误差比估算经验收敛阶。

## R05

核心数据结构：
- `x: np.ndarray`，网格点
- `y: np.ndarray`，函数采样值
- `errors: dict[str, float]`，三种格式的最大绝对误差
- `results: list[tuple[h, n, err_fwd, err_bwd, err_ctr]]`，多分辨率汇总表

该结构足够轻量，便于调试和扩展。

## R06

正确性要点：
- 代码直接按三种离散公式实现，不依赖黑盒微分器。
- 误差计算严格对齐“公式定义所在网格位置”：
  - 前向差分对应 `x[:-1]`
  - 后向差分对应 `x[1:]`
  - 中心差分对应 `x[1:-1]`
- 通过网格加密（`h` 逐步减半）观察经验阶：
  - 前/后向约为 1 阶
  - 中心差分约为 2 阶

## R07

复杂度分析（单次分辨率）：
- 时间复杂度：`O(N)`（向量化差分与误差计算都线性）
- 空间复杂度：`O(N)`（保存网格、函数值、导数近似）

多分辨率实验复杂度为各次 `O(N)` 之和。

## R08

边界与异常处理：
- `x/y` 不是一维数组：抛 `ValueError`
- `len(x) != len(y)`：抛 `ValueError`
- 点数小于 3：抛 `ValueError`（中心差分需要至少 3 点）
- `x/y` 含 `nan` 或 `inf`：抛 `ValueError`
- `x` 非严格递增：抛 `ValueError`
- 网格非均匀（超出容差）：抛 `ValueError`
- 步长非正：抛 `ValueError`

## R09

MVP 取舍：
- 使用 `numpy` 进行数组运算，避免手写循环噪音；
- 聚焦一阶导数，先把最核心有限差分思想讲清；
- 不引入高阶模板系数自动生成、非均匀网格一般公式、噪声正则化等进阶内容。

## R10

`demo.py` 函数职责：
- `validate_uniform_grid`：校验网格并返回 `h`
- `forward_difference`：前向差分
- `backward_difference`：后向差分
- `central_difference`：中心差分
- `max_abs_error`：最大绝对误差
- `run_single_resolution`：单个分辨率实验
- `estimate_orders`：相邻分辨率经验阶估计
- `print_sample_rows`：打印样例行
- `main`：组织全流程实验与输出

脚本无需交互输入，会自动打印收敛表、经验阶和样例明细。

## R12

输出解读：
- `n_intervals`：区间划分数（越大表示网格越细）
- `h`：网格步长
- `err_forward / err_backward / err_central`：三种格式的最大绝对误差
- `estimated order p`：由误差比估算出的经验收敛阶

理论预期：`central` 的阶数显著高于 `forward/backward`。

## R13

建议最小测试集：
- 收敛性：`n_intervals = 20, 40, 80, 160`
- 非法输入：
  - `x` 非递增
  - 非均匀网格
  - `len(x) != len(y)`
  - `x` 或 `y` 含 `nan`
- 边界样例：仅 3 个点（允许中心差分但分辨率极粗）

## R14

关键可调参数：
- `n_intervals_list`：分辨率序列
- 区间端点 `[x_start, x_end]`
- 样例输出行数 `rows`

工程上可先用粗网格做功能检查，再用细网格观察收敛阶。

## R15

方法对比：
- 前向/后向差分：
  - 优点：公式简单、边界点易处理
  - 缺点：一阶误差，精度较低
- 中心差分：
  - 优点：二阶精度，常见场景更优
  - 缺点：需要两侧点，边界处需搭配单边公式

## R16

应用场景：
- 仅有离散采样数据，需要估计变化率
- PDE 离散化中的空间导数近似
- 物理实验/传感器数据的速度、梯度估计基线

后续扩展方向：

## R18

源码级算法流（对应 `demo.py`，8 步）：
1. `main` 定义分辨率列表 `n_intervals_list`，依次执行实验。  
2. `run_single_resolution` 在 `[0, 2π]` 生成均匀网格 `x`，并计算 `y=sin(x)` 与真导数 `cos(x)`。  
3. 调用 `validate_uniform_grid(x, y)` 校验输入并得到步长 `h`。  
4. `forward_difference` 用切片 `y[1:] - y[:-1]` 计算前向差分。  
5. `backward_difference` 用切片 `y[1:] - y[:-1]` 计算后向差分（位置对齐到 `x[1:]`）。  
6. `central_difference` 用切片 `y[2:] - y[:-2]` 计算中心差分。  
7. 对三种近似分别与 `cos(x)` 在对应位置求 `max_abs_error`，形成误差表。  
8. `main` 汇总所有分辨率误差，并用 `estimate_orders` 输出经验阶，最后打印样例行。
