## R01

法拉第效应描述线偏振光沿磁场方向传播时，偏振面发生旋转的磁光现象。常用一阶关系式：

`theta = V(lambda) * B * L + b`

- `theta`：旋转角（rad 或 deg）
- `V(lambda)`：Verdet 常数（与波长相关）
- `B`：沿传播方向的磁感应强度（T）
- `L`：介质有效光程（m）
- `b`：仪器零点偏移或系统偏置

本条目关注“如何从多组带噪观测反推出可解释的法拉第模型参数”。

## R02

MVP 目标：

1. 生成可复现的合成实验数据 `(lambda, B, L, theta)`；
2. 拟合简化色散模型 `V(lambda)=K/lambda^2`；
3. 估计 `K` 和仪器偏移 `b`，并输出拟合误差；
4. 演示已知旋转角时对未知磁场强度 `B` 的反演流程。

强调小而透明，不依赖黑盒非线性优化器。

## R03

采用的一阶观测模型：

`theta_i = (K / lambda_i^2) * B_i * L_i + b + epsilon_i`

其中：

- `K`：待估参数（与材料和单位体系相关）
- `epsilon_i`：测量噪声（近似高斯）

将特征记为 `x_i = B_i * L_i / lambda_i^2`，则模型线性化为：

`theta_i = K * x_i + b + epsilon_i`

因此可直接用线性最小二乘求解。

## R04

物理解释与现象特征：

1. `B` 反号会导致旋转角反向（在偏置项不变时）；
2. `L` 增大时旋转角幅值线性增大；
3. 波长越短，`|V(lambda)|` 往往越大（在该简化模型下随 `1/lambda^2` 增大）；
4. 法拉第效应是非互易效应，广泛用于光隔离器与磁场测量。

## R05

数值算法设计（显式可追踪）：

- 构造特征 `x=B*L/lambda^2`；
- 组装设计矩阵 `X=[x, 1]`；
- 用 `np.linalg.lstsq` 求解 `theta ≈ K*x + b`；
- 计算 `RMSE` 与 `R^2` 评估拟合质量；
- 将估计参数回代到反演公式，得到未知磁场估计值。

## R06

`demo.py` 会输出：

- 合成数据前几行（便于检查量纲与分布）；

## R10

正确性直觉：

1. 观测方程在 `K,b` 上线性，可全局最小二乘求解；
2. `B,L,lambda` 的可变性提供参数可辨识性；
3. 拟合后若 `RMSE` 小且 `R^2` 高，说明模型解释力足够；
4. 反演 `B` 的误差可直接验证参数估计是否有工程可用性。

## R11

数值稳定性处理：

- 避免 `lambda` 接近 0（本例使用固定可见光离散波长）；
- 使用 `float64` 与 `np.linalg.lstsq` 避免手写矩阵求逆；
- `R^2` 在总方差近零时做保护；
- 固定随机种子使结果可复现；
- 断言阈值设置为“可解释且可通过”的合理范围。

## R12

关键参数（见 `FaradayConfig`）：

- `n_samples`：样本数量；
- `noise_std_rad`：旋转角噪声标准差；
- `true_k`、`true_offset_rad`：生成数据真值；

- `b_min_t/b_max_t`、`l_min_m/l_max_m`：实验采样范围；
- `unknown_*`：未知样品反演配置。

调参建议：

- 噪声高时增加 `n_samples`；
- 若参数不稳定，增大 `B` 与 `L` 的变化范围；
- 扩展到真实数据时先检查单位是否一致。

## R14

常见失败模式：

1. 单位混乱（nm/m、rad/deg 混用）导致参数数量级异常；
2. 忽略偏置项 `b`，反演磁场出现系统偏差；
3. `B` 或 `L` 变化过小，导致 `K` 可辨识性不足；
4. 数据中存在明显异常点却仍用普通最小二乘；
5. 把第三方拟合器当黑盒，无法定位误差来源。

## R18

`demo.py` 的源码级流程（8 步）：

1. `generate_synthetic_dataset` 构造 `(lambda_nm, B_tesla, L_m, theta_obs_rad)` 含噪样本。  
2. `verdet_constant` 用 `K/lambda^2` 计算每个波长下的 Verdet 常数。  
3. `forward_rotation_rad` 把 `V(lambda), B, L, b` 组装成旋转角前向模型。  
4. `fit_faraday_model` 调用 `build_feature` 生成线性特征 `x=B*L/lambda^2`。  
5. `fit_faraday_model` 中的 `solve_linear_least_squares` 显式求解 `theta ≈ K*x+b`。  
6. 拟合后计算 `RMSE` 与 `R^2`，评估模型解释力。  
7. `infer_magnetic_field` 用估计参数将未知样品旋转角反演为 `B_hat`。  
8. `main` 打印核心结果并执行断言（`K` 相对误差、`R^2`、`B` 反演误差）。  

第三方库边界：`numpy/pandas` 仅用于数组线性代数和表格展示；参数估计流程、特征构造和反演公式均在源码中明确实现，不依赖专用黑盒磁光拟合包。
