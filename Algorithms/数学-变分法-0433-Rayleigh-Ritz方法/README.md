## R01

Rayleigh-Ritz 方法是变分法里最经典的“有限维逼近”思路：
把无限维函数优化/特征值问题，投影到一个有限维试探空间里，再把连续问题变成线性代数问题求解。

本目录 MVP 选择的模型是 Sturm-Liouville 最小特征值问题：

- 方程：`-u'' = λu`，`x ∈ (0,1)`
- 边界：`u(0)=u(1)=0`
- 目标：近似最小特征值 `λ1` 与对应模态

## R02

该问题的 Rayleigh 商为：

`R(u) = (∫_0^1 |u'(x)|^2 dx) / (∫_0^1 |u(x)|^2 dx)`，其中 `u != 0` 且满足齐次 Dirichlet 边界。

最小特征值满足：

`λ1 = min_{u∈H0^1(0,1), u!=0} R(u)`。

精确解已知：`λ1 = π^2`，`u1(x) ∝ sin(πx)`。

## R03

Rayleigh-Ritz 的核心变分思想：

1. 选有限维子空间 `V_N ⊂ H0^1(0,1)`。
2. 只在 `V_N` 内做最小化：`λ1^(N) = min_{v∈V_N, v!=0} R(v)`。
3. 得到离散特征值 `λ1^(N)` 及离散特征函数 `u_N`。

当 `V_N` 随 `N` 增大逐步稠密时，`λ1^(N)` 会收敛到 `λ1`。

## R04

本实现使用试探基函数：

`φ_k(x)=x(1-x)x^k,  k=0,...,N-1`

其优势：

- 自动满足 `φ_k(0)=φ_k(1)=0`，天然满足边界条件；
- 多项式结构简单，导数可显式写出；
- 便于快速搭建最小可运行 MVP。

近似解写作：

`u_N(x)=Σ_{k=0}^{N-1} c_k φ_k(x)`。

## R05

代入弱形式后，得到广义特征值问题：

`Kc = λMc`

其中

- `K_ij = ∫_0^1 φ_i'(x)φ_j'(x) dx`（刚度矩阵）
- `M_ij = ∫_0^1 φ_i(x)φ_j(x) dx`（质量矩阵）

`K` 与 `M` 都是对称矩阵；在基函数线性无关时，`M` 正定。

## R06

矩阵装配采用高斯-勒让德求积（`numpy.polynomial.legendre.leggauss`）：

- 先在 `[-1,1]` 取节点/权重；
- 仿射映射到 `[0,1]`；
- 用加权内积计算 `K` 与 `M`。

特征值求解使用 `scipy.linalg.eigh(K, M)`，取最小特征值作为 `λ1^(N)`。

## R07

理论性质（本问题中可观察到）：

- `λ1^(N) >= λ1`（上界性质，因在受限子空间最小化）；
- 子空间扩充时最小离散特征值非增：`λ1^(N+1) <= λ1^(N)`；
- 若试探空间足够丰富，则 `λ1^(N) -> λ1`。

## R08

误差来源主要有两类：

- 截断误差：有限维空间无法完全表示真解；
- 积分误差：数值求积替代精确积分。

本实现把求积点设为 `400`，对当前低维多项式基已较充分，通常截断误差占主导。

## R09

设基函数数为 `N`，求积点数为 `Q`。

- 基函数与导数评估：`O(NQ)`
- 装配矩阵：`O(N^2Q)`
- 广义特征值分解：`O(N^3)`

在本 MVP 里，`N` 很小（2~10），计算非常快。

## R10

数值稳定性注意点：

- 多项式基在较大 `N` 下可能变得病态，`M` 条件数上升；
- 若要更大规模，建议改用正交基（如正交多项式或正弦基）；
- 需要保证 `M` 正定，否则广义特征值问题可能不稳定。

## R11

边界条件处理方式：

- 当前采用“内建边界”的基函数（每个基函数自动为零于边界）；
- 替代做法是惩罚法或拉格朗日乘子，但实现更复杂；
- 对教学型 MVP，内建边界最直接可靠。

## R12

`demo.py` 包含以下核心函数：

- `gauss_legendre_on_unit_interval`：生成 `[0,1]` 求积规则
- `basis_and_derivative`：评估 `φ_k` 与 `φ_k'`
- `assemble_matrices`：装配 `K, M`
- `rayleigh_ritz_smallest_eigenpair`：求最小特征对
- `l2_error_to_exact_mode`：计算与精确一阶模态的 `L2` 误差
- `run_experiment`：对多个 `N` 批量实验并汇总

## R13

脚本无交互输入，参数在代码中固定：

- 基函数规模：`[2, 3, 4, 6, 8, 10]`
- 求积点：`400`

输出为表格，列包括：

- `N`：基函数数量
- `lambda_1`：Ritz 近似最小特征值
- `rel_err(lambda)`：相对特征值误差（对比 `π^2`）
- `L2_err(mode)`：近似模态与精确模态的 `L2` 误差

结果解读要点：

## R18

`demo.py` 的源代码级算法流程可拆为 8 步：

1. 在 `main` 里设定一组基函数维度 `basis_sizes=[2,3,4,6,8,10]`。
2. `run_experiment` 对每个 `N` 调用 `rayleigh_ritz_smallest_eigenpair(N)`。
3. `rayleigh_ritz_smallest_eigenpair` 先调用 `assemble_matrices` 生成 `K,M`。
4. `assemble_matrices` 通过 `gauss_legendre_on_unit_interval(Q)` 得到 `[0,1]` 上求积点和权重。
5. `basis_and_derivative` 在求积点上计算全部 `φ_k`、`φ_k'`，并用加权内积装配 `K_ij=∫φ_i'φ_j'`、`M_ij=∫φ_iφ_j`。
6. 调用 `scipy.linalg.eigh(K, M)` 解对称广义特征值问题，取最小特征值/特征向量。
7. `l2_error_to_exact_mode` 将 Ritz 模态重构到稠密网格，做归一化与符号对齐，再与 `sqrt(2)sin(πx)`计算 `L2` 误差。
8. `print_results` 按表格输出 `N`、`lambda_1`、相对特征值误差和模态误差，形成可复现实验结论。
