# 动量守恒定律 (Law of Conservation of Momentum)

## R01

动量守恒定律表述为：在外力合冲量为零（或可忽略）的系统中，总动量保持不变。

连续时间形式：

- `dP/dt = F_ext`
- 若 `F_ext = 0`，则 `dP/dt = 0`，故 `P = 常量`

冲量形式（更适合碰撞问题）：

- `P_after - P_before = J_ext`
- 若 `J_ext = 0`，则 `P_after = P_before`

## R02

该定律是经典力学中处理碰撞、爆炸、反冲、喷射推进等问题的第一条守恒约束。即使能量在非弹性过程中损失，动量守恒通常仍成立（在封闭系统下），因此它经常是求解速度分配、校验数值模拟正确性的主方程。

## R03

本条目 MVP 聚焦一维两体碰撞：

- 输入：`m1, m2, u1, u2, e`，可选外部冲量 `J_ext`
- 输出：碰撞后速度 `v1, v2`，以及动量平衡残差
- 验证目标：
  - 闭系统：`m1*u1 + m2*u2 = m1*v1 + m2*v2`
  - 有外冲量：`m1*v1 + m2*v2 = m1*u1 + m2*u2 + J_ext`
  - 弹性碰撞（`e=1`）在闭系统下额外验证动能守恒

## R04

MVP 使用两条方程联立求解碰撞后速度。

1. 动量方程（闭系统）
   `m1*u1 + m2*u2 = m1*v1 + m2*v2`

2. 恢复系数方程
   `v2 - v1 = e * (u1 - u2)`, `e ∈ [0, 1]`

3. 闭式解
   - `v1 = ((m1 - e*m2) * u1 + (1+e) * m2 * u2) / (m1 + m2)`
   - `v2 = ((m2 - e*m1) * u2 + (1+e) * m1 * u1) / (m1 + m2)`

4. 外冲量场景
   - 采用“碰前速度修正”等价实现：`u1_eff = u1 + J_ext/m1`
   - 然后按闭系统碰撞求解
   - 结果满足 `P_after = P_before + J_ext`

## R05

设测试样本数为 `N`：

- 批量随机验证时间复杂度 `O(N)`
- 空间复杂度 `O(1)`（若仅保留统计量）

MVP 默认 400 组随机样本，开销非常小。

## R06

`demo.py` 包含三个层次：

## R10

正确性直觉：

1. 内力作用成对出现，系统内部交换动量但不改变总量；
2. 若外冲量为零，总动量前后必须相同；
3. 恢复系数只控制“相对速度回弹程度”，不替代动量方程；
4. 弹性碰撞是额外满足动能守恒的特例，非弹性则动能可下降。

## R14

常见错误模式：

1. 把恢复系数方程方向写反（`v1-v2` 与 `v2-v1` 混淆）；
2. 在有外冲量时仍错误套用 `P_after=P_before`；
3. 将负质量或 `e>1` 作为输入导致非物理结果；
4. 仅凭视觉看速度变化，不检查守恒残差；
5. 把动量守恒误当作动能守恒（非弹性碰撞会损失动能）。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 构造三个确定性样例（弹性闭系统、非弹性闭系统、含外冲量样例）。
2. `run_case` 先把外冲量等价为碰前速度修正：`u1_eff = u1 + J_ext/m1`。
3. `run_case` 调用 `solve_two_body_collision_1d`，在函数内检查 `m1,m2,e` 合法性。
4. `solve_two_body_collision_1d` 用动量方程与恢复系数方程的闭式解直接计算 `v1,v2`。
5. `run_case` 计算 `p_before`、`p_after`、`expected_p_after = p_before + J_ext`，得到动量残差。
6. `run_case` 同时计算碰前有效动能与碰后动能，供弹性/非弹性断言使用。
7. `main` 对每个样例执行断言：动量残差足够小，且弹性样例动能守恒、非弹性样例动能不增。
8. `main` 运行 `run_random_stress_test` 做 400 组随机一致性检查，输出最大残差并给出总体通过结论。

该流程中 `numpy` 只用于随机数与基础数值工具，碰撞求解与守恒校验逻辑均在源码中显式实现，没有将核心物理步骤委托给黑盒库。
