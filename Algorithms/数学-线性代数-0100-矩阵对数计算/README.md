# 矩阵对数计算

## R01

矩阵对数（matrix logarithm）是矩阵指数的逆问题：给定可逆矩阵 `A`，寻找矩阵 `X` 使

- `exp(X) = A`

其中 `exp` 是矩阵指数而非逐元素指数。与标量情形类似，矩阵对数通常不唯一；在满足谱条件时可定义“主值矩阵对数（principal log）”。

## R02

本条目 MVP 的定位：

- 目标问题：计算实对称正定矩阵（SPD）的主值矩阵对数。
- 选择该子问题的原因：
  - SPD 矩阵一定可正交对角化；
  - 全部特征值为正，主值对数实数且唯一；
  - 可用 `eigh + log + 回代` 给出透明实现。

这比“任意复矩阵的一般矩阵对数”更窄，但更稳健、可验证。

## R03

输入输出定义（对应 `demo.py`）：

- 输入：`A ∈ R^{n×n}`，要求
  - 方阵；
  - 元素有限；
  - `A = A^T`（数值近似）；
  - 最小特征值 `> 0`。
- 输出：
  - `logA`：矩阵主值对数；
  - `exp(logA)`：用于回代验证；
  - 误差指标（重构误差、对称性误差、谱一致性误差、交换子误差）。

## R04

时间复杂度（稠密矩阵）：

- 核心操作为对称特征分解 `eigh(A)`，复杂度 `O(n^3)`；
- 回代 `Q @ diag(log(λ)) @ Q^T` 也是 `O(n^3)` 量级；
- 因此总复杂度为 `O(n^3)`。

## R05

空间复杂度：

- 存储矩阵本体、特征向量矩阵及中间量，均为 `O(n^2)`；
- 附加向量（特征值、对角项）为 `O(n)`。

总空间复杂度 `O(n^2)`。

## R06

微型示例（对角矩阵）：

设

- `A = diag(1, e^2)`

则

- `log(A) = diag(log 1, log e^2) = diag(0, 2)`。

对 SPD 矩阵可先正交对角化 `A = Q diag(λ_i) Q^T`，再把 `log` 作用到每个 `λ_i`，最后回代。

## R07

算法意义：

- 在线性系统与控制中，可通过 `A = exp(FΔt)` 反求连续时间生成元 `F = log(A)/Δt`；
- 在信息几何与 SPD 流形计算中，`log` 把曲面点映射到切空间；
- 在数值分析中，矩阵函数计算（log/sqrt/exp）是许多高级算法组件。

## R08

核心数学关系（SPD 场景）：

1. `A` 为 SPD，则存在正交矩阵 `Q` 与正特征值 `λ_i>0`，使 `A = Q diag(λ_i) Q^T`。
2. 定义
   - `log(A) = Q diag(log λ_i) Q^T`。
3. 由矩阵函数谱映射性质：
   - `exp(log(A)) = Q diag(exp(log λ_i)) Q^T = A`。
4. 该 `log(A)` 为实对称矩阵，且是主值解。

## R09

适用条件与局限：

适用：

- 实对称正定矩阵；
- 需要稳定、可解释、可验证的最小实现。

局限：

- 不直接覆盖非对称矩阵、含负实轴特征值的情形；
- 不处理 Jordan 块缺陷矩阵的广义情况；
- 对接近奇异（最小特征值极小）的 SPD 矩阵，`log(λ_min)` 可能带来数值放大。

## R10

正确性要点（工程可验证版）：

1. `validate_spd_matrix` 保证输入满足 SPD 前提。  
2. `eigh` 给出 `A = Q diag(λ) Q^T`。  
3. 对角上逐元素取 `log` 得到 `diag(log λ)`。  
4. 回代得到 `logA = Q diag(log λ) Q^T`。  
5. 再以同样谱法计算 `exp(logA)` 并比较 `A`，若相对误差足够小，则实现与理论一致。

## R11

数值稳定性讨论：

- `eigh`（对称特征分解）通常比一般 `eig` 更稳；
- 当 `λ_min` 很接近 0 时，`log(λ_min)` 的幅值很大，会增大舍入误差敏感性；
- 实现中设置 `eig_floor` 做合法性判断，拒绝非正定或近退化输入；
- 最终采用多指标校验，而非只看单一残差。

## R12

性能视角：

- 主耗时来自两次特征分解（一次对 `A`，一次对 `logA` 做指数回代）；
- 全部操作向量化，避免 Python 层双重循环；
- 对大规模稀疏问题，应改为专门稀疏矩阵函数算法，本 MVP 不覆盖该优化。

## R13

本目录 MVP 的可验证保证：

- 用固定随机种子生成可复现 SPD 测试矩阵；
- 打印并断言以下指标：
  - `||exp(logA)-A||_F / ||A||_F`；
  - `||logA-logA^T||_F / ||logA||_F`；
  - `eig(logA)` 与 `log(eig(A))` 的最大差；
  - `||A logA - logA A||_F`。
- 任一核心指标超阈值即抛出异常，保证失败可见。

## R14

常见失效模式：

- 输入不是方阵或含 `NaN/Inf`；
- 输入不是对称矩阵（数值误差或数据错误）；
- 存在非正特征值，主值实矩阵对数不成立；
- 把逐元素 `np.log(A)` 误当矩阵对数（这是不同运算）。

## R15

实现设计（`demo.py`）：

- `MatrixLogReport`：集中保存结果矩阵与诊断指标；
- `validate_spd_matrix`：输入合法性与 SPD 检查；
- `matrix_log_spd`：特征分解路线实现矩阵对数；
- `matrix_exp_symmetric`：用于回代验证 `exp(logA)`；
- `build_spd_matrix`：生成可复现测试矩阵；
- `run_checks`：统一阈值断言；
- `main`：串联执行并打印报告。

## R16

相关算法链路：

- 一般矩阵对数：Schur 分解 + inverse scaling and squaring + 有理逼近；
- 矩阵平方根：Denman-Beavers、Schur 法；
- 矩阵指数：Padé 逼近 + scaling and squaring；
- 李群李代数计算中：`log` 与 `exp` 互为关键映射。

本 MVP 只实现其中最透明、最稳定的一条分支（SPD 谱分解法）。

## R18

`demo.py` 的源码级算法流程（9 步，非黑盒）：

1. `main` 固定随机种子，调用 `build_spd_matrix` 生成 `A = Q diag(λ) Q^T`（`λ>0`）。  
2. `validate_spd_matrix` 检查 `A` 的方阵性、有限值、对称性与最小特征值阈值。  
3. `matrix_log_spd` 调用 `numpy.linalg.eigh(A)` 得到 `Q, λ`（NumPy 底层转调 LAPACK 对称特征求解器）。  
4. 在特征值层面逐元素计算 `log(λ)`，形成对角矩阵 `diag(log λ)`。  
5. 回代 `logA = Q diag(log λ) Q^T`，并做一次 `(logA + logA^T)/2` 抑制舍入非对称。  
6. `matrix_exp_symmetric` 再次对 `logA` 做 `eigh`，逐元素 `exp` 后回代，得到 `exp(logA)`。  
7. 计算重构相对误差、对称性误差、谱一致性误差与交换子误差。  
8. `run_checks` 对上述指标执行阈值断言，失败则抛出异常并退出。  
9. 断言通过后打印主要数值结果，形成“构造 -> 计算 -> 回代验证”的完整闭环。

补充：这里第三方线代库只提供“特征分解原语”，矩阵对数的定义、回代、校验流程均由源码显式展开。
