## R01

线性判别分析（Linear Discriminant Analysis, LDA）是一类监督学习方法，核心目标是：

- 在降维时，寻找一个线性投影，使“类间距离尽量大、类内离散尽量小”；
- 在分类时，假设各类服从高斯分布且共享协方差矩阵，得到线性判别函数进行分类。

从线性代数角度看，LDA 的关键是把分类问题转化为广义特征值问题。

## R02

本目录 MVP 解决的问题：

- 输入：多分类数据集 `X, y`（脚本内自动构造，无交互输入）；
- 输出：
  - 训练得到的投影矩阵 `W`（维度 `d x m`）；
  - 投影后的数据表示；
  - 基于共享协方差的线性判别分类器；
  - 训练/测试准确率与 Fisher 比率（类间/类内分离度）。

脚本会同时验证“降维分离效果”与“分类性能”。

## R03

符号约定：

- `X ∈ R^{n×d}`：`n` 个样本，`d` 维特征；
- `y ∈ {0,1,...,K-1}`：`K` 个类别标签；
- `μ_k`：第 `k` 类均值，`μ`：全局均值；
- `S_w`：类内散度矩阵（within-class scatter）；
- `S_b`：类间散度矩阵（between-class scatter）；
- `W ∈ R^{d×m}`：投影矩阵，`m <= K-1`。

定义：

- `S_w = Σ_k Σ_{x_i in C_k} (x_i-μ_k)(x_i-μ_k)^T`
- `S_b = Σ_k n_k (μ_k-μ)(μ_k-μ)^T`

## R04

LDA 降维目标可写为 Fisher 准则：

`J(W) = tr((W^T S_w W)^{-1}(W^T S_b W))`

可转化为广义特征值问题：

`S_b v = λ S_w v`

在实现中常用等价形式：

`M = pinv(S_w + λ_reg I) @ S_b`

再对 `M` 做特征分解，取最大特征值对应的前 `m` 个特征向量组成 `W`。

## R05

本实现采用“手写 NumPy LDA”流程：

1. 计算每类均值和全局均值；
2. 构造 `S_w` 与 `S_b`；
3. 对 `S_w` 做小幅对角正则，避免奇异；
4. 求解 `pinv(S_w)S_b` 的特征值/特征向量；
5. 选取前 `m` 个特征向量作为投影方向；
6. 在原空间与投影空间分别训练共享协方差线性判别分类器；
7. 输出准确率与 Fisher 比率，并执行断言检查。

## R06

二维直觉示例：

- 若两类数据在 `x1` 方向重叠明显、在某个斜方向可分，LDA 会学习该斜方向向量 `w`；
- 投影后 `z = Xw`，类均值距离变大、类内方差变小；
- 对二分类，最多只需 `1` 个判别方向（`K-1=1`）。

这体现了 LDA 与 PCA 的区别：LDA 用标签信息决定“最有判别力”的方向。

## R07

算法价值：

- 可解释：投影向量可直接查看各特征权重；
- 低维可视化：多分类数据常可投影到 `2D` 观察可分性；
- 训练代价低：核心是统计量和矩阵特征分解；
- 工程常用：作为分类基线、特征压缩前置步骤、模式识别经典方法。

## R08

复杂度（`n` 样本，`d` 维，`K` 类）：

- 计算均值与散度矩阵：约 `O(nd^2)`；
- 求解 `d×d` 特征分解：约 `O(d^3)`；
- 预测阶段（`t` 个样本）：约 `O(tKd^2)`（含判别分数计算）。

当 `d` 很大时，`d^3` 的特征分解是主要成本。

## R09

数值稳定性要点：

- `S_w` 在高维小样本下可能奇异；本实现加入 `reg * I` 正则；
- 使用 `pinv` 比直接 `inv` 更稳健；
- 特征分解结果可能出现微小虚部，代码取 `real` 并排序；
- 判别函数采用对数先验 `log(pi_k)`，避免概率下溢。

## R14

预期现象：

- `n_components <= K-1`，多分类下最多 `K-1` 个有效判别轴；
- 投影空间通常保留主要类别分离信息，测试准确率接近原空间；
- 投影后 Fisher 比率通常高于原空间，说明“每个维度的判别密度”更高；
- 若类间均值差异减弱或噪声增大，准确率会下降。

## R15

常见实现错误：

- 把 `S_w` 与 `S_b` 的定义写反；
- 忽略 `n_components <= K-1` 约束；
- 在病态矩阵上直接 `inv(S_w)` 导致数值爆炸；
- 特征值未按降序选取，导致判别方向错误；
- 只跑训练集，不做测试集验证，无法判断泛化。

## R16

可扩展方向：

- Shrinkage LDA（协方差收缩）提升高维稳定性；
- Kernel LDA 处理非线性可分问题；
- 与 PCA 级联（先 PCA 降噪，再 LDA 判别）；
- 对接真实数据集（如 Iris/Wine/手写数字子集）做可视化分析。

## R17

与相关方法对比：

- PCA：无监督，只最大化方差，不保证分类可分性；
- QDA：各类协方差不同，边界是二次曲线，参数更多；
- Logistic Regression：直接判别建模，不显式做类内/类间散度分解；
- LDA：在“线性边界 + 共享协方差 + 可解释投影”上具有平衡优势。

## R18

`demo.py` 的源码级 LDA 算法流（8 步，非黑盒）如下：

1. `main` 读取 `LDAConfig`，调用 `generate_synthetic_dataset` 生成多类高斯数据并做分层切分。  
2. `fit_lda_projection` 在训练集上计算每类均值与全局均值，构造 `S_w`、`S_b`。  
3. 对 `S_w` 加 `reg*I` 后构造 `M = pinv(S_w_reg) @ S_b`，再用 `np.linalg.eig` 求特征分解。  
4. 按特征值降序取前 `m` 个特征向量组成投影矩阵 `W`，得到 `X_train_proj`、`X_test_proj`。  
5. `fit_lda_classifier` 在原空间估计 `μ_k`、共享协方差 `Σ`、先验 `π_k`，并缓存 `Σ^{-1}`。  
6. `predict_lda_classifier` 用线性判别函数 `δ_k(x)=x^TΣ^{-1}μ_k-0.5μ_k^TΣ^{-1}μ_k+logπ_k` 进行分类。  
7. 在投影空间重复训练与预测，计算原空间/投影空间的训练与测试准确率。  
8. `fisher_ratio` 与 `run_checks` 统一验证分离度和精度阈值，全部通过后打印 `All LDA checks passed.`。
