# 唯一性定理 (Uniqueness Theorem)

- UID: `PHYS-0153`
- 学科: `物理`
- 分类: `静电学`
- 源序号: `154`
- 目标目录: `Algorithms/物理-静电学-0154-唯一性定理_(Uniqueness_Theorem)`

## R01

唯一性定理说明：在给定区域内，若电势满足拉普拉斯方程（或泊松方程）并且边界条件完全给定，则该边值问题的解是唯一的。  
本条目用二维方形区域上的数值离散例子做最小可运行验证：相同的狄利克雷边界、不同初值，最终收敛到同一个电势场。

## R02

数学表述（狄利克雷边界）：

- 设区域 \(\Omega\) 有边界 \(\partial\Omega\)。
- 若 \(V_1, V_2\) 均满足 \(\nabla^2 V = 0\)（或 \(\nabla^2 V = -\rho/\epsilon_0\)）且在 \(\partial\Omega\) 上取同一边界值 \(f\)。
- 则 \(V_1 \equiv V_2\) 于整个 \(\Omega\) 内。

证明常用思路：令 \(u = V_1 - V_2\)，则 \(\nabla^2 u = 0\) 且 \(u|_{\partial\Omega}=0\)。由最大值原理或能量积分可得 \(u\equiv0\)。

## R03

本 MVP 的具体问题定义：

- PDE：\(\nabla^2 V = 0\)（无自由电荷静电区）。
- 区域：\([0,1]\times[0,1]\)。
- 边界：
  - \(V(x,0)=0\)
  - \(V(0,y)=0\)
  - \(V(1,y)=0\)
  - \(V(x,1)=\sin(\pi x)\)
- 数值方法：Jacobi 迭代有限差分。

## R04

离散化公式（五点模板）：

- 网格步长 \(h=1/(N-1)\)。
- 对内点 \((i,j)\) 使用
  \[
  V_{i,j}^{(k+1)} = \frac{1}{4}\left(V_{i-1,j}^{(k)} + V_{i+1,j}^{(k)} + V_{i,j-1}^{(k)} + V_{i,j+1}^{(k)}\right)
  \]
- 每轮更新后重新钉住边界值（Dirichlet 边界不参与自由演化）。

## R05

为什么这个数值实验能对应“唯一性”：

- Jacobi 对该离散椭圆问题收敛到唯一离散解。
- 若从不同初值（全零、随机）迭代，收敛结果应一致。
- 因而可用 \(\max|V_A - V_B|\) 验证“同边界下解唯一”。

## R06

伪代码：

```text
输入 N, tol, max_iter
构造边界矩阵 B
for init_mode in {zeros, random}:
    设 V 为初值
    固定 V 的四条边界为 B
    repeat:
        V_new(内点) = 邻域四点平均
        固定 V_new 的四条边界为 B
        delta = max_abs(V_new - V)
        V = V_new
    until delta < tol or 达到 max_iter
输出两组解 V_A, V_B 与 max_abs(V_A - V_B)
```

## R07

正确性要点：

- 离散 Laplace 算子对应线性方程组 \(A\mathbf{v}=\mathbf{b}\)。
- 在标准矩形网格+Dirichlet 条件下，矩阵 \(A\) 正定，离散解唯一。
- Jacobi 收敛后得到该唯一离散解；初值只影响收敛路径，不影响极限点。
- 因此两个初值对应的收敛解差值应接近机器误差级别。

## R08

复杂度（单次求解）：

- 时间：\(O(KN^2)\)，其中 \(N^2\) 为网格点数，\(K\) 为迭代轮数。
- 空间：\(O(N^2)\)（当前解与下一轮解）。

示例中运行两次（不同初值），总时间约翻倍。

## R09

数值稳定与误差来源：

- Jacobi 是稳定但较慢的平滑迭代。
- 收敛阈值 `tol` 越小，迭代轮数越大，解越接近离散极限。
- 总误差由两部分组成：
  - 迭代截断误差（未完全收敛）
  - 空间离散误差（有限网格近似连续 PDE）

## R10

边界与异常处理：

- `n < 5` 直接报错，避免过粗网格导致验证无意义。
- 初值模式仅允许 `zeros` 或 `random`。
- 每步都强制覆盖边界，避免边界漂移污染结论。

## R11

解析解对照（用于 sanity check）：

本边值问题解析解为
\[
V(x,y) = \sin(\pi x)\frac{\sinh(\pi y)}{\sinh(\pi)}
\]
`demo.py` 计算该解析场，并输出数值解与解析解的最大绝对误差，确保结果物理上可信。

## R12

参数建议：

- `n=81`：在速度与精度间折中。
- `tol=1e-12`：进一步压低迭代停止误差，使两次求解差值更接近机器极限。
- `max_iter=40000`：确保随机初值场景也能达到上述容差。

## R17

局限与可扩展方向：

- Jacobi 迭代收敛慢，可替换为 Gauss-Seidel / SOR / Multigrid。
- 当前为 2D 规则网格，可扩展到不规则区域（有限元法）。
- 可进一步增加 Neumann/混合边界条件下的唯一性讨论（需额外约束，如总电荷或参考电势）。

## R18

`demo.py` 的源代码级算法流程（8 步）：

1. 读取网格规模 `n`，创建边界矩阵 `boundary`，只在顶边赋值 `sin(pi x)`，其余边界为 0。  
2. 计算同边值问题的解析解 `exact`，用于后续误差对照。  
3. 第一次调用 `solve_laplace_jacobi(..., init_mode="zeros")`：内部从全零场起步，并钉住边界。  
4. 在每次迭代中，对所有内点执行四邻点平均更新，形成 `vn`。  
5. 用 `max(abs(vn - v))` 作为收敛指标 `delta`，小于 `tol` 即停止并返回。  
6. 第二次调用同一求解器但 `init_mode="random"`，仅改变初值，不改变 PDE 与边界。  
7. 主程序计算两组收敛解的最大差值 `uniqueness_gap = max|V_A - V_B|`，这就是唯一性的数值证据。  
8. 同时计算两组解对解析解的最大误差并输出 `PASS/CHECK`，保证不仅“相同”，且“物理上正确”。
