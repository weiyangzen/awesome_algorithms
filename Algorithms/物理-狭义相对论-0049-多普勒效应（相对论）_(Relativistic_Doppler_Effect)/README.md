# 多普勒效应（相对论） (Relativistic Doppler Effect)

- UID: `PHYS-0049`

- 分类: `狭义相对论`

## R01

相对论多普勒效应描述的是：当光源与观测者沿视线方向有相对运动时，观测到的频率/波长会发生系统性偏移，且必须使用狭义相对论而非经典近似。

对光信号而言，纵向（line-of-sight）情形最常用，正速度（`beta>0`）定义为“光源远离观测者”，对应红移；负速度（`beta<0`）对应蓝移。

## R02

本条目 MVP 的最小目标：

- 实现纵向相对论多普勒频率比与波长比；
- 计算红移参数 `z`（可为负，代表蓝移）；
- 根据观测 `z` 反推速度比 `beta=v/c`；
- 与经典近似 `z≈beta` 做对比，展示高速下偏差；
- 用非交互断言验证可逆性和物理一致性。

## R03

使用定义与公式如下（光源远离观测者取 `beta>0`）：

- `beta = v/c`, 约束 `|beta| < 1`
- 频率比（观测/发射）：
  `D_f(beta) = sqrt((1-beta)/(1+beta))`
- 波长比（观测/发射）：
  `D_lambda(beta) = sqrt((1+beta)/(1-beta)) = 1/D_f`
- 红移：
  `z = D_lambda - 1`
- 由红移反推速度：
  `beta(z) = ((1+z)^2 - 1) / ((1+z)^2 + 1)`, 定义域 `z > -1`
- 低速经典近似（`|beta|<<1`）：
  `z_classical ≈ beta`

## R04

`demo.py` 的计算流程：

1. 定义常数（`c`）与样例速度数组 `betas`；
2. 校验 `betas` 为有限实数且满足 `|beta|<1`；
3. 计算相对论频率比、波长比、红移 `z`；
4. 用解析逆公式由 `z` 反推 `beta`；
5. 计算经典近似红移并与相对论结果比较误差；
6. 以给定谱线（如 H-alpha）计算观测波长；
7. 组织为 `pandas.DataFrame` 并打印；
8. 执行断言验证物理约束与可逆性。

## R10

正确性直觉：

1. 远离（`beta>0`）应导致频率降低、波长增大，即 `D_f<1`、`D_lambda>1`、`z>0`；
2. 接近（`beta<0`）应导致蓝移，即 `z<0`；
3. `D_f * D_lambda = 1` 是频率与波长互逆关系；
4. 解析逆 `beta(z)` 应恢复原始 `beta`；
5. 在小速度下，相对论结果应接近 `z≈beta`。

## R11

数值与实现策略：

- 全部使用 `float64`；
- 对 `beta` 做 `|beta|<1` 显式约束，防止分母接近零或出现非法值；
- 对 `z` 做 `z>-1` 检查，保证逆变换定义有效；
- 用 `np.allclose` 做可逆性验证，容差覆盖浮点误差；
- 用排序后的 `beta` 检查 `z` 单调性，避免符号约定写错。

## R12

关键参数：

- `betas`：采样速度点，建议覆盖负值、零、正值；
- `emitted_wavelength_nm`：用于把比例量映射到可读观测波长；
- `C_LIGHT`：光速常数（m/s）。

调参建议：

- 增加 `|beta|` 到 `0.8~0.95` 可明显看到经典近似偏差；
- 增加低速点（如 `0.01`）可验证经典极限；
- 更换谱线（如 500nm、656.28nm）不改变比例规律，只影响绝对波长数值。

## R13

- 近似比保证：N/A（非优化问题）。
- 随机成功率保证：N/A（确定性计算）。

可验证保证：

- 可逆性：`beta_from_redshift(relativistic_redshift(beta)) ≈ beta`；
- 互逆性：`doppler_frequency_ratio * doppler_wavelength_ratio ≈ 1`；
- 符号一致：`beta>0 => z>0`，`beta<0 => z<0`；
- 低速一致：`|beta|<=0.1` 时 `z` 与 `beta` 误差较小。

## R14

常见失效模式：

1. 符号约定混乱（把“远离”误写成负号）导致红蓝移反转；
2. 使用 `|beta|>=1` 触发数学奇异；
3. 把波长比和频率比公式混用，造成 `D_f*D_lambda != 1`；
4. 将经典近似误当作高速精确公式。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 定义速度数组 `betas`（含接近、静止、远离三类样本）与发射波长常量。  
2. `validate_beta_array` 检查输入数组维度、有限性和 `|beta|<1` 约束。  
3. `doppler_frequency_ratio` 根据 `sqrt((1-beta)/(1+beta))` 计算频率比。  
4. `doppler_wavelength_ratio` 根据 `sqrt((1+beta)/(1-beta))` 计算波长比。  
5. `relativistic_redshift` 由波长比得到 `z = D_lambda - 1`。  
6. `beta_from_redshift` 用解析逆公式 `((1+z)^2-1)/((1+z)^2+1)` 反求 `beta`，形成闭环验证。  
7. `build_summary_table` 计算速度（km/s）、观测波长、经典近似误差，并组织 `pandas.DataFrame`。  
8. `main` 打印结果并执行断言（可逆性、互逆性、符号与单调性、小速度近似），构成可复验的非交互 MVP。  

第三方库说明：`numpy` 与 `pandas` 只负责数组计算和结果展示；相对论多普勒的核心公式与逆推流程均在源码中显式展开，不依赖黑箱物理函数。
