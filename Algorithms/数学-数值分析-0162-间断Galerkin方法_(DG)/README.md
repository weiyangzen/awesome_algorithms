# 间断Galerkin方法 (DG)

- UID: `MATH-0162`
- 学科: `数学`
- 分类: `数值分析`
- 源序号: `162`
- 目标目录: `Algorithms/数学-数值分析-0162-间断Galerkin方法_(DG)`

## R01

本条目实现间断 Galerkin（Discontinuous Galerkin, DG）方法的最小可运行版本，针对一维线性对流方程：

`u_t + a u_x = 0, x in [0,1]`（周期边界）。

实现目标：
- 明确 DG 的“单元内多项式 + 单元间数值通量”结构；
- 给出可运行的 `P1`（每单元线性）离散与显式时间推进；
- 输出 `L2` 误差、质量守恒漂移和网格加密收敛阶，形成可验证 MVP。

## R02

问题定义（本 MVP）：
- 输入：
  - 单元数 `N`；
  - 对流速度 `a`；
  - 终止时间 `T`；
  - CFL 参数 `cfl`。
- 方程与边界：
  - PDE：`u_t + a u_x = 0`；
  - 边界：周期边界；
  - 初值：`u(x,0)=sin(2*pi*x)`。
- 输出：
  - `DGResult`（步数、`dt`、绝对/相对 `L2` 误差、初末质量）。

## R03

离散空间与基函数：
- 区间 `[0,1]` 均匀剖分为 `N` 个单元，单元长度 `h=1/N`；
- 每个单元映射到参考区间 `xi in [-1,1]`；
- 局部近似采用 `P1` 基：`{1, xi}`；
- 单元内解表示为：
  `u_h(x,t)|_{K_j} = c_{j,0}(t) + c_{j,1}(t)*xi`。

`c_{j,0}` 对应单元平均，`c_{j,1}` 对应局部斜率模态。

## R04

DG 弱形式（线性通量 `f(u)=a*u`）：
- 单元测试函数 `v` 取局部基函数；
- 采用分部积分后形式：
  `int_{K_j} u_t v dx - int_{K_j} a*u*v_x dx + f_hat_{j+1/2} v^-_{j+1/2} - f_hat_{j-1/2} v^+_{j-1/2} = 0`；
- 单元界面通过数值通量 `f_hat` 耦合相邻单元。

此结构允许单元间解不连续，是 DG 的关键特征。

## R05

数值通量选择：
- 实现采用 Rusanov（局部 Lax-Friedrichs）通量：
  `f_hat(uL,uR)=0.5*(a*uL+a*uR)-0.5*|a|*(uR-uL)`；
- 对本标量线性对流方程，该通量与上风通量等价；
- 兼容 `a>0` 与 `a<0`，无需手写分支。

## R06

时间推进：
- 空间离散得到常微分方程组 `dU/dt = L(U)`；
- 采用三阶 SSP-RK3：
  1. `U1 = U^n + dt*L(U^n)`
  2. `U2 = 3/4*U^n + 1/4*(U1 + dt*L(U1))`
  3. `U^{n+1} = 1/3*U^n + 2/3*(U2 + dt*L(U2))`
- 时间步按 `dt = cfl * h / ((2p+1)|a|)` 估计（`p=1`，因子为 `3`）。

## R07

质量矩阵与半离散更新（`{1,xi}` 基）要点：
- 局部质量矩阵对角可显式消元：
  - `M00 = h`，`M11 = h/3`；
- 在实现中直接写成两条 RHS 更新：
  - `dc0/dt = (flux_left - flux_right)/h`
  - `dc1/dt = (3/h)*(2*a*c0 - flux_right - flux_left)`

这样避免了通用矩阵装配，保留 DG 核心机制且足够透明。

## R08

初值投影与误差评估：
- 初值不直接点值赋值，而是做 `L2` 投影到 `P1` 空间；
- 投影积分使用 3 点 Gauss-Legendre；
- `L2` 误差积分使用 5 点 Gauss-Legendre；
- 精确解为平移：`u(x,t)=sin(2*pi*(x-a*t))`（按周期取模）。

## R09

复杂度分析（单次仿真）：
- 每步 RHS 计算仅包含向量运算与 `roll`，复杂度 `O(N)`；
- 总步数 `Nt ~ O(T/dt) ~ O(N)`（固定 CFL）；
- 总时间复杂度约 `O(N^2)`；
- 空间复杂度 `O(N)`（存储 `N x 2` 系数与若干工作数组）。

## R10

`demo.py` 结构：
- `DGConfig` / `DGResult`：配置与结果容器；
- `project_to_p1`：初值 `L2` 投影；
- `rusanov_flux`：单界面数值通量；
- `dg_rhs`：构造半离散右端；
- `ssp_rk3_step`：单步时间推进；
- `l2_error`：计算绝对/相对 `L2` 误差；
- `run_dg`：完整仿真流程；
- `main`：执行三组网格并输出收敛信息。

## R11

运行方式：

```bash
cd Algorithms/数学-数值分析-0162-间断Galerkin方法_(DG)
python3 demo.py
```

脚本无需交互输入，会自动运行 `N=40,80,160` 三组网格。

## R12

输出字段说明：
- `cells`：单元数 `N`；
- `steps`：时间步数 `Nt`；
- `dt`：实际时间步长（会对齐到 `T`）；
- `L2_error`：绝对 `L2` 误差；
- `rel_L2`：相对 `L2` 误差；
- `mass_drift`：`mass_final - mass_initial`，用于观察守恒误差。

## R13

最小测试集（脚本已内置）：
- 固定物理参数：`a=1.0`、`T=0.5`、`cfl=0.18`；
- 网格序列：`N in {40, 80, 160}`；
- 验证点：
  - 误差随网格加密下降；
  - 观测 `L2` 收敛阶约接近 2（`P1` 空间）；
  - 质量漂移保持接近机器误差量级。

## R14

关键参数建议：
- `num_cells`：先用 `40~80` 快速检查，再增大做收敛试验；
- `cfl`：建议 `0.1~0.2`，过大可能破坏显式稳定性；
- `final_time`：建议先用 `0.5` 或 `1.0` 观察误差与守恒。

如果换成更陡初值（如方波），可进一步加入限制器（本 MVP 未包含）。

## R15

与相近方法对比：
- 对比有限体积（FV）：
  - FV 单元内通常常数重构；
  - DG 在单元内保留多项式自由度，精度更高但实现更复杂。
- 对比连续有限元（CG）：
  - CG 在单元接口强制连续；
  - DG 通过数值通量弱耦合，适合双曲问题与局部自适应。
- 对比谱方法：
  - 谱方法全局高阶但对间断敏感；
  - DG 局部化强、并行友好、工程鲁棒性更好。

## R16

典型应用场景：
- 可压缩流体力学中的守恒律离散；
- 电磁波传播（Maxwell 方程）的时域计算；
- 伴随间断/激波的双曲 PDE 数值求解；
- 高性能并行计算中的局部装配与通信友好离散。

## R18

源码级算法流（对应 `demo.py`，8 步）：
1. `main` 定义参数（`a,T,cfl`）和网格级别 `N=40,80,160`，逐个调用 `run_dg`。  
2. `run_dg` 先调用 `project_to_p1`，把初值 `sin(2*pi*x)` 通过 3 点高斯积分投影到每个单元的 `P1` 系数 `(c0,c1)`。  
3. `run_dg` 按 CFL 规则计算 `dt` 与总步数 `Nt`，构造右端算子 `rhs = dg_rhs`。  
4. 每个时间步调用 `ssp_rk3_step`，内部三次评估 `dg_rhs` 完成 SSP-RK3 更新。  
5. `dg_rhs` 在每个单元求左右迹值 `u^-`,`u^+`，并通过 `rusanov_flux` 计算所有界面通量。  
6. `dg_rhs` 使用解析化质量矩阵结果，组装 `dc0/dt` 与 `dc1/dt` 两个模态更新并返回。  
7. 时间推进完成后，`run_dg` 调用 `l2_error`（5 点高斯积分）与精确平移解比较，得到绝对/相对 `L2` 误差。  
8. `main` 汇总打印各网格误差、质量漂移和相邻网格收敛阶，完成可验证输出。
