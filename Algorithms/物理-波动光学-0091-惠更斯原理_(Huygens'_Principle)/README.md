# 惠更斯原理 (Huygens' Principle)

- UID: `PHYS-0091`
- 学科: `物理`
- 分类: `波动光学`
- 源序号: `91`
- 目标目录: `Algorithms/物理-波动光学-0091-惠更斯原理_(Huygens'_Principle)`

## R01

惠更斯原理的核心表述是：**波前上的每一点都可看作次级球面波源，下一时刻波前是这些次级波的包络面**。

在数值实现里，这可转写为“把孔径面离散为很多小源元，然后在观察屏上对所有源元的复振幅贡献做相干叠加”。这正是本条目 `demo.py` 的主体思想。

## R02

本条目选择一个最小但完整的可验证闭环：

1. 用离散次级波叠加直接计算传播场（不走黑盒传播器）；
2. 在单缝场景下，与 Fraunhofer `sinc^2` 包络做误差对比；
3. 在双缝场景下，验证干涉条纹间距 `Δx = λz/d`；
4. 输出阈值检查并给出 `Validation: PASS/FAIL`。

## R03

MVP 的建模边界与假设：

- 使用 1D 标量波模型（孔径横向 `x`，传播方向 `z`）；
- 采用次级球面波核 `exp(i k r)/r`，可选倾斜因子 `z/r`；
- 光源为单色相干波，孔径透过率为理想矩形；
- 不建模偏振、介质吸收、相机噪声和器件像差；
- 比较解析式时使用远场单缝近似 `sinc^2`。

## R04

`demo.py` 的核心方程：

1. 波数与几何距离  
`k = 2π/λ`，`r(x_a, x_s) = sqrt((x_s - x_a)^2 + z^2)`

2. 惠更斯叠加（离散形式）  
`U(x_s) ≈ Σ_a U0(x_a) * exp(i k r) / r * (z/r) * Δx_a`

3. 强度与归一化  
`I(x_s) = |U(x_s)|^2`，`I_norm = I / max(I)`

4. 单缝解析包络  
`I_ana(x) ∝ sinc^2(a x / (λ z))`

5. 双缝条纹理论间距  
`Δx_theory = λ z / d`

## R05

设孔径采样点数为 `N_a`，屏幕采样点数为 `N_s`：

- 距离矩阵和传播核计算是 `O(N_a N_s)`；
- 复数求和也是 `O(N_a N_s)`；
- 峰值检测、误差统计、表格输出是 `O(N_s)`。

总体时间复杂度 `O(N_a N_s)`，空间复杂度主要由核矩阵主导，也为 `O(N_a N_s)`。

## R06

脚本输出三类验收信息：

- **单缝衍射验证**：中心带 RMSE、第一暗纹位置误差；
- **双缝干涉验证**：条纹间距相对误差、可见度；
- **样例剖面表**：若干 `x` 位置的数值/解析归一化强度对照。

最终统一阈值判断并输出 `Validation: PASS` 或 `Validation: FAIL`。

- 同时覆盖衍射包络与干涉条纹两类经典现象；

## R10

正确性直觉链：

1. 若每个孔径点都发次级球面波，则屏上场是复振幅矢量和；
2. 单缝中，不同路径相位差形成中心亮纹和旁瓣衰减；
3. 远场下该衰减应近似 `sinc^2`，因此可用解析式核对；
4. 双缝中，两束主贡献波叠加形成近似等间距条纹；
5. 条纹间距随 `λz/d` 缩放，数值若与此一致则机制闭环成立。

## R11

数值稳定策略：

- 使用显式 `1e-15` 小量避免除零与归一化异常；
- 第一暗纹搜索只在理论附近窗口进行，减少离群误判；
- 条纹间距用 `find_peaks` + 中心区域峰筛选，降低边缘畸变影响；
- 同时使用多指标（RMSE、暗纹位置、间距、可见度）避免单指标偶然通过。

- 第一暗纹位置与理论值一致性达标；

## R14

常见失效模式：

1. 采样太粗导致暗纹和峰值定位跳变；
2. 屏幕范围太窄，无法覆盖足够多条纹用于间距估计；
3. 峰值检测阈值不合适，导致漏检/误检；
4. 把远场解析式用于明显近场参数时，单缝对比误差会升高；
5. 参数单位混淆（`μm` 与 `m`）会直接破坏尺度关系。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `make_grids` 生成孔径面与屏幕面的离散坐标，固定离散积分网格。  
2. `build_single_slit_aperture` / `build_double_slit_aperture` 构造透过函数 `U0(x)`。  
3. `huygens_propagate_1d` 显式计算每个源点到每个观察点距离 `r`，构建 `exp(i k r)/r`（含 `z/r`）传播核。  
4. 对所有源点复振幅贡献求和并乘 `Δx`，得到屏幕复场 `U(x_s)`；`normalize_intensity` 转为归一化强度。  
5. `analytic_single_slit_intensity` 计算单缝 `sinc^2` 参考；`estimate_first_minimum` 在理论附近窗口定位第一暗纹。  
6. `build_single_slit_report` 统计中心带 RMSE 和暗纹相对误差，形成单缝验证报告。  
7. `estimate_fringe_spacing` 用 `scipy.signal.find_peaks` 在中心条纹群估计平均间距，`estimate_visibility` 评估可见度，`build_double_slit_report` 汇总双缝指标。  
8. `main` 统一输出表格与阈值判定，打印 `Validation: PASS/FAIL` 并以进程退出码表示是否通过。  

第三方包职责拆解：

- `numpy`：复数传播核、矢量化叠加、误差与归一化计算；
- `scipy.signal`：条纹峰值检测（`find_peaks`）；
- `pandas`：样例剖面表格输出。

算法主逻辑（次级波核、几何距离、叠加积分、误差定义、阈值判定）都在源码显式实现，不是黑盒函数一键替代。
