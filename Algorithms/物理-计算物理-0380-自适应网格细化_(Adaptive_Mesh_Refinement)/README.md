# 自适应网格细化 (Adaptive Mesh Refinement)

- UID: `PHYS-0361`
- 学科: `物理`
- 分类: `计算物理`
- 源序号: `380`
- 目标目录: `Algorithms/物理-计算物理-0380-自适应网格细化_(Adaptive_Mesh_Refinement)`

## R01

自适应网格细化（Adaptive Mesh Refinement, AMR）是一类“把计算资源放在误差大的局部区域”的数值策略。

核心思想：
- 先在较粗网格上求解；
- 用误差指示子定位高误差单元；
- 只对被标记单元做局部加密（h-refinement）；
- 迭代到误差或预算满足为止。

这比全域均匀加密更节省自由度，尤其适合含局部剧烈变化的物理场。

## R02

本条目 MVP 采用 1D Poisson 方程：

`-u''(x) = f(x), x in [0, 1], u(0)=u(1)=0`

并使用制造解（manufactured solution）作为真值：

`u_exact(x) = x(1-x) * (sin(3πx) + 0.5*exp(-120*(x-0.70)^2))`

该解在 `x≈0.70` 附近有局部尖特征，适合展示 AMR 相对均匀网格的优势。

## R03

离散化方法使用 1D 线性有限元（FEM）：
- 网格节点可非均匀分布；
- 单元刚度矩阵为 `[[1/h,-1/h],[-1/h,1/h]]`；
- 右端项使用 2 点 Gauss 积分组装；
- 边界条件 `u(0)=u(1)=0` 直接施加。

这样 AMR 只需改变节点分布，不需改写求解主干。

## R04

误差指示子使用“通量跳跃”型近似：
- 先算单元斜率 `s_e = (u_{e+1}-u_e)/h_e`；
- 在内部节点计算跳跃 `jump_i = |s_i - s_{i-1}|`；
- 单元指标定义为 `eta_e = h_e * sqrt(jump_left^2 + jump_right^2)`。

`eta_e` 越大，表示该区域局部误差越可能偏大，应优先细化。

## R05

单元标记策略采用 Dörfler marking：
- 将 `eta_e` 按降序排列；
- 选取最少的单元集合，使其指标和达到总和的 `theta` 比例；
- 本实现 `theta = 0.60`。

该策略可避免“全标记”，并保持 AMR 的局部性与效率。

## R06

网格细化方式为中点二分：
- 被标记单元 `[x_l, x_r]` 插入新节点 `0.5*(x_l+x_r)`；
- 非标记单元保持不变；
- 更新得到新的非均匀网格。

这是 1D AMR 最小可运行实现，逻辑清晰且稳定。

## R07

`demo.py` 主流程：
1. 从 `initial_cells=10` 的均匀网格开始。
2. 在当前网格上 FEM 求解。
3. 计算通量跳跃指标并记录历史误差。
4. 用 Dörfler 规则标记高指标单元。
5. 中点细化生成新网格。
6. 迭代到 `max_iters` 或 `max_cells` 或指标阈值。
7. 最终把 AMR 结果与“同单元数均匀网格”做误差对比。

## R08

复杂度（设最终单元数为 `N`，迭代轮数为 `K`）：
- 1D 线性 FEM 装配与求解：`O(N)` 到 `O(N^3)`（当前代码使用稠密线性代数；在 1D 小规模下可接受）。
- 指示子与标记：`O(N log N)`（排序主导）。
- 总体近似：`O(K * N log N)`（不考虑稠密求解常数项）。

空间复杂度约为 `O(N)` 到 `O(N^2)`（由稠密矩阵暂存决定）。

## R09

数值稳定与工程处理：
- 对网格单调性做显式检查，避免负单元长度；
- 用高斯积分替代粗糙点值积分，降低载荷积分误差；
- 用制造解直接评估 `L2/Linf`，避免只看残差造成误判；
- 加入 `max_cells` 与 `max_iters` 预算上限，防止无限细化。

## R10

MVP 技术栈：
- Python 3
- `numpy`：向量运算、网格与线性代数
- `pandas`：AMR 历史与摘要表输出

未依赖专用 AMR 框架（如大型 PDE 库），关键算法均在源码中可直接审计。

## R11

运行方式：

```bash
cd Algorithms/物理-计算物理-0380-自适应网格细化_(Adaptive_Mesh_Refinement)
uv run python demo.py
```

脚本无需交互输入，会打印：
- 迭代历史表（网格规模与误差演化）
- AMR vs 均匀网格对照摘要
- 阈值检查与 `Validation: PASS/FAIL`

## R12

主要输出字段说明：
- `cells`：当前迭代单元数。
- `l2_error / linf_error`：相对制造解的误差。
- `max_indicator / indicator_sum`：本轮误差指标规模。
- `L2/Linf improvement ratio (uniform/AMR)`：同自由度下均匀网格误差除以 AMR 误差（>1 表示 AMR 更优）。
- `max(h)/min(h)`：网格非均匀程度。
- `cell fraction in [0.58, 0.82]`：局部特征区间的单元占比。

## R13

demo 内置判定阈值：
1. AMR 至少执行 2 轮。
2. 最终单元数大于初始单元数。
3. 非均匀性满足 `max(h)/min(h) > 1.2`。
4. `AMR L2 <= Uniform L2`。
5. `AMR Linf <= Uniform Linf`。
6. 局部区间 `[0.58, 0.82]` 的单元占比 `>= 0.20`。

全部通过即输出 `Validation: PASS`，否则进程以非零码退出。

## R14

当前 MVP 局限：
- 仅 1D，未覆盖 2D/3D 块结构或八叉树/四叉树数据结构。
- 求解器为稠密 `numpy.linalg.solve`，未接入稀疏矩阵与多重网格。
- 指示子是通量跳跃近似，未实现更严格的残差后验估计。
- 未做 coarsening（仅 refine，不回收网格）。

## R15

可扩展方向：
- 改为稀疏组装 + `scipy.sparse.linalg`，提升大规模性能。
- 扩展到 2D（四叉树）并加入 hanging-node 约束处理。
- 增加 refine/coarsen 双向自适应，控制自由度波动。
- 引入时间依赖 PDE（如对流扩散）并做时空自适应。

## R16

典型应用：
- 冲击波、边界层、尖峰源项等局部梯度很大的流体/传热问题。
- 等离子体、电磁场中局部强场区分辨。
- 天体物理中的塌缩核、激波前沿追踪。
- 多物理耦合场景中“局部高精度 + 全局可承受成本”的平衡。

## R17

与均匀网格法对比：
- 均匀网格：实现最简单，但会在“平滑区”浪费自由度。
- AMR：实现更复杂（需要指标、标记、重构），但同预算下误差通常更低。

本条目选择 AMR MVP 的原因是：
- 问题规模小，便于完整展示算法闭环；
- 能直接量化“同单元数下 AMR 优于均匀网格”的收益。

## R18

`demo.py` 源码级算法流程（10 步）：
1. `exact_solution` 定义含局部高曲率结构的制造真解，满足零边界。
2. `forcing_term` 对真解做解析二阶导并构造 `f(x)`，保证 PDE 一致性。
3. `solve_poisson_fem_1d` 在任意非均匀节点数组上组装线性 FEM 系统并求解。
4. `compute_element_indicators` 用相邻单元斜率跳跃计算每个单元的 `eta_e`。
5. `dorfler_mark` 依据指标排序，选取累计达到 `theta` 的最小标记集。
6. `refine_mesh_midpoint` 仅对标记单元做中点二分，产生新网格。
7. `run_amr` 循环执行“求解-估计-标记-细化”，并记录每轮误差和指标历史。
8. `l2_error` 与 `linf_error` 用制造解评估数值误差（分别用高斯积分和密集采样）。
9. `run_uniform_baseline` 在与 AMR 终态同单元数的均匀网格上求解，形成公平对照。
10. `main` 汇总表格、执行阈值检查并输出 `Validation: PASS/FAIL`。
