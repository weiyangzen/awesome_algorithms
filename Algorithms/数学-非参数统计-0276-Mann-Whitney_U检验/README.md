## R01

Mann-Whitney U 检验（又称 Wilcoxon rank-sum 检验）用于比较两组**独立样本**是否来自同一总体分布，或更常见地，是否存在整体位置（location）差异。它是独立样本 t 检验在非正态、重尾、含离群值场景下的常用非参数替代方法。

## R02

设两组独立样本分别为 \(X=\{x_1,\dots,x_{n_1}\}\) 与 \(Y=\{y_1,\dots,y_{n_2}\}\)。
- 原假设 \(H_0\)：两组分布相同（在常见解释下可理解为位置无系统差异）
- 备择假设 \(H_1\)：
  - 双侧：两组分布不同
  - 单侧：\(X\) 倾向于更大或更小

当两组分布形状近似相同时，Mann-Whitney U 常被解释为对“中位数位置差异”的检验。

## R03

典型前提条件：
- 两组样本相互独立，组内观测也应独立
- 数据至少可排序（ordinal 及以上）
- 允许非正态、偏态与重尾
- 若希望解释为“位置差异”，最好满足两组分布形状相近（仅位置发生平移）
- ties（并列值）允许存在，但方差近似需做并列修正

## R04

适用场景：
- A/B 两组相互独立且不满足正态假设
- 样本量中小且存在离群值，不希望均值被极端值主导
- 量表型或评分型数据（可排序但不一定等距）
- 示例：两种教学法的独立班级成绩比较、两种药物在不同患者组上的反应评分比较

## R05

本目录 `demo.py` 的输入输出定义：
- 输入：两个一维独立样本数组 `x`, `y`
- 预处理：去除各自样本中的 `NaN`
- 中间量：
  - 合并样本后的平均秩 `ranks`
  - 秩和 \(R_1, R_2\)
  - 统计量 \(U_1, U_2, U=\min(U_1,U_2)\)
- 输出：
  - \(U_1/U_2/U\)
  - 正态近似 `z` 与双侧近似 `p`
  - ties 修正后的方差
  - SciPy 对照结果（若环境可用）

## R06

核心思想：
1. 将两组合并后按数值大小排序并赋秩。
2. 若两组分布一致，来自两组的秩应“混合均匀”，秩和不会系统偏向某一组。
3. 若某组整体更大，它将获得更多高秩，导致对应 \(U\) 或秩和显著偏离原假设期望。
4. 因而不依赖原始值线性尺度，而基于排序信息完成检验。

## R07

关键公式（双样本）：
1. 合并后对全部 \(N=n_1+n_2\) 个观测赋秩（并列取平均秩）。
2. 记第一组秩和为 \(R_1\)，第二组秩和为 \(R_2\)。
3. 统计量：
\[
U_1=R_1-\frac{n_1(n_1+1)}{2},\quad
U_2=R_2-\frac{n_2(n_2+1)}{2},\quad
U=\min(U_1,U_2).
\]
且 \(U_1+U_2=n_1n_2\)。
4. 大样本近似下：
\[
\mu_U=\frac{n_1n_2}{2},
\]
\[
\sigma_U^2=\frac{n_1n_2}{12}\left[(N+1)-\frac{\sum_j(t_j^3-t_j)}{N(N-1)}\right],
\]
其中 \(t_j\) 是第 \(j\) 个并列组大小。
5. 正态近似统计量（对 \(U_1\)）：
\[
z=\frac{U_1-\mu_U-\text{cc}}{\sigma_U},
\]
`cc` 为连续性修正项（常取 \(\pm0.5\) 或 0）。

## R08

伪代码：

```text
input: x[1..n1], y[1..n2]
drop NaN in x and y
merged <- concat(x, y)
labels <- [group1 for x, group2 for y]
ranks <- rank(merged, ties=average)
R1 <- sum(ranks where label==group1)
R2 <- sum(ranks where label==group2)
U1 <- R1 - n1*(n1+1)/2
U2 <- R2 - n2*(n2+1)/2
U <- min(U1, U2)
compute tie-corrected var(U)
z <- (U1 - mean(U1) - continuity_correction)/sqrt(var)
p <- 2 * sf(|z|)
return U1, U2, U, z, p
```

## R09

正确性直觉：
- 把每个观测与另一组观测比较，\(U_1\) 本质上等价于“第一组比第二组大的配对次数（含并列折半）”。
- 在 \(H_0\) 下，第一组赢/输应近似对称，\(U_1\) 围绕 \(n_1n_2/2\) 波动。
- 当第一组整体更大时，较大值会获得更高秩，\(R_1\) 与 \(U_1\) 系统增大，进而在 z/p 上体现显著偏离。
- 因此 U 统计量能稳定反映两独立样本的位置相对关系。

## R10

时间与空间复杂度：
- 主成本为排序赋秩：\(O((n_1+n_2)\log(n_1+n_2))\)
- 其余求秩和与统计量：\(O(n_1+n_2)\)
- 额外空间：\(O(n_1+n_2)\)

因此算法瓶颈明确且易于在中等规模数据上应用。

## R11

边界与鲁棒性注意事项：
- 任一组清洗后样本量为 0：应抛出异常
- 全部数值都相同：方差可能退化或极小，需防止除零
- ties 很多时必须做并列修正，否则近似 p 值偏差会明显
- 单侧/双侧备择需在实现中明确，避免解释错误
- 若样本很小，可优先 exact 方法；大样本用渐近近似更高效

## R12

与独立样本 t 检验（含 Welch）比较：
- t/Welch 检验关注均值差并依赖分布假设（或大样本近似）
- Mann-Whitney 关注秩序信息，对离群与偏态更稳健
- 若数据确实接近正态且方差条件满足，t 检验统计效率通常更高
- 若数据明显偏态、重尾、存在离群点，Mann-Whitney 更可靠

## R15

常见误用：
- 将配对数据错用 Mann-Whitney（配对应使用 Wilcoxon 符号秩）
- 将“分布不同”直接过度解读为“均值不同”
- 忽略 ties 修正，导致近似显著性偏差
- 把双侧 p 值当单侧结论使用
- 仅报告 p 值而不报告效应方向（例如看 \(U_1\) 相对 \(\mu_U\) 的偏向）

## R16

可扩展方向：
- 增加 `alternative` 参数支持 `greater/less`
- 小样本加入 exact p-value 计算或动态规划计数
- 输出效应量（如 rank-biserial correlation, Cliff's delta）
- 支持批量特征检验与多重比较校正（Benjamini-Hochberg 等）

## R18

`demo.py` 的源码级算法流（非黑盒）：
1. 输入校验与清洗：将 `x/y` 转为一维浮点数组，分别去除 `NaN`，检查两组都非空。
2. 合并并标记分组：拼接为 `merged=[x,y]`，同步构造组别标签数组用于回填秩和。
3. 排秩实现：优先使用 `scipy.stats.rankdata(method="average")`；若无 SciPy，则使用 NumPy 版平均秩实现。
4. 秩和计算：根据标签切分秩向量，求出 `R1` 与 `R2`。
5. U 统计量计算：按公式得到 `U1`、`U2` 与 `U=min(U1,U2)`，并利用 `U1+U2=n1*n2` 做一致性检查。
6. ties 修正方差：统计合并样本中各并列组大小 `t_j`，计算 \(\sum(t_j^3-t_j)\)，代入渐近方差公式。
7. 正态近似显著性：计算 \(\mu_U,\sigma_U^2\)，结合连续性修正得到 `z` 与双侧 `p`。
8. 库函数对照：调用 `scipy.stats.mannwhitneyu(..., alternative="two-sided", method="asymptotic")`，打印并与手工结果比对。
9. 结论输出：按 `alpha=0.05` 输出是否拒绝原假设，并给出方向解释（`U1` 偏大或偏小）。
