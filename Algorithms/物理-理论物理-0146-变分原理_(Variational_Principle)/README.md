## R01

变分原理的核心思想是：在所有满足边界条件的候选轨迹中，真实物理轨迹使作用量
\(S=\int L\,dt\) 取驻值（通常是极小值或鞍点）。

本条目给出一个可运行 MVP：
- 物理系统：一维简谐振子
- 拉氏量：\(L=\frac12 m\dot x^2-\frac12 kx^2\)
- 任务：固定 \(x(0)=x_0\), \(x(T)=x_T\) 时最小化离散作用量
- 方法：将轨迹离散为节点后，用数值优化求内部节点

## R02

理论背景：
- 经典力学中，牛顿方程可由最小作用量原理推出。
- 对函数泛函做变分，得到 Euler-Lagrange 方程：
  \[
  \frac{d}{dt}\left(\frac{\partial L}{\partial \dot x}\right)-\frac{\partial L}{\partial x}=0.
  \]
- 对简谐振子代入可得：\(m\ddot x + kx = 0\)。

该例可直接对比“数值变分解”和“解析解”，便于验证实现正确性。

## R03

数学建模：
- 时间区间：\([0,T]\)
- 离散网格：\(t_i=i\Delta t,\; i=0,\dots,N-1\), \(\Delta t=T/(N-1)\)
- 决策变量：内部节点 \(x_1,\dots,x_{N-2}\)（两端固定）
- 离散速度：\(v_i=(x_{i+1}-x_i)/\Delta t\)
- 离散作用量：
  \[
  S_d=\sum_{i=0}^{N-2}\left(\frac12 m v_i^2-\frac12 kx_i^2\right)\Delta t.
  \]

优化问题即最小化 \(S_d\)。

## R04

离散 Euler-Lagrange 形式可写为（内部点）：
\[
m\frac{x_{i+1}-2x_i+x_{i-1}}{\Delta t^2}+k x_i=0.
\]

在 `demo.py` 中，优化后会计算该残差向量并报告最大绝对值，作为“是否接近驻值轨迹”的证据。

## R05

实现策略（MVP）：
- 初始轨迹取两端点连线（线性插值），保证满足边界条件。
- 仅优化内部点，避免显式约束求解器。
- 使用 `scipy.optimize.minimize(..., method="L-BFGS-B")` 优化离散作用量。
- 计算并打印：
  - 初始/优化后作用量
  - 与解析解误差（max, RMS）
  - 离散 EL 残差
  - 随机扰动后作用量变大比例

## R06

数据与参数（默认值）：
- `MASS = 1.0`
- `SPRING_K = 1.0`
- `X0 = 1.0`
- `XT = 0.2`
- `T_FINAL = 1.2`（避免 \(\sin(\omega T)\approx 0\) 的退化边界）
- `N_GRID = 80`

说明：当 \(T<\pi/\omega\) 时，作用量解在该分支下更稳定地表现为最小值。

## R07

复杂度分析：
- 变量维度：\(d=N-2\)
- 每次目标函数评估复杂度：\(O(N)\)
- L-BFGS-B 单次迭代近似成本：\(O(N)\)
- 总体：\(O(I\cdot N)\)，其中 \(I\) 为迭代轮数
- 空间复杂度：\(O(N)\)

对当前 `N=80` 的教学规模，运行开销很小。

## R08

数值稳定性与误差来源：
- 时间离散误差：来自 \(\Delta t\) 有限。
- 优化容差：`ftol` 与迭代上限决定收敛精度。
- 边界退化：若 \(\sin(\omega T)\to 0\)，解析边界匹配会病态。

本实现通过较细网格和严格 `ftol` 降低误差，并显式检查退化情况。

## R12

失败模式与排查：
- 优化失败：可能由极端参数或迭代上限不足引起。
- 解析解异常：`sin(omega*T)` 接近 0。
- 误差偏大：网格太粗或容差太松。

排查建议：
- 增大 `N_GRID`；
- 提高 `maxiter`；
- 调整 `T_FINAL` 远离退化点。

## R13

与常见黑箱方案的区别：
- 不是直接调用“现成物理求解器”一步到位。
- 先显式构建离散作用量，再把内部节点作为优化变量。
- 结果再通过解析解和离散 EL 方程双重校验。

因此该 MVP 保留了变分法的核心可解释链路。

## R14

## R18

`demo.py` 的源级算法流程（非黑箱，7 步）：

1. `optimize_path` 构造时间网格与线性初始轨迹，抽取内部节点作为优化变量。  
2. `discrete_action` 将内部节点与边界拼接成完整轨迹，计算每段速度与拉氏量并求和得到离散作用量。  
3. `scipy.optimize.minimize`（L-BFGS-B）反复调用 `discrete_action`，在内部节点空间搜索更小作用量路径。  
4. 收敛后重建 `x_opt`，并和初始线性路径比较作用量改进幅度。  
5. `analytic_path` 用边界条件解出简谐振子解析轨迹，作为“真值参考”。  
6. `discrete_el_residual` 计算离散 Euler-Lagrange 残差，检查驻值方程满足程度。  
7. `perturbation_check` 对最优路径做随机扰动，统计作用量升高比例，给出局部极值证据。

这个流程把“物理变分定义 -> 离散泛函 -> 数值优化 -> 物理校验”完整打通。
