## R01

热力学积分（Thermodynamic Integration, TI）用于计算两个体系之间的自由能差 `ΔF`。  
设有路径参数 `λ∈[0,1]`，并定义插值势能 `U_λ(x)`，则

`ΔF = F_B - F_A = ∫_0^1 <∂U_λ/∂λ>_λ dλ`

其中 `<...>_λ` 表示在 `U_λ` 对应正则系综下的统计平均。  
本条目实现经典一维情形：`x` 为连续坐标，`U_λ=(1-λ)U_A+λU_B`，故 `∂U_λ/∂λ = U_B-U_A`。

## R02

TI 的价值在于把“难直接算的配分函数比值”改写成“沿路径的可采样平均量积分”：

- 在分子模拟、软物质、材料中可用于计算溶剂化自由能、结合自由能、相变自由能；
- 相比一次性重叠重加权，TI 只要求相邻 `λ` 状态可采样，工程上更稳健；
- 可用多个参考值（解析、配分函数数值积分、双向路径）交叉验证正确性。

## R03

本 MVP 输入输出（无交互，参数写在脚本内）：

- 输入：
1. 两个谐振子势 `U_A(x)=0.5*k_A*(x-μ_A)^2+c_A`、`U_B(x)=0.5*k_B*(x-μ_B)^2+c_B`；
2. 逆温度 `β`；
3. 空间离散网格 `x_grid`；
4. 积分路径网格 `lambda_grid`。
- 输出：
1. `ΔF_TI`：TI 主结果；
2. `ΔF_partition`：由端点配分函数数值积分得到的参考值；
3. `ΔF_exact`：谐振子解析自由能差；
4. 两个误差 `|ΔF_TI-ΔF_partition|`、`|ΔF_TI-ΔF_exact|`；
5. 自动断言通过后输出 `All checks passed.`。

## R04

MVP 使用的核心公式：

1. 路径势能（线性耦合）：
`U_λ(x)=(1-λ)U_A(x)+λU_B(x)`。

2. TI 被积函数：
`g(λ)=<∂U_λ/∂λ>_λ=<U_B-U_A>_λ`。

3. Boltzmann 平均：
`<O>_λ = (∫ O(x) exp(-β U_λ(x)) dx) / (∫ exp(-β U_λ(x)) dx)`。

4. TI 积分：
`ΔF_TI=∫_0^1 g(λ) dλ`（脚本中用 Simpson 积分）。

5. 端点自由能：
`F=-β^{-1} ln Z,  Z=∫ exp(-βU(x)) dx`。

6. 谐振子解析自由能（1D 经典）：
`F = c + (1/(2β)) ln(βk/(2π))`，因此
`ΔF_exact = (c_B-c_A) + (1/(2β)) ln(k_B/k_A)`。

## R05

记 `N_x = |x_grid|`、`N_λ = |lambda_grid|`：

- 每个 `λ` 计算一次加权平均，时间复杂度 `O(N_x)`；
- 全路径时间复杂度 `O(N_λ * N_x)`；
- 空间复杂度 `O(N_x)`（复用 `U_A/U_B` 与权重向量）；
- 参考自由能端点积分额外 `O(N_x)`，不改变主阶。

## R06

`demo.py` 包含三个最小闭环：

- **闭环 A（TI 主链路）**：逐个 `λ` 计算 `g(λ)=<U_B-U_A>_λ`，再沿 `λ` 积分；
- **闭环 B（独立数值参考）**：直接算端点配分函数 `Z_A/Z_B` 得 `ΔF_partition`；
- **闭环 C（解析参考）**：用谐振子闭式公式给 `ΔF_exact` 并与 TI 对比。

## R07

优点：

- 同时提供“数值参考 + 解析参考”双校验；

## R10

正确性直觉：

1. `ΔF` 是状态函数，与积分路径无关；只要 `g(λ)` 估计准确，路径积分应还原真实差值。  
2. 对每个 `λ`，`<U_B-U_A>_λ` 是在该系综下对“瞬时导数”的平均，属于 TI 的标准恒等式。  
3. 端点 `Z_A/Z_B` 给出自由能差的独立表达，可作为 TI 的数值交叉验证。  
4. 对谐振子，存在闭式解；若 TI 与闭式一致，说明实现链路（权重、积分、归一化）正确。

## R11

数值稳定策略：

- 对 `exp(-βU)` 使用能量平移：`U -> U - U_min`，避免下溢；
- `lambda_grid` 与 `x_grid` 强制单调递增，防止积分器输入异常；
- 对分母 `∫exp(-βU)dx` 做正值检查，提前捕获病态参数；
- 端点与解析双参考同时断言，避免“某一条链路偶然对齐”。

## R12

关键参数：

- `beta`：温度倒数，越大分布越尖，积分对网格更敏感；
- `x_grid` 范围与密度：范围过小会截断尾部，密度过低会有离散误差；
- `lambda_grid` 密度：太稀会放大路径积分误差；
- `k_A,k_B,μ_A,μ_B,c_A,c_B`：控制势能曲率、位置和自由能偏移。

调参建议：

- 先保证 `x` 范围覆盖主要概率质量（尾部贡献很小）；
- 再增大 `N_x` 或 `N_λ` 观察误差收敛；
- 若 `β` 很大，优先提高 `x_grid` 密度并缩小步长。

## R14

常见失效模式：

1. `k<=0` 导致势能非束缚，配分函数发散；
2. `x` 截断区间太窄，端点自由能和 TI 都出现系统偏差；
3. `lambda_grid` 太稀，路径积分低精度；
4. 忽略指数平移导致 `exp(-βU)` 下溢，出现 0/0 或数值噪声；
5. 参数极端（超大 `β`、超大曲率）但未同步提升网格精度。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `HarmonicState` / `TIProblem` 定义端点势参数、`β`、`x_grid`、`lambda_grid`，并在 `validate` 中做参数合法性检查。  
2. `harmonic_potential` 显式计算 `U_A(x)` 与 `U_B(x)`，`interpolate_potential` 构造 `U_λ`。  
3. `boltzmann_average` 对每个 `λ` 计算 `<U_B-U_A>_λ`：先做 `U_min` 平移，再用 `exp(-β(U-U_min))` 形成稳定权重。  
4. `ti_profile` 在所有 `λ` 网格上循环调用 `boltzmann_average`，得到离散曲线 `g(λ)`（以 `pandas.DataFrame` 持有）。  
5. `solve` 对 `g(λ)` 用 `scipy.integrate.simpson` 积分，得到 `ΔF_TI`。  
6. `log_partition` 与 `free_energy_from_partition` 在 `λ=0,1` 端点计算 `Z_A,Z_B` 和 `ΔF_partition`，作为独立数值参考。  
7. `analytic_harmonic_free_energy` 计算谐振子闭式 `F`，得到 `ΔF_exact`，再统计 TI 与两类参考的误差。  
8. `main` 打印全部指标并断言误差阈值，最终输出 `All checks passed.`。  

说明：`scipy` 仅提供 Simpson 求积；关键的 TI 物理流程（路径构造、Boltzmann 加权、导数平均、端点参考与误差验证）全部在源码中显式实现，不是库黑盒调用。
