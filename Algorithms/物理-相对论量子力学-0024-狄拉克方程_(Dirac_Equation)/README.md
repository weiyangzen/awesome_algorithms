## R01

狄拉克方程是将量子力学与狭义相对论统一后的自由费米子方程。自然单位制 (`hbar=c=1`) 下：

`i * d(psi)/dt = H psi,  H = -i * alpha * d/dx + beta * m + V(x)`

其中 `psi` 是旋量波函数，`alpha`、`beta` 满足反对易关系，保证能谱满足相对论色散关系 `E^2 = p^2 + m^2`，并自然包含正负能量支。

## R02

在相对论量子力学中，狄拉克方程是电子/正电子动力学、量子场论量子化前的核心桥梁。它解释了：

- 自旋 `1/2` 粒子的动力学结构；
- 粒子-反粒子分支的对称能谱；
- 高频“颤振”（zitterbewegung）等典型相对论量子效应。

对算法工程而言，它也是构建格点量子模拟器和高能物理数值实验的基础模型。

## R03

本条目 MVP 解决三个最小闭环问题：

- 构建 1D 有限差分狄拉克哈密顿量，并验证 Hermitian 性与正负能谱对称性；
- 用平面波本征态做时间推进，从相位斜率反推能量，验证离散色散关系；
- 用高斯波包推进，检查概率与能量守恒，并量化颤振指标。

目标是完成“方程 -> 离散算子 -> 时间传播 -> 可检验物理量”的可运行链路。

## R04

`demo.py` 使用的核心公式：

1. 1D 狄拉克哈密顿量（两分量旋量）  
`H = -i * sigma_x * d/dx + sigma_z * m + V(x) * I_2`
2. 中心差分（周期边界）  
`(d psi/dx)_j ~= (psi_{j+1} - psi_{j-1}) / (2 dx)`
3. 时间推进（定常哈密顿量）  
`psi(t+dt) = exp(-i H dt) psi(t)`
4. 归一化与概率密度  
`sum_x |psi|^2 dx = 1,  rho(x)=|psi_up|^2+|psi_down|^2`
5. 能量期望  
`<H> = <psi|H|psi> dx`
6. 平面波离散色散（中心差分）  
`p_d = sin(k dx)/dx,  E = sqrt(m^2 + p_d^2)`

## R05

设网格点数为 `N`，自旋分量 2，总状态维度 `2N`：

- 构造导数矩阵与哈密顿量：`O(N^2)`（MVP 用稠密矩阵）；
- 计算传播子 `U=expm(-iHdt)`：稠密矩阵指数约 `O((2N)^3)`；
- 每步传播 `psi <- U psi`：`O((2N)^2)`；
- 总传播 `T` 步：`O(T * N^2)`；
- 线性回归拟合相位斜率：`O(T)`。

空间复杂度主要由 `H` 与 `U` 决定，为 `O(N^2)`。

## R10

正确性直觉：

1. `H` 若严格 Hermitian，则 `U=exp(-iHdt)` 应近似幺正，范数守恒；
2. 自由狄拉克系统应存在关于 0 对称的正负能谱；
3. 本征态只积累相位，`phase(t)` 斜率就是 `-E`；
4. 时间无关哈密顿量下，`<H>` 应守恒（仅有数值舍入误差）；
5. 混合正负能分量的局域波包会出现高频颤振分量。

## R11

稳定性与数值精度策略：

- 使用中心差分 + 周期边界，保证离散动量结构清晰；
- 通过矩阵指数构造一步传播子，避免显式 Euler 的稳定性问题；
- 平面波验证采用离散动量 `sin(kdx)/dx`，消除“连续公式直套”误差；
- 所有守恒量检查使用最大漂移指标并设置严格阈值。

## R12

关键参数：

- `num_points`：空间离散精度，越大越接近连续极限；
- `length`、`dx`：决定动量分辨率与 Nyquist 上限；
- `mass`：能隙与颤振频率尺度；
- `dt`、`num_steps`：传播时间分辨率与观察窗口；
- `mode`（平面波模态）：设置测试动量；
- `sigma`、`x0`、`p0`（高斯包）：控制局域性、初始位置和平均动量。

调参建议：

- 先固定 `mass`，通过增大 `num_points` 看能量误差收敛；
- 若颤振不明显，调大 `p0` 或减小 `sigma` 以增强正负能混合；
- 想降低矩阵指数成本，可减小 `N` 并缩短 `num_steps`。

## R14

常见失效模式：

1. 误用非周期边界却沿用本导数矩阵，导致边界伪影；
2. 把连续动量 `k` 直接代入色散而忽略离散修正 `sin(kdx)/dx`；
3. `dx` 或 `dt` 选取不合理，造成谱截断误差或时间分辨不足；
4. 状态向量归一化遗漏 `dx` 因子，导致守恒量判断偏差；
5. 大规模 `N` 下继续用稠密 `expm`，出现性能瓶颈。

## R18

`demo.py` 源码级算法流程（9 步）：

1. `periodic_central_difference_matrix` 构造周期中心差分算子 `D`，把连续 `d/dx` 离散为矩阵。  
2. `dirac_hamiltonian_1d` 用 `sigma_x/sigma_z` 与 Kronecker 积组装 `H`（动能项、质量项、势能项）。  
3. `run_spectrum_consistency_demo` 先测 `H` 的 Hermitian 缺陷，再用 `numpy.linalg.eigvalsh` 与 `torch.linalg.eigvalsh` 双后端算谱，并检查 `lambda_i + lambda_{-i} ~= 0` 的正负能对称。  
4. `positive_energy_spinor` 给定离散动量 `p_d` 构造正能本征旋量，`build_plane_wave_state` 生成平面波初态。  
5. `expm(-iHdt)` 生成一步传播子 `U`，随后循环执行 `psi <- U psi` 完成时间推进。  
6. `run_plane_wave_phase_demo` 每步计算与初态重叠 `c(t)=<psi(0)|psi(t)>`，对 `arg(c(t))` 解缠后用 `LinearRegression` 拟合斜率，得到 `E_est=-dphi/dt`，并与离散色散 `E_ref=sqrt(m^2+p_d^2)` 比较。  
7. `gaussian_wave_packet` 构造局域高斯包，`run_gaussian_packet_demo` 记录 `<x>(t)`、`||psi||`、`<H>`，验证守恒量漂移在阈值内。  
8. 对 `<x>(t)` 进行线性去趋势，残差 RMS 作为颤振（zitterbewegung）量化指标。  
9. `print_report` 用 `pandas` 输出三组指标表格，`main` 汇总并断言通过后结束。  

第三方库没有被当黑盒：`scipy` 只用于矩阵指数，`sklearn` 只用于一元线性拟合斜率，`torch` 只作为谱计算交叉验证后端；物理建模、离散算子构造、守恒量与诊断逻辑均在源码中显式实现。
