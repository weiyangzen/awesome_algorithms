## R01

爱因斯坦模型（Einstein Model）是晶体热容理论的早期量子模型。核心假设是：

- 晶体中每个原子都可近似为 3 个独立、同频率 `omega_E` 的量子简谐振子；
- 频率统一映射为特征温度 `Theta_E = hbar*omega_E/k_B`；
- 由玻色占据数得到内能和定容热容随温度变化。

常用摩尔热容表达式为：

`C_V(T) = 3R * (Theta_E/T)^2 * exp(Theta_E/T) / (exp(Theta_E/T)-1)^2`

其中 `R` 是气体常数。

## R02

该模型在固体物理中的意义：

- 它是“量子化晶格振动解释热容”的起点；
- 能解释高温下恢复 Dulong-Petit 定律 `C_V -> 3R`；
- 低温给出热容指数抑制（比经典常数热容更合理）。

局限是频谱过于理想化（单频），对低温实验通常不如德拜模型准确，但作为最小可计算模型非常适合教学与基线验证。

## R03

本目录 MVP 目标：

- 输入：`Theta_E` 与温度网格 `T`；
- 输出：`C_V(T)`、`C_V/(3R)`、热内能 `U_th(T)`（去掉零点能）；
- 验证：
  1. 低温区满足指数抑制渐近行为；
  2. 高温区逼近 `3R`；
  3. `C_V(T)` 随温度单调上升并饱和。

实现不调用黑盒物理求解器，全部公式与数值流程在 `demo.py` 明确展开。

## R04

本实现使用的关键公式：

1. 特征变量：`x = Theta_E / T`
2. 玻色占据数：
   `n_B(x) = 1/(exp(x)-1)`
3. 热内能（零点能剥离）：
   `U_th = 3R*Theta_E*n_B(x)`
4. 定容摩尔热容：
   `C_V = 3R * x^2 * exp(x)/(exp(x)-1)^2`
5. 数值稳定重写：
   `C_V/(3R) = x^2 * exp(-x)/(1-exp(-x))^2`

当 `x` 很小时，`demo.py` 使用级数近似避免高温端的消减误差。

## R05

复杂度（设温度点数 `N_T`）：

- 每个温度点用闭式公式计算一次，复杂度 `O(1)`；
- 全温区扫描复杂度 `O(N_T)`；
- 空间复杂度 `O(N_T)`（主要为结果表存储）。

默认 `N_T=140`，运行开销极小，适合自动化批量验证。

## R10

正确性直觉：

1. 爱因斯坦模型把晶格自由度映射为等频量子振子；
2. 按玻色占据数可直接写出 `U(T)` 与 `C_V(T)`；
3. 当 `T>>Theta_E`（`x->0`），应回到经典极限 `C_V->3R`；
4. 当 `T<<Theta_E`（`x->∞`），热激发稀少，`C_V` 指数衰减；
5. `C_V(T)` 应从低温近零单调上升并最终饱和。

`demo.py` 的断言对应这几条可执行物理检查。

## R11

数值稳定性处理：

- 使用 `exp(-x)` 与 `expm1` 重写分母，避免大 `x` 上溢和小 `x` 消减；
- 在 `x<1e-4` 使用级数展开近似：
  - `n_B(x) ≈ 1/x - 1/2 + x/12 - x^3/720`
  - `C_V/(3R) ≈ 1 - x^2/12 + x^4/240`
- 对温度与 `Theta_E` 做正值检查，避免无意义输入。

这些处理保证从低温到高温的稳定计算。

## R12

## R14

常见失败模式：

1. `Theta_E` 或温度使用错误单位（非 K）；
2. 使用 `exp(x)` 直接表达导致大 `x` 溢出；
3. 忽略小 `x` 数值抵消，导致高温端精度下降；
4. 把包含零点能的 `U` 与实验热量数据直接比较；
5. 温度采样太稀，误判单调性或渐近行为。

本实现分别通过输入检查、稳定重写与断言机制缓解这些问题。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 创建 `EinsteinConfig`，调用 `build_result_table` 执行温度扫描。  
2. `build_result_table` 生成几何温度网格，并分别调用热容与热内能函数。  
3. `einstein_heat_capacity_molar` 计算 `x=Theta_E/T`，再调用 `cv_ratio_over_3r`。  
4. `cv_ratio_over_3r` 对 `x` 分段处理：小 `x` 用级数，大 `x` 用 `x^2 exp(-x)/(1-exp(-x))^2`。  
5. `einstein_internal_energy_thermal_molar` 调用 `bose_occupancy`，计算 `U_th=3R*Theta_E*n_B(x)`。  
6. `bose_occupancy` 对小 `x` 用级数、其余区间用 `exp(-x)` 与 `expm1` 稳定计算占据数。  
7. `run_sanity_checks` 分别做低温渐近、高温 `3R`、单调性与超低温热内能检查并断言。  
8. `main` 打印样例表与检查指标，形成可复现、可审查的最小交付。  

第三方库角色说明：`numpy` 仅用于数组计算，`pandas` 仅用于表格组织与打印；模型核心（公式、分段稳定计算、物理断言）都在源码中显式实现，不依赖黑盒库。
