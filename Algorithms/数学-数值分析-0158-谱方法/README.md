## R01

谱方法（Spectral Method）通过全局基函数（如 Fourier、Chebyshev、Legendre）展开数值解，适合求解光滑问题并可达到高阶甚至指数收敛。  
本条目给出一个最小可运行 MVP：在周期区间上用 Fourier 谱方法求解一维 Poisson 方程 `-u''(x)=f(x)`，并用网格加密展示误差快速衰减。

## R02

本目录解决的问题定义为：
- 方程：`-u''(x)=f(x)`，`x in [0, 2π)`，周期边界。
- 输入：离散采样点上的 `f(x_j)` 与区间长度 `L=2π`。
- 输出：离散数值解 `u(x_j)`（取零均值规范），以及 `L2/L∞` 误差与收敛速率。

`demo.py` 使用固定解析解 `u(x)=1/(1.1-cos x)` 自动构造右端项并完成验证，不需要交互输入。

## R03

Fourier 谱方法核心关系：
- 将解展开为 `u(x)=sum_k u_hat_k e^(ikx)`。
- 二阶导数在频域中满足：`(u'')_hat_k = -(k^2) u_hat_k`。
- 对 Poisson 方程 `-u''=f`，频域变为 `k^2 u_hat_k = f_hat_k`（`k != 0`）。
- 零频 `k=0` 对应常数模态，方程不可唯一确定；MVP 采用 `u_hat_0=0`（即零均值解）。

数值实现中通过 FFT/IFFT 在物理空间与频域之间往返，实现“物理空间采样 + 频域求解”的谱离散。

## R04

算法高层流程：
1. 选定离散点数 `N`，在 `[0, L)` 构造均匀周期网格。
2. 在网格上计算解析右端项 `f(x)`。
3. 对 `f` 做 FFT 得到 `f_hat`。
4. 构造波数数组 `k` 与 `k^2`。
5. 对全部非零模态执行 `u_hat_k = f_hat_k / k^2`。
6. 将零模态设为 `u_hat_0 = 0`（固定常数自由度）。
7. IFFT 回到物理空间得到 `u_num`。
8. 与零均值参考解比较，输出误差和网格加密收敛率。

## R05

核心数据结构：
- `x: np.ndarray`：周期网格点。
- `f: np.ndarray`：右端项离散值。
- `f_hat/u_hat: np.ndarray(complex)`：频域系数。
- `k: np.ndarray`：角波数（由 `np.fft.fftfreq` 生成后乘 `2π`）。
- `results: list[dict[str, float]]`：不同 `N` 下的误差与收敛率记录。

数据结构全部保持扁平数组，便于追踪每一步线性代数含义。

## R06

正确性要点：
- 频域更新 `u_hat_k = f_hat_k / k^2` 直接来自方程 `-u''=f` 的 Fourier 系数关系。
- 使用 FFT/IFFT 的离散变换对偶保证频域求解与物理空间表示一致。
- 通过强制 `u_hat_0=0` 去除常数不唯一性，得到可比较的唯一数值解。
- 用解析解（减去均值）做误差对照，能验证实现逻辑与符号方向是否正确。

## R07

复杂度分析（单次求解，网格点数 `N`）：
- 时间复杂度：`O(N log N)`，主耗时在 FFT 与 IFFT。
- 空间复杂度：`O(N)`，存储网格、右端项与频域系数。

对比稠密全局矩阵谱离散（通常 `O(N^2)` 存储），FFT 版本更适合周期问题的快速 MVP。

## R08

边界与异常处理：
- `N < 4` 直接拒绝（分辨率不足）。
- 输入数组必须是一维、长度为 `N`、且全为有限数（非 `nan/inf`）。
- `L <= 0` 直接抛出异常。
- 零模态 `k=0` 不做除法，避免 `1/0`。

方法适用边界：本 MVP 面向“周期 + 光滑”问题；若是非周期边界或存在尖峰/间断，误差表现会显著变差。

## R09

MVP 取舍：
- 只实现 Fourier 周期谱法，不混入 Chebyshev/Legendre 版本，保持最小实现。
- 不调用高层 PDE 黑盒求解器，直接写出 FFT 到模态求解的关键路径。
- 只依赖 `numpy`，确保环境要求低、复现简单。

## R10

`demo.py` 函数职责：
- `periodic_grid`：构造 `[0, L)` 周期网格。
- `exact_solution`：给出解析解 `u(x)=1/(1.1-cos x)`。
- `forcing_term`：由解析解显式计算 `f(x)`。
- `spectral_poisson_periodic`：核心谱求解器（FFT -> 模态求解 -> IFFT）。
- `error_metrics`：统一计算 `L2/L∞` 误差。
- `run_convergence_study`：对多个 `N` 做实验并估计收敛率。
- `print_results_table`：输出表格化实验结果。
- `main`：组织执行、判定是否通过。

## R12

输出字段与 R04 流程对应：
- `N`：流程第 1 步的网格规模。
- `L2_error / Linf_error`：流程第 8 步与参考解比较得到的误差。
- `rate(L2) / rate(Linf)`：相邻网格尺度的误差衰减速率（`log2(e_old/e_new)`）。
- `max_error_all`：全部实验中的最大 `L∞` 误差。
- `pass`：是否满足预设阈值（默认看最细网格误差）。

## R13

最小实验集（`demo.py` 已内置）：
- 网格序列：`N = [16, 32, 64, 128, 256]`。
- 验证对象：`u(x)=1/(1.1-cos x)`，比较 `u_num` 与 `u_exact - mean(u_exact)`。

## R15

方法对比：
- 对平滑周期问题：谱方法通常比低阶有限差分/有限体积更高精度（同等自由度下误差更小）。
- 对非光滑问题：谱方法可能出现 Gibbs 现象，局部振荡明显。
- 对非周期边界：应改用 Chebyshev/Legendre 谱方法或谱元方法，不应直接套本 MVP。

## R16

典型应用场景（与 R08 适用边界一致）：
- 周期边界的 Poisson/Helmholtz/扩散类 PDE 子问题。
- 流体力学中的周期方向离散（如伪谱 Navier-Stokes 的空间离散部件）。
- 需要高精度解析平滑结构（波动、稳定性分析、模式分解）的数值实验。

不建议直接用于强间断或复杂非周期几何场景。

## R17

可扩展方向：
- 从 1D 扩展到 2D/3D（张量积 FFT）。
- 引入时间推进，组成完整的伪谱 PDE 求解器。
- 对非线性项加入 2/3 去混叠（dealiasing）策略。
- 加入 Chebyshev 版本处理非周期边界。

这些扩展都建立在本 MVP 的“频域模态求解主干”之上。

## R18

`demo.py` 源码级算法流程（8 步）：
1. `main` 设置 `grid_sizes`、`L` 和验收阈值，然后调用 `run_convergence_study`。  
2. `run_convergence_study` 对每个 `N` 调用 `periodic_grid` 生成离散点 `x_j`。  
3. 在 `x_j` 上调用 `forcing_term` 得到离散右端项 `f_j`，随后进入 `spectral_poisson_periodic`。  
4. `spectral_poisson_periodic` 先做输入检查（维度、长度、有限性、`L>0`）。  
5. 对 `f_j` 执行 `np.fft.fft` 得到 `f_hat_k`，并用 `np.fft.fftfreq` 构造波数 `k` 与 `k^2`。  
6. 对全部 `k != 0` 的模态执行 `u_hat_k = f_hat_k / k^2`，并显式设定 `u_hat_0 = 0`。  
7. 用 `np.fft.ifft` 反变换回物理空间得到 `u_num`；`run_convergence_study` 再构造 `u_exact-mean(u_exact)` 并计算误差。  
8. `print_results_table` 输出各 `N` 的误差与收敛率，`main` 基于最细网格误差给出 `PASS/FAIL`。
