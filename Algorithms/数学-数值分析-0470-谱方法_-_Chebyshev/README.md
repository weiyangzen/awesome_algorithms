# 谱方法 - Chebyshev

- UID: `MATH-0470`
- 学科: `数学`
- 分类: `数值分析`
- 源序号: `470`
- 目标目录: `Algorithms/数学-数值分析-0470-谱方法_-_Chebyshev`

## R01

本条目实现 Chebyshev 谱方法（配点/伪谱形式）的最小可运行版本：

- 在 `[-1,1]` 上用 Chebyshev-Gauss-Lobatto（CGL）节点离散；
- 用 Chebyshev 微分矩阵近似导数；
- 求解一维边值问题 `u''(x)=f(x)`（Dirichlet 边界）；
- 用解析解对比验证误差与收敛速度。

## R02

问题定义（MVP）：

- 输入：
  - 配点参数 `N`（节点数为 `N+1`，要求 `N>=2`）；
  - 光滑右端函数 `f(x)`；
  - 两端边界值 `u(-1), u(1)`（本 demo 由制造解给出）。
- 输出：
  - CGL 节点 `x_j`；
  - 数值解向量 `u_num`；
  - 与解析解 `u_exact` 的误差 `||u_num-u_exact||_inf`。

## R03

核心数学对象：

1. Chebyshev 节点（CGL）：
   `x_j = cos(pi*j/N), j=0,...,N`。
2. 一阶微分矩阵：
   `u'(x_j) ~ (D u)_j`。
3. 二阶微分矩阵：
   `u''(x_j) ~ (D^2 u)_j`。
4. 边值问题离散后变为线性方程组：
   `A u_interior = rhs`，其中边界值通过右端修正进入。

## R04

算法流程总览：

1. 生成 CGL 节点；
2. 构造一阶 Chebyshev 微分矩阵 `D`；
3. 计算 `D2 = D @ D`；
4. 取内部块 `A = D2[1:N,1:N]`；
5. 组装 `rhs = f_i - D2_ib * u_b`；
6. 解线性系统得到内部节点上的 `u`；
7. 拼回边界值得到完整离散解；
8. 计算无穷范数误差并做收敛展示。

## R05

数据结构设计（`demo.py`）：

- `numpy.ndarray`：
  - `x`: 节点；
  - `d1`, `d2`: 一阶/二阶微分矩阵；
  - `u_num`, `u_exact`: 数值解与解析解；
  - `rhs`: 线性系统右端。
- `SolveResult`（`dataclass`）：
  - 汇总 `n`、`x`、`u_num`、`u_exact`、`error_inf`，便于打印与扩展。

## R06

正确性要点：

- CGL 节点上的插值多项式导数可由微分矩阵离散表示；
- 对足够低次多项式，微分矩阵导数在机器误差内可近似精确；
- 对光滑目标函数，Chebyshev 配点法常见谱收敛（误差快速下降）；
- Dirichlet 边界通过“已知边界移项”嵌入线性系统，满足边界一致性。

## R07

时间复杂度（稠密实现）：

- 构造 `D`：`O(N^2)`；
- 形成 `D2 = D @ D`：`O(N^3)`；
- 求解内部线性系统（规模 `N-1`）：`O(N^3)`；
- 单次求解主导项为 `O(N^3)`。

本 MVP 重点是透明实现，不追求大规模最优常数。

## R08

空间复杂度：

- 存储 `D`、`D2` 等稠密矩阵：`O(N^2)`；
- 存储向量（节点、右端、解）：`O(N)`。

若用于大规模计算，应改为更节省内存的算子形式或快速变换路线。

## R09

边界与异常处理：

- `N < 2`：抛出 `ValueError`；
- 若 `D` 或 `D2` 出现 `nan/inf`：抛出 `RuntimeError`；
- 线性系统奇异或病态时会在 `np.linalg.solve` 抛异常；
- 该 MVP 假定解与右端足够光滑，不覆盖非光滑/间断问题。

## R10

MVP 取舍说明：

- 只依赖 `numpy`，不引入更重框架；
- 不调用黑箱边值求解器（如 `scipy.integrate.solve_bvp`）；
- 显式写出节点、微分矩阵、方程组组装与求解流程；
- 以内置断言保障“可运行且可验证”，而非工程化通用库。

## R13

输出指标解释：

- `inf-norm error`：`max_j |u_num(x_j)-u_exact(x_j)|`；
- `prev/cur`：上一组误差与当前误差比值（越大表示收敛越快）；
- 自检误差：检验 `D @ p` 是否匹配多项式解析导数。

对解析光滑解，随着 `N` 增加，误差应快速下降并接近机器精度平台。

## R15

与常见方法对比：

- 对比二阶有限差分：
  - 差分矩阵稀疏、单步便宜；
  - Chebyshev 谱法在光滑解上常能用更小自由度达到高精度。
- 对比黑箱 BVP 求解器：
  - 黑箱接口更通用；
  - 本实现更适合教学、原理验证和可解释调试。

## R16

适用场景：

- 有限区间上的光滑边值问题；
- 需要高精度但自由度希望较小的模型问题；
- 谱方法课程中的“从公式到代码”演示；
- 后续扩展到谱元、Legendre/Fourier 谱法的基线实现。

## R17

## R18

源码级算法流（`solve_poisson_dirichlet_chebyshev`，8 步）：

1. 调用 `chebyshev_diff_matrix(n)`，拿到 CGL 节点 `x` 与一阶矩阵 `D`。  
2. 通过矩阵乘法得到二阶离散算子 `D2 = D @ D`。  
3. 在节点上计算解析解 `u_exact(x)` 与右端 `f(x)`。  
4. 取内部未知对应的子矩阵 `A = D2[1:n, 1:n]`。  
5. 用已知边界值修正右端：`rhs = f_i - D2_i0*u_left - D2_in*u_right`。  
6. 调用 `np.linalg.solve(A, rhs)` 求出内部节点解 `u[1:n]`。  
7. 把两端边界值与内部解拼装成完整 `u_num`。  
8. 计算 `||u_num-u_exact||_inf`，封装 `SolveResult` 返回给收敛验证流程。  

说明：本实现没有把核心离散步骤委托给第三方黑箱；节点构造、微分矩阵、边界移项和线性求解在源码中全部显式可见。
