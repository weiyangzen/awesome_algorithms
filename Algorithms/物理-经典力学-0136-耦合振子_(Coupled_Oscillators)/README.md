# 耦合振子 (Coupled Oscillators)

- UID: `PHYS-0136`
- 学科: `物理`
- 分类: `经典力学`
- 源序号: `136`
- 目标目录: `Algorithms/物理-经典力学-0136-耦合振子_(Coupled_Oscillators)`

## R01

耦合振子是经典力学中最基本的多自由度振动模型之一。  
本条目实现的是两质量块-三弹簧的 1D 线性耦合系统（无阻尼、无外驱动），目标是把：

- 连续方程 `M x¨ + K x = 0`
- 正则模态分解 `K*phi = omega^2*M*phi`
- 时域数值积分（RK4）

串成一个可执行、可校验、可复现实验的最小闭环。

## R02

问题定义（本目录 MVP）：

- 输入：
  - 物理参数 `m1, m2, k1, k2, kc`
  - 数值参数 `dt, steps`
  - 初始条件 `x0=[x1(0),x2(0)]`, `v0=[v1(0),v2(0)]`
- 输出：
  - 时间序列 `t`
  - 数值解 `x_num(t), v_num(t)`（RK4）
  - 解析模态解 `x_exact(t), v_exact(t)`
  - 误差指标与能量漂移指标

系统结构为 `wall - k1 - m1 - kc - m2 - k2 - wall`，只考虑小振幅线性近似。

## R03

动力学方程写成矩阵形式：

`M = [[m1, 0], [0, m2]]`

`K = [[k1+kc, -kc], [-kc, k2+kc]]`

`x = [x1, x2]^T`

则有：

`M x¨ + K x = 0`

展开后：

- `m1*x1¨ + (k1+kc)*x1 - kc*x2 = 0`
- `m2*x2¨ + (k2+kc)*x2 - kc*x1 = 0`

这是典型线性常系数二阶耦合 ODE，可用模态分解或状态空间积分求解。

## R04

正则模态解法核心：

1. 求广义特征值问题 `K*phi = lambda*M*phi`，其中 `lambda = omega^2`。  
2. 得到模态矩阵 `Phi=[phi1,phi2]` 与角频率 `omega1, omega2`。  
3. 用质量正交关系把初值投影到模态坐标：
   - `q0 = Phi^T * M * x0`
   - `qdot0 = Phi^T * M * v0`
4. 每个模态满足标量简谐方程：
   - `q_i(t) = q0_i*cos(omega_i*t) + (qdot0_i/omega_i)*sin(omega_i*t)`
5. 回到物理坐标：`x(t)=Phi*q(t)`，`v(t)=Phi*qdot(t)`。

`demo.py` 同时给出该解析解与 RK4 数值解并做一致性检查。

## R05

复杂度（自由度固定为 2）：

- 模态分解：常数规模矩阵特征分解，近似 `O(1)`；
- 时间积分：`steps = N` 时，RK4 每步常数次矩阵向量运算，时间复杂度 `O(N)`；
- 空间复杂度：保存全轨迹为 `O(N)`。

若推广到 `d` 自由度、`N` 步：

- 模态分解约 `O(d^3)`；
- 每步矩阵向量乘法约 `O(d^2)`，总计 `O(N*d^2)`。

## R06

守恒量检查（无阻尼无外力）：

总能量应守恒：

`E(t) = 0.5 * v^T M v + 0.5 * x^T K x`

MVP 会输出相对能量漂移：

`max_t |E(t)-E(0)| / |E(0)|`

这既验证了方程实现，也验证了数值步长是否足够小。

## R07

方法优点：

- 物理解释直接：可清楚看到“本征频率 + 模态叠加”机制；
- 代码小：几十行核心逻辑即可跑通全流程；
- 可验证性强：解析解与数值解可逐点比对。

局限：

- 仅适用于线性小振幅模型；
- 未包含阻尼、非线性弹簧、外驱动；
- 目前只演示 2 自由度系统。

## R08

前置知识建议：

- 二阶常微分方程与状态空间表示；
- 线性代数中的特征值/特征向量；
- 简谐振动与相位叠加；
- Python `numpy/pandas/scipy` 基础用法。

其中 `scipy.linalg.eigh` 用于解 `K*phi=lambda*M*phi`，其余流程在源码中显式实现。

## R09

适用场景：

- 机械振动教学（模态概念、能量交换）；
- 多自由度系统建模的最小原型；
- 更复杂 FEM/结构动力学前的 sanity check。

不适用场景：

- 大位移非线性振动；
- 强阻尼或随机激励主导问题；
- 需要高维大规模稀疏求解的工程级生产场景。

## R10

`demo.py` 的函数职责：

- `CoupledOscillatorParams`：统一管理物理和数值参数；
- `validate_params`：参数合法性与有限值检查；
- `mass_matrix / stiffness_matrix`：构造 `M/K`；
- `normal_modes`：求正则模态频率与模态向量；
- `rhs`：把二阶方程改写为一阶状态方程；
- `rk4_step`：单步四阶 Runge-Kutta；
- `simulate_rk4`：完整时域仿真；
- `analytic_solution`：根据模态叠加给出解析轨迹；
- `total_energy`：计算能量守恒指标；
- `main`：组织实验、打印结果、执行断言。

## R12

输出指标说明：

- `omega1_rad_s`, `omega2_rad_s`：两模态角频率；
- `freq1_hz`, `freq2_hz`：对应赫兹频率；
- `max_abs_position_error_m`：`x_num` 与 `x_exact` 的最大绝对位移误差；
- `max_abs_velocity_error_mps`：速度误差；
- `relative_energy_drift`：数值积分的相对能量漂移；
- `Trajectory sample`：前若干行 `t, x1, x2, v1, v2`，用于快速人工审计。

## R13

建议最小测试集：

1. 正常参数：验证脚本能稳定输出且误差在阈值内。  
2. 边界参数：`kc=0` 时应退化为两个独立振子。  
3. 异常参数：`m<=0`、`dt<=0`、`steps<=0` 应触发 `ValueError`。  
4. 稳定性检查：减小 `dt` 后，误差与能量漂移应进一步降低。  
5. 物理一致性：交换初始位移符号，轨迹应对应镜像变化（线性系统叠加性）。

## R14

关键可调参数与影响：

- `kc`（耦合刚度）：越大，两个质量的能量交换越明显；
- `k1, k2`：决定各自固有尺度与模态分裂程度；
- `m1, m2`：改变惯性比，影响模态形状；
- `dt`：控制积分精度与运行时间；
- `steps`：决定观测时长，过短可能看不出 beat 现象。

经验上先固定物理参数，再通过减小 `dt` 做收敛检查更稳妥。

## R15

与相关方法对比：

- 纯数值积分（仅 RK4）：
  - 优点：可扩展到阻尼/外力/非线性；
  - 缺点：缺少解析基准时不易判断绝对正确性。
- 纯模态解析：
  - 优点：高精度、物理解释强；
  - 缺点：依赖线性系统假设，不易直接处理复杂非线性。
- 本实现（解析 + RK4 对照）：
  - 把“可解释性”和“可扩展性”结合，适合教学与算法基线验证。

## R16

工程落地建议：

- 将 `M/K/C/F`（质量、刚度、阻尼、外力）模块化，便于从无阻尼扩展到一般二阶系统；
- 长时仿真时监控能量或相空间轨迹，及时发现数值漂移；
- 若自由度提升，可替换为稀疏矩阵与专用本征求解器；
- 把当前断言阈值纳入 CI，作为动力学回归测试。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 固定参数 `m1,m2,k1,k2,kc,dt,steps` 与初值 `x0,v0`，构造实验输入。  
2. `simulate_rk4` 先做参数与初值合法性检查，再构造 `M,K` 以及 `M^{-1}`。  
3. 把二阶方程改写为一阶状态 `s=[x1,x2,v1,v2]`，并在 `rhs` 中计算 `a=-M^{-1}Kx`。  
4. `rk4_step` 对该状态方程执行四次斜率评估（`k1,k2,k3,k4`）并更新到下一时刻。  
5. 迭代 `steps` 次得到数值轨迹 `x_num(t), v_num(t)`。  
6. `analytic_solution` 调用 `scipy.linalg.eigh(K,M)` 得到 `omega, Phi`，把初值投影到模态坐标，再用 `cos/sin` 闭式公式重建 `x_exact(t), v_exact(t)`。  
7. 在 `main` 中计算 `max|x_num-x_exact|`、`max|v_num-v_exact|`，并用 `total_energy` 计算相对能量漂移。  
8. 结果以 `pandas.DataFrame` 打印，同时执行阈值断言，形成“建模 -> 积分 -> 解析对照 -> 自动验收”闭环。
