## R01

磁偶极辐射描述的是**随时间变化的磁偶极矩**在远场辐射电磁能量的机制。对标量形式的轴向磁偶极矩 `m(t)`，常用总辐射功率表达式为：

`P(t) = mu0/(6*pi*c^3) * |m''(t)|^2`

若 `m(t) = m0 cos(omega t)`，时间平均功率为：

`<P> = mu0 * omega^4 * m0^2 / (12*pi*c^3)`

这与电偶极辐射呈现类似的“二阶时间导数驱动辐射”结构。

## R02

该模型是小回路天线、核磁共振辐射估算、等离子体磁扰动辐射近似等问题的基础模块。它在工程上最常见的作用是：

- 快速估算小环天线辐射能力和辐射电阻；
- 判断频率变化对辐射功率的 `omega^4` 强依赖；
- 为更高保真全波仿真提供数量级基线。

## R03

本条目 MVP 解决三个最小闭环问题：

- 从时域 `m(t)` 出发，数值计算 `m''(t)`，再得到 `P(t)` 与 `<P>`；
- 从角分布 `dP/dOmega` 出发做球面积分，验证回总功率；
- 将磁偶极与小回路天线 `m0 = N A I0` 关联，验证辐射电阻公式一致性。

目标是打通“物理定律 -> 数值实现 -> 自检断言”的完整链路。

## R04

`demo.py` 使用的核心方程：

1. 磁偶极辐射瞬时总功率：`P(t) = mu0/(6*pi*c^3) * |m''(t)|^2`。
2. 谐振磁偶极平均功率：`<P> = mu0*omega^4*m0^2/(12*pi*c^3)`。
3. 角分布（轴对称谐振）：`dP/dOmega = mu0*omega^4*m0^2/(32*pi^2*c^3) * sin^2(theta)`。
4. 球面积分：`P = ∫(4pi) dP/dOmega dOmega = 2pi ∫[0,pi] dP/dOmega * sin(theta) dtheta`。
5. 小回路关系：`m0 = N A I0`。
6. 辐射电阻：`Rrad = <P>/Irms^2 = mu0*omega^4*(N*A)^2/(6*pi*c^3)`，并可写成近似
   `Rrad ~= 31171 * (N*A/lambda^2)^2 (ohm)`。

## R05

设时间采样点数为 `Nt`，角向采样点数为 `Ntheta`：

- 二阶差分求 `m''`：`O(Nt)`；
- 功率序列与均值：`O(Nt)`；
- 角分布构造与积分：`O(Ntheta)`；
- 总体时间复杂度：`O(Nt + Ntheta)`；
- 空间复杂度：`O(Nt + Ntheta)`。

## R10

正确性直觉：

1. 若 `m(t)` 为常量，则 `m''(t)=0`，辐射功率应为 0；
2. 对正弦驱动，二阶导数仍为正弦，平均平方可闭式计算；
3. 辐射图样应满足 `sin^2(theta)`，极轴方向为零、赤道方向最大；
4. 对小回路，频率翻倍功率和辐射电阻应放大 16 倍（`f^4`）。

本 MVP 的断言正是围绕这四个物理不变量设计。

## R11

数值稳定与精度策略：

- 要求均匀时间网格，避免差分失真；
- 二阶导采用中心差分，边界使用二阶单边差分；
- 计算均值时剔除边界点，减少端点公式误差影响；
- 角积分使用高分辨率 `theta` 网格，保证积分收敛；
- 误差判断采用相对误差并设置防零 `EPS`。

## R12

关键参数与含义：

- `turns (N)`：线圈匝数；
- `area_m2 (A)`：回路面积；
- `current_amp (I0)`：电流峰值，决定 `m0 = N A I0`；
- `freq_hz/omega`：频率或角频率，功率主导缩放为 `omega^4`；
- `Nt/Ntheta`：时域与角域采样密度。

调参建议：

- 若误差偏大，优先增大每周期采样点；
- 角积分精度不足时提高 `Ntheta`；
- 高频案例应同步加密时间采样，避免差分误差放大。

## R14

常见失效模式：

1. 时间采样过稀，导致 `m''(t)` 数值噪声过大；
2. 非均匀时间网格误用中心差分；
3. 单位错误（`MHz` 当 `Hz`、面积单位混淆）；
4. 将该模型用于非小源或强近场耦合场景；
5. 使用极少角点积分导致“积分不守恒”假象。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `SmallLoop.magnetic_moment_amplitude` 把线圈参数映射为 `m0 = N A I0`。  
2. `central_second_derivative` 在均匀时间网格上计算 `m''(t)`，为辐射功率提供核心输入。  
3. `instantaneous_total_power_from_mddot` 按 `P(t)=mu0/(6*pi*c^3)|m''|^2` 生成时序功率。  
4. `average_power_harmonic_closed_form` 给出谐振情形闭式 `<P>`，与步骤 3 的数值均值对比。  
5. `angular_density_harmonic` 构造 `sin^2(theta)` 方向图的 `dP/dOmega`。  
6. `integrate_over_solid_angle` 对 `dP/dOmega` 做 `4pi` 积分并回到总功率，与闭式交叉校验。  
7. `radiation_resistance_from_power_formula` 与 `radiation_resistance_small_loop_textbook` 分别计算 `Rrad`，再检查二者一致及 `f^4` 缩放。  
8. `main` 顺序执行三个 Demo、打印指标、触发断言，形成可重复的非交互验证流程。  

这里没有把第三方包当黑盒：`numpy` 仅用于数组、差分与积分，磁偶极辐射核心公式和验证逻辑均由源码显式实现。
