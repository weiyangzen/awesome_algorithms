# 天线理论 (Antenna Theory)

- UID: `PHYS-0177`
- 学科: `物理`
- 分类: `电磁学`
- 源序号: `178`
- 目标目录: `Algorithms/物理-电磁学-0178-天线理论_(Antenna_Theory)`

## R01

本条目实现一个“可跑、可核对、不过度包装”的天线理论 MVP：
- 天线模型选用**细线中心馈电偶极子**（默认半波振子，`L = lambda/2`）；
- 通过远场方向函数 `F(theta)` 计算辐射强度方向图；
- 通过数值积分估计方向性系数（Directivity）；
- 用 Friis 公式做自由空间链路接收功率计算。

## R02

问题定义（本 MVP）：
- 输入参数：工作频率、天线电长度 `L/lambda`、角度采样点数、发射功率、链路距离。
- 输出结果：
  - 归一化方向图 `U(theta)/U_max`；
  - 方向性系数 `D`（线性值与 dBi）；
  - 半功率波束宽度（HPBW）；
  - 有效口径 `A_e`；
  - Friis 接收功率 `P_r`（W 与 dBm）。

## R03

核心公式：

1. 细线偶极子远场角向因子（忽略常数前因子）

`F(theta) = [cos((kL/2)cos(theta)) - cos(kL/2)] / sin(theta)`

2. 辐射强度（比例量）

`U(theta) ∝ |F(theta)|^2`

3. 总辐射功率（轴对称）

`P_rad = 2pi * integral_0^pi U(theta) sin(theta) dtheta`

4. 方向性系数

`D = 4pi * U_max / P_rad`

5. Friis 传输方程

`P_r = P_t * G_t * G_r * (lambda/(4piR))^2`

## R04

目标与边界：
- 目标：把“方向图 -> 方向性 -> 链路预算”这条经典天线分析链打通。
- 边界：
  - 只做自由空间远场，不含地面反射、多径、极化失配等工程复杂项；
  - 默认辐射效率取 1（即 `G≈D`），便于聚焦理论主干。

## R05

复杂度分析（`N = theta_samples`）：
- 方向图采样计算：`O(N)`；
- 轴对称积分（Simpson）：`O(N)`；
- 根求解（HPBW，`brentq`）每次代价取决于插值调用，整体近似 `O(log eps)` 次函数评估；
- 总体时间复杂度约 `O(N)`，空间复杂度约 `O(N)`。

## R06

微型数值示例（与脚本同结构）：
- 设 `f=2.4 GHz`，`lambda≈0.1249 m`；
- 半波振子方向性理论值约 `1.64`（约 `2.15 dBi`）；
- 若 `P_t=1 W`，收发天线均取 `G=1.64`，`R=100 m`，则 Friis 给出 `P_r` 在微瓦量级（约 `-30 dBm` 量级）。

这和 `demo.py` 的输出数量级一致。

## R07

为什么这个条目重要：
- 偶极子是很多复杂天线（阵列、口径天线等）的基础构件；
- “方向性 + Friis”是射频链路预算最常见第一性估算框架；
- 对后续学习增益、有效口径、波束宽度、阵列因子有直接衔接价值。

## R08

理论基础与假设：
- 使用远场近似，辐射场满足 `1/r` 衰减主项；
- 偶极子取理想细导体、正弦电流分布模型；
- 方向图对方位角 `phi` 近似无关（线天线轴对称）；
- 链路预算假定自由空间、匹配、同极化且无额外系统损耗。

## R09

适用范围与局限：
- 适用：教学、初步估算、算法验证、参数敏感性演示；
- 不适用：
  - 近场耦合问题；
  - 宽带匹配/输入阻抗精确建模；
  - 有地效应、遮挡、多径的真实传播环境；
  - 强损耗介质或复杂平台共形天线。

## R10

正确性依据（工程可验证版）：
1. 由偶极子解析角向函数得到 `F(theta)`；
2. 用 `|F|^2` 构造相对辐射强度 `U(theta)`；
3. 对 `U(theta)sin(theta)` 在 `theta∈[0,pi]` 数值积分，得到 `P_rad`；
4. 用定义式 `D = 4pi U_max/P_rad` 得到方向性；
5. 将 `D` 代入 Friis，得到接收功率。

该流程与电磁学教材中的解析推导链一致，仅把积分环节改为数值实现。

## R11

误差来源：
- 数值误差：角度离散和积分误差（通过较密采样降低）；
- 模型误差：理想偶极子近似、效率设为 1 的简化；
- 链路误差：Friis 未纳入真实系统损耗（馈线、极化、指向、环境）。

实践上，本 MVP 的结果应被解释为“理论上限附近的基线估计”。

## R12

性能与实现取舍：
- 仅用 `numpy + scipy + pandas`，无重型电磁全波求解器；
- 优先透明可读：每个物理量都有显式公式与对应函数；
- 输出 `pattern_samples.csv` 便于后续画图或外部核验。

## R13

可验证性（`demo.py` 已内置）：
- 输出方向性应接近半波振子常识值（约 `1.64`）；
- 输出 HPBW 应落在合理范围（脚本设置了宽松断言）；
- 自动写出 0°~180° 每 15° 的方向图采样表；
- 无需交互输入，运行结束打印 `All checks passed.`。

## R14

鲁棒性处理：
- 在 `theta≈0, pi` 时对 `sin(theta)` 分母做安全除法，避免数值奇异；
- 对 `U_max<=0` 或非有限值直接报错；
- HPBW 通过“插值 + 有界根查找”获取，避免手工阈值跳点误差；
- 对功率 dB 换算使用下限截断（防止 `log10(0)`)。

## R15

代码结构（对应 `demo.py`）：
- `AntennaConfig`：参数容器；
- `dipole_field_factor`：实现偶极子角向场函数；
- `radiation_intensity`：生成 `U(theta)`；
- `find_half_power_beamwidth`：求 -3 dB 交点与 HPBW；
- `compute_dipole_metrics`：汇总方向性、口径和 Friis 链路指标；
- `build_pattern_table`：构建可读表格并导出 CSV；
- `main`：串联执行与断言检查。

## R16

- 更系统链路：可叠加极化损耗、系统噪声、阴影衰落、快衰落模型；

## R18

源码级算法流程拆解（`compute_dipole_metrics` + `main`，8 步）：
1. `main` 构造 `AntennaConfig`，固定频率、电长度、采样点与链路参数。  
2. `compute_dipole_metrics` 生成 `theta` 均匀网格（避开 0 与 `pi` 的奇点端点）。  
3. 调 `radiation_intensity`，内部先在 `dipole_field_factor` 计算 `F(theta)`，再取平方得到 `U(theta)`。  
4. 归一化得到 `U_norm = U/U_max`，并用 `simpson` 对 `U(theta)sin(theta)` 积分得到 `P_rad`。  
5. 按定义计算 `D = 4piU_max/P_rad`，得到方向性（并默认效率 1 转为增益）。  
6. 调 `find_half_power_beamwidth`，用“线性插值 + `brentq` 根查找”得到左右 -3 dB 角和 HPBW。  
7. 用 `scipy.constants.c` 算波长，再按 Friis 公式得到 `P_r`，同时计算有效口径 `A_e`。  
8. `main` 把结果整理成 `pandas.DataFrame` 采样表，打印指标、保存 `pattern_samples.csv`，最后执行数值断言。  

说明：本实现没有把核心算法托管给黑箱天线库；方向图、积分、波束宽度和链路预算均在源码中逐步展开。
