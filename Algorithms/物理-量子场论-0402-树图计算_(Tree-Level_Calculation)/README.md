## R01

树图计算（Tree-Level Calculation）是微扰量子场论里最基础的一层：
只保留无闭环（loop-free）的费曼图来近似散射振幅。它通常给出最低阶（leading order）结果，是后续圈修正、重整化和精密比较的基线。

本条目用一个可审计的 toy 标量理论演示 `2->2` 散射树图振幅：

- 四点接触图（contact）
- `s/t/u` 三个交换道（exchange）

并把振幅进一步映射到微分截面与总截面。

## R02

要解决的核心问题：
如何把“画费曼图 -> 写振幅 -> 得到可观测截面”这条链路，做成一个最小、可运行、可验证的 Python MVP。

MVP 目标：

1. 给出固定模型下树图振幅的逐项分解；
2. 在一组散射角上计算 `dσ/dΩ`；
3. 数值积分得到总截面 `σ_total`；
4. 给出两个自动校验：
- `t<->u` 交换对称性（同种标量散射）；
- 纯接触项极限下的解析截面对照。

## R03

`demo.py` 的输入输出约定（无交互）：

- 输入（脚本内部配置）：
1. 外线质量 `m`、介子质量 `m_med` 与宽度 `Gamma_med`；
2. 耦合常数 `lambda4`（四点）与 `g3`（三点）；
3. 质心能量 `E_cm`；
4. 角度网格点数 `n_theta`。
- 输出：
1. 每个角度的 `(s,t,u)`、各图贡献、总振幅、`dσ/dΩ` 表格；
2. 数值总截面；
3. 对称性与解析极限误差；
4. 断言通过后打印 `All checks passed.`。

## R04

MVP 使用的数学模型（标量 toy 理论，教学用途）：

1. 树图总振幅（去掉整体 `i` 约定后的约化写法）：
`A_tree = A_contact + A_s + A_t + A_u`

2. 各项定义：
- `A_contact = -lambda4`
- `A_x = g3^2 / (x - m_med^2 + i m_med Gamma_med)`, `x in {s,t,u}`

3. 等质量 `2->2` 运动学（COM）：
- `s = E_cm^2`
- `t = -2 p^2 (1-cos theta)`
- `u = -2 p^2 (1+cos theta)`
- `p^2 = s/4 - m^2`

4. 微分截面：
`dσ/dΩ = |A_tree|^2 / (64 pi^2 s)`（等质量初末态，`p_f/p_i=1`）

5. 总截面：
`σ_total = 2pi * integral_0^pi sin(theta) * dσ/dΩ dtheta`

## R05

设角度采样数为 `N`：

- 振幅逐点计算：`O(N)`；
- 角积分（Simpson）：`O(N)`；
- 总体时间复杂度：`O(N)`；
- 空间复杂度：`O(N)`（保存结果表）。

## R06

算法闭环：

1. 由 `E_cm,m` 构造每个角度点的 `(s,t,u)`；
2. 计算接触图与 `s/t/u` 三个交换图贡献；
3. 求和得到 `A_tree`；
4. 计算 `dσ/dΩ`；
5. 汇总为 `pandas` 表格；
6. 对 `sin(theta) * dσ/dΩ` 做 Simpson 积分得到 `σ_total`；
7. 检查 `theta` 与 `180-theta` 的振幅一致性（`t<->u`）；
8. 在 `g3=0` 极限下，对比解析 `σ = lambda4^2/(16pi s)`。

## R10

正确性直觉：

1. 树图是各独立图振幅的线性叠加；
2. `2->2` 等质量运动学保证 `s+t+u=4m^2`；
3. 截面由 `|A|^2` 决定，因此必须非负；
4. 同种标量下 `t/u` 交换不改物理振幅，故应满足角反射对称；
5. 在 `g3=0` 时退化为常数振幅，可与解析积分精确对照。

## R11

数值稳定性策略：

- 交换道传播子加入有限宽度 `Gamma_med`，避免极点奇异；
- 检查阈值 `E_cm > 2m`，防止非物理动量；
- 使用 Simpson 积分提高角积分稳定性；
- 断言 `dσ/dΩ >= 0` 与关键误差阈值，防止静默失败。

## R12

关键参数与影响：

- `lambda4`：控制各向同性接触背景强度；
- `g3`：控制三交换道的能角依赖结构；
- `m_med`：决定传播子共振位置；
- `Gamma_med`：控制共振峰平滑程度；
- `E_cm`：决定可达 `s` 与角分布形状；
- `n_theta`：影响积分精度与输出分辨率。

调参建议：

- 若想看“纯树图干涉”结构，增大 `g3/lambda4`；
- 若积分噪声偏大，增加 `n_theta`（建议奇数点用于 Simpson）。

## R14

常见失效模式：

1. 把传播子写成错误符号（`x-m^2` 与 `m^2-x` 混淆）；
2. 遗漏 `s/t/u` 某一道贡献；
3. 阈值以下仍强行计算散射；
4. 总截面角积分漏掉 `sin(theta)` 权重；
5. 宽度设为 0 且刚好踩到极点导致数值发散。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `TreeLevelConfig` 固定质量、耦合、宽度、能量与角网格配置。
2. `mandelstam_2to2_equal_mass` 根据 `E_cm,theta,m` 构造 `s,t,u`。
3. `scalar_propagator` 计算 `1/(q^2-m_med^2+i m_med Gamma_med)`。
4. `tree_level_components` 分别计算 `A_contact, A_s, A_t, A_u` 并求和为 `A_tree`。
5. `differential_cross_section` 把 `A_tree` 映射到 `dσ/dΩ`。
6. `build_scattering_table` 在角网格上批量计算并整理成 `pandas` DataFrame。
7. `integrate_total_cross_section` 对 `sin(theta) dσ/dΩ` 做 Simpson 积分得到 `σ_total`。
8. `verify_crossing_symmetry` 与 `contact_only_consistency_check` 分别执行 `t<->u` 对称校验和纯接触解析校验；`main` 汇总打印并断言通过。

说明：第三方库只用于基础数值与表格；树图计算核心（运动学、传播子、分图求和、截面积分）都在源码中逐步展开，没有把物理主流程交给黑盒函数。
