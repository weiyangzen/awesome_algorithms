## R01

半经验质量公式（Semi-Empirical Mass Formula, SEMF，也称 Weizsaecker 公式）用少量“可解释项”近似原子核总结合能：

`B(A, Z) = a_v A - a_s A^(2/3) - a_c Z(Z-1)/A^(1/3) - a_a (A-2Z)^2/A + delta(A,Z)`

其中：

- `A` 是质量数，`Z` 是质子数，`N=A-Z` 是中子数；
- 体积项 `a_v A` 表示短程核力导致的总体束缚；
- 表面项 `-a_s A^(2/3)` 修正表面核子近邻不足；
- 库仑项 `-a_c Z(Z-1)/A^(1/3)` 表示质子静电排斥；
- 对称项 `-a_a (A-2Z)^2/A` 惩罚 `N` 与 `Z` 偏离；
- 配对项 `delta(A,Z)` 体现奇偶效应（偶偶核更稳定，奇奇核更不稳定）。

## R02

SEMF 是液滴模型的核心定量表达，常用于：

- 快速估计核素结合能和质量趋势；
- 解释稳定线（beta-stability valley）的 `N/Z` 演化；
- 粗估裂变/聚变反应的 `Q` 值数量级；
- 为更复杂壳模型或密度泛函计算提供基准对照。

它不是微观第一性原理，但在“低成本 + 可解释”之间提供了高性价比折中。

## R03

本目录 MVP 目标：

1. 实现可直接调用的 SEMF 结合能计算函数；
2. 用一组代表性核素样本拟合系数 `a_v, a_s, a_c, a_a, a_p`；
3. 比较 SciPy 最小二乘与线性回归解的一致性；
4. 计算稳定线近似 `Z*(A)` 与离散最优 `Z`；
5. 扫描 `A` 预测 `B/A` 峰值（铁峰附近）并做自动断言。

## R04

公式细节与离散实现：

1. 配对项定义：
   - 偶偶核：`delta = +a_p / sqrt(A)`
   - 奇奇核：`delta = -a_p / sqrt(A)`
   - 奇 `A` 核：`delta = 0`
2. 预测结合能后，核质量可由
   `M(A,Z)c^2 = Z m_p c^2 + (A-Z)m_n c^2 - B(A,Z)`
   获得（本 MVP 聚焦 `B`，不额外展开原子电子修正）。
3. 稳定线连续近似来自 `dB/dZ = 0`（忽略配对离散跳变）：
   `Z*(A) = [4 a_a A + a_c A^(2/3)] / [8 a_a + 2 a_c A^(2/3)]`。

## R05

设样本数为 `N`，质量数扫描区间长度为 `M`：

- 单次结合能评估：`O(1)`；
- 样本批量预测：`O(N)`；
- 最小二乘拟合（小参数维度）：近似 `O(N)` 到 `O(N * iter)`；
- 稳定线扫描：`O(M * k)`（`k` 为每个 `A` 的小候选 `Z` 数，常数级）。

MVP 的 `N`、`M` 均很小，运行时间通常小于 1 秒。

## R11

数值注意事项：

- 使用 `float64` 以减小拟合误差；
- 小样本拟合时，系数相关性会导致参数可辨识性下降；
- 数据噪声或样本偏置会把误差转移到某些系数；
- 配对项为离散符号函数，连续最优化只能做近似折中。

## R14

常见失效模式：

1. 把 `Z^2` 与 `Z(Z-1)` 混用后又与数据定义不一致；
2. 忘记配对项奇偶符号，导致偶偶/奇奇趋势错判；
3. 用过少样本拟合，参数漂移严重；
4. 把 SEMF 外推到极端轻核或极端滴线区域并过度解读；
5. 单看 MAE 不看物理趋势（稳定线、铁峰）导致“数值看似可用但物理失真”。

## R18

`demo.py` 源码级算法流程（9 步）：

1. `build_reference_dataset` 构建核素样本 `(A, Z, B_exp)` 表。  
2. `pairing_sign` 依据奇偶性返回 `+1/0/-1`，并在 `design_matrix` 中生成配对特征。  
3. `design_matrix` 生成五个显式基函数列（体积、表面、库仑、对称、配对）。  
4. `fit_params_scipy` 用 `scipy.optimize.least_squares` 最小化 `Xc - B_exp`，得到系数。  
5. `fit_params_linear_regression` 用线性回归（或 `numpy.linalg.lstsq` 回退）做独立拟合。  
6. `evaluate_on_dataset` 计算逐核素预测误差并汇总 `MAE/RMSE/max_abs`。  
7. `best_integer_z_for_a` 与 `stable_z_continuous` 在每个 `A` 上寻找最稳定 `Z`，再由 `scan_iron_peak` 扫描 `B/A` 峰值。  
8. `torch_cross_check`（若可用）以张量路径重算预测，验证与 NumPy 路径数值一致。  
9. `main` 打印关键表格与指标，并执行断言（误差上限、拟合一致性、铁峰区间）后退出。

说明：脚本没有把第三方库当黑盒。核心物理表达、特征构造、拟合目标、稳定线搜索与验证条件都在源码中逐步展开。
