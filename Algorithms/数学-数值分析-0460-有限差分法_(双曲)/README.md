# 有限差分法 (双曲)

## R01

有限差分法（双曲）用于求解以“信息沿特征线传播”为核心的双曲型偏微分方程。典型方程是线性平流方程
\[
u_t + a u_x = 0,\quad a\in\mathbb{R}
\]
它描述形状沿速度 \(a\) 平移。本文给出一个最小可运行 MVP：一维显式迎风差分（Upwind）+ 周期边界 + 误差评估。

## R02

问题设定：
- 空间区间：\(x\in[0,1)\)
- 时间区间：\(t\in[0,T]\)
- 初值：\(u(x,0)=u_0(x)\)
- 周期边界：\(u(0,t)=u(1,t)\)

精确解（周期意义下）：
\[
u(x,t)=u_0(x-a t)
\]
因此该问题非常适合验证数值格式的相位误差与数值耗散。

## R03

网格离散：
- 空间网格：\(x_j=j\Delta x,\ j=0,\dots,N-1,\ \Delta x=1/N\)
- 时间网格：\(t^n=n\Delta t,\ n=0,1,\dots,N_t\)
- 网格函数：\(u_j^n\approx u(x_j,t^n)\)

CFL 数定义为
\[
\nu=\frac{a\Delta t}{\Delta x}
\]
对显式双曲格式，\(|\nu|\) 控制稳定性与传播准确性。

## R04

一阶迎风格式：

当 \(a>0\) 时（信息从左向右）：
\[
u_j^{n+1}=u_j^n-\nu\left(u_j^n-u_{j-1}^n\right)
\]

当 \(a<0\) 时（信息从右向左）：
\[
u_j^{n+1}=u_j^n-\nu\left(u_{j+1}^n-u_j^n\right)
\]

实现上可统一为“按速度符号选择上风方向”，并利用循环移位处理周期边界。

## R05

稳定性（线性情形）：
- 迎风显式格式满足 CFL 条件时稳定：\(|\nu|\le 1\)
- 若 \(|\nu|>1\)，数值解通常出现振荡并发散
- 本项目在 `demo.py` 中显式检查 `abs(cfl) <= 1`

工程上通常取 `cfl=0.5~0.9`，在稳定与时间步效率之间折中。

## R06

一致性与收敛性：
- 时间一阶、空间一阶（局部截断误差 \(O(\Delta t,\Delta x)\)）
- 线性稳定 + 一致性可推出收敛（Lax 等价定理）
- 缺点是数值耗散较强，峰值会逐渐“抹平”

因此它是“稳健基线方法”，常作为更高阶 TVD/WENO 方法的对照。

## R07

边界条件处理：
- 本例使用周期边界，适合纯平移测试
- 代码中用 `np.roll` 实现邻点访问，天然满足首尾相接
- 对非周期问题可替换为外推、固定值、特征边界或吸收边界

## R08

复杂度分析：
- 每个时间步仅做常数次向量运算：\(O(N)\)
- 总时间复杂度：\(O(NN_t)\)
- 内存复杂度：\(O(N)\)（若只保留当前/下一步）

MVP 只存当前解与最终解，便于大网格扩展。

## R09

伪代码：

```text
input: u0(x), a, nx, t_end, cfl
dx = 1 / nx
dt = cfl * dx / |a|
n_steps = round(t_end / dt), then dt = t_end / n_steps
nu = a * dt / dx
u = sample u0 on grid
repeat n_steps:
    if a > 0:
        u = u - nu * (u - roll_right(u, 1))
    else:
        u = u - nu * (roll_left(u, 1) - u)
u_exact = periodic_shift(u0, a * t_end)
compute L1, L2, Linf errors
```

## R10

关键参数建议：
- `nx`: 100~1000（网格越密误差越小）
- `cfl`: 建议 `0.8`
- `t_end`: 取 0.5 或 1.0 可观察完整平移
- `a`: 平流速度，正负决定迎风方向

若 `a=0`，方程退化为静态，演化步骤无意义，代码会直接报错提示。

## R11

MVP 默认实验：
- `a=1.0`, `nx=200`, `t_end=0.5`
- 初值：高斯包 + 小幅正弦扰动
- 输出指标：最终 `CFL`、`L1/L2/Linf` 误差、质量守恒误差（离散积分差）

预期现象：
- 数值解整体向右平移约 0.5 个区间长度
- 波峰略变矮（数值耗散）
- 质量守恒误差接近机器精度量级。

## R12

`demo.py` 结构说明：
- `initial_condition(x)`: 构造光滑周期初值
- `upwind_step(u, a, cfl)`: 单步更新
- `solve_advection_upwind(...)`: 时间推进主循环
- `exact_solution_periodic(...)`: 构造精确解
- `compute_error_metrics(...)`: 计算 L1/L2/Linf
- `main()`: 设置参数并打印实验摘要

## R13

运行方式：

```bash
python3 demo.py
```

输出包含：
- 网格与时间离散参数
- 实际 CFL（因步数取整后会微调）

所有内容非交互，一次执行即可完成。

## R14

常见误差来源：
- `cfl > 1` 导致不稳定
- 忘记按 `a` 符号切换上风方向
- 周期位移计算错误（精确解对比失真）
- 时间步取整后未回写 `dt`，导致目标终止时刻偏离

当前实现逐项规避了这些问题。

## R15

可扩展方向：
- 二阶 Lax-Wendroff（降低数值耗散）
- TVD + 限制器（兼顾高分辨率与无振荡）
- WENO（处理激波/间断更稳健）
- 二维线性双曲系统（分裂法或多维迎风）

## R16

验证策略：
- 网格收敛：增大 `nx`，观察误差下降趋势
- 参数扫面：`cfl` 从 0.2 到 0.95，比较耗散程度
- 正负速度：测试 `a>0` 与 `a<0` 两种方向一致性
- 守恒检查：比较初末时刻离散积分

## R17

适用场景与边界：
- 适用：对鲁棒性优先、允许一阶精度的快速基线
- 不适用：强间断且追求高分辨率细节（应考虑高阶非振荡格式）
- 在教学/原型阶段价值高：公式简单、调试成本低、可解释性强

## R18

`demo.py` 的源级算法流（非黑盒）：
1. 构造均匀网格 `x_j`，并调用 `initial_condition` 得到离散初值 `u^0`。
2. 依据 `dx`、目标 `cfl`、速度 `a` 计算初始 `dt`，再按 `t_end` 反推整数步数 `n_steps` 并修正 `dt`。
3. 用 `nu = a*dt/dx` 得到实际 CFL，若 `|nu|>1` 立即报错，防止显式格式失稳。
4. 时间推进循环中，若 `a>=0`，通过 `np.roll(u, 1)` 读取 \(u_{j-1}^n\)；若 `a<0`，通过 `np.roll(u, -1)` 读取 \(u_{j+1}^n\)。
5. 按迎风公式更新每个网格点，形成 `u^{n+1}`，并覆盖当前解 `u`。
6. 循环结束后，使用周期平移 `x_shift=(x-a*t_end)%1.0` 构造精确解 `u_exact`，避免插值黑盒。
7. 逐点计算误差向量 `e=u-u_exact`，输出 `L1/L2/Linf` 三个范数，分别刻画平均误差、能量误差与最大点误差。
8. 计算离散质量 `sum(u)*dx` 的初末差值，检验守恒性质与实现正确性。
