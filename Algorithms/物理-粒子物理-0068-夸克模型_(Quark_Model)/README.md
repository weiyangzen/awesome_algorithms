## R01

夸克模型（Quark Model）的核心思想是：
可观测强子由价夸克组成，介子是 `q\bar{q}`，重子是 `qqq`。在最小构成夸克模型里，强子质量可以近似拆成：

- 组分夸克质量和（constituent masses）
- 自旋相关的超精细相互作用（hyperfine term）

本条目实现一个可运行的最小版本：
对一组常见轻/重介子与重子，拟合少量模型参数并给出质量预测与一致性校验。

## R02

要解决的问题是把“粒子物理概念”转成“可执行算法流程”：

1. 用统一数据结构表示强子（夸克组成、自旋、实验质量）；
2. 把夸克模型公式编码为可计算特征；
3. 在训练集上拟合参数；
4. 在测试集上验证质量误差与物理排序（如 `rho > pi`）；
5. 同时验证电荷与重子数守恒是否与夸克内容一致。

## R03

- 输入（脚本内置）：

3. 训练/测试 MAE、自旋分裂检查、量子数一致性检查；
4. 断言通过后输出 `All checks passed.`。

## R04

MVP 使用的最小构成夸克质量公式：

1. 质量模型：
`M_pred = sum_i m_i + delta_type + A_type * H`

2. 其中：
- `type in {meson, baryon}`
- `H = S_factor * sum_{i<j} 1/(m_i m_j)`

3. 自旋因子：
- 介子：`S=0 -> S_factor=-3/4`, `S=1 -> +1/4`
- 重子：`J=1/2 -> S_factor=-3/4`, `J=3/2 -> +3/4`

4. 量子数计算：
- 电荷 `Q = sum q_i`（反夸克电荷取反）
- 重子数 `B = sum B_i`（夸克 `+1/3`，反夸克 `-1/3`）

该式是教学级近似，不包含完整 QCD 动力学与圈修正。

## R05

设样本数为 `N`，每个强子组分数最多为 `k`（介子 `k=2`，重子 `k=3`）：

- 构造特征：`O(N * k^2)`（配对求和，`k` 很小可视作常数）；
- 最小二乘拟合（4 参数）：`O(N)` 量级；
- 评估与检查：`O(N)`。

总体时间复杂度 `O(N)`，空间复杂度 `O(N)`。

## R06

算法闭环（`demo.py`）：

1. 定义夸克属性与强子样本数据结构；
2. 校验样本合法性（介子必须一夸克一反夸克、重子必须三夸克）；
3. 计算每个样本的基线质量 `sum m_i`；
4. 计算超精细特征 `H`；
5. 构造线性方程组 `X beta = y`（目标是实验质量减基线质量）；
6. 用 `numpy.linalg.lstsq` 拟合 4 个参数；
7. 生成全样本预测表并统计 MAE；
8. 执行物理一致性断言（自旋排序、量子数一致、误差阈值）。

## R07

优点：

- 结构清晰：从“夸克内容 -> 特征 -> 参数 -> 质量预测”逐层透明；
- 计算轻量：仅依赖 `numpy` 和 `pandas`；
- 有物理约束检查，不只是数值拟合。

局限：

- 这是构成夸克模型的简化版，不是第一性原理 QCD；
- 未显式处理 SU(3) 破缺细节、混合态、衰变宽度；
- 参数集小，精度目标是“数量级正确 + 结构正确”，非高精度谱学拟合。

## R09

适用场景：

- 粒子物理教学中的“夸克模型最小可运行演示”；
- 需要快速构建强子质量近似预测 baseline 的任务；
- 后续更复杂模型（如包含更多耦合项）的起点。

不适用场景：

- 要求精密实验拟合（MeV 级精度）；
- 需要完整 QCD 圈修正、格点 QCD 级别严谨计算；
- 涉及复杂混合态与衰变道全局分析。

## R10

正确性直觉：

1. 组分夸克质量和给出强子质量的主尺度；
2. 超精细项对不同自旋态产生分裂（例如向量介子通常重于赝标介子）；
3. 同组分但高总自旋态常更重（如 `Sigma* > Sigma`）；
4. 电荷与重子数由组分可直接求和，是硬约束；
5. 训练/测试误差同时受控，说明模型至少抓住了主导结构。

## R11

数值稳定性与工程策略：

- 参数拟合采用线性最小二乘，避免非线性优化初值敏感；
- 输入样本先做结构校验，防止物理非法数据污染拟合；
- 使用显式断言限制 MAE 与物理排序，避免“数值跑完但物理错误”；
- 输出完整表格，便于定位离群样本误差来源。

## R12

关键参数与影响：

- `delta_meson`、`delta_baryon`：通道基准偏移（吸收未显式建模项）；
- `a_meson`、`a_baryon`：超精细耦合强度，决定自旋分裂幅度；
- `m_u,m_d,m_s,m_c,m_b`：构成夸克质量，决定整体质量尺度与 flavor 依赖。

调参直觉：

- 增大 `a_meson` 会放大介子 `S=1` 与 `S=0` 质量差；
- 增大 `a_baryon` 会放大重子 `J=3/2` 与 `J=1/2` 分裂；
- 改动夸克质量会全局平移并重塑不同 flavor 的相对误差。

## R14

常见失效模式：

1. 把反夸克标记漏掉，导致电荷/重子数错误；
2. 自旋标签填错（例如把 `S=0` 写成 `S=1`），会破坏分裂结构；
3. 数据集中加入不适配该简化模型的复杂态，误差会显著增大；
4. 把拟合和评估样本混淆，导致指标解读失真。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `HadronSample` 编码每个强子的类型、价夸克组成、自旋、实验质量和数据分割标签。  
2. `validate_sample` 检查介子/重子结构约束，避免非法输入进入拟合。  
3. `base_mass_gev` 与 `hyperfine_feature` 计算 `sum m_i` 和 `S_factor * sum 1/(m_i m_j)` 两个核心量。  
4. `build_linear_system` 将每个样本转为线性方程：`observed - base = delta_type + A_type * H`。  
5. `fit_model` 调用 `numpy.linalg.lstsq` 拟合四个参数 `delta_meson, delta_baryon, a_meson, a_baryon`。  
6. `predict_mass_gev` 用拟合参数对全部样本做质量预测；`evaluate_samples` 产出含误差和特征的 `pandas` 表格。  
7. `check_spin_ordering` 验证关键自旋分裂方向（如 `rho > pi`, `Sigma* > Sigma`）；`check_quantum_numbers` 验证电荷与重子数由夸克组分严格求和一致。  
8. `main` 汇总打印参数、明细表、训练/测试 MAE，并执行断言，确保脚本在非交互模式下可重复通过。  

说明：第三方库只用于基础线性代数与表格组织；夸克模型的物理建模（特征构造、通道拆分、量子数计算、一致性校验）都在源码中显式展开，没有交给黑盒物理包。
