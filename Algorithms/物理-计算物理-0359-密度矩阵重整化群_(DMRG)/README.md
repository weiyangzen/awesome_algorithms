## R01

密度矩阵重整化群（Density Matrix Renormalization Group, DMRG）是求解一维强关联量子多体系统基态的经典算法。核心思想是：

1. 把链分成左右块，构造超块（superblock）基态；
2. 对超块基态做部分迹，得到块的约化密度矩阵 `rho`；
3. 保留 `rho` 最大本征值对应的少量基矢（截断）；
4. 在截断后的有效空间继续扩展系统。

本条目给出一个可运行 MVP：用无限系统 DMRG（infinite-system DMRG）求解一维自旋 `1/2` 反铁磁 Heisenberg 链的基态能量。

## R02

DMRG 的工程价值：

- 对 1D 局域哈密顿量，能在远小于全希尔伯特空间维度的条件下达到高精度；
- 把“指数维度爆炸”转化为“受控截断误差”；
- 既可以做教学验证（小系统与精确对角化比对），也能做中等规模链长的快速估计。

在本实现中，链长 `L=20`、保留态数 `m=24`，即可得到稳定的基态能量与极小截断误差。

## R03

`demo.py` 的输入输出（无交互，参数写在脚本内）：

- 输入参数：
1. `target_length`：目标链长（偶数）；
2. `m_keep`：密度矩阵截断后保留态数；
3. `coupling_j`：Heisenberg 耦合常数 `J`。
- 输出结果：
1. DMRG 生长过程表（步数、链长、块维数、超块维数、能量、截断误差）；
2. 目标链长下总能量与每格点能量；
3. 小系统 `L=10` 的 DMRG 与精确对角化能量对比；
4. 校验通过时打印 `All checks passed.`。

## R04

本 MVP 使用的关键公式：

1. 开边界 Heisenberg 哈密顿量：
`H = J * Σ_i (Sx_i Sx_{i+1} + Sy_i Sy_{i+1} + Sz_i Sz_{i+1})`。

2. 用升降算符重写交换项：
`SxSx + SySy = 0.5 * (S+S- + S-S+)`。

3. 超块基态：
`H_super |psi0> = E0 |psi0>`。

4. 左块约化密度矩阵：
`rho_L = Tr_R (|psi0><psi0|)`。

5. 截断准则：
保留 `rho_L` 最大的 `m` 个本征值对应本征矢构成变换矩阵 `U`。

6. 算符投影：
`O' = U^T O U`（本实现为实数表示，因此使用转置）。

7. 截断误差：
`eps_trunc = 1 - Σ_{kept} lambda_i`。

## R05

令局域维数 `d=2`、保留态数为 `m`：

- 扩展后单侧块维度约为 `d*m`（早期增长阶段小于该值）；
- 超块维度约为 `(d*m)^2`；
- 每一步主开销来自稀疏本征求解 `eigsh`，近似随超块维度超线性增长；
- 密度矩阵对角化开销约为 `O((d*m)^3)`；
- 内存主要由超块哈密顿量稀疏表示与投影阶段的稠密小矩阵组成。

因此 DMRG 相比精确对角化（维度 `2^L`）可显著降低大链长计算成本。

## R06

`demo.py` 由四个闭环组成：

- 闭环 A（模型构建）：`initial_block`、`enlarge_block`、`superblock_hamiltonian`；
- 闭环 B（变分求解）：`ground_state` 用 `eigsh` 取超块基态；
- 闭环 C（重整化截断）：`truncate_left_block` 生成约化密度矩阵并投影算符；
- 闭环 D（可验证性）：`exact_heisenberg_energy` 对 `L=10` 做精确对角化对照。

## R07

优点：

- 代码短小，完整呈现 DMRG 的“扩展-求基态-密度矩阵截断”主链路；
- 稀疏求解 + 小规模投影，能在普通 CPU 上快速运行；
- 自带小系统精确解校验，不是只输出单一路径结果。

局限：

- 采用无限系统 DMRG，未实现有限系统 sweeping；
- 未显式使用量子数分块（如 `Sz` 守恒扇区），效率仍有提升空间；
- 投影步骤为教学可读性，使用了 `toarray()` 的稠密化方式，不适合超大 `m`。

## R10

正确性直觉：

1. 超块基态包含左右块纠缠信息；
2. 约化密度矩阵的主本征空间携带对目标态最重要的信息；
3. 保留最大本征权重对应“最相关子空间”，可在固定维度下最大限度保真；
4. 重复执行后，块基不断贴近热力学极限下的有效低能子空间；
5. 与小系统精确对角化一致，说明实现链路正确。

## R11

数值稳定策略：

- `ground_state` 对超小矩阵走 `numpy.linalg.eigh`，其余使用 `eigsh`；
- 对约化密度矩阵做对称化 `0.5*(rho+rho^T)`，降低浮点噪声；
- 归一化基态向量并检查零范数异常；
- 截断误差强制非负：`max(0, 1-sum_kept)`；
- 通过 `L=10` 精确对角化误差阈值断言保证回归稳定性。

## R12

关键参数：

- `target_length`：目标链长，越大越接近热力学极限；
- `m_keep`：保留态数，越大误差越小但开销越高；
- `coupling_j`：耦合强度，改变能量尺度但不改变算法流程。

调参建议：

- 先固定 `L`，增大 `m_keep` 观察截断误差 `truncation_error` 是否收敛；
- 再提升 `L` 观察 `energy_per_site` 的有限尺寸收敛趋势；
- 若误差不降，优先增加 `m_keep`，其次再加链长。

## R14

常见失效模式：

1. `length` 不是偶数或过小，导致生长流程定义无效；
2. `m_keep` 太小导致截断误差积累，能量明显偏高；
3. `eigsh` 在病态参数下收敛慢或失败；
4. 若把 `m_keep` 设得很大，投影阶段稠密化可能导致内存压力；
5. 仅使用 infinite-system DMRG 时，某些模型需要 finite sweeping 才能进一步降能。

## R18

`demo.py` 源码级算法流程（9 步）：

1. `initial_block` 建立 1 个格点的初始块，包含 `H` 与边界算符 `S+ / S- / Sz`。  
2. `enlarge_block` 把块向右扩展 1 个格点：`H' = H⊗I + H_bond`，并更新新边界算符。  
3. `superblock_hamiltonian` 把左右扩展块拼成超块：`H_super = H_L⊗I + I⊗H_R + H_center`。  
4. `ground_state` 对 `H_super` 求最低本征对（`eigsh` 或小维 `eigh`），得到 `E0, psi0`。  
5. `truncate_left_block` 将 `psi0` reshape 成左右矩阵，构造 `rho_L = psi psi^T` 并本征分解。  
6. 在 `rho_L` 本征谱中选最大 `m_keep` 权重的本征矢作为变换基 `U`，计算截断误差。  
7. `project_operator` 把 `H` 和边界算符投影到新基：`O' = U^T O U`，得到新的重整化块。  
8. `infinite_dmrg` 循环执行步骤 2-7，直到达到目标链长，同时用 `pandas.DataFrame` 记录过程。  
9. `main` 额外调用 `exact_heisenberg_energy` 做 `L=10` 精确对角化对照并断言误差阈值。  

说明：`scipy` 仅用于稀疏本征求解与稀疏矩阵基础运算；DMRG 的关键逻辑（扩展、超块构造、密度矩阵截断、投影更新）都在源码中显式实现，非黑盒调用。
