# 迎风格式

- UID: `MATH-0464`
- 学科: `数学`
- 分类: `数值分析`
- 源序号: `464`
- 目标目录: `Algorithms/数学-数值分析-0464-迎风格式`

## R01

迎风格式（Upwind Scheme）是求解一维对流方程的经典有限差分方法。它根据信息传播方向选择空间差分方向，核心思想是“沿着来流方向取差分”，因此比中心差分更稳定、物理可解释性更强。

## R02

目标 PDE 取线性常系数对流方程：

\[
\frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = 0,
\]

其中速度 `a` 为常数。该方程描述量 `u` 以速度 `a` 沿空间平移。

## R03

网格记号：

- 空间网格：`x_j = j\Delta x`
- 时间网格：`t^n = n\Delta t`
- 网格函数：`u_j^n \approx u(x_j, t^n)`
- CFL 数：`\nu = a\Delta t / \Delta x`

迎风思想：

- 若 `a > 0`，信息从左向右传播，使用后向差分 `u_j - u_{j-1}`。
- 若 `a < 0`，信息从右向左传播，使用前向差分 `u_{j+1} - u_j`。

## R04

离散格式写成分段形式：

- `a > 0`：
\[
u_j^{n+1} = u_j^n - \nu\left(u_j^n-u_{j-1}^n\right)
\]

- `a < 0`：
\[
u_j^{n+1} = u_j^n - \nu\left(u_{j+1}^n-u_j^n\right)
\]

统一写法（推荐实现）：

\[
u_j^{n+1} = u_j^n - \nu^+ (u_j^n-u_{j-1}^n) - \nu^- (u_{j+1}^n-u_j^n)
\]

其中 `\nu^+=max(\nu,0)`, `\nu^-=min(\nu,0)`。

## R05

局部截断误差：时间一阶、空间一阶，因此整体是 `O(\Delta t + \Delta x)` 的一阶方法。它的优点是稳定性和单调性；代价是数值耗散较明显，间断会被涂抹。

## R06

Von Neumann 稳定性条件（线性常系数场景）：

\[
|\nu| \le 1
\]

即 `|a|\Delta t/\Delta x <= 1`。这也是代码里的 CFL 约束。

## R07

收敛性结论（Lax 等价定理框架）：

- 该格式与对流方程一致（consistency）
- 在 `|\nu|<=1` 下稳定（stability）
- 因而收敛（convergence）

对光滑解常观察到接近一阶收敛率。

## R08

边界处理：

- 本目录 MVP 采用周期边界 `u_{-1}=u_{N-1}, u_N=u_0`，对应“环形域”。
- 实现上可用数组循环移位（`numpy.roll`）高效完成。

## R09

算法伪代码：

1. 给定 `u^0`, `a`, `\Delta x`, `\Delta t`, 总步数 `N_t`。
2. 计算 `\nu = a\Delta t/\Delta x` 并检查 `|\nu|<=1`。
3. 计算 `\nu^+=max(\nu,0)` 与 `\nu^-=min(\nu,0)`。
4. 每个时间步：
   - `u_left = roll(u, +1)`
   - `u_right = roll(u, -1)`
   - `u = u - \nu^+(u-u_left) - \nu^-(u_right-u)`
5. 返回终态。

## R10

复杂度：

- 时间复杂度：`O(N_x N_t)`
- 空间复杂度：`O(N_x)`（原地更新时）

## R11

误差与数值特性：

- 优点：
  - 对流方向正确，稳定区间明确
  - 对间断不会产生中心差分那种非物理振荡
- 缺点：
  - 数值黏性较强，波峰会衰减
  - 对高频成分耗散明显

## R12

参数建议：

- `CFL` 推荐在 `0.6 ~ 0.9`，常用 `0.8`。
- 当追求更高精度时，先加密网格（减小 `\Delta x`），再考虑高阶迎风类格式（如 MUSCL/WENO）。

## R13

`demo.py` 包含两组实验：

- 光滑初值收敛实验：`sin(2πx)+0.5sin(4πx)`，比较数值解与解析平移解，输出 `L1/Linf` 误差与估计阶。
- 间断初值单调性观察：方波初值，统计最小值/最大值变化，验证迎风格式不会产生明显过冲。

## R16

适用边界：

- 适用于对流主导、速度方向明确的问题。
- 对扩散占优或强非线性守恒律，需组合更多机制（Riemann 求解、限制器、时间高阶推进等）。

## R17

可扩展方向：

- 时间推进：从显式 Euler 升级到 SSP-RK2/3。
- 空间离散：从一阶迎风升级到二阶 TVD/MUSCL。
- 方程类型：从线性对流扩展到 Burgers 方程或浅水方程通量离散。

## R18

`demo.py` 的源码级算法流程（非黑箱）如下：

1. **网格与初值构造**：根据 `N_x` 生成周期网格 `x`，并构造光滑/间断初值数组 `u0`。
2. **时间步确定**：先按目标 `CFL` 给出候选 `dt`，再用总时间 `T` 反推整数步数 `N_t` 并修正 `dt`，保证正好推进到 `T`。
3. **CFL 安全检查**：计算 `\nu=a dt/dx`，若 `|\nu|>1` 直接抛错，阻止不稳定计算。
4. **迎风系数拆分**：计算 `\nu^+` 与 `\nu^-`，把“按速度方向选差分”转换成统一向量表达式。
5. **邻点提取**：每一步用 `roll` 得到左邻点 `u_{j-1}` 与右邻点 `u_{j+1}`，天然满足周期边界。
6. **显式更新**：执行 `u <- u - \nu^+(u-u_left) - \nu^-(u_right-u)` 完成一层时间推进。
7. **解析解对照**：利用平移解析解 `u(x,t)=u0((x-a t) mod L)` 计算终态真值。
8. **误差与阶评估**：输出 `L1/Linf` 误差，并对相邻网格做 `log2(e_h/e_{h/2})` 估计收敛阶。
9. **间断行为检查**：对方波实验输出 min/max 对比，展示迎风格式“稳但有耗散”的核心特征。
