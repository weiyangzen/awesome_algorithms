## R01

主方程（Master Equation）用于描述离散状态随机系统的概率随时间演化。对连续时间马尔可夫过程，若 `p_i(t)` 表示处于状态 `i` 的概率，则有：

`dp_i/dt = sum_{j!=i} [w_{i<-j} p_j - w_{j<-i} p_i]`

矩阵形式写为：

`dp/dt = K p`

其中 `K` 是生成元（generator）矩阵，满足列和为 0（在本实现的列向量约定下）。

## R02

本目录的目标是实现一个可运行、可审计、无黑盒依赖的最小 MVP：

- 构建一个有限状态、可逆（满足局域细致平衡）的跃迁模型；
- 用 RK4 数值积分主方程；
- 用特征分解构造“解析参考解”做误差对照；
- 自动校验概率守恒、平衡分布收敛和熵产生非负性。

## R03

`demo.py` 采用 4 状态环形网络作为示例系统：

- 状态能级：`E = [0.0, 0.35, 0.9, 1.4]`；
- 拓扑：仅与左右相邻状态连通（环）；
- 速率构造：
  `w(i->j) = gamma * exp[-0.5 * beta * (E_j - E_i)]`；
- 对应局域细致平衡：
  `w(i->j)/w(j->i) = exp[-beta*(E_j-E_i)]`。

因此平衡分布应是玻尔兹曼分布 `pi_i ∝ exp(-beta E_i)`。

## R04

核心方程与实现关系：

1. 速率矩阵约定：`W[dst, src] = w(src->dst)`。
2. 生成元构造：
   - `K[i,j] = W[i,j] (i!=j)`
   - `K[j,j] = -sum_{i!=j} W[i,j]`
3. 概率演化：`p(t+dt)` 由 RK4 近似。
4. 精确参考：
   - `p(t) = V exp(Lambda t) V^{-1} p(0)`（`K=V Lambda V^{-1}`）。
5. 诊断量：
   - `L1(p(t), pi)`、`KL(p(t)||pi)`；
   - 熵产生率 `sigma = sum_{i<j} J_ij * ln(F_ij/F_ji)`。

## R05

复杂度分析（`n` 为状态数，`T` 为时间步数）：

- 构建速率和生成元：`O(n^2)`；
- RK4 每步需若干次矩阵向量乘：`O(n^2)`，总计 `O(T*n^2)`；
- 特征分解一次：`O(n^3)`，随后每个时间点重建 `p(t)` 约 `O(n^2)`；
- 总体在本 MVP（`n=4`）下开销很小，适合教学和验证。

## R06

运行后会输出：

- 玻尔兹曼平衡分布 `pi_eq` 与由 `K` 零特征向量提取的平衡分布对照；
- 若干时间快照（`p0..p3`、`l1_to_eq`、`kl_to_eq`、`entropy_prod`、`rk4_exact_linf`）；
- 汇总校验项：列和残差、细致平衡残差、最终收敛误差、RK4-精确解误差。

## R10

正确性直觉：

1. 生成元列和为 0，保证总概率守恒。
2. 局域细致平衡保证玻尔兹曼分布为平衡态。
3. 若积分器实现正确，`p(t)` 应从任意初值收敛到平衡分布。
4. 数值解应与特征分解参考解一致（误差受时间步控制）。
5. 可逆链下熵产生率应非负，`KL(p||pi)` 应不增（允许微小数值扰动）。

## R11

数值稳定性处理：

- 每步 RK4 后调用 `normalize_prob` 做裁剪+归一化，避免浮点误差导致微负概率；
- 用 `KL` 与 `L_inf` 双指标监控演化质量；
- 断言阈值设置为“严格但可通过”的数量级，便于自动化验证；
- 对对数项加入微小 `eps`，避免 `log(0)`。

## R12

关键参数（`MasterEquationConfig`）：

- `n_states`：状态数；
- `beta`：逆温度，控制平衡分布尖锐程度；
- `gamma`：速率整体时间尺度；
- `energies`：离散能级；
- `initial_prob`：初始概率；
- `dt`：时间步长；
- `n_steps`：积分步数（总时长 `dt*n_steps`）。

调参建议：

- 若 `rk4_exact_linf` 偏大，先减小 `dt`；
- 若收敛不充分，增大 `n_steps`；
- 通过修改 `energies/beta` 可观察更强或更弱的平衡偏置。

## R14

常见失效模式：

1. 速率矩阵下标方向写反，导致 `K` 构造错误；
2. 漏掉对角项 `-sum(outgoing)`，破坏概率守恒；
3. 时间步过大导致数值振荡或负概率；
4. 直接把特征分解复数舍入处理不当，造成伪误差；
5. 误把“可逆链结论”外推到非平衡驱动链。

## R18

`demo.py` 源码级算法流程（8 步，非黑盒）：

1. `MasterEquationConfig` 给出状态数、能级、`beta/gamma`、初始分布与积分时间网格。  
2. `build_ring_adjacency` 生成 4 状态环拓扑；`build_rate_matrix` 按局域细致平衡构造 `W[dst,src]`。  
3. `build_generator` 将 `W` 转为主方程生成元 `K`，并设置对角项为负外流总和。  
4. `boltzmann_distribution` 计算理论平衡分布；`stationary_distribution_from_generator` 从 `K` 的零特征向量提取数值平衡分布并对照。  
5. `solve_master_rk4` 用显式 RK4 迭代 `dp/dt=Kp`，每步做概率归一化投影以抑制浮点漂移。  
6. `solve_master_exact_eig` 通过 `p(t)=V exp(Lambda t)V^{-1}p(0)` 逐时刻重建参考解，作为积分误差标尺。  
7. `build_observation_table` 汇总 `p_i(t)`、`L1`、`KL`、熵产生率与 RK4-精确解误差，并在 `main` 打印关键时间快照。  
8. `main` 执行断言：守恒残差、细致平衡残差、平衡分布一致性、最终收敛误差、KL 单调性与熵产生非负性，全通过后输出 `All checks passed.`。

第三方库说明：`numpy` 仅用于数组/线性代数，`pandas` 仅用于结果表格展示；主方程建模、积分、诊断和自检均在源码内显式实现，没有调用现成黑盒主方程求解器。
