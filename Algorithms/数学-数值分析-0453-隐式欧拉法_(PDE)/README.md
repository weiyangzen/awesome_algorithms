# 隐式欧拉法 (PDE)

- UID: `MATH-0453`
- 学科: `数学`
- 分类: `数值分析`
- 源序号: `453`
- 目标目录: `Algorithms/数学-数值分析-0453-隐式欧拉法_(PDE)`

## R01

隐式欧拉法（Backward/Implicit Euler）是处理抛物型 PDE 的经典时间离散方法。和显式欧拉相比，它把未知量放在下一时刻，单步需要解线性系统，但换来更强的稳定性。

本条目 MVP 选用一维热方程作为载体：
- 时间推进用隐式欧拉；
- 空间离散用二阶中心差分；
- 每步形成三对角线性系统，用 Thomas 算法直接求解。

## R02

`demo.py` 求解的问题：

- PDE：`u_t = alpha * u_xx`, `x in (0,1)`, `t>0`
- 边界：`u(0,t)=0, u(1,t)=0`（齐次 Dirichlet）
- 初值：`u(x,0)=sin(pi x)`

该问题有解析解：
`u(x,t)=exp(-alpha*pi^2*t)*sin(pi x)`，可用于数值误差验证。

## R03

离散网格与符号：
- 空间网格 `x_i=i*dx`, `i=0,...,N-1`；
- 时间层 `t^n=n*dt`；
- `r = alpha*dt/dx^2`。

隐式欧拉 + 中心差分得到内点离散方程：
`-r*u_{i-1}^{n+1} + (1+2r)*u_i^{n+1} - r*u_{i+1}^{n+1} = u_i^n`。

因此每一步都需要解一个三对角线性系统。

## R04

线性系统结构（内点维度 `N-2`）：
- 下对角：`-r`
- 主对角：`1+2r`
- 上对角：`-r`

齐次 Dirichlet 边界在本算例下恒为零，因此右端项就是上一时刻内点值 `u^n[1:-1]`，不需要额外边界修正项。

## R05

Thomas 算法（追赶法）在本实现中的作用：
1. 前向消元，逐行消掉下对角元素；
2. 回代求解，得到新时刻内点解。

复杂度优势：
- 时间复杂度 `O(N)`；
- 额外空间 `O(N)`。

与通用稠密求解 `O(N^3)` 相比，三对角结构可显著降本。

## R06

正确性验证采用“解析解对照 + 物理量趋势”：
- 误差指标：`L2` 与 `Linf`；
- 能量指标：离散能量 `E=dx*sum(u^2)`。

热方程扩散会衰减振幅，正常情况下：
- `max|u|` 随时间下降；
- 离散能量整体不增；
- 数值解与解析解误差保持在合理范围。

## R07

复杂度分析：
- 单步：构造 RHS + Thomas 求解，`O(N)`；
- 总步数为 `steps`，总复杂度 `O(steps*N)`；
- 存储主变量和三对角系数，空间复杂度 `O(N)`。

这里 `N=nx`，本示例默认 `nx=101`，计算成本很低。

## R08

稳定性与精度特征：
- 隐式欧拉对线性扩散问题 A-稳定，时间步不受显式 CFL 限制；
- 时间精度一阶 `O(dt)`；
- 空间精度二阶 `O(dx^2)`。

因此在工程上常见策略是：
- 先用较大 `dt` 获取稳定结果；
- 再根据误差要求细化 `dt` 与 `dx`。

## R09

本 MVP 的边界与局限：
- 只实现 1D、常系数、齐次 Dirichlet 条件；
- 未实现非线性扩散、变系数和高维情形；
- 未引入自适应时间步与网格加密。

定位是“算法流程清晰可审计”的基线实现，而不是通用 PDE 工具箱。

## R10

`demo.py` 主要函数分工：
- `make_grid`：构建 1D 网格；
- `initial_condition` / `exact_solution`：初值与解析解；
- `assemble_tridiagonal`：组装三对角矩阵系数；
- `thomas_solve`：显式实现追赶法；
- `implicit_euler_heat`：时间推进主循环；
- `collect_snapshot`：计算误差、能量、幅值；
- `run_checks`：自动一致性检查；
- `print_report`：输出可读报告。

## R11

运行方式：

```bash
cd Algorithms/数学-数值分析-0453-隐式欧拉法_(PDE)
python3 demo.py
```

脚本无交互输入，默认直接输出配置、采样误差表和收尾检查结果。

## R12

输出字段说明：
- `step/time`：离散步与物理时间；
- `L2_error`：与解析解的均方根误差；
- `Linf_error`：最大绝对误差；
- `energy`：离散能量 `dx*sum(u^2)`；
- `max|u|`：当前最大振幅。

最后会给出 `Final L2/Linf error` 并打印 `All checks passed.`。

## R13

建议最小实验：
- 收敛趋势：固定 `alpha, final_time`，比较 `nx=51/101/201`；
- 时间误差：固定 `nx`，比较 `dt_target=0.004/0.002/0.001`；
- 退化场景：`final_time=0` 检查是否返回初值；
- 参数健壮性：测试非法参数是否触发 `ValueError`。

## R14

关键参数与影响：
- `nx`：空间分辨率；
- `alpha`：扩散系数，越大衰减越快；
- `final_time`：总演化时间；
- `dt_target`：目标时间步，决定 `steps` 与误差级别；
- `snapshot_count`：报告采样密度。

虽然隐式格式稳定，但过大的 `dt` 仍会降低精度。

## R15

和常见替代方法对比：
- 显式欧拉：实现更简单，但受稳定步长严格约束；
- Crank-Nicolson：时间二阶，精度更高，但实现和分析更复杂；
- 隐式欧拉：最稳健、流程直接，是刚性扩散问题常见起点。

本条目选择隐式欧拉，强调“稳定 + 透明 + 易扩展”。

## R16

典型应用场景：
- 传热与扩散过程的原型仿真；
- 刚性抛物型 PDE 的稳健时间推进；
- 作为更复杂隐式时间积分器（CN、BDF）的教学起点；
- 作为多物理场耦合中的扩散子步基线。

## R17

可扩展方向：
- 扩展到 2D/3D（稀疏矩阵 + 迭代求解器）；
- 支持非齐次边界和源项 `f(x,t)`；

## R18

源码级算法流（`demo.py`，8 步）：
1. `main` 设定 `nx/alpha/final_time/dt_target`，调用 `implicit_euler_heat`。
2. `make_grid` 生成 `[0,1]` 均匀网格，`initial_condition` 构造 `sin(pi x)` 初值，并施加边界点为 0。
3. 按 `dt_target` 计算离散步数 `steps` 和实际 `dt`，得到 `r=alpha*dt/dx^2`。
4. `assemble_tridiagonal` 构造三对角系数 `(-r, 1+2r, -r)`，对应隐式欧拉离散系统。
5. 每个时间步把上一层内点 `u[1:-1]` 作为 RHS，调用 `thomas_solve` 做前向消元与回代，得到 `u^{n+1}` 内点。
6. 回写新解并重置边界，随后检查是否出现 `NaN/Inf`。
7. 在采样步调用 `collect_snapshot`，计算 `L2/Linf` 误差、离散能量和 `max|u|`。
8. 时间循环结束后 `run_checks` 验证能量不增和最终误差阈值，再由 `print_report` 打印结构化结果。

说明：核心离散与线性求解均在源码中展开实现，没有调用黑盒 PDE 求解器。
