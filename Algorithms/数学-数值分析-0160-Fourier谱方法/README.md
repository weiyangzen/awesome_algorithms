# Fourier谱方法

## R01

Fourier 谱方法（Fourier Spectral Method）用于求解**周期边界**上的微分方程。  
核心思想是把函数表示成有限 Fourier 模态之和，把“微分算子”转成频域里的代数乘法：
- 一阶导数对应乘以 `ik`；
- 二阶导数对应乘以 `-(k^2)`。

本目录给出一个最小可运行 MVP，覆盖三件事：
- 谱导数计算；
- 周期 Poisson 方程 `-u_xx=f` 求解；
- 1D 热方程 `u_t = nu*u_xx` 的频域推进。

## R02

问题设置（本 MVP）：
- 空间区间：`x ∈ [0, L)`，默认 `L = 2π`；
- 网格：`N` 个等距点，`x_j = j*L/N`；
- 周期条件：`u(0)=u(L)`，且各阶导数周期一致；
- 数值工具：`numpy`（FFT 与基础数组运算）。

输出目标：
- 对已知解析函数，验证谱导数误差；
- 对已知解析解构造的 Poisson 右端项，验证反演误差；
- 对可解析热方程初值，验证时域演化误差。

## R03

Fourier 截断表示（离散视角）：

`u(x_j) <-> û_k,  k = -N/2,...,N/2-1`

使用离散 FFT 对偶关系（由 `numpy.fft.fft/ifft` 实现）：
- 正变换：`û_k = FFT(u_j)`；
- 逆变换：`u_j = IFFT(û_k)`。

对谱方法最关键的算子映射：
- `d/dx  <->  ik`；
- `d²/dx² <-> -(k²)`。

因此：
- 谱导数：`u_x = IFFT((ik) * û)`；
- 周期 Poisson `-u_xx=f`：`k != 0` 时 `û = f̂/(k²)`，`k=0` 由零均值约束固定。

## R04

算法主流程（统一模板）：
1. 生成均匀周期网格 `x`；
2. 构造波数数组 `k = 2π*fftfreq(N, d=L/N)`；
3. 把实空间变量做 FFT 得到频谱；
4. 在频域应用代数算子（乘 `ik`、除 `k²`、乘扩散因子）；
5. IFFT 回到实空间；
6. 与解析解做 `L2` 与 `L∞` 误差验证并断言阈值。

## R05

`demo.py` 的核心数据结构：
- `SpectralGrid`（`dataclass`）：
  - `N`：网格点数；
  - `L`：区间长度；
  - `x`：物理网格；
  - `k`：对应波数（与 FFT 排序一致）。
- `dict[str, float | int]`：每个实验返回误差指标与配置。

该设计保证代码短小、可读、可复用，不引入额外框架。

## R06

正确性要点：
- 频域微分等价于 Fourier 基函数求导：`d/dx e^{ikx} = ik e^{ikx}`；
- Poisson 反演中 `k=0` 模态不可除，必须单独处理（设为 0 等价固定零均值解）；
- 热方程在频域每个模态满足常微分方程  
  `dû_k/dt = -nu*k²*û_k`，可用精确因子 `exp(-nu*k²*dt)` 推进；
- 若初值由有限模态组成，谱法可达到接近机器精度的误差。

## R07

复杂度（`N` 点）：
- 一次 FFT / IFFT：`O(N log N)`；
- 频域逐点乘除：`O(N)`；
- 谱导数或 Poisson 一次求解：总体 `O(N log N)`；
- 热方程 `T` 个时间步：`O(T * N + N log N)`（每步只做频域逐点缩放，FFT 仅首尾各一次）。

## R08

边界与异常处理：
- `N <= 0`：抛出 `ValueError`；
- `L <= 0`：抛出 `ValueError`；
- `dt <= 0` 或 `t_end < 0`：抛出 `ValueError`；
- Poisson 输入若非零均值，本实现会先减去均值再反演，避免不可解模态污染；
- 所有输入数组长度必须与 `N` 一致。

## R09

MVP 取舍说明：
- 只做 1D、周期边界、均匀网格；
- 不做 Chebyshev 非周期谱法；
- 不做非线性项去混叠（`2/3` 规则）；
- 目标是“透明的谱方法基线”，而不是通用 PDE 框架。

## R10

`demo.py` 函数分工：
- `build_grid`：构造 `x,k`；
- `spectral_derivative`：计算任意阶（默认 1 阶）谱导数；
- `solve_periodic_poisson`：求解 `-u_xx=f` 的零均值周期解；
- `solve_heat_equation`：频域精确模态推进；
- `l2_error`：`L2` 误差；
- `run_*_case`：三个可验证实验；
- `main`：组织运行、打印结果、执行断言。

## R12

输出字段解释：
- `N`：网格分辨率；
- `l2_error`：离散 `L2` 误差；
- `linf_error`：最大绝对误差；
- `steps`：热方程时间步数；
- `t_end`：总演化时间。

判定标准（`demo.py` 内部断言）：
- 导数误差和 Poisson 误差应达到高精度（`~1e-10` 量级或更好）；
- 热方程误差应显著小于工程阈值。

## R13

最小测试集（内置）：
1. 谱导数测试  
   `u(x)=sin(3x)+0.2cos(5x)`，对比解析 `u_x`。
2. 周期 Poisson 测试  
   设真解 `u_true=sin(2x)+0.3cos(4x)`，构造 `f=-u_true_xx` 后反求 `u`。
3. 热方程测试  
   `u0=sin(3x)+0.5cos(6x)`，与解析衰减解比较。

这三类覆盖了谱方法最常见的“导数-椭圆-抛物”基础模块。

## R14

调参建议：
- `N`：优先增大到 64/128/256 观察误差衰减；
- `dt`：本实现每步采用频域精确衰减，稳定性不受 CFL 限制，但过大步长会降低时间采样精细度；
- `t_end`：用于控制模式衰减程度，值越大高频衰减越明显。

经验上：对光滑周期函数，误差随 `N` 增长通常呈指数级下降（谱精度特征）。

## R15

与常见方法对比：
- 对比有限差分：
  - 差分局部、实现直观；
  - Fourier 谱法对光滑周期问题精度更高（相同 `N` 下常明显更优）。
- 对比有限元：
  - 有限元更擅长复杂几何与非均匀网格；
  - Fourier 谱法更适合规则周期域与高精度需求。
- 对比纯黑箱 PDE 求解器：
  - 黑箱开发快；
  - 本 MVP 更透明，便于教学与二次扩展。

## R16

应用场景：
- 周期流体/湍流的伪谱离散基础模块；
- 周期信号平滑与高阶导数计算；
- 周期边界下的 Poisson/Helmholtz 子问题；
- 作为更复杂谱元、伪谱时间推进器的入门基线。

## R18

源码级算法流（对应 `demo.py`，8 步）：
1. `main` 先调用 `build_grid(N, L)` 生成均匀网格 `x` 与波数 `k`。  
2. 在导数案例中，`spectral_derivative` 对 `u(x)` 执行 `û = FFT(u)`。  
3. 同函数把频谱乘上 `(ik)^order`（导数算子在频域的代数形式）。  
4. 再做 `IFFT` 回到物理空间，取实部得到 `u_x` 数值解并计算误差。  
5. 在 Poisson 案例中，`solve_periodic_poisson` 先对 `f` 去均值，再做 `f̂ = FFT(f)`。  
6. 对 `k != 0` 模态执行 `û = f̂/(k²)`，并把 `k=0` 模态置零以固定零均值解。  
7. 在热方程案例中，先算 `û0 = FFT(u0)`，每一步做 `û <- û * exp(-nu*k²*dt)`，最后 `IFFT` 得到 `u(x,t_end)`。  
8. `main` 汇总三组 `L2/L∞` 误差并断言阈值，确保算法链路“离散化-频域算子-逆变换-验证”闭环成立。
