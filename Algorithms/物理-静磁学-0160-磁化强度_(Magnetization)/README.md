# 磁化强度 (Magnetization)

- UID: `PHYS-0159`
- 学科: `物理`
- 分类: `静磁学`
- 源序号: `160`
- 目标目录: `Algorithms/物理-静磁学-0160-磁化强度_(Magnetization)`

## R01

磁化强度 `M` 描述单位体积材料内部等效磁偶极矩总和，是材料响应磁场 `H` 的核心场量（单位 `A/m`）。

本条目 MVP 采用两类可计算模型。

- 线性各向同性模型：`M = chi * H`
- 饱和无磁滞近似：`M = Ms * tanh(H / H0)`
- 磁感应强度关系：`B = mu0 * (H + M)`

输入与输出定义。

- 输入：
  - 磁场样本 `H`（一维数组，单位 `A/m`）
  - 线性材料参数 `chi`
  - 饱和材料参数 `Ms, H0`
  - 可选带噪观测数据 `(H_i, M_i)`（用于参数反演）
- 输出：
  - 磁化曲线 `M(H)`
  - 磁感应曲线 `B(H)`
  - 拟合参数估计值 `chi_hat, Ms_hat, H0_hat`
  - 数值一致性与阈值检查结果

## R02

磁化概念的历史脉络可简述为：

- 1820 年代：电流磁效应被系统研究，静磁学定律逐步形成。
- 19 世纪中期：场论框架下 `H` 与材料响应被分离描述，`M` 作为介质内部响应量被标准化。
- 20 世纪：SI 单位制下 `M`（`A/m`）与磁化率 `chi`（无量纲）成为工程建模常用参数。

## R03

问题背景是“给定外加磁场，如何快速计算并标定材料磁响应”。

在工程中，常见需求是：

- 由少量实验点估计低场磁化率 `chi`
- 用简化非线性模型估计饱和值 `Ms` 与特征场强 `H0`
- 将 `M(H)` 转换为 `B(H)` 供线圈、磁路或传感器前期设计使用

完整铁磁模型可能涉及磁滞回线、历史依赖、温度依赖；MVP 先给出“无磁滞静态曲线”这一最小可运行闭环。

## R04

设低场线性数据点数为 `N1`，饱和曲线点数为 `N2`，探针点评估数为 `P`。

- 线性磁化计算：`O(N1)`
- 线性回归拟合 `chi`：`O(N1)`（单特征、无截距）
- 饱和模型函数评估：每次 `O(N2)`
- `curve_fit` 迭代总成本：`O(I * N2)`，`I` 为迭代次数
- 构建对照表与 `B(H)`：`O(P)`

总成本主要由非线性拟合项 `O(I * N2)` 决定。

## R05

空间复杂度（额外工作内存）主要由数组存储决定。

- `H`、`M`、`B` 各自线性存储：`O(max(N1, N2, P))`
- 非线性拟合内部临时向量：`O(N2)`
- 总体额外空间为线性量级 `O(N)`

本实现不构造大型矩阵，内存占用较平稳。

## R06

`demo.py` 固定随机种子 `20260407` 的可复现实例结果如下（`uv run python demo.py`）：

- 低场线性回归：

  - `chi_hat  = 7.838277e-03`
  - `R2       = 0.948058`
- 饱和参数拟合：

  - `H0_true = 1.200000e+05 A/m`, `H0_hat = 1.211003e+05 A/m`

示例还给出 `H={-3e5,-1e5,0,1e5,3e5}` 上三类材料（抗磁、顺磁、饱和铁磁近似）的 `M` 与 `B` 对照表。

## R07

该算法条目的意义：

- 物理建模意义：把材料磁响应从“概念定义”转成“可计算函数 `M(H)`”。
- 工程意义：支持快速参数估计与初步磁路量级判断。
- 方法意义：展示线性拟合 + 非线性拟合 + 多框架一致性校验的最小闭环。

## R08

依赖的先导知识与方法：

- 静磁学基础关系：`B = mu0(H + M)`
- 线性介质假设：`M = chi H`
- 非线性饱和近似：`tanh` 型无磁滞模型
- 数值回归：
  - 最小二乘线性回归
  - 非线性最小二乘曲线拟合
- 向量化数组计算与单位一致性管理（SI）

## R09

适用前提与边界：

- 适用：
  - 静态或准静态场景
  - 仅关注主磁化曲线（不追踪磁滞回线历史）
  - 温度近似恒定
- 边界：
  - 线性模型只在低场小信号范围可靠
  - `tanh` 饱和模型无法表达矫顽力、剩磁、路径依赖
- 不适用：
  - 精确铁磁回线仿真（应使用 Jiles-Atherton / Preisach 等模型）
  - 强耦合多物理场（应引入更完整 PDE 框架）

## R10

正确性依据来自三层：

1. 公式层：`M(H)` 与 `B=mu0(H+M)` 的实现直接对应模型定义。
2. 参数层：
   - 线性部分使用无截距最小二乘估计 `chi`，符合 `M=chi H` 假设。
   - 饱和部分通过残差最小化估计 `Ms,H0`。
3. 实现层：
   - 用固定随机种子保证可复现。
   - 通过阈值断言（拟合相对误差、`R2`、跨框架差值）形成自动化正确性回归。

## R11

数值稳定性与误差传播：

- 低场线性拟合的误差主要受观测噪声影响；`H` 采样范围过窄时会放大参数不确定性。
- 非线性拟合对初值与噪声敏感；本实现使用有界参数和合理初值降低发散风险。
- `tanh(H/H0)` 在大 `|H|` 区域梯度变小，参数相关性可能增强，需足够覆盖中场区样本。
- NumPy 与 PyTorch 一致性误差约 `1e-16 T`，处于双精度舍入量级。

## R12

真实成本（工程视角）包括：

- CPU 成本：非线性拟合迭代是主要耗时；本规模（几百点）通常秒级内完成。
- 内存成本：仅线性数组，远低于网格 PDE 求解。
- 维护成本：参数和单位含义清晰，便于接入实验数据批处理。
- 依赖成本：仅使用通用科学计算库，不依赖专用电磁黑箱求解器。

## R13

误差与保证口径：

- 该流程是确定性数值算法，不涉及随机采样概率保证。
- 在线性真模型 + 零均值噪声下，`chi_hat` 是最小二乘意义下的一致估计。
- 非线性拟合收敛到局部最优；有界约束与多点覆盖可降低劣解风险。
- 本 MVP 用阈值化回归标准提供“工程可接受”保证：
  - `rel_err(chi) < 8%`
  - `R2 > 0.94`
  - `rel_err(Ms) < 5%`
  - `rel_err(H0) < 7%`

## R14

鲁棒性与失效模式：

- 失效 1：数据仅在极小场强区，`chi` 拟合受噪声主导。
- 失效 2：真实材料存在强磁滞，`tanh` 模型系统性偏差较大。
- 失效 3：单位混用（`kA/m` 与 `A/m`）导致参数量级错误。
- 失效 4：非线性拟合初值不当可能导致慢收敛或局部不良解。

缓解方式：扩展采样范围、分段建模、单位审计、增加多起点拟合。

## R15

工程实现要点：

- 统一 SI 单位并显式打印参数量纲。
- 线性与非线性模型函数分离，便于单测与替换。
- 使用固定随机种子构建可重复基准数据。
- 保留阈值检查并在失败时 `SystemExit(1)`，便于 CI 自动判定。
- 用表格输出关键点 `M,B`，方便人工快速审阅模型行为。

## R18

源码级算法流追踪（含第三方库非黑箱拆解，8 步）：

1. `main` 先按固定种子生成两批合成数据：低场线性数据和饱和曲线数据，确保回归可复现。
2. `estimate_susceptibility_linear_regression` 调用 `LinearRegression(fit_intercept=False)`：
   - 构造设计矩阵 `X=H[:,None]`
   - 以最小二乘准则求解 `min ||X*chi - M||^2`
   - 底层通过线性代数解算（等价于单参数 OLS）返回 `chi_hat` 与 `R2`。
3. `fit_saturation_parameters` 调用 `scipy.optimize.curve_fit`，目标函数为 `Ms*tanh(H/H0)`。
4. `curve_fit` 在有界参数空间中迭代执行“残差评估 -> Jacobian 近似 -> 步长更新”，直到收敛得到 `Ms_hat,H0_hat`。
5. 用 `magnetic_flux_density` 把各模型 `M(H)` 转换为 `B(H)=mu0(H+M)`，形成统一物理输出口径。
6. `build_material_table` 在指定探针场强点汇总抗磁、顺磁、饱和模型的 `M` 与 `B`，生成 `pandas` 表格。
7. `torch_consistency_check` 用 `torch.float64` 重算同一 `B(H)`，比较 `max|B_torch-B_numpy|`，验证跨实现一致性。
8. 汇总误差阈值并做自动断言，全部通过则输出 `Validation: PASS`，否则失败退出。
