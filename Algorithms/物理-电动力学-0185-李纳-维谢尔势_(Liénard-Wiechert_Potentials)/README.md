# 李纳-维谢尔势 (Liénard-Wiechert Potentials)

- UID: `PHYS-0184`

## R01

本条目实现李纳-维谢尔势的最小可运行算法版本：

- 给定匀速运动点电荷，求观测点在观测时刻 `t_obs` 的迟滞时间 `t_r`；
- 用 `t_r` 构造标量势 `phi` 与矢势 `A`；
- 进一步计算速度场项 `E, B`，并与等价闭式解进行数值一致性校验。

目标不是完整辐射仿真器，而是“公式 -> 数值流程 -> 自动验证”的可审计 MVP。

## R02

李纳-维谢尔势用于描述任意运动点电荷在迟滞时刻对场点的影响。核心难点不在代入公式，而在：

1. `t_r` 出现在几何距离中，需解隐式方程；
2. 势与场的因果结构依赖 `kappa = 1 - n·beta`；
3. 数值实现中必须验证因果残差和不同等价表达的一致性。

本 MVP 选取“沿 `+x` 匀速运动”的可控场景，既保留迟滞求解难点，也有可用闭式基准解。

## R03

`demo.py` 的实验目标：

1. 固定点迭代求解迟滞时间，避免黑盒根求解器；
2. 用匀速情形二次方程解析根做交叉验证；
3. 计算 `phi`、`A` 并检查关系 `A = (phi/c) beta`；
4. 计算李纳-维谢尔速度场并对比 Heaviside 闭式场；
5. 在低速极限 (`beta=1e-4`) 下验证势趋近库仑势。

## R04

脚本采用 SI 单位，核心方程如下。

1. 迟滞时间方程：
`c (t_obs - t_r) = |r_obs - r_q(t_r)|`

2. 匀速轨道：
`r_q(t) = v t`, `beta = v/c`

3. 李纳-维谢尔势：
`phi = q / (4 pi epsilon0 kappa R)`
`A = (mu0 q / (4 pi)) * v / (kappa R)`
其中 `R = |r_obs - r_q(t_r)|`, `n = (r_obs - r_q(t_r))/R`, `kappa = 1 - n·beta`。

4. 匀速时速度场（加速度项为 0）：
`E = (q/(4 pi epsilon0)) * (1-beta^2) * (n-beta) / (kappa^3 R^2)`
`B = (n x E)/c`

5. 匀速电荷闭式场（Heaviside 形式）用于对照：
`E_ref = (q/(4 pi epsilon0)) * (1-beta^2) * R_now / (R_now^2 - |beta x R_now|^2)^(3/2)`
`B_ref = (beta x E_ref)/c`

## R05

设观测点数为 `N`，固定点迭代步数为 `K`：

- 迟滞时间求解复杂度：`O(NK)`；
- 势与场计算：`O(N)`；
- 诊断量与表格输出：`O(N)`。

总体时间复杂度 `O(NK)`，空间复杂度 `O(N)`。

## R06

脚本输出包含三层信息：

1. 全局诊断：
   - `max |t_r_num - t_r_ref|`
   - 因果方程残差 `max |t_obs - t_r - R/c|`
   - 势/场等价关系误差

2. 逐点样本表：
   - 观测点坐标、`t_r`、`kappa`、`R`
   - `phi`、`|A|`、`|E|` 与相对误差

3. 阈值检查与最终 `Validation: PASS/FAIL`。

## R07

优点：

- 迟滞时间由源码迭代显式求解，流程透明；
- 同时具备数值解与解析基准，易于定位错误；
- 势与场都给出，并提供多条物理一致性检查。

局限：

- 仅覆盖匀速运动（不含加速度辐射项）；
- 未计算时域波形传播与频谱；
- 未处理导体边界、介质色散、束流集体效应。

## R10

正确性直觉：

1. 若 `t_r` 正确，必满足 `t_obs - t_r = R/c`；
2. 若势公式实现正确，`A` 与 `phi` 应满足 `A=(phi/c)beta`；
3. 对匀速运动，李纳-维谢尔速度场与 Heaviside 闭式场是同一物理对象，应逐点一致；
4. 当 `beta -> 0`，系统应回到静电库仑势近似。

因此，脚本通过“方程残差 + 等价公式互验 + 极限行为”给出可审计正确性证据。

## R11

数值稳定性设计：

- 迟滞时间用固定点迭代，匀速 `|beta|<1` 下具有收敛性；
- 观测点远离电荷轨迹奇点，避免 `R=0` 与分母病态；
- 对判别式 `disc` 使用 `max(disc,0)` 消除浮点微负值；
- 相对误差分母使用 `max(norm, 1e-30)` 防止零除。

## R12

关键参数与影响：

- `beta`：越接近 1，`kappa` 结构越尖锐，数值更敏感；
- `t_obs_second`：决定电荷当前位置和迟滞几何；
- `retarded_tol_second` 与 `retarded_max_iter`：控制迟滞求解精度与成本；
- `beta_for_low_speed_check`：低速极限校验强度。

调参建议：

- 若 `retarded equation residual` 偏大，先缩小 `retarded_tol_second`；
- 若高 `beta` 下失败，优先增加 `retarded_max_iter` 并拉远观测点。

## R13

保证类型：

- 近似比保证：N/A（非优化问题）；
- 随机成功率：N/A（流程完全确定性）；
- 可验证保证（以阈值形式）：
  1) 迟滞数值解与解析解一致；
  2) 迟滞方程残差接近 0；
  3) `A` 与 `phi` 的关系成立；
  4) `E,B` 与 Heaviside 闭式一致；
  5) 低速势接近库仑势。

## R14

常见失效模式：

1. 误把 `beta` 当 `v` 使用，导致维度错误；
2. 使用 `t_r` 的“前向根”而非迟滞根，造成因果错误；
3. 观测点过近导致 `R` 太小，数值发散；
4. 忘记 `kappa` 因子，势与场数量级严重偏差；
5. `beta>=1` 时固定点迭代不再物理可用。

## R18

`demo.py` 源码级算法流程（9 步）：

1. `make_observation_points` 依据 `beta,t_obs` 构造非奇异观测点集合。  
2. `solve_retarded_time_fixed_point` 对每个观测点迭代求解隐式方程 `c(t_obs-t_r)=|r-r_q(t_r)|`。  
3. `retarded_time_analytic_uniform_x` 用匀速运动二次方程闭式根计算 `t_r_ref`。  
4. `potentials_for_uniform_case` 比较 `t_r_num` 与 `t_r_ref`，并计算因果残差 `t_obs-t_r-R/c`。  
5. `lw_potentials_from_retarded` 在 `t_r_num` 上求 `R,n,kappa`，进而计算 `phi` 与 `A`。  
6. `lw_velocity_fields` 用李纳-维谢尔速度场公式计算 `E_lw,B_lw`。  
7. `heaviside_fields_uniform` 用匀速闭式表达计算 `E_ref,B_ref`，形成独立基准。  
8. `main` 计算 `A-(phi/c)beta`、`E/B` 相对误差、低速库仑极限误差并执行阈值判定。  
9. 输出全局统计、逐点表格和 `Validation: PASS/FAIL`，失败时返回非零退出码。

第三方库未被当作黑盒算法：

- `numpy` 用于向量化数组运算；
- `pandas` 仅用于结果表格化展示；
- `scipy.constants` 仅提供物理常数；
- 迟滞时间求解、势/场构造、残差/阈值判断都在源码中显式实现。
