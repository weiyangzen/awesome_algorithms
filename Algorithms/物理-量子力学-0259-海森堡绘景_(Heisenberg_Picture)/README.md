# 海森堡绘景 (Heisenberg Picture)

- UID: `PHYS-0256`
- 学科: `物理`
- 分类: `量子力学`
- 源序号: `259`
- 目标目录: `Algorithms/物理-量子力学-0259-海森堡绘景_(Heisenberg_Picture)`

## R01

海森堡绘景（Heisenberg Picture）是量子力学的三种等价表述之一（另两种是薛定谔绘景与相互作用绘景）。其核心思想是：

- 态（state）固定不随时间演化；
- 可观测量对应的算符随时间演化；
- 物理可测量量（期望值）与薛定谔绘景严格一致。

若设时间演化算符 `U(t)=exp(-iHt/ħ)`，则海森堡算符为：

- `A_H(t) = U†(t) A_S U(t)`

其中 `A_S` 是薛定谔绘景中的时间无关算符（若本身无显含时项）。

## R02

背景与意义：

- 由海森堡在矩阵力学框架中提出，强调“可观测量”而非“波函数”的时间变化；
- 与薛定谔波动力学在数学上等价，但关注对象不同；
- 在量子场论与多体理论中，海森堡绘景常更自然，因为场算符/产生湮灭算符的时间演化更直接。

## R03

该方法回答的问题是：

- 如何在保持态固定的前提下追踪系统随时间变化的物理量？
- 如何通过算符动力学直接得到测量期望值与守恒量？

其关键方程（无显含时时）为海森堡运动方程：

- `dA_H/dt = (i/ħ)[H, A_H]`

若 `A` 显含时，还需加 `+(∂A/∂t)_H`。

## R04

本目录 MVP 的数学模型：二维自旋 `1/2` 系统。

- 取 `ħ=1`；
- 哈密顿量：`H = (ω/2) σ_z`；
- 研究算符：`σ_x, σ_y, σ_z`；
- 初态（固定态）：`ρ0 = |+x><+x|`。

由理论可得：

- `σx_H(t) = cos(ωt) σx - sin(ωt) σy`
- `σy_H(t) = sin(ωt) σx + cos(ωt) σy`
- `σz_H(t) = σz`

因此在 `ρ0` 上：

- `<σx>(t)=cos(ωt)`，`<σy>(t)=sin(ωt)`，`<σz>(t)=0`。

## R05

复杂度（对本 MVP 的一般维度 `d` 分析）：

- 通过本征分解构造 `U(t)`：`O(d^3)`；
- 单次算符变换 `A_H = U†AU`：`O(d^3)`；
- 若计算 `T` 个时间点：`O(T*d^3)`；
- 空间复杂度约 `O(d^2)`。

本示例 `d=2`，计算代价几乎可忽略，重点在验证关系与可读性。

## R06

直观例子：令 `ω=1`，当 `t=π/2` 时：

- `cos(ωt)=0`，`sin(ωt)=1`；
- `σx_H(π/2) = -σy`；
- 若初态 `|+x>`，则 `<σx>(π/2)=0`、`<σy>(π/2)=1`。

这表示“测量坐标系在转动”：态不动，但算符发生绕 `z` 轴进动。

## R07

优点：

- 直接描述可观测量动力学，便于识别守恒量；
- 与对易关系结合紧密，适合代数推导；
- 在场算符演化问题中表达自然。

局限：

- 对初学者不如“态在变化”直观；
- 对大维度系统，显式算符演化成本高；
- 含时哈密顿量时需要时间序算符，公式更复杂。

## R08

前置知识：

- 线性代数：厄米矩阵、幺正变换、本征分解；
- 量子基础：密度矩阵、期望值、对易子；
- 数值实现：复矩阵计算与误差容忍。

## R09

适用场景：

- 关注算符演化、守恒律、响应函数；
- 需要和对易关系直接联立推导；
- 小到中等维度系统教学与验证。

不适用或需谨慎：

- 只关心态演化、且已有成熟态矢仿真流程时，薛定谔绘景可能更方便；
- 大规模 Hilbert 空间中显式算符矩阵过大，需要稀疏化或张量网络等技巧。

## R10

实现正确性的关键检查：

1. `U(t)` 是否幺正：`U†U=I`；
2. 海森堡变换是否按 `U†AU` 而不是 `UAU†`；
3. 期望值等价性：`Tr(ρ0 A_H(t)) == Tr(ρ_S(t) A_S)`；
4. 与解析解 `cos/sin` 是否一致；
5. 数值导数是否满足 `dA/dt ≈ i[H,A]`。

`demo.py` 对以上检查均有数值断言。

## R11

数值稳定性要点：

- 使用厄米矩阵本征分解（`np.linalg.eigh`）生成 `U(t)`，避免一般分解的额外不稳定性；
- 期望值保留实部并检查虚部接近 0；
- 有限差分步长 `dt` 取中等量级（如 `1e-6`），平衡截断误差与舍入误差；
- 断言阈值用 `1e-8 ~ 1e-5` 量级，不追求不现实的机器零误差。

## R12

参数选择（本 MVP）：

- `omega`：进动角频率；
- `num_steps`：离散时间点数；
- `t_final`：总演化时间（示例取一个完整周期 `2π/ω`）；
- `fd_dt`：有限差分步长。

调参建议：

- `num_steps` 增大可更细看波形；
- `fd_dt` 太小会放大浮点误差，太大则导数近似变粗。

## R13

理论性质：

- 海森堡与薛定谔绘景是严格等价重表述，不是近似算法；
- 若 `H` 与 `A` 对易，则 `A_H(t)` 不随时间变（守恒）；
- 本任务无“近似比/随机成功率”概念；
- 数值误差仅来自有限精度与离散导数。

## R14

常见失效模式与防护：

- 失效：把变换顺序写反（`UAU†`）。
  防护：用 `t=0` 与解析解双重校验。
- 失效：哈密顿量非厄米导致“演化不幺正”。
  防护：运行前检查 `H≈H†`。
- 失效：期望值出现明显虚部。
  防护：输出并断言 `|Im(Tr(ρA))|` 足够小。
- 失效：有限差分残差过大。
  防护：调整 `fd_dt` 并确认公式与单位一致（此处 `ħ=1`）。

## R15

工程实践建议：

- 先从 2x2 系统验证流程，再扩展到高维；
- 把“幺正性检查 + 绘景等价检查”作为回归测试；
- 对大系统优先考虑稀疏表示与 Krylov 子空间近似；
- 在文档中明确单位制（`ħ=1` 或 SI 单位），避免系数错误。

## R16

与相关方法关系：

- 与薛定谔绘景：物理等价，计算焦点不同（态 vs 算符）；
- 与相互作用绘景：把自由演化与相互作用拆分，常用于微扰理论；
- 与经典哈密顿动力学：`dA/dt=(i/ħ)[H,A]` 对应经典中的泊松括号结构。

## R17

`demo.py` 的 MVP 输出内容：

- 每个时间点的 `<σx>, <σy>, <σz>`（海森堡/薛定谔/解析值）；
- 两种绘景的最大期望值差异；
- 与解析解的最大误差；
- 海森堡运动方程有限差分残差。


## R18

`demo.py` 源码级算法流程拆解（8 步）：

1. 构造泡利矩阵与单位矩阵，建立 `H=(ω/2)σz` 和初态 `ρ0=|+x><+x|`。  
2. 用 `np.linalg.eigh(H)` 做厄米本征分解，得到本征值与本征矢。  
3. 在每个 `t` 计算 `U(t)=V·diag(exp(-iEt))·V†`。  
4. 用 `A_H(t)=U†AU` 计算 `σx,σy,σz` 的海森堡算符。  
5. 用固定态 `ρ0` 计算海森堡期望值 `Tr(ρ0 A_H(t))`。  
6. 同时计算 `ρ_S(t)=Uρ0U†`，并求 `Tr(ρ_S(t) A)`，验证与第 5 步一致。  
7. 将第 5 步结果与解析解 `cos(ωt), sin(ωt), 0` 比较，统计最大误差。  
8. 对 `σx_H(t)` 做中心差分，验证 `dA/dt = i[H,A]`，输出残差并断言通过。  

该实现未调用黑盒量子仿真库，演化算符、算符变换、期望值和验证流程都可在源码中逐行追踪。
