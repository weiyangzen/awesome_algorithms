## R01

克雷姆斯-克罗尼格关系（Kramers-Kronig, K-K）把**线性响应函数**的实部与虚部严格耦合起来。  
对因果系统（响应只在激励之后出现），频域响应 `chi(omega)` 在上半复平面解析，导致：

- 色散（实部）与吸收/耗散（虚部）不能独立指定；
- 只要知道一部分并满足适当衰减与因果条件，就能通过主值积分恢复另一部分；
- 因果性约束可转化为可计算的数值校验。

本条目用一个最小 MVP 直接实现离散主值积分，双向验证 `Re(chi) <-> Im(chi)`。

## R02

计算任务定义：

- 输入：
  - 频率网格 `omega in [-omega_max, omega_max]`（均匀离散）；
  - 一个因果模型（本实现为洛伦兹振子）给出的 `chi(omega)`；
  - 数值积分与验收阈值配置。
- 输出：
  - 由 `Im(chi)` 重建的 `Re(chi)`；
  - 由 `Re(chi)` 重建的 `Im(chi)`；
  - 重建误差指标（相对 L2、最大绝对误差）与样例点对照表；
  - 自动化通过/失败结论。

运行方式无交互输入，`uv run python demo.py` 即可完成整个流程。

## R03

K-K 核心公式（全频轴形式）：

设 `chi(omega) = chi'(omega) + i chi''(omega)`，则

- `chi'(omega) = (1/pi) P ∫_{-inf}^{+inf} chi''(omega') / (omega' - omega) d omega'`
- `chi''(omega) = -(1/pi) P ∫_{-inf}^{+inf} chi'(omega') / (omega' - omega) d omega'`

其中 `P` 表示柯西主值（principal value），用于处理 `omega' = omega` 处的奇异核。

若时域响应为实函数，通常还满足：

- `chi'(omega)` 为偶函数；
- `chi''(omega)` 为奇函数。

## R04

本 MVP 使用的解析基准模型：单洛伦兹振子响应

`chi(omega) = A / (omega_0^2 - omega^2 - i gamma omega)`

分解得到：

- `Re(chi) = A (omega_0^2 - omega^2) / [ (omega_0^2 - omega^2)^2 + (gamma omega)^2 ]`
- `Im(chi) = A gamma omega / [ (omega_0^2 - omega^2)^2 + (gamma omega)^2 ]`

该模型天然满足因果结构，且 `Re` 偶、`Im` 奇，适合做 K-K 数值校验基线。

## R05

算法策略（最小但可验证）：

1. 构建对称均匀频率网格。  
2. 计算解析 `chi(omega)` 的 `Re` 与 `Im`。  
3. 用离散主值积分实现 K-K 变换：
   - `Im -> Re`（正号核）；
   - `Re -> Im`（负号核）。  
4. 在中心频段统计误差，避免边界截断主导。  
5. 输出指标与样例点，并执行阈值断言。

关键点：实现中没有调用现成黑盒 Hilbert 函数，而是显式写出分母核与奇点剔除。

## R06

微型数值示例（`A=1, omega_0=2.5, gamma=0.7`）：

取 `omega=1`：

- 分母平方和 `D = (2.5^2-1^2)^2 + (0.7*1)^2 = 28.0525`
- `Re(chi) = 5.25 / 28.0525 ≈ 0.1872`
- `Im(chi) = 0.7 / 28.0525 ≈ 0.0250`

可见低频处实部主导、虚部较小；在共振附近（`omega ≈ omega_0`）虚部会显著增强。

## R07

为什么 K-K 关系在统计物理/凝聚态里重要：

- 线性响应理论（涨落-耗散、输运系数、介电函数）都依赖因果结构；
- 实验常只能测到吸收谱或折射谱的一侧，K-K 提供互推桥梁；
- 可用于数据质量审计：若严重违背 K-K，通常意味着非因果、截断误差或预处理问题；
- 在介电、光谱、声学、黏弹与磁响应分析中广泛使用。

## R08

理论基础（工程化摘要）：

1. 因果性要求时域核 `g(t)=0 (t<0)`。  
2. 频域响应是 `g(t)` 的傅里叶/拉普拉斯变换边值，具备复解析性。  
3. 解析函数实虚部满足 Hilbert 变换对偶。  
4. 对实频轴边值可写成 K-K 主值积分。  
5. 任何“独立指定实部与虚部”的做法若不满足该关系，就会破坏因果约束。

## R09

适用条件与局限：

- 适用：线性、时不变、因果的响应系统；
- 需要频谱在高频足够衰减，积分可定义；
- 数值上常见难点：
  - 有限频窗截断（无法积分到 `inf`）；
  - 采样噪声放大（Hilbert 核对高频误差敏感）；
  - 奇点附近主值离散化误差。

因此工程中常配合外推、窗函数、基线校正或带物理约束拟合。

## R10

本实现的正确性验证框架：

1. 解析真值：由洛伦兹模型直接算 `Re_true`, `Im_true`；
2. K-K 重建：得到 `Re_from_Im`, `Im_from_Re`；
3. 在中心区 `|omega| <= eval_omega_limit` 计算：
   - 相对 L2 误差（双向）；
   - 最大绝对误差（双向）；
4. 对称性检查：
   - `Re` 偶对称误差；
   - `Im` 奇对称误差；
5. 阈值断言全部通过后输出 `All checks passed.`。

## R11

误差来源与稳定性说明：

- 主值积分离散误差：网格越密，误差越低；
- 频窗截断误差：边界附近误差会更显著；
- 核函数近奇异性：`omega'≈omega` 时数值敏感；
- 双精度舍入：对称误差通常在 `1e-15` 量级。

本 MVP 通过：
- 使用较密网格（3001 点）；
- 对端点用梯形权重；
- 在评估时剔除靠边区间；
来保持结果稳定。

## R12

复杂度：

设频点数为 `N`。

- 每个目标频点都需扫描一次全频网格，复杂度 `O(N^2)`；
- 额外存储主要是若干 `O(N)` 向量；
- 总体空间复杂度 `O(N)`（未构造全核矩阵）。

对 `N=3001` 的 MVP，普通 CPU 上可在较短时间完成。

可复现与可验收特性：

- 标准输出包含：

## R14

常见失效模式与排查：

- 网格不是严格均匀：主值积分离散公式不再适配；
- `n_points` 太小：相对误差明显增大；
- `eval_omega_limit` 过接近 `omega_max`：边界截断污染统计；
- 模型参数非法（`gamma<=0`, `omega_0<=0` 等）：应先触发配置校验；
- 若把符号写反（`+/-` 搞错），双向重建会系统性失真。

## R15

`demo.py` 结构：

- `KKConfig`：集中参数与合法性校验；
- `build_frequency_grid`：生成对称频率网格；
- `lorentz_susceptibility`：解析基准谱；
- `kk_principal_value_transform`：离散主值积分核心；
- `relative_l2_error`：相对 L2 指标；

- `main`：打印报告并进行断言验收。

## R18

源码级算法流程拆解（`demo.py`，8 步）：

1. `main` 创建 `KKConfig` 并调用 `validate()`，先把参数边界问题拦截。  
2. `run_mvp` 用 `build_frequency_grid` 生成 `[-omega_max, omega_max]` 的均匀对称网格。  
3. `lorentz_susceptibility` 在每个频点计算复响应 `chi(omega)`，拆出 `Re_true` 与 `Im_true`。  
4. 第一次 K-K：`kk_principal_value_transform(..., Im_true, sign=+1)`，重建 `Re_from_Im`。  
5. 第二次 K-K：`kk_principal_value_transform(..., Re_true, sign=-1)`，重建 `Im_from_Re`。  
6. 在积分核心中逐点构造核 `1/(omega'-omega_i)`，通过 `np.divide(..., where=denominator!=0)` 显式执行主值奇点剔除，再做梯形求和。  
7. `run_mvp` 计算中心区相对 L2 与最大绝对误差，并附带偶/奇对称一致性指标。  
8. `main` 输出 `pandas` 表格并执行阈值断言；全部通过后打印 `All checks passed.`。

补充：本实现没有把 K-K 当第三方黑盒调用，核心积分核、主值处理、误差评估都在源码中可直接审计。
