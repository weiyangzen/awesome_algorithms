## R01

伯努利方程用于描述稳态、不可压、无粘（或将损失单独建模）的流体沿同一流线的机械能守恒。经典形式为：

`p + 0.5*rho*v^2 + rho*g*z = constant`

其中 `p` 为静压，`rho` 为密度，`v` 为流速，`z` 为几何高度，`g` 为重力加速度。  
本目录 MVP 采用“连续性方程 + 广义伯努利方程”求解两截面管流问题。

## R02

任务目标是实现一个最小且可验证的伯努利计算器：

- 给定上游/下游压力、截面积、高程和损失系数；
- 反求上下游速度与体积流量；
- 扫描下游压力，观察压差与流量关系；
- 验证能量平衡残差接近 0（数值误差范围内）。

实现强调可解释推导，不依赖第三方流体求解黑盒。

## R03

本实现采用的模型与符号：

- 连续性方程（不可压）：`A1*v1 = A2*v2`
- 广义伯努利（含局部损失）：
  `p1 + 0.5*rho*v1^2 + rho*g*z1 = p2 + 0.5*rho*v2^2 + rho*g*z2 + 0.5*rho*K*v2^2`
- 损失模型：`h_loss = K*v2^2/(2g)`，写成压强单位即 `0.5*rho*K*v2^2`

MVP 假设流体密度常数、流动稳态、截面速度均匀近似。

## R04

把 `v1 = (A2/A1)*v2` 代入伯努利方程，可得下游速度闭式解：

`v2 = sqrt( 2*[(p1-p2)+rho*g*(z1-z2)] / (rho*(1-alpha^2+K)) )`

其中 `alpha = A2/A1`。随后：

- `v1 = alpha*v2`
- `Q = A2*v2`

只要驱动项 `[(p1-p2)+rho*g*(z1-z2)] > 0` 且分母 `rho*(1-alpha^2+K) > 0`，即可得到实数解。

## R05

`demo.py` 的计算流程是：

1. 读取配置（`rho, g, A1, A2, z1, z2, K, p1`）。
2. 对给定 `p2` 用闭式公式求 `v2`，再由连续性求 `v1` 和 `Q`。
3. 计算伯努利能量残差（Pa）用于一致性检查。
4. 对一组 `p2` 进行扫描，生成 `pandas` 表格。
5. 执行断言：连续性误差、残差、单调性、损失项影响。

## R06

脚本输出两类结果：

- 参考工况：`p1/p2, v1/v2, Q, residual`
- 扫描表：`delta_p, v_upstream, v_downstream, flow_rate, continuity_error, energy_residual`

预期现象：

- `delta_p` 增大时 `Q` 增大；
- 含损失系数 `K>0` 时，同压差下流量低于 `K=0` 理想情况；
- 连续性误差与能量残差接近机器精度。

## R07

优点：

- 公式透明，容易核查；
- 计算量小，适合教学和自动化验证；
- 把能量方程残差显式输出，便于定位错误。

局限：

- 未处理可压缩流和粘性边界层细节；
- 损失项用单参数 `K` 汇总，不能替代完整管网模型；
- 默认截面平均速度模型，不含速度分布修正系数。

## R10

正确性直觉来自两个守恒约束：

1. 质量守恒给出 `A1*v1 = A2*v2`；
2. 能量守恒（加损失项）约束上下游总机械能；
3. 两者联立即可唯一确定 `v1, v2, Q`（在参数可解条件下）。

因此，若程序同时满足连续性误差近零与能量残差近零，结果在该模型内就是自洽的。

## R11

数值与工程稳健性处理：

- 在求根前检查驱动项和分母正性，避免 `sqrt` 非法；
- 显式输出 `continuity_error` 与 `energy_residual`；
- 全部计算使用 `float64`；
- 断言阈值设置为严格但可达的量级（如 `1e-6 Pa`、`1e-12 m^3/s`）。

## R12

核心参数（`BernoulliConfig`）：

- `rho, g`：流体与重力参数；
- `area_upstream, area_downstream`：上下游截面积；
- `z_upstream, z_downstream`：几何高程；
- `loss_k`：局部损失系数；
- `p_upstream, p_downstream_*`：压力工况与扫描区间；
- `n_sweep_points`：扫描分辨率。

调参建议：若要研究损失敏感性，优先调整 `loss_k`；若要研究管径影响，调整面积比 `A2/A1`。

## R14

常见失败模式：

1. 混用单位（kPa 与 Pa、L/s 与 m^3/s）。
2. 忽略高程项导致竖向工况偏差。
3. 把损失项符号写反，出现“损失越大流量越大”的反物理结果。
4. 面积比写反（`A1/A2` 与 `A2/A1`）导致速度放大/缩小错误。
5. 在驱动项非正时仍强行开方。

本实现通过显式公式、参数命名和断言来减少上述错误。

## R18

`demo.py` 源码级算法流程（7 步）：

1. `main` 构造 `BernoulliConfig`，给定流体参数、几何参数、损失系数和压力范围。  
2. `solve_from_pressure_pair` 先用连续性写出 `v1=alpha*v2`，再代入广义伯努利得到 `v2` 闭式解。  
3. 同函数继续计算 `v1` 与 `Q=A2*v2`，并返回单工况结果。  
4. `bernoulli_residual_pa` 按能量方程左右端重算一次，得到残差（用于自检）。  
5. `build_pressure_sweep` 扫描一组 `p2`，逐点调用步骤 2-4，汇总成 `pandas.DataFrame`。  
6. `main` 额外构造 `K=0` 的对照工况，比较同压差下有损失/无损失流量。  
7. `main` 执行断言（连续性、能量残差、压差-流量单调性、损失抑制流量）并打印通过信息。  

第三方库边界说明：`numpy` 仅用于数组扫描，`pandas` 仅用于表格展示；核心水力学求解全部由源码显式公式完成，无黑盒流体求解器调用。
