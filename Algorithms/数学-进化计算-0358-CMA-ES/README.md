## R01

CMA-ES（Covariance Matrix Adaptation Evolution Strategy，协方差矩阵自适应进化策略）是一类面向连续变量黑盒优化的随机搜索算法。  
它不需要目标函数的梯度信息，通过“采样候选解 -> 按适应度排序 -> 更新采样分布”反复迭代，在复杂、非凸、不可导目标上通常比固定高斯扰动更稳健。

## R02

核心思想是让“搜索分布”跟着成功样本的统计结构一起进化：

- 均值 `m` 向优秀样本的加权中心移动（决定搜索中心）。
- 全局步长 `sigma` 根据进展快慢自适应放大/缩小（决定搜索尺度）。
- 协方差 `C` 学习变量间相关性与主方向（决定搜索形状）。

因此 CMA-ES 不是直接“更新一个点”，而是更新“产生点的概率模型”。

## R03

标准采样模型：

`x_k = m + sigma * y_k`，其中 `y_k ~ N(0, C)`。

常见实现将 `C = B * diag(D^2) * B^T`，先采样 `z ~ N(0, I)`，再做线性变换得到 `y = B * diag(D) * z`。  
每代选择前 `mu` 个优秀样本，按权重 `w_i` 更新：

- 均值：`m <- sum_i w_i * x_i`
- 协方差（简化写法）：`C <- (1-c1-cmu)C + c1*pc*pc^T + cmu*sum_i w_i*y_i*y_i^T`
- 步长：`sigma <- sigma * exp((cs/ds)*(||ps||/E||N(0,I)|| - 1))`

其中 `pc`、`ps` 为演化路径（evolution paths），用来累计跨代趋势信息。

## R04

输入（本目录 MVP）：

- 目标函数 `f(x)`，默认用 8 维 Rosenbrock 函数。
- 初始均值 `init_mean`、初始步长 `init_sigma`。
- 最大迭代代数 `max_iter`。

输出：

- `best_x`：搜索过程中最好解。
- `best_f`：最好目标值。
- 收敛轨迹摘要（每 25 代打印一次关键指标）。

## R05

伪代码（最小可运行版本）：

```text
初始化 m, sigma, C=I, pc=0, ps=0
for t = 1..T:
  采样 lambda 个候选 x_k = m + sigma * y_k, y_k ~ N(0, C)
  计算每个候选的适应度 f(x_k)
  选择前 mu 个最优样本并排序
  用加权均值更新 m
  用白化位移更新 ps，并据此更新 sigma
  更新 pc
  用 rank-one + rank-mu 公式更新 C
  周期性对 C 做特征分解，更新 B, D, C^{-1/2}
返回历史最优解
```

## R06

关键超参数及本实现默认值：

- `lambda`（种群规模）: `4 + floor(3*log(n))`
- `mu`（选择数量）: `lambda // 2`
- `weights`：对前 `mu` 个样本做对数权重归一化
- `init_sigma`：`0.9`（初始搜索尺度）
- `max_iter`：`350`

经验上：

- 维度升高时可适当提高 `lambda` 增强稳定性。
- `init_sigma` 过小会陷入局部区域，过大收敛变慢。

## R07

CMA-ES 没有“全局最优必达”的通用保证，但在连续黑盒优化中具备较强鲁棒性：

- 对变量尺度和线性变换具有较好适应性（来自协方差学习）。
- 比固定各向同性高斯扰动更能利用“谷地方向”信息。
- 常用于难调参、无梯度、目标噪声存在的场景。

本 MVP 在 Rosenbrock 上通常能把目标值降到很小量级（接近 0）。

## R08

设维度为 `n`、每代种群为 `lambda`：

- 采样与适应度评估：`O(lambda * n)`（不含目标函数内部成本）
- 协方差累计更新：`O(mu * n^2)`
- 特征分解（周期执行）：`O(n^3)`，是高维主要开销

空间复杂度主要来自协方差及其分解矩阵：`O(n^2)`。

## R09

适用场景：

- 连续参数优化，目标函数不可导或梯度代价高。
- 多峰、非凸、噪声较强问题。
- 超参数搜索、控制参数调优、仿真器黑盒优化。

## R10

不适用或需谨慎场景：

- 超高维（例如数千维）且预算很紧：`O(n^2)`/`O(n^3)` 可能过重。
- 离散组合优化：应考虑遗传算法、局部搜索或 MCTS 等离散方法。
- 有高质量梯度可用时，L-BFGS/Adam 等通常更高效。

## R11

常见实现坑：

- `C` 数值误差导致非对称或非正定：需对称化并截断特征值下界。
- 忽略步长路径 `ps` 的归一化项，导致 `sigma` 更新失真。
- 把“最优点更新”误写成“最优点替换均值”，会损失分布学习能力。
- 目标函数量级极端时，建议做输入标准化或变量重参数化。

## R12

与常见方法对比：

- 相比基础 ES：CMA-ES 会学习协方差，不再只做各向同性扰动。
- 相比 Nelder-Mead：CMA-ES 在高噪声和多峰下通常更稳。
- 相比粒子群 PSO：CMA-ES 更强调统计分布建模，参数更少但矩阵计算更重。
- 相比梯度法：无需梯度，但样本效率通常低于高质量梯度优化器。

## R18

源码级算法流程拆解（对应 `demo.py`）：

1. 在 `CMAES.__init__` 初始化 `m, sigma, C, B, D, pc, ps` 与学习率参数 `c1, cmu, cs, ds`。
2. `ask()` 中采样标准高斯 `z`，经 `B*D` 变换得到 `y`，再构造候选 `x = m + sigma*y`。
3. 主循环里计算每个候选的目标值并按适应度排序，截取前 `mu` 个样本。
4. 在 `tell()` 用权重均值更新 `m`，并计算标准化位移 `delta_mean`。
5. 用 `invsqrtC @ delta_mean` 更新步长路径 `ps`，再依据其范数指数更新 `sigma`。
6. 用 `delta_mean` 与 `hsig` 更新协方差路径 `pc`，保留跨代方向信息。
7. 用 `c1 * pc*pc^T + cmu * Σ(w_i y_i y_i^T)` 更新协方差 `C`，并做对称化避免数值漂移。
8. 周期调用 `_update_eigensystem()` 对 `C` 特征分解，刷新 `B, D, invsqrtC` 供下一代采样与白化使用。
