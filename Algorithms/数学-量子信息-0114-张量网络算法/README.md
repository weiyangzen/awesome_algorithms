# 张量网络算法

- UID: `MATH-0114`
- 学科: `数学`
- 分类: `量子信息`
- 源序号: `114`
- 目标目录: `Algorithms/数学-量子信息-0114-张量网络算法`

## R01

张量网络算法是一类把高维张量分解为低阶张量并通过网络收缩计算物理量的方法。对于 `N` 个量子比特的纯态，完整态向量维度是 `2^N`，直接存储和计算会指数爆炸。  
本条目采用最常见的一维张量网络形式：`MPS (Matrix Product State)`，并用 `TT-SVD`（Tensor-Train SVD）把全态张量分解成一串三阶张量核（cores）。

## R02

背景脉络（简化版）：
- 量子多体计算中，早期 DMRG 思想推动了 MPS 的工程化使用；
- 数值线性代数社区用 TT（Tensor Train）描述与 MPS 等价的结构；
- 在量子信息与量子模拟里，MPS/TT 成为“用多项式资源近似低纠缠态”的核心工具。

本 MVP 不追求完整 DMRG 优化流程，而聚焦一个可复现、可解释的最小路径：`随机量子态 -> TT-SVD 分解 -> 收缩计算振幅/观测量 -> 误差评估`。

## R03

本条目解决的问题定义：
1. 输入：`N` 比特量子态 `|psi>`（代码中用复数向量表示，并 reshape 成 `(2,2,...,2)` 张量）。
2. 输出：受键维上限 `chi` 约束的 MPS 近似表示。
3. 目标：在可控误差下显著减少参数量，并支持不重建全向量的局域观测量收缩计算。

## R04

MVP 的算法流程如下：
1. 生成归一化随机复数态向量；
2. 转为 `N` 阶二值物理维张量；
3. 逐站点做 SVD（TT-SVD）：把当前展开矩阵分成 `U S V^H`；
4. 按 `cutoff` 与 `max_bond=chi` 截断奇异值；
5. `U` reshape 为当前 MPS 核，`S V^H` 传递到下一站点；
6. 得到完整 MPS cores 后，可做：
   - 全态重构与保真度评估；
   - 基矢振幅 `<b|psi>` 链式收缩；
   - 局域 `Z_i` 期望值的转移矩阵收缩。

## R05

复杂度（设物理维 `d=2`，键维上限 `chi`）：
- 从全态做 TT-SVD：受每步 SVD 规模影响，整体仍与 `2^N` 相关，适合中小 `N` 的教学/验证；
- 已有 MPS 后，单个基矢振幅收缩成本约 `O(N * chi^2)`；
- 局域单点观测量（如 `Z_i`）转移矩阵收缩成本约 `O(N * chi^3)`；
- 参数存储从 `O(2^N)` 降为 `O(N * d * chi^2)`（边界核更小）。

## R06

直观例子：
- `N=10` 时，全态参数量是 `2^10=1024`；
- 若限制 `chi=8`，MPS 参数量远少于高键维精确表示，通常可保持较高保真度；
- 提高到 `chi=32`（`N=10` 中心切分理论上界）可达到数值精度上的“近似精确重构”。

`demo.py` 会直接输出不同 `chi` 下的：
- 相对误差、保真度；
- 参数压缩比；
- 中心键的纠缠熵估计。

## R07

优点：
- 把指数维对象转成链式局部结构，计算与存储显著降维；
- 可解释性强，奇异值谱直接反映纠缠结构；
- 便于做可控截断（`chi` 和 `cutoff`）。

局限：
- 对高纠缠（体积律）态，低 `chi` 近似误差会明显上升；
- 本 MVP 从全态起步，仍需要先持有 `2^N` 向量，不适合超大系统；
- 未实现 DMRG/TEBD 等“在 MPS 空间直接演化或优化”的高级流程。

## R08

前置知识建议：
- 线性代数：SVD、正交分解、奇异值截断；
- 量子信息：态向量、基矢振幅、Pauli 算符期望值；
- 张量操作：reshape、tensordot、Kronecker 积、链式收缩。

## R09

适用边界：
- 适用：1D 量子链、低到中等纠缠态、教学验证、算法原型；
- 谨慎：强体积律随机态在很小 `chi` 下；
- 不适用：直接处理超大规模系统且无法显式构造全态时（应转向纯 MPS 演化算法）。

## R10

正确性要点（实现层面）：
1. TT-SVD 每步都保持代数分解 `M = U S V^H`；
2. 不截断时（足够大 `chi`）理论上可精确重构，代码通过 `chi=32` 场景断言验证；
3. 基矢振幅采用 MPS 链式矩阵乘，和全态索引值逐点对比；
4. 局域 `Z_i` 期望值采用转移矩阵收缩，并与全态真值逐站点对齐。

## R11

数值稳定性措施：
- SVD 优先用 `scipy.linalg.svd`，缺失时回退 `numpy.linalg.svd`；
- 截断时保底保留至少一个奇异值，避免秩坍塌为 0；
- 比较误差前先把重构态重新归一化；
- 断言阈值采用 `1e-10 ~ 1e-8` 量级，兼顾浮点误差与可复现性。

## R12

性能与调参经验：
- `chi` 是最关键旋钮：增大 `chi` 会提升精度但增加参数与收缩成本；
- `cutoff` 可过滤极小奇异值，减少冗余键维；
- 对固定 `N`，建议先跑 `chi` 网格（如 `2,4,8,16,32`）观察误差-成本曲线。

## R13

理论保证说明：
- 该实现不属于“近似比算法”范畴，故近似比 `N/A`；
- 也不是随机成功率算法，`N/A`；
- 但对“给定全态 + 给定截断规则”的 TT-SVD 过程是确定性的，结果可复现（固定随机种子后输入态固定）。

## R14

常见失效模式与防护：
1. `chi` 太小导致误差过大。  
   防护：脚本输出误差表并在压缩质量越界时抛异常。
2. 输入张量物理维不是 2。  
   防护：`tt_svd` 显式校验每个 mode 必须为 2。
3. 位串长度和 qubit 数不一致。  
   防护：`mps_amplitude` 参数检查后报错。
4. 数值漂移导致范数偏离。  
   防护：重构后归一化，并用转移矩阵再验 `<psi|psi>`。

## R15

工程实践建议：
- 小规模先做“正确性闭环”：重构误差、振幅对比、局域观测量对比三重验证；
- 再根据任务预算调 `chi/cutoff`；
- 若进入真实量子模拟任务，应进一步加入规范化（canonical form）、截断误差累计监控与算符 MPO 支持。

## R16

相关算法关系：
- 本实现：`MPS / TT-SVD`（一维链式网络）；
- 常见扩展：
  - DMRG（基态优化）；
  - TEBD / TDVP（时间演化）；
  - PEPS（二维网络）；
  - MERA（分层网络）。

可把本条目视为这些高级方法的“底层张量核与收缩习题”。

## R18

源码级流程拆解（`demo.py`，8 步）：
1. `random_quantum_state` 生成可复现复数态并归一化，`vector_to_tensor` 转成 `(2,)*N` 张量。  
2. `summarize_compression` 在 `chi` 网格上循环调用 `tt_svd`，评估误差、保真度、参数量与中间键纠缠熵。  
3. `tt_svd` 内部按站点重排矩阵后做 SVD，并根据 `max_bond/cutoff` 截断奇异值，逐步生成 MPS cores。  
4. `mps_to_state` 用 `tensordot` 把所有 cores 收缩回全态向量，用于重构误差与保真度验证。  
5. `mps_amplitude` 按位串选择每个 core 的物理切片，执行链式矩阵乘得到 `<b|psi>`。  
6. `_single_site_transfer` 构造转移矩阵，`mps_local_z_expectation` 通过环境传播计算 `<psi|Z_i|psi>`。  
7. `exact_local_z_expectation` 直接从全态张量计算真值，与 MPS 收缩结果逐站点比较。  
8. `main` 汇总打印并执行断言：误差随 `chi` 单调不增、`chi=32` 近似精确、`chi=8` 保持可用压缩质量。

说明：第三方库仅提供线性代数基础算子，MPS/TT-SVD 分解逻辑、振幅收缩与观测量收缩都在源码中显式实现，不是黑盒调用。
