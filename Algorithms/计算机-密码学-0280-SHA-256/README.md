## R01

`SHA-256`（Secure Hash Algorithm 256-bit）是 `SHA-2` 家族中的经典密码学哈希函数。它接收任意长度输入，输出固定 `256 bit`（32 字节）摘要，常用于完整性校验、数字签名预处理、区块链数据结构等场景。

## R02

本条目的目标是提供一个“可运行、可审阅、可验证”的最小 MVP：
- 在 `demo.py` 中源码实现 SHA-256 的填充、消息扩展和 64 轮压缩；
- 使用标准测试向量与 `hashlib.sha256` 做交叉验证；
- 保证 `uv run python demo.py` 无交互直接执行。

## R03

核心机制（Merkle-Damgard 迭代压缩）：
- 将消息按 `512 bit`（64 字节）分块；
- 每块扩展为 `64` 个 32-bit 消息字；
- 用 8 个 32-bit 工作寄存器执行 64 轮布尔/位运算混合；
- 每块结束后把工作寄存器回加到链值，形成下一块输入状态。

## R04

关键参数：
- 分组大小：`512 bit`；
- 输出大小：`256 bit`；
- 字长：`32 bit`；
- 轮数：`64`；
- 初始向量（8 个 32-bit 常量）为：
  - `6a09e667 bb67ae85 3c6ef372 a54ff53a`
  - `510e527f 9b05688c 1f83d9ab 5be0cd19`
- 每轮使用一个常量 `K[t]`（共 64 个常量）。

## R05

`demo.py` 对外接口：
- `sha256_digest(data: bytes) -> bytes`：返回 32 字节摘要；
- `sha256_hexdigest(data: bytes) -> str`：返回 64 字符十六进制摘要；
- `main()`：执行自检并打印样例摘要。

## R06

实现约束与数据表示：
- 仅依赖 Python 标准库（`hashlib` 仅用于结果对照，不参与主算法计算）；
- 32-bit 运算使用 `& 0xFFFFFFFF` 显式截断；
- 所有字节解析与输出采用 big-endian；
- 消息长度字段按 64-bit 无符号整数编码（符合 SHA-2 规范语义）。

## R08

复杂度分析：
- 设输入长度为 `n` 字节；
- 分块数约为 `ceil((n+9)/64)`；
- 每块固定 64 轮，所以总时间复杂度为 `O(n)`；
- 额外空间主要是 64 个消息字与少量寄存器，空间复杂度为 `O(1)`（不计输入输出本身）。

## R09

正确性要点（直观）：
- 填充阶段附加原始比特长度，避免不同长度消息在块边界上产生歧义；
- 消息扩展和轮函数把局部输入快速扩散到全局状态；
- 链式回加（feed-forward）使每一块都不可被后续块“覆盖”，最终摘要依赖全部历史块。

## R13

安全注意事项：
- SHA-256 适合抗碰撞完整性场景，但它本身不提供“密钥化认证”；
- 需要抗篡改认证时应使用 `HMAC-SHA-256` 等构造；
- 口令存储不应直接用单次 SHA-256，应使用 `Argon2 / scrypt / PBKDF2` 等慢哈希方案。

## R14

与相近算法对比（简述）：
- 对比 SHA-1：SHA-256 输出更长且安全裕量更高；
- 对比 SHA-512：SHA-512 在 64-bit 平台常更快，但输出更长（512 bit）；
- 对比 SHA-3：SHA-3 使用海绵结构，内部设计路线与 SHA-2 不同。

## R18

源码级算法流程拆解（对应 `demo.py`，非第三方黑盒）：
1. 读取输入字节串，计算原始比特长度 `len(data) * 8`。  
2. 执行填充：追加 `0x80`，补 `0x00` 直到长度满足 `mod 64 == 56`，再追加 8 字节长度字段。  
3. 将填充后的消息按 64 字节切块，每块解析为 16 个 big-endian 的 32-bit 字。  
4. 用小 sigma 函数扩展消息字：`W[16..63]` 由 `W[t-16], W[t-15], W[t-7], W[t-2]` 组合得到。  
5. 以当前链值初始化 8 个工作寄存器 `(a,b,c,d,e,f,g,h)`。  
6. 进行 64 轮压缩：每轮计算 `Ch/Maj`、大 Sigma、`temp1/temp2`，并更新 8 个寄存器。  
7. 块结束后将寄存器回加到链值 `H0..H7`（模 `2^32`），再处理下一块。  
8. 所有分块完成后，把 `H0..H7` 按 big-endian 拼接成 32 字节摘要；`main` 再与 `hashlib.sha256` 做一致性验算。
