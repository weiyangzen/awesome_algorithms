# 同调群计算

- UID: `MATH-0247`
- 学科: `数学`
- 分类: `计算拓扑`
- 源序号: `247`
- 目标目录: `Algorithms/数学-计算拓扑-0247-同调群计算`

## R01

同调群计算是计算拓扑中的基础任务：把几何对象离散成链复形，通过边界算子提取连通分支、环路和空腔等拓扑不变量。

本目录提供一个最小可运行 MVP：
- 输入有限单纯复形（用极大单纯形给出）
- 自动补全闭包并构造链群边界矩阵
- 在 `Z/2Z` 系数下计算各维 Betti 数
- 以 `H_k ≅ (Z/2Z)^{beta_k}` 形式输出同调群摘要

## R02

数学背景（有限单纯复形）：
- `C_k`：由所有 `k` 维单纯形生成的链群（在本实现里是 `Z/2Z` 向量空间）
- `d_k: C_k -> C_{k-1}`：边界算子
- `Z_k = ker(d_k)`：`k`-循环群
- `B_k = im(d_{k+1})`：`k`-边界群
- `H_k = Z_k / B_k`：第 `k` 同调群

在域系数下有维数公式：
`beta_k = dim C_k - rank(d_k) - rank(d_{k+1})`。

## R03

MVP 的输入输出约定：
- 输入：极大单纯形列表，例如
- `[(0,1),(1,2),(0,2)]` 表示三角形边界（`S^1`）
- `[(0,1,2)]` 表示填充三角形（可收缩）
- 输出：
- 各维单纯形个数 `n_k`
- 边界矩阵秩 `rank(d_k)`
- Betti 数 `beta_k`
- 同调群摘要 `H_k ≅ (Z/2Z)^{beta_k}`
- 链复形一致性检查 `d_{k-1} d_k = 0`

## R04

核心数据结构：
- `Simplex = Tuple[int, ...]`：排序后的顶点元组
- `ComplexByDim = Dict[int, List[Simplex]]`：按维组织单纯形
- `BoundaryMap = Dict[int, np.ndarray]`：保存各阶边界矩阵
- `HomologyResult`：单个样例的完整统计结果（计数、秩、Betti、群摘要）

边界矩阵使用 `numpy.uint8`，所有运算在 `GF(2)`（模 2）下进行。

## R05

算法主流程：
1. 由极大单纯形构造闭包，得到所有非空子面。
2. 按维度分组，得到链群基 `C_0, C_1, ..., C_n`。
3. 对每个 `k>=1` 构造边界矩阵 `d_k`。
4. 使用 `GF(2)` 高斯消元求 `rank(d_k)`。
5. 用 `beta_k = n_k - rank(d_k) - rank(d_{k+1})` 计算 Betti 数。
6. 输出 `H_k ≅ (Z/2Z)^{beta_k}`。
7. 验证 `d_{k-1} d_k = 0`，确认链复形关系成立。

## R06

正确性要点：
- 闭包构造保证链复形完整，不会缺少低维面。
- 边界矩阵按“删去一个顶点得到余维 1 面”逐列构建，符合单纯同调定义。
- 在 `Z/2Z` 下方向符号抵消，边界条目只需 `0/1`。
- 高斯消元提供精确的模 2 秩，代入 Betti 公式即可得同调维数。
- 检查 `d_{k-1} d_k = 0` 是实现正确性的关键代数约束。

## R07

复杂度分析（`N_k` 为 `k` 维单纯形个数）：
- 闭包构造：与输入极大单纯形的全部子面总数同阶。
- 边界矩阵构造：`d_k` 约 `O(k * N_k)` 个面访问与索引查询。
- 秩计算：若矩阵规模 `m x n`，GF(2) 消元约 `O(min(m,n) * m * n)`。

该 MVP 面向教学、验证与中小规模数据，不针对超大复形优化。

## R08

实现细节约定：
- 系数固定为 `Z/2Z`，不计算整数同调中的扭结分量。
- 所有单纯形统一排序并转为 tuple，保证去重与哈希稳定。
- 通过字典索引加速“面 -> 行号”映射。
- 边界矩阵可能为空矩阵，秩按 0 处理。

## R09

能力边界：
- 支持对象：有限单纯复形。
- 不支持：一般 CW 复形、流形网格定向符号跟踪、整数 SNF 扭结分解。
- 输出的是 `Z/2Z` 系数同调群摘要，而非 `Z` 系数完整结构。
- 对高维/高规模复形，运行时间会受组合爆炸影响。

## R10

为何选择 `Z/2Z` 系数：
- 可以避免定向符号处理，最小实现更稳健。
- 便于直接用位运算完成矩阵消元。
- 常用于拓扑数据分析入门和快速原型验证。

如果要得到 `Z` 系数同调群（含扭结），需要引入 Smith Normal Form 等整数矩阵算法。

## R11

`demo.py` 内置 4 个标准样例：
1. `S^1`（三角形边界）
2. 填充三角形（二维圆盘）
3. 四面体边界（`S^2`）
4. 两个不相交圆环（`S^1 ⊔ S^1`）

这些样例覆盖 `beta0/beta1/beta2` 的常见组合，适合回归测试。

## R13

输出字段解读：
- `simplex counts n_k`：各维链群维数
- `rank(d_k)`：边界算子秩
- `betti beta_k`：同调群自由维数（在 `Z/2Z` 下）
- `homology summary`：`H_k ≅ (Z/2Z)^{beta_k}`
- `boundary consistency`：是否满足 `d_{k-1} d_k = 0`

当一致性检查为 `True` 且 Betti 数与已知拓扑一致时，可认为实现可信。

## R14

与“直接调用黑盒库”的差异：
- 本实现没有把同调计算交给一行 API。
- 边界构造、GF(2) 消元、Betti 计算都在源码中显式展开。
- 便于代码审计、教学演示、以及后续替换为持久同调版本。

## R15

最小测试清单（`demo.py` 已覆盖）：

## R18

`demo.py` 源码级算法流（8 步，非黑盒）：
1. `closure_of_maximal` 把极大单纯形展开成完整闭包（包含所有子面）。
2. `group_simplices_by_dim` 将闭包按维度分组，确定链群基。
3. `boundary_matrix_mod2` 对每个 `k` 构造 `d_k: C_k -> C_{k-1}` 的模 2 矩阵。
4. `gf2_rank` 对每个 `d_k` 进行 GF(2) 高斯消元，得到 `rank(d_k)`。
5. `compute_betti_numbers` 使用维数公式逐维求 `beta_k`。
6. `verify_boundary_square_zero` 检查相邻边界复合是否全零（链复形约束）。
7. `format_homology_groups` 将 Betti 数转为 `H_k ≅ (Z/2Z)^{beta_k}` 字符串摘要。
8. `analyze_complex` 汇总并打印结果，`main` 依次运行四个标准拓扑样例。
