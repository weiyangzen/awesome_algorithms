## R01

单纯复形构造（Simplicial Complex Construction）是计算拓扑的基础步骤：
把离散数据（点云、邻接关系）组织为由 0-单纯形（点）、1-单纯形（边）、2-单纯形（三角形）等组成的组合结构，并满足“若一个单纯形在复形中，则它的全部面也在复形中”。

本目录采用最常见且工程上易实现的 Vietoris-Rips（VR）复形作为 MVP。

## R02

问题建模：
- 输入：点云 `X in R^(n x d)` 与尺度参数 `epsilon`。
- 输出：截至某一维度（本实现到 2 维）的单纯复形 `K`。
- 判定规则（VR）：
  - 顶点：每个点对应一个 0-单纯形；
  - 边：若两点距离 `<= epsilon`，加入 1-单纯形；
  - 三角形：若三点两两均有边（即形成 3-clique），加入 2-单纯形。

## R03

为什么需要“构造单纯复形”而不是直接看点云：
- 点云本身只给几何坐标，不直接给“拓扑连接关系”；
- 复形将局部几何关系转为可代数处理的链群结构；
- 后续同调、Betti 数、持久同调都建立在复形与边界算子之上。

## R04

VR 复形的关键直觉是“距离阈值导致连接”，并在更高维上闭包：
- `epsilon` 小：复形稀疏，连通分量多；
- `epsilon` 大：边和高维单纯形迅速增多，孔洞可能被填平。

因此单纯复形构造本质上是“尺度驱动的拓扑结构生成”。

## R05

与常见复形类型对比：
- Cech 复形：基于球体交集，几何解释强，但实现代价较高；
- Vietoris-Rips 复形：仅依赖两两距离，构造简单，工程中最常用；
- Alpha 复形：依赖 Delaunay 结构，维度高时代价上升明显。

本任务选 VR 是为了保持 MVP 透明、可复现、易扩展。

## R06

数学对象（本实现对应）：
- `C0, C1, C2`：分别由顶点、边、三角形生成的链群（实系数）；
- 边界算子：
  - `d1: C1 -> C0`（边映射到端点差）；
  - `d2: C2 -> C1`（三角形映射到有向边和）。
- 链复形一致性：`d1 * d2 = 0`。

`demo.py` 会优先使用 PyTorch（若可用）做秩计算；若环境缺失则回退到 NumPy，但都会显式验证该恒等式。

## R07

复杂度（构造到 2-单纯形）：
- 距离矩阵：`O(n^2 d)`；
- 边枚举：`O(n^2)`；
- 三角形枚举（朴素）：`O(n^3)`；
- 边界矩阵构造：`O(|E| + |T|)`。

当 `n` 增大时，三角形数量可能爆炸，通常需要控制 `epsilon` 或使用稀疏邻接策略。

## R08

本 MVP 的工程取舍：
- 只构造到 2-维，避免高维 clique 枚举失控；
- 用 `k` 近邻距离分位数自动选 `epsilon`，避免手工调参；
- 用紧凑数据结构（元组列表 + 索引字典）支持边界矩阵快速建立。

目标是“可读 + 可跑 + 可验证”，而不是大规模性能最优。

## R09

依赖栈与职责：
- `numpy`：组合逻辑与数值处理；
- `scipy`：`pdist/squareform` 计算距离矩阵；
- `pandas`：输出维度统计表；
- `scikit-learn`：生成样本点云、估计 `epsilon`；
- `torch`：边界算子秩计算的优先后端（缺失时自动回退到 `numpy`）。

这样可以避免把“复形构造”完全外包给黑箱库函数。

## R10

输出指标（`demo.py` 默认打印）：
- 各维单纯形计数 `n0, n1, n2`；
- Euler 示性数 `chi = n0 - n1 + n2`；
- Betti 数估计（实系数）`beta0, beta1, beta2`；
- `max|d1*d2|`（链复形一致性检查，应为 0）。

这些指标足以作为最小正确性回归基线。

## R14

结果解释建议：
- `beta0` 近似连通分量数，通常阈值增大时会下降到 1；
- `beta1` 反映 1 维孔洞数量，常在中等 `epsilon` 区间出现峰值；
- 当 `max|d1*d2| = 0` 时，说明边界算子方向和索引构造一致。

若该值非 0，优先排查三角形边界符号或边索引映射。

## R15

常见失败模式：
- `epsilon` 太小：几乎无边，复形过稀，拓扑信息不足；
- `epsilon` 太大：复形趋于“填满”，`beta1` 快速衰减；
- 点数太多：`O(n^3)` 三角形枚举过慢；
- 边界符号约定不统一：会破坏 `d1*d2=0`。

应对策略：固定随机种子，先调 `epsilon`，再做规模扩展。

## R18

源码级算法流程（对应本目录 `demo.py`，9 步）：
1. 调用 `make_circles` 生成二维点云，并用 `StandardScaler` 标准化。  
2. 使用 `NearestNeighbors` 求每个点的第 `k` 近邻距离，并取分位数得到 `epsilon`。  
3. 用 `scipy.spatial.distance.pdist/squareform` 构造全量欧氏距离矩阵 `D`。  
4. 基于阈值 `D[i,j] <= epsilon` 生成所有 1-单纯形（边）。  
5. 枚举三点组合 `(i,j,k)`，仅当三条边都存在时加入 2-单纯形（三角形）。  
6. 组装 `d1`：每条边在端点行写入 `-1/+1`，得到 `C1 -> C0` 的边界矩阵。  
7. 组装 `d2`：每个三角形按 `((b,c), -(a,c), +(a,b))` 写入，得到 `C2 -> C1` 的边界矩阵。  
8. 用矩阵乘法（优先 PyTorch，缺失时 NumPy）计算 `d1 @ d2` 并检查最大绝对值，验证链复形恒等式 `d1*d2=0`。  
9. 统计 `n0,n1,n2`、Euler 示性数与 Betti 估计，最终打印摘要与单纯形预览。
