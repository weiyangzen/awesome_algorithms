# 磁偶极子场 (Magnetic Dipole Field)

- UID: `PHYS-0020`
- 学科: `物理`
- 分类: `静磁学`
- 源序号: `20`
- 目标目录: `Algorithms/物理-静磁学-0020-磁偶极子场_(Magnetic_Dipole_Field)`

## R01

磁偶极子场用于描述“局域电流分布在远场中的主导磁场分量”。

在点偶极子近似下，偶极矩为 `m`（单位 `A·m^2`），观测点位置向量为 `r`，则在 `r != 0` 处：

`B(r) = (mu0 / (4*pi*r^3)) * [3(m·r_hat)r_hat - m]`

其中 `r_hat = r / |r|`，`mu0` 为真空磁导率。该模型是静磁学多极展开的首个非零项（单极项不存在）。

## R02

- 理论背景: 麦克斯韦方程组与磁矢势方法。
- 历史来源: 19 世纪电磁理论建立后，磁偶极近似被系统用于条形磁体、线圈远场与地磁建模。
- 在算法层的定位: 给定 `m` 与一批空间点，批量计算 `B(r)` 的向量化核函数。

## R03

问题背景是“如何用最小参数描述复杂电流源的远场磁分布”。

对任意有限尺度电流源，若观测距离 `r` 远大于源尺寸 `a`，磁场可做多极展开，偶极项通常主导。相比直接做 Biot-Savart 体积分，偶极模型只需一个向量 `m`，计算成本低且可解释性强。

## R04

核心数学关系：

1. 偶极矩定义（连续电流）:
   `m = (1/2) * ∫ r' x J(r') dV`
2. 磁矢势（点偶极子）:
   `A(r) = (mu0 / (4*pi)) * (m x r) / r^3`
3. 磁场:
   `B = ∇ x A`
4. 展开后得到 R01 的闭式公式。

本目录 MVP 直接实现闭式公式，同时用数值检验验证其关键物理性质。

## R05

时间复杂度（`N` 个采样点）：

- 向量化点场计算: `O(N)`
- 经验散度检验（`nx*ny*nz = G` 个网格点）: `O(G)`
- 远场对照表（`K` 个轴向采样）: `O(K)`

总复杂度近似线性，主常数来自浮点向量运算。

## R06

空间复杂度：

- 点集与磁场数组各存 `N x 3` 浮点数，额外空间 `O(N)`。
- 网格验证阶段需要 `O(G)` 存储 `B(x,y,z)` 与中间梯度。

## R07

示例（轴线上解析值）：若 `m` 沿 `+z` 方向，取 `r = (0, 0, z)`，则

`Bz = mu0 * m / (2*pi*z^3)`，`Bx = By = 0`。

可见磁场强度随距离按 `1/z^3` 衰减，这也是偶极远场最关键的标度律。

## R08

物理意义：

- 将复杂电流源压缩为“强度 + 朝向”两个层面的等效描述（向量 `m`）。
- 解释磁体远场、线圈远场、原子/分子磁矩等问题时非常高效。
- 与电偶极场在数学结构上高度对应，便于跨学科迁移。

## R09

适用前提与边界：

- 适用: 静磁或准静态条件、观测点远离电流源几何尺度。
- 不适用: 观测点进入源内部细节区（`r ~ a`）时，点偶极近似误差明显。
- 奇点: 点偶极模型在 `r = 0` 处发散，数值实现必须显式避开或做正则化。

## R10

正确性依据（本条目实现层）：

1. 公式层: 使用标准闭式 `B(r)`（见 R01）。
2. 物理约束: 在无源区域应满足 `∇·B = 0`。
3. 标度约束: 固定方向上 `|B(2r)| = |B(r)| / 8`。
4. 交叉验证: 与有限半径圆电流环轴线精确解对比，确认“远场趋近偶极”。

## R11

数值稳定性与误差来源：

- 近奇点误差: `1/r^3` 在小 `r` 下放大舍入误差，故需最小半径阈值。
- 网格散度检验误差: 由有限差分截断误差和浮点误差共同决定。
- 远场对照误差: 来自模型误差（偶极近似本身），非数值不稳定。

## R12

真实计算成本（工程视角）：

- 单点评估仅含点积、标量幂和向量线性组合，CPU 缓存友好。
- 批量点计算天然适合 NumPy 向量化。
- 若需要百万级点云，可进一步分块以控制峰值内存。
- 本 MVP 不依赖大型仿真框架，部署成本低。

## R13

误差/保证口径：

- 点偶极闭式本身在其模型假设下是确定性表达，不是随机算法。
- 与真实有限尺度电流源的误差随 `a/r` 增大而变大；通常 `r >> a` 时误差快速下降。
- 本 demo 给出圆环轴线的相对误差表，作为可复现实证。

## R14

鲁棒性与失效模式：

- 输入点包含原点或过近点时会触发半径阈值异常。
- 若 `m = 0`，场恒为零，缩放比检验会失去意义（分母为零风险）。
- 网格太粗会导致散度估计偏大；太密则增加计算与内存开销。

## R15

工程实现要点：

- 统一使用 SI 单位（`mu0`、`A·m^2`、`m`、`T`）。
- 将磁场核函数写成“批量点 -> 批量向量”的纯函数，便于复用。
- 对奇点做输入防护，而不是沉默地产生 `inf/nan`。
- 验证逻辑与计算逻辑分离，便于单测与扩展。

## R16

前驱/后继关系与应用：

- 前驱理论: Biot-Savart 定律、磁矢势、向量分析。
- 后继扩展:
  - 更高阶多极项（四极、八极等）。
  - 由离散线圈/电流元反演等效偶极矩。
  - 与磁力、力矩模型耦合（如 `tau = m x B`）。
- 应用场景: 地磁近似、磁传感器标定、磁操控与教育仿真。

## R17

Python MVP 方案（见 `demo.py`）：

- 依赖: `numpy` + `pandas`。
- 核心函数:
  - `dipole_field(points, m, min_radius)`：向量化计算点偶极磁场。
  - `divergence_check(m)`：在远离原点的盒域上做 `∇·B` 有限差分检验。
  - `inverse_cubic_scaling_check(m)`：验证 `1/r^3` 标度律。
  - `loop_axis_field_exact / loop_axis_field_dipole`：圆环轴线精确解与偶极近似对照。
- 运行方式:
  - `uv run python demo.py`
- 输出: 采样点场值、缩放检验表、圆环远场误差表、PASS/FAIL。

## R18

`demo.py` 源码级流程拆解（8 步）：

1. 在 `dipole_field` 中读取 `points` 与偶极矩 `m`，计算每个点的 `r = |x|`。
2. 先做半径阈值检查，拒绝奇点附近输入，避免 `1/r^3` 溢出。
3. 计算 `r_hat` 与 `m·r_hat`，按闭式 `B = c * (3(m·r_hat)r_hat - m)` 向量化得到全部点的磁场。
4. `divergence_check` 在不含原点的三维网格上评估 `B`，用 `np.gradient` 近似三方向偏导并求和得到离散散度。
5. `inverse_cubic_scaling_check` 对固定方向集分别在 `r` 与 `2r` 计算场强范数，比对理论比例 8。
6. `loop_axis_field_exact` 给出有限半径圆电流环轴线精确解；`loop_axis_field_dipole` 给出其偶极远场近似。
7. `build_loop_comparison_table` 生成 `z/a` 与相对误差表，量化“远场收敛到偶极”的速度。
8. `main` 汇总三类验证指标并判定阈值，全部通过输出 `Validation: PASS`，否则非零退出。
