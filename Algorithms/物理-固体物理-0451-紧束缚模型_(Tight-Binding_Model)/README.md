# 紧束缚模型 (Tight-Binding Model)

## R01

紧束缚模型（Tight-Binding Model, TB）是固体物理中描述电子在晶格中运动的经典近似方法。  
核心思想：电子主要局域在原子轨道上，只通过邻近原子发生“跃迁”（hopping），由此构造哈密顿量并求能带色散关系。  
本条目实现一个一维单原子链、最近邻跃迁的最小可运行 MVP。

## R02

问题设定（本 MVP）：

1. 晶格：一维周期链，共 `N` 个格点；
2. 基底：每个格点仅保留一个轨道（单带）；
3. 近似：只保留最近邻跃迁，跃迁幅 `t`，在位能 `epsilon_0`；
4. 边界：实空间采用周期边界条件（PBC）。

目标是同时从两条路径得到同一组本征能：
- 路径 A：构造实空间哈密顿矩阵后数值对角化；
- 路径 B：使用解析色散 `E(k)=epsilon_0-2t cos(ka)`。

## R03

数学模型：

`H = Σ_i epsilon_0 c_i^† c_i - t Σ_<i,j> (c_i^† c_j + c_j^† c_i)`  

其中 `<i,j>` 表示最近邻。对于一维链，动量表象可得

`E(k) = epsilon_0 - 2t cos(ka)`。

这是一条典型余弦能带，带宽为 `4t`。

## R04

离散化与可计算形式：

- 令格点编号 `i=0,...,N-1`；
- 哈密顿矩阵 `H` 为 `N x N` 实对称矩阵；
- 对角元：`H[i,i]=epsilon_0`；
- 最近邻元：`H[i,i+1]=H[i+1,i]=-t`；
- 周期闭合：`H[0,N-1]=H[N-1,0]=-t`。

此时可使用厄米矩阵本征值求解器（`scipy.linalg.eigh`）稳定计算本征能级。

## R05

本条目采用的最小工具栈与职责分工：

- `numpy`：数组运算、构造 `k` 网格与解析色散；
- `scipy`：数值对角化实空间哈密顿量；
- `pandas`：结果表格化输出；
- `scikit-learn`：线性回归拟合 `k≈0` 区域抛物线系数；
- `torch`：用独立线代后端复核本征值一致性。

这些库仅承担数值基础能力，算法逻辑（模型构建、校验、物理量提取）在源码中显式实现。

- `onsite_energy = 0.5`
- `hopping = 1.2`

## R09

复杂度分析：

- 构造哈密顿矩阵：`O(N)` 非零条目（若按稠密数组写入为 `O(N^2)` 存储）；
- 厄米矩阵本征分解（稠密）：时间 `O(N^3)`，空间 `O(N^2)`；
- 解析色散计算：`O(N)`；
- 回归拟合：`O(M)`（`M` 为拟合窗口样本数）。

对教学规模 (`N<=10^3`) 足够直接清晰。

## R12

物理解释要点：

- `E(k)` 为余弦形，`k=0` 处为带底（当 `t>0`）；
- 群速度 `v(k)=dE/dk = 2ta sin(ka)`；
- 曲率 `d^2E/dk^2 = 2ta^2 cos(ka)`；
- 带底附近可抛物线近似：`E ≈ E_0 + (hbar^2 k^2)/(2m*)`，因此  
  `m* = hbar^2 / (2 t a^2)`（脚本使用 `hbar=1` 归一化单位）。

## R13

适用范围：

- 周期晶体中窄能带近似；
- 教学/原型阶段的能带与态密度定性分析；
- 作为更复杂多轨道 TB、Hubbard、k·p 方法的起点。

局限性：

- 忽略电子-电子相互作用；
- 忽略远程跃迁与自旋轨道耦合；
- 对强关联材料通常不足。

## R14

常见错误与排查：

1. 周期边界漏连 `H[0,N-1]` 导致谱不匹配；
2. `k` 网格排序与本征值排序不一致，比较时误判；
3. 拟合窗口太大，`m*` 偏离理论值；
4. 混用单位（`a`、`hbar`、`t`）导致有效质量量纲错误。

## R18

`demo.py` 源码级算法流程（8 步）：

1. 读取 `TightBindingConfig` 参数，固定一维链规模与模型常数。  
2. `build_real_space_hamiltonian()` 按最近邻与周期边界构造 `N x N` 实对称矩阵。  
3. `allowed_k_points()` 生成离散布里渊区 `k` 网格，`analytic_band_energy()` 计算解析 `E(k)`。  
4. 调用 `scipy.linalg.eigh` 求实空间本征值并排序，得到 `E_real`。  
5. 将 `E_real` 与排序后的 `E_analytic` 做逐点比较，计算最大绝对误差。  
6. 用 `torch.linalg.eigvalsh` 对同一哈密顿量再次求谱，并与 SciPy 结果比较，形成后端一致性指标。  
7. 在 `|k|` 小窗口中构造特征 `k^2`，用 `sklearn.linear_model.LinearRegression` 拟合 `E≈E0+alpha*k^2`，再由 `m*=1/(2alpha)`（`hbar=1`）得到有效质量估计。  
8. 使用 `pandas` 输出能带采样表与关键指标，并执行断言（谱一致性、有效质量误差阈值），全部通过后打印 `All checks passed.`。  

说明：第三方库只提供线代/表格/回归基础算子；TB 模型构造、物理量提取、误差评估与验证逻辑均在源码中逐步显式实现，不是黑盒调用。
