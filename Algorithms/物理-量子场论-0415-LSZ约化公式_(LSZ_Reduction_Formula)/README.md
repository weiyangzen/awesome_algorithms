## R01

LSZ 约化公式（Lehmann-Symanzik-Zimmermann Reduction Formula）给出从时序关联函数（Green 函数）到散射振幅（S-matrix 元）的桥梁：

- 对每条外线做“截肢”（乘以 `p_i^2-m^2` 并取壳极限 `p_i^2 -> m^2`）；
- 再除以场强重整化残数因子（常写作 `Z^{1/2}` 每条外线）；
- 得到可观测的散射振幅 `M`。

它的核心意义是把“场论路径积分/算符语言”与“实验可测散射截面语言”连接起来。

## R02

本条目要解决的问题：
如何用一个最小、可运行、可审计的 Python 脚本，把 LSZ 约化从公式变成可验证的数值流程。

MVP 目标：

- 构造一个含单粒子极点的 2 点函数并从中估计 `Z`；
- 构造含外线传播子的 4 点连通函数 `G4`；
- 对 `G4` 执行 LSZ 截肢与 `Z` 因子归一；
- 恢复“真实设定的截肢振幅”并量化误差。

## R03

`demo.py` 的输入输出约定（无交互）：

- 输入（脚本内固定）：
1. 标量质量 `m=1.0`、极点调节 `epsilon=1e-3`；
2. 真实残数 `Z_true=0.82`；
3. 一个 toy 振幅（接触项 + `s` 道 Breit-Wigner 项）；
4. 一组壳附近 `p^2` 样本用于估计 `Z`；
5. 多个散射角与随机小离壳外线用于 LSZ 还原测试。
- 输出：
1. `Z_est` 与 `|Z_est-Z_true|`；
2. 每个测试点的 `(s,t,u)`、真实振幅、LSZ 还原振幅、相对误差表；
3. 最大/平均误差摘要；
4. 断言通过后打印 `All checks passed.`。

## R04

MVP 使用的数学模型（toy、可解释）：

1. 两点函数（壳附近）
`G2(p^2) = Z/(p^2-m^2+i eps) + regular(p^2)`。

2. 令
`y(delta) = (delta+i eps) G2, delta = p^2-m^2`，
则 `delta -> 0` 时 `y -> Z`，可用线性外推估计残数。

3. 四点连通函数（显式带外线传播子）
`G4 = Z^2 * M(s,t,u) / prod_i (p_i^2-m^2+i eps)`。

4. LSZ 截肢还原
`M_LSZ = prod_i (p_i^2-m^2+i eps) * G4 / Z_est^2`。

当离壳足够小且 `Z_est` 准确时，`M_LSZ ~ M_true`。

## R05

复杂度（设测试样本数为 `N`）：

- `Z` 估计（线性最小二乘，常数维）：`O(N)`；
- 每个样本做一次振幅构造与 LSZ 还原：`O(1)`；
- 全部样本表格汇总：`O(N)`；
- 空间复杂度：`O(N)`。

本条目属于“低维解析结构 + 轻量数值验证”，主成本在样本遍历与结果汇总。

## R06

算法闭环：

1. 生成壳附近 `p^2` 样本并构造 `G2`；
2. 计算 `y=(delta+i eps)G2`，拟合 `y≈a0+a1 delta`，取 `a0` 为 `Z_est`；
3. 生成多个 `2->2` 运动学点 `(s,t,u)`；
4. 为每个点随机生成小离壳外线 `p_i^2`；
5. 用 toy 模型构造 `G4`；
6. 用 LSZ 截肢公式计算 `M_LSZ`；
7. 与 `M_true` 比较并记录相对误差；
8. 通过断言检查残数误差和 LSZ 还原误差。

## R10

正确性直觉：

1. 外线传播子的极点行为被 `prod_i (p_i^2-m^2+i eps)` 抵消；
2. 剩余的残数因子由 `Z` 控制，除以 `Z^2` 后得到截肢振幅；
3. 若 `Z_est` 接近 `Z_true` 且离壳量小，则 `M_LSZ` 应稳定逼近 `M_true`；
4. 因此 `|Z_est-Z_true|` 与 `|M_LSZ-M_true|` 是自然的两级正确性指标。

## R11

数值稳定性策略：

- 使用有限 `epsilon` 避免严格极点除零；
- 只在小离壳区采样，控制高阶项污染；
- 残数通过线性外推而非单点估计，降低偶然噪声影响；
- 全流程使用复数计算，避免手工拆分实虚部误差。

## R12

关键参数与影响：

- `epsilon`：过小易放大数值噪声，过大偏离壳极限；
- 外线离壳范围 `|p_i^2-m^2|`：越小越贴近 LSZ 极限；
- `delta_grid`：决定 `Z` 外推精度；
- toy 振幅参数（`lambda_contact`, `resonance_g`, `resonance_mass`, `resonance_width`）：决定振幅复结构与误差尺度。

调参建议：

- 若误差偏大，先收紧离壳范围，再适度减小 `epsilon`；
- 若残数估计波动大，增加 `delta_grid` 点数并保持对称采样。

## R14

常见失效模式：

1. 把 LSZ 截肢因子写成错误幂次（少乘/多乘外线因子）；
2. 漏掉 `Z` 因子或幂次错误；
3. `epsilon` 选取不当导致数值不稳定或偏差过大；
4. 离壳样本太大，导致“极限公式”不再近似成立；
5. `(s,t,u)` 运动学构造不合法（如 `E_cm <= 2m`）。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `LSZToyConfig` 固定 toy 场论参数（`m`, `epsilon`, `Z_true`, 振幅参数等）。
2. `two_point_connected` 构造壳附近两点函数 `G2 = Z/(p^2-m^2+i eps) + regular`。
3. `estimate_field_strength_z` 计算 `y=(delta+i eps)G2`，并用复数线性最小二乘拟合 `y≈a0+a1 delta`，取截距 `a0` 作为 `Z_est`。
4. `mandelstam_2to2_equal_mass` 在给定 `E_cm, theta` 下构造 `(s,t,u)`，保证 `s+t+u=4m^2`。
5. `amputated_amplitude` 生成 toy 截肢振幅 `M_true(s,t,u)`（接触项 + `s` 道共振项）。
6. `connected_four_point` 显式拼接外线传播子，形成 `G4 = Z^2 M_true / prod_i(p_i^2-m^2+i eps)`。
7. `lsz_reduce_amplitude` 对 `G4` 乘外线截肢因子并除以 `Z_est^2`，得到 `M_LSZ`。
8. `main` 批量比较 `M_LSZ` 与 `M_true`、打印误差表并用断言验证算法闭环。

说明：`numpy`/`pandas` 只用于向量计算与报告整理；LSZ 的关键物理步骤（残数提取、外线截肢、`Z` 归一）均在源码中逐步展开，没有把核心逻辑交给第三方黑盒。
