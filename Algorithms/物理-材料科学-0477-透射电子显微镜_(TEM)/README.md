# 透射电子显微镜 (TEM)

## R01

透射电子显微镜（TEM）的核心思想是：高能电子透过超薄样品后形成出射波，出射波经过物镜相位调制（离焦、球差、孔径）后在像平面形成强度分布，进而反映材料内部结构。

本条目给出一个最小可运行的 TEM 前向仿真 MVP：用二维晶格投影势生成出射波，再通过简化物镜传递函数形成图像，并配套自动断言验证关键物理关系。

## R02

TEM 在材料科学中的典型用途：

- 晶格成像与缺陷识别（空位、杂质、位错附近局域对比）；
- 通过离焦序列理解相位-强度传递与 CTF（Contrast Transfer Function）行为；
- 为更复杂的多片层（multislice）或实验拟合提供可审计的算法基线。

本 MVP 重点是“可解释算法路径”，不是工业级定量显微镜模拟器。

## R03

MVP 目标闭环：

1. 从加速电压计算相对论电子波长；
2. 构造含掺杂与空位扰动的晶格投影势；
3. 将投影势映射为复数出射波；
4. 施加物镜传递函数（离焦 + 球差 + 孔径）；
5. 形成像强度与功率谱，并自动做物理一致性检查。

## R04

本实现的核心公式：

1. 相对论电子波长：
`lambda = h / sqrt(2 m e V * (1 + eV/(2mc^2)))`

2. 弱相位风格出射波（简化写法）：
`psi_exit(x,y) = exp(i * alpha * V_proj(x,y))`

3. 物镜像差相位函数：
`chi(k) = pi * lambda * Deltaf * k^2 + (pi/2) * Cs * lambda^3 * k^4`

4. 相干传递函数：
`H(k) = A(k) * exp(-i * chi(k))`
其中 `A(k)` 为孔径掩膜（由最大散射角决定）。

5. 像平面波函数与强度：
`psi_img = F^-1(F(psi_exit) * H)`，`I = |psi_img|^2`

6. 频域主峰估计：
在去掉中心直流分量后，从功率谱中取主非零峰，验证其与 `1/a` 的一致性。

## R05

设网格边长 `N`：

- 构建投影势与高斯原子核展宽：`O(N^2)`；
- FFT 成像（一次前向一次逆向）：`O(N^2 log N)`；
- 功率谱与径向统计：`O(N^2)`。

总时间复杂度主导项为 `O(N^2 log N)`，空间复杂度约 `O(N^2)`。

## R07

优点：

## R10

正确性直觉：

1. `200 kV` 电子波长应约 `2.5 pm`；
2. 晶格常数为 `a` 时，主倒易峰应接近 `1/a`；
3. 正负离焦应导致对比传递发生变化，图像相关性不应无限接近 1；
4. 强度图应非负且有限，且存在可观测对比度。

这些直觉在代码中都有对应断言。

## R11

数值实现细节：

- 对势场和强度做归一化，防止量纲差异造成比较失真；
- 主谱峰搜索排除低频中心区，避免直流分量干扰；
- 使用 `np.isfinite` 和非负性断言保障结果可用性；
- 以确定性参数运行，不依赖随机交互输入。

## R12

关键参数与影响：

- `accelerating_voltage_kv`：决定电子波长，从而影响可传递空间频率；
- `defocus_nm`：改变 CTF 零点位置，显著影响晶格条纹对比；
- `spherical_aberration_mm`：高频相位项，影响高分辨信息传递；
- `aperture_mrad`：限制最大散射角，相当于频域带宽裁剪；
- `lattice_constant_angstrom`：决定功率谱主峰位置；
- `thickness_scale` 与 `phase_gain_rad_per_norm_potential`：控制相位调制强度和图像对比。

## R14

常见失效模式：

1. 像素尺寸或单位误填（Å 与 m 混淆）导致频率标定错误；
2. 孔径太小导致高频信息全部截断，晶格峰消失；
3. 相位增益过小导致图像接近常数，无法形成有效对比；
4. 网格过小导致频谱分辨率不足，峰位估计偏差大；
5. 仅用单一离焦解释实验图像，忽略像差和样品厚度效应。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. 在 `TEMConfig` 中固定电压、离焦、球差、孔径、晶格常数和采样网格，形成确定性仿真上下文。  
2. `build_projected_potential` 先在规则晶格点放置离散原子脉冲，再插入一个掺杂点与一个空位，并通过 `gaussian_filter` 生成连续投影势场。  
3. `electron_wavelength_relativistic` 按相对论公式计算电子波长 `lambda`，把显微镜高压映射到成像尺度。  
4. `build_exit_wave` 使用 `psi_exit = exp(i * alpha * V_proj)` 把实值势场转换为复数出射波，显式保留相位调制。  
5. `objective_transfer_function` 在频域网格上计算像差相位 `chi(k)`，再结合孔径掩膜得到 `H(k)=A(k)exp(-i chi)`。  
6. `simulate_tem_image` 对出射波做 FFT，乘以 `H(k)` 后逆 FFT 得到像波函数，并取模平方形成归一化强度图；同时计算中心化功率谱。  
7. `estimate_primary_spatial_frequency` 与 `radial_profile` 从功率谱提取主非零峰和径向统计，用于检查晶格频率与频谱结构。  
8. `run_validations` 执行四类断言（波长窗口、主峰误差、对比度阈值、正负离焦差异），`main` 输出统计表并在全部通过时打印成功信息。  

第三方库仅提供基础数值能力：`numpy` 负责数组与 FFT，`scipy` 提供物理常数和高斯滤波，`pandas` 用于结构化报告；TEM 物理流程本身在源码中逐步显式实现，不依赖黑箱调用。
