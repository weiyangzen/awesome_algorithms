## R01

阿尔芬波（Alfvén waves）是磁化等离子体中的横向磁流体波。  
在均匀背景磁场 `B0`、低频、理想 MHD 线性近似下，扰动速度 `v_perp` 与扰动磁场 `b_perp` 耦合传播，其相速为阿尔芬速度：

`v_A = B0 / sqrt(mu0 * rho0)`

这里 `rho0` 是质量密度，`mu0` 是真空磁导率。

## R02

本条目聚焦最小可运行的一维线性问题（沿 `x` 传播）：

- 背景场 `B0` 沿 `x`；
- 仅考虑横向分量 `v_y, b_y`；
- 采用周期边界，避免复杂边界反射处理；
- 用解析平移解对照数值结果，验证“波形传播速度与幅值关系”。

## R03

MVP 目标：

- 实现线性化阿尔芬波方程的显式数值推进；
- 使用 Elsässer 变量将耦合系统拆为两个对流方程；
- 给出 `v_y` 与 `b_y` 的数值/解析误差；
- 检查扰动总能量漂移与小振幅线性条件；
- 脚本可通过 `uv run python demo.py` 直接运行，无需交互输入。

## R04

`demo.py` 使用的连续方程（SI 单位）：

1. `d_t v_y = (B0/(mu0*rho0)) d_x b_y`
2. `d_t b_y = B0 d_x v_y`

令 `b_bar = b_y / sqrt(mu0*rho0)`，定义 Elsässer 变量：

- `z_plus = v_y + b_bar`
- `z_minus = v_y - b_bar`

可化为两条独立线性对流方程：

- `d_t z_plus  - v_A d_x z_plus  = 0`（向左传播）
- `d_t z_minus + v_A d_x z_minus = 0`（向右传播）

本实现初始化 `z_minus` 正弦波、`z_plus=0`，得到单向传播的阿尔芬波包。

## R05

离散与复杂度（网格数 `N`，时间步数 `T`）：

- 空间：均匀网格、周期边界；
- 时间推进：二阶 Lax-Wendroff 离散对流方程；
- 每步使用 `roll` 计算邻点差分，复杂度 `O(N)`；
- 总时间复杂度 `O(N*T)`；
- 存储 `z_plus/z_minus/v_y/b_y` 等数组，空间复杂度 `O(N)`。

## R10

正确性直觉：

1. 线性系统可拆为左右传播的特征变量；
2. 若数值对流离散正确，`z_minus` 应按速度 `v_A` 平移而不改变基本波形；
3. 由 `z_plus/z_minus` 反算的 `v_y` 和 `b_y` 应同时匹配解析解；
4. 在稳定 CFL 下，总扰动能量只应有小幅数值耗散；
5. 若 `max(|v_y|)/v_A << 1`，线性假设自洽。

`demo.py` 内置阈值对应上述 2-5 条。

## R11

数值稳定性与鲁棒性处理：

- 显式限制 `cfl <= 0.95`；
- 每步 `dt = min(dt_nominal, t_end - t)`，确保终止时间精确命中；
- 参数合法性检查：`n_grid`、`B0`、`rho0`、幅值必须为正；
- 误差分母加 `1e-12` 防止零除；
- 验证阶段检查数组有限性（无 NaN/Inf）。

## R12

关键参数（`AlfvenConfig`）：

- `background_b0_t`：背景磁场强度，决定 `v_A`；
- `mass_density_kg_m3`：质量密度，和 `B0` 一起决定 `v_A`；
- `domain_length_m`：空间周期长度；
- `n_grid`：网格分辨率，决定误差与计算开销；
- `cfl`：时间步安全系数；
- `wave_amplitude_m_s`：初始 `z_minus` 幅值；
- `n_periods`：模拟总时长（以波通过一个周期长度为单位）。

默认参数下，`max(|v_y|)/v_A` 明显小于 1，保持线性波近似。

## R14

常见失效模式：

1. `cfl` 过大导致显式推进不稳定；
2. 网格太粗，平移误差和色散/耗散误差明显增大；
3. 输入 `B0 <= 0` 或 `rho0 <= 0` 导致 `v_A` 无定义；
4. 幅值过大使线性近似不再成立；
5. 解析解符号或传播方向写错会导致系统性相位偏差。

脚本对 1、3、4 提供了直接检查或诊断指标。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `AlfvenConfig` 固化背景磁场、密度、网格、CFL、扰动幅值和模拟时长参数。  
2. `alfven_speed` 根据 `v_A = B0/sqrt(mu0*rho0)` 计算特征传播速度。  
3. `build_initial_state` 构造周期网格与初值：`z_minus` 为正弦波、`z_plus=0`。  
4. `lax_wendroff_periodic` 对每个 Elsässer 变量执行一次二阶 Lax-Wendroff 显式推进（周期边界用 `np.roll` 实现）。  
5. `simulate` 在时间循环中分别以速度 `-v_A` 和 `+v_A` 更新 `z_plus/z_minus`，直至 `t_end`。  
6. `exact_solution` 生成同一时刻的解析平移解；`to_velocity_and_magnetic_perturbation` 将 `z_plus/z_minus` 还原为 `v_y/b_y`。  
7. 计算 `v_y` 与 `b_y` 的 L2 相对误差、扰动总能量漂移、以及 `max(|v_y|)/v_A` 线性小参数。  
8. `main` 打印采样表与检查项，全部达标输出 `Validation: PASS`，否则以非零状态退出。

第三方库未被当作黑盒求解器：

- `numpy` 仅用于数组与离散运算；
- `pandas` 仅用于结构化打印结果表；
- 线性 MHD 模型、变量变换、时间推进与验证逻辑均在源码中显式实现。
