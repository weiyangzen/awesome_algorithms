## R01

陈数（Chern Number）是二维绝缘体能带在布里渊区上的拓扑不变量。对单个占据能带可写为：

`C = (1 / 2π) * ∫_BZ Ω(kx, ky) dkx dky`

其中 `Ω` 是 Berry 曲率。`C` 必须是整数，且在能隙不闭合时对连续扰动不变。

## R02

在拓扑物理中，陈数直接区分不同拓扑相：

- `C = 0` 通常对应平庸绝缘相；
- `C != 0` 对应非平庸拓扑相；
- 在量子反常霍尔体系中，霍尔电导满足 `σ_xy = C * e^2 / h`。

所以陈数是“从哈密顿量判断是否存在拓扑保护边界态”的核心计算对象。

## R03

本条目 MVP 选择最小可复现路径：

- 使用 Qi-Wu-Zhang 两能带格点模型；
- 在离散布里渊区上用 Fukui-Hatsugai-Suzuki (FHS) 格点规范算法计算陈数；
- 对多个参数 `m` 输出相图切片并做自动断言。

重点是实现“可追踪、可运行、可验证”的基础算法闭环，而不是大而全框架。

## R04

`demo.py` 对应的数学对象：

1. 模型哈密顿量：
   `H(k) = sin(kx)σx + sin(ky)σy + (m + cos(kx) + cos(ky))σz`。
2. 每个 `k` 点取最低能带本征矢 `|u(k)>`。
3. 链变量（U(1) 规范）：
   `Ux(k) = <u(k)|u(k+Δx)> / |<u(k)|u(k+Δx)>|`，
   `Uy(k) = <u(k)|u(k+Δy)> / |<u(k)|u(k+Δy)>|`。
4. 单元格 Berry 通量：
   `F(k) = Arg[ Ux(k) Uy(k+Δx) Ux(k+Δy)^(-1) Uy(k)^(-1) ]`。
5. 陈数离散和：
   `C ≈ (1 / 2π) * Σ_k F(k)`。

## R05

设动量网格大小为 `N x N`：

- 构造占据带本征矢：`O(N^2)` 次 `2x2` 本征分解；
- 计算链变量：`O(N^2)`；
- 计算每个 plaquette 通量并求和：`O(N^2)`；
- 总体时间复杂度 `O(N^2)`，空间复杂度 `O(N^2)`。

在本 MVP 中，`N=61` 已足够稳定得到整数化结果。

## R10

正确性直觉：

1. 陈数是 Berry 曲率在闭合二维流形上的总通量，理论上是整数；
2. FHS 把连续 Berry 曲率积分改写成每个离散小方格的规范不变量相位；
3. 每个小方格相位都是局域规范不变的，求和后得到全局拓扑整数；
4. 网格足够细且体系有能隙时，离散结果会稳定靠近整数。

## R11

数值稳定策略：

- 选取避开相变点的 `m`（不取 `-2, 0, 2`）；
- 对重叠 `<u|u'>` 做 `z/|z|` 的 U(1) 投影，抑制幅值漂移；
- 使用周期边界索引 `(i+1) % N` 保证 BZ 闭合；
- 通过 `abs(chern_raw - round(chern_raw)) < 1e-3` 监控离散误差。

## R12

关键参数：

- `m`：控制拓扑相，决定陈数所在分支；
- `n_k`：动量网格分辨率，越大通常越接近整数极限；
- `eps`（`safe_u1_phase` 中）：极小重叠时的数值保护阈值。

调参建议：

- 若 `chern_raw` 偏离整数，先提高 `n_k`；
- 若扫描相图，优先远离相变点再逐步逼近边界。

## R14

常见失效模式：

1. 在 `m=-2,0,2` 附近能隙闭合，数值结果可能不稳定；
2. 未做 U(1) 归一化，直接累乘重叠会引入幅值误差；
3. plaquette 方向顺序写反，陈数符号会整体翻转；
4. 忘记周期边界，导致 BZ 非闭合、通量统计错误。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `qwz_hamiltonian` 按 `sin/cos` 与泡利矩阵组合显式构造 `2x2` 厄米哈密顿量。  
2. `occupied_eigenvector` 对每个 `k` 点做 `np.linalg.eigh`，取最低本征值对应本征矢作为占据带波函数。  
3. `build_occupied_grid` 在 `[-π, π)` 的 `N x N` 周期网格上生成全部占据带本征矢。  
4. `safe_u1_phase` 把相邻点重叠 `<u|u'>` 投影到 U(1) 相位，得到无幅值污染的链变量。  
5. `fhs_chern_number` 计算 `Ux, Uy` 两组链变量，并在每个 plaquette 上按固定方向构造 Wilson 小回路相位。  
6. 对每个 plaquette 取 `np.angle(loop)` 得到离散 Berry 通量 `F(k)`，再执行 `ΣF/(2π)` 得到 `chern_raw`。  
7. 用 `np.rint` 得到 `chern_rounded`，并记录最大通量幅值等数值诊断信息。  
8. `main` 对四个 `m` 值重复上述流程，逐行打印结果并执行整数性与相区正确性的断言，全部通过后结束。  

说明：虽然 `numpy` 提供本征分解等基础算子，但拓扑计算核心（链变量、plaquette 回路、离散通量求和）是源码显式实现，不是黑盒一行调用。
