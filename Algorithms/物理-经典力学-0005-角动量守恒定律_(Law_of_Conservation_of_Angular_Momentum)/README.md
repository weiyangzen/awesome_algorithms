# 角动量守恒定律 (Law of Conservation of Angular Momentum)

- UID: `PHYS-0005`
- 学科: `物理`
- 分类: `经典力学`
- 源序号: `5`
- 目标目录: `Algorithms/物理-经典力学-0005-角动量守恒定律_(Law_of_Conservation_of_Angular_Momentum)`

## R01

角动量守恒定律研究的问题是：当系统所受合外力矩为零时，系统角动量 `L` 是否保持常量；当存在外力矩 `tau` 时，是否满足 `dL/dt = tau`。

本条目的 MVP 目标是把这个定律变成可运行代码，并通过数值实验给出可审计的验证指标，而不是仅停留在文字公式。

## R02

在经典力学中，角动量守恒与线动量守恒、能量守恒并列为核心守恒律。它广泛出现于：

- 天体运动（开普勒第二定律可由角动量守恒导出）；
- 刚体旋转（花样滑冰收臂加速、自旋稳定）；
- 机械系统设计（飞轮、陀螺、姿态控制）。

从工程实现角度，角动量守恒也是数值仿真“健康度”的关键监控量。

## R03

本 MVP 分三组可重复实验（均无需交互输入）：

1. 中心力闭轨道：检验封闭系统中 `L` 近似守恒；
2. 常外力矩转子：检验 `L(t)` 斜率是否等于 `tau`；
3. 零外力矩 + 时变转动惯量：检验 `I*omega` 是否守恒。

输出包括：

- 角动量相对漂移、能量相对漂移；
- 面积速度一致性与轨道周期误差；
- `tau` 拟合误差、线性残差；
- NumPy 与 PyTorch 角动量计算一致性误差。

## R04

核心方程如下：

- 定义：`L = r x p`，平面情况下 `Lz = m*(x*vy - y*vx)`；
- 角动量演化：`dL/dt = tau_ext`；
- 无外力矩条件：`tau_ext = 0 -> L = const`；
- 中心力：`F(r) || r`，因此 `tau = r x F = 0`，可推出角动量守恒；
- 面积速度关系：`dA/dt = L/(2m)`（开普勒第二定律的等价形式之一）。

`demo.py` 使用这些公式构造验证闭环，而非黑盒库直接给结论。

## R05

本条目采用三种最小可行建模：

- 中心力轨道：二维 `1/r^2` 力场，使用 Velocity-Verlet 积分；
- 常力矩刚体：离散积分 `L_{k+1}=L_k + tau*dt`；
- 时变惯量零力矩刚体：解 `I*omega' + I'*omega = 0`，采用 RK4。

选择理由：

- 覆盖“守恒”和“非守恒”两种工况；
- 既有轨道动力学，也有刚体旋转动力学；
- 每一步都能在源码中逐行追踪。

## R06

记时间步数为 `T`：

- 中心力轨道仿真：每步常数成本，时间复杂度 `O(T)`，空间复杂度 `O(T)`（保存全轨迹）；
- 常力矩仿真：时间复杂度 `O(T)`，空间复杂度 `O(T)`；
- 时变惯量 RK4：每步固定 4 次导数评估，总体仍为 `O(T)`。

本目录默认参数规模较小，普通 CPU 数秒内即可完成。

## R07

直观物理解释：

- 行星绕恒星：引力沿径向，外力矩近零，所以轨道角动量近似不变；
- 滑冰选手收臂：转动惯量 `I` 变小，在 `tau≈0` 下必须通过增大 `omega` 保持 `L=I*omega`；
- 电机驱动飞轮：施加外力矩后，`L` 会随时间线性增加（理想模型）。

这三类现象在 `demo.py` 中分别对应三组数值实验。

## R11

数值稳定策略：

- 轨道积分使用 Velocity-Verlet，减少长期漂移；
- 时变惯量方程使用 RK4，提升守恒量精度；
- 全程使用 `float64`，降低累积舍入误差；
- 所有关键参数（`dt`、`steps`、惯量正定性）做显式校验；
- 用守恒量漂移而非单点误差作为质量判据。

## R12

默认参数（可在 `demo.py` 顶部 dataclass 修改）：

- `OrbitConfig`: `mu=1`, `mass=2`, `dt=0.01`, `steps=6000`；
- `TorqueConfig`: `inertia=2.5`, `torque=0.15`, `dt=0.02`, `steps=2500`；
- `VariableInertiaConfig`: `inertia0=3.0`, `alpha=0.45`, `freq_hz=0.2`, `dt=0.002`, `steps=8000`。

调参建议：

- 想要更小守恒漂移：优先减小 `dt`；
- 想看更长演化：增加 `steps`，并同步收紧/放宽阈值；
- 保证时变惯量始终正值：`0 <= alpha < 1`。

## R13

理论性质总结：

- 在连续模型中，若外力矩恒为零，角动量严格守恒；
- 中心力系统天然满足零外力矩条件；
- 有外力矩时，角动量变化率与外力矩一一对应（线性关系）；
- 离散仿真中“守恒”体现为小漂移，误差阶由积分器和步长决定。

因此本条目采用“漂移阈值 + 关系拟合”作为工程可验证标准。

## R14

常见失效模式与应对：

- `dt` 过大导致轨道能量与角动量漂移扩大。  
  应对：减小 `dt` 或改更高阶方法。
- 惯量函数参数非法（如 `alpha >= 1`）导致 `I(t)` 接近或低于零。  
  应对：限制参数域并在代码中断言。
- 单位或符号方向错误（力矩符号、叉乘顺序）。  
  应对：保持统一坐标约定，并增加 NumPy/Torch 交叉校验。
- 仅看轨迹图像不看守恒指标，易漏掉缓慢数值发散。  
  应对：把漂移指标纳入自动化门禁。

## R15

工程实践建议：

- 把角动量、能量漂移作为 CI 回归指标；
- 新增力模型时先验证 `tau=0` 情况，再引入外力矩工况；
- 对外展示时同时给“图像结果 + 数值指标”，避免只凭视觉判断；
- 多体扩展前先固定二维最小样例，确保守恒基线正确。

## R16

与相关方法关系：

- 牛顿方程是本条目实现入口，直接按力更新状态；
- 拉格朗日/哈密顿方法在理论上等价，但实现层通常仍要落到数值积分；
- 辛积分法族在长期守恒方面更强，Velocity-Verlet 可视为入门级辛结构方法；
- 本 MVP 处于“守恒律验证”层，不追求高阶多体天文学精度。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 创建三组配置：中心力轨道、常力矩转子、零力矩时变惯量转子。  
2. `simulate_central_orbit` 用 Velocity-Verlet 逐步更新 `r,v`，并计算 `Lz`、能量、累计扫面积。  
3. `evaluate_central_orbit` 统计 `L`/`E` 相对漂移，利用 `linregress` 拟合 `A(t)` 斜率验证面积速度常量，并估计轨道周期误差。  
4. `simulate_constant_torque` 按 `L_{k+1}=L_k+tau*dt` 积分，得到 `L(t), omega(t)`。  
5. `evaluate_constant_torque` 用 `LinearRegression` 与 `linregress` 对 `L(t)` 做斜率拟合，验证 `dL/dt=tau`。  
6. `simulate_zero_torque_variable_inertia` 构造 `I(t)`，以 RK4 求解 `I*omega' + I'*omega = 0`，得到 `L=I*omega`。  
7. `evaluate_zero_torque_variable_inertia` 计算 `L` 漂移与 `omega` 比值误差，验证无外力矩下守恒关系。  
8. `torch_consistency_lz` 对同一轨道数据分别用 NumPy/PyTorch 计算 `Lz` 并比较误差，`build_summary` 汇总指标后由 `main` 执行阈值门禁。  

说明：第三方库只用于基础数值运算、线性拟合与表格输出；守恒律相关动力学更新与校验逻辑均在源码中显式展开。
