# 顶点探测器 (Vertex Detectors)

- UID: `PHYS-0407`
- 学科: `物理`
- 分类: `粒子物理实验`
- 源序号: `427`
- 目标目录: `Algorithms/物理-粒子物理实验-0427-顶点探测器_(Vertex_Detectors)`

## R01

问题定义：在粒子物理实验中，顶点探测器通过高精度硅像素/条形传感器记录带电粒子的空间命中点（hits），目标是从多条轨迹反推出粒子相互作用位置（vertex）。本 MVP 聚焦二维横截面 `x-y` 主顶点重建，模拟一次事件并输出“真实顶点 vs 重建顶点”的误差。

## R02

输入：每条轨迹在多个探测层上的离散命中点 `[(x_k, y_k)]`，其中包含测量噪声和少量离群轨迹。输出：
- 每条轨迹的直线参数（方向向量 `u`、线上一点 `p`）
- 全局主顶点估计 `v = (x_v, y_v)`
- 顶点协方差近似、残差统计、与真值偏差

## R03

物理与几何简化假设：
- 只做横向平面近似，忽略磁场弯曲和 `z` 方向螺旋结构
- 单条轨迹在局部可近似为直线
- 命中误差服从零均值高斯噪声
- 离群轨迹可由鲁棒权重抑制

这些假设适合教学级 MVP，不代表完整实验软件链（如 ACTS/CMSSW）精度。

## R04

数学模型：
- 对第 `i` 条轨迹，线方向单位向量 `u_i`，法向量 `n_i = (-u_{iy}, u_{ix})`
- 若顶点为 `v`，轨迹到顶点的有符号横向距离残差

`r_i = n_i^T (v - p_i)`

目标函数（加权最小二乘）：

`min_v  sum_i w_i * (r_i / sigma_i)^2`

其中 `w_i` 为鲁棒迭代权重，`sigma_i` 为单轨迹横向分辨率近似。

## R05

模拟数据生成：
- 随机生成一个真主顶点 `v_true`
- 生成 `N_primary` 条来自 `v_true` 的主轨迹，角度均匀采样
- 每条轨迹在多层半径上采样理想命中点并叠加高斯测量噪声
- 额外生成 `N_outlier` 条来自偏移次级顶点的离群轨迹

这样可测试算法对 pile-up / 二次衰变污染的鲁棒性。

## R06

单轨迹拟合方法：总最小二乘（TLS, PCA 视角）
- 输入该轨迹的多层命中点矩阵 `H (m x 2)`
- 取质心 `c = mean(H)`
- 对中心化矩阵做 SVD，最大奇异值对应方向即轨迹主方向 `u`
- `p` 取质心 `c`

优势：相比普通 `y=ax+b` 回归，不依赖“谁是自变量”，适合任意方向直线。

## R07

主顶点闭式求解：
- 组装线性系统 `A v ≈ b`，其中第 `i` 行为 `A_i = n_i / sigma_i`，`b_i = n_i^T p_i / sigma_i`
- 给定权重 `w_i`，解加权系统 `sqrt(w_i) A_i, sqrt(w_i) b_i`
- 用 `scipy.linalg.lstsq` 得到 `v_hat`

并计算协方差近似：`Cov(v_hat) ≈ (A_w^T A_w)^(-1)`。

## R08

鲁棒拟合（IRLS）：
- 初值：普通最小二乘 (`w_i=1`)
- 计算残差 `r_i` 与稳健尺度 `s = 1.4826 * MAD(r)`
- 用 Cauchy 权重更新

`w_i = 1 / (1 + (r_i / (c*s))^2)`

- 重复 5-10 轮迭代直到收敛

效果：离群轨迹残差大，权重自动降低，主顶点估计更稳定。

## R09

PyTorch 微调：
- 在 IRLS 解附近，用 `torch` 对二维变量 `v` 做少量梯度优化
- 损失使用 pseudo-Huber：

`L = sum_i delta^2 * (sqrt(1 + (r_i/delta)^2) - 1)`

- Adam 优化几十到几百步

意义：展示传统重建与自动微分框架可组合，便于后续接入可学习不确定度模型。

## R10

复杂度（设轨迹数 `N`，每轨迹命中数 `M`，IRLS 迭代 `T`）：
- 轨迹 TLS：`O(N * M)`（2D 小矩阵 SVD 常数很小）
- 顶点每轮线性求解：`O(N)`（未知量仅 2 维）
- IRLS 总体：`O(T * N)`
- Torch 微调：`O(S * N)`，`S` 为优化步数

瓶颈通常在事件数而非单事件规模。

## R11

数值稳定性策略：
- 对方向向量归一化，避免尺度漂移
- MAD 过小时使用最小阈值 `eps` 防止除零
- 采用 `lstsq` 而非显式矩阵求逆
- 对轨迹命中点数做最小约束（至少 2 点）

## R12

关键超参数（demo.py 默认值）：
- `sigma_hit = 0.015 mm`：单命中噪声
- `n_primary = 12`，`n_outlier = 4`
- `irls_iters = 8`
- Cauchy 常数 `c = 2.5`
- Torch `steps = 120`, `lr = 0.08`, `delta = 0.04`

可通过修改 `main()` 中参数直接做灵敏度实验。

## R13

运行方式：

```bash
uv run python demo.py
```

程序会打印：
- 真顶点、OLS 顶点、IRLS 顶点、Torch 微调顶点
- 各方法到真值的欧氏距离误差
- 每条轨迹残差与鲁棒权重表（DataFrame）

无需交互输入，可直接用于流水线验证。

## R14

结果解读建议：
- 若存在离群轨迹，`IRLS` 通常优于 `OLS`
- `Torch` 微调通常在 IRLS 附近小幅改进，不应大幅偏离
- 若三者都显著偏离，优先检查：离群比例、命中噪声、轨迹层数

## R15

局限与失效模式：
- 强磁场弯曲下直线模型失效，应改用螺旋参数化
- 多主顶点（高 pile-up）时单顶点模型会产生折中解
- 近乎平行轨迹集合会导致几何约束退化
- 真实实验还需材料效应、散射、对准误差建模

## R18

源码级算法流程拆解（对应 `demo.py` 实现）：
1. `simulate_event(...)` 生成真顶点、主轨迹与离群轨迹命中点。
2. 对每条轨迹调用 `fit_track_tls(...)`：中心化命中矩阵，SVD 取第一主方向，得到 `(p_i, u_i)`。
3. `build_constraints(...)` 把 `(p_i, u_i)` 转换为法向约束 `(n_i, p_i, sigma_i)`。
4. `solve_vertex_wls(...)` 构建加权线性系统 `A_w v ≈ b_w`，用 `scipy.linalg.lstsq` 解出顶点。
5. `robust_vertex_irls(...)` 以 OLS 为初值，循环执行“算残差 -> MAD 标度 -> Cauchy 重加权 -> 重新最小二乘”。
6. 在 IRLS 解附近调用 `torch_refine_vertex(...)`，把残差写成 pseudo-Huber 可导损失，用 Adam 做少量迭代。
7. 用 `evaluate_solution(...)` 计算各估计与真值偏差，并统计加权残差 RMSE（`sklearn.metrics.mean_squared_error`）。
8. 用 `pandas.DataFrame` 输出每条轨迹的残差与权重，便于检查哪些轨迹被判为离群。

以上 8 步已经把第三方库调用拆到可追踪粒度，而不是把“顶点重建”交给单个黑盒 API。
