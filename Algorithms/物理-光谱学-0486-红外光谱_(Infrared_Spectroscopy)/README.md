## R01

红外光谱（Infrared Spectroscopy, 常见为 FTIR）通过测量样品对不同波数 `\tilde{\nu}`（单位 `cm^-1`）的吸收强度来识别化学键并进行定量分析。其核心思想是分子振动与红外光相互作用，特定官能团会在特定波数出现吸收带。

在定量场景中，常写为 Beer-Lambert 形式：

- `A(\tilde{\nu}) = \epsilon(\tilde{\nu}) * l * c`

其中 `A` 为吸光度，`\epsilon` 为吸收系数，`l` 为光程，`c` 为浓度。

## R02

IR 光谱长期用于官能团识别（如 `C=O`、`O-H`、`N-H`、`C-H` 等），早期工作多依赖人工判峰和经验解释。随着数字信号处理和机器学习普及，工程实践从“单峰定性/单点定量”逐步发展为：

- 光谱预处理标准化（平滑、基线校正、归一化）；
- 多波段联合建模（线性回归、PLS、神经网络）；
- 对批次变化和仪器漂移的鲁棒验证（ID/OOD 双测试）。

因此 IR 是“物理可解释 + 数据建模”结合非常典型的一类算法问题。

## R03

本任务目标：给定一组 IR 光谱曲线，估计目标分析物浓度 `c_target`。实际难点通常不是公式本身，而是谱图质量和干扰因素：

1. 基线漂移：仪器漂移、散射、背景吸收会抬高或倾斜整条曲线；
2. 峰重叠：干扰物在相近波数有吸收带，单峰法偏差明显；
3. 噪声与分布偏移：不同批次的噪声水平、湿度水汽峰、样品状态变化会影响泛化。

MVP 采用“单波段物理估计 + 全谱统计学习”的组合策略。

## R04

本目录 `demo.py` 的建模设定：

- 合成光谱：`A(\tilde{\nu}) = c_a*\epsilon_a(\tilde{\nu}) + c_b*\epsilon_b(\tilde{\nu}) + b(\tilde{\nu}) + n(\tilde{\nu})`
- 其中 `c_a` 为目标组分浓度，`c_b` 为干扰组分浓度；
- `b(\tilde{\nu})` 表示慢变基线与水汽伪峰，`n(\tilde{\nu})` 是高斯噪声。

算法分三条线：

1. 单波段 Beer-Lambert 估计（解释性强）；
2. 全谱 Ridge 回归（线性稳健基线）；
3. 小型 PyTorch MLP（补充非线性能力）。

## R05

记样本数 `N`、波数点数 `M`、AsLS 迭代次数 `K`、MLP 训练轮数 `E`：

- Savitzky-Golay 平滑：`O(N*M)`；
- AsLS 基线校正（稀疏线性求解，近似）：`O(N*K*M)` 到 `O(N*K*M log M)`（与求解器/稀疏结构有关）；
- 单波段估计：`O(N)`；
- Ridge（常见量级）：`O(N*M^2 + M^3)`；
- MLP 训练：`O(E*N*M*H)`（`H` 为隐藏层宽度规模）。

MVP 数据规模较小，CPU 即可在较短时间完成。

## R06

单波段估计示例（简化单位）：

- 选定分析带 `\tilde{\nu}*=1720 cm^-1`；
- 基线校正后吸光度 `A_corr(\tilde{\nu}*)=0.56`；
- 设 `\epsilon_a(\tilde{\nu}*)=1.40`，光程 `l=1 cm`。

则浓度估计：

- `c_hat = 0.56 / (1.40 * 1) = 0.40`

这体现了 IR 定量的可解释优势，但在峰重叠时会出现系统偏差。

## R07

优点：

- 对官能团与化学结构有直接物理含义；
- 仪器普及、测试速度快、可在线监测；
- 可与多变量模型组合，兼顾解释性与精度。

局限：

- 单波段对基线漂移和峰重叠敏感；
- 环境因素（湿度、水汽、温度）会引入伪峰与偏移；
- 高度依赖预处理策略，跨仪器迁移不总是稳定。

## R08

建议前置知识：

- 振动光谱基础（特征吸收带、峰形、波数区间）；
- Beer-Lambert 定律与线性回归；
- 基础信号处理（平滑、基线校正、归一化）；
- Python 科学计算工具链（NumPy/SciPy/pandas/scikit-learn/PyTorch）。

## R09

适用场景：

- 化工/制药过程中的组分浓度监测；
- 食品与材料质控中的快速无损检测；
- 光谱算法教学与原型验证。

不适用或需谨慎：

- 样品散射极强且难以稳定制样；
- 目标峰太弱、信噪比不足；
- 标定数据覆盖不足却直接部署复杂模型。

## R10

正确性检查要点：

1. 预处理后基线应显著回落，曲线不再整体倾斜；
2. 自动选择分析带应落在目标吸收带附近；
3. 所有模型输出应非负且数值有限（无 NaN/Inf）；
4. Ridge/MLP 在 ID 集上应明显优于单波段基线；
5. Shifted 集性能下降应“可解释”而非数值崩溃。

## R11

数值稳定性处理策略：

- Savitzky-Golay 采用奇数窗口与低阶多项式，防止过拟合噪声；
- AsLS 使用稀疏矩阵与权重迭代，抑制峰区对基线估计的污染；
- 单波段除法分母加下界 `1e-12` 防止零除；
- MLP 前使用标准化，避免特征尺度导致训练不稳。

## R12

本 MVP 的调参经验：

- AsLS 的 `lam` 越大，基线越平滑；过大可能吞掉宽峰信息；
- AsLS 的 `p` 越小，算法越倾向把峰当作“正偏离”保留；
- Ridge 的 `alpha` 决定偏差-方差折中；
- MLP 的 `epochs` 与 `lr` 需要配合，过快会振荡，过小会欠拟合。

## R13

理论性质（本任务相关）：

- Beer-Lambert 单波段法在“线性吸收 + 无干扰 + 基线已校正”前提下可解释且一致；
- Ridge 目标函数是凸优化（`alpha>0` 时解唯一且稳定）；
- MLP 为非凸优化，不保证全局最优，但可刻画非线性混合效应；
- 本问题不涉及近似比证明，重点是物理一致性与泛化性能验证。

## R14

常见失效模式与防护：

- 失效：峰位轻微漂移导致固定单波段偏差。  
  防护：动态选带 + 全谱模型并行。
- 失效：强基线漂移掩盖真实吸收强度。  
  防护：AsLS/多项式基线校正后再建模。
- 失效：干扰组分与目标峰重叠。  
  防护：用 Ridge/MLP 融合多波段信息。
- 失效：测试分布偏移（噪声、湿度）导致性能退化。  
  防护：引入 shifted 验证集并做稳健性门槛。

## R15

工程实践建议：

- 先固定采样流程（光程、分辨率、背景采集），再谈模型优化；
- 将单波段物理估计作为可审计基线，不直接依赖黑盒模型；
- 保存预处理和训练参数（窗口、`lam`、`alpha`、随机种子）以便复现；
- 部署前必须同时通过 ID 与 shifted 的回归指标阈值。

## R16

相关方法对比：

- 单波段 Beer-Lambert：解释性最高，抗干扰较弱；
- Ridge/线性多元回归：稳定、实现简单、可作为强基线；
- PLS：化学计量学常用，对共线谱变量友好；
- 神经网络：可建模更复杂关系，但数据与验证要求更高。

本 MVP 选择 “Beer-Lambert + Ridge + MLP” 三模型并列，目的是展示从物理先验到数据驱动的完整链路。

## R17

`demo.py` 提供可直接运行的最小可行实现：

- 生成训练、ID 测试、shifted 测试三套 FTIR 合成数据；
- 执行平滑、AsLS 基线校正、归一化和自动选带；
- 对比单波段、Ridge、Torch-MLP 三类方法并输出 `MAE/RMSE/R2`。

运行方式：

```bash
cd Algorithms/物理-光谱学-0486-红外光谱_(Infrared_Spectroscopy)
uv run python demo.py
```

脚本无需交互输入，运行后直接打印指标与样例预测。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `ir_absorptivity_profiles` 构造目标物与干扰物在不同波数上的吸收系数曲线 `\epsilon_a(\tilde{\nu}), \epsilon_b(\tilde{\nu})`。  
2. `simulate_ftir_dataset` 根据 `A=c_a\epsilon_a+c_b\epsilon_b+b+n` 合成光谱，并注入乘性散射、基线漂移、水汽伪峰和噪声。  
3. `preprocess_spectra` 对光谱做 Savitzky-Golay 平滑，再调用 `asls_baseline`（稀疏线性系统迭代求解）进行基线扣除。  
4. 同一函数中执行向量归一化，减少路径长度/整体强度波动对回归器的影响。  
5. `select_analytical_band` 用 `find_peaks` 找目标候选峰，再以 `\epsilon_a/(\epsilon_b+\delta)` 评分挑选分离度最佳分析带。  
6. `beer_lambert_estimate` 在选定波段按 `c=A_corr/(\epsilon*l)` 进行单波段浓度估计，作为可解释基线。  
7. `train_ridge_model` 与 `train_torch_mlp` 分别训练线性全谱模型和非线性 MLP，全流程显式可追踪，不依赖黑盒一键求解。  
8. `main` 在 ID 与 shifted 数据集上计算 `MAE/RMSE/R2`，打印对比表并执行断言检查（有限值、选带区间、性能门槛）。

该实现将 IR 定量问题拆解为“可解释物理估计 + 可学习多变量模型”，并在代码中保留了每一步的可审计逻辑。
