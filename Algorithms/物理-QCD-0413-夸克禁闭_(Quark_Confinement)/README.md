# 夸克禁闭 (Quark Confinement)

- UID: `PHYS-0394`
- 学科: `物理`
- 分类: `QCD`
- 源序号: `413`
- 目标目录: `Algorithms/物理-QCD-0413-夸克禁闭_(Quark_Confinement)`

## R01

夸克禁闭（Quark Confinement）的核心现象是：
颜色荷不能以孤立自由粒子形式出现在低能可观测谱中，实验上只观察到颜色中性的强子态。

对最小数值 MVP 而言，可把它压缩为一个可计算判据：
- 若静态夸克-反夸克势 `V(r)` 在大距离区间近似线性增长；
- 且对应力 `F(r)=dV/dr` 在大 `r` 逼近正常数 `sigma>0`；
则分离夸克需要持续增加能量，体现禁闭趋势。

## R02

本条目要解决的问题是：
如何用一个不依赖大型格点框架、但仍保留物理可解释性的脚本，演示“禁闭势 -> 束缚态 -> 量化校验”的闭环。

本实现采用 Cornell 势：
`V(r) = -kappa/r + sigma*r + c0`

其中：
- `-kappa/r` 代表短程类库仑行为；
- `sigma*r` 代表长程线性势（弦张力项）；
- `c0` 为常数零点。

## R03

`demo.py` 的输入输出约定（无交互）：

- 输入（脚本内部配置）：
1. 势参数 `kappa, sigma, c0`；
2. 夸克质量 `m_q`（用于约化质量 `mu=m_q/2`）；
3. 径向网格参数 `r_max, grid_points`；
4. 低能级数量 `n_levels` 与大距离拟合起点 `tail_fit_min_fm`。

- 输出：
1. 若干半径采样点上的 `V(r)` 与 `F(r)`；
2. 有限差分径向薛定谔方程的低激发 `nS` 能级和 `<r>`；
3. 大距离线性拟合得到的 `sigma_fit` 与 `R^2`；
4. 断言通过后打印 `All checks passed.`。

## R04

数学模型与离散化：

1. Cornell 势：
`V(r) = -kappa/r + sigma*r + c0`

2. 禁闭力：
`F(r) = dV/dr = kappa/r^2 + sigma`

3. `l=0` 径向薛定谔方程（`u(r)=rR(r)`）：
`[-(hbar*c)^2/(2mu) * d^2/dr^2 + V(r)] u(r) = E u(r)`

4. 在 `(0, r_max)` 上做二阶中心差分并施加 Dirichlet 边界：
- 主对角：`2A/dr^2 + V(r_i)`
- 次对角：`-A/dr^2`
- `A = (hbar*c)^2/(2mu)`

5. 得到实对称三对角哈密顿矩阵后，求最小特征值作为低能束缚态。

## R05

算法主流程（高层）：

1. 校验参数合法性；
2. 构建径向网格并离散哈密顿矩阵；
3. 用稀疏特征值求解器提取低能 `nS` 态；
4. 归一化波函数并计算 `<r>`、`r_rms`；
5. 在采样半径上计算 `V(r)` 与 `F(r)`；
6. 对大距离段 `V(r)` 做线性拟合得到 `sigma_fit`；
7. 执行数值与物理一致性断言；
8. 打印表格与拟合结果。

## R06

复杂度（网格点数 `N`，求 `k` 个本征态）：

- 构建三对角矩阵：`O(N)`；
- 稀疏本征求解（Lanczos/ARPACK）：约 `O(k*N*iter)`，本 MVP 中近似线性可扩展；
- 统计与拟合：`O(N)`；
- 空间复杂度：`O(N)`（三对角存储 + 本征向量）。

在默认 `N=1000, k=5` 下，运行开销很小，适合作为可审计最小样例。

## R07

模型直觉：

- 短程：`-kappa/r` 使势阱在小 `r` 更深，支持束缚谱；
- 长程：`sigma*r` 保证 `r` 增大时势能持续上升；
- 因而拉开夸克对所需能量随距离增长，不出现“有限势垒后自由分离”的库仑式行为。

该脚本不声称还原完整 QCD，但能把“禁闭机制的有效势表征”明确地数值化。

## R08

运行环境与依赖：

- Python `>=3.10`；
- `numpy`：向量化与数组计算；
- `scipy`：稀疏矩阵与本征值求解；
- `pandas`：表格整理与输出。

本实现故意保持最小工具栈，不依赖黑箱物理专用大库。

## R09

适用场景：

- QCD 入门/课程中演示“禁闭势”与“束缚态”之间的数值联系；
- 作为更复杂格点或势模型的前置验证 baseline；
- 快速做参数敏感性实验（`kappa/sigma/m_q` 改动对谱和尺度的影响）。

不适用场景：

- 精密强子谱拟合；
- 动态夸克、弦断裂、耦合道等非微扰全细节研究；
- 需要实验级误差传播与系统不确定度分析的任务。

## R10

`demo.py` 中的函数职责：

- `validate_config`：参数合法性与物理/数值边界检查；
- `cornell_potential`：计算 `V(r)`；
- `confinement_force`：计算 `F(r)`；
- `build_radial_grid`：构造 `(0, r_max)` 内部网格；
- `build_radial_hamiltonian`：组装三对角哈密顿矩阵；
- `solve_s_wave_levels`：求低激发本征态并计算态统计量；
- `fit_large_r_string_tension`：大距离线性拟合提取 `sigma_fit`；
- `build_potential_report`：输出采样点势能/力表；
- `run_quality_checks`：执行断言门禁；
- `main`：组织运行并打印结果。

## R11

数值稳定性与鲁棒性策略：

- 避开 `r=0` 奇点：网格从 `dr` 开始，不直接取 `0`；
- 使用稀疏三对角格式，避免构建大稠密矩阵；
- 波函数逐态重新归一化，防止求解器尺度漂移；
- 线性拟合只在大距离区间进行，避免短程库仑项污染；
- 对关键物理量设置定量断言，防止“能跑但错误”的静默失败。

## R12

输出字段说明（对应 R04 的模型量）：

1. 势/力采样表：
- `r_fm`：夸克间距；
- `V_total_gev`：`V(r)`；
- `F_total_gev_per_fm`：`F(r)`；
- `V_coulomb_only_gev`：仅 `-kappa/r + c0`；
- `V_linear_only_gev`：仅 `sigma*r + c0`。

2. 束缚态表：
- `state`：`1S,2S,...`；
- `energy_gev`：离散哈密顿本征值；
- `mean_r_fm`：`<r>`；
- `rms_r_fm`：`sqrt(<r^2>)`；
- `norm`：离散归一化检验值。

3. 线性拟合结果：
- `sigma_fit_gev_per_fm`：大距离有效弦张力；
- `intercept_fit_gev`：拟合截距；
- `fit_r2`：拟合质量。

## R13

本条目不提供近似比保证（非组合优化问题），但提供可执行质量门槛：

1. 本征能级必须严格递增；
2. 每个态的离散归一化误差需在阈值内；
3. `sigma_fit` 必须处于目标 `sigma` 的合理比例区间；
4. 大距离线性拟合 `R^2` 需足够高（默认 `>=0.98`）；
5. 采样半径上分离能代价必须显著增长。

这些门槛用于保证“禁闭信号”来自可验证数值结构，而不是主观叙述。

## R14

常见失效模式与对应处置：

1. 网格太粗导致本征值抖动：
- 现象：能级排序不稳、归一化漂移；
- 处置：增大 `grid_points`。

2. `tail_fit_min_fm` 设太小：
- 现象：`sigma_fit` 被库仑短程项拉偏；
- 处置：提升拟合起点到更大 `r`。

3. 参数尺度失配（如 `sigma` 过小）：
- 现象：大距离势增长不足，弱化禁闭信号；
- 处置：回到物理量纲检查并重设参数。

4. 忽略单位换算：
- 现象：动力学项和势项数量级失衡；
- 处置：保持 `hbar*c` 显式出现在离散哈密顿里。

## R18

`demo.py` 的源码级算法流程（8 步，非黑盒）：

1. `main` 创建 `ConfinementConfig` 并调用 `validate_config`，约束参数范围与网格条件。  
2. `build_radial_grid` 在 `(0,r_max)` 构造内部离散点，自动规避 `r=0` 奇点。  
3. `build_radial_hamiltonian` 依据 R04 的有限差分公式构建实对称三对角矩阵。  
4. `solve_s_wave_levels` 调用 `scipy.sparse.linalg.eigsh` 求最低 `n_levels` 本征值/本征向量，并做离散归一化与 `<r>, r_rms` 统计。  
5. `build_potential_report` 在若干 `r` 采样点计算 `V_total`、`F_total` 以及库仑/线性分量对照。  
6. `fit_large_r_string_tension` 在 `r>=tail_fit_min_fm` 区间对 `V(r)` 线性拟合，提取 `sigma_fit`、截距与 `R^2`。  
7. `run_quality_checks` 依次检查能级单调性、波函数归一化、拟合质量、力的渐近行为与分离能代价。  
8. 全部检查通过后，`main` 打印配置、表格和拟合摘要，并输出 `All checks passed.`。

说明：第三方库仅用于基础数值线代与表格展示；禁闭相关的物理主流程（势模型、离散哈密顿、状态统计、弦张力提取与校验）都在源码里逐步展开，没有把结论交给黑盒 API。
