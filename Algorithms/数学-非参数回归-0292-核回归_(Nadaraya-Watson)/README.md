## R01

核回归（Nadaraya-Watson, NW）是一种经典非参数回归方法。它不预先假设全局函数形式（如线性、多项式），而是在每个查询点附近对观测样本做局部加权平均，得到该点的函数估计值。直观上，离查询点越近的样本权重越高，离得越远的样本权重越低。

## R02

给定训练样本 \((x_i, y_i)\)（\(i=1,\dots,n\)），目标是估计回归函数：

\[
 m(x) = \mathbb{E}[Y\mid X=x]
\]

并对任意查询点 \(x\) 预测 \(\hat y = \hat m(x)\)。

## R03

核心思想是“核加权平均”：

1. 计算查询点与每个训练点的距离。
2. 用核函数把距离映射为非负权重。
3. 用归一化后的权重对训练标签做加权求和。

因此 NW 本质上是“局部常数模型”：在查询点附近，用一个常数（加权均值）近似真实函数值。

## R04

Nadaraya-Watson 估计器公式：

\[
\hat m(x)=\frac{\sum_{i=1}^{n} K\!\left(\frac{x-x_i}{h}\right) y_i}{\sum_{i=1}^{n} K\!\left(\frac{x-x_i}{h}\right)}
\]

其中：

- \(K(\cdot)\) 是核函数。
- \(h>0\) 是带宽（smoothing parameter）。
- 分子是加权标签和，分母是权重归一化因子。

## R05

常见核函数：

- 高斯核：\(K(u)=\frac{1}{\sqrt{2\pi}}\exp(-u^2/2)\)（实现时可省略常数因子，归一化时会抵消）。
- Epanechnikov 核：\(K(u)=\frac{3}{4}(1-u^2)\mathbf{1}(|u|\le1)\)。
- 均匀核：\(K(u)=\frac{1}{2}\mathbf{1}(|u|\le1)\)。

本目录 MVP 在 `demo.py` 中支持 `gaussian`、`epanechnikov`、`uniform` 三种核。

## R06

带宽 \(h\) 决定平滑程度：

- \(h\) 太小：方差大，曲线抖动，容易过拟合。
- \(h\) 太大：偏差大，曲线过平，容易欠拟合。

实践中常用交叉验证或验证集搜索带宽。`demo.py` 采用候选带宽集合上的简单 hold-out 评估（以测试 MSE 选择最佳值）来演示这一点。

## R07

算法流程（概念级）：

1. 读入训练集 \(X_{train}, y_{train}\)。
2. 对每个查询点 \(x\)：
3. 计算与所有训练点的缩放距离 \(u_i=(x-x_i)/h\)。
4. 计算核权重 \(w_i=K(u_i)\)。
5. 计算加权和 \(\sum_i w_i y_i\) 和权重和 \(\sum_i w_i\)。
6. 输出 \(\hat y=\frac{\sum_i w_i y_i}{\sum_i w_i}\)。
7. 若权重和过小，则用回退策略（如全局均值）防止数值异常。

## R08

复杂度（朴素实现）：

- 训练阶段：\(O(1)\)（仅存储样本）。
- 预测阶段：若预测 \(m\) 个点、训练集大小为 \(n\)、维度为 \(d\)，则约为 \(O(mnd)\)。
- 内存：主要是训练数据 \(O(nd)\)，若向量化批量计算距离，临时矩阵约 \(O(mn)\)。

## R09

数值与工程注意事项：

- 当查询点远离所有训练样本时，所有核权重可能接近 0，分母会非常小。
- 实现中应加入 `eps` 阈值，避免除零。
- 本 MVP 在分母 \(\le eps\) 时回退到训练标签均值 `y_mean_`，保证输出稳定。

## R10

适用场景：

- 关系非线性明显，且不想强行设定函数形式。
- 样本规模中小（预测时可接受 \(O(mn)\) 成本）。
- 需要直观、可解释的局部平滑估计。

## R11

优点：

- 模型假设弱，适配复杂非线性。
- 原理简单，易实现、易调试。
- 局部加权直观，便于解释。

缺点：

- 对带宽敏感。
- 维度升高时受“维度灾难”影响明显。
- 预测成本随训练集线性增长，不适合超大规模在线推断。

## R12

与相关方法对比：

- 对比 kNN 回归：kNN 用“固定样本数邻居”，NW 用“固定距离尺度（带宽）+连续权重”，预测通常更平滑。
- 对比局部线性回归：NW 是局部常数，边界偏差通常更大；局部线性可降低边界偏差但实现更复杂。
- 对比参数模型（线性回归等）：NW 灵活但缺少全局参数表达，外推能力通常较弱。

## R13

本目录 MVP 设计：

- `demo.py` 定义 `NadarayaWatsonRegressor`。
- `fit` 仅保存训练数据并计算全局均值。
- `predict` 使用 NumPy 向量化计算所有查询点到训练点的距离、权重和预测值。
- 主程序生成 1D 非线性合成数据，切分训练/测试，搜索带宽并输出指标与样例预测。

## R15

输出解读（示例结构）：

- 打印候选带宽及对应测试 MSE。
- 打印最佳带宽。
- 打印 `MSE / MAE / R2` 三个回归指标。
- 打印若干条 `(x, y_true, y_pred)` 预测样例，便于人工检查拟合质量。

## R16

边界条件与异常处理：

- `fit` 要求 `X` 为二维、`y` 为一维且样本数一致。
- 带宽必须为正数。
- `predict` 前必须先 `fit`。
- 当 `sum(weights)` 过小，使用 `y_mean_` 回退，避免 `NaN/Inf`。

## R17

可扩展方向：

- 自动带宽选择（K 折交叉验证、Silverman/Scott 规则、自适应带宽）。
- KD-Tree / Ball-Tree 或近邻截断加速预测。
- 局部线性/局部多项式扩展，降低边界偏差。
- 支持样本权重、异方差建模和置信区间估计（如 bootstrap）。

## R18

`demo.py` 的源码级算法流（非黑盒）可拆为 7 步：

1. `make_synthetic_data` 生成 \(x\) 与噪声观测 \(y\)，构造“真函数 + 噪声”的回归任务。
2. `fit` 保存 `X_train_`、`y_train_` 并计算 `y_mean_`，作为极端稀疏权重场景的稳定回退值。
3. `predict` 中调用 `_pairwise_scaled_distance`，对每个查询点与每个训练点计算 \(\|x-x_i\|/h\) 的矩阵。
4. 调用 `_kernel_weights` 将距离矩阵映射为核权重矩阵（高斯/ Epanechnikov / 均匀）。
5. 用逐元素乘法并按行求和 `np.sum(weights * y_train_[None, :], axis=1)` 计算分子，`weights.sum(axis=1)` 计算分母。
6. 使用带 `where` 的 `np.divide` 执行稳定除法；分母过小位置直接填充 `y_mean_`。
7. 主程序对多个带宽重复步骤 2-6，按 MSE 选择最优带宽并报告最终指标与样例预测。
