## R01

毕奥-萨伐尔定律给出“稳恒电流分布在空间某点产生磁场”的积分表达式。

- 连续形式:
  - `B(r) = (mu0 / 4pi) * ∫ I (dl × R) / |R|^3`
  - 其中 `R = r - r'`，`r'` 是电流元位置，`dl` 是沿电流方向的线元。
- 输入:
  - 电流几何（导线曲线或离散线段）
  - 电流大小 `I`
  - 观测点集合 `r`
- 输出:
  - 每个观测点的磁感应强度向量 `B = (Bx, By, Bz)`
- 数值实现主步骤（本目录 `demo.py`）:
  1. 将连续导线离散为 `N` 个小线段。
  2. 取每段中点作为积分采样点，计算 `dl` 与 `R`。
  3. 累加 `dB = (mu0/4pi) I (dl×R)/|R|^3` 得到总场。
- 关键机制:
  - 方向由叉积决定（右手定则）。
  - 强度随距离按 `1/r^2`（来自 `|R|^3` 与线元长度共同作用）衰减。

## R02

- 提出年代: 1820 年。
- 提出者: Jean-Baptiste Biot 与 Félix Savart。
- 历史节点: 在奥斯特发现电流磁效应（1820）后不久，二人通过实验与数学归纳建立该定律，后由安培-麦克斯韦理论统一解释。

## R03

在 19 世纪初，关键问题是“已知电流分布，如何定量预测磁针偏转与空间磁场”。早期只能做定性描述，无法计算任意几何导线的磁场。毕奥-萨伐尔定律把磁场贡献分解为可叠加的电流元，解决了复杂导线场计算问题，并成为电磁学从经验规律走向可计算理论的关键台阶。

## R04

对离散化后的数值积分，设观测点数为 `M`、线段数为 `N`。

- 时间复杂度: `O(MN)`。
  - 每个点都要累加所有线段贡献。
  - 单项贡献包含向量差、范数、叉积、标量乘法。
- 最坏/典型: 在均匀离散、全空间求场场景下通常就是 `O(MN)`。
- 若做快速多极子、树算法或对称性化简，可降到次二次规模，但不属于本 MVP 范围。

## R05

- 向量化实现（本 demo）:
  - 临时张量 `r` 规模约 `M×N×3`，空间复杂度 `O(MN)`。
- 流式累加实现:
  - 每次只处理一个观测点或一个线段块，可降到 `O(M)` 或 `O(N)` 额外空间。
- 本条目 MVP 选择 `O(MN)` 空间换取代码简洁和 NumPy 向量化吞吐。

## R06

示例: 半径 `R=0.1 m` 的圆形电流环，电流 `I=5 A`，观测中心点 `(0,0,0)`。

- 解析解（轴线上 `z=0`）:
  - `Bz = mu0 I / (2R) ≈ 3.141593e-05 T`
- 数值解（`N=1600` 分段，中点积分）:
  - `Bz_numeric ≈ 3.141585e-05 T`
- 相对误差:
  - 约 `3.21e-06`（实际由 `demo.py` 运行结果给出）

该示例展示了离散 Biot-Savart 积分对标准解析场的收敛性。

## R07

毕奥-萨伐尔定律的意义主要有三点。

- 理论意义:
  - 提供从电流分布到磁场分布的直接映射，是静磁学基本定律之一。
- 方法意义:
  - 体现“局部贡献 + 线性叠加”的场论计算范式。
- 工程意义:
  - 广泛用于线圈设计、磁传感器标定、MRI/粒子束线圈建模、电机与电源磁兼容分析。

## R08

该算法路径依赖如下先导知识/方法。

- 向量微积分:
  - 叉积、向量范数、曲线线积分。
- 物理原理:
  - 稳恒电流假设与线性叠加原理。
- 数值方法:
  - 曲线积分离散化（中点法/梯形法等求积思想）。
- 几何建模:
  - 将连续导线参数化并切分为有限线段。

## R09

适用前提与边界条件。

- 适用前提:
  - 稳恒电流（时间不变或准静态）。
  - 介质近似线性、均匀（本式使用真空磁导率 `mu0`）。
  - 导线可抽象为细线电流（忽略截面内电流分布细节）。
- 边界条件:
  - 观测点不能落在理想线电流奇点上（`|R|=0`）。
- 不适用场景:
  - 高频辐射问题、强时变电磁场（需全 Maxwell 方程或延迟势）。
  - 导体尺寸/集肤效应不可忽略的精细电流分布问题。

## R10

正确性依据来自两层。

1. 连续层: 毕奥-萨伐尔定律本身是静磁学的基础规律，可由 Maxwell 方程组（静态条件）一致导出。
2. 离散层: 数值算法把线积分写成黎曼和；当分段长度 `Δl -> 0` 且观测点远离奇点时，离散和收敛到连续积分。

本 demo 通过“圆环轴线解析解 vs 数值解”做可执行正确性校验。

## R11

数值稳定性与误差传播分析。

- 截断误差:
  - 来自积分离散化；中点法在被积函数足够平滑时通常具二阶收敛趋势。
- 舍入误差:
  - 大规模求和时存在浮点累积误差，尤其在对称抵消场景（`Bx, By` 接近 0）更明显。
- 病态来源:
  - 观测点接近导线时 `|R|^3` 很小，导致数值放大。
- 本实现处理:
  - 使用 `eps` 防止除零；并在轴线上检查横向分量是否接近零以监控误差。

## R12

真实计算成本（超越渐近阶）。

- 常数项主耗时:
  - `np.cross`、`np.linalg.norm` 与大张量广播。
- 内存带宽:
  - `M×N×3` 张量读写通常比标量运算更早成为瓶颈。
- 并行化:
  - NumPy 底层已做向量化；进一步可按观测点分块并行。
- GPU 可行性:
  - 若 `M,N` 很大，可迁移到 PyTorch/CUDA；但小规模问题 CPU+NumPy 更简洁。
- I/O 成本:
  - 本 MVP 仅终端打印表格，I/O 成本可忽略。

## R13

- 概率保证: N/A（算法为确定性数值积分，不涉及随机过程）。
- 误差界说明:
  - 在观测点不接近奇点且曲线足够光滑时，中点离散误差可写成 `O(Δl^2)` 量级（全局）。
  - 若观测点逼近导线，误差常数急剧增大，需自适应加密或解析奇异项处理。

## R14

鲁棒性与失效模式。

- 失效模式 1: 观测点落在导线上或极近导线，导致奇异/爆大值。
- 失效模式 2: 分段过粗，几何弯曲处积分误差显著。
- 失效模式 3: 多段近乎相消时，浮点抵消造成有效数字损失。
- 失效模式 4: 单次向量化规模过大触发内存压力。
- 缓解方式:
  - 近场加密分段、分块累加、补充解析基准点回归测试。

## R15

工程实现要点。

- 数据结构:
  - 用 `seg_starts/seg_ends` 两个 `N×3` 数组表达导线。
- 初始化:
  - 几何构造函数先验证 `radius > 0`、`n_segments >= 8`。
- 数值细节:
  - 统一使用 `float64`，对 `|R|^3` 使用 `eps` 防除零。
- 验证策略:
  - 至少做两类检查: 中心点收敛、轴线解析曲线拟合。
- 常见陷阱:
  - 叉积顺序写反会导致磁场方向整体翻转。
  - 单位混用（mm 与 m）会导致数量级错误。

## R16

前驱/后继算法与典型应用。

- 前驱关系:
  - 由库仑型反平方积分思想与线性叠加范式发展而来（与 R08 一致）。

  - 边界元/FEM/FDTD 在复杂介质与时变问题中的推广。
- 典型应用:
  - 圆线圈/亥姆霍兹线圈设计。
  - 电流回路磁场快速评估与标定。
  - 低频电磁兼容近场估算。

## R18

源码追踪与 3-10 步算法拆解（本条目未依赖黑箱求解器，核心实现位于本目录 `demo.py`）。

`biot_savart_field_from_segments` 的源级流程可拆为 8 步。

1. 读取观测点数组与线段起止点数组，做形状校验并转为 `float64`。
2. 计算每个线段的向量 `dl = end - start` 与中点 `mid`。
3. 通过广播构造 `r = obs - mid`，得到每个“点-线段”对的相对位移。
4. 计算 `|r|` 与 `|r|^3`，并对过小距离使用 `inf` 分母屏蔽奇点。
5. 计算叉积 `dl × r`，编码右手定则下的方向信息。
6. 形成每段贡献 `cross / |r|^3`，再乘以常数因子 `(mu0*I)/(4*pi)`。
7. 沿线段维度求和，得到每个观测点总磁场向量 `B`。
8. 将数值结果与解析轴线公式对照，并用断言给出误差闭环验证。

对应到第三方库层面，`numpy` 仅提供基础张量算子（广播、叉积、范数、求和），并未替代算法本身。
