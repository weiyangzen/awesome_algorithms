# 幂法 (Power Iteration)

- UID: `MATH-0057`
- 学科: `数学`
- 分类: `线性代数`
- 源序号: `57`
- 目标目录: `Algorithms/数学-线性代数-0057-幂法_(Power_Iteration)`

## R01

幂法（Power Iteration）用于近似求矩阵的主特征值与对应特征向量。

给定方阵 `A` 与非零初始向量 `x0`，反复执行：

- `y_{k+1} = A x_k`
- `x_{k+1} = y_{k+1} / ||y_{k+1}||`

当满足一定谱条件时，`x_k` 会收敛到主特征向量方向，Rayleigh 商
`lambda_k = x_k^T A x_k`
收敛到主特征值。

## R02

历史定位（简要）：

- 幂法是最早期且最经典的特征值迭代法之一；
- 20 世纪数值线性代数教材中长期作为“谱迭代方法”入门案例；
- 现代很多方法（如逆幂法、Arnoldi、Lanczos）都可视作在幂法思想上的结构化扩展。

## R03

本条目解决的问题（MVP 版本）：

- 输入：
  - 实方阵 `A in R^{n x n}`；
  - 最大迭代次数 `max_iters`；
  - 残差阈值 `tol`；
  - 随机种子 `seed`（用于可复现初始向量）。
- 输出：
  - 主特征值近似 `lambda_hat`；
  - 单位主特征向量近似 `v_hat`；
  - 残差 `||A v_hat - lambda_hat v_hat||`；
  - 是否收敛与迭代次数。

## R04

时间复杂度：

- 稠密矩阵下，每步核心是一次矩阵-向量乘 `A @ x`，成本 `O(n^2)`；
- 迭代 `k` 步，总成本 `O(k n^2)`；
- 若 `A` 为稀疏矩阵，单步可降为 `O(nnz(A))`，总成本 `O(k nnz(A))`。

## R05

空间复杂度：

- 存储矩阵 `A`：稠密为 `O(n^2)`；
- 迭代向量和中间量（`x, y, Ax`）：`O(n)`；
- 收敛历史（本实现记录残差序列）为 `O(k)`。

若只看迭代额外开销（不含输入矩阵），是 `O(n + k)`。

## R06

2x2 微型示例：

令
`A = [[2,1],[1,2]]`，主特征值为 `3`，主特征向量方向为 `[1,1]^T`。

取 `x0 = [1,0]^T`：

1. `y1 = A x0 = [2,1]^T`，归一化得 `x1 ≈ [0.8944, 0.4472]^T`。  
2. `y2 = A x1 ≈ [2.2360, 1.7888]^T`，归一化后更接近 `[1,1]` 方向。  
3. 持续迭代后，`xk` 收敛到主方向，`lambda_k = xk^T A xk` 收敛到 `3`。

## R07

算法意义：

- 是理解“谱分解 + 迭代收敛”最直接的入口；
- 在只需要主特征对而非全部特征值时，通常比全分解更省；
- 在 PageRank、PCA 主方向估计、图谱分析中具有基础地位；
- 也是 Krylov 子空间方法的认知前置。

## R08

核心理论（收敛机制）：

设 `A` 可对角化，特征值按绝对值排序为
`|lambda1| > |lambda2| >= ... >= |lambdan|`，且初始向量在主特征向量上分量不为 0。

将 `x0` 展开到特征向量基：
`x0 = c1 v1 + c2 v2 + ... + cn vn`，其中 `c1 != 0`。

则
`A^k x0 = c1 lambda1^k v1 + c2 lambda2^k v2 + ...`。

除以主尺度 `lambda1^k` 后，其它项按比率 `(|lambda_i|/|lambda1|)^k` 衰减，因此方向趋向 `v1`。
收敛速度由谱间隙（常见度量 `|lambda2/lambda1|`）决定。

## R09

适用条件与局限：

适用：

- 需要主特征值/向量；
- 存在明显主特征值（按绝对值唯一最大）；
- 能高效执行矩阵-向量乘。

局限：

- 若 `|lambda1|` 与 `|lambda2|` 接近，收敛会慢；
- 若主特征值不唯一（绝对值并列），可能不收敛到唯一方向；
- 初始向量若恰与主特征向量正交，会失败（随机初始化下概率通常极低）。

## R10

正确性说明（工程可验证版）：

1. 迭代本质上构造 `x_k = A^k x0 / ||A^k x0||`。  
2. 在 R08 条件下，`x_k` 的方向收敛到主特征向量方向。  
3. 当 `x_k -> v1` 时，Rayleigh 商 `x_k^T A x_k -> lambda1`。  
4. 因此残差 `rk = ||A x_k - lambda_k x_k||` 应趋近于 0。  
5. `demo.py` 用残差阈值作为停止标准，并用 `numpy.linalg.eig` 结果做交叉校验。

## R11

误差与稳定性：

- 幂法是迭代近似法，截断误差由停止阈值与迭代上限决定；
- 浮点舍入会引入微小扰动，但每步归一化可抑制向量爆炸/下溢；
- 对负主特征值，向量方向可能出现符号翻转，比较向量时应做“符号无关”处理；
- 实践中使用残差 `||Av-lambda v||` 比仅看 `||x_{k+1}-x_k||` 更稳健。

## R12

性能视角：

- 单步只做一次 matvec 和少量向量运算，常数因子小；
- 对稀疏问题，matvec 可充分利用稀疏存储与并行 BLAS；
- 若仅取一个主特征对，通常优于全特征分解的 `O(n^3)` 成本；
- 真正瓶颈通常是“迭代次数是否被谱间隙拖慢”。

## R13

可验证保证（本目录 MVP）：

- 固定随机种子，结果可复现；
- 输出包含：估计特征值、残差、迭代次数、与 NumPy 参考解误差；
- 自动检查：
  - 算法收敛；
  - 残差足够小；
  - 与参考主特征值一致到高精度；
  - 与参考主特征向量方向高度一致（符号无关）。

## R14

鲁棒性与常见失效模式：

- 输入不是方阵或含 `nan/inf`：直接报错；
- 迭代中出现 `||A x|| = 0`（例如退化输入）：报错退出；
- 达到 `max_iters` 仍未满足阈值：返回 `converged=False`，提示放宽预算或更换方法；
- 谱间隙过小时，建议改用更强方法（如 Lanczos/Arnoldi/QR 类）。

## R15

实现设计（对应 `demo.py`）：

- `power_iteration`：幂法主过程（显式实现迭代，不依赖黑箱特征值例程）；
- `dominant_eigenpair_numpy`：仅用于最终对照验证；

## R16

相关算法链路：

- 直接扩展：逆幂法（找最小特征值方向）、位移逆幂法（定位指定谱区间）；
- 子空间扩展：Lanczos、Arnoldi（一次逼近多个特征值）；
- 全谱方法：QR 迭代；
- 工程实践：大规模稀疏主特征问题常由幂法/Arnoldi 家族处理。

## R18

源码级流程拆解（`demo.py` 中 `power_iteration` 主路径，8 步）：

1. `validate_inputs` 检查 `A` 为有限实方阵，`max_iters` 与 `tol` 合法。  
2. 用固定 `seed` 生成随机初始向量 `x` 并归一化，保证可复现。  
3. 进入迭代循环，先做 `y = A @ x`（唯一重计算热点）。  
4. 计算 `||y||` 并归一化得到 `x_next = y / ||y||`。  
5. 计算 Rayleigh 商 `lambda_hat = x_next^T A x_next`。  
6. 计算残差 `r = ||A x_next - lambda_hat x_next||`，记录到历史。  
7. 若 `r < tol`，立即返回收敛结果；否则继续下一轮（并做符号对齐避免方向抖动）。  
8. 若超过迭代上限仍未达阈值，返回 `converged=False` 的结果供上层决策。  

说明：`numpy.linalg.eig` 仅在 `main` 中用于“验算”，并未替代幂法核心迭代逻辑。
