## R01

Wilcoxon符号秩检验（Wilcoxon signed-rank test）用于比较两组配对样本（如治疗前后、同一对象两种条件）是否存在系统性位置差异。它是配对 t 检验在非正态场景下的重要替代方案，核心利用“差值绝对值的秩”而不是原始数值大小。

## R02

给定配对样本 \((x_i, y_i), i=1,\dots,n\)，定义差值 \(d_i=x_i-y_i\)。检验目标通常是中位数差异是否为 0：
- 原假设 \(H_0\)：差值分布以 0 为中心（常见表述：差值中位数为 0）
- 备择假设 \(H_1\)：差值分布中心不为 0（双侧）或大于/小于 0（单侧）

## R03

常见假设前提：
- 样本是成对且独立抽取（不同 pair 之间独立）
- 差值至少是可排序的（通常为连续或近似连续）
- 在经典解释下，差值分布应近似对称（围绕中位数）
- 零差值（\(d_i=0\)）需按指定规则处理（常见 `wilcox`：剔除）

## R04

适用场景：
- 样本量中小、正态性不满足，或存在离群值
- 关注“方向 + 排名强度”，而非均值差线性模型
- 典型例子：药物干预前后评分、A/B 条件下同一受试者反应时

## R05

输入与输出定义（对应本目录 `demo.py`）：
- 输入：两个等长一维数组 `x`, `y`（配对观测）
- 中间量：`d=x-y`、非零差值筛选、绝对差值秩 `r_i`
- 输出：
  - \(W^+\)：正差值秩和
  - \(W^-\)：负差值秩和
  - \(T=\min(W^+,W^-)\)（常用检验统计量）
  - 近似 z 值与 p 值（双侧）
  - SciPy 对照结果（用于校验）

## R06

核心思想：
1. 仅看差值符号（正/负）会损失幅度信息。
2. 仅看幅度又会忽略方向。
3. Wilcoxon 将二者结合：先按 \(|d_i|\) 排秩，再按符号分别累加秩和。
4. 若 \(H_0\) 成立，正负秩和应大致平衡；若偏离明显，说明存在系统性差异。

## R07

关键公式（`zero_method='wilcox'`）：
1. 去除零差值后得到 \(m\) 个有效差值。
2. 对 \(|d_i|\) 从小到大赋秩 \(r_i\)（并列取平均秩）。
3. 计算
\[
W^+=\sum_{d_i>0} r_i,\quad
W^-=\sum_{d_i<0} r_i,\quad
T=\min(W^+,W^-).
\]
4. 大样本近似时，可对 \(W^+\) 做正态近似：
\[
\mu=\frac{m(m+1)}{4},\quad
\sigma^2=\frac{m(m+1)(2m+1)-\sum_j t_j(t_j+1)(2t_j+1)}{24},
\]
其中 \(t_j\) 为并列组大小。然后
\[
z=\frac{W^+-\mu-\text{cc}}{\sigma},
\]
`cc` 是连续性修正项（取决于方向与实现约定）。

## R08

伪代码：

```text
input: x[1..n], y[1..n]
d <- x - y
remove d_i == 0
abs_d <- |d|
r <- rank(abs_d, ties=average)
W_plus <- sum(r_i for d_i > 0)
W_minus <- sum(r_i for d_i < 0)
T <- min(W_plus, W_minus)
compute p-value (exact for small n or normal approximation)
return T, W_plus, W_minus, p-value
```

## R09

正确性直觉：
- 在“无系统差异”下，正负号应随机分布在各秩上，正负秩和接近。
- 当 `x` 普遍大于 `y`，大量较大秩会落入正号侧，\(W^+\) 显著偏大、\(W^-\) 偏小。
- 因此以 \(T=\min(W^+,W^-)\) 或等价统计量衡量“失衡程度”即可检验假设。

## R10

复杂度：
- 排秩是主成本：\(O(m\log m)\)
- 其余扫描求和：\(O(m)\)
- 额外空间：\(O(m)\)
其中 \(m\) 为去零后的有效配对数。

## R11

边界与鲁棒性注意点：
- 全部差值为 0：检验无意义，应报错或返回不可判定
- 样本太小时建议 exact p-value；样本较大可用近似
- ties（并列绝对差值）会影响方差，近似计算需做 tie correction
- 缺失值需先清洗，且 `x`/`y` 长度必须一致

## R12

与配对 t 检验比较：
- 配对 t 检验：针对均值差，依赖差值近似正态
- Wilcoxon：针对中位位置差异/分布中心，更抗离群与偏态
- 代价：当正态假设严格成立时，Wilcoxon 统计效率可能略低于 t 检验

## R15

常见误用：
- 把独立样本误当配对样本（应改用 Mann-Whitney U）
- 忽略“成对关系”打乱了配对顺序
- 不说明零差值处理策略（`wilcox` / `pratt` / `zsplit`）
- 只报告 p 值，不报告效应方向（可结合 `W+` 与中位差）

## R16

可扩展方向：
- 增加 `alternative` 参数（`two-sided`/`greater`/`less`）
- 增加精确分布 p 值计算（小样本递推/枚举）
- 输出效应量（如 rank-biserial correlation）
- 批量评估多组配对特征并做多重比较校正

## R17

最小测试清单：
1. 正常双侧案例：应返回有限统计量与 p 值
2. 含零差值案例：确认按 `wilcox` 被剔除
3. 含 ties 案例：方差修正后结果稳定
4. 全零差值：抛出可读异常
5. 长度不等：抛出参数错误

## R18

`demo.py` 的源码级算法流（非黑盒）：
1. 构造差值：`d = x - y`，先进行输入长度检查与一维化。
2. 去零处理：按 `wilcox` 规则丢弃 `d == 0` 的 pair，得到有效差值 `d_nz`。
3. 绝对值排秩：对 `abs(d_nz)` 使用平均秩（并列同秩均分），得到 `ranks`。
4. 分符号求秩和：`W+ = sum(ranks[d_nz > 0])`，`W- = sum(ranks[d_nz < 0])`。
5. 形成统计量：`T = min(W+, W-)`，作为双侧检验常见统计量。
6. 计算期望与方差：基于有效样本数 `m` 计算 \(\mu,\sigma^2\)，并加入 ties 修正项。
7. 正态近似 p 值：对 `W+` 做连续性修正后换算 `z`，再得到双侧 `p`。
8. 库函数对照验证：调用 `scipy.stats.wilcoxon` 输出 `statistic/pvalue`，用于交叉核对。
9. 结论输出：根据设定显著性水平（如 0.05）给出拒绝/不拒绝 \(H_0\) 的解释。
