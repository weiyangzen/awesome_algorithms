## R01

平均场理论（Mean Field Theory, MFT）是一类把“多体相互作用”近似为“单体在平均有效场中运动”的方法。  
对 Ising 铁磁模型，平均场近似把邻居自旋的涨落替换为平均磁化 `m=<s>`，得到自洽方程：

`m = tanh(beta * (z J m + h))`

其中 `beta=1/T`（取 `k_B=1`）、`z` 为配位数、`J` 为耦合常数、`h` 为外场。该方程是本目录 MVP 的核心。

## R02

本任务目标是做一个“可运行、可解释、非黑盒”的最小实现：

- 输入一组温度 `T`；
- 对每个温度求平均场自洽方程的全部固定点；
- 判定固定点稳定性并用自由能选择平衡解；
- 输出磁化与磁化率随温度的变化，验证 `Tc=zJ` 附近的相变特征。

MVP 聚焦算法与物理机制，不追求高阶修正（如涨落修正、重整化群）。

## R03

本实现使用经典 Ising 平均场模型（单组分磁化）：

- 序参量：`m in [-1, 1]`
- 自洽方程：`m = tanh(beta * (z J m + h))`
- 自由能密度（每自旋）：
  `f(m) = -0.5*z*J*m^2 - h*m + T*[p_up ln p_up + p_dn ln p_dn]`
  其中 `p_up=(1+m)/2`, `p_dn=(1-m)/2`
- 稳定性判据（固定点迭代斜率）：
  `| beta*z*J*(1-m^2) | < 1`

`demo.py` 会显式给出稳定根数量（例如 `T<Tc` 且 `h=0` 时通常有三根，其中两根稳定）。

## R04

关键理论结论：

1. 临界温度：`Tc = z J`（本实现默认 `z=4, J=1 => Tc=4`）
2. `h=0` 时：
   - `T > Tc`：唯一稳定解 `m=0`（顺磁相）
   - `T < Tc`：出现对称非零稳定解 `±m`（自发对称性破缺）
3. 磁化率（由隐函数求导）
   `chi = dm/dh = beta*(1-m^2) / [1 - beta*z*J*(1-m^2)]`
   在 `Tc` 附近出现增强。

## R05

数值算法采用“网格扫描 + 二分法”而非第三方黑盒求根器：

- 在 `[-1,1]` 上离散扫描残差 `g(m)=m-tanh(beta*(zJm+h))`；
- 找到符号变号区间后用二分法收敛到根；
- 去重得到固定点集合；
- 对每个固定点计算稳定性与自由能；
- 选自由能最低的稳定根作为平衡磁化 `m_eq`。

这样每一步都可追踪，便于教学和验证。

## R06

`demo.py` 输出两类信息：

- 温度扫描表（`temperature`, `m_equilibrium`, `susceptibility`, `n_fixed_points`, `n_stable_points` 等）；
- 摘要指标：理论 `Tc=zJ`、数值 `chi` 峰值温度、低温/高温磁化检查结果。

并内置断言：

- 低温平衡磁化显著非零；
- 高温平衡磁化接近 0；
- 低温区存在多稳定分支；
- 数值峰值温度接近理论 `Tc`。

## R10

正确性直觉：

1. 平衡态满足自洽方程固定点条件 `g(m)=0`；
2. 固定点是否可观测，还需满足稳定性（微扰不发散）；
3. 若多个稳定根并存，物理平衡态由自由能最小原则决定；
4. 因而“求根 + 稳定性 + 自由能比较”三步缺一不可。

本 MVP 正是按这三步执行。

## R11

数值稳定性处理：

- 根搜索区间采用 `[-0.999999, 0.999999]`，避免对数项触及 `ln(0)`；
- 自由能中的概率项 `p_up/p_dn` 做截断 `clip`；
- 二分迭代有最大步数与容差双重终止；
- 固定点去重避免重复计数；
- 计算磁化率时对奇异分母做 `nan/inf` 保护。

## R12

核心参数（`MeanFieldConfig`）：

- `coordination_number` (`z`)：配位数；
- `coupling_j` (`J`)：相互作用强度；
- `external_field` (`h`)：外场；
- `temperature_min/max/n_temperatures`：扫描范围与分辨率；
- `scan_points`：根扫描网格密度；
- `root_tol`、`max_bisect_iter`：二分求根精度控制。

调参建议：若漏根，可提高 `scan_points`；若速度优先，可适度降低。

## R14

常见失败模式：

1. 只用单初值迭代，导致漏掉多分支固定点；
2. 不做稳定性判据，把不稳定根当作物理解；
3. 不做自由能比较，无法正确处理简并或多稳态；
4. `m` 取到 `±1` 导致熵项数值溢出；
5. 温度网格过粗，`Tc` 峰位估计偏差大。

本实现分别用“全区间扫描、稳定性过滤、自由能选根、clip 与较密网格”应对。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 构造 `MeanFieldConfig`，定义 `z, J, h` 与温度扫描区间。  
2. `run_temperature_scan` 逐温度调用 `find_fixed_points`，构造每个温度的固定点集合。  
3. `find_fixed_points` 在 `m` 网格上计算残差 `g(m)`，检测变号区间。  
4. 对每个变号区间调用 `bisect_root`（手写二分法）求根，并做去重。  
5. `stability_slope` 计算每个根的局部斜率，判定稳定/不稳定。  
6. `select_equilibrium_root` 对稳定根计算 `free_energy_density`，以最小自由能选出平衡根 `m_eq`。  
7. 基于 `m_eq` 用解析公式计算 `susceptibility`，并汇总为 `pandas.DataFrame`。  
8. `main` 打印表格与 `Tc` 比较，并执行断言（低温有序、高温无序、低温多稳态、峰值温度接近理论）。  

第三方库使用边界：`numpy/pandas` 仅承担数值数组与表格展示；固定点搜索、稳定性判断、自由能筛选都在源码中显式实现，不依赖黑盒平均场求解器。
