- UID: `PHYS-0189`

## R01

德拜屏蔽（Debye shielding）描述的是：在等离子体中放入一个测试电荷后，周围带电粒子会重新分布，从而削弱其远场库仑势。  
在线性近似（`|q_s phi| << k_B T_s`）下，点电荷 `Q` 的势变为 Yukawa 形式：

`phi(r) = (Q / (4*pi*epsilon_0*r)) * exp(-r/lambda_D)`

其中 `lambda_D` 是德拜长度，控制屏蔽范围。

## R02

该机制是等离子体“准中性但局部可带电”的核心桥梁：

- `r << lambda_D`：势接近裸库仑势，局部电场仍强；
- `r ~ lambda_D`：势被显著压低；
- `r >> lambda_D`：测试电荷影响指数衰减，等离子体宏观看近似中性。

因此德拜长度常被用作判断“流体近似/静电近似是否成立”的第一尺度。

## R03

本目录 MVP 目标：

- 计算双组分等离子体（电子 + 单价离子）的 `lambda_D`；
- 用有限差分求解球对称线性化方程并与解析解对照；
- 输出径向势剖面与屏蔽因子 `phi/phi_Coulomb`；
- 通过断言验证数值正确性与基本物理规律。

脚本是“可运行、可审查、可扩展”的最小实现，而非大型 PIC/MHD 框架。

## R04

核心方程：

1. 线性化 Poisson-Boltzmann：
   `nabla^2 phi = phi/lambda_D^2`（点源外区域）
2. 德拜长度（多组分形式）：
   `1/lambda_D^2 = sum_s n_s q_s^2 / (epsilon_0 k_B T_s)`
3. 球对称变换 `u(r)=r*phi(r)` 后：
   `u'' - u/lambda_D^2 = 0`
4. 解析势（点电荷）：
   `phi(r) = Q/(4*pi*epsilon_0*r) * exp(-r/lambda_D)`

`demo.py` 对第 3 式做离散求解，再与第 4 式逐点比较。

## R05

复杂度（设网格点数 `N`）：

- 有限差分离散得到三对角线性系统，规模 `N-2`；
- Thomas 算法求解复杂度 `O(N)`；
- 解析解与误差评估复杂度 `O(N)`；
- 空间复杂度 `O(N)`。

默认 `N=800`，可在普通 CPU 上毫秒级到秒级完成。

## R10

正确性直觉：

1. 线性响应使电荷扰动满足指数屏蔽，而非纯 `1/r` 长程；
2. `u=r*phi` 变换后方程是常系数二阶 ODE，离散后应得到三对角系统；
3. 若离散和求解正确，数值解应贴近解析 Yukawa 势；
4. 在 `r=lambda_D` 处，屏蔽因子应约为 `e^-1`；
5. 密度提高会缩短德拜长度，比例为 `1/sqrt(n)`。

`demo.py` 断言分别对应 3-5 条。

## R11

数值稳定性处理：

- 径向网格从 `r_min>0` 开始，避开点电荷 `r=0` 奇点；
- 均匀网格并显式检查 `dr` 一致性；
- 三对角求解在前消元阶段检查主元，防止奇异除法；
- 相对误差分母使用下限 `1e-30`，防止极小值放大。

## R12

关键参数（`DebyeConfig`）：

- `n_e_m3`, `n_i_m3`, `z_i`：决定电荷响应强度；
- `t_e_eV`, `t_i_eV`：温度越高，热运动越强，屏蔽越弱（`lambda_D` 增大）；
- `r_min_over_lambda`, `r_max_over_lambda`：控制近场和远场采样范围；
- `n_grid`：离散精度与耗时平衡；
- `density_multiplier_probe`：用于标度关系 sanity check。

调参建议：若追求更高数值精度，优先增大 `n_grid`。

## R14

常见失效模式：

1. 温度或密度输入非正，导致 `lambda_D` 无定义；
2. 在 `r=0` 直接离散，触发奇点数值爆炸；
3. 网格过粗导致指数衰减拟合偏差；
4. 忘记多组分求和，只算电子项或离子项；
5. 单位混淆（eV 与 K、SI 与 cgs）。

脚本对 1 和网格合法性做了显式检查。

## R18

`demo.py` 源码级算法流程（9 步）：

1. `main` 构造 `DebyeConfig`，调用 `build_profiles` 生成势剖面与误差指标。  
2. `build_profiles` 先调用 `debye_length_two_species` 计算 `lambda_D`，该函数按 `sum_s n_s q_s^2/(epsilon_0 k_B T_s)` 显式累加电子/离子贡献。  
3. 同函数按 `r_min_over_lambda` 到 `r_max_over_lambda` 构造均匀径向网格。  
4. `solve_potential_fd` 把方程写成 `u=r*phi` 的离散形式：`u_{i-1}-(2+alpha)u_i+u_{i+1}=0`，形成三对角线性系统。  
5. `solve_potential_fd` 在两端使用解析势对应的 `u` 作为 Dirichlet 边界，把边界项并入 RHS。  
6. `thomas_solve_tridiagonal` 执行前消元 + 回代，`O(N)` 求得内点 `u`，再除以 `r` 还原数值势 `phi_num`。  
7. `build_profiles` 计算解析势 `phi_analytic`、裸库仑势 `phi_coulomb`、屏蔽因子 `phi_num/phi_coulomb` 与相对误差，整理成 `pandas.DataFrame`。  
8. `run_sanity_checks` 做四个断言：离散误差阈值、`r=lambda_D` 处 `exp(-1)` 检验、单调递减检验、`n->4n` 时 `lambda_D->lambda_D/2` 标度检验。  
9. `main` 打印样本点与检查结果，形成可复现、可审查的最小交付。  

第三方库角色说明：`numpy` 仅用于数组运算和离散网格，`pandas` 仅用于结果表展示；德拜长度计算、有限差分离散、三对角求解和物理断言均在源码中显式实现，没有调用黑盒求解流程。
