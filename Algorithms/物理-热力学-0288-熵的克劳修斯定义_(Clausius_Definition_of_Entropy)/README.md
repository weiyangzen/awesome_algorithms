# 熵的克劳修斯定义 (Clausius Definition of Entropy)

## R01

克劳修斯定义把熵引入为可逆传热的积分量：

`dS = δQ_rev / T`

因此任意两个平衡态 `1 -> 2` 的熵差定义为

`ΔS = ∫(1->2, rev) (δQ_rev / T)`。

要点：

- 熵 `S` 是状态函数，`ΔS` 只依赖初末态；
- 积分路径必须取“可逆参考路径”；
- 实际过程可逆与否，不影响状态函数 `ΔS` 的值。

## R02

该定义是热力学第二定律的核心工程接口，直接连接三类问题：

- 状态函数计算：用可逆积分求 `ΔS`；
- 不可逆过程诊断：用熵平衡识别熵产生；
- 上界约束设计：为效率、最小耗功提供判据。

本条目 MVP 目标：用最小代码验证“路径无关 + 不可逆熵产生”这两个核心结论。

## R03

`demo.py` 输入输出（脚本内置参数，无交互）：

- 输入：
1. 理想气体两个状态 `state_1, state_2`（`T, V, n, Cv`）；
2. 两条可逆参考路径（A: 等容+等温，B: 等温+等容）；
3. 一个不可逆自由膨胀案例（绝热、真空）。
- 输出：
1. 三种方法计算的 `ΔS`（闭式、路径A积分、路径B积分）；
2. 数值积分误差估计；
3. 自由膨胀的 `ΔS`、`∫δQ/T` 与 `S_gen`；
4. 自动断言与 `All checks passed.`。

## R04

MVP 使用的核心关系：

1. 克劳修斯定义：
`dS = δQ_rev/T`。

2. 理想气体熵差闭式：
`ΔS = nCv ln(T2/T1) + nR ln(V2/V1)`。

3. 可逆等容段：
`δQ_rev = nCv dT`，故 `ΔS_isochoric = ∫ nCv/T dT`。

4. 可逆等温段：
`δQ_rev = nRT dV/V`，故 `ΔS_isothermal = ∫ nR/V dV`。

5. 熵平衡（系统）：
`ΔS = ∫(actual) δQ/T + S_gen`，且 `S_gen >= 0`。

## R05

复杂度分析（单次评估）：

- 闭式公式：`O(1)`；
- 两条路径积分：每次 `quad` 为自适应求积，函数评估次数记为 `K`，则约 `O(K)`；
- 空间复杂度：`O(1)`（仅少量标量和小表格）。

## R06

`demo.py` 的最小算法闭环：

- 闭环 A（状态函数）：`entropy_change_closed_form` 直接给出 `ΔS` 参考真值；

- 闭环 C（不可逆诊断）：`irreversible_free_expansion_budget` 计算 `S_gen` 并验证其为正。

## R10

正确性直觉：

1. 克劳修斯定义把熵差固定为“可逆热量除温度”的积分；
2. 若 `S` 真是状态函数，则任意可逆路径积分必须相同；
3. 代码中路径 A 与路径 B 对同一初末态给出相同 `ΔS`，就是状态函数性质的离散验证；
4. 对不可逆自由膨胀，实际路径 `∫δQ/T=0` 但 `ΔS>0`，故必有 `S_gen>0`，与第二定律一致。

## R11

数值稳定性与实现策略：

- 所有状态量强制正值检查（`T,V,n,Cv > 0`）；
- 状态对检查 `n` 与 `Cv` 一致，避免不自洽积分；
- 使用 `float64` 和对数表达式减少放缩误差；
- 对路径等价性断言使用小容差 `1e-9`。

## R14

常见失效模式：

1. 温度误用摄氏度而非开尔文；
2. 体积或温度填入非正值导致对数或积分失效；
3. 混淆“实际路径积分”和“可逆参考路径积分”；
4. 把 `Cv` 当常数用于温区过宽的真实工质，导致模型偏差；
5. 忽略数值容差，把机器误差当作物理差异。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `IdealGasState` 封装 `T,V,n,Cv`，`validate` 强制全部为正，保证 `ln` 与积分定义域合法。  
2. `_validate_state_pair` 检查两状态 `n_mol` 与 `cv_molar` 一致，保证比较的是同一封闭系统模型。  
3. `entropy_change_closed_form` 用 `nCv ln(T2/T1) + nR ln(V2/V1)` 给出状态函数基准值。  
4. `_integrand_isochoric` 与 `_integrand_isothermal` 分别实现 `nCv/T`、`nR/V`，把 `δQ_rev/T` 明确成可积函数。  
5. `entropy_change_path_a` 调用 `quad` 分别积分等容段和等温段并求和，得到路径 A 的 `ΔS`。  
6. `entropy_change_path_b` 交换积分段顺序再次积分，得到路径 B 的 `ΔS`，用于验证路径无关。  
7. `irreversible_free_expansion_budget` 设定实际过程 `∫δQ/T=0`，再由状态函数 `ΔS` 反推 `S_gen = ΔS - ∫δQ/T`。  
8. `main` 汇总表格输出并断言：`ΔS_closed ≈ ΔS_A ≈ ΔS_B` 且自由膨胀 `S_gen > 0`，形成最小自证闭环。  

补充：`scipy.integrate.quad` 仅用于一维数值积分执行；被积函数、物理量构造、结果判定都在源码显式展开。
