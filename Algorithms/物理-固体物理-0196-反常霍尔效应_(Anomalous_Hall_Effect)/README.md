# 反常霍尔效应 (Anomalous Hall Effect)

- UID: `PHYS-0195`
- 学科: `物理`
- 分类: `固体物理`
- 源序号: `196`

## R01

反常霍尔效应（AHE）指在无外磁场时，磁性体系仍出现横向电导 `σ_xy` 的现象。其内禀机制来自布里渊区中的 Berry 曲率：

`σ_xy = -(e^2/h) * (1/(2π)) * Σ_n ∫ f_n(k) Ω_n(k) d^2k`。

本条目给出一个最小可运行数值实现：采用二维 massive Dirac 模型，显式计算能带、Berry 曲率与 Fermi-Dirac 占据，再做二维数值积分得到 `σ_xy`。

## R02

物理建模假设：

1. 两能带有效模型：`H(k)=kx*σx + ky*σy + m*σz`（`ħ=v_F=1`）；
2. 仅计算内禀 AHE（Berry 曲率项），不包含杂质导致的 skew scattering / side jump；
3. 有限温度 `T` 下用 Fermi-Dirac 分布；
4. 在有限 `k` 截断盒 `[-k_max, k_max]^2` 上离散积分（MVP 近似）。

## R03

`demo.py` 的输入输出（无交互，参数在脚本内）：

- 输入参数：`mass`, `temperature`, `k_max`, `nk`, `mu_values`；
- 主输出：每个化学势 `μ` 对应的 `sigma_xy_e2_over_h`；
- 参考输出：`sigma_t0_continuum`（连续极限 `T=0` 理论式）；
- 误差列：`abs_err_vs_t0_continuum`；
- 终止标志：全部断言通过后打印 `All checks passed.`。

## R04

核心公式（对应代码中的显式实现）：

1. 能带色散：`E_±(k)=±sqrt(kx^2+ky^2+m^2)`；
2. Berry 曲率：
   `Ω_-(k)= -m / (2*(kx^2+ky^2+m^2)^(3/2))`，`Ω_+=-Ω_-`；
3. 费米分布：`f(E)=1/(exp((E-μ)/T)+1)`；
4. 电导积分：
   `σ_xy = -(1/(2π)) ∫ [f_+Ω_+ + f_-Ω_-] d^2k`（结果以 `e^2/h` 为单位）；
5. 理论参考（单 Dirac 锥，`T=0` 连续极限）：
   - `|μ|<=|m|` 时 `σ_xy = 0.5*sign(m)`；
   - `|μ|>|m|` 时 `σ_xy = m/(2|μ|)`。

## R05

设 `N = nk`，`M = len(mu_values)`：

- 构网格与预计算 `E, Ω`：`O(N^2)`；
- 每个 `μ` 计算占据与积分：`O(N^2)`；
- 总时间复杂度：`O(M*N^2)`；
- 空间复杂度：`O(N^2)`（存储二维网格场）。

## R06

`demo.py` 代码结构：

- `AHEConfig`：集中管理参数并做合法性检查；
- `DiracGrid`：保存离散 `k` 轴、能量与 Berry 曲率缓存；
- `build_dirac_grid`：构造二维网格和物理场；
- `fermi_dirac`：稳定计算占据（含 `temperature=0` 分支）；
- `intrinsic_sigma_xy`：执行二维 Simpson 积分；
- `run_sweep`：遍历 `μ` 形成 `pandas.DataFrame`；
- `run_consistency_checks`：执行物理一致性断言；
- `main`：运行主任务并打印表格。

## R07

优点：

- 代码短小可运行，便于教学与单元测试；

## R10

正确性直觉：

1. `m` 决定 Berry 曲率手性，故 `σ_xy` 应随 `m` 反号；
2. `μ` 落在能隙内时，低温下横向电导接近平台值（单锥约半量子）；
3. `|μ|` 远离能隙时两带占据趋向抵消，`|σ_xy|` 应下降；
4. 代码中检查 `σ(μ)≈σ(-μ)`（该模型粒子-空穴对称下的结果）与质量翻转反号。

## R11

数值稳定策略：

- 费米分布指数输入截断到 `[-80, 80]`，避免溢出；
- `nk` 强制奇数以匹配 Simpson 规则精度；
- 预计算 `E(k)` 与 `Ω(k)`，减少重复计算带来的噪声与开销；
- 用相对宽松但明确的误差阈值（`mean_abs_err <= 0.06`）容纳有限截断误差。

## R12

关键参数及影响：

- `mass`：控制 Berry 曲率符号与强度，决定 `σ_xy` 的符号；
- `temperature`：越高越平滑占据，平台越钝化；
- `k_max`：积分截断上限，太小会低估平台值；
- `nk`：网格分辨率，太低会增加积分离散误差；
- `mu_values`：扫描区间和采样密度，决定能否清晰观察平台到衰减过渡。

调参建议：

- 若平台值偏低，优先增大 `k_max`；
- 若曲线不平滑，增大 `nk`；
- 若想更接近 `T=0` 理论，降低 `temperature` 并适度加密网格。

## R14

常见失效模式：

1. `mass=0`：模型退化，Berry 曲率公式失去有效意义；
2. `nk` 太小：二维积分误差大，趋势不稳定；
3. `k_max` 太小：截断误差明显，平台值被系统性压低；
4. `temperature` 过高：平台结构被强烈热展宽；
5. `mu_values` 未排序：参数检查失败。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `AHEConfig.validate()` 检查输入参数合法性（`mass`、`k_max`、`nk`、`mu_values`）。
2. `build_dirac_grid()` 在 `[-k_max,k_max]^2` 上建网格，并显式计算 `E(k)` 与两条能带的 Berry 曲率。
3. `fermi_dirac()` 对给定 `μ,T` 计算 `f_+(k), f_-(k)`，并通过指数截断保证数值稳定。
4. `intrinsic_sigma_xy()` 构造被积函数 `-(f_+Ω_+ + f_-Ω_-)`。
5. `intrinsic_sigma_xy()` 用 `scipy.integrate.simpson` 先沿 `ky` 再沿 `kx` 做二维积分，并乘 `1/(2π)` 得到 `σ_xy`。
6. `run_sweep()` 对所有 `μ` 重复步骤 3-5，并用 `sigma_xy_t0_continuum()` 计算 `T=0` 连续理论参考值。
7. `run_consistency_checks()` 检查物理趋势：隙内同号、`|μ|` 增大时衰减、`σ(μ)≈σ(-μ)`、平均参考误差受控。
8. `main()` 再做一次 `mass -> -mass` 的反号验证，打印结果表并在全部通过后输出成功标志。

说明：`scipy` 只用于通用数值积分；AHE 的物理主流程（哈密顿量、曲率、占据、积分构造和一致性检查）均在源码中显式展开，不是第三方黑盒调用。
