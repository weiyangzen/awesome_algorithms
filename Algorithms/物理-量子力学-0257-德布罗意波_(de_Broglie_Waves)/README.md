## R01

德布罗意波假设指出：任何具有动量 `p` 的微观粒子都可对应一条物质波，其波长为

`lambda = h / p`

其中 `h` 是普朗克常数。这个关系把“粒子动力学量（动量）”与“波动学量（波长）”直接连接起来，是量子力学从经典力学过渡的核心桥梁之一。

## R02

该关系解释了电子衍射、中子衍射、原子干涉等实验现象，也给出电子显微镜分辨能力的量级基础。工程上，常用它快速估算“给定粒子能量时波长有多短”，进而判断是否能分辨某个空间尺度。

## R03

本条目 MVP 解决四个最小闭环问题：

- 由给定电子动能（eV）计算非相对论动量与德布罗意波长；
- 用线性回归验证 `lambda` 与 `1/p` 的线性关系，并恢复斜率 `h`；
- 比较非相对论与相对论电子波长在不同加速电压下的偏差；
- 使用 NumPy 与 PyTorch 两条计算路径交叉核验数值一致性。

目标是实现“物理公式 -> 数值实现 -> 自动断言验证”的可运行最小样例。

## R04

本实现使用的核心方程：

1. 非相对论动量：`p_nr = sqrt(2 m K)`；
2. 德布罗意波长：`lambda = h / p`；
3. 非相对论电子加速电压模型：`K = e V`；
4. 相对论总能量关系：`E^2 = (pc)^2 + (mc^2)^2`；
5. 相对论动量：`p_rel = sqrt((mc^2 + K)^2 - (mc^2)^2) / c`；
6. 自由粒子（非相对论）群速度/相速度关系：`v_g = p/m`, `v_p = v_g / 2`。

## R05

若采样能量点数为 `n`、电压点数为 `m`：

- 物理量逐点计算复杂度为 `O(n)`；
- 线性回归（单特征）复杂度为 `O(n)`；
- 相对论修正对比复杂度为 `O(m)`；
- PyTorch 一致性核验复杂度为 `O(n)`。

总时间复杂度 `O(n + m)`，空间复杂度 `O(n + m)`（用于保留表格与报告）。

## R10

正确性直觉：

1. 固定质量下，动能增大 `=>` 动量增大 `=>` 波长缩短；
2. `lambda` 对 `1/p` 必须是过原点直线，斜率应等于 `h`；
3. NumPy 与 PyTorch 对同一算式应得到机器精度内一致结果；
4. 当 `K` 接近 `mc^2` 的较大比例时，非相对论近似误差必须明显上升。

## R11

数值稳定与精度策略：

- 全流程使用 `float64` 以降低单位转换和开方误差；
- 输入动量/能量/电压均强制正值，避免非法开方；
- 回归同时检查斜率、截距与相关系数，避免单指标误判；
- 通过低压/高压两端断言检查相对论修正趋势是否合理。

## R14

常见失效模式：

1. 把 eV 直接当作 J 使用，造成数量级错误；
2. 在高能区仍只用 `p=sqrt(2mK)`，低估相对论修正；
3. 输入包含非正动能或电压，导致开方无定义；
4. 用 `float32` 或过松阈值导致验证敏感性不足。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `ParticleBatch` 定义粒子质量与能量样本，作为一次批量计算输入。  
2. `momentum_nonrelativistic` 把 `K` 转为非相对论动量 `p_nr=sqrt(2mK)`。  
3. `de_broglie_wavelength` 按 `lambda=h/p` 计算物质波波长。  
4. `compute_de_broglie_table` 组合 `K, p, lambda, k, v_g, v_p` 并输出 `pandas.DataFrame`。  
5. `fit_lambda_inverse_p` 用 `LinearRegression` 拟合 `lambda=a*(1/p)+b`，提取 `a,b` 并与理论 `h,0` 对照，同时用 `scipy.stats.pearsonr` 检查线性相关性。  
6. `torch_numpy_consistency_check` 用 PyTorch 重走同一公式，计算与 NumPy 结果的最大绝对差，验证实现一致性。  
7. `electron_wavelength_vs_voltage` 依据相对论能量动量关系计算 `lambda_rel`，并与 `lambda_nonrel` 比较相对误差。  
8. `main` 串联四个 demo，打印结果并执行断言（斜率、截距、相关性、框架一致性、低压/高压误差趋势），形成非交互自动验证。  

第三方库没有被当作黑盒：核心物理映射（`K->p->lambda`、相对论修正）和校验逻辑都在源码中显式实现；`sklearn` 与 `scipy` 仅承担标准统计/回归算子。
