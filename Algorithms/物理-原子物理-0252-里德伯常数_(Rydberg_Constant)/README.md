## R01

里德伯常数 `R` 描述氢样原子谱线波数尺度，是原子物理中连接量子数与可观测光谱线位置的核心常数。对无限重核（`R_inf`）有：

`R_inf = alpha^2 * m_e * c / (2h)`

其中 `alpha` 为精细结构常数，`m_e` 电子质量，`c` 光速，`h` 普朗克常数。

## R02

该条目重要性在于：

- 它把微观常数与宏观可测光谱波长直接连接；
- 是验证量子理论与常数一致性的经典入口；
- 也是谱线反演、元素识别、精密计量的基础步骤；
- 可作为线性回归与物理建模结合的教学示例。

## R03

## R04

MVP 使用公式：

1. 无限重核里德伯常数：
`R_inf = alpha^2 * m_e * c / (2h)`。

2. 有限核质量修正（核质量 `M`）：
`R_M = R_inf / (1 + m_e/M)`。

3. Rydberg 公式（Balmer 系列）：
`1/lambda = R_M * (1/n1^2 - 1/n2^2), n1=2, n2>2`。

4. 线性化：令 `x = (1/4 - 1/n2^2)`，`y = 1/lambda`，则 `y = R_M * x`。

5. 非线性拟合形式：
`lambda(n2; R) = 1 / (R * (1/4 - 1/n2^2))`。

## R14

1. 把 `R_inf` 与 `R_H` 混用，忘记有限核质量修正；

3. 波长单位不统一（nm 与 m 混用）；

## R18

`demo.py` 源码级算法流程（8 步）：

1. `rydberg_infinite_from_constants` 从 `alpha,m_e,c,h` 按公式显式计算 `R_inf`，再用 `rydberg_with_reduced_mass` 得到 `R_H`。  
2. `generate_balmer_dataset` 先按 `1/lambda = R_H*(1/4-1/n2^2)` 生成无噪声数据，再注入固定种子的 ppm 级噪声，构造“观测波长”。  
3. 构建线性特征 `x=(1/4-1/n2^2)` 与目标 `y=1/lambda_obs`，写入 `pandas.DataFrame` 便于审计。  
4. `estimate_rydberg_sklearn` 调用 `LinearRegression(fit_intercept=False)`：内部执行最小二乘，求解单参数斜率 `R_hat`；源码中显式指定无截距以匹配物理模型。  
5. `estimate_rydberg_curve_fit` 在波长域拟合 `lambda(n2;R)=1/(R*(1/4-1/n2^2))`：`curve_fit` 迭代更新 `R`，最小化残差平方和。  
6. `estimate_rydberg_torch_optional`（若可用）将同一线性模型写成一参数张量回归，通过 Adam 梯度下降迭代收敛到 `R_hat`。  
7. `build_estimate_summary` 汇总各估计器结果，计算相对误差（相对 `R_H`）并打印对照表。  
8. `main` 统一执行断言：常数一致性、拟合误差阈值、跨方法一致性；全部通过后输出 `All checks passed.`。  

说明：第三方库仅提供“数值求解器与表格容器”；常数推导、特征构造、物理方程、阈值判定都在源码中展开，不是黑盒调用即结束。
