## R01

后验误差估计（a posteriori error estimation）指的是：

- 在已经得到数值解 `u_h` 之后，
- 仅利用离散解和方程数据（不依赖精确解），
- 计算一个可操作的误差指标 `eta`，用于估计真实误差大小与分布。

它是自适应网格算法（`SOLVE -> ESTIMATE -> MARK -> REFINE`）的核心环节。

## R02

本条目采用 1D Poisson 问题作为最小可运行场景：

- 方程：`-u''(x) = pi^2 sin(pi x), x in (0,1)`
- 边界：`u(0)=u(1)=0`
- 精确解：`u(x)=sin(pi x)`（仅用于离线验证，不用于估计器定义）

输入：单元数序列 `N=(8,16,32,64,128)` 与若干积分阶数。  
输出：每个网格上的

- `L2` 误差
- 能量范误差 `||u'-u_h'||_{L2}`
- 后验估计量 `eta`
- 比值 `eta / energy_error`
- 最大局部指示子 `max_etaK`

## R03

离散模型为 1D P1 有限元：

- 网格节点 `x_0=0 < x_1 < ... < x_n=1`
- 试探空间 `V_h` 为分片线性连续函数并满足齐次 Dirichlet 边界
- 弱形式：求 `u_h in V_h`，使

`int_0^1 u_h'(x) v_h'(x) dx = int_0^1 f(x) v_h(x) dx, forall v_h in V_h`

实现中显式组装刚度矩阵与载荷向量，再求解线性方程组。

## R04

采用残差型后验估计量（residual estimator）：

`eta_K^2 = h_K^2 ||f||_{L2(K)}^2 + 0.5*h_K*(J_left^2 + J_right^2)`

其中：

- `h_K` 是单元长度
- `J` 是节点处导数跳跃：`J_i = (u_h'|_{K+}) - (u_h'|_{K-})`
- 对 P1 元，单元内部 `u_h''=0`，体残差项可写为 `h_K^2 ||f||^2`

全局估计量：`eta = sqrt(sum_K eta_K^2)`。

## R05

复杂度（单次求解）在本 MVP 中为：

- 组装：`O(N)`（每个单元固定阶积分）
- 稀疏求解（SciPy 可用时）：约 `O(N)` 到 `O(N log N)` 量级
- 稠密回退（无 SciPy）：`O(N^3)`（仅为依赖缺失时兜底）
- 误差与估计量评估：`O(N)`

对演示规模（最大 128 单元）开销很小。

## R06

微型直观例（仅说明跳跃项含义）：

- 设网格有两个相邻单元 `K1=[x0,x1], K2=[x1,x2]`
- FEM 解在每个单元上的斜率分别为 `s1=(u1-u0)/h1`, `s2=(u2-u1)/h2`
- 中间节点 `x1` 的跳跃为 `J_1 = s2 - s1`

如果 `|J_1|` 大，说明该处导数不连续程度高、局部误差风险大；估计器会通过 `0.5*h*J^2` 项给出更高指示值，驱动局部加密。

## R07

后验误差估计的工程价值：

- 不需要精确解即可做误差控制
- 能定位误差集中区域（用于局部加密，而非全局盲目细化）
- 可以作为停止准则（当 `eta` 低于阈值即停止）
- 能连接“数学可靠性”与“可执行网格策略”

## R08

理论上常关注两类性质：

1. 可靠性（Reliability）：`||u-u_h|| <= C_rel * eta`（估计量不低估误差过多）
2. 效率性（Efficiency）：`eta <= C_eff * ||u-u_h|| + osc`（估计量不过分保守）

本 demo 没有证明常数，但通过数值实验观察 `eta / energy_error` 的稳定性，验证其作为可用误差代理的实际价值。

## R09

适用前提与限制：

- 当前仅覆盖 1D、线性椭圆、齐次 Dirichlet、P1 元
- 估计器形式针对该场景写死，未抽象到一般 PDE
- 非光滑解、强系数跳变、多维网格会改变常数与表现
- 结果可用于“方法演示与验证”，不是通用工业级求解器

## R10

正确性检查策略（`demo.py` 内置断言）：

- 能量误差随网格加密单调下降
- 估计量 `eta` 也随网格加密单调下降
- 最细网格能量误差达到阈值（`< 3e-2`）
- `eta / energy_error` 在经验区间内（`3.5 ~ 6.5`）

这些检查保证了数值流程和估计量行为与理论预期一致。

## R11

一次实测输出（`python3 demo.py`）显示：

- `EnergyErr`：`2.51e-1 -> 1.57e-2`（单调下降）
- `eta`：`1.23e+0 -> 7.71e-2`（单调下降）
- `eta/energy_error`：约 `4.88 ~ 4.90`，跨网格基本稳定
- 能量误差收敛阶（每次 `h` 折半）约 `0.997 ~ 1.000`

说明该后验估计器在本模型上是“稳定且偏保守”的误差指示器。

## R14

常见失效模式与对策：

- 积分阶太低：会污染装配和误差评估，导致指标偏差
- 网格过粗：`eta/error` 的经验常数可能暂时波动
- 高维或复杂边界直接套用本公式：会失真
- 仅看全局 `eta` 不看局部 `eta_K`：会丢失自适应价值

## R15

`demo.py` 结构说明：

- `solve_poisson_p1`：组装并求解 1D P1 FEM 线性系统
- `residual_estimator`：计算 `eta_K` 与全局 `eta`
- `exact_errors`：用解析解计算 `L2` 与能量误差（仅验证用）
- `run_case`：单个网格实验
- `run_suite`：批量实验
- `validate_results`：自动断言校验
- `print_report`：输出指标表与观测收敛阶
- `main`：非交互入口

## R16

与相邻算法主题的关系：

- 本条目是“后验估计器”本体实现；

## R18

源码级算法流程拆解（`demo.py`，8 步）：

1. `main()` 读取 `Config`，设定网格序列 `8,16,32,64,128`。  
2. `run_suite()` 逐个网格调用 `run_case()` 执行求解与估计。  
3. `run_case()` 用 `make_uniform_mesh()` 生成一维均匀网格。  
4. `solve_poisson_p1()` 对每个单元计算局部刚度与载荷，汇总成全局系统并施加 Dirichlet 边界后求解 `u_h`。  
5. `residual_estimator()` 计算单元斜率、节点跳跃 `J_i`，再按 `h^2||f||^2 + 0.5h(J_l^2+J_r^2)` 得到每个 `eta_K` 与全局 `eta`。  
6. `exact_errors()` 在高斯点上积分，得到 `L2` 与能量误差 `||u'-u_h'||`（仅用于验证，不参与估计器定义）。  
7. `print_report()` 打印每个网格的误差、估计量、比值和观测收敛阶。  
8. `validate_results()` 依次检查单调下降、终阶精度和 `eta/error` 区间，全部通过后输出 `All checks passed.`。

说明：实现没有调用“后验误差估计黑盒 API”，估计器每个组成项都在源码中显式展开。
