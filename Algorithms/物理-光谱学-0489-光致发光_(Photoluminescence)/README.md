# 光致发光 (Photoluminescence)

- UID: `PHYS-0466`
- 学科: `物理`
- 分类: `光谱学`
- 源序号: `489`
- 目标目录: `Algorithms/物理-光谱学-0489-光致发光_(Photoluminescence)`

## R01

光致发光（PL）描述材料吸收激发光后，经辐射复合发射光子的过程。实验上常记录发射谱 `I(λ)`，用于判断材料带隙、缺陷态、能量转移和浓度变化。

在定量任务中，目标通常是从整段发射光谱估计某一组分浓度 `c_target`。理想化下可写成：

- `I(λ) ≈ K * Φ * c_target * S(λ)`

其中 `Φ` 为量子效率，`S(λ)` 为归一化发射线型，`K` 包含激发功率、光路耦合和探测效率。

## R02

PL 从早期“峰位/峰强人工读数”逐步演化到“全谱建模 + 自动预处理”的分析流程，关键推动来自：

- CCD 光谱仪普及，使高分辨全谱采集成为常规；
- 数字信号处理（平滑、基线扣除）替代手工读图；
- 机器学习回归可在峰重叠和分布漂移下保持更稳健定量。

因此 PL 问题天然适合“物理可解释基线 + 数据驱动增强”的混合方案。

## R03

本条目要解决的问题：给定含噪声、基线漂移、干扰发射和猝灭效应的 PL 光谱，估计目标发光体浓度。

主要困难：

1. 猝灭（Stern-Volmer）使峰强与浓度关系偏离简单线性；
2. 干扰发光体峰位重叠，单峰法可分性下降；
3. 仪器基线、暗电流与随机噪声会抬高误差。

本目录 MVP 采用三层策略：

- 单波长线性标定（可解释、可审计）；
- 全谱 Ridge（线性多波长融合）；
- 全谱 PyTorch MLP（非线性补偿猝灭与漂移）。

## R04

MVP 使用的合成物理模型：

- `I(λ) = A_t * E_t(λ) + A_i * E_i(λ) + B(λ) + N(λ)`
- `A_t = G * c_t * exp(-β(c_t + c_i)) / (1 + Ksv * q)`
- `A_i = G * c_i * (1 - 0.15 q)`

符号含义：

- `E_t(λ), E_i(λ)`：目标/干扰发射线型；
- `q`：猝灭剂浓度；
- `β`：内滤效应强度；
- `Ksv`：Stern-Volmer 常数；
- `B(λ)`：低阶多项式基线；
- `N(λ)`：高斯噪声。

## R05

记样本数 `N`、波长数 `M`、MLP 训练轮数 `E`、隐藏维 `H`：

- 光谱预处理（平滑 + 基线拟合）：`O(N*M)`；
- 单波长线性标定与预测：`O(N)`；
- Ridge 训练：约 `O(N*M^2 + M^3)`；
- MLP 训练：约 `O(E*N*M*H)`。

本实现中 `M=361`，CPU 即可快速运行。

## R06

单波长标定示例（简化）：

- 选分析波长 `λ*=548 nm`；
- 基线校正后峰强 `I_corr(λ*) = 0.62`；
- 标定线 `c = a * I + b`，若 `a=0.95, b=0.04`，则：

- `c_hat = 0.95 * 0.62 + 0.04 = 0.629`

该方法可解释性强，但在峰重叠与猝灭增强时会受限。

## R07

优点：

- 与发光机理一致，便于追踪误差来源；
- 能直接连接实验参数（激发、猝灭、内滤）；
- 便于与全谱机器学习组合构建稳健管线。

局限：

- 强重叠或显著非线性时，单波长法偏差较大；
- 模型外分布（shifted 仪器状态）会降低纯数据模型泛化；
- 需要足够覆盖的标定样本，避免欠拟合/过拟合。

## R09

适用场景：

- 荧光探针浓度或反应转化监测；
- 材料发光性能筛选（批量快速比对）；
- 算法教学与验证（物理先验 + ML）。

需谨慎场景：

- 强自吸收或光漂白主导，且无对应校正；
- 目标峰过弱，信噪比不足；
- 缺少覆盖生产工况的标定集却直接上线复杂模型。

## R10

正确性检查点：

1. 预处理后边缘波段基线应接近 0；
2. 自动选出的 `λ*` 应位于目标发射主峰附近；
3. 单波长预测应保持非负且量级合理；
4. Ridge/MLP 在 ID 集上应显著优于常数基线；
5. shifted 集性能下降应可解释，而非数值发散。

## R11

数值稳定性设计：

- 使用奇数窗口 Savitzky-Golay（`window=9, poly=3`）抑制高频噪声；
- 基线拟合仅使用低分位点，减少峰区污染；
- 所有输出做 `clip(0, +inf)` 避免不合理负浓度；
- MLP 输入标准化，目标做 z-score，提高训练稳定性；
- 固定随机种子，保证可复现。

## R12

调参建议（对应 demo）：

- `window_length` 过大将抹平窄峰，过小则去噪不足；
- Ridge `alpha` 增大可抑制方差但会提高偏差；
- MLP `epochs/lr/weight_decay` 需联合调节；
- shifted 数据可通过增加训练分布覆盖与数据增强改善。

## R13

理论性质：

- 单波长线性标定属于一元线性回归，解析解唯一；
- Ridge 目标函数为强凸（二次项 + `alpha>0`），全局最优可得；
- MLP 为非凸优化，不保证全局最优，但能逼近非线性映射；
- 本任务是监督回归，不涉及近似比或随机化可行性证明。

## R14

常见失效模式与防护：

- 失效：目标峰与干扰峰严重重叠。  
  防护：由单峰升级到全谱 Ridge/MLP。
- 失效：猝灭强度超出训练范围。  
  防护：扩展标定区间并加入 shifted 验证。
- 失效：基线大幅漂移导致峰强虚高。  
  防护：先做基线校正再回归。
- 失效：模型对新仪器泛化差。  
  防护：做跨仪器标准化与迁移标定。

工程实践建议：

- 固定预处理超参数并版本化；
- 同时维护“可解释模型（单峰）”与“高性能模型（全谱）”；

## R16

相关方法对比：

- 单峰线性标定：解释性最好，复杂场景鲁棒性较弱；
- Ridge/PLS：适合高维共线光谱，训练稳定；
- 树模型（如 CatBoost）：对非线性友好，但解释路径较碎片化；
- 神经网络：表达能力最强，但更依赖数据规模与正则化。

本 MVP 选择“单峰 + Ridge + MLP”，覆盖从可解释到高表达能力的连续方法谱。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `build_emission_profiles` 构建目标与干扰发光体的参考发射线型 `E_t(λ), E_i(λ)`。  
2. `simulate_pl_dataset` 按 `A_t/A_i` 模型生成光谱，显式加入猝灭、内滤效应、谱线平移、基线漂移和噪声。  
3. `preprocess_spectra` 对每条光谱执行 Savitzky-Golay 平滑，再用低分位点二次多项式拟合并扣除基线。  
4. `select_analytical_wavelength` 用 `scipy.signal.find_peaks` 找目标主峰，并按 `target/interferent` 分离度选定分析波长 `λ*`。  
5. `fit_single_wavelength_calibration` 在训练集峰强上拟合一元线性标定，`predict_single_wavelength` 给出可解释浓度估计。  
6. `train_ridge_model` 通过 `StandardScaler + Ridge` 在全谱特征上训练线性多波长回归器。  
7. `train_torch_mlp` 构建两层 MLP（PyTorch），采用 AdamW + MSE 在标准化特征上学习非线性映射。  
8. `main` 在 ID/shifted 集上计算 `MAE/RMSE/R2`，并用 `pandas` 打印模型对比与样例预测。  

实现没有把 PL 定量交给单一黑盒函数，关键流程（建模、预处理、选峰、标定、训练、评估）都可在源码中逐步追踪。
