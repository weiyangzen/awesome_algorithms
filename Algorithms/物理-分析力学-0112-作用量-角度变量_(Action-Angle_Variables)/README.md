## R01

作用量-角度变量是可积哈密顿系统中的一类规范变量变换，用来把周期运动写成“常量 + 匀速相位”。

- 一般定义（1 自由度束缚运动）:
  - `J = (1 / 2pi) ∮ p dq`
  - `theta = ∂S(q, J) / ∂J`（`S` 为 Hamilton-Jacobi 生成函数）
- 典型性质:
  - 在新坐标中 `H = H(J)`（不显含 `theta`）
  - 因而 `J_dot = -∂H/∂theta = 0`
  - `theta_dot = ∂H/∂J = omega(J)`，角变量线性增长

本目录 MVP 选取 1D 简谐振子作为最小可运行实例:

- `H(q,p)=p^2/(2m)+m*omega^2*q^2/2`
- 显式变换:
  - `q = sqrt(2J/(m*omega)) * sin(theta)`
  - `p = sqrt(2m*omega*J) * cos(theta)`
  - `J = p^2/(2m*omega) + m*omega*q^2/2`
  - `theta = atan2(m*omega*q, p)`

`demo.py` 的算法流程是:
1. 用 `(J0, theta0)` 构造初始 `(q0, p0)`。
2. 用 velocity-Verlet 积分哈密顿方程得到轨线 `(q(t), p(t))`。
3. 每步回算 `J(t), theta(t)`，验证 `J` 守恒、`theta` 线性增长。
4. 做 `(q,p)->(J,theta)->(q,p)` 往返一致性检验。

## R02

历史脉络（简版）:

- 1830s: Hamilton 建立哈密顿形式，为后续规范变换奠定基础。
- 1860s: Jacobi 发展 Hamilton-Jacobi 方程，给出“变量分离 + 生成函数”路线。
- 19 世纪末到 20 世纪: Delaunay、Poincare 等把作用-角变量系统化用于天体力学摄动问题。
- 20 世纪后半: 在 KAM 理论、半经典量子化（`J ~ n*h`）等领域成为核心语言。

## R03

要解决的核心问题是: 对周期性系统，原变量 `(q,p)` 的轨线是振荡的，长期分析不直观，且摄动计算困难。

作用量-角度变量将动力学压缩为:

- 振幅信息进 `J`（常量或慢变量）
- 相位信息进 `theta`（近似匀速）

这样可以把“复杂振荡轨迹”改写成“环面上匀速流”，显著降低长期演化和共振分析难度。

## R04

- 时间复杂度:

  - 回算 `J(t), theta(t)`: `O(N)`
  - 线性拟合与误差评估: `O(N)`

- 最坏/典型:
  - 本 MVP 为固定 1 自由度，最坏与典型同阶，均为线性。

## R05

空间复杂度取决于是否保存全轨线:

- 本实现保存 `t,q,p,J,theta` 全部序列: `O(N)`
- 若仅做在线统计（守恒量最大偏差、频率估计）: 可降为 `O(1)` 额外空间

MVP 选择 `O(N)`，目的是让验证表格直观可读。

## R06

`demo.py` 默认参数:

- `m=1.3`, `omega=2.4`, `J0=1.1`, `theta0=0.35`
- `dt=0.0015`, `n_steps=8000`, 总时长 `T=12.0`

一次实际运行得到（同机可复现）:

- `max |J(t)-J0|/J0 = 2.859053e-06`
- `RMS(theta_num - theta_expected) = 8.554790e-06`
- 由线性拟合得到角速度 `omega_fit = 2.40000129`
- 频率相对误差 `5.383249e-07`
- 往返一致性:
  - `max |dq| = 2.664535e-15`
  - `max |dp| = 7.327472e-15`

这表明数值积分与动作-角变量映射在双精度下达到高一致性。

## R09

适用前提与边界:

- 适用前提:
  - 系统可积，且轨道是周期/准周期闭合（至少局部可定义角变量）。
  - 本 MVP 限定为 1D 简谐振子。
- 边界条件:
  - `m > 0`, `omega > 0`, `J >= 0`。
  - `J -> 0` 时振幅趋零，角度定义会出现数值退化（相位不敏感）。
- 不适用:
  - 强混沌非可积系统无法全局定义动作-角变量。
  - 非平滑势或碰撞型系统需额外事件处理。

## R10

正确性依据分为解析层与离散层。

1. 解析层（模型正确）:
   - 对简谐振子，代入
     `q = sqrt(2J/(m*omega)) sin(theta)`,
     `p = sqrt(2m*omega*J) cos(theta)`
     可得 `H = omega * J`。
   - 因此哈密顿方程直接给出
     `J_dot = 0`, `theta_dot = omega`，与动作-角理论一致。
2. 离散层（数值实现正确）:
   - Velocity-Verlet 是二阶辛格式，长期保持几何结构，能把守恒量误差限制在有界振荡。
   - `demo.py` 通过三组可执行断言闭环验证:
     - `J` 相对漂移阈值
     - `theta` 线性跟踪误差阈值
     - 变量映射往返误差阈值

## R11

稳定性与误差传播:

- 离散误差:
  - Verlet 全局误差 `O(dt^2)`，局部截断 `O(dt^3)`。
- 长期行为:
  - 辛格式通常表现为“能量/动作量有界振荡”，而非单调漂移。
- 角变量误差:
  - `atan2` 在跨 `pi` 处有跳变，需 `unwrap` 处理连续相位。
- 数值风险:
  - `dt` 过大时会出现相位漂移增大甚至失稳。
  - `J` 很小时 `theta` 对噪声极敏感。

## R14

鲁棒性与失效模式:

- 失效模式 1:
  - 输入 `J<0` 或 `m,omega<=0`，物理/数学定义失效。
- 失效模式 2:
  - 步长过大导致相位误差明显、守恒量漂移变大。
- 失效模式 3:
  - `J≈0` 时角变量受舍入误差放大。
- 失效模式 4:
  - 忘记 `unwrap` 导致角度序列出现 `2pi` 跳变伪误差。

缓解方式:

- 严格参数校验（本实现已做）。
- 用守恒量和频率拟合作为自动回归指标。
- 近零振幅时优先分析 `J` 与能量，弱化 `theta` 指标。

## R18

源码级算法拆解（本条目未调用黑箱求解器，核心逻辑全部在 `demo.py`）:

1. 读取物理参数 `m, omega` 与初始动作-角 `(J0, theta0)`，做合法性检查。
2. 用显式解析公式把 `(J0, theta0)` 映射成 `(q0, p0)`。
3. 在 `velocity_verlet_harmonic` 中逐步推进:
   - 半步更新动量
   - 一步更新坐标
   - 再半步更新动量
4. 对每个时间点由 `(q,p)` 计算 `J(t)`，检查动作量是否守恒。
5. 对每个时间点由 `atan2(m*omega*q, p)` 计算 `theta(t)`，并 `unwrap` 成连续相位。
6. 构造解析参考 `theta_ref(t)=theta0+omega*t`，计算 RMS 相位误差。
7. 用 `polyfit` 估计 `theta` 的斜率，得到数值频率并比较相对误差。
8. 执行往返一致性 `(q,p)->(J,theta)->(q,p)` 检查与断言，给出最终通过/失败结论。
