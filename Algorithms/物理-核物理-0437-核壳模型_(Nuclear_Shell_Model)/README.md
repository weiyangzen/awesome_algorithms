# 核壳模型 (Nuclear Shell Model)

- UID: `PHYS-0417`
- 学科: `物理`
- 分类: `核物理`
- 源序号: `437`
- 目标目录: `Algorithms/物理-核物理-0437-核壳模型_(Nuclear_Shell_Model)`

## R01

本条目实现“核壳模型”的最小可运行 MVP：
- 用单粒子轨道量子数 `(N,l,j)` 构造简化平均场能量模型；
- 用三条数值路径（`scikit-learn` / `SciPy` / `PyTorch`）拟合同一组参数；
- 依据轨道能级排序后的大能隙自动提取壳闭合候选，并与经验幻数比较。

目标是提供可复现、可解释的算法流程，而不是高精度核数据库替代品。

## R02

核壳模型的核心思想是：核子在平均势场中近似独立运动，能级呈壳层结构；当某些壳层填满时，出现异常稳定的“幻数”核。

简化历史脉络：
- 1949 年前后，Mayer 与 Jensen 等工作指出强自旋-轨道耦合是解释 `28, 50, 82, 126` 的关键；
- 因此，教学型 MVP 的最小机制应包含：主壳量子数项 + 轨道角动量项 + `l·s` 自旋轨道项。

## R03

本实现采用线性能量参数化：

\[
E = b_0 + b_N N + b_{L2} l(l+1) + b_{LS}\langle l\cdot s \rangle
\]

其中（取 `s=1/2`）：

\[
\langle l\cdot s\rangle = \frac{1}{2}\left[j(j+1)-l(l+1)-\frac{3}{4}\right]
\]

并用轨道简并度 `2j+1` 作为该轨道可容纳核子数，按能量从低到高累计填充，利用相邻能级差
`gap_i = E_{i+1}-E_i` 的大值识别壳闭合边界。

## R04

`demo.py` 内置一份紧凑轨道表（到 `1j15/2`），每条包含：
- `orbital` 名称（如 `1f7/2`）
- `N,l,j`
- 容量 `2j+1`
- 参考能量 `E_ref_MeV`（用于拟合）

脚本输出四类结果：
- 拟合参数与误差汇总表（3 条算法路径）
- 轨道参考能量与各模型预测能量对照表
- 最优模型下的能级排序表（含累计核子数与下一个能隙）
- 幻数比较表（预测、经验、匹配集合与 `precision/recall/F1`）

## R05

变量与物理量说明：
- `N`：主壳量子数（无量纲）
- `l`：轨道角动量量子数（无量纲）
- `j`：总角动量量子数（无量纲）
- `2j+1`：轨道简并度/可容纳核子数（个）
- `E_ref_MeV` / `E_pred_MeV`：轨道能量（`MeV`）
- `b0, bN, bL2, bLS`：模型参数
- `gap_MeV`：相邻轨道能隙（`MeV`）
- `magic_candidate`：由大能隙边界诱导的壳闭合候选核子数

## R06

特征工程与壳填充规则：
- 特征向量固定为 `[1, N, l(l+1), <l·s>]`，不做额外黑箱嵌入。
- 轨道容量由 `2j+1` 明确给出，累计和定义填充后核子数。
- 闭壳检测不是“直接硬编码幻数”，而是从最优拟合能谱中选择前 `k` 个最大 `gap` 的边界。

这一设计把“壳模型假设 -> 能谱排序 -> 幻数候选”连成一条可审计链路。

## R07

第一条拟合路径：`fit_sklearn_linear`
- 使用 `LinearRegression(fit_intercept=False)` 直接拟合上式参数；
- 输出 `b0, bN, bL2, bLS` 与 `RMSE/R^2/Spearman`；
- 作为解析最小二乘基线，便于与其他优化路径做一致性检查。

## R08

为避免单一路径依赖，脚本额外包含两条独立实现：
- `fit_scipy_least_squares`：`scipy.optimize.least_squares` 最小化残差 `Xθ-y`；
- `fit_torch_adam`：`torch` + `Adam` 的显式梯度下降（含微小 `L2` 正则稳定项）。

三条路径拟合同一线性物理模型，若参数和误差接近，说明实现层一致性良好。

## R09

复杂度分析（`M` 为轨道数，`K` 为 torch 迭代步数）：
- 构造特征与预测：`O(M)`
- `sklearn` / `scipy` 线性规模残差评估：主流程近似 `O(M)`
- `torch` 路径：`O(KM)`
- 壳闭合提取（排序）：`O(M log M)`

当前 `M=23`，`K=4000`，开发机上运行通常为秒级。

## R10

数值稳定性处理：
- `fit_scipy_least_squares` 采用 `numpy.linalg.lstsq` 结果作为初值，降低坏初值风险；
- `fit_torch_adam` 使用极小 `L2` 正则项，抑制参数漂移；
- `R^2` 在 `SST=0` 的退化场景有保护分支；
- Spearman 相关系数若遇 `NaN`，回退到安全值避免报表中断。

## R11

正确性检查思路：
- 参数一致性：三条拟合路径的 `b0,bN,bL2,bLS` 应接近；
- 谱序一致性：能量排序应体现自旋-轨道导致的关键分裂；
- 结构一致性：由大能隙提取的闭壳候选应覆盖主要经验幻数集合。

该 MVP 在内置样本上可恢复主幻数序列 `2,8,20,28,50,82,126`。

## R14

物理假设与边界：
- 使用教学型、参数化平均场能量，不是 Woods-Saxon 全微观求解；
- 轨道参考能量为演示用紧凑样本，非 ENSDF/NuDat 全量评估值；
- 不含配对、形变、残余相互作用与同位旋细节。

因此结果定位为“机制可解释演示”，不应直接用于精密核结构预测。

非黑箱说明：
- 代码显式实现了 `N`, `l(l+1)`, `<l·s>` 物理特征，不是端到端神经网络替代；

## R18

`demo.py` 源码级流程（8 步）：

1. `build_orbitals()` 构造轨道样本，给出 `(N,l,j,2j+1,E_ref)`。
2. `build_feature_matrix()` 把每个轨道转换为 `[1,N,l(l+1),<l·s>]` 及目标能量。
3. `fit_sklearn_linear()` 用线性回归得到第一组参数与预测能谱。
4. `fit_scipy_least_squares()` 以残差最小化得到第二组参数，形成独立数值校验。
5. `fit_torch_adam()` 用梯度下降再求一次同一模型，完成第三条验证路径。
6. 汇总三组参数与误差，按最小 `RMSE` 选定当前样本的最优模型。
7. `extract_shell_closures()` 对最优能谱排序、计算相邻 `gap`、取前 `k` 大能隙并映射为闭壳候选核子数。
8. `compare_magic()` 将预测闭壳集合与经验幻数比较，输出匹配集合及 `precision/recall/F1`。
