## R01

降阶法（Reduction of Order）用于二阶线性齐次常微分方程：
`y'' + p(x)y' + q(x)y = 0`。  
当已知一个非零解 `y1(x)` 时，可把原问题降到一阶积分问题，构造第二个线性无关解 `y2(x)`。

本条目给出可运行 MVP：
- 采用数值积分实现降阶公式，而非符号计算；
- 使用固定测试方程 `y'' - y = 0`（`p=0, q=-1`）与已知解 `y1=e^x`；
- 验证 `y2` 与 `sinh(x)` 的误差、ODE 残差和 Wronskian 非退化性。

## R02

问题定义（本目录实现）：
- 输入：
  - 一维严格递增网格 `x_grid`；
  - 系数函数 `p(x)`、`q(x)`；
  - 已知非零解 `y1(x)`。
- 输出：
  - 通过降阶法构造的 `y2(x)`；
  - 中间量 `∫p`、积分核、累积积分；
  - 验证指标：`y1/y2` 的 ODE 残差、Wronskian、与参考解误差。

`demo.py` 为固定样例，不需要命令行参数和交互输入。

## R03

数学基础：

1. 原方程：`y'' + p(x)y' + q(x)y = 0`。  
2. 令 `y = v(x) y1(x)`，其中 `y1` 已知且满足原方程。  
3. 代入并化简后可得到关于 `w=v'` 的一阶方程：
   `w' + (2y1'/y1 + p) w = 0`。  
4. 解得：
   `w = C * exp(-∫(2y1'/y1 + p)dx) = C * exp(-∫p dx) / y1^2`。  
5. 再积分一次得第二解（忽略常数倍）：
   `y2(x) = y1(x) * ∫(exp(-∫p)/y1^2) dx`。

## R04

算法流程（高层）：
1. 检查 `x_grid` 单调性、长度和有限性。  
2. 在网格上评估 `p(x), q(x), y1(x)` 并做形状与数值检查。  
3. 计算 `integral_p(x) = ∫ p(s) ds`（累积梯形积分）。  
4. 构造积分核 `kernel(x) = exp(-integral_p) / y1(x)^2`。  
5. 计算 `J(x) = ∫ kernel(s) ds`。  
6. 得到 `y2(x) = y1(x) * J(x)`。  
7. 用数值微分计算 `y1/y2` 的一、二阶导，并代回 ODE 得残差。  
8. 计算 Wronskian 和误差指标，给出通过/失败判断。

## R05

核心数据结构：
- `ReductionResult`（`dataclass`）：
  - `x, p, q`：网格与方程系数；
  - `y1, y2`：已知解与降阶构造解；
  - `integral_p, integrand, integral_integrand`：降阶中间量；
  - `wronskian`：`W(y1,y2)`；
  - `residual_y1, residual_y2`：代回原方程后的离散残差。
- `pandas.DataFrame`：抽样输出关键点，便于肉眼检查。

## R06

正确性要点：
- 理论正确性：`y2 = y1 * ∫(exp(-∫p)/y1^2)dx` 是经典降阶公式。  
- 数值正确性：`y2` 通过累积积分直接构造，不依赖黑盒 ODE 求解器。  
- 线性无关验证：Wronskian 在内部网格上保持远离 0。  
- 方程一致性：`y1` 与 `y2` 代回 `y'' + py' + qy` 后残差应足够小。

## R07

复杂度分析（网格点数 `n`）：
- 累积积分两次：`O(n)`；
- 数值求导与残差计算：`O(n)`；
- 总时间复杂度：`O(n)`；
- 额外空间复杂度：`O(n)`（保存中间数组和结果）。

## R08

边界与异常处理：
- `x_grid` 非一维、非严格递增、存在 `nan/inf`：抛 `ValueError`。  
- `p/q/y1` 与网格形状不匹配：抛 `ValueError`。  
- `y1` 在网格上过小（接近 0）导致分母不稳定：抛 `ValueError`。  
- `y1_floor <= 0`：抛 `ValueError`。  
- 指标未达阈值：抛 `RuntimeError`，防止“静默失败”。

## R09

MVP 取舍说明：
- 只实现二阶线性齐次 ODE 的“已知一个解”场景。  
- 使用 `numpy + scipy` 完成积分与微分，不引入大型框架。  
- 使用 `pandas` 仅做结果表格展示，增强可读性。  
- 不实现符号推导、自适应网格、多方程批处理，保持最小而诚实。

## R15

方法对比：
- 对比“直接数值解 ODE（如 RK45）”：
  - 直接求解更通用；
  - 降阶法在“已知一个解”时结构更清晰、可解释性更强。  
- 对比“待定系数法/特征方程法”：
  - 后者依赖特定方程形式；
  - 降阶法适用于一般线性齐次二阶 ODE（前提是已知 `y1`）。

## R18

`demo.py` 源码级算法流程（8 步）：
1. `main` 构造固定网格与方程 `y'' - y = 0`，并给出已知解 `y1=e^x`。  
2. 调用 `reduction_of_order_second_solution`，先做网格、形状、有限性和 `y1` 非零检查。  
3. 使用 `cumulative_trapezoid` 对 `p(x)` 做一次前缀积分，得到 `integral_p`。  
4. 逐点构造积分核 `integrand = exp(-integral_p)/(y1^2)`。  
5. 对积分核再做一次前缀积分得到 `integral_integrand`，并计算 `y2 = y1 * integral_integrand`。  
6. 用 `_first_second_derivative` 对 `y1/y2` 计算数值一、二阶导，组装 ODE 残差 `y''+py'+qy`。  
7. 计算 Wronskian `W = y1*y2' - y1'*y2`，并与参考 `sinh(x)` 计算相对 L2 误差。  
8. `main` 打印抽样表格与汇总指标，依据阈值输出 `PASS`；未通过则抛异常中止。
