## R01

质能方程是狭义相对论的核心结论之一：质量与能量是同一物理量的两种表述。
最常见形式是静质量对应的静能：

- `E0 = m c^2`

其中 `m` 为静质量，`c` 为真空光速。它给出一个直接可计算的数量级结论：即使很小的质量也对应极大的能量。

## R02

本条目 MVP 的最小目标：

- 实现 `E0 = m c^2`（质量转静能）；
- 实现反变换 `m = E0 / c^2`（能量转等效质量）；
- 在给定速度 `beta=v/c` 下补充 `E_total = gamma m c^2` 与 `K = (gamma-1)mc^2`，展示“静能 vs 总能 vs 动能”关系；
- 通过非交互断言验证公式一致性、可逆性与基本物理约束。

## R03

使用 SI 单位制，采用精确定义常数：

- `c = 299792458 m/s`
- `gamma(beta) = 1/sqrt(1-beta^2), |beta|<1`
- `E0(m) = m c^2`
- `m(E0) = E0/c^2`
- `E_total(m,beta) = gamma(beta) m c^2`
- `K(m,beta) = E_total - E0 = (gamma-1)mc^2`

脚本还提供单位换算，便于读数：

- 焦耳转千瓦时：`kWh = J / 3.6e6`
- 焦耳转 TNT 当量（吨）：`ton_TNT = J / 4.184e9`

## R04

`demo.py` 的计算流程：

1. 定义物理常数、样例质量数组与速度参数 `beta`；
2. 校验输入质量是否有限且非负；

7. 执行可逆性与物理一致性断言；
8. 输出结果表格与关键标量。

## R10

正确性直觉：

1. `c^2` 是固定比例因子，因此 `E0` 与 `m` 必须严格线性；
2. 反变换 `m(E0)=E0/c^2` 应当恢复原质量；
3. 当 `beta=0` 时 `gamma=1`，总能退化为静能，动能应为 0；
4. 当 `0<|beta|<1` 时 `gamma>1`，总能应大于静能，动能为正。

脚本中的断言正是这四条物理约束的程序化检查。

## R11

数值与实现策略：

- 全部采用 `float64`，对本问题数量级足够；
- 对 `beta` 做 `|beta|<1` 显式检查，避免 `sqrt(1-beta^2)` 非法；
- 对质量与能量做有限性检查（非 `NaN`、非 `inf`）；
- 通过 `np.allclose` 验证可逆关系时使用紧容差，兼顾浮点误差与稳定性。

## R12

关键参数：

- `masses_kg`：示例质量数组，覆盖微小质量到人体量级；
- `beta`：速度与光速比，用于展示相对论总能；
- `C_LIGHT`：光速常量；
- `JOULE_PER_KWH`、`JOULE_PER_TNT_TON`：输出可读化换算系数。

调参建议：

- 增加更小质量（如 `1e-9 kg`）可观察线性关系在小量级仍成立；
- 提高 `beta`（如 `0.95`）可直观看到动能激增；
- 扩展样本数可用于批量换算，但复杂度仍线性。

## R13

- 近似比保证：N/A（非优化问题）。

可验证保证：

- 线性比例：`E0(2m) ≈ 2E0(m)`；
- 可逆性：`mass_from_energy(rest_energy(m)) ≈ m`；
- 边界一致：`beta=0` 时 `E_total=E0` 且 `K=0`；
- 正能性：`m>=0` 时 `E0>=0`，且 `0<|beta|<1` 时 `K>=0`。

## R14

常见失效模式：

1. `beta` 越界（`|beta|>=1`）导致 `gamma` 无定义；
2. 误用单位（把克当千克）导致能量差 1000 倍；
3. 输入中含 `NaN/inf` 导致结果污染；
4. 忽略静能与动能区别，把 `E=mc^2` 误当所有情形下唯一总能表达。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 定义样例质量数组与 `beta=0.8`，并声明 SI 常数（`c`、单位换算因子）。  
2. `validate_mass_array` 检查质量数组维度、有限性与非负性，防止非法输入进入计算链。  
3. `rest_energy_joule` 按 `E0=mc^2` 逐元素计算静能（`numpy` 向量化，不是黑箱物理库）。  
4. `lorentz_gamma` 计算 `gamma`，并在 `|beta|>=1` 时抛异常。  
5. `total_energy_joule` 与 `kinetic_energy_joule` 依公式 `gamma mc^2`、`(gamma-1)mc^2` 计算总能与动能。  
6. `mass_from_energy_joule` 执行逆变换 `m=E/c^2`，用于可逆性校验。  
7. `build_summary_table` 将焦耳值换算为 `kWh` 与 TNT 当量，组装 `pandas.DataFrame` 并输出。  
8. `main` 执行断言（可逆性、线性性、`beta=0` 边界、正能性），保证该 MVP 在非交互模式下可自动验证。  

第三方库使用说明：`numpy/pandas` 仅承担数组计算与表格展示；核心物理公式和流程均在源码中逐步展开并可追踪。
