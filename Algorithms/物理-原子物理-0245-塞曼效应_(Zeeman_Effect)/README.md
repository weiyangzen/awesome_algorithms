# 塞曼效应 (Zeeman Effect)

- UID: `PHYS-0242`
- 学科: `物理`
- 分类: `原子物理`
- 源序号: `245`
- 目标目录: `Algorithms/物理-原子物理-0245-塞曼效应_(Zeeman_Effect)`

## R01

塞曼效应描述的是原子能级在外磁场 `B` 中发生分裂，导致谱线从单线变为多分量线。一级近似下，磁子能量修正为：

`Delta E = mu_B * g_J * m_J * B`

其中 `mu_B` 是玻尔磁子，`g_J` 是 Landé 因子，`m_J` 是磁量子数。对应频率位移：

`Delta nu = Delta E / h = (mu_B / h) * g_J * m_J * B`

## R02

该效应是连接量子角动量与原子光谱实验的关键桥梁，主要价值：

- 验证角动量量子化与选择定则；
- 用可测的谱线位移反推 `g_J`、量子数结构与耦合模型；
- 区分正常塞曼效应（通常三重线）与反常塞曼效应（多于三条）；
- 为磁场诊断（实验室等离子体、天体磁场）提供标定手段。

## R03

本条目 MVP 输入输出约定（无交互）：

- 输入（脚本内固定）：
1. 两组谱线跃迁参数：
   - 正常例：`1P1 -> 1S0`，`lambda0=500 nm`；
   - 反常例：`2P3/2 -> 2S1/2`，`lambda0=589 nm`。
2. 外磁场：`B=0.8 T`（另含 `0, 0.4, 0.8 T` 对比校验）。
3. 选择定则：`Delta m_J in {0, ±1}`。

- 输出：
1. Landé 因子与分裂分量表（频移、波长、偏振类别）；
2. 线性随 `B` 比例校验；
3. `B=0` 退化校验；
4. 正常塞曼三峰合成光谱峰值计数；
5. `All checks passed.` 自动断言结果。

## R04

MVP 使用的核心物理与数值公式：

1. Landé 因子（LS 耦合）：
`g_J = 1 + [J(J+1)+S(S+1)-L(L+1)] / [2J(J+1)]`（`J=0` 时按零处理）。

2. 一阶频率位移：
`Delta nu = (mu_B / h) * g_J * m_J * B`。

3. 跃迁频率与波长：
`nu = nu0 + (Delta nu_upper - Delta nu_lower)`，`lambda = c / nu`。

4. 合成光谱：
`I(lambda) = sum_i w_i * exp(-(lambda-lambda_i)^2/(2 sigma^2))`。

## R05

令上、下能级简并度分别为 `Nu=2Ju+1`、`Nl=2Jl+1`：

- 枚举候选跃迁复杂度 `O(Nu * Nl)`；
- 选择定则过滤后仍是同阶规模；
- 分量合并与排序复杂度 `O(M log M)`（`M` 为有效分量数）；
- 合成光谱复杂度 `O(M * G)`（`G` 为波长网格点数）；
- 空间复杂度约 `O(M + G)`。

## R06

`demo.py` 的最小闭环：

- **闭环 A（物理参数）**：从 `L,S,J` 计算 `g_J`，并打印可审计值；
- **闭环 B（分裂谱线）**：基于 `m_J` 组合 + 选择定则输出塞曼分量；

## R10

正确性直觉：

1. 外磁场下每个 `m_J` 子能级线性平移，斜率由 `g_J` 决定；
2. 正常塞曼例中分量应等间距三重线；
3. 磁场强度翻倍，频率位移应近似翻倍；
4. `B=0` 时所有分量应退化回中心频率。

## R11

数值稳定与工程策略：

- 对 `L,S,J` 做半整数合法性与三角不等式校验；
- 用频率域 `nu0 + Delta nu` 再反算波长，避免直接在波长域线性近似带来的偏差；
- 简并分量合并时使用可控小数位舍入键，抑制浮点噪声；
- 峰值检测设置最小间距 `distance`，避免同一峰被重复计数。

## R12

关键参数：

- `B_t`：磁场强度，直接决定分裂尺度；
- `lambda0_nm`：无场中心波长；
- `rounding_digits`：分量合并时的频移舍入精度；
- `fwhm_pm`：合成谱线展宽，过大可能导致多峰并峰。

调参建议：

- 峰数检测失败时，先减小 `fwhm_pm` 再增加 `span_pm`；
- 若合并结果不稳，可提高 `rounding_digits` 或改用绝对阈值聚类；
- 做实验拟合时应将简化 `weight` 替换为基于矩阵元的强度权重。

## R14

常见失效模式：

1. 输入 `J` 非法（非半整数）导致 `m_J` 枚举错误；
2. 把 `Delta m` 定义方向写反，导致偏振标记错误；
3. 在波长域直接做线性叠加，忽略 `nu-lambda` 非线性关系；
4. 线宽过大使三重峰并为单峰，误判“没有分裂”。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `validate_level` 检查每个能级的 `L,S,J` 是否是整数/半整数并满足 `|L-S|<=J<=L+S`。  
2. `lande_g` 按 Landé 公式显式计算 `g_J`（`J=0` 特判为 0），不是黑盒调用。  
3. `mj_values` 生成离散 `m_J` 序列，`zeeman_shift_frequency` 计算单能级 `Delta nu=(mu_B/h)g_Jm_JB`。  
4. `zeeman_components` 双重遍历 `(m_u,m_l)`，按 `Delta m_J in {0,±1}` 过滤并构造每条跃迁分量的频移、波长、偏振标签。  
5. `collapse_degenerate_components` 通过频移舍入键做 groupby 聚合，得到实验上可分辨的简并合并分量。  
6. `synthesize_spectrum` 将离散分量与高斯线型卷积，生成连续的合成光谱 `I(lambda)`。  
7. `find_peaks` 在合成谱上检测峰数，用于正常塞曼三重线的自动验证。  
8. `main` 同时跑正常/反常示例，输出表格并执行断言（等间距、线性随 `B`、`B=0` 退化、峰数校验），最后打印 `All checks passed.`。  

说明：`scipy` 在本实现里只提供常数与峰检测工具；核心物理计算（`g_J`、位移、选择定则、分量合并）均在源码中逐步展开。
