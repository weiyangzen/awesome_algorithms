# μ子自旋旋转 (μSR)

- UID: `PHYS-0459`

## R01

μ子自旋旋转（muon spin rotation, μSR）是在材料中注入极化 μ 子后，
通过其衰变正电子角分布反推自旋极化函数 `P(t)` 的实验方法。
常见可观测量是非对称度 `A(t)`，在横场（TF-μSR）近似下可写成阻尼振荡：

`A(t) = A0 * exp(-lambda * t) * cos(2*pi*f*t + phi) + A_bg`

其中 `f` 与局域磁场 `B` 成正比（`f = (gamma_mu/2pi) * B`）。

## R02

- 1950-60 年代 μ 子束流技术成熟后，μSR 开始用于固体内部磁性探测。
- 1970 年代后，μSR 成为与 NMR、中子散射互补的微观磁探针。
- 在非常规超导、磁性序与量子材料研究中，μSR 常用于识别内部磁场分布与涨落时间尺度。

## R03

本条目将“μSR 技术”落成一个可运行的算法 MVP：

1. 生成一组带噪声的 μSR 时域信号（模拟实验数据）。
2. 从信号中估计初值（频率、弛豫率、幅值、相位、背景）。
3. 用有界非线性最小二乘拟合阻尼振荡模型。
4. 输出参数恢复误差、拟合质量以及推断的局域磁场 `B`。

目标是展示 μSR 数据分析的核心反演路径，而非做完整实验管线。

## R04

MVP 使用的核心关系：

1. 时域模型
`A(t) = A0 * exp(-lambda*t) * cos(2*pi*f*t + phi) + A_bg`

2. 局域磁场换算（`f` 以 MHz）
`B(T) = f / 135.5`

3. 残差最小化目标
`min_theta sum_i [A_model(t_i; theta) - A_obs(t_i)]^2`

参数向量 `theta = [A0, lambda, f, phi, A_bg]`。

## R05

脚本输出包括：

- 初值估计向量 `[A0, lambda, f_MHz, phi, A_bg]`。
- 真值 vs 拟合值对照表（含误差列）。
- 拟合诊断指标：`RMSE`、`R^2`。
- 由 `f` 反推的局域磁场 `B`（Tesla）。

## R06

`demo.py` 的主流程：

1. 构造时间轴 `t`（0 到 8 微秒）和“真值”参数。
2. 合成无噪声 μSR 曲线并叠加高斯噪声。
3. 用 FFT 峰值给出频率初值。
4. 用 Hilbert 包络 + 线性回归给出弛豫率初值。
5. 估计背景、幅值、相位初值。
6. 调用 `scipy.optimize.least_squares` 在有界参数空间中拟合。
7. 计算误差指标并输出结果表。

## R07

正确性依据：

- 物理一致性：输出信号形式与 TF-μSR 常用阻尼进动模型一致。
- 统计一致性：最小二乘在高斯噪声假设下对应极大似然估计。
- 可验证性：使用已知真值生成数据，可直接检验参数恢复误差。
- 可解释性：`f -> B` 的换算关系显式写在代码中，不依赖隐藏规则。

## R08

设采样点数为 `n`，迭代次数为 `k`。

- FFT 初值估计：`O(n log n)`。
- 包络与线性回归初值：`O(n)`。
- 每次最小二乘迭代需评估残差和数值 Jacobian，约 `O(n)`；总体 `O(k*n)`。
- 空间复杂度：`O(n)`（主要是时序数组和中间残差向量）。

## R09

数值稳定性处理：

- 对时间步长做递增检查，避免频域估计失效。
- Hilbert 包络取 `max(envelope, 1e-6)`，避免 `log(0)`。
- 拟合参数设置上下界，防止进入非物理解（如负频率、过大背景）。
- 相位初值由 `arccos` 计算前做 `clip([-1, 1])`，避免浮点越界。

## R13

输出解释：

- `Initial guess`: 初值器给出的参数起点，用于判断是否离真值太远。
- `Parameter recovery table`: 每个参数的真值、拟合值与偏差。
- `RMSE`: 时域点残差均方根，越小越好。
- `R^2`: 拟合解释度，接近 1 表示模型与数据一致性高。
- `Inferred local field B`: 由拟合频率直接换算的局域磁场。

## R14

自检建议：

1. 增大噪声 `noise_sigma`，应看到 `RMSE` 上升、参数误差增大。
2. 缩短时间窗（如改到 4 微秒），频率与弛豫率的区分会变差。
3. 把真值频率改高或改低，检查 FFT 初值是否跟随变化。
4. 把参数边界收得更紧，验证拟合是否出现“贴边界”行为。

## R15

模型局限：

- 仅覆盖单组分阻尼余弦，不含多组分信号与场分布卷积。
- 未显式处理计数统计（Poisson）和探测器几何校正。
- 未做不确定度估计（如协方差/Bootstrap/MCMC）。
- 不适用于零场 Kubo-Toyabe 或纵场动态模型的精细反演。

## R16

可扩展方向：

- 引入多组分模型（磁有序体积分数 + 顺磁背景）。
- 用高斯/洛伦兹场分布（或 Kubo-Toyabe）替代单指数阻尼。
- 增加参数置信区间评估（Fisher、Bootstrap、Bayesian）。

## R17

与凝聚态研究问题的关系：

- 通过 `lambda` 可追踪自旋涨落或无序程度变化。
- 通过 `f` 和 `B` 可识别内部磁场尺度，进而判断磁序形成。
- 在超导体中，横场 μSR 拟合出的场分布参数可关联穿透深度与超导能隙对称性。
- 本 MVP 是“最小反演骨架”，可作为更复杂 μSR 分析脚手架。

## R18

本实现的源码级算法流（8 步）：

1. `simulate_data` 用给定真值参数生成 `A(t)`，再叠加高斯噪声形成观测序列。
2. `estimate_initial_guess` 先用尾段均值估计 `A_bg`，得到去背景信号。
3. 对去背景信号做 `rfft`，在非零频段找谱峰，得到频率初值 `f0`。
4. 对同一信号做 Hilbert 变换取包络，回归 `log(envelope)` 对 `t` 的斜率，得到 `lambda0`。
5. 用分位数和首点关系估计 `A0`、`phi`，组装初始参数向量。
6. `fit_mu_sr` 显式定义残差 `r_i(theta)=A_model(t_i;theta)-A_obs_i`，并设置每个参数上下界。
7. `least_squares(method="trf", jac="2-point")` 迭代：
   - 用当前参数计算残差；
   - 用有限差分近似 Jacobian；
   - 在信赖域内解线性化子问题并更新参数，直到收敛。
8. 用最终参数计算 `RMSE`、`R^2` 与 `B=f/135.5`，并输出真值-拟合对照表。
