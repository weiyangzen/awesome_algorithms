# 激波理论 (Shock Wave Theory)

- UID: `PHYS-0088`
- 学科: `物理`
- 分类: `气体动力学`
- 源序号: `88`
- 目标目录: `Algorithms/物理-气体动力学-0088-激波理论_(Shock_Wave_Theory)`

## R01

激波理论研究可压缩流体中跨越极薄间断层的状态突变。对理想气体，激波可由守恒律和热力学第二定律共同约束：质量、动量、能量在波前后守恒，而熵必须增加。该条目的 MVP 聚焦最基础且可验证的场景: 一维定常法向激波（normal shock）。

## R02

控制方程采用一维欧拉方程守恒形式（无粘、无热传导）：

- 质量守恒: `d(rho*u)/dx = 0`
- 动量守恒: `d(rho*u^2 + p)/dx = 0`
- 能量守恒: `d(h + u^2/2)/dx = 0`（定常、绝热）

其中 `rho` 为密度，`u` 为速度，`p` 为静压，`h` 为比焓。

## R03

法向激波的 Rankine-Hugoniot 跳跃关系可写成上游马赫数 `M1` 的闭式函数（`gamma` 为比热比，且 `M1>1`）：

- `p2/p1 = 1 + 2*gamma/(gamma+1) * (M1^2 - 1)`
- `rho2/rho1 = ((gamma+1)*M1^2) / ((gamma-1)*M1^2 + 2)`
- `T2/T1 = (p2/p1) / (rho2/rho1)`
- `M2^2 = (1 + 0.5*(gamma-1)*M1^2) / (gamma*M1^2 - 0.5*(gamma-1))`

## R04

热力学可逆损失通过总压与熵增刻画：

- `p0/p = (1 + 0.5*(gamma-1)*M^2)^(gamma/(gamma-1))`
- `p02/p01 = (p2/p1) * (p0/p at M2) / (p0/p at M1)`
- `Delta s / R = ln(p2/p1) - gamma*ln(rho2/rho1)`

对真实压缩激波，预期 `p02/p01 < 1` 且 `Delta s / R > 0`。

## R05

本条目目标不是搭建完整 CFD 框架，而是提供“最小可运行且物理可核对”的算法原型：

- 输入一组 `M1`，批量计算法向激波前后状态比；
- 反问题演示：给定 `p2/p1` 反求 `M1`；
- 输出 CSV 并做自动一致性检查（亚音速下游、总压下降、熵增）。

## R06

`demo.py` 的输入/输出约定：

- 输入参数（代码内默认）: `gamma=1.4`，`M1` 网格为 `[1.05, 5.0]` 上 40 点；
- 主输出文件: `shock_table.csv`；
- CSV 列: `Mach1, Mach2, p2/p1, rho2/rho1, T2/T1, u2/u1, p02/p01, ds/R`。

## R07

反问题部分用 `scipy.optimize.root_scalar(brentq)`，求解
`f(M1)=p2/p1(M1)-target=0`。

这样做的意义：

- 展示“实验可测压强比 -> 上游马赫数”的工程流程；
- 在保留闭式正向公式的同时，演示数值求根作为通用反演手段；
- 避免把第三方库当黑箱，目标函数和区间都在源码中显式给出。

## R08

复杂度分析（`N` 为马赫采样数）：

- 正向批量计算: `O(N)`；
- 反问题求根: `O(K)`，`K` 为 Brent 迭代次数，通常很小；
- 总体空间复杂度: `O(N)`（DataFrame + 中间数组）。

## R09

鲁棒性与边界处理：

- 要求 `gamma > 1`；
- 要求 `M1 > 1`（法向激波物理解）；
- 非有限数输入会直接报错；
- 反问题要求 `p2/p1 > 1`，否则拒绝求解。

## R10

`demo.py` 的模块拆分：

- `_validate_inputs`: 输入物理域检查；
- `normal_shock_relations`: 闭式跳跃关系核心计算；
- `infer_mach_from_pressure_ratio`: 压强比反求 `M1`；
- `build_table`: 批量构建 `pandas.DataFrame`；
- `summarize`: 自动一致性诊断；
- `main`: 串联执行与结果落盘。

## R12

默认实验设置：

- 气体模型: 理想气体
- 比热比: `gamma=1.4`（空气常用近似）
- 上游马赫数范围: `1.05` 到 `5.0`
- 采样数: `40`

该范围覆盖弱激波到中强激波，足以观察各状态比的单调变化趋势。

## R13

输出解释：

- `Mach2`: 激波后马赫数，理论上应全部 `<1`；
- `p2/p1, rho2/rho1, T2/T1`: 静压、密度、温度突增比例；
- `u2/u1`: 速度比，激波后减速应 `<1`；
- `p02/p01`: 总压损失比例，应 `<1`；
- `ds/R`: 无量纲熵增，应 `>0`。

## R14

最小正确性检查：

1. `Mach2` 全部小于 1（超声速到亚声速跃迁）。
2. `p2/p1` 与 `rho2/rho1` 全部大于 1（压缩）。
3. `p02/p01` 全部小于 1（不可逆损失）。
4. `ds/R` 最小值大于 0（满足第二定律）。
5. 反问题恢复的 `Mach1` 与原样本近似一致。

## R15

典型趋势（默认参数下）应满足：

- `M1` 越大，`p2/p1` 越大；
- `M1` 越大，`Mach2` 趋向更低；
- `M1` 越大，熵增 `ds/R` 通常增大；
- `p02/p01` 逐步减小，表示激波损失增强。

若这些单调性被破坏，通常是公式抄写或变量定义出错。

## R16

常见实现错误：

- 把 `M` 与 `M^2` 混用，导致温度比或总压比公式错误；
- 忽略 `M1>1` 物理约束，把亚音速状态送入激波公式；
- 误把等熵关系当作激波跨越关系，得到零熵增；
- 反问题求根区间未包根，或目标函数符号定义反向。

## R17

可扩展方向：

- 增加斜激波（`theta-beta-M` 关系）并比较弱解/强解分支；
- 联立喷管面积变化，构造“等熵段 + 激波段”组合流动；
- 用实验压力点做参数反演（`gamma` 或上游状态）；
- 将法向激波关系嵌入 1D Riemann 求解器作为更完整 CFD 组件。

## R18

`demo.py` 源码级算法流程可拆为 8 步：

1. `main` 设定 `gamma` 与 `Mach1` 采样网格，调用 `build_table` 进行批量计算。
2. `build_table` 内部调用 `normal_shock_relations`，用 NumPy 向量化执行全部闭式跳跃公式。
3. `normal_shock_relations` 先经 `_validate_inputs` 校验 `gamma>1` 且 `Mach1>1`，防止非物理输入。
4. 同一函数计算 `p2/p1`、`rho2/rho1`、`T2/T1`、`Mach2`、`u2/u1`，并基于总压定义得到 `p02/p01`。
5. 通过 `Delta s / R = ln(p2/p1) - gamma ln(rho2/rho1)` 计算熵增，形成完整状态跳跃描述。
6. `main` 从表中抽取一个样本压强比，调用 `infer_mach_from_pressure_ratio`。
7. `infer_mach_from_pressure_ratio` 用 `root_scalar(brentq)` 在 `[1.000001, 20]` 上求解 `p2/p1(M1)-target=0`，得到反演 `Mach1`。
8. `main` 将结果保存为 `shock_table.csv`，再由 `summarize` 输出一致性诊断（下游亚音速、总压下降、熵增为正）。
