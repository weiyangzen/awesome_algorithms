# Chebyshev谱方法

## R01

Chebyshev谱方法是把未知函数表示到 Chebyshev 多项式基上的高阶数值方法。常见离散化方式是 Chebyshev-Gauss-Lobatto 配点（collocation）：

- 节点 `x_j = cos(pi*j/N), j=0..N`
- 在节点上用微分矩阵近似导数
- 将微分方程转成代数方程组求解

本目录 MVP 选择的是一维边值问题：

`u''(x) = f(x), x in [-1, 1]`，并配 Dirichlet 边界。

## R02

历史与定位：

- 谱方法在 20 世纪中后期成为高精度 PDE/ODE 离散的重要路线。
- Chebyshev 基因其端点加密节点、快速变换结构和稳定插值性质，在非周期区间问题上很常用。
- 与 Fourier 谱方法相比，Chebyshev 更适合有限区间与非周期边界条件。
- 在“解足够光滑、且要求高精度”的场景，Chebyshev 谱方法通常比低阶差分法更高效。

## R03

它要解决的核心矛盾是：

- 低阶方法（如二阶差分）提高精度往往只能靠大幅加密网格。
- 谱方法希望在“未知函数光滑”时，用较少自由度达到高精度。

工程直觉：

- 与其在等距网格上慢慢提阶，不如直接用全局基函数捕捉整体形状。
- Chebyshev 节点在端点更密，能缓解高次插值的端点振荡问题。

## R04

在本 MVP（稠密矩阵实现）中的复杂度：

- 构造一阶微分矩阵 `D`：`O(N^2)`。
- 形成二阶矩阵 `D2 = D @ D`：`O(N^3)`。
- 求解内部线性系统（大小约 `N-1`）：`O(N^3)`。

所以单次求解总体主导成本是 `O(N^3)`。如果固定 `D2` 并重复求多组右端，矩阵分解可复用。

## R05

空间复杂度：

- 存储 `D`、`D2` 等稠密矩阵需要 `O(N^2)` 内存。
- 状态向量与右端向量是 `O(N)`。

本实现是教学型“直接矩阵法”，优先代码透明度；大规模问题通常会用更节省内存的算子或快速变换实现。

## R06

微型例子（`N=4`）：

- 节点为 `x = [1, cos(pi/4), 0, -cos(pi/4), -1]`。
- 已知边界 `u(1)` 和 `u(-1)` 后，只需解 3 个内部点未知量。
- 通过 `D2` 的内部子块 `A = D2[1:4, 1:4]` 与修正右端 `rhs`，得到
  `A * u_interior = rhs`。

这就是“配点法把微分方程离散成线性代数方程”的核心过程。

## R07

方法重要性：

- 是谱方法体系中的基础成员，广泛用于流体、波动、量子和科学计算教学。
- 在平滑解场景常表现出“误差随 N 超快衰减（谱收敛）”。
- 为理解高阶离散、条件数、边界处理与全局近似提供了标准范式。

## R08

理论基础：

- Chebyshev 多项式：`T_k(x) = cos(k arccos x)`。
- 在 CGL 节点上构造插值多项式，导数可由微分矩阵作用得到：
  `u'(x_j) ~ (D u)_j`, `u''(x_j) ~ (D2 u)_j`。
- 把微分方程在每个内部配点强制满足，即得到代数系统。
- 对解析函数，谱方法通常可达到指数型误差衰减（比代数阶 `h^p` 更快）。

## R09

适用条件与限制：

- 适用于定义在有限区间、且解较光滑的问题。
- 边界条件需要明确（本例是 Dirichlet）。
- 对非光滑解、间断系数或尖峰行为，谱收敛会退化。
- 稠密矩阵版本在大 `N` 时会遇到计算和内存压力。

## R10

正确性框架（本实现）：

1. 用 CGL 节点与 Chebyshev 微分矩阵近似导数。
2. 将 `u''=f` 离散成 `D2 u ~ f`。
3. 把边界点移到右端，保留内部未知量，得到线性系统 `A u_i = rhs`。
4. 解线性系统获得离散解，并在节点上与解析解比较。
5. 通过网格加密观测误差快速下降，验证谱收敛行为。

## R11

误差与稳定性要点：

- 截断误差：对解析解可非常快下降（通常呈指数趋势）。
- 舍入误差：`N` 很大时，稠密矩阵运算与条件数会放大浮点误差。
- 实践上常见现象是：误差先快速下降，再在机器精度附近进入平台。
- 本 demo 选择中等 `N`（8 到 16）展示“下降阶段”，避免纯舍入主导区。

## R12

性能视角：

- 与二阶差分相比，单次大 `N` 成本更高（差分常是稀疏结构）。
- 但 Chebyshev 谱方法在“高精度目标”下，往往用更小 `N` 即可达标。
- 本 MVP 的成本瓶颈在 `D2` 构造与线性求解（均为 `O(N^3)`）。
- 这与 `demo.py` 中的实现路径一致：先建 `D`/`D2`，再一次线性求解。

## R13

可验证保证（`demo.py` 内置断言）：

- 多项式导数自检：
  - 取 `p(x)=x^4-2x^3+x`，在 `N=12` 下检查 `D p` 与解析导数一致。
  - 断言无穷范数误差 `< 1e-10`。
- BVP 收敛自检：
  - 用 `N=[8,10,12,14,16]` 计算误差。
  - 断言误差严格下降。
  - 断言末端误差 `< 1e-10`。
  - 断言首末误差比 `> 1e4`，确保存在显著收敛。

## R14

鲁棒性与常见失效：

- `n < 2` 会直接报错（节点与矩阵无法定义）。
- 若构造出的 `D` 或 `D2` 出现非有限值，代码会抛出 `RuntimeError`。
- 极大 `N` 下，稠密矩阵可能带来病态与内存压力，不适合直接扩展。
- 非光滑问题会破坏谱收敛预期，误差可能不再快速下降。

## R16

相关方法链路：

- 同属谱方法：Legendre 谱方法、Fourier 谱方法。
- 同类离散思想：伪谱法（pseudospectral）、谱元法（spectral element）。
- 对照数值路线：有限差分法、有限元法。
- 在本题语境下，Chebyshev 谱方法的关键优势是对光滑解的高精度效率；这与 R08 的理论基础和 R12 的性能讨论一致。

## R18

源码级算法流程拆解（`solve_poisson_dirichlet_chebyshev`，8 步）：

1. 调用 `chebyshev_diff_matrix(n)`，得到 CGL 节点 `x` 和一阶微分矩阵 `D`。  
2. 计算 `D2 = D @ D`，作为二阶导离散算子。  
3. 在节点上计算解析解 `u_exact(x)` 与右端 `f(x)`。  
4. 提取内部子矩阵 `A = D2[1:n, 1:n]`，只保留内部未知量。  
5. 根据已知边界值 `u(1), u(-1)` 修正右端：`rhs = f_i - D2_ib * u_b`。  
6. 调用 `np.linalg.solve(A, rhs)` 解内部未知量 `u_interior`。  
7. 将边界值与内部解拼回完整 `u_num`。  
8. 计算 `||u_num - u_exact||_inf` 并返回 `SolveResult`，供收敛验证使用。  

本 MVP 没有调用黑箱边值求解器（如 `scipy.integrate.solve_bvp`），核心离散和求解流程均在源码中显式展开。
