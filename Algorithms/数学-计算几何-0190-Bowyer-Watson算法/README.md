## R01

Bowyer-Watson 算法是构造 Delaunay 三角剖分的经典增量法：
每次插入一个新点，删除其外接圆包含该点的“坏三角形”，再用空腔边界与新点重建局部三角网。

本目录给出可运行 MVP：
- `demo.py` 手写 Bowyer-Watson 主流程（不调用黑盒 Delaunay API）；
- 自动处理重复点、共线退化等基础情况；
- 输出三组固定样例的三角形数量和 Delaunay 空圆性质检查结果。

## R02

问题定义（本实现）：
- 输入：二维点集 `P = {(x_i, y_i)}`，形状为 `n x 2`。
- 输出：三角形索引集合 `T = {(i, j, k)}`，满足：
  - 每个三角形顶点来自输入去重点后的点集；
  - 三角形按逆时针（CCW）存储；
  - 尽量满足 Delaunay 空圆性质（没有其它点落入任一三角形外接圆内部）。

脚本内置数据，不需要命令行参数和交互输入。

## R03

数学基础：

1. Delaunay 空圆准则：对任意三角形，其外接圆内部不包含其它样本点。  
2. 点在外接圆内测试（`incircle`）：对点 `p` 与三角形顶点 `a,b,c`，可用行列式符号判断。  
3. Bowyer-Watson 增量思想：
   - 用超三角形覆盖全部输入点；
   - 逐点插入，维护当前合法三角剖分。  
4. 局部重建正确性直觉：
   - 删除“坏三角形”后形成多边形空腔；
   - 空腔边界与新点连边，得到新的局部三角化。

## R04

算法流程（高层）：
1. 检查输入点数组维度与有限性。  
2. 以容差 `eps` 去重，减少退化情形。  
3. 若点数 `< 3` 或全部共线，直接返回空三角形集。  
4. 构造覆盖全部点的超三角形并初始化三角网。  
5. 依次插入每个点，找出其外接圆包含该点的坏三角形集合。  
6. 删除坏三角形，统计其边出现次数，提取只出现一次的空腔边界。  
7. 对每条边界边与新点建三角形（保证 CCW），加入三角网。  
8. 所有点插入完成后，删除包含超三角形顶点的三角形，得到最终结果。

## R05

核心数据结构：
- `Point = (x, y)`：二维点。  
- `Triangle = (i, j, k)`：三角形顶点索引三元组。  
- `Edge = (u, v)`：边索引（规范化为 `u < v` 便于计数）。  
- `TriangulationResult`：
  - `unique_points: np.ndarray` 去重后的点；
  - `triangles: list[Triangle]` 最终三角形列表；
  - `removed_duplicates: int` 去除的重复点数量。

## R06

正确性要点：
- 输入合法性：`check_points` 保证 `n x 2`、数值有限。  
- 方向一致性：`make_ccw_triangle` 统一三角形方向为 CCW。  
- 空腔提取：坏三角形边计数后只保留出现 1 次的边，即空腔边界。  
- 退化防护：面积近零三角形直接跳过，避免污染网格。  
- 结果校验：`verify_delaunay_empty_circumcircle` 对最终网格做空圆性质复核。

## R07

复杂度分析（`n` 个点）：
- 朴素实现下，每次插点都可能扫描全部现有三角形；
- 坏三角形检测最坏可近似 `O(n)` 级别，累计最坏 `O(n^2)` 到 `O(n^2 log n)`（取决于网格规模常数）；
- 空间复杂度主要为点集、三角集、临时边计数，约 `O(n)` 到 `O(n + |T|)`。

说明：本 MVP 优先清晰与可审计，未实现四叉树/邻接加速结构。

## R08

边界与异常处理：
- 维度错误（非 `n x 2`）或含 `nan/inf`：抛 `ValueError`。  
- 点数不足 3：返回空三角形集。  
- 全部共线：返回空三角形集。  
- 重复点：按 `eps` 量化键去重并记录删除数量。  
- 局部退化（三点近共线）：构三角时抛错并在主流程中跳过该候选。

## R09

MVP 取舍说明：
- 仅做二维平面三角剖分，不扩展到 3D 四面体化。  
- 使用 `numpy` 实现核心几何计算，避免重依赖。  
- 不调用 `scipy.spatial.Delaunay`，确保算法流程可见。  
- 采用固定样例打印摘要，不做图形渲染，以保证无交互、易验证。

## R10

`demo.py` 主要函数职责：
- `check_points`：输入点集形状与数值检查。  
- `deduplicate_points`：按容差去重。  
- `orientation` / `are_all_collinear`：方向与共线判断。  
- `build_super_triangle`：构造包围输入点的超三角形。  
- `circumcircle_contains`：点在三角形外接圆内测试。  
- `bowyer_watson`：主算法实现。  
- `verify_delaunay_empty_circumcircle`：结果复核。  
- `summarize_case` / `main`：组织样例并输出摘要。

## R14

可调参数：
- `eps`（`bowyer_watson` / `circumcircle_contains`）：
  - 控制去重容差；
  - 控制“点在外接圆内”与退化判断阈值。

调参建议：
- 若重复点未合并：适当增大 `eps`。  
- 若误判退化太多导致三角形偏少：适当减小 `eps`。  
- 若数据尺度很大或很小：先做坐标归一化再三角化。

## R15

方法对比：
- 对比分治 Delaunay：
  - 分治法理论和实现较复杂；
  - Bowyer-Watson 更直观，适合教学与原型。  
- 对比 Lawson 边翻转：
  - 边翻转依赖初始三角化质量与邻接维护；
  - Bowyer-Watson 直接以增量方式构建。  
- 对比现成库调用：
  - 库调用更快更稳；
  - 本实现可完整展示几何核心机制，便于审计与学习。

## R18

`demo.py` 源码级算法流程（9 步）：
1. `main` 固定随机种子，构造三组二维点集（随机、扰动网格、含重复点）。  
2. `summarize_case` 调用 `bowyer_watson` 执行三角剖分并汇总结果。  
3. `bowyer_watson` 先用 `check_points` 校验输入，再用 `deduplicate_points` 去重。  
4. 若点数不足或全共线（`are_all_collinear`），直接返回空结果。  
5. `build_super_triangle` 构造覆盖全部点的超三角形，初始化三角网。  
6. 对每个待插入点，遍历当前三角网，用 `circumcircle_contains` 找到坏三角形集合。  
7. 删除坏三角形并统计其边出现次数，出现 1 次的边构成空腔边界。  
8. 对每条边界边与新点调用 `make_ccw_triangle` 建立新三角形并加入网格。  
9. 插点结束后移除含超三角形顶点的三角形；`summarize_case` 再调用 `verify_delaunay_empty_circumcircle` 做空圆复核并打印摘要。
