# 狄拉克半金属 (Dirac Semimetal)

## R01

狄拉克半金属是三维动量空间中存在离散狄拉克点（四重简并、线性色散）的量子材料相。其低能激发可由三维 Dirac 方程近似描述，导带与价带在有限个 `k` 点相交，且在这些点附近满足 `E ~ v_F |delta k|`。

本条目使用一个最小 4x4 晶格模型演示 Dirac 点的数值定位与低能性质抽取。

## R02

该模型在凝聚态物理中的核心价值：

- 提供从“对称性保护的带交叉”到“可计算能带特征”的最小桥梁；
- 解释为何低能态密度在 Dirac 点附近被强抑制（理想 3D 情况下 `DOS(E) ~ E^2`）；
- 为后续拓扑相变（Dirac -> Weyl 或开隙绝缘体）提供可操作基线。

## R03

本目录 MVP 目标：

- 构造可显式对角化的 4x4 晶格 Dirac 哈密顿量；
- 在第一布里渊区做网格扫描并用连续优化精修 Dirac 节点；
- 对节点附近色散做线性回归，估计有效费米速度；
- 采样低能态密度并输出可复现表格。

实现坚持“小而诚实”：避免大型框架，只做可验证核心流程。

## R04

MVP 使用的哈密顿量：

`H(k) = v * [sin(kx) * Gamma1 + sin(ky) * Gamma2 + M(k) * Gamma3]`

其中

`M(k) = m0 - cos(kx) - cos(ky) - cos(kz)`。

能谱为（双重简并）：

`E(k) = +- v * sqrt(sin^2(kx) + sin^2(ky) + M(k)^2)`。

Dirac 点条件：`sin(kx)=0`、`sin(ky)=0` 且 `M(k)=0`，在 `1 < m0 < 3` 时得到一对 `kz = +-k0` 节点。

## R05

设布里渊区网格边长为 `N`，色散采样数为 `S`，DOS 采样数为 `D`：

- 网格扫描复杂度：`O(N^3)`；
- 节点精修（L-BFGS-B）复杂度：与迭代次数相关，实践中近似常数级；
- 色散拟合复杂度：`O(S)`；
- DOS 统计复杂度：`O(D)`。

默认参数下（`N=41, S=1500, D=100000`）可在普通 CPU 快速完成。

## R06

`demo.py` 运行后输出：

- 两个 Dirac 节点坐标与残余带隙；
- 每个节点的线性色散拟合结果（斜率、截距、`R^2`、RMSE）；
- 低能 DOS 采样表（前若干 bin）；
- 一组数值 sanity checks（节点近零能、镜像对称、线性拟合质量、NumPy/Torch 本征值一致性）。

并写出三个 CSV：`nodes_summary.csv`、`dispersion_fit.csv`、`dos_curve.csv`。

## R10

正确性直觉：

1. 当 `1 < m0 < 3`，质量项方程允许一对 `+-k0` 解；
2. 节点附近哈密顿量按 `delta k` 一阶展开，能量近似线性；
3. 3D 线性色散对应低能 `DOS ~ E^2`，在 `E->0` 处抑制；
4. NumPy 与 PyTorch 对同一矩阵本征值应一致（数值误差内）。

脚本中的断言逐条对应上述物理与数值检查。

## R11

数值稳定处理：

- 统一将动量折返到 `[-pi, pi]`，避免周期边界漂移；
- 节点先做离散粗扫再做连续优化，降低局部最小风险；
- 用随机种子固定 Monte Carlo 与拟合采样，保证复现；
- 用 `complex128` 执行矩阵对角化，减小舍入误差。

## R12

关键参数（见 `DiracConfig`）：

- `m0`：质量控制参数，决定是否有 Dirac 点；
- `velocity`：整体速度尺度，线性缩放能量；
- `grid_n`：布里渊区扫描分辨率；
- `fit_radius`/`fit_samples`：节点附近回归半径与样本数；
- `dos_samples`/`dos_bins`：DOS 采样精度；
- `seed`：随机过程复现控制。

建议：先固定 `m0=2.2` 跑通，再改参数观察节点消失或开隙。

## R14

常见失效模式：

1. `m0` 取值越出 `1 < m0 < 3`，Dirac 点消失并开隙；
2. 网格太粗导致初值偏差大，优化收敛慢；
3. `fit_radius` 过大使高阶非线性污染线性拟合；
4. DOS 样本过少造成低能 bin 抖动大；
5. 误把第三方库当黑盒，忽略模型方程与流程追踪。

本实现对 1 和 3 有显式参数约束或质量检查。

## R18

`demo.py` 的源码级算法流程（9 步）：

1. `main` 读取 `DiracConfig` 并检查 `m0` 是否处于半金属窗口。  
2. `locate_nodes` 在三维网格上计算 `gap_formula`，分别在 `kz>0` 与 `kz<0` 半区找最小 gap 粗初值。  
3. `refine_node` 以粗初值为起点，调用 SciPy `L-BFGS-B` 最小化 gap，得到连续动量下的 Dirac 节点。  
4. `build_node_dataframe` 汇总每个节点的 `(kx, ky, kz, gap)`，并计算理论 `k0` 与数值结果偏差。  
5. `fit_linear_dispersion` 在节点附近随机采样 `delta k`，计算能量后用 scikit-learn 线性回归，输出斜率/截距/`R^2`/RMSE。  
6. `estimate_dos_curve` 在整个 BZ 做蒙特卡洛采样，按能量分箱估计 DOS 曲线并组织为 pandas 表格。  
7. `torch_eigenvalue_consistency` 对同一哈密顿量分别用 NumPy 与 PyTorch 求本征值，报告最大差异。  
8. `run_sanity_checks` 执行节点近零能、镜像对称、拟合质量、NumPy/Torch 一致性等断言。  
9. `main` 打印关键表格并将 `nodes_summary.csv`、`dispersion_fit.csv`、`dos_curve.csv` 写入当前目录。

第三方库不是黑盒求解器：`numpy`/`torch`仅用于线性代数与数组运算，`scipy`仅用于通用最小化器，`sklearn`仅用于线性回归，模型方程、目标函数、采样策略、检查指标均在源码中显式实现。
