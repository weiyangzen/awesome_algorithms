## R01

双线性插值（Bilinear Interpolation）用于在二维规则网格上，从离散像素/网格值估计任意连续坐标点的数值。

它等价于“先沿 x 方向做两次线性插值，再沿 y 方向做一次线性插值”，是图像缩放、纹理采样、栅格重采样中的基础算子。

## R02

设二维网格 `G` 的形状为 `(H, W)`，索引方式为 `G[y, x]`。  
给定连续查询点 `p=(x, y)`（网格索引坐标系），先找到包围它的四邻域：

- `x0 = floor(x)`，`x1 = min(x0 + 1, W - 1)`
- `y0 = floor(y)`，`y1 = min(y0 + 1, H - 1)`

四个角点分别为：`(x0,y0)、(x1,y0)、(x0,y1)、(x1,y1)`。

## R03

定义局部小数偏移：

- `xd = x - x0`
- `yd = y - y0`

对应四个角点值：

- `f00 = G[y0, x0]`
- `f10 = G[y0, x1]`
- `f01 = G[y1, x0]`
- `f11 = G[y1, x1]`

## R04

双线性插值可分两层线性组合：

```text
fx0 = f00*(1-xd) + f10*xd
fx1 = f01*(1-xd) + f11*xd
f   = fx0*(1-yd) + fx1*yd
```

也可写成四角点加权和：

```text
f = f00*(1-xd)*(1-yd)
  + f10*xd*(1-yd)
  + f01*(1-xd)*yd
  + f11*xd*yd
```

四个权重非负且和为 1。

## R05

本目录 MVP 目标：

## R06

`demo.py` 核心函数：

- `_validate_grid` / `_validate_points`：输入维度与类型检查；
- `_handle_bounds`：统一处理越界点；
- `bilinear_interpolate`：向量化批量插值；
- `bilinear_interpolate_naive`：朴素循环实现；
- `build_linear_grid` / `analytic_linear_field`：构造可精确验证的线性测试场。

## R07

正确性不变量：

1. `grid.ndim == 2`；
2. `points.shape == (N, 2)` 且列顺序为 `(x, y)`；
3. 插值权重位于 `[0, 1]` 且总和为 `1`；
4. 在线性场上，插值结果应与解析表达式一致（仅有浮点误差）；
5. 向量化实现与朴素实现在相同输入下结果一致。

## R08

复杂度分析（`N` 为查询点数）：

- 向量化版本：时间 `O(N)`，空间 `O(N)`；
- 朴素循环版本：时间 `O(N)`，但 Python 循环常数较大；
- 单点插值仅涉及 4 个邻点与固定次算术运算。

工程中通常优先选向量化版本。

## R09

数值与工程注意事项：

- 接近上边界时 `x1/y1` 必须钳到合法最大索引；
- 坐标语义 `(x,y)` 与数组索引 `G[y,x]` 必须严格区分；
- 浮点比较应使用容差（`np.allclose`）；
- 插值不是外推，越界行为要显式定义；
- 批量查询时向量化可显著减少 Python 调度开销。

## R10

实验设置（`demo.py`）：

- 线性场：`f(x,y)=ax+by+c`；
- 查询点：`80,000` 个随机合法点（随机种子 `2026`）；

- 边界演示：构造越界点，分别测试 `clip` 与 `raise`。

## R13

脚本内置质量门禁：

1. 线性场误差必须低于阈值；
2. 向量化与朴素实现必须一致；
3. `mode='raise'` 遇到越界必须抛出异常；
4. 非法输入形状需给出明确 `ValueError`。

## R14

边界与异常策略：

- `mode='raise'`：任一点越界即报错，并给出部分坏点索引；
- `mode='clip'`：将越界坐标裁剪到合法区间后再插值；
- 非法 `mode` 触发 `ValueError`；
- 网格或点集维度错误在入口处立即失败。

## R18

源码级算法链路（8 步）：

1. `main()` 先调用 `build_linear_grid` 构造二维线性场 `G[y,x]`，并随机生成连续查询点 `queries`。  
2. `bilinear_interpolate` 入口调用 `_validate_grid` 与 `_validate_points`，确保输入为二维网格和 `(N,2)` 点集。  
3. `_handle_bounds` 根据 `mode` 处理越界：`raise` 直接报错，`clip` 将坐标钳到合法区间。  
4. 对每个查询点分解整数索引与小数偏移：`x0/x1, y0/y1` 与 `xd/yd`，并保证 `x1/y1` 不越界。  
5. 用 NumPy 高级索引一次性取出四角点数组 `f00/f10/f01/f11`，避免 Python 逐点访存。  
6. 先沿 x 做两次线性合并得到 `fx0/fx1`，再沿 y 合并得到最终插值结果 `f`。  
7. `main()` 用 `analytic_linear_field` 计算解析真值，统计 `max_abs_err` 与 `rmse` 并执行阈值断言。  
8. `main()` 运行 `bilinear_interpolate_naive` 进行一致性和耗时对比，最后打印样本表、边界表、指标表，形成可验证闭环。
