## R01

本条目实现静电学电势理论的最小可运行 MVP：

- 以点电荷系为输入，在二维网格上计算电势场 `V(x,y)`。
- 同时给出电场 `E=-∇V`，并与库仑定律直接求和结果对照。
- 通过可量化误差指标验证“叠加原理”和“势场-电场一致性”。

核心目标不是做复杂几何求解器，而是把电势理论的基本计算链路完整跑通：
`电荷分布 -> 电势 -> 电场 -> 可验证指标`。

## R02

背景与理论位置（简要）：

- 电势理论是经典静电学的基础框架，强调用标量势 `V` 描述电场。
- 在静电条件下，电场是保守场，可写为 `E=-∇V`。
- 对离散点电荷，电势可直接由库仑势叠加得到；对连续电荷则对应势积分与 Poisson/Laplace 方程（后续条目展开）。

因此“电势理论”是“点电荷公式”和“偏微分方程电势求解”之间的桥梁层。

## R03

本 MVP 的问题定义：

- 已知若干点电荷 `q_i` 及位置 `r_i=(x_i,y_i)`。
- 在矩形网格上求
  - 电势：`V(r)=sum_i k*q_i/|r-r_i|`
  - 电场：`E(r)=sum_i k*q_i*(r-r_i)/|r-r_i|^3`
- 同时由离散梯度计算 `E_grad=-∇V`，并比较 `E_grad` 与 `E_direct`。

输出包含：

- 电势范围（最小/最大值）
- 叠加原理相对误差
- 电场一致性相对误差
- 物理对称性检查（偶极子在原点的势）
- 中心线采样表

## R04

数学模型与符号：

- 库仑常数：`k = 1/(4*pi*epsilon_0*epsilon_r)`。
- 点电荷叠加电势：
  - `V(r)=sum_i k*q_i/|r-r_i|`
- 点电荷叠加电场：
  - `E(r)=sum_i k*q_i*(r-r_i)/|r-r_i|^3`
- 势场与电场关系：
  - `E=-∇V`

数值实现中，为避免网格点与电荷位置重合导致奇异，使用软化半径 `softening`：

- `|r-r_i| -> sqrt(|r-r_i|^2 + softening^2)`

该处理是数值稳定性手段，不改变算法主流程。

## R05

算法流程（MVP）：

1. 构建均匀网格 `xx, yy`。
2. 对所有电荷向量化广播，计算每个网格点到各电荷距离。
3. 对 `q_i/r_i` 求和得到 `V_total`。
4. 对同一电荷集逐个单独求势并累加，验证叠加原理。
5. 用库仑公式直接求 `E_direct`。
6. 用 `numpy.gradient` 计算 `E_grad=-∇V_total`。
7. 在远离奇异点的掩膜区域比较两种电场，得到相对误差。
8. 对理想偶极子测试 `V(0,0)=0` 对称性。

## R07

物理意义与工程价值：

- 电势是静电分析中的低维核心变量，先求 `V` 再求 `E` 常更稳定。
- 多电荷系统下，线性叠加提供了直接、可解释的计算路径。
- 本条目可作为 Poisson/Laplace 数值求解前的概念与实现基础。

## R09

适用范围与边界：

- 适用于静态电场（时间变化可忽略）。
- 当前实现是二维观测平面上的点电荷叠加演示，不含介质边界与导体边界条件求解。

## R10

正确性依据：

1. 物理层正确性：使用经典库仑势和线性叠加原理。
2. 数值层正确性：
   - 叠加恒等式检查：`V(all) ?= sum V(single)`。
   - 场势关系检查：`E_direct ?= -∇V`（掩膜区）。
   - 对称性检查：理想偶极子在原点电势应为 0。

三类检查共同构成“可计算 + 可验收”的闭环。

## R11

数值误差与稳定性：

- 误差来源：
  - 网格离散误差（`numpy.gradient` 二阶差分）。
  - 奇异点邻域的高梯度误差放大。
  - 浮点舍入误差。
- 稳定措施：
  - 使用 `softening` 避免除零与极端数值爆炸。
  - 用距离掩膜排除靠近电荷的最不稳定区域后再评估场一致性。

## R12

复杂度与真实开销：

设网格点数 `N = nx*ny`，电荷数 `M`：

- 电势计算：`O(MN)`。
- 直接电场计算：`O(MN)`。
- 梯度计算：`O(N)`。
- 空间复杂度：`O(N + MN)`（向量化中会临时持有 `dx/dy/r` 的 `M x nx x ny` 数组）。

本 MVP 默认 `M=3`、`N=25921`，CPU 单次运行在秒级以内。

## R14

鲁棒性与失效模式：

- `epsilon_r <= 0`：物理参数非法，直接报错。
- 网格过粗：梯度误差偏大，场一致性可能超阈值。
- `softening` 过小且采样点接近电荷：可能产生极端数值并恶化误差。
- 电荷规模过大：会放大数值动态范围，影响浮点稳定性。

应对策略：参数校验、掩膜评估、阈值断言和可读诊断输出。

## R18

源码级算法拆解（非黑盒，8 步）：

1. `run_mvp` 定义介电常数比 `epsilon_r`、软化半径和三组点电荷。  
2. `build_grid` 生成 `161 x 161` 均匀网格以及步长 `hx/hy`。  
3. `potential_from_charges` 将 `q/x/y` 组织成 `(M,1,1)`，与网格广播计算 `dx,dy,r`，再按 `k*sum(q/r)` 得到 `V_total`。  
4. 逐电荷重复第 3 步并累加为 `v_sum`，计算 `||V_total-v_sum||/||v_sum||` 作为叠加恒等式误差。  
5. `electric_field_direct` 用同样的广播张量计算 `E_direct = k*sum(q*dr/r^3)`。  
6. `electric_field_from_potential` 调用 `numpy.gradient` 以 `(hx,hy)` 做二阶有限差分，返回 `E_grad=-∇V`。  
7. `min_distance_to_charges` 生成“远离奇异点”掩膜，基于掩膜计算电场相对误差 `sqrt(sum||ΔE||^2)/sqrt(sum||E||^2)`。  
8. `potential_at_points` 对理想偶极子计算 `V(0,0)`；`quality_checks` 统一判定三项指标是否达标，`main` 最终打印报告。  

第三方库使用说明：

- `numpy` 负责数组广播、梯度和范数计算；
- `scipy.constants` 仅提供物理常数，不替代算法；
- `pandas` 仅做结果表展示，不参与核心数值逻辑。
