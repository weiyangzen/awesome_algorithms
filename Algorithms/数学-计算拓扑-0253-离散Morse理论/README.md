# 离散Morse理论

- UID: `MATH-0253`

- 分类: `计算拓扑`

## R01

离散 Morse 理论（Forman）把光滑 Morse 理论离散化到胞腔复形或单纯复形上。
核心目标是构造一个离散向量场（离散梯度），用更少的临界胞元近似原拓扑结构。
在计算拓扑里，它常被用于：
- 复形简化（减少单元数）
- 同调计算前预处理
- 拓扑特征提取与可视化

## R02

设 `K` 为有限单纯复形。
离散向量场 `V` 是若干对 `(alpha^p, beta^(p+1))` 的集合，满足：
- `alpha` 是 `beta` 的一个余维 1 面
- 每个胞元最多出现在一对中

未被配对的胞元称为临界胞元。
若 `V` 不含闭合 `V`-path（等价地，对应方向化 Hasse 图无有向环），则称 `V` 为无环离散向量场。

## R03

本目录 MVP 的输入是“极大单纯形列表”，例如：
- `[(0,1),(1,2),(0,2)]` 表示一个 1 维环
- `[(0,1,2)]` 表示一个填充三角形

输出包括：
- 各维胞元数量 `n_k`
- 离散向量场匹配对数量
- 各维临界胞元数量 `c_k`
- Betti 数 `beta_k`（在 `Z2` 系数下）
- `c_k >= beta_k` 的弱 Morse 不等式检查
- Euler 特征一致性检查

## R04

MVP 使用的数据结构：
- `Simplex = Tuple[int, ...]`：单纯形
- `cells_by_dim: Dict[int, List[Simplex]]`：按维分组胞元
- `Pair = (alpha, beta)`：离散向量场中的配对
- 有向邻接表 `adj`：用于检测无环性
- `numpy.uint8` 矩阵：`Z2` 下边界矩阵

## R05

算法主线：
1. 从极大单纯形生成闭包，得到完整复形。
2. 枚举所有 `(p, p+1)` 关联对作为候选匹配。
3. 按固定顺序做贪心尝试：若两端胞元尚未使用，则临时加入匹配。
4. 将 Hasse 图按 Forman 规则定向：
   - 未匹配边高维指向低维
   - 匹配边低维指向高维
5. 检测有向环；若无环则接受该匹配，否则回退。
6. 最终未匹配胞元即临界胞元。
7. 构造 `Z2` 边界矩阵并高斯消元，得到 Betti 数。
8. 验证 Morse 不等式与 Euler 一致性。

## R06

正确性要点（MVP 级别）：
- 每个胞元最多出现在一次匹配中，由 `used_cells` 保证。
- 每次新增匹配都显式做有向环检测，因此最终向量场无环。
- 临界胞元定义由“未配对”直接实现。
- Betti 数采用链复形公式 `beta_k = n_k - rank(B_k) - rank(B_{k+1})`。
- 对于离散 Morse 理论，弱不等式 `c_k >= beta_k` 与 Euler 恒等式是基础一致性检查。

## R07

设：
- `N` 为胞元总数
- `E` 为 Hasse 图关联边数（候选匹配数同阶）

复杂度（保守估计）：
- 闭包构造：与输入极大单纯形子面总数成正比。
- 贪心匹配：每次尝试需一次环检测，约 `O(E * (N + E))`。
- 同调计算：每个边界矩阵高斯消元约 `O(mn*min(m,n))`（GF(2) 版本）。

该实现面向教学与小规模验证，不是最优大规模实现。

## R08

实现细节约定：
- 系数域使用 `Z2`，避免符号定向处理，便于最小实现。
- 所有单纯形先排序并统一为 tuple，保证可哈希与可复现。
- 候选匹配顺序固定，输出可重复。
- 环检测使用 DFS 三色标记，逻辑直观。

## R09

MVP 边界：
- 仅支持有限单纯复形（非一般 CW 复形）。
- 仅做贪心匹配，不保证临界胞元全局最少。
- Betti 数只在 `Z2` 下计算，不输出扭结信息。
- 适合小中规模示例，不针对超大复形优化。

## R10

与连续 Morse 理论关系：
- 连续情形里，临界点索引对应拓扑变化。
- 离散情形中，临界胞元扮演离散化后的“临界点”角色。
- 临界胞元数量控制拓扑下界（Morse 不等式），并保留 Euler 信息。

## R11

`demo.py` 内置 3 个测试复形：
- `S^1` 的三角剖分（只有边界环）
- 一个填充 2-单形（可收缩）
- 四面体边界（`S^2`）

程序会逐个打印：`n_k`、`c_k`、`beta_k`、不等式检查、Euler 检查。

## R13

结果解读要点：
- 若 `weak Morse` 全为 `True`，表示 `c_k >= beta_k` 成立。
- `Euler(胞元计数)` 与 `Euler(临界计数)` 相等，说明离散向量场与临界统计在 Euler 层面自洽。
- `c_k` 通常不会等于最优值，因为这里用的是贪心匹配。

## R14

与常见“直接同调计算”流程比较：
- 直接法：直接建边界矩阵并约化。
- 离散 Morse 法：先做拓扑简化，再计算同调。
- 本 MVP 同时保留两者：先构造离散向量场，再用边界矩阵做验证。

## R15

典型应用场景：
- TDA（拓扑数据分析）预简化
- 网格/体素数据的拓扑骨架提取
- 教学中演示“临界单元与拓扑不变量”对应关系

## R16

可改进方向：
- 使用更强的匹配策略（如启发式优先高维消去）
- 替换为增量式环检测，降低重复建图成本
- 扩展到一般胞腔复形
- 加入持久同调前的 Morse 简化管线

## R17

最小测试清单：
- 环 `S^1`：`beta_0=1, beta_1=1`
- 填充三角形：`beta_0=1, beta_1=0, beta_2=0`
- 四面体边界 `S^2`：`beta_0=1, beta_1=0, beta_2=1`
- 每个案例都应满足：
  - `c_k >= beta_k`
  - Euler 两种计数相等

## R18

`demo.py` 的源码级算法流（非黑盒，8 步）：
1. `closure_of_maximal`：把输入极大单纯形展开为完整闭包胞元集。
2. `group_cells_by_dim`：按维度整理胞元，形成链群基。
3. `incidence_pairs`：枚举所有 `(alpha^p, beta^(p+1))` 关联边。
4. `greedy_acyclic_matching`：逐个尝试候选匹配，并维护胞元唯一使用约束。
5. `build_orientation`：按“匹配边上行、未匹配边下行”构造方向化 Hasse 图。
6. `has_cycle`：每次试探匹配后做 DFS 环检测，拒绝引入有向环的方案。
7. `critical_cells`：输出未参与匹配的胞元，得到 `c_k`。
8. `boundary_matrix_mod2 + rank_mod2 + betti_numbers`：构造 `Z2` 边界矩阵并做 GF(2) 消元，计算 `beta_k`，最后在 `analyze_complex` 中校验 Morse 与 Euler 条件。
