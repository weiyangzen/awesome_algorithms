## R01

热力学第三定律的常见表述包含三层：

1. Planck 表述：完美晶体在 `T -> 0 K` 时熵趋于 `0`；
2. Nernst 热定理：对完美晶体，任意等温过程的熵变 `delta S` 在 `T -> 0 K` 时趋于 `0`；
3. 不可达性表述：有限步有限时的物理过程无法把系统冷却到绝对零度。

它给出了低温热力学的边界条件：熵在零温附近不是任意的，而是被强烈约束。

## R02

工程与建模意义：

- 为低温性质表、熵基准和热容外推提供零点约束；
- 解释为何超低温实验只能“逼近 0 K”而不能“到达 0 K”；
- 给出区分“完美晶体 vs 无序体系（残余熵）”的判据；
- 在代码中可转化为三个可验证目标：`S(T)->0`、`|delta S|->0`、有限时温度保持正值。

本条目实现一个可运行 MVP，重点是可审计的低温熵计算流程，而不是复杂制冷机细节仿真。

## R03

`demo.py` 输入输出（无交互，参数写在脚本里）：

3. 有限时冷却轨迹采样表（`T(t)` 与 `S(T)`）；
4. 断言通过后打印 `All checks passed.`。

## R04

MVP 使用的模型与公式：

1. 低温热容近似：`C(T) = a*T^3 + b*T`；
2. 熵定义：`S(T) = S_res + integral_0^T C(T')/T' dT'`；
3. 对该模型可得解析式：`S(T) = S_res + (a/3)*T^3 + b*T`；
4. 完美晶体条件：`S_res = 0` 且 `b = 0`，因此 `S(T) ~ (a/3)T^3 -> 0`；
5. 两完美晶体熵差：`|S_A-S_B| = |a_A-a_B|/3 * T^3 -> 0`；
6. 不可达性演示：采用 `T(t) = T0*exp(-t/tau)`，任意有限 `t` 下均有 `T(t) > 0`。

说明：脚本没有调用黑盒热力学求解器，熵的解析推导与积分对照在源码中逐项展开。

## R05

复杂度（设材料数 `m`、温度点数 `n`、积分网格 `g`、冷却步数 `s`）：

- 熵表构建：`O(m*n*g)`（每个温度点做一次数值积分）；
- 熵差表：`O(n)`；

- 总时间复杂度：`O(m*n*g + s)`；

在教学级数据规模下（几十到几百行）开销很小。

## R06

`demo.py` 的最小闭环：

- 闭环 A（热容 -> 熵）：`MaterialModel.heat_capacity` + `entropy_analytic`；
- 闭环 B（积分校验）：`entropy_numerical` 与解析解逐点对照；
- 闭环 C（第三定律验证）：`S(T)->0` 与 `|delta S|->0` 的断言；
- 闭环 D（不可达性演示）：`simulate_exponential_cooling` 验证有限时 `T>0`。

## R09

适用场景：

- 课堂演示第三定律在低温模型中的可计算含义；
- 快速检查某个低温热容拟合是否满足 `T->0` 熵约束；
- 作为后续低温算法（Debye/Einstein/制冷循环）的最小测试底座。

不适用场景：

- 真实材料全温区高精度性质拟合；
- 工程级低温系统设计（需要设备级传热与控制模型）；
- 接近量子临界点或相变附近的精细实验解释。

## R10

正确性直觉：

1. 第三定律约束的是“零温极限行为”；
2. 当完美晶体满足 `S_res=0,b=0`，`S(T)` 的最低阶是 `T^3`，所以自然趋零；
3. 两个完美晶体只要都服从该结构，其熵差也按 `T^3` 缩小；
4. 指数冷却是连续渐近过程，有限时间只会得到正温度下界，不会触及 `0 K`；
5. 把以上结论写成断言后，脚本即可成为第三定律的自动化检查器。

## R11

数值稳定性与鲁棒策略：

- 积分下限采用 `eps=1e-8`，避免显式 `0/0`；
- 全部计算使用 `float64`；
- 解析熵与数值熵做误差门限校验（`max_abs_error < 2e-6`）；
- 单调性检查采用差分 `np.diff(...) < 0`；
- 输入参数均带正值/非负值检查，防止物理无意义输入。

## R12

关键参数与影响：

- `a_cubic`：决定 `T^3` 项强度，主导完美晶体低温熵曲线弯曲度；
- `b_linear`：决定线性低温项，非零常见于无序/缺陷主导系统；
- `s_residual`：残余熵，直接决定 `T->0` 时是否保持非零熵；
- `t_grid`：熵曲线采样范围，越低温越能看到第三定律极限；
- `tau_s`：冷却时间常数，越大则降温越慢；
- `n_steps`：轨迹采样分辨率，越高则轨迹更细致。

调参建议：

- 若要突出第三定律，优先降低 `t_grid` 最小温度并保留完美晶体对照组；
- 若要突出不可达性，固定 `t0_k` 后增大 `tau_s` 或缩短总时间窗口；
- 若要观察残余熵影响，逐步提高 `s_residual` 并比较低温平台高度。

## R14

常见失效模式：

1. 把“任意材料”误写成“完美晶体”而错误设置 `s_residual`；
2. 直接在 `T=0` 数值积分 `C/T`，导致除零问题；
3. 温度单位混乱（摄氏度误当开尔文）；
4. 仅看单点而忽略极限趋势，误判第三定律是否满足；
5. 把指数冷却演示当成普适制冷机性能模型。

## R18

`demo.py` 源码级算法流程（9 步）：

1. `main` 初始化三个材料参数：两个完美晶体（`s_residual=0,b_linear=0`）和一个无序材料（`s_residual>0`）。  
2. `build_entropy_frame` 遍历每个材料和温度点，调用 `heat_capacity`、`entropy_analytic`、`entropy_numerical` 生成对照表。  
3. `entropy_numerical` 在 `eps..T` 网格上计算 `C/T` 并用 `np.trapezoid` 积分，得到数值熵。  
4. `build_entropy_gap_frame` 对两个完美晶体逐温度计算 `abs(S_A-S_B)`，构建 Nernst 趋势表。  
5. `simulate_exponential_cooling` 按 `T(t)=T0*exp(-t/tau)` 生成有限时冷却轨迹。  
6. `main` 打印三张表：熵表、熵差表、冷却轨迹采样表。  
7. 断言积分一致性：`max_abs_error < 2e-6`，确保解析与数值流程一致。  
8. 断言第三定律趋势：完美晶体 `S(T)` 低温趋零，且完美晶体间熵差随降温收缩并趋零。  
9. 断言不可达性演示：冷却轨迹在全部有限时间步上 `T>0` 且单调下降，最后打印 `All checks passed.`。  

说明：第三方库仅用于数组、积分和表格展示（`numpy/pandas`）；第三定律判定逻辑、熵计算结构和验证门槛都在源码中显式实现，不是黑盒调用。
