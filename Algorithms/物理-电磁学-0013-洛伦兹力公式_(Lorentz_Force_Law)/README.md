## R01

洛伦兹力描述带电粒子在电磁场中的受力：

`F = q (E + v × B)`

其中：

- `q` 为电荷量；
- `E` 为电场强度；
- `B` 为磁感应强度；
- `v` 为粒子速度。

力可拆成两部分：

- 电场力 `F_E = qE`（可做功，改变动能）；
- 磁场力 `F_B = q(v × B)`（始终垂直于 `v`，单独作用时不做功，只改变速度方向）。

## R02

洛伦兹力是经典电磁学与粒子运动学的接口方程，贯穿：

- 回旋加速器与质谱仪；
- 等离子体约束与磁镜；
- 霍尔效应、电磁偏转与束流控制。

在非相对论范围（`|v| << c`）里，配合牛顿第二定律 `m dv/dt = F` 就能得到大量工程可用轨迹模型。

## R03

本条目 MVP 目标：

- 实现洛伦兹力向量计算，并验证基本物理性质；
- 在匀强磁场（可选匀强电场）中数值推进粒子轨迹；
- 验证关键解析结论：
  - 纯磁场下动能守恒；
  - 回旋角频率 `ω_c = |q|B/m`；
  - 拉莫尔半径 `r_L = m v_perp / (|q| B)`；
  - 平行分量近似匀速形成螺旋轨道。

## R04

核心方程组：

1. 受力方程  
   `F = q(E + v × B)`
2. 运动方程  
   `m dv/dt = q(E + v × B)`，`dx/dt = v`
3. 纯磁场性质（`E=0`）  
   `d/dt(1/2 m|v|^2) = v · F = q v·(v×B) = 0`
4. 匀强磁场回旋参数（`v_perp` 为垂直于 `B` 的速度分量）  
   `ω_c = |q|B/m`，`r_L = v_perp/ω_c = m v_perp/(|q|B)`

数值实现采用 Boris 推进（非相对论版本），其核心是把电场半步加速与磁场旋转拆开，保持纯磁场下能量稳定。

## R10

正确性直觉：

1. 公式层面：`v×B` 天生同时垂直于 `v` 和 `B`。  
2. 能量层面：纯磁场下 `v·(v×B)=0`，故磁场不直接改动速率。  
3. 轨道层面：垂直分量做匀速圆周，平行分量匀速前进，叠加成螺旋。  
4. 数值层面：Boris 把磁场作用视作“速度旋转”，天然更稳地保留纯磁场能量行为。

## R11

数值稳定与精度策略：

- 使用 `float64`；
- 采用 Boris 推进替代显式 Euler，降低纯磁场能量漂移；
- 以相位线性拟合估计 `ω_c`，减小单点噪声影响；
- 用相对误差阈值断言，而非追求机器零误差。

## R14

常见失效模式：

1. 单位混用（如 `B` 用高斯制、其余用 SI）；
2. 时间步长过大导致相位误差积累；
3. 将 `v×B` 误写为 `B×v` 导致方向反号；
4. `q` 符号处理错误，电子与离子轨迹方向颠倒；
5. 仅看位置图像不看守恒量，难以及时发现数值漂移。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `lorentz_force` 按 `q(E + v×B)` 计算总力，并返回电场力、磁场力拆分结果。  
2. `run_force_property_demo` 构造多组测试样例，逐条验证磁力正交性、符号反转和幅值关系。  
3. `boris_push` 先做电场半步更新 `v_minus`，再进入磁场旋转阶段。  
4. 在 `boris_push` 中计算 `t = (qB/m) dt/2` 与 `s = 2t/(1+|t|^2)`，完成速度旋转 `v_minus -> v_plus`。  
5. `boris_push` 做第二个电场半步，得到 `v_new`，并用 `x_new = x + v_new dt` 更新位置。  
6. `simulate_particle` 循环调用 `boris_push`，记录 `time/position/velocity/kinetic` 时序数组。  
7. `analyze_uniform_b_case` 从时序中估计数值回旋角频率、拉莫尔半径与轴向速度，并与解析值比较。  
8. `main` 打印关键结果并执行断言门槛，确保脚本在非交互模式下一次运行即可完成验证。

流程中没有把第三方库当黑盒：`numpy` 仅承担向量运算，物理方程离散步骤由源码显式展开。
