# 行列式计算算法

- UID: `MATH-0072`
- 学科: `数学`
- 分类: `线性代数`
- 源序号: `72`
- 目标目录: `Algorithms/数学-线性代数-0072-行列式计算算法`

## R01

问题定义: 给定一个 `n x n` 实矩阵 `A`，计算其行列式 `det(A)`，并在数值计算场景下尽量保持稳定性与可复现性。

## R02

核心数学背景: 行列式可看作线性变换对体积的缩放因子。若矩阵经过初等行变换转为上三角矩阵 `U`，则 `det(A)` 可由对角线乘积与行交换符号共同确定。

## R03

算法选择: 采用“带部分主元选取的高斯消元”。相比按定义展开（复杂度 `O(n!)`）或朴素消元（数值不稳定），该方法复杂度 `O(n^3)` 且在工程上更可靠。

## R04

输入输出约定:
- 输入: 二维方阵 `numpy.ndarray`，形状为 `(n, n)`。
- 输出: `float` 类型行列式估计值。
- 非法输入: 非方阵抛出 `ValueError`。

## R05

边界条件:
- `1x1` 矩阵直接返回单元素值。
- 若某列主元绝对值小于阈值 `pivot_tol`，视为奇异或近奇异，返回 `0.0`。
- 行交换会改变符号，必须显式记录。

## R06

MVP 步骤:
1. 复制输入矩阵到浮点数组，避免原地污染用户数据。
2. 对每一列做部分主元选取（当前列绝对值最大的候选行）。
3. 必要时交换行，并将符号因子 `sign` 乘以 `-1`。
4. 对主元下方各行做消元，构造上三角矩阵。
5. 返回 `sign * prod(diag(U))`。

## R07

正确性直觉:
- 行加减变换不改变行列式值；
- 行交换只改变符号；
- 上三角矩阵行列式等于对角线乘积；
- 因此“交换符号累计 + 三角化后对角乘积”与原矩阵行列式一致。

## R08

复杂度分析:
- 时间复杂度: `O(n^3)`（双重消元循环主导）。
- 空间复杂度: `O(n^2)`（保存矩阵副本）。

## R09

数值稳定性说明:
- 使用部分主元可以减少除以极小数造成的误差放大。
- 结果仍受浮点舍入影响；`demo.py` 中以相对误差阈值进行一致性检查而非追求位级完全相等。

## R10

与库函数关系:
- `numpy.linalg.det` 底层会调用 LAPACK 的 LU 分解路径（典型为 `dgetrf/zgetrf`）。
- 本实现是该思想的“可读版 MVP”，便于理解行交换、消元与行列式恢复过程。

## R11

代码结构:
- `determinant_gaussian_elimination`: 核心算法实现。
- `run_examples`: 固定矩阵示例，展示非奇异与奇异情形。
- `run_stress_test`: 随机矩阵压力测试，并与 `numpy.linalg.det` 对照。
- `main`: 串行执行示例与测试。

## R14

测试覆盖点:
- 一般可逆矩阵（`A_3x3`, `B_4x4`）；
- 明确奇异矩阵（`C_singular`）；
- 多尺寸随机矩阵（2, 3, 5, 8 阶）；
- 与 `numpy.linalg.det` 做数值对照，确保误差在可接受区间。

## R15

常见错误与规避:
- 忘记记录行交换符号，导致结果符号错误；
- 直接原地修改输入，造成调用方数据被污染；
- 未做主元检查，出现除零或巨量数值误差；
- 将“绝对误差”当作唯一标准，忽略规模效应。

## R16

可扩展方向:
- 对整数矩阵可扩展为分式/高精度路径（如 `fractions` 或符号计算）。
- 对大规模稀疏矩阵可切换到稀疏 LU。
- 对批量矩阵可向量化或并行化以提升吞吐量。

## R17

适用场景:

- 需要可解释实现而非仅调用黑盒 API 的场景。

## R18

源码级算法流（对应 `demo.py` 与 NumPy/LAPACK 路径）:
1. 入口函数检查输入是否为二维方阵，否则立即报错。
2. 将输入复制为浮点矩阵 `a`，初始化 `sign = 1`。
3. 对每一列 `col`，在 `a[col:, col]` 中寻找绝对值最大的行作为主元行。
4. 若主元过小（`< pivot_tol`），判定为奇异并返回 `0.0`。
5. 若主元行不是当前行，交换两行并将 `sign *= -1`。
6. 对主元下方每一行计算消元系数 `factor`，执行 `row -= factor * pivot_row`，把列下方元素消成 0。
7. 全部列处理完成后，取上三角对角线乘积并乘 `sign` 返回行列式。
8. `numpy.linalg.det` 的底层同样先做 LU（LAPACK `getrf`），再由置换奇偶性与 `U` 对角线乘积恢复行列式；本 MVP 与其核心流程一一对应。
