# 含时密度泛函理论 (TDDFT)

- UID: `PHYS-0209`
- 学科: `物理`
- 分类: `计算物理`
- 源序号: `210`
- 目标目录: `Algorithms/物理-计算物理-0210-含时密度泛函理论_(TDDFT)`

## R01

含时密度泛函理论（TDDFT）用于描述多电子体系在外场驱动下的实时动力学。  
核心思想是把真实相互作用电子体系映射到一个随时间演化的 Kohn-Sham（KS）非相互作用体系，并让两者电子密度 `n(x,t)` 一致。

本条目的 MVP 选择 1D 实空间玩具模型，重点展示：
- 基态 SCF（静态 KS）；
- 实时 TD-KS 传播；
- 外场响应量（偶极、吸收能）提取。

## R02

该算法条目的价值：

- 把“静态 DFT -> 动态响应”连成一条可运行链路；
- 演示 TDDFT 的关键环节：`v_eff[n](t)` 的自洽时间更新；
- 提供一个不依赖大型电子结构软件的教学基线；
- 为后续升级到频域线性响应、Casida 方程、实材料计算打基础。

## R03

本 MVP 求解的方程（原子单位）：

`i d/dt psi(x,t) = [-1/2 d^2/dx^2 + v_ext(x) + x E(t) + v_H[n](x,t) + v_xc[n](x,t)] psi(x,t)`

其中密度由单占据空间轨道（自旋配对）给出：

`n(x,t) = N_e |psi(x,t)|^2`，本实现取 `N_e=2`。

外场使用高斯包络脉冲：

`E(t) = E0 * exp(-(t-t0)^2/(2 sigma^2)) * sin(omega t)`。

## R04

本实现的物理近似：

- 空间离散为 1D 均匀网格；
- 外势为软核原子势 `v_ext(x) = -Z / sqrt(x^2 + a_nuc^2)`；
- Hartree 势使用软库仑核卷积；
- 交换-相关势使用局域绝热近似（ALDA-like）：
  `v_xc(n) = -C * n^(1/3)`；
- 交换-相关能密度与之匹配为 `e_xc(n) = -3/4 * C * n^(4/3)`。

这是教学型模型，不追求量化化学精度。

## R05

输入输出约定（无交互）：

- 输入（脚本内固定）：
1. 网格、时间步长、脉冲参数；
2. SCF 收敛参数；
3. 振幅扫描列表 `E0 in (0.004, 0.006, 0.008, 0.010)`。
- 输出：
1. 振幅扫描结果表（吸收能、最大偶极、主频、范数漂移）；
2. 最大振幅下的时间轨迹预览表；
3. 回归斜率（`sklearn` / `scipy` / 可选 `torch`）；
4. 自动断言与 `All checks passed.`。

## R06

算法主流程（高层）：

1. 构建实空间网格与动能三对角离散算符；
2. 用静态 SCF 求基态轨道 `psi0`；
3. 对每个脉冲振幅做实时传播；
4. 每个时间步根据当前密度更新 `v_H` 与 `v_xc`；
5. 用 Crank-Nicolson 步进推进波函数；
6. 记录偶极、内能、归一化误差；
7. 计算吸收能并做振幅标度拟合；
8. 执行一致性检查并给出 PASS/FAIL。

## R07

时间与空间复杂度（`N` 网格点，`T` 时间步，`A` 振幅点）：

- Hartree 卷积（核矩阵乘密度）每步约 `O(N^2)`；
- Crank-Nicolson 三对角求解每步 `O(N)`；
- 单次传播主导复杂度约 `O(T * N^2)`；
- 扫描总复杂度约 `O(A * T * N^2)`；
- 空间复杂度主导为 Hartree 核矩阵 `O(N^2)`。

本 MVP 选小规模参数，确保在普通 CPU 上可快速运行。

## R08

数值稳定策略：

- 每一步传播后显式归一化波函数，控制累积误差；
- `n` 在 `v_xc` 计算前做下限裁剪，避免低密度病态；
- 使用 Crank-Nicolson 保证线性算子层面的时间推进稳定性；
- 采用中等脉冲振幅，保持线性响应近似可用；
- 用范数漂移阈值作为强制质量门禁。

## R09

技术栈说明：

- `numpy`：网格、势能、密度与 FFT 频谱；
- `scipy.linalg`：基态三对角本征求解 + CN 带状线性求解；
- `scipy.optimize.curve_fit`：吸收能标度拟合；
- `scipy.signal.windows.hann`：频谱窗口；
- `pandas`：结果表格化输出；
- `scikit-learn`：线性回归对照；
- 可选 `torch`：一参数梯度下降拟合复核。

第三方库只提供基础数值算子，TDDFT 流程由源码显式拼装。

## R10

正确性直觉与可观测判据：

1. 基态 SCF 收敛后电子数应保持 `N_e`；
2. 实时传播范数应接近 1；
3. 吸收能应随脉冲振幅增大而增大（近似二次标度）；
4. `E_abs ~ beta * E0^2` 的回归系数应为正；
5. 偶极响应频谱应给出非零主频。

## R11

关键参数与默认值：

- 空间：`x in [-12, 12]`，`N=220`；
- 时间：`dt=0.06`，`n_steps=300`；
- SCF：`mix=0.32`，`max_iter=120`，`tol=2e-6`；
- 脉冲：`omega=1.2`，`t0=6.0`，`sigma=1.6`；
- 振幅：`(0.004, 0.006, 0.008, 0.010)`。

调参建议：
- 若 SCF 不收敛，降低 `mix` 或增大 `max_iter`；
- 若范数漂移偏大，减小 `dt`；
- 若吸收能非单调，减小振幅范围并增加步数。

## R12

内置验证项：

1. `ground-state SCF` 必须收敛；
2. `|∫n dx - N_e| < 1e-8`；
3. 吸收能对振幅严格单调上升；
4. 最大范数漂移 `< 2e-9`；
5. `sklearn` 与 `curve_fit` 斜率一致；
6. 可选 `torch` 斜率与前两者近似一致；
7. 主频 `omega_dom > 0.1`。

全部通过才输出 `All checks passed.`。

- 相同环境下结果可复现；
- 关键中间量（密度、势、偶极、能量）都可追踪；
- 不是黑盒“单函数 TDDFT 求解”，而是分步显式计算。

不提供严格数学收敛证明；这里只提供工程可验证性与物理一致性检查。

## R14

常见失效模式：

1. 时间步过大导致数值色散或范数漂移；
2. 忘记更新 `v_H[n(t)]/v_xc[n(t)]`，退化成固定哈密顿量传播；
3. 单位混用（场强、频率、时间非原子单位）；
4. 误把带场时刻能量直接当吸收能；
5. 脉冲过强导致非线性效应显著，线性拟合失真。

## R15

局限性：

- 1D 软库仑模型仅用于教学；
- 只含一个空间轨道（闭壳层映射）；
- `v_xc` 为简化 ALDA-like，不是可发表精度的泛函；
- 未包含离子运动、耗散、开边界与多体超越 TDDFT 修正。

因此输出应理解为算法演示，不是材料预测结论。

## R18

`demo.py` 源码级算法流程（9 步）：

1. `Grid1D/TDDFTConfig` 固化离散网格、SCF 参数和脉冲参数，形成可复现实验配置。  
2. `build_kinetic_tridiagonal` 构造 `T=-1/2 d^2/dx^2` 三对角离散算符，`build_hartree_kernel` 预计算软库仑核矩阵。  
3. `ground_state_scf` 迭代执行 `density -> v_H/v_xc -> H_KS -> 最低本征态 -> density`，得到基态轨道 `psi0`。  
4. `run_tddft_response` 对给定振幅进入时间循环：每步先由当前密度更新 `v_H[n(t)]` 与 `v_xc[n(t)]`，再组装含场 `H(t)`。  
5. `crank_nicolson_step` 用三对角带状线性系统求解 `(I+i dt H/2)psi_{t+dt}=(I-i dt H/2)psi_t`。  
6. 每步计算并记录 `dipole`、`internal_energy`、`norm`，并在指定间隔写入 `pandas` 轨迹表。  
7. 扫描全部振幅后，构建响应表 `amplitude -> absorbed_energy`，并提取最大振幅的主导频率 `omega_dom`。  
8. `fit_absorbed_energy_sklearn`、`fit_absorbed_energy_curve_fit`、`fit_absorbed_energy_torch_optional` 对同一 `E_abs ~ beta E0^2` 做多实现交叉拟合。  
9. `main` 统一执行断言：SCF 收敛、电子数守恒、范数漂移阈值、单调吸收能、回归一致性、主频有效，最后输出 `All checks passed.`。  

说明：第三方库仅负责通用线性代数/回归算子；TDDFT 的有效势更新、传播、能量定义和验收逻辑都在源码中逐步展开。
