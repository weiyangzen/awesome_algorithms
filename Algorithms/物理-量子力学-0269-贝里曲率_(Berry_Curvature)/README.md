# 贝里曲率 (Berry Curvature)

- UID: `PHYS-0266`
- 学科: `物理`
- 分类: `量子力学`
- 源序号: `269`

## R01

贝里曲率（Berry Curvature）是参数空间（这里是布里渊区动量空间）中的“几何磁场”，定义自贝里联络：

- `A_n(k) = i <u_n(k) | ∇_k u_n(k)>`
- `Omega_n(k) = ∂_{k_x} A_{n,y}(k) - ∂_{k_y} A_{n,x}(k)`

它决定了绝热演化中的几何相位，也直接关联反常速度、量子霍尔效应与 Chern 数等拓扑量。

## R02

本条目目标是给出一个可运行、可审计的最小实现：在二维两能带模型中计算贝里曲率，并积分得到 Chern 数。  
实现同时给出两种独立计算路径：

- 基于本征态重叠的离散规范不变算法（Fukui-Hatsugai-Suzuki, FHS）；
- 两能带 `H(k)=d(k)·sigma` 的解析曲率公式。

两路结果相互校验，避免“单一路径自洽但错误”。

## R03

MVP 具体问题设定为 Qi-Wu-Zhang (QWZ) Chern insulator：

`H(k) = sin(kx) sigma_x + sin(ky) sigma_y + (m + cos(kx) + cos(ky)) sigma_z`

任务是对下能带在离散 `nk x nk` 布里渊区网格上：

- 计算局域贝里曲率 `Omega(k)`；
- 数值积分得到 `C = (1/2pi) ∫ Omega(k) d^2k`；
- 在多个质量参数 `m` 上验证相图的 Chern 数分段常值行为。

## R04

本实现采用的核心方程：

1. 两能带解析曲率（下能带）：
`Omega(k) = -1/2 * d · (∂_{kx} d x ∂_{ky} d) / |d|^3`

2. FHS 链变量：
`U_mu(k) = <u(k)|u(k+mu)> / |<u(k)|u(k+mu)>|`

3. 离散小方格贝里通量：
`phi(k) = arg[ U_x(k) * U_y(k+dx) / ( U_x(k+dy) * U_y(k) ) ]`

4. 曲率密度近似：
`Omega_FHS(k) = phi(k) / (dk^2)`

5. 离散 Chern 数：
`C = (1/2pi) * Σ_k phi(k) = (1/2pi) * Σ_k Omega_FHS(k) dk^2`

## R05

设网格边长为 `N=nk`：

- 构造本征态网格：`N^2` 个 2x2 本征分解，复杂度 `O(N^2)`；
- FHS 曲率计算：每格常数次内积与相位，`O(N^2)`；
- 解析曲率计算：每格常数次向量运算，`O(N^2)`。

总时间复杂度 `O(N^2)`，空间复杂度 `O(N^2)`（存储本征态与曲率图）。

## R06

`demo.py` 执行一个非交互相图扫描：

- 质量参数 `m in {-3, -1, 1, 3}`；
- 每个 `m` 用 `nk=61` 计算一张曲率图；
- 输出 `Chern(FHS)`、`Chern(analytic)`、最小能隙、曲率图均值误差。

并通过断言验证：

- Chern 数与已知 QWZ 相图一致；
- FHS 结果具有整数化；
- 能隙未闭合（避开拓扑相变点）；
- 拓扑相（`m=1`）下离散与解析曲率图误差足够小。

## R07

优点：

- 使用最小依赖（`numpy`）即可复现实用拓扑不变量计算；
- FHS 方法规范不变，避免直接数值微分本征矢导致的规范噪声；
- 同时保留解析通道，方便物理解释与结果交叉验证。

局限：

- 仅覆盖二维两能带模型；
- 相变临界点（`m=-2,0,2`）附近能隙趋零，数值条件变差；
- 当前输出为文本指标，不含可视化图像。

## R09

适用场景：

- 量子拓扑课程中的最小可运行示例；
- 在更复杂材料模型前做算法链路预验证；
- 检查“某个二维两能带模型是否可能有非平庸 Chern 数”。

不适用场景：

- 多带强简并系统需要非阿贝尔 Berry 曲率；

- 要求高性能大规模 `k` 网格并行计算的生产环境。

## R10

正确性直觉：

1. FHS 通过闭合回路相位求“离散磁通”，天然对应几何曲率积分；
2. 链变量归一化后只保留相位，去除了局部规范选择引起的振幅影响；
3. Chern 数是全局通量的 `2pi` 归一化，理论上应量子化为整数；
4. 解析 `d·(∂d x ∂d)` 公式提供了独立的连续模型基准；
5. 两者对同一 `m` 给出一致积分值，说明实现链路可信。

## R11

数值稳定性策略：

- 在构造链变量时检测重叠模长，若接近 0 立即报错；
- 采样避开能隙闭合点，减少本征矢不连续导致的相位跳跃；
- 使用周期边界索引 `(i+1) % nk` 保持布里渊区拓扑闭合；
- 用 `endpoint=False` 保证网格不重复采样 `-pi` 与 `pi`。

## R12

关键参数：

- `mass (m)`：决定拓扑相，`(-2,0)` 与 `(0,2)` 的 Chern 数符号不同；
- `nk`：动量网格密度，越大曲率图更细、误差更小但计算更慢；
- `curvature_mae` 阈值：控制离散方法与解析方法一致性标准。

调参建议：

- 快速验证可用 `nk=31`；
- 需要更平滑曲率分布可用 `nk=101` 或更高；
- 若出现重叠接近零的异常，先远离临界 `m` 再加密网格。

## R13

该算法不属于近似优化，不提供“近似比”概念；它给出的是几何-拓扑量的数值估计。  
可给出的确定性保证：

- 对固定 `m`、`nk`、浮点环境，输出完全可复现；
- FHS 离散总通量在非闭合能隙区域呈稳定整数化；
- 脚本内断言确保结果至少满足相图一致性与基本数值健康性。

## R14

常见失效模式：

1. 在 `m=-2,0,2` 附近采样导致能隙接近 0，本征态不稳定；
2. `nk` 太小造成曲率图过粗，解析-离散误差变大；
3. 忽略周期边界会破坏闭环相位计算，导致 Chern 漂移；
4. 把本征矢直接做有限差分而不做规范处理，容易出现伪奇异。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `qiwuzhang_d_vector` 与 `qiwuzhang_hamiltonian` 定义二维两能带模型 `H(k)`。  
2. `lower_band_state_and_gap` 在单个 `k` 点做 2x2 本征分解，取下能带态并记录局域能隙。  
3. `sample_lower_band_states` 在整个 `nk x nk` 周期网格采样下能带本征态，得到 `states[kx,ky]`。  
4. `link_variable` 计算相邻本征态重叠并归一化为 U(1) 相位链变量，显式检查近零重叠异常。  
5. `berry_curvature_fhs` 逐小方格计算 `arg(Ux*Uy_x/(Ux_y*Uy))`，得到离散曲率图并积分出 `Chern(FHS)`。  
6. `analytic_berry_curvature` 与 `berry_curvature_analytic` 用 `-1/2 * d·(∂d x ∂d)/|d|^3` 生成解析曲率图和 `Chern(analytic)`。  
7. `run_single_case` 汇总每个质量参数下的 `Chern`、最小能隙、曲率图平均绝对误差。  
8. `main` 扫描 `m={-3,-1,1,3}`，打印表格并断言相图一致性、整数化与误差阈值，形成一次性非交互验收。  

说明：第三方库仅使用 `numpy` 的基础线性代数与数组能力；贝里曲率/FHS/Chern 的算法逻辑全部在源码内显式展开，不是黑盒调用。
