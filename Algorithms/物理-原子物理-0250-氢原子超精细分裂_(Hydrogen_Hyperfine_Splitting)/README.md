# 氢原子超精细分裂 (Hydrogen Hyperfine Splitting)

- UID: `PHYS-0247`
- 学科: `物理`
- 分类: `原子物理`
- 源序号: `250`

## R01

氢原子基态 `1S1/2` 的超精细分裂来自电子自旋 `J=1/2` 与质子自旋 `I=1/2` 的磁偶极耦合。零磁场下形成：

- 三重态 `F=1`（三重简并）；
- 单重态 `F=0`（单态）。

二者能量差对应著名 `21 cm` 线频率：

`nu_HFS ≈ 1420.405751768 MHz`。

## R02

该条目的核心价值：

- 是原子物理中“自旋耦合 -> 可观测谱线”的经典例子；
- 对天体物理中的中性氢 `21 cm` 射电观测至关重要；
- 也是 Breit-Rabi 图、低场 Zeeman 分裂和量子态混合的最小教学模型；
- 适合作为“哈密顿量构造 + 对角化 + 物理量验证”的 MVP 模板。

## R03

本条目 MVP 输入输出约定（无交互）：

- 输入（脚本内固定）：
1. 超精细常数通过 `A = h * nu_HFS` 设定；
2. 物理常数：`mu_B, mu_N, g_J, g_p`（来自 `scipy.constants`）；
3. 磁场扫描 `B = [0, 1, 2, 5, 10] mT`。

- 输出：
1. `B=1 mT` 的 4 条本征能级表（含 `mI/mJ/mF` 期望值）；
2. 从低能 `mF=0` 分支到三重态分支的跃迁频率表（`sigma-`, `pi`, `sigma+`）；
3. 自动断言与 `All checks passed.`。

## R04

MVP 使用的核心模型与公式：

1. 总哈密顿量：
`H = A (I·J) + g_J mu_B B J_z - g_p mu_N B I_z`。

2. 自旋 `1/2` 算符（无量纲）：
`Sx = sigma_x/2, Sy = sigma_y/2, Sz = sigma_z/2`。

3. 张量积构造（基底 `|m_I, m_J>`）：
- `I_k = S_k ⊗ 1`，`J_k = 1 ⊗ S_k`；
- `I·J = I_xJ_x + I_yJ_y + I_zJ_z`。

4. 零场本征值：
- `E(F=1)=A/4`（三重）；
- `E(F=0)=-3A/4`（单重）；
- `Delta E = A = h nu_HFS`。

5. `mF=0` 两支解析式（由 `2x2` 子块精确对角化）：
`E_0^± = -A/4 ± (1/2)*sqrt(A^2 + ((g_J mu_B + g_p mu_N)B)^2)`。

## R05

设磁场采样点数为 `K`：

- 单个 `B` 点对角化的是固定 `4x4` 矩阵，时间复杂度 `O(1)`；
- 全扫描成本 `O(K)`；
- 存储每个点常数条能级和频率，空间复杂度 `O(K)`。

虽然这里规模很小，但流程可直接迁移到更大角动量维度。

## R06

`demo.py` 的最小闭环：

- **闭环 A（模型）**：由 Pauli 矩阵和 Kronecker 积显式构造 `H(B)`；
- **闭环 B（数值）**：`numpy.linalg.eigh` 求本征值/本征矢，输出可审计能级；
- **闭环 C（物理校验）**：验证零场分裂、解析解一致性、弱场线性 Zeeman 斜率。

## R10

正确性直觉：

1. 零场应出现 `1+3` 的简并结构；
2. 平均三重态与单重态间隔应回到 `nu_HFS`；
3. `mF=0` 双分支在外场下通过平方根项分离；
4. 弱场下 `sigma±` 频移应对称线性偏移，`pi` 分支主要受二阶项影响。

## R11

数值稳定与工程策略：

- 先在能量域计算，最后统一转频率域（`nu = E/h`），避免单位混乱；
- 对哈密顿量执行 Hermitian 检查，防止算符拼接符号错误；
- 用解析结果（`mF=0` 子块、`mF=±1` 纯态）做强约束回归；
- 弱场断言采用 Hz 级绝对容差，避免浮点舍入误伤。

## R12

## R14

常见失效模式：

1. 电子/质子 Zeeman 项符号写反，导致频移方向错误；
2. 把 `g_p` 与 `mu_N`、`mu_B` 混用，造成量级错误；
3. 忘记 `S=1/2` 算符是 `sigma/2` 而非 `sigma`；
4. 仅调用对角化函数但不做解析互校，导致“数值正确性”不可审计。

## R18

`demo.py` 源码级算法流程（8 步）：

1. 定义常量：读取 `mu_B, mu_N, g` 因子，并将 `A = h * nu_21cm` 显式写出，不依赖黑盒原子结构库。  
2. 构造 `sigma_x/y/z` 与 `S_k=sigma_k/2`，再通过 `kron` 形成 `I_k`、`J_k`，把 `I·J` 展开为 `I_xJ_x + I_yJ_y + I_zJ_z`。  
3. `build_hamiltonian(B)` 逐项组装 `H = A(I·J) + g_J mu_B B J_z - g_p mu_N B I_z`，并检查 `H=H^dagger`。  
4. `diagonalize_levels(B)` 调用 `numpy.linalg.eigh` 对 `4x4` Hermitian 矩阵求本征值/本征矢；然后以 `v^dagger O v` 计算 `mI/mJ/mF` 期望值。  
5. `classify_branches` 使用 `mF` 期望值识别 `mF=+1,-1` 两条纯态分支，并按能量排序得到 `mF=0` 的低/高两支。  
6. `build_transition_table` 对磁场数组逐点计算 `sigma-`、`pi`、`sigma+` 三类跃迁频率，形成 `pandas` 表格输出。  
7. `run_checks` 执行三类校验：零场 `1+3` 简并、有限场解析式（`mF=0` 平方根分支与 `mF=±1` 线性式）、弱场一阶 Zeeman 斜率。  
8. `main` 打印常量、示例能级表、频率表并触发断言；全部通过后输出 `All checks passed.`。  

说明：第三方库只承担线性代数和常数提供；核心物理建模（哈密顿量项、算符结构、分支判别、解析互校）都在源码中显式实现。
