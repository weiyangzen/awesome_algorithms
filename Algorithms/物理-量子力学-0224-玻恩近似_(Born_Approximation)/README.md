# 玻恩近似 (Born Approximation)

- UID: `PHYS-0223`

## R01

玻恩近似用于求解“弱势散射”中的近似散射振幅与微分散射截面。

在量子力学散射问题中，完整解通常需要解 Lippmann-Schwinger 方程或偏波展开。第一玻恩近似把散射态波函数近似成入射平面波，从而把问题简化为对势函数的傅里叶型积分，得到可计算、可解释的闭式或数值结果。

## R02

本条目 MVP 聚焦三维、定态、单粒子、中心势散射：
- 输入：粒子质量 `m`、约化普朗克常数 `hbar`、入射能量 `E`、角度网格 `theta`、势函数参数（此处为 Yukawa 势 `g, mu`）。
- 输出：动量转移 `q`、第一玻恩散射振幅 `f_B(theta)`、微分截面 `dσ/dΩ = |f_B|^2`。
- 验证：数值积分结果与 Yukawa 势解析结果对比，并给出误差阈值判定。

这样既覆盖“通用数值流程”，又有“可校验的解析基准”。

## R03

第一玻恩近似的核心表达式（坐标表象）为：

`f_B(k' <- k) = -(m / (2*pi*hbar^2)) * ∫ exp(-i q·r) V(r) d^3r`

其中 `q = k' - k` 为动量转移，`|k| = |k'| = sqrt(2mE)/hbar`（弹性散射）。

物理含义：散射振幅正比于势函数的空间傅里叶分量，说明玻恩近似本质上是“弱势下的一次散射”。

## R04

对中心势 `V(r)`，振幅可化为一维径向积分：

`f_B(theta) = -(2m / (hbar^2 * q)) * ∫[0,∞] r V(r) sin(qr) dr`

`q = 2k sin(theta/2)`

当 `q -> 0` 时，上式用极限：

`f_B(0) = -(2m / hbar^2) * ∫[0,∞] r^2 V(r) dr`

因此数值实现必须显式处理 `q≈0`，避免除零不稳定。

## R05

本 MVP 选用 Yukawa 势：

`V(r) = g * exp(-mu r) / r`

其玻恩振幅解析解为：

`f_B^analytic(theta) = -(2m g) / (hbar^2 * (q^2 + mu^2))`

这给了一个理想的“真值对照”，可以直接验证数值积分模块的正确性与精度。

## R06

`demo.py` 输入输出约定：
- 输入：脚本内部固定参数（`m=1, hbar=1, E=2, g=-1, mu=1.5`）与角度网格 `0°..180°`。
- 输出：
1. 参数回显；
2. 角度采样表（`theta_deg, q, f_numeric, f_analytic, rel_err, dsigma_*`）；
3. 全局误差统计；
4. `Validation: PASS/FAIL`。

脚本不读取命令行参数，也不等待交互输入。

## R07

算法主流程（高层）：
1. 根据入射能量计算波数 `k`。
2. 由散射角计算动量转移 `q(theta)`。
3. 在径向网格上构造势函数 `V(r)`。
4. 用玻恩径向积分数值计算 `f_numeric(theta)`。
5. 用 Yukawa 解析式计算 `f_analytic(theta)`。
6. 计算微分截面与相对误差。
7. 通过阈值规则给出 `PASS/FAIL`。

## R08

设角度点数为 `N_theta`，径向点数为 `N_r`：
- 势函数采样：`O(N_r)`
- 对每个角度做一次径向积分：`O(N_theta * N_r)`
- 误差统计与表格输出：`O(N_theta)`

总复杂度约 `O(N_theta * N_r)`，空间复杂度约 `O(N_theta * N_r)`（向量化 `sin(qr)` 矩阵）或 `O(N_r)`（若改为逐角循环）。

## R09

数值稳定性处理：
- `q≈0` 使用极限表达式，避免 `sin(qr)/q` 的不稳定计算。
- 径向截断 `r_max` 取足够大，减少无限区间截断误差。
- 径向分辨率 `N_r` 足够细，以控制振荡积分误差。
- 相对误差计算加入小正数 `eps` 防止分母接近零导致异常放大。

这些措施确保 MVP 在可接受时间内给出可信结果。

- 相比只给理论公式：本条目提供可运行脚本与误差量化，能直接用于回归测试。

## R18

`demo.py` 源码级流程拆解（9 步）：
1. `BornConfig` 定义物理参数、角度与径向离散配置。
2. `wave_number` 按 `k = sqrt(2mE)/hbar` 计算入射波数。
3. `momentum_transfer` 由角度数组生成 `q = 2k sin(theta/2)`。
4. `yukawa_potential` 在径向网格上计算 `V(r)`。
5. `born_amplitude_numeric_central` 计算 `∫ rV(r)sin(qr)dr`，并对 `q≈0` 切换到极限积分 `∫ r^2V(r)dr`。
6. `born_amplitude_yukawa_analytic` 按闭式 `-(2mg)/(hbar^2(q^2+mu^2))` 生成解析振幅。
7. `differential_cross_section` 分别计算数值/解析 `|f|^2`。
8. `relative_error` 与 `build_report_table` 生成误差数组和可读表格。
9. `validate_results` 汇总阈值检查并在 `main` 输出 `Validation: PASS/FAIL`。

核心线性流程都在源码中显式实现；第三方库仅提供基础数组运算与表格能力，不掩盖算法本体。
