## R01

本条目实现“费米跃迁”最小可运行算法闭环，聚焦允许跃迁（allowed Fermi transition）中的三件事：
1. 显式计算统计速率函数 `f`（相空间积分 + 费米函数修正）。
2. 计算 `ft` 与辐射/核结构修正后的 `Ft`。
3. 用统计检验验证“`Ft` 近似常数”的一致性。

目标是把公式、数值积分、拟合与验证都展开到源码级，不依赖黑盒单函数结论。

## R02

费米跃迁由矢量流主导，经典选择定则可写成：
- `ΔJ = 0`
- `Δπ = 0`
- 同位旋结构通过费米矩阵元 `M_F` 进入跃迁强度。

在超允许 `0+ -> 0+` 语境下，实验分析常使用比较半衰期 `ft` 和修正量后的 `Ft` 进行跨核素一致性检验（CVC 相关检验思路）。

## R03

MVP 使用的核心公式：

1. 统计速率函数（无量纲）

a = `f = ∫ p W (W0 - W)^2 F(±Z, W) dW`

其中 `W = E/(m_e c^2)`，`p = sqrt(W^2 - 1)`，`W0 = 1 + T0/m_e`。

2. 费米函数近似

b = `F(±Z, W) ≈ x / (1 - exp(-x))`，`x = 2π η`，`η = ± α Z W / p`

3. 比较半衰期与修正量

c = `ft = f * t`

d = `Ft = ft * (1 + δR') * (1 + δNS - δC)`

`demo.py` 用上述关系构成完整数值流程。

## R04

`demo.py` 的输入输出设计（无交互）：
- 输入：脚本内部固定的 10 条理想化跃迁配置（`Z`、末态动能端点 `T0`）与随机种子。
- 输出：
1. 每条跃迁的 `f`、`t`、`ft`、`Ft`、残差百分比。
2. 全局加权 `Ft`（解析公式）与 PyTorch 拟合 `Ft`。
3. `chi2/ndf`、`p-value`、`Ft-Z` 线性趋势斜率。

## R05

变量与单位约定：
- `endpoint_kinetic_mev` (`T0`)：端点动能，单位 `MeV`
- `W`：总能量的无量纲化变量 `E/(m_e c^2)`
- `f_stat`：统计速率函数 `f`（无量纲）
- `t_partial_obs_s`：部分半衰期（或等效部分寿命），单位 `s`
- `ft_obs_s`：`f*t`，单位 `s`
- `Ft_corr_s`：修正后的 `Ft`，单位 `s`

## R06

数据策略：
- 该 MVP 使用可复现的合成数据，而不是外部实验数据库。
- 先设定参考常数 `Ft_ref = 3072.27 s`。
- 再结合每条跃迁的 `f` 与修正因子反推“理想 `t`”，并加入小幅测量噪声，生成观测样本。

这样可以验证算法流程本身，避免把重点转移到数据抓取与清洗。

## R07

数值积分算法（`f` 计算）实现细节：
1. 把积分变量离散成 `W ∈ [1+eps, W0]`。
2. 逐点评估 `p(W)`、`F(±Z,W)` 与 `(W0-W)^2`。
3. 用 `scipy.integrate.simpson` 做一维数值积分。
4. 输出每条跃迁的 `f_stat`。

该实现直接对应公式，不把物理步骤交给黑盒包隐藏。

## R08

一致性检验采用三条互补路径：
1. 解析加权平均：`Ft_mean = sum(w_i Ft_i)/sum(w_i)`，`w_i = 1/sigma_i^2`。
2. `scikit-learn` 线性回归：拟合 `Ft` 随 `Z` 的残余斜率，理想应接近 0。
3. `PyTorch` 单参数拟合：最小化加权 MSE，估计常数 `Ft`。

三条路径结果一致时，说明实现内部自洽。

## R09

复杂度（`N` 条跃迁、每条积分网格 `G`）：
- `f` 计算：`O(N*G)`
- 表格统计与残差计算：`O(N)`
- 线性回归与单参数优化：`O(N)` 到 `O(N*iter)`（此处 `iter` 固定）

当前默认规模 (`N=10`, `G=8000`) 在普通 CPU 上可快速完成。

## R10

数值稳定性处理：
- `p = sqrt(W^2-1)` 前使用 `clip`，避免浮点负值导致域错误。
- 费米函数分母 `1-exp(-x)` 使用 `-expm1(-x)`，减小小 `x` 区域消减误差。
- `|x|` 很小时使用 `1 + x/2` 的一阶展开，避免 `0/0` 不稳定。
- 权重计算 `1/sigma^2` 对 `sigma` 设置下界，避免极端放大。

## R11

正确性检查（脚本内断言 + 输出诊断）：
1. `f_stat > 0`，`t_partial_obs_s > 0`。
2. PyTorch 拟合常数接近设定参考值（允许小噪声偏差）。
3. `Ft-Z` 斜率绝对值小于阈值，验证“近似无系统漂移”。
4. 输出 `chi2/ndf` 和 `p-value` 作为一致性量化指标。

## R12

## R18

`demo.py` 的源码级流程可拆为 8 步：

1. `main()` 固定随机种子，调用 `build_synthetic_dataset()` 生成跃迁样本配置。  
2. `build_synthetic_dataset()` 对每条跃迁调用 `statistical_rate_function()` 数值积分得到 `f_stat`。  
3. 同函数内根据参考 `Ft_ref` 与修正项反推观测 `t`，再计算 `ft_obs` 与 `Ft_corr`。  
4. `weighted_ft_estimate()` 计算加权平均 `Ft`、不确定度、`chi2/ndf` 与 `p-value`。  
5. `fit_ft_constant_torch()` 用单参数加权 MSE 优化得到梯度法版本的常数 `Ft`。  
6. `run_linear_trend_test()` 使用 `LinearRegression` 拟合 `Ft` 对 `Z` 的斜率与 `R^2`。  
7. `main()` 汇总每条跃迁残差并打印结构化表格与全局诊断指标。  
8. `main()` 执行断言（正值、常数接近、斜率阈值），全部通过后输出 `All checks passed.`。
