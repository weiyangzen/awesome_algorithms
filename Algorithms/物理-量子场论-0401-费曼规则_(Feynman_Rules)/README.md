## R01

费曼规则（Feynman Rules）是把“拉氏量”转成“可计算散射振幅”的机械化翻译规则。

在微扰量子场论中，它回答的核心问题是：
给定模型以后，如何把每条外线、每个顶点、每条内部线变成代数因子，并按图拓扑相乘求和得到振幅 `M`。

本条目给出一个最小可运行 MVP：
- toy 标量理论 `phi^3 + phi^4`；
- 显式构建传播子与顶点规则；
- 用规则自动拼接 `2->2` 树图的 `contact + s/t/u` 四项；
- 输出微分截面与总截面，并做一致性校验。

## R02

要解决的问题：
把“规则表”与“数值结果”连成闭环，避免只停留在公式描述。

MVP 的具体目标：
1. 从交互项参数自动生成三点/四点顶点因子；
2. 对给定能量和角度网格计算树图总振幅；
3. 计算 `dσ/dΩ` 和积分总截面 `σ_total`；
4. 用断言验证费曼规则实现没有符号级错误。

## R03

`demo.py` 输入输出约定（无交互）：

- 输入（脚本内固定配置）：
1. 外线质量 `m`；
2. 内线传播子质量 `m_med` 与有限宽度 `Gamma_med`；
3. 耦合常数 `g3`（三点）与 `lambda4`（四点）；
4. 质心能量 `E_cm`；
5. 角度采样点数 `n_theta`。

- 输出：
1. 费曼规则表（传播子与顶点因子）；
2. 逐角度散射表（`s,t,u`、各图贡献、总振幅、`dσ/dΩ`）；
3. 总截面；
4. 规则一致性、交换对称与接触极限的误差指标。

## R04

本 MVP 采用的 toy 模型与规则：

1. 交互项：
`L_int = - g3/3! * phi^3 - lambda4/4! * phi^4`

2. 规则翻译：
- 三点顶点：`V3 = -i g3`
- 四点顶点：`V4 = -i lambda4`
- 标量传播子：`Delta(q^2) = i / (q^2 - m_med^2 + i m_med Gamma_med + i eps)`

3. `2->2` 树图振幅：
`M_tree = M_contact + M_s + M_t + M_u`
- `M_contact = V4`
- `M_x = V3 * Delta(x) * V3`, `x in {s,t,u}`

4. COM 等质量运动学：
- `s = E_cm^2`
- `p^2 = s/4 - m^2`
- `t = -2 p^2 (1-cos theta)`
- `u = -2 p^2 (1+cos theta)`

5. 微分截面与总截面：
- `dσ/dΩ = |M_tree|^2 / (64 pi^2 s)`
- `σ_total = 2pi * integral_0^pi sin(theta) * dσ/dΩ dtheta`

## R05

设角度网格点数为 `N`：

- 规则构建：`O(1)`；
- 每个角度计算四个图贡献：`O(1)`；
- 全角度扫描：`O(N)`；
- Simpson 角积分：`O(N)`；
- 总时间复杂度：`O(N)`；
- 空间复杂度：`O(N)`（保存表格）。

## R06

算法流程：

1. 根据 `g3, lambda4` 生成交互项对象；
2. 将交互项翻译成顶点因子（`-i g`）；
3. 由 `E_cm, m, theta` 生成 `s,t,u`；
4. 按图拓扑计算 `contact/s/t/u` 四项振幅；
5. 线性求和得到 `M_tree`；
6. 计算 `dσ/dΩ`；
7. 在角网格上形成结果 DataFrame；
8. 对 `sin(theta) dσ/dΩ` 做 Simpson 积分得到 `σ_total`；
9. 执行规则和物理一致性检查并断言。

## R10

正确性直觉：

1. 每张费曼图对应一个代数乘积因子；
2. 总振幅是所有同阶图的线性和；
3. 截面由 `|M|^2` 决定，必须非负；
4. 同种标量散射应满足 `theta <-> 180-theta` 对称（`t<->u`）；
5. 当 `g3=0` 时退化为常数接触振幅，存在可对照解析总截面。

## R11

数值稳定与鲁棒策略：

- 阈值检查 `E_cm > 2m`，避免非物理动量；
- 传播子加入 `Gamma_med` 与 `eps`，避免极点数值爆炸；
- 使用 Simpson 积分提高角积分稳定性；
- 对关键性质加断言（非负性、对称性、解析极限误差）。

## R12

关键参数：

- `g3`：控制交换道强度与角分布结构；
- `lambda4`：控制接触项背景；
- `m_med`：控制传播子极点位置；
- `Gamma_med`：控制极点附近平滑程度；
- `E_cm`：决定相空间与 `s,t,u` 区间；
- `n_theta`：决定积分精度和输出分辨率。

调参建议：

- 观察交换道主导时增大 `g3/lambda4`；
- 若总截面积分误差偏大，提升 `n_theta`（奇数点更适配 Simpson）；
- 若出现尖峰，适度增大 `Gamma_med`。

可验证保证（脚本断言）：

- `g3=0` 接触极限下，总截面相对误差 `< 1e-3`。

## R14

常见失效模式：

1. 顶点符号写错（`-i g` 写成 `+i g`）；
2. 漏掉 `s/t/u` 某一道；
3. 忘记传播子 `i` 因子或极点符号；
4. 角积分遗漏 `sin(theta)` 权重；
5. 在阈值以下强行计算导致虚动量问题。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `QFTConfig` 固定 toy 理论参数（质量、耦合、宽度、能量、角网格）。
2. `build_interactions` 构建交互项清单（`phi^3` 与 `phi^4`）。
3. `vertex_factor` 把交互项翻译成顶点因子 `-i*coupling`，`build_rule_table` 汇总规则表。
4. `mandelstam_2to2_equal_mass` 计算每个散射角的 `s,t,u`。
5. `propagator_factor` 计算内部线传播子 `i/(q^2-m^2+i m Gamma+i eps)`。
6. `diagram_amplitudes` 按拓扑分别计算 `M_contact`, `M_s`, `M_t`, `M_u`，并线性求和成 `M_tree`。
7. `build_scattering_table` 与 `integrate_total_cross_section` 先得到 `dσ/dΩ`，再做 Simpson 积分得到 `σ_total`。
8. `verify_vertex_rules`、`verify_crossing_symmetry`、`contact_only_check` 分别执行规则校验、`t/u` 对称校验和接触极限解析校验；`main` 汇总并断言通过。

说明：`numpy/pandas/scipy` 仅用于基础数值、表格和积分；“费曼规则生成 + 图级振幅拼接”完全在源码中显式实现，没有把主流程交给黑盒库。
