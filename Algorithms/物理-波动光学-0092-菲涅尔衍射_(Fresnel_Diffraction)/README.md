# 菲涅尔衍射 (Fresnel Diffraction)

- UID: `PHYS-0092`
- 学科: `物理`
- 分类: `波动光学`

## R01

菲涅尔衍射描述的是有限传播距离下的近场衍射现象。
与夫琅禾费衍射的远场傅里叶极限不同，菲涅尔衍射保留了传播过程中的二次相位项，
因此屏上强度分布会随传播距离 `z` 明显变化。

本条目采用一维单缝模型，给出一个可运行、可验证的 MVP：
用离散 Fresnel 积分计算数值结果，并与单缝 Fresnel 积分解析式进行逐点对比。

## R02

最小可验证问题：
给定波长 `lambda`、缝宽 `a`、传播距离 `z`，在观察面 `x'` 上计算归一化强度曲线 `I(x')`，并检查：

- 数值积分曲线与解析曲线的中心区 RMSE 是否足够小；
- 两曲线中心区相关系数是否接近 1；
- 中心区最大绝对误差是否受控。

这三个指标同时通过时，脚本输出 `Validation: PASS`。

## R03

MVP 边界与假设：

- 一维标量波近似（不考虑偏振与矢量电磁效应）；
- 入射波为均匀平面波；
- 孔径是理想矩形单缝；
- 仅验证单一波长和单一传播距离；
- 不做探测器噪声、像差、部分相干等工程因素。

## R04

`demo.py` 使用的核心方程：

- 数值 Fresnel 积分（忽略全局常数，仅保留形状）：
  `U_num(x') ~ integral A(x) * exp(i*pi*(x'-x)^2/(lambda*z)) dx`
- 强度定义：
  `I(x') = |U(x')|^2`
- 单缝 Fresnel 解析解（`A(x)=1, x in [-a/2,a/2]`）形状：
  `U_theory(x') ~ (C(u2)-C(u1)) + i*(S(u2)-S(u1))`
- 其中：
  `u1 = sqrt(2/(lambda*z)) * (-a/2 - x')`
  `u2 = sqrt(2/(lambda*z)) * ( a/2 - x')`

## R05

脚本输出内容：

- 参数回显（`lambda`, `a`, `z`, 采样数、空间窗口）；
- Fresnel 数 `N_F = a^2/(lambda*z)`；
- 三项验证指标（RMSE、相关系数、最大绝对误差）；
- 观察面若干采样点的数值/解析强度对比表；
- 最终 `Validation: PASS/FAIL`。

脚本无交互输入，适合直接自动化调用。

## R06

实现策略：

- `FresnelConfig` 统一管理物理参数、离散化规模与阈值；
- `build_grids` 构造孔径面与观察面网格，并生成矩形单缝透过函数；
- `compute_numeric_fresnel_field` 用矩阵化离散积分计算数值场；
- `compute_theory_field_single_slit` 用 `scipy.special.fresnel` 构造解析参考场；
- `normalize_intensity` 将两条曲线独立归一化；
- `build_validation_report` 在中心区域做定量误差评估。

## R07

优点：

- 直接保留近场二次相位项，能体现 Fresnel 衍射本质；
- 同时提供数值解与解析解，验证目标清晰；
- 代码短小、确定性强、可复现实验结果。

局限：

- 只覆盖 1D 单缝，不含二维孔径；
- 未建模真实系统噪声与光源相干特性；
- 直接积分为 `O(N_obs * N_ap)`，不适合超大规模网格。

## R09

设孔径采样点数为 `N_ap`，观察点数为 `N_obs`：

- 核心数值积分复杂度：`O(N_obs * N_ap)`；
- 误差统计与表格构建：`O(N_obs)`；
- 总体时间复杂度：`O(N_obs * N_ap)`；
- 空间复杂度（显式核矩阵）：`O(N_obs * N_ap)`。

当前默认规模（`1400 x 900`）用于 MVP 验证是可接受的。

## R10

数值稳定性处理：

- 两条曲线分别按峰值归一化，避免全局常数影响比较；
- 指标仅在中心窗口 `|x'| <= 0.35 mm` 统计，减少边缘截断影响；
- 网格跨度与采样密度分开控制，便于在精度和成本间折中；
- 对关键步骤加输入检查（样本不足时显式报错）。

## R11

`demo.py` 默认参数：

- `lambda = 532 nm`
- `a = 120 um`
- `z = 3.0 cm`
- 孔径采样 `N_ap = 1400`，观察采样 `N_obs = 900`
- 孔径窗口 `x in [-0.6, 0.6] mm`
- 观察窗口 `x' in [-0.8, 0.8] mm`

对应 Fresnel 数：
`N_F = a^2/(lambda*z) ~= 0.902`，处于典型近场过渡区。

## R12

内置判定阈值：

1. 中心区 `RMSE < 0.025`；
2. 中心区 `Correlation > 0.995`；
3. 中心区 `MaxAbsError < 0.08`。

三项全部通过才判定 `PASS`，否则脚本以非零状态退出。

## R14

常见失效模式：

1. 单位混用（nm / um / mm / m）导致数量级错误；
2. `num_aperture_samples` 太小，积分离散误差放大；
3. 观察窗口过大但采样不足，造成中心区分辨率下降；
4. `z` 取值极端时（很近或很远）当前参数组合不再匹配阈值。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `FresnelConfig` 定义物理量（`lambda,a,z`）、离散网格规模和判定阈值。  
2. `build_grids` 在孔径面生成 `x`、在观察面生成 `x'`，并构造矩形单缝 `A(x)`。  
3. `compute_numeric_fresnel_field` 显式构造核 `exp(i*pi*(x'-x)^2/(lambda*z))`，对 `x` 做离散积分得到数值复振幅。  
4. `compute_theory_field_single_slit` 先把 `x'` 映射到 `u1,u2`，再用 `fresnel` 计算 `C/S` 差值得到解析复振幅。  
5. `normalize_intensity` 将 `|U|^2` 分别按峰值归一化，消除比较中无关的全局幅度常数。  
6. `build_validation_report` 在中心窗口统计 RMSE、相关系数和最大绝对误差。  
7. `build_profile_table` + `print_profile_preview` 用 `pandas` 组织并输出局部采样点，便于人工审计。  
8. `main` 汇总阈值检查并输出 `Validation: PASS/FAIL`（失败时返回非零退出码）。

第三方库的职责是可解释且非黑盒的：

- `numpy`：网格、复指数核、向量化积分与误差统计；
- `scipy.special.fresnel`：仅用于 Fresnel 积分 `C/S` 数值求值；
- `pandas`：结果表格化展示。

物理建模与验证逻辑（核构造、积分路径、指标阈值、通过条件）均在源码中显式实现。
