# α衰变理论 (Alpha Decay Theory)

## R01

α衰变理论的核心是“量子隧穿导致的有限衰变概率”：
- 在经典图像里，α 粒子能量低于库仑势垒，无法越出核外
- 在量子图像里，波函数在禁阻区有非零穿透概率，形成可测衰变常数

本目录 MVP 把该理论转成可执行算法，输入母核参数后估计半衰期量级：
- 输入: `A_parent, Z_parent, Q_MeV, preformation`
- 步骤1: 计算子核 `A_d=A-4, Z_d=Z-2`
- 步骤2: 由 `Q` 得 α 动能 `E_alpha ≈ Q * A_d / A_parent`
- 步骤3: 建立势垒区间 `[R, b]`，其中 `R` 为接触半径、`b` 为外转折点
- 步骤4: 计算 WKB 作用量 `G = sqrt(2μ)/ħ * ∫[R,b] sqrt(V(r)-E) dr`
- 步骤5: `P = exp(-2G)`、`λ = S_alpha * ν * P`、`t1/2 = ln2 / λ`
- 输出: `G, penetrability, lambda, t1_2_s, log10_t1_2_s`

## R02

理论时间线可压缩为三个关键节点：
- 1900s 初期: Rutherford 等通过放射性实验建立 α 粒子与核衰变现象学
- 1928: Gamow 独立提出 α 衰变量子隧穿解释；Gurney 与 Condon 同年给出等价隧穿图像
- 后续几十年: 发展出 Geiger-Nuttall 定律、预形成因子与角动量阻滞修正等工程化版本

因此本算法的“理论起点年份”通常锚定在 `1928`。

## R03

提出背景是经典核模型无法解释“低能仍可衰变”悖论：
- 实验观测到许多重核会自发发射 α 粒子
- 但 α 粒子典型动能小于库仑势垒峰值，经典力学预测应被困在核内

量子力学提供的穿透机制同时解释了两件事：
- 为什么衰变是概率过程（对应指数衰变律）
- 为什么半衰期随 `Q` 值变化极其敏感（指数依赖 `G`）

## R04

设有 `N` 个核素样本，`quad` 每次积分平均评估 `M` 次被积函数：
- 时间复杂度（典型）: `O(N*M)`
- 若用闭式近似 Gamow 因子，理论上可降为近似 `O(N)`

本 MVP 的主要计算量来自每个样本的一维自适应积分。

## R05

空间复杂度：
- 单样本只保留常数个中间变量（`R,b,E,G,λ`），为 `O(1)`
- 汇总 `N` 条结果到 DataFrame 为 `O(N)`

## R06

以 `238U -> 234Th + α` 为例（脚本内置）：
- 输入近似: `A=238, Z=92, Q=4.27 MeV`
- 计算 `E_alpha`、`R` 与 `b`，形成禁阻区积分区间
- 得到 `G` 后，穿透率 `P=exp(-2G)` 极小
- 再结合撞壁频率 `ν` 与预形成因子 `S_alpha`，得到 `λ` 和 `t1/2`

脚本同时给出若干核素，展示模型可跨越从微秒到亿年以上的量级变化。

## R07

算法意义：
- 物理层面: 直接体现“量子隧穿 -> 放射性寿命”这一核物理核心机制
- 方法层面: 将势垒几何、WKB 指数因子和统计衰变律串成可复现实验链路
- 工程层面: 可作为更复杂核结构模型（含角动量与壳效应）的 baseline

## R08

直接依赖的理论与数值组件：
- 量子隧穿/WKB 半经典近似
- 库仑势垒模型 `V(r)=2*Z_d*e^2/r`
- 两体约化质量与动能-频率换算
- 指数衰变律 `t1/2 = ln2 / λ`
- 一维数值积分（SciPy `integrate.quad`，底层为 QUADPACK 自适应求积）

## R09

适用前提与边界：
- 本实现假设 `l=0`（未显式加入离心势垒项）
- 预形成因子 `S_alpha` 视为外部给定/经验参数
- 势垒只保留简化库仑外区，不做核内微观势精细拟合
- 适合做“量级估计与机制演示”，不等价于核数据评估库的精密模型

## R10

正确性来自三层一致性：
- 物理结构一致: 隧穿路径、转折点、衰变常数关系符合标准 α 衰变理论
- 数学链路一致: `Q -> E -> (R,b) -> G -> P -> λ -> t1/2` 无跳步
- 程序控制一致: 先做输入合法性检查，再做积分与指数变换，避免伪结果

## R11

数值稳定性注意点：
- `exp(-2G)` 可能严重下溢；代码用对数链路先算 `log_lambda`
- 当 `Q` 很小，`b` 很大，积分区间变宽，`quad` 调用成本上升
- `b <= R` 说明参数不落在可隧穿区间，脚本会返回无效原因
- 最终误差通常由模型近似（如 `S_alpha`、`l=0`）主导，而非积分器舍入误差

## R12

真实运行成本（超越大 O）：
- 每个样本只做一次 1D 积分，CPU 成本低
- 常数开销主要在 Python 函数调用和自适应求积迭代
- DataFrame 构造与打印对小样本几乎可忽略
- 大批量核素扫描时，积分次数是主要瓶颈

## R13

误差界与保证：
- 算法本身是确定性流程，不含随机采样
- `quad` 的数值容差在代码中显式设置（`epsabs=1e-10, epsrel=1e-8`）
- 结果的不确定性主要来自物理建模简化，而非数值求积本身
- 对工程使用应关注“对数半衰期量级”而非逐位绝对值

## R14

鲁棒性与失效模式：
- `Q<=0`、`A<=4`、`Z<=2`、`S_alpha` 越界会被拒绝
- 非物理速度（`beta<=0` 或 `beta>=1`）会被拦截
- 近阈值衰变中，指数敏感导致小参数误差可放大为巨大寿命差异
- 若用于高角动量衰变，因缺少离心项会系统低估寿命

## R15

工程实现要点：
- 用 `dataclass` 固化输入字段，降低参数错位风险
- 把常量集中定义，便于后续做参数扫描
- 先做“规则过滤”再做积分，避免无意义计算
- 输出保留 `log10_t1_2_s` 与观测对比误差，便于跨数量级分析

## R18

源码级算法流（非黑箱，8 步）：
1. 读取 `AlphaDecayCase`，先执行输入合法性检查（`A,Z,Q,S_alpha`）。
2. 由母核得到子核 `(A-4, Z-2)`，并做反冲修正得到 `E_alpha`。
3. 计算接触半径 `R` 与外转折点 `b`，若 `b<=R` 则判为无有效隧穿区。
4. 构造被积函数 `sqrt(V(r)-E)`，其中 `V(r)=2*Z_d*e^2/r`。
5. 调用 `scipy.integrate.quad` 做区间 `[R,b]` 的自适应求积，得到积分值。
6. 用 `G = sqrt(2μ)/ħ * integral` 形成 WKB 指数项，并计算 `log_penetrability=-2G`。
7. 计算撞壁频率 `ν`，合成 `log_lambda = log(S_alpha)+log(ν)-2G`，再得 `t1/2`。
8. 汇总结果到 pandas 表格，并输出与观测 `log10(t1/2)` 的差值用于快速校验。

第 5 步补充说明：虽然外层只是一行 `quad(...)`，但其内部是可解释的自适应求积流程（误差估计驱动区间细分），并非机器学习黑箱。
