## R01

本条目实现 1D Haar 小波的快速小波变换（Fast Wavelet Transform, FWT）最小可运行版本。

核心目标：
- 给定离散实信号，完成多层分解，得到近似系数与细节系数；
- 支持逆变换并验证“可重构性”（分解后可恢复原信号）；
- 在同一脚本中给出一个简单去噪示例，展示小波阈值法的实用价值。

## R02

问题定义（有限精度计算版）：
- 输入：
  - 一维实信号 `x in R^n`；
  - 分解层数 `L`（`1 <= L <= floor(log2(m))`，`m` 为补零后长度）；
- 输出：
  - 多层近似系数 `a_L`；
  - 多层细节系数列表 `[d_1, d_2, ..., d_L]`（`d_1` 为最细层）；
  - 逆变换恢复信号 `x_hat`；
  - 去噪后信号与误差指标（MSE）。

补充约束：
- 若原长度不是 2 的幂，MVP 用零填充到最近的 2 的幂，再在输出阶段裁回原长度。

## R03

数学基础与公式（Haar 小波）：

1) 单层分析（analysis）
- 对偶样本对 `(x_{2k}, x_{2k+1})`，定义
  - `a_k = (x_{2k} + x_{2k+1}) / sqrt(2)`（低频/近似）
  - `d_k = (x_{2k} - x_{2k+1}) / sqrt(2)`（高频/细节）

2) 单层合成（synthesis）
- 给定 `a_k, d_k`，可恢复
  - `x_{2k}   = (a_k + d_k) / sqrt(2)`
  - `x_{2k+1} = (a_k - d_k) / sqrt(2)`

3) 多层递归
- 仅对近似分量 `a` 继续分解，得到金字塔结构：
  `x -> (a_1, d_1) -> (a_2, d_2, d_1) -> ... -> (a_L, d_L, ..., d_1)`。

4) 复杂度
- 每层处理元素数减半，总工作量为几何级数，整体 `O(n)`。

## R04

算法总览（MVP）：
1. 输入检查（维度、有限值、层数合法）。
2. 若长度非 2 的幂，补零到下一幂次长度。
3. 执行 Haar 前向 FWT：每层按偶/奇抽样计算 `a` 与 `d`。
4. 保存全部细节系数与最终近似系数。
5. 执行 Haar 逆变换 IFWT：从最粗层开始逐层上采样合成。
6. 将重构结果裁剪回原始长度并计算重构误差。
7. 估计噪声方差并对细节系数做软阈值，执行去噪重构。
8. 输出关键统计量（长度、层数、误差、部分系数预览）。

## R05

核心数据结构：
- `numpy.ndarray`：
  - `signal`：原始或补零后的信号；
  - `approx`：最终层近似系数；
  - `details[i]`：第 `i+1` 层细节系数；
- `FWTResult`（`dataclass`）：
  - `original_length`、`padded_length`、`levels`；
  - `approx`、`details`；
  - `reconstructed`（裁剪后重构信号）。

## R06

正确性要点：
- Haar 分析/合成是互逆线性变换；
- 分析矩阵是正交矩阵的分块递归形式，满足能量守恒（在浮点下近似）；
- 若不做阈值截断，`IFWT(FWT(x))` 在数值误差范围内应接近 `x`；
- 去噪步骤改变了细节系数，故“去噪重构”不再等于原信号，而是偏向平滑估计。

## R07

复杂度分析：
- 前向 FWT：`O(n)` 时间，`O(n)` 额外存储（保存系数）；
- 逆向 IFWT：`O(n)` 时间，`O(n)` 存储；
- 软阈值处理：遍历所有细节系数，`O(n)`；
- 总体：时间 `O(n)`，空间 `O(n)`。

## R08

边界与异常处理：
- 非一维输入：抛出 `ValueError`；
- 含 `nan/inf`：抛出 `ValueError`；
- 空信号：抛出 `ValueError`；
- `levels < 1` 或超过允许最大层数：抛出 `ValueError`；
- 逆变换时若细节层级尺寸与近似尺寸不匹配：抛出 `ValueError`。

## R09

MVP 取舍：
- 使用 `numpy` + 标准库，不依赖额外小波包；
- 选用 Haar 作为最小可解释基，避免复杂滤波器边界处理细节；
- 聚焦“算法透明 + 可运行 + 可验证”，不追求完整信号处理框架；
- 去噪只实现通用软阈值，不覆盖高级自适应阈值策略。

## R10

`demo.py` 函数职责：
- `next_power_of_two`：计算补零目标长度；
- `pad_to_power_of_two`：完成零填充与原长记录；
- `validate_signal`：检查一维有限输入；
- `haar_fwt`：执行多层 Haar 前向快速小波变换；
- `haar_ifwt`：执行 Haar 逆变换；
- `universal_threshold`：用最细层细节估计噪声并给出阈值；
- `soft_threshold`：对系数做软阈值收缩；
- `wavelet_denoise`：阈值化细节后逆变换得到去噪信号；
- `build_demo_signal`：生成固定随机种子的测试信号；
- `run_demo`：组织实验并打印重构误差、去噪效果与系数摘要。

## R12

输出解读：
- `original_length` / `padded_length`：原长度与补零后长度；
- `levels`：分解层数；
- `max_abs_reconstruction_error`：不阈值时重构最大绝对误差（应接近机器精度）；
- `mse_noisy`：噪声信号相对干净信号的均方误差；
- `mse_denoised`：去噪后相对干净信号的均方误差；
- `improvement_ratio`：`mse_noisy / mse_denoised`，大于 1 表示去噪有效；
- `detail_preview`：每层细节系数前若干项，便于观察多尺度特征。

## R13

建议最小测试集：
- 可逆性：随机向量长度为 2 的幂，验证 `||x - IFWT(FWT(x))||_inf` 很小；
- 非幂长输入：长度如 150，验证补零与裁剪逻辑；
- 平滑+突变混合信号：观察细节系数在突变点附近更大；
- 异常输入：空数组、二维数组、含 `nan`、非法 `levels`。

## R14

可调参数：
- `levels`：分解层数（越高越偏向全局尺度）；
- `seed`：随机噪声种子；
- `noise_std`：噪声强度；
- `threshold_scale`：阈值缩放（默认 1.0，可调小保细节、调大更平滑）；
- `preview_k`：日志输出时每层预览系数数量。

## R15

方法对比：
- 与 FFT：
  - FFT 频域全局基，适合平稳周期结构；
  - 小波是时频局部基，更适合局部突变和非平稳信号。
- 与移动平均去噪：
  - 移动平均简单但会抹平边缘；
  - 小波阈值通常能更好保留尖峰/边缘结构。
- 与直接调用第三方 `pywt.wavedec`：
  - 第三方实现更完整更高效；
  - 本实现更适合教学和源码级理解。

## R18

源码级算法流（对应 `demo.py`，8 步）：
1. `build_demo_signal` 生成“低频正弦 + 局部高频 + 台阶突变”的干净信号，并按固定种子叠加高斯噪声。  
2. `validate_signal` 检查输入是一维且全有限，再由 `pad_to_power_of_two` 统一到 2 的幂长度。  
3. `haar_fwt` 在每一层把当前序列按偶/奇位分组，计算 `a=(even+odd)/sqrt(2)` 与 `d=(even-odd)/sqrt(2)`。  
4. 递归只对 `a` 继续分解，累计得到 `[d_1, d_2, ..., d_L]` 与最终 `a_L`。  
5. `haar_ifwt` 从 `(a_L, d_L)` 开始逐层逆推，用 `x_even=(a+d)/sqrt(2)`、`x_odd=(a-d)/sqrt(2)` 重建上一层。  
6. 完成逆推后裁剪到原长度，计算 `max_abs_reconstruction_error` 验证前向/逆向一致性。  
7. `universal_threshold` 用最细层细节的 MAD 估计噪声 `sigma`，再按 `tau=sigma*sqrt(2log n)` 得阈值；`soft_threshold` 对每层细节收缩。  
8. 用阈值化后的细节执行一次 `haar_ifwt` 得到去噪信号，输出 `mse_noisy`、`mse_denoised` 与改善倍数。
