## R01

马尔可夫随机场（Markov Random Field, MRF）是定义在无向图上的联合概率模型。

给定图 `G=(V,E)` 与随机变量集合 `X={X_i}_{i in V}`，MRF 满足局部马尔可夫性：

- `X_i` 在给定其邻域 `N(i)` 后，与其他非邻居条件独立；
- 联合分布可写成团势函数乘积（Hammersley-Clifford 形式）。

常见写法：

- `P(x) = (1/Z) * exp(-E(x))`
- `E(x) = sum_c psi_c(x_c)`，`Z` 是配分函数。

## R02

MRF 是统计物理（Ising/Potts 模型）与概率图模型在无向建模上的结合。

典型发展路径：

- 物理中的格点相互作用模型提供了“局部耦合 + 全局分布”框架；
- 统计机器学习将其推广到图像恢复、分割、空间统计、结构化预测；
- 推断方面形成了多条主线：ICM、Gibbs/MCMC、Belief Propagation、Graph Cut、变分近似。

本条目采用最小可运行路线：二值 Ising MRF + ICM 近似 MAP 推断。

## R03

MRF 解决的问题是：当变量之间存在明显“邻近相关性”时，如何把这种结构写入概率模型并做推断。

在图像去噪里，观测 `Y` 含噪，潜在真值 `X` 通常空间上更平滑。我们希望：

- `X` 不要偏离观测太远（数据一致性）；
- 邻近像素倾向取相同标签（先验平滑性）。

MRF 正好把这两件事统一为一个能量最小化问题。

## R04

本目录 `demo.py` 使用二值 Ising 观测模型：

- 隐变量 `x_ij in {-1, +1}`；
- 观测 `y_ij in {-1, +1}`；
- 后验能量
  `E(x|y) = -eta * sum_ij (x_ij y_ij) - beta * sum_(i,j)-(u,v) in E (x_ij x_uv)`。

其中：

- `eta` 控制“贴近观测”的强度；
- `beta` 控制“邻域平滑”的强度；
- 当 `beta` 增大时，结果更平滑；当 `eta` 增大时，结果更接近观测。

推断算法用 ICM（Iterated Conditional Modes）逐点坐标下降近似 MAP。

## R05

设网格大小为 `H x W`，总变量数 `N = H*W`，ICM 迭代轮数为 `T`。

- 单次 sweep（遍历全部像素）：`O(N)`；
- 总时间复杂度：`O(T*N)`；
- 额外空间复杂度：`O(N)`（存储当前状态与观测）。

本 MVP 的默认规模是 `28x28`，在普通 CPU 上可瞬时完成。

## R06

示例配置（见 `demo.py` 默认参数）：

- 网格 `28x28`；
- 噪声翻转概率 `0.18`；
- `beta=1.1`，`eta=1.6`；
- 最大迭代 `25`。

程序会生成一个“可解释结构”的二值真值场（圆形+条带区域），先加噪得到观测，再用 ICM 恢复。

## R07

优点：

- 能清晰表达“局部相关 + 全局一致”的结构先验；
- 能量函数可解释，参数含义明确；
- ICM 实现简单、运行快，适合教学和原型。

局限：

- ICM 只保证收敛到局部最优，不保证全局最优；
- 配分函数 `Z` 通常难算，精确概率计算困难；
- 参数 `beta/eta` 需要根据任务调节，过强会过平滑。

## R09

适用场景：

- 图像/栅格数据的二值或多类标签平滑问题；
- 空间统计中的邻域依赖建模；
- 结构化先验明显、局部关系主导的问题。

不适用场景：

- 高维连续变量且复杂长程依赖主导；
- 需要全局最优且误差容忍极低的任务（仅 ICM 不够）；
- 大规模实时推断且图结构复杂时（需更强近似或并行算法）。

## R10

本 MVP 的正确性直觉：

1. 固定其他变量时，单个节点的最优标签由局部场 `eta*y_i + beta*sum_neighbors x_j` 决定；
2. ICM 每次把该节点替换为局部最优标签，因此不会增大后验能量；
3. 对所有节点反复更新，能量序列单调不增；
4. 状态空间有限（`2^N`），因此迭代最终停在某个局部稳定点。

`demo.py` 中会显式检查能量是否单调不增。

## R11

数值与实现稳定性策略：

- 标签统一使用 `{-1,+1}`，避免 0/1 与符号公式混淆；
- 采用棋盘顺序（红黑更新）降低更新偏置；
- 设置最大迭代上限，防止异常输入下长循环；
- 对能量单调性和精度改进做断言，及时暴露退化配置。

## R12

关键参数调节建议：

- `noise_flip_prob`：观测噪声强度（实验难度）；
- `beta`：平滑先验强度，越大越“连片”；
- `eta`：观测置信度，越大越“贴观测”；
- `max_iters`：收敛预算。

经验上：

- `beta` 过小，去噪不足；
- `beta` 过大，细节被抹平；
- 一般先固定 `eta`，在 `beta` 上做网格搜索。

## R13

理论性质说明：

- 本实现是确定性 ICM，不是随机近似算法，因此“随机成功率保证”不适用；
- ICM 提供的是局部最优保证，不提供全局最优保证；
- 该问题不是近似比算法场景，因此“近似比”不适用。

可保证的部分：

- 在固定参数与固定初始化下，结果可复现；
- 能量沿迭代不增（在当前离散更新规则下）。

## R14

常见失效模式：

1. `beta` 过大导致过平滑，边界细节丢失；
2. `eta` 过大导致几乎不改动噪声观测；
3. 初始化较差时陷入次优局部极小值；
4. 真实数据非二值结构，Ising 假设过强。

诊断方式：

- 比较 `noisy_acc` 与 `map_acc` 是否提升；
- 观察 `boundary_disagreement_count` 是否异常降低（可能过平滑）；
- 查看能量历史是否很快停滞。

## R15

工程落地建议：

- 从该二值 MVP 起步，先验证“先验确实有益”；
- 需要多类标签时扩展到 Potts 模型；
- 需要更好全局解时替换为 Graph Cut / alpha-expansion 或更强近似；
- 在真实任务中将 `beta/eta` 纳入验证集调参流程。

## R16

相关方法与关系：

- CRF：条件建模 `P(X|Y)`，常用于判别式结构化预测；
- HMM：链式有向图，强调时间序列方向依赖；
- Gibbs 采样：可做后验采样而不只做 MAP；
- Mean-Field / Loopy BP：常见近似推断方法。

MRF 的核心特色是无向邻域耦合与能量视角。

## R17

`demo.py` 的 MVP 功能清单：

- 构造 2D 二值真值场并生成翻转噪声观测；
- 用 Ising MRF + ICM 执行 MAP 去噪；
- 输出并对比：
  - 观测准确率与去噪后准确率；
  - 初始/最终能量与迭代轮数；
  - 平滑度指标（邻域不一致边数）；
- 执行自动断言：能量单调不增、精度提升、输出标签合法。

## R18

`demo.py` 源码级算法流程拆解（8 步）：

1. `generate_ground_truth` 生成可控结构的二值真值场 `x_true`（圆形与条带区域）。  
2. `add_bernoulli_flip_noise` 以固定翻转概率对 `x_true` 加噪，得到观测 `y_obs`。  
3. `ising_posterior_energy` 定义后验能量 `E(x|y)`，由 unary（观测一致性）和 pairwise（邻域平滑）两项组成。  
4. `icm_map_inference` 以 `y_obs` 作为初始化，进行红黑棋盘顺序的逐点更新。  
5. 每次节点更新通过 `local_neighbor_sum` 汇总 4 邻域，计算局部场并执行 `x_i <- sign(eta*y_i + beta*sum_neighbors)`。  
6. 每轮 sweep 结束后记录能量与改动像素数，若无像素变化则提前收敛。  
7. `main` 计算噪声观测与 MAP 结果的准确率、平滑度与能量改变量，并打印核心统计。  
8. `main` 最后执行断言：能量历史单调不增、MAP 准确率明显高于噪声基线、输出仅含 `{-1,+1}`。  

该实现未调用黑盒 MRF 包；建模、推断和验证都在源码中可逐行追踪。
