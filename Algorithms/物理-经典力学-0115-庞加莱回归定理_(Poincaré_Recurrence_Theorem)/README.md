## R01

庞加莱回归定理讨论的是保测动力系统中的“回到原来附近”现象。

设 `(X, Sigma, mu)` 是有限测度空间（`mu(X) < +infinity`），`T: X -> X` 保持测度（`mu(T^{-1}(A)) = mu(A)`）。则对任意可测集 `A`，几乎处处的 `x in A` 会在某些时刻 `n >= 1` 满足 `T^n(x) in A`。等价地说：几乎所有点会无限多次回到任意给定邻域。

## R02

它在经典力学中的意义是：对封闭、体积保持（李乌维尔体积不变）的哈密顿系统，长期演化不会把“绝大多数状态”永远散失到某个新区域，而会反复接近先前状态。

这不等于“短时间回归”或“精确周期”，但说明了有限相空间与保测演化下的长期可返性。

## R03

本条目 MVP 的目标是把定理变成可计算、可验证的最小实验：

- 离散严格验证：在有限相空间 `Z_N^2` 上构造保测双射，证明每个状态都落在某个周期轨道中，从而必然精确回归。
- 连续数值验证：在二维环面 `[0,1)^2` 上迭代面积保持映射，统计样本点是否在有限步内回到初态的 `epsilon` 邻域。

## R04

`demo.py` 选用 Arnold cat map（猫映射）：

- 连续形式：
  `T(x, y) = (x + y, x + 2y) mod 1`
- 离散形式（模 `N`）：
  `T_N(x, y) = (x + y, x + 2y) mod N`

其线性部分矩阵 `A=[[1,1],[1,2]]`，`det(A)=1`，因此保持面积（Lebesgue 测度）并在模 `N` 下可逆。

## R05

核心理论连接：

1. `det(A)=1` 保证连续映射是面积保持（保测）。
2. 在有限网格 `Z_N^2` 上，`T_N` 是双射，双射在有限集合上可分解为若干不相交循环。
3. 每个循环都对应一个周期轨道，因此每个离散状态都存在精确回归时间。
4. 连续情况下一般只保证“几乎处处、任意近”而非“每点精确周期”。

## R06

程序输出两组结果：

- **离散回归报告**：状态总数、循环数量、最小/最大/平均周期、周期长度分布。
- **连续回归报告**：样本数、邻域半径 `epsilon`、最大迭代步数、首次回归比例、首次回归步数统计、最小回归距离统计。

并附带自动断言，失败时直接退出非零状态。

## R09

复杂度：

- 离散循环分解：`O(N^2)` 时间、`O(N^2)` 空间（访问标记）。
- 连续首次回归统计：`O(M * T)` 时间、`O(M)` 额外空间。

其中 `N` 为离散网格边长，`M` 为连续样本点数量，`T` 为最大迭代步数。

## R10

数值设计与稳定性：

- 连续映射每步都取 `mod 1`，状态始终留在紧致空间 `[0,1)^2`。
- 环面距离采用 `min(|dx|, 1-|dx|)` 处理跨边界最近距离。
- 首次回归只记录第一次命中，避免后续覆盖。
- 使用固定随机种子保证实验可复现。

## R11

关键参数（`demo.py` 默认）：

- `N = 37`：离散网格大小，状态数 `N^2 = 1369`。
- `sample_size = 400`：连续样本数量。
- `epsilon = 0.025`：回归邻域半径（环面欧氏距离）。
- `max_steps = 3000`：连续迭代上限。
- `seed = 20260407`：随机种子。

## R12

正确性检查（脚本内断言）：

1. `det(A) == 1`（面积保持条件）。
2. 模 `N` 映射是双射（像集大小等于 `N^2`）。
3. 循环分解覆盖全部状态（周期和等于 `N^2`）。
4. 连续样本在给定步数内回归比例达到阈值（默认 `>= 0.99`）。

## R14

常见失效模式：

1. 忘记 `mod 1` 或 `mod N`，状态越界导致模型失真。
2. 距离若不用环面最短距离，会误判“未回归”。
3. `epsilon` 过小或 `max_steps` 过短，导致统计回归比例偏低。
4. 将连续统计结果误解为“每个点都有短周期”。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `cat_map_mod_n_step` 与 `is_bijection_mod_n` 在 `Z_N^2` 上构造并验证猫映射是双射。  
2. `decompose_cycles_mod_n` 对每个未访问状态沿映射前进直到回到起点，得到该循环周期并标记整条轨道。  
3. 汇总所有循环周期，得到“周期和 = `N^2`”与“每点回归”的离散严格证据。  
4. `cat_map_continuous` 在 `[0,1)^2` 上实现连续猫映射迭代。  
5. `torus_l2_distance` 计算环面最短欧氏距离，正确处理周期边界。  
6. `first_return_times` 对每个样本点逐步迭代，记录首次进入 `epsilon` 邻域的时间以及全过程最小距离。  
7. `summarize_periods` 把离散周期长度计数成表格（`period`、`count`）便于审计。  
8. `main` 统一打印两类报告并执行断言（面积保持、双射、覆盖、高回归比例），形成一次性非交互验证。  

整个流程没有把第三方库当黑盒：`numpy/pandas` 只做数组与表格，核心映射、循环分解、回归统计都在源码中显式实现。
