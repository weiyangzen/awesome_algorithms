# 拉普拉斯方程 (Laplace's Equation)

- UID: `PHYS-0152`
- 学科: `物理`
- 分类: `静电学`
- 源序号: `153`
- 目标目录: `Algorithms/物理-静电学-0153-拉普拉斯方程_(Laplace's_Equation)`

## R01

本条目实现二维静电学中的拉普拉斯方程最小可运行版本（MVP），目标是把“边界条件 -> 离散求解 -> 误差验证”完整打通。  
采用矩形区域上的有限差分离散，并使用 SOR（Successive Over-Relaxation）迭代求解电势 `V(x, y)`，同时与解析解对比以验证数值正确性。

## R02

问题定义（MVP 范围）：
- 连续方程：`∇²V = 0`（无自由电荷区域的静电势）。
- 计算区域：`(x, y) ∈ [0,1] × [0,1]`。
- 边界条件（Dirichlet）：
  - `V(x,0)=0`
  - `V(0,y)=0`
  - `V(1,y)=0`
  - `V(x,1)=sin(πx)`
- 目标输出：
  - 数值解 `V_num`；
  - 解析解 `V_exact`；
  - 收敛与精度指标（迭代次数、残差、最大绝对误差、RMSE）。

## R03

数学模型：
1. 拉普拉斯方程：`∂²V/∂x² + ∂²V/∂y² = 0`。  
2. 本边界值问题的解析解为：
`V_exact(x,y) = sin(πx) * sinh(πy) / sinh(π)`。  
3. 数值上在均匀网格使用中心差分，将连续方程转成离散代数方程组。  
4. 迭代法求稳态电势，相当于离散调和函数求解。

## R04

离散与算法流程：
1. 生成 `nx × ny` 均匀网格，初始化电势矩阵 `V`。  
2. 写入四条边界的 Dirichlet 条件。  
3. 对内点应用五点格式离散：  
`V(i,j)` 由四个邻点按网格步长加权平均得到。  
4. 使用 SOR 更新：`V_new = (1-ω)V_old + ωV_GS`。  
5. 每轮统计最大更新量 `max_update`，若小于 `tol` 判定收敛。  
6. 收敛后计算离散残差 `||Δ_h V||_∞` 与解析解误差。

## R05

核心数据结构：
- `numpy.ndarray`：
  - `x`, `y`：网格坐标；
  - `potential_num`：SOR 得到的数值解；
  - `potential_exact`：解析解；
  - 中间变量 `err`、`laplace_residual`。
- `LaplaceResult`（`dataclass`）：
  - 统一封装网格、解场与诊断指标（迭代次数、收敛状态、误差）。
- `pandas.DataFrame`：
  - 输出中心线剖面采样表，便于终端快速检查。

## R06

正确性要点：
- 五点差分是二阶一致离散，网格细化时离散误差按 `O(h^2)` 下降。  
- SOR 本质上是在 Gauss-Seidel 基础上做超松弛加速，`1 < ω < 2` 时可显著提升收敛速度。  
- 对当前边界条件有已知解析解，因此可直接做“真值对照”，不依赖黑盒库。  
- 同时检查两类量：
  - 方程满足程度：离散残差 `||Δ_h V||_∞`；
  - 解的准确性：`max_abs_error`、`RMSE`（对 interior 节点）。

## R07

复杂度分析（`N = nx * ny`, `K = 迭代轮数`）：
- 时间复杂度：`O(K * N)`，每轮需更新全部内点。  
- 空间复杂度：`O(N)`，主要存储数值场与解析场。  
- 在本 MVP 默认网格（`81 × 81`）下，CPU 上可快速完成求解。

## R08

边界与异常处理：
- `nx < 5` 或 `ny < 5`：拒绝（网格过粗，不利于稳定验证）。  
- `omega` 不在 `(1,2)`：拒绝（SOR 参数非法）。  
- `tol <= 0` 或 `max_iter < 1`：拒绝。  
- 求解后若未收敛：`run_checks` 抛出异常，避免静默输出无效结果。  
- 若残差或误差超过阈值：立即失败，确保 MVP 质量可追踪。

## R09

MVP 取舍：
- 选用二维矩形 + Dirichlet 边界，避免几何和边界拟合复杂度。  
- 选用 SOR 而非多重网格/有限元，以最小代码实现可解释数值求解。  
- 不依赖 PDE 黑盒求解器（如直接调用高层函数一步出结果），所有关键步骤在 `demo.py` 明确展开。  
- 重点是“可运行 + 可验证 + 可读”，而不是通用 PDE 框架。

## R10

`demo.py` 模块职责：
- `initialize_grid`：创建网格并注入边界条件。  
- `sor_solve_laplace`：执行核心 SOR 迭代求解。  
- `residual_inf_norm`：计算离散拉普拉斯残差无穷范数。  
- `exact_solution`：生成解析解做误差对照。  
- `solve_laplace_mvp`：串联求解与评估流程。  
- `run_checks`：对收敛性和误差阈值做断言。  
- `sample_profile_table`：输出中线剖面对比表。  
- `main`：无交互运行入口。

## R11

运行方式：

```bash
cd Algorithms/物理-静电学-0153-拉普拉斯方程_(Laplace's_Equation)
uv run python demo.py
```

脚本无需交互输入，会自动求解并打印收敛与误差报告。

## R12

输出字段说明：
- `Grid size`：网格规模 `nx × ny`。  
- `Converged`：是否在 `max_iter` 内满足 `max_update < tol`。  
- `Iterations`：迭代轮数。  
- `Max update (last iter)`：最后一次迭代中的最大单点更新幅度。  
- `Residual inf-norm`：离散拉普拉斯残差 `||Δ_hV||_∞`。  
- `Max abs error vs exact (interior)`：内点最大绝对误差。  
- `RMSE vs exact (interior)`：内点均方根误差。  
- `Centerline profile sample`：`y=0.5` 截线上的数值解与解析解对比表。

## R13

建议最小测试集：
- 默认参数：`nx=81, ny=81, omega=1.92, tol=1e-10`，应收敛并通过误差断言。  
- 网格加密：例如 `121 × 121`，预期误差进一步降低。  
- 参数敏感性：
  - `omega` 接近 1（退化到近 GS），收敛更慢；
  - `omega` 过大接近 2，可能震荡甚至不稳定。  
- 异常输入：
  - `nx=4`；
  - `omega=2.1`；
  - `tol<=0`。

## R14

关键可调参数：
- `nx`, `ny`：网格密度，影响精度与耗时。  
- `omega`：SOR 松弛因子，默认 `1.92`。  
- `tol`：收敛阈值（最大更新量）。  
- `max_iter`：最大迭代次数上限。  

经验建议：
- 若机器性能允许，先增大网格密度再收紧 `tol`；
- `omega` 通常在 `1.7~1.95` 间试探较稳健。

## R15

方法对比：
- Jacobi：
  - 实现最简单，但收敛慢。  
- Gauss-Seidel：
  - 比 Jacobi 快，但仍可能需要较多迭代。  
- SOR（本实现）：
  - 在低成本实现下显著加速，是教学和工程入门中的常用折中。  
- 多重网格 / 有限元：
  - 可处理更复杂问题，但超出本条目 MVP 范围。

## R16

应用场景：
- 平行板/屏蔽结构的静电势快速估算；  
- 课程实验中的边值问题离散求解演示；  
- 作为更复杂泊松方程、有限元电场仿真的基线验证算例；  
- 数值 PDE 求解器开发中的单元测试基准问题。

## R17

可扩展方向：
- 从 `∇²V=0` 扩展到含源项的泊松方程 `∇²V=-ρ/ε`；  
- 支持 Neumann/Robin 混合边界条件；  
- 引入非均匀介质 `∇·(ε(x,y)∇V)=0`；  
- 升级到三维网格与并行加速（Numba/CUDA）；  
- 用多重网格替代单层 SOR 提升大规模问题效率。

## R18

源码级算法流（对应 `demo.py`，8 步）：
1. `main` 调用 `solve_laplace_mvp`，设定网格、SOR 参数与收敛阈值。  
2. `initialize_grid` 创建 `x/y` 网格与电势矩阵 `V`，写入 `V(x,1)=sin(πx)` 等边界值。  
3. `sor_solve_laplace` 进入迭代循环，对每个内点计算五点差分下的 `V_GS`。  
4. 同函数按 `V_new=(1-ω)V_old+ωV_GS` 做超松弛更新，并追踪本轮 `max_update`。  
5. 若 `max_update < tol` 则提前收敛返回；否则继续直到 `max_iter`。  
6. `solve_laplace_mvp` 调用 `exact_solution` 构造解析场，并在内点上计算 `max_abs_error` 与 `RMSE`。  
7. `residual_inf_norm` 计算离散拉普拉斯残差 `||Δ_hV||_∞`，用于检验方程满足程度。  
8. `run_checks` 校验收敛、残差与误差阈值，随后 `sample_profile_table` 输出中线采样表，`main` 打印最终报告。
