# 磁流体动力学 (Magnetohydrodynamics, MHD)

## R01

磁流体动力学（MHD）描述导电流体（如等离子体）与电磁场耦合演化。  
在理想 MHD 近似下，流体由守恒律控制，磁场通过感应方程与动量/能量方程耦合，形成快/慢磁声波、阿尔芬波与激波等结构。

## R02

本条目聚焦一维理想 MHD 的最小可运行问题：

- 空间仅沿 `x` 方向离散；
- `Bx` 取常数（满足一维下 `div B = 0` 约束）；
- 使用 Brio-Wu 激波管初值，验证算法能稳定推进到有限时间并保持物理可行状态。

## R03

MVP 目标是“数值可跑 + 结果可核验”，具体包括：

- 实现 1D 理想 MHD 守恒方程组；
- 用有限体积法 + Rusanov 数值通量推进；
- 输出密度、压强、速度、横向磁场剖面；
- 通过内置阈值自动给出 `Validation: PASS/FAIL`。

## R04

`demo.py` 使用的守恒变量为：

- `U = [rho, rho*vx, rho*vy, rho*vz, By, Bz, E]`

其中 `Bx` 为常数。理想 MHD 通量写作：

- `F_rho = rho*vx`
- `F_mx = rho*vx^2 + p_tot - Bx^2`
- `F_my = rho*vx*vy - Bx*By`
- `F_mz = rho*vx*vz - Bx*Bz`
- `F_By = By*vx - Bx*vy`
- `F_Bz = Bz*vx - Bx*vz`
- `F_E = (E + p_tot)*vx - Bx*(v·B)`

并使用：

- `p_tot = p + 0.5*(Bx^2 + By^2 + Bz^2)`
- `E = p/(gamma-1) + 0.5*rho*|v|^2 + 0.5*|B|^2`

## R05

离散策略：

- 空间：一阶有限体积（cell-centered）；
- 界面通量：Rusanov（局部 Lax-Friedrichs）
  `F* = 0.5*(F_L+F_R) - 0.5*s_max*(U_R-U_L)`；
- 时间步：显式 CFL 约束
  `dt = cfl*dx/max(|vx|+c_f)`；
- 边界：零梯度外流边界（复制端点单元）。

快磁声速 `c_f` 由局部本征速度公式计算，确保 `s_max` 覆盖最大传播速度。

## R07

优点：

- 结构小而完整：方程、通量、推进、验收在单文件闭环；
- 适合快速验证理想 MHD 基础离散流程；
- 所有关键物理量可直接审计。

局限：

- 一阶格式耗散较大，接触间断会被抹宽；
- 未引入高阶重构（MUSCL/WENO）和近似 Riemann 求解器（HLLD 等）；
- 未包含电阻项、黏性项和 2D/3D `div B` 控制技术。

## R09

设网格数为 `N`，时间步数为 `T`。

- 每步需计算通量与波速，时间复杂度 `O(N)`；
- 总时间复杂度 `O(N*T)`；
- 存储 `U`、通量和若干派生数组，空间复杂度 `O(N)`。

## R10

数值稳定性处理：

- 时间步由最大局部 `|vx|+c_f` 动态限制；
- 每步更新后对 `rho`、`p` 应用下限并回写守恒量，避免出现非物理负值；
- 波速异常（非有限或非正）立即抛错；
- 设定 `max_steps`，防止意外死循环。

## R11

默认参数（`demo.py`）：

- `gamma = 2.0`
- `Bx = 0.75`
- 区间：`x in [-0.5, 0.5]`
- `n_cells = 400`
- `cfl = 0.35`
- `t_end = 0.08`

初值采用 Brio-Wu 激波管：

- 左态：`rho=1.0, p=1.0, By=1.0`
- 右态：`rho=0.125, p=0.1, By=-1.0`
- 初始速度与 `Bz` 均为 0。

## R13

## R14

常见失效模式：

1. `cfl` 过大导致更新不稳定，出现负压或崩溃；
2. 网格过粗时激波结构过度扩散，诊断量偏差变大；
3. 若省略 floors，强间断附近可能出现非物理状态；
4. 边界过近或终止时间过长会放大外流边界对守恒量统计的影响。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `MHDConfig` 固化方程参数、网格参数、CFL 和数值下限。  
2. `build_brio_wu_initial_state` 构造 Brio-Wu 左右 Riemann 初值，并转为守恒变量。  
3. `primitive_to_conserved` / `conserved_to_primitive` 在两种变量表示间互转，统一能量与压强计算口径。  
4. `mhd_flux` 根据理想 MHD 通量公式计算每个单元的物理通量。  
5. `fast_magnetosonic_speed` 计算局部最大特征速度 `|vx|+c_f`，供 CFL 和数值粘性系数使用。  
6. `rusanov_flux` 在每个界面组合左右状态，得到稳定的一阶数值通量。  
7. `solve_1d_ideal_mhd` 执行时间循环：外流边界填充 -> 通量差更新 -> 正性修复 -> 统计守恒量漂移。  
8. `main` 汇总诊断指标，执行阈值检查并打印 `Validation: PASS/FAIL`。

第三方库没有作为“黑盒求解器”直接给答案：

- `numpy` 仅用于数组运算与向量化；
- `pandas` 仅用于结构化输出表格；
- MHD 方程、通量、波速、时间推进与验收逻辑都在源码中显式实现。
