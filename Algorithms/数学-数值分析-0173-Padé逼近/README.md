## R01

Padé逼近（Padé approximation）用一个有理函数
`R_{m,n}(x)=P_m(x)/Q_n(x)`
去逼近目标函数的幂级数。它不是只截断多项式，而是同时学习“分子 + 分母”，因此在同样系数预算下，常比泰勒截断在更大区间内更准确。

## R02

简要历史：
- 1892年前后：Henri Padé系统化提出该逼近形式。
- 20世纪：Padé逼近在渐近分析、微分方程、控制、量子物理中广泛应用。
- 现代数值计算中：常作为“从局部级数恢复全局行为”的基础工具，也常与连分式和有理逼近技术联用。

## R03

设目标函数在 `x=0` 的幂级数为：
`f(x)=sum_{k=0}^{∞} c_k x^k`。

寻找 `deg(P)<=m, deg(Q)<=n, Q(0)=1`，满足：
`Q(x)f(x)-P(x)=O(x^{m+n+1})`。

把 `Q(x)=1+q_1x+...+q_nx^n` 代入后，可得线性条件：
- 对 `k=m+1..m+n`：`c_k + sum_{j=1}^{n} q_j c_{k-j} = 0`
- 解出 `q_j` 后，再由
  `p_k = sum_{j=0}^{min(k,n)} q_j c_{k-j}` (`q_0=1`)
  得到 `P(x)` 系数。

## R04

输入输出定义（对应本目录MVP）：
- 输入：
  - 幂级数系数 `c_0..c_{m+n}`
  - 阶数 `m, n`
- 输出：
  - 分子系数 `p_0..p_m`
  - 分母系数 `q_0..q_n`（其中 `q_0=1`）
  - 在测试区间上的误差指标（RMSE、最大绝对误差）

## R05

核心思想：
- 局部约束：强制匹配到 `m+n` 阶展开系数。
- 全局收益：有理函数能表达“极点/弯折”结构，通常比同阶多项式外推更稳。
- 参数经济：同样是 `m+n+1` 个自由度时，Padé经常优于纯多项式截断。

## R06

标准计算流程：
1. 准备幂级数系数 `c_0..c_{m+n}`。
2. 设定 `Q(0)=1`，未知量为 `q_1..q_n`。
3. 构造 `n×n` 线性系统并求解 `q`。
4. 用卷积关系计算 `p_0..p_m`。
5. 得到 `R_{m,n}(x)=P/Q`。
6. 在目标区间评估误差并检查分母零点风险。

## R07

伪代码：

```text
function pade_from_series(c, m, n):
    # c 至少包含 c[0..m+n]
    solve A * q_tail = b     # q_tail = [q1..qn]
    q = [1, q_tail...]

    for k in 0..m:
        p[k] = sum_{j=0..min(k,n)} q[j] * c[k-j]

    return p, q
```

```text
function eval_rational(p, q, x):
    num = polyval_ascending(p, x)
    den = polyval_ascending(q, x)
    return num / den
```

## R08

正确性要点：
- `Qf-P=O(x^{m+n+1})` 的含义是：`x^0..x^{m+n}` 的系数全部相等。
- 分母系数由高阶匹配条件唯一确定（在非奇异情形）。
- 分子再由低阶卷积回代得到，因此构造出的 `R_{m,n}` 必然满足定义条件。

## R09

复杂度分析：
- 构造并求解线性系统（规模 `n`）：`O(n^3)`。
- 计算分子系数：`O(mn)`。
- 在 `N` 个点上评估：`O((m+n)N)`。

通常 `m,n` 都不大，瓶颈更多在批量评估或参数扫描。

## R10

数值稳定性与工程注意事项：
- 若矩阵病态或近奇异，`solve` 可能不稳定；可退化到 `lstsq`。
- 当分母在目标区间接近零时，误差可能暴涨，需要监控 `|Q(x)|`。
- 高阶Padé可能出现“杂散极零对（Froissart doublets）”，实务上常控制阶数并做鲁棒筛查。

## R11

典型应用：
- 从泰勒级数改进函数值计算（指数、对数、特殊函数）
- 常微分方程/偏微分方程离散中的稳定有理近似
- 控制与信号处理中传递函数近似
- 渐近展开重求和（resummation）

## R12

不太适合的场景：
- 函数在目标区间内存在复杂分支切线且局部级数信息不足。
- 数据非解析或噪声很大，仅靠少量级数系数很难稳定恢复。
- 强约束要求“绝对无极点”时，需要额外约束而非直接Padé。

## R13

参数建议：
- `m,n` 选择：先从平衡阶数（如 `[4/4]`,`[5/5]`）开始，再根据误差曲线调整。
- 如果更看重无穷远行为，可适当提高分母阶数。
- 检查项：
  - 系数匹配残差 `max|Qf-P|_{0..m+n}`
  - 评估区间上的 `min|Q(x)|`
  - 与同参数预算下泰勒截断的对照误差

## R14

与相近方法对比：
- 泰勒截断：
  - 优点：实现最简单。
  - 缺点：离展开点稍远时误差常快速变大。
- Padé逼近：
  - 优点：同等自由度下常更准、外推能力更强。
  - 风险：可能引入极点，需监控分母。
- Chebyshev/Remez最小最大逼近：
  - 直接优化区间误差上界，区间最优性更强；
  - 但构造流程通常比Padé更重。

## R18

`demo.py` 的源码级流程可拆为 8 步：
1. 生成 `exp(x)` 的幂级数系数 `c_0..c_8`（对应 `[4/4]`）。
2. 根据 `c_k + sum(q_j c_{k-j})=0`（`k=5..8`）构造线性系统 `Aq=b`。
3. 先用 `np.linalg.solve` 求 `q_1..q_4`，若失败则回退 `np.linalg.lstsq`。
4. 设 `q_0=1`，按卷积关系回代得到分子系数 `p_0..p_4`。
5. 用 Horner 方式评估 `P(x)` 与 `Q(x)`，计算 `R(x)=P/Q`。
6. 构造同样使用 9 个系数的 8 阶泰勒多项式作为基线对照。
7. 在 `[-4,4]` 网格上计算两者的 RMSE 与最大绝对误差，并检查 `min|Q(x)|`。
8. 打印关键结果与采样点表格，形成“构造-评估-验证”的最小闭环。
