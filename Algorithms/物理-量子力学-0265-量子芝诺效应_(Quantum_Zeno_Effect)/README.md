# 量子芝诺效应 (Quantum Zeno Effect)

- UID: `PHYS-0262`
- 学科: `物理`
- 分类: `量子力学`
- 源序号: `265`
- 目标目录: `Algorithms/物理-量子力学-0265-量子芝诺效应_(Quantum_Zeno_Effect)`

## R01

量子芝诺效应指：对量子系统进行足够频繁的测量，会抑制其从初态跃迁到其他态。  
本任务的算法目标是数值展示并验证该效应：在总演化时间 `T` 固定时，测量次数 `N` 增大，初态存活概率 `P_survive(N)` 上升并趋近 1。

## R02

采用最小两能级模型（Rabi 振荡）：

- 初态：`|0>`  
- 哈密顿量：`H = (Ω/2) σ_x`（设 `ħ=1`）  
- 无测量时演化：`|ψ(T)> = exp(-iHT)|0>`

无测量初态存活概率解析式：

`P_free(T) = |<0|exp(-iHT)|0>|^2 = cos^2(ΩT/2)`

## R03

数值任务输入输出定义：

- 输入：
1. `Ω`（Rabi 频率）
2. `T`（总演化时间）
3. `N`（总时间内等间隔测量次数）

- 输出：
1. 数值仿真存活概率 `P_sim(N)`（逐段演化 + 投影测量）
2. 解析存活概率 `P_ana(N) = [cos^2(ΩT/(2N))]^N`
3. 误差 `|P_sim - P_ana|`
4. 参考值 `P_free(T)`（无测量）

## R04

离散测量公式：

设每段时长 `τ = T/N`。每个测量周期：

1. 先做幺正演化 `U(τ)=exp(-iHτ)`
2. 再对投影算符 `Π0 = |0><0|` 做“是否仍在初态”测量

单步存活概率：

`p_step = ||Π0 U(τ)|0>||^2 = cos^2(Ωτ/2)`

`N` 次都存活（条件乘积）概率：

`P_survive(N) = (p_step)^N = [cos^2(ΩT/(2N))]^N`

## R05

MVP 算法步骤：

1. 构造泡利矩阵 `σ_x` 与哈密顿量 `H=(Ω/2)σ_x`。
2. 通过厄米本征分解得到 `U(τ)=exp(-iHτ)`。
3. 初始化 `ψ=|0>`，累计概率 `P=1`。
4. 循环 `N` 次：先 `ψ <- Uψ`，再求 `ψ_keep = Π0ψ`。
5. 计算单步存活概率 `p = <ψ_keep|ψ_keep>`，更新 `P <- P*p`。
6. 条件归一化 `ψ <- ψ_keep/sqrt(p)`，进入下一步。
7. 并行计算解析式 `P_ana(N)` 与无测量基线 `P_free(T)` 进行对比。

## R06

正确性依据：

1. 量子测量公设给出投影后概率 `p = ||Πψ||^2`，循环乘积即“连续成功存活”概率。
2. 对两能级 Rabi 模型，单步解析值 `p_step=cos^2(Ωτ/2)` 可直接推导。
3. 数值实现按“幺正演化 + 投影测量”逐步执行，与物理过程一一对应。
4. 当 `N→∞` 且 `T` 固定，`P_survive(N)→1`，构成量子芝诺效应的理论判据。

## R07

复杂度（单个 `N`）：

- 时间复杂度：`O(N)`  
每一步只有 `2x2` 复矩阵与向量运算，常数开销固定。

- 空间复杂度：`O(1)`  
只维护当前态向量和少量常量矩阵，不需要保存轨迹。

## R08

数值稳定性处理：

1. 概率使用实部 `max(real(p), 0)` 防止浮点负零误差。
2. 当 `p < 1e-15` 时提前中止并返回 0，避免归一化除零。
3. 对解析式使用 `log1p` 形式  
`N * log1p(-sin^2(ΩT/(2N)))` 再指数化，减少大 `N` 时精度损失。
4. 最终概率可用 `clip` 到 `[0,1]` 防止极小越界。

## R09

验证策略：

1. 解析一致性：检查 `|P_sim - P_ana|` 在浮点误差量级。
2. 芝诺趋势：固定 `Ω,T`，验证 `N` 增大时 `P_survive` 单调上升（至少总体上升）。
3. 基线对照：对比 `P_free(T)`，展示“频繁测量显著抑制跃迁”。
4. 极限检查：`N=1` 时应接近单次末端测量；`N` 很大时接近 1。

## R10

实现取舍：

1. 仅依赖 `numpy`，保持最小可运行栈。
2. 使用“本征分解重建矩阵指数”而非引入更重框架，便于源码追踪。
3. 默认参数写死在 `main()`，保证 `uv run python demo.py` 无交互执行。
4. 输出文本表格，便于自动校验脚本抓取。


默认参数：

- `T = π`（无测量时从 `|0>` 完全翻转到 `|1>`，`P_free≈0`）
- `N ∈ {1,2,5,10,20,50,100,200,500,1000}`

## R12

输出字段含义：

1. `N`：总测量次数。
2. `P_sim`：数值仿真的存活概率（逐步投影累乘）。
3. `P_analytic`：解析公式结果。
4. `abs_err`：二者绝对误差。
5. `short_time_approx`：`exp(-Ω^2 T^2 /(4N))` 的大 `N` 近似。

附加打印：

- `P_free(T)`：无测量存活概率基线。
- `max_abs_err`：全表最大误差。

## R13

预期数值现象：

1. `N=1` 时 `P_survive` 很小（本例接近 0）。
2. 随 `N` 增大，`P_survive` 明显上升并逐步逼近 1。
3. `P_sim` 与 `P_analytic` 在双精度内高度一致（常见 `1e-14` 量级）。
4. `short_time_approx` 在大 `N` 区间与精确值接近。

## R14

边界情况与失败模式：

1. `N <= 0`：物理上不合法，代码应抛出异常。
2. `Ω=0`：系统不演化，任意 `N` 下存活概率应为 1。
3. 极端大 `N`：可能出现累乘下溢，建议在工程版改为对数域累加。
4. 非投影测量（弱测量/POVM）不在本 MVP 范围，结论不可直接套用。

## R15

最小测试建议：

1. 解析回归：固定 `Ω=1,T=π`，对每个 `N` 检查 `|P_sim-P_ana| < 1e-10`。
2. 趋势回归：验证 `N=1000` 的存活概率显著高于 `N=1`。
3. 物理极限：`Ω=0` 时任意 `N` 返回 `1.0`。
4. 输入校验：`N=0` 时确认抛 `ValueError`。

## R18

`demo.py` 源码级算法流（不把线性代数调用当黑箱）：

1. 构造 `H=(Ω/2)σ_x`，并通过 `numpy.linalg.eigh(H)` 求厄米本征对 `(V, Λ)`。
2. 根据谱分解 `exp(-iHτ)=V·diag(exp(-iλ_kτ))·V†` 组装单步幺正算符 `U(τ)`。
3. 初始化 `ψ=|0>`、投影算符 `Π0=|0><0|`、累计概率 `P=1`。
4. 每个测量周期先做 `ψ <- Uψ`（薛定谔演化），再做 `ψ_keep <- Π0ψ`（投影筛选）。
5. 计算 `p = <ψ_keep|ψ_keep>` 作为该周期“存活”概率，乘入累计 `P <- P*p`。
6. 若 `p` 很小则提前终止；否则做条件归一化 `ψ <- ψ_keep/sqrt(p)`。
7. 循环 `N` 次得到 `P_sim(N)`，并与解析式 `P_ana(N)`、近似式和无测量基线比较。
