# B-H曲线 (B-H Curve)

- UID: `PHYS-0162`
- 学科: `物理`
- 分类: `静磁学`
- 源序号: `163`
- 目标目录: `Algorithms/物理-静磁学-0163-B-H曲线_(B-H_Curve)`

## R01

B-H 曲线描述的是磁性材料在外加磁场 `H` 作用下，磁感应强度 `B` 随 `H` 的响应关系。对于铁磁材料，`B-H` 关系通常呈现磁滞回线：上升、下降扫描路径不重合。

本目录给出一个可运行 MVP，用于演示以下关键现象：
- 饱和（`|H|` 很大时 `B` 趋于平台）；
- 剩磁（`H=0` 时 `B` 非零）；
- 矫顽力（`B=0` 时所需反向 `H`）；
- 回线面积（与磁滞损耗相关）。

## R02

本实现使用的基础物理关系：

- `B = mu0 * (H + M)`
- `mu0 = 4*pi*1e-7 H/m`

其中：
- `H`：外加磁场强度（A/m）；
- `M`：材料磁化强度（A/m）；
- `B`：磁感应强度（T）。

MVP 重点不是还原复杂微观机制，而是用一个透明、可解释的离散模型复现回线形状和关键指标。

## R03

任务定义（demo.py）：

- 输入：一条主回线激励轨迹 `H: -Hmax -> +Hmax -> -Hmax`；
- 输出：
  - 序列数据 `H, M, B`；
  - 指标 `Bsat, Br+, Br-, Hc+, Hc-, dB/dH|H~0, loop area`；
  - 幅值扫描汇总表（不同 `Hmax` 下的回线特征）。

脚本无交互输入，直接运行并打印报告，同时导出 `csv` 文件。

## R04

模型选择：分支目标 + 一阶松弛（非黑盒）。

1. 上升支（`dH>0`）目标磁化：
   - `M_target = Ms * tanh((H - Hc)/a)`
2. 下降支（`dH<0`）目标磁化：
   - `M_target = Ms * tanh((H + Hc)/a)`
3. 离散更新：
   - `M_{k+1} = M_k + beta * (M_target - M_k)`

该模型可解释地编码“方向依赖 + 路径记忆”，能够形成闭合磁滞回线。

## R05

参数物理意义：

- `Ms`：饱和磁化强度上限；
- `a`：`tanh` 过渡宽度，控制回线陡峭程度；
- `Hc`：上下分支的水平偏移，主导矫顽特征；
- `beta`：离散步内响应速度，越小表示动态滞后越强。

默认参数是“演示友好”而非某个具体牌号钢材标定值。

## R06

离散流程：

1. 构造 `H` 三角波扫描路径；
2. 从负饱和 `M(0)=-Ms` 出发；
3. 每一步根据 `dH` 判断分支方向；
4. 计算分支目标 `M_target`；
5. 用一阶松弛更新 `M`；
6. 代入 `B=mu0*(H+M)` 得到 `B`。

该流程只依赖 `numpy/pandas`，便于审查和迁移。

## R07

指标提取方法：

- `Br`（剩磁）：在 `H=0` 处对相邻离散点做线性插值，得到 `B(H=0)` 的正/负分支值；
- `Hc`（矫顽力）：在 `B=0` 处线性插值得到 `H(B=0)` 的正/负分支值；
- `loop_area_BdH`：数值积分 `|∮ B dH|`，作为损耗代理量；
- `mu_diff_origin`：在小 `|H|` 区间线性拟合斜率估计 `dB/dH`。

## R08

复杂度分析（`N` 为离散点数）：

- 回线仿真：`O(N)`；
- 过零插值扫描：`O(N)`；
- 线性拟合（近原点子集）：`O(N)`；
- 空间复杂度：`O(N)`。

幅值扫描若有 `K` 组激励，整体复杂度约 `O(KN)`。

## R09

默认实验设置：

- 主回线：`Hmax=1200 A/m`，每半周 700 点；
- 幅值扫描：`Hmax in {300, 500, 800, 1200, 1600} A/m`；
- 输出文件：
  - `bh_curve_samples.csv`（主回线逐点样本）；
  - `bh_sweep_summary.csv`（多幅值指标表）。

可以直接据此画图或做后处理。

## R10

`demo.py` 函数职责：

- `BHParams`：管理模型参数；
- `build_h_trajectory`：构造主回线激励；
- `target_magnetization`：按扫描方向给出分支目标；
- `simulate_bh_curve`：推进 `M` 与 `B`；
- `segment_crossings`：线性插值过零点；
- `slope_near_origin`：估计近原点微分磁导；
- `compute_metrics`：汇总单回线指标；
- `amplitude_sweep`：批量幅值实验；
- `main`：组织执行、输出、断言与保存文件。

## R12

输出解读建议：

- `Bsat_T`：趋于饱和时的 `|B|`；
- `Br_pos/Br_neg`：剩磁正负分支；
- `Hc_pos/Hc_neg`：矫顽力正负分支；
- `loop_area_BdH`：回线面积，通常随激励增强而增大；
- `sat_ratio_absM_over_Ms`：磁化是否接近饱和（接近 1 表示接近饱和）。

若 `Hmax` 较小，回线会偏窄，`Br/Hc` 与面积也会相对较小。

## R13

脚本内置最小正确性门禁：

1. 幅值扫描结果必须全部为有限数值（无 `NaN/Inf`）；
2. `sat_ratio_absM_over_Ms <= 1.02`，避免数值发散；
3. `loop_area_BdH >= 0`，确保面积定义一致；
4. 主回线文件与扫描文件必须正常写出。

这保证 MVP 结果可被自动化流程消费。

## R14

常见失效模式：

- 点数过少：回线折线化严重，过零插值不稳；
- `beta` 过大：系统过快贴合目标分支，动态滞后被削弱；
- `beta` 过小：回线过宽且不易稳定；
- `a` 过小：转折过于陡峭，离散噪声被放大；
- `Hmax` 过低：未进入饱和区，`Bsat` 参考意义降低。

## R15

工程实践建议：

- 若要拟合真实材料，优先用实测 `B-H` 数据反标 `Ms/a/Hc/beta`；
- 先固定 `Hc` 控制宽度，再调 `a` 控制陡峭度，最后调 `beta` 控制动态滞后；
- 报告中同时给出 `Br`、`Hc`、回线面积，避免单一指标误导；
- 可将 `csv` 输出接入统一可视化模板，以便批量比较。

## R16

与常见模型关系：

- 相比 Jiles-Atherton：本实现更轻量，参数更少，但微观机理刻画较弱；
- 相比 Preisach：本实现不构建继电器分布，计算和实现复杂度更低；
- 相比纯静态 `B(H)` 曲线拟合：本实现显式区分上/下支，能表达磁滞闭环。

因此它适合“教学验证/流程联调”的最小模型，而非高精度材料仿真终版。

## R18

`demo.py` 的源码级算法流程（8 步，非黑盒）如下：

1. `main` 创建 `BHParams`，并用 `build_h_trajectory` 生成主回线 `H` 扫描序列。  
2. `simulate_bh_curve` 从 `M(0)=-Ms` 起步，逐点读取 `H_k`。  
3. 每步根据 `dH = H_k - H_{k-1}` 判定当前处于上升支或下降支。  
4. `target_magnetization` 按分支公式计算 `M_target = Ms*tanh((H - direction*Hc)/a)`。  
5. 同步执行一阶松弛更新 `M_k = M_{k-1} + beta*(M_target - M_{k-1})`，形成路径记忆。  
6. 通过 `B_k = mu0*(H_k + M_k)` 显式得到 `B-H` 序列，构成闭合回线。  
7. `compute_metrics` 调用 `segment_crossings` 进行线性插值，提取 `Br` 与 `Hc`，并计算 `|∮B dH|` 和近原点斜率。  
8. `amplitude_sweep` 对多个 `Hmax` 重复上述流程，`main` 汇总打印并保存 `csv`，同时执行断言门禁。
