## R01

线性回归（Linear Regression）是监督学习里最经典的回归算法，用线性函数刻画输入特征与连续目标值之间的关系。

设样本为 `X in R^(n x d)`、目标为 `y in R^n`，模型写作：

`y_hat = Xw + b`

其中：
- `w in R^d` 是回归系数；
- `b` 是截距；
- `y_hat` 是预测值。

最常见目标是最小化残差平方和（RSS）或均方误差（MSE）：

`min_{w,b} ||y - (Xw + b)||_2^2`

## R02

历史脉络（简要）：
- 1805 年前后，高斯与勒让德分别发展了最小二乘思想。
- 19 世纪后，线性模型成为天文学、测量学和统计建模基础。
- 20 世纪中后期，回归推广到带正则化的岭回归、Lasso 等方法。
- 现代机器学习中，线性回归仍是可解释性和训练效率都很高的强基线。

## R03

线性回归解决的问题：给定一组有标签样本，学习一个可泛化的连续值映射。

输入输出：
- 输入：`X`（特征矩阵）、`y`（连续目标）。
- 输出：`w`、`b`，以及对新样本 `x_new` 的预测 `y_new = x_new^T w + b`。

核心价值：
- 训练快、推理快；
- 系数可解释；
- 适合作为复杂模型前的基线与误差下界参考。

## R04

线性回归（最小二乘）常见两类求解：

1. 闭式解（Normal Equation）
- 将偏置并入扩展矩阵 `X_bar = [1, X]`。
- 求解 `theta = (X_bar^T X_bar)^(-1) X_bar^T y`（或带微小正则的稳定版本）。

2. 迭代解（梯度下降）
- 计算损失 `L = (1/n) ||y - y_hat||_2^2`；
- 对参数求梯度并更新：`theta <- theta - eta * grad`；
- 迭代直到收敛。

本目录 MVP 采用闭式解手写实现，并用 sklearn 做对照。

## R05

复杂度分析：

- 闭式解：
  - 计算 `X_bar^T X_bar` 约 `O(n d^2)`；
  - 求解线性系统约 `O(d^3)`；
  - 总体约 `O(n d^2 + d^3)`。
- 预测：`O(n_test d)`。
- 空间：`O(n d)`（存储数据）与 `O(d^2)`（中间矩阵）。

当 `d` 很大时，闭式解会被 `d^3` 项拖慢，工程中常改用迭代法或稀疏优化器。

## R06

二维简化例子：

样本：`(x, y) = (1, 3), (2, 5), (3, 7)`，显然接近关系 `y = 2x + 1`。

令模型 `y_hat = wx + b`，最小化：

`sum_i (y_i - (w x_i + b))^2`

求解后可得 `w ≈ 2, b ≈ 1`，对 `x=4` 预测 `y_hat≈9`。这体现了线性回归把数据趋势压缩成“斜率 + 截距”的能力。

## R07

优势：
- 计算高效，适合快速建模。
- 参数可解释，便于业务沟通。
- 对中小规模数据常有稳定表现。

局限：
- 只能拟合线性关系（或需人工特征变换后线性）。
- 对异常值敏感（平方损失放大大残差）。
- 多重共线性会导致参数不稳定。
- 假设噪声结构较理想时统计性质更好。

前置知识与依赖：

- 微积分：最小二乘目标与梯度。
- 评估指标：MSE、MAE、RMSE、R²。

MVP 依赖：

## R09

适用场景：
- 需要可解释连续值预测，如价格估计、需求预测、打分回归。
- 关系近似线性或可通过特征工程线性化。
- 需要低延迟、可复现和易部署模型。

不适用或需谨慎：
- 强非线性关系且无特征变换；
- 离群点比例较高；
- 特征高度共线但又强依赖单个系数解释。

## R10

正确性检查清单：
1. 训练后 `coef_` 维度是否等于特征维度。
2. 预测输出长度是否与输入样本数一致。
3. 训练集上 MSE 是否显著小于目标方差。
4. 与成熟实现（如 sklearn）参数/指标是否接近。
5. 是否存在异常数值（`NaN`、`Inf`）。
6. 在可控合成数据上能否恢复已知真实系数。

## R11

数值稳定性注意点：
- 直接求逆 `(X^T X)^(-1)` 容易数值不稳定，优先 `solve/pinv`。
- 特征尺度差异大时建议标准化。
- 对病态矩阵可加入微小 `L2` 正则（Tikhonov 稳定化）。
- 用 `float64` 降低累积误差。
- 缺失值、极端值应先清洗，否则解会被污染。

## R12

真实工程中的成本与调参：
- 线性回归本身几乎无超参，主要成本在数据准备与特征工程。
- 若泛化不足，优先检查：
  - 特征是否漏掉关键变量；
  - 是否需加入交互项/多项式项；
  - 是否存在时间泄漏与分布漂移。
- 当解释性优先时，线性回归通常优于复杂黑盒。

## R13

理论性质（简述）：
- 在经典线性模型假设下，OLS 是无偏估计。
- 满足高斯-马尔可夫条件时，OLS 是线性无偏估计量中方差最小（BLUE）。
- 若噪声近似高斯，参数估计与置信区间推断更方便。
- 当假设不满足（异方差、自相关、共线性）时，需稳健标准误、加权回归或正则化改造。

## R14

常见失效模式与防护：
- 失效：异常值严重拉偏参数。  
  防护：异常值检测、稳健回归、分位数回归。
- 失效：训练集与线上分布偏移。  
  防护：按时间切分验证、监控漂移、周期重训。
- 失效：特征共线导致系数震荡。  
  防护：降维、去冗余、岭回归。
- 失效：误把相关当因果。  
  防护：明确目标是预测还是因果解释，必要时引入因果设计。

## R15

工程落地建议：
- 固定随机种子与数据切分，保证复现。
- 统一特征处理流水线（训练/推理一致）。
- 日志中记录模型系数、截距、指标与数据时间窗。
- 通过基线门槛（如 `R²`、RMSE）做自动化回归测试。
- 新模型上线前必须与当前线上模型做 A/B 或回放对照。

## R16

相关方法与演化：
- 岭回归（Ridge）：L2 正则，缓解共线性。
- Lasso：L1 正则，可做特征选择。
- Elastic Net：L1+L2 折中。
- 逻辑回归：线性模型用于分类概率建模。
- 广义线性模型（GLM）：通过链接函数扩展线性回归框架。

## R18

源码级流程拆解（`demo.py` + `scikit-learn`，7 步）：
1. `make_synthetic_linear_data` 生成 `X`、`y` 以及已知真值参数 `w_true/b_true`，并加入可控高斯噪声。  
2. `train_test_split_numpy` 用固定随机种子打乱索引并切分训练/测试集，确保可复现。  
3. `LinearRegressionNormalEq.fit` 将偏置并入扩展矩阵 `X_bar=[1, X]`，构建正规方程 `(X_bar^T X_bar + lambda I) theta = X_bar^T y`。  
4. `fit` 用 `np.linalg.solve` 直接求解参数向量 `theta`，再拆分为 `intercept_` 与 `coef_`；`predict` 按 `X @ coef_ + intercept_` 输出预测。  
5. `main` 同时训练 `sklearn.linear_model.LinearRegression` 作为基线，对两者分别计算训练集和测试集指标。  
6. 在当前 sklearn 实现（`sklearn/linear_model/_base.py`）中，`LinearRegression.fit` 会先做数据校验与预处理（含截距处理），随后调用 `numpy.linalg.lstsq`/稀疏求解器得到最小二乘解，再还原截距。  
7. `main` 汇总参数差距、指标差距与预测样例，并通过断言守卫确保 MVP 的正确性与可用性。
