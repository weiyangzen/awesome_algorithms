# 奇异值分解 (SVD)

- UID: `MATH-0055`
- 学科: `数学`
- 分类: `线性代数`
- 源序号: `55`
- 目标目录: `Algorithms/数学-线性代数-0055-奇异值分解_(SVD)`

## R01

奇异值分解（SVD）把任意实矩阵分解为：

`A = U Σ V^T`

- `U`：左奇异向量（列正交）；
- `Σ`：奇异值对角矩阵（非负且按降序）；
- `V`：右奇异向量（列正交）。

本条目实现“紧致 SVD（compact SVD）”的 MVP：只保留数值非零奇异值对应的分量，直接服务于重构、降维和伪逆。

## R02

历史定位（简要）：

- SVD 是现代数值线性代数最核心的分解之一；
- 在最小二乘、低秩近似、PCA、推荐系统中都属于基础工具；
- 工程实现通常基于双对角化 + QR 等流程；
- 本目录先做可审计的最小版本：通过 `A^T A` 的特征分解恢复 SVD 主体结构。

## R03

MVP 输入输出约定（无交互输入）：

- 输入（脚本内部固定构造）：
1. 一个 `6x4` 实矩阵 `A`（可复现，且设定为秩亏）；
2. 阈值 `tol`（数值秩判定）。
- 输出：
1. 估计秩 `rank`；
2. 紧致 SVD 因子 `U_r, s_r, V_r^T`；
3. 重构误差、正交误差；
4. 与 `numpy.linalg.svd/pinv` 的误差对照；
5. 自动校验通过标记 `All checks passed.`。

## R04

典型应用：

- 低秩近似与压缩（图像/信号/特征矩阵）；
- 伪逆计算与秩亏最小二乘；
- 特征提取（PCA 里的主方向与能量解释）；
- 条件数与数值稳定性评估（由奇异值谱直接给出）。

## R05

数学要点：

- `A^T A` 是对称半正定矩阵；
- 若 `A^T A v_i = λ_i v_i`，则奇异值 `σ_i = sqrt(λ_i)`；
- 对 `σ_i > 0`，左奇异向量可由 `u_i = A v_i / σ_i` 得到；
- 由此得到紧致分解：
  `A = U_r diag(s_r) V_r^T`；
- 截断到前 `k` 个奇异值可给出最佳秩 `k` 近似（Eckart-Young 定理意义下）。

## R06

本实现的核心流程：

1. 构造 Gram 矩阵 `G = A^T A`；
2. 对 `G` 做特征分解，得到 `(λ_i, v_i)`；
3. 排序并裁剪负零噪声（`λ_i <- max(λ_i, 0)`）；
4. 取 `σ_i = sqrt(λ_i)`；
5. 用阈值筛出数值非零奇异值形成紧致秩 `r`；
6. 计算 `U_r = A V_r / s_r`；
7. 计算重构与正交性误差，并做自动断言。

## R07

工程细节（MVP 里的关键判据）：

- 数值秩阈值：`σ_i > tol * max(σ_1, 1)`；
- 对由浮点误差产生的极小负特征值做截断，避免 `sqrt` 非法；
- 对 `rank=0`（全零矩阵）路径单独处理；
- 只输出紧致 SVD，避免对零奇异值做无意义除法；
- 使用 `pinv` 与 Moore-Penrose 恒等式做双重交叉校验。

## R08

时间复杂度（设 `A in R^{m x n}`，且以 `n <= m` 讨论）：

- 构造 `A^T A`：`O(m n^2)`；
- 对 `n x n` 对称矩阵做特征分解：`O(n^3)`；
- 恢复 `U_r = A V_r / s_r`：`O(m n r)`，最坏 `O(m n^2)`；
- 综合可视作 `O(m n^2 + n^3)`。

若 `m < n`，可对 `A A^T` 做同构流程，复杂度按较小维度立方项主导。

## R09

空间复杂度：

- 存储输入矩阵 `A`：`O(mn)`；
- Gram 矩阵 `G`：`O(n^2)`；
- 紧致因子 `U_r, s_r, V_r^T`：`O(mr + r + rn)`；
- 总体受 `A` 与 `G` 主导，约 `O(mn + n^2)`。

## R12

输出字段说明：

- `rank`：按阈值判定的数值秩；
- `singular_values_true`：构造矩阵时设置的目标奇异值；
- `singular_values_est`：本实现估计的非零奇异值；
- `singular_values_ref`：NumPy `svd` 参考值（含零奇异值）；
- `reconstruction_error`：`||A - U_r Σ_r V_r^T||_F`；
- `u_orthogonality_error`：`||U_r^T U_r - I||_F`；
- `v_orthogonality_error`：`||V_r^T V_r - I||_F`（等价写法）；
- `pinv_rel_error`：与 `numpy.linalg.pinv` 的相对误差；
- `mp_identity_error`：`||A A^+ A - A||_F`；
- `trunc_err_rank1/rank2`：截断低秩近似误差，用于验证误差随秩提升不增。

## R13

鲁棒性与边界处理：

- 输入校验：必须是二维有限矩阵，`tol > 0`；
- 特征值负零清理：`eigvals = max(eigvals, 0)`；
- `rank=0` 分支返回空因子，避免除零；
- 所有关键检查失败时主动抛异常，而不是静默继续；
- 通过 Moore-Penrose 恒等式和参考伪逆双校验，避免“看起来能跑但数学性质错误”。

## R14

当前局限：

- 采用 `A^T A` 路径，理论上会放大条件数（相比工业级双对角化 SVD 稳定性略弱）；
- 主要面向中小规模教学/验证，不追求超大规模性能；
- 未实现稀疏矩阵优化或随机化 SVD；
- 仅处理实数矩阵场景（复数版本可扩展但本 MVP 未覆盖）。

## R15

可扩展方向：

- 增加随机化 SVD（Randomized SVD）用于大规模近似；
- 增加截断能量比选择策略（如累计能量达 99% 自动选秩）；
- 支持复数矩阵（共轭转置版本）；
- 增加与 `scipy.sparse.linalg.svds` 的稀疏对比基准；
- 增加基于 SVD 的最小二乘与降噪应用示例。

## R16

最小测试建议：

- 正确性：与 `numpy.linalg.svd` 的奇异值对比；
- 重构性：`||A - U_r Σ_r V_r^T||_F` 足够小；
- 正交性：`U_r^T U_r` 与 `V_r^T V_r` 接近单位阵；
- 伪逆性：`A A^+ A ≈ A`；
- 单调性：rank-2 截断误差不应大于 rank-1 截断误差；
- 边界：全零矩阵、满秩矩阵、近秩亏矩阵。

## R17

方案对比：

- `numpy.linalg.svd`：最直接且稳健，但主算法细节不可见；
- 本方案（`A^T A` + `eigh`）：流程透明、实现短小，便于教学与审计；
- 幂法/子空间法：更适合只取前几个奇异值的大规模场景；
- 工业 SVD（双对角化 + QR）：稳定性和性能更强，但实现复杂度明显更高。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `build_demo_matrix()` 用可复现随机正交矩阵构造 `6x4` 测试矩阵，并预置目标奇异值。  
2. `compact_svd_from_gram()` 先做输入检查，再构造 `G = A^T A`。  
3. 对 `G` 调用 `numpy.linalg.eigh` 得到全部特征对，按特征值降序排列。  
4. 将特征值裁剪到非负后开方，得到候选奇异值序列 `σ`。  
5. 用阈值判定数值秩 `r`，筛出 `V_r` 与 `s_r`（即非零奇异值子空间）。  
6. 按 `u_i = A v_i / σ_i` 逐列恢复 `U_r`，形成紧致分解 `A ≈ U_r diag(s_r) V_r^T`。  
7. `run_checks()` 进行重构误差、正交性、伪逆误差、Moore-Penrose 恒等式和截断误差单调性验证。  
8. `main()` 打印估计值与 `numpy.linalg.svd/pinv` 参考差异，全部断言通过后输出 `All checks passed.`。  

说明：第三方库在此仅提供“对称特征分解与验算算子”；SVD 的主控制流程、秩判定、向量恢复、校验逻辑均在源码中显式展开，不是黑箱一键调用。
