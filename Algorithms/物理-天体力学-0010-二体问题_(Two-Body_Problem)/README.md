# 二体问题 (Two-Body Problem)

- UID: `PHYS-0010`
- 学科: `物理`
- 分类: `天体力学`
- 源序号: `10`
- 目标目录: `Algorithms/物理-天体力学-0010-二体问题_(Two-Body_Problem)`

## R01

二体问题研究两个只受相互万有引力作用的天体运动。经典结论是：在质心系下可化为一个“约化质量粒子”绕中心势 `-mu/r` 的运动，其中 `mu = G(m1+m2)`。轨道形状是圆锥曲线（椭圆/抛物线/双曲线），本条目聚焦有界椭圆轨道。

## R02

该问题是天体力学、航天轨道设计与长期积分算法的基础模型。很多复杂模型（摄动、N 体、相对论修正）都以二体问题为零阶基线。若最小模型都不能保持能量、角动量等守恒量，复杂模型会更不可信。

## R03

本目录 MVP 的目标不是做完整轨道力学库，而是打通最小闭环：

- 给定 `m1,m2,a,e`，构造近日点初值；
- 用数值 ODE 积分一整周期二维轨道；
- 验证守恒量（比机械能、比角动量、偏心率向量）；
- 用圆锥曲线公式 `r=p/(1+e cos(theta-omega))` 拟合轨道并打分；
- 输出可读表格和断言，形成非交互回归脚本。

## R04

核心公式：

1. 相对运动方程：
   `r'' = -mu * r / |r|^3,  mu=G(m1+m2)`。
2. 近日点初值（椭圆）：
   `r_p = a(1-e)`，
   `v_p = sqrt(mu(1+e)/(a(1-e)))`。
3. 开普勒周期：
   `T = 2*pi*sqrt(a^3/mu)`。
4. 比机械能：
   `eps = v^2/2 - mu/r`，理论常值 `-mu/(2a)`。
5. 比角动量（二维 z 分量）：
   `h = x*vy - y*vx`，理论常值 `sqrt(mu*a*(1-e^2))`。
6. 偏心率向量：
   `e_vec = ((v^2-mu/r)r - (r·v)v)/mu`，其模长为轨道偏心率。
7. 圆锥曲线极坐标：
   `r(theta)=p/(1+e*cos(theta-omega))`，`p=h^2/mu`。

## R05

算法思路（高层）：

1. 由 `(a,e,mu)` 生成近日点初值 `[x,y,vx,vy]`；
2. 以 `DOP853` 积分一个周期得到离散轨道；
3. 在全轨迹上计算 `eps,h,e_vec` 等不变量；
4. 用初始 `h,e_vec` 反推 `p,omega`，构造解析半径 `r_model`；
5. 计算 `MAE/R^2`、闭轨误差、守恒量漂移；
6. 断言阈值，失败即抛异常。

## R06

设采样点数为 `n`：

- ODE 积分在固定 `t_eval` 下输出 `n` 个状态，主导复杂度近似 `O(n)`；
- 守恒量计算、拟合残差、表格组装均为 `O(n)`；
- 空间复杂度 `O(n)`（保存整条轨道用于诊断）。

因此该 MVP 是线性规模，适合做基线回归。

## R07

正确性直觉：

- 中心力 `~1/r^2` 必然给出角动量守恒；
- 时间不显含的势能系统对应机械能守恒；
- 二体椭圆轨道应满足圆锥曲线关系；
- 从近日点起步积分一个周期，末状态应回到初状态附近。

脚本正是围绕这四个“物理不变量/几何不变量”做联合校验。

## R08

数值稳定策略：

- 使用高阶显式积分器 `DOP853`，并设置严格容差 `rtol=1e-10, atol=1e-12`；
- 避免奇点：若 `r` 接近 0 直接抛错；
- 仅在椭圆区间 `0<=e<1` 构造初值；
- 同时检查“相对漂移”和“理论常值误差”，避免只看单一指标。

- 演示 B：守恒量漂移评估（能量、角动量、偏心率）；
- 演示 C：圆锥曲线半径拟合及 `R^2/MAE` 评估；

## R11

输出关键指标：

- `rel_energy_drift`：比机械能相对漂移最大值；
- `rel_h_drift`：比角动量相对漂移最大值；
- `max_abs_ecc_drift`：偏心率绝对漂移最大值；
- `radius_mae`、`radius_r2`：轨道半径拟合质量；
- `closure_pos_error`、`closure_vel_error`：一周期闭轨误差；
- `com_max_norm`、`momentum_max_norm`：质心与总动量守恒误差；
- `torch_energy_span`：用 PyTorch 统计的能量跨度。

## R12

默认参数（可在 `OrbitConfig` 调整）：

- `m1=1.0`，`m2=0.2`，`G=1.0`（无量纲演示）；
- `a=2.0`，`e=0.35`（中等偏心率椭圆）；
- `samples=4001`（一周期离散点数）。

调参建议：

- 若增大偏心率接近 1，应提高采样密度；
- 若加入更长仿真窗口，建议同步收紧误差阈值并检查漂移积累；
- 若替换为双曲线轨道，需要单独修改初值构造与通过条件。

## R14

常见失效模式：

1. 参数非法：`e>=1` 但仍按椭圆公式初始化；
2. 数值容差过松，导致长周期守恒量漂移明显；
3. 轨道近碰撞但仍使用固定阈值，出现加速度奇异；
4. 只看轨迹形状不看不变量，可能掩盖积分器误差；
5. 只看某一个指标（如 `R^2`）导致误判。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `OrbitConfig` 定义质量、轨道参数和采样数，并提供 `mu` 与周期 `T` 的计算属性。  
2. `make_periapsis_initial_state` 根据椭圆解析公式构造近日点状态 `r_p, v_p`，形成积分初值。  
3. `integrate_relative_orbit` 调用 `solve_ivp(DOP853)` 对 `two_body_relative_rhs` 做一周期积分，得到离散 `state(t)`。  
4. `compute_invariants` 在每个时间点显式计算 `r, eps, h, e_vec, true_anomaly`，不依赖黑盒轨道元素库。  
5. `run_two_body_demo` 取初始 `h` 与 `e_vec` 计算 `p, omega`，再用 `conic_radius_model` 构建解析半径 `r_model(theta)`。  
6. `summarize_metrics` 计算守恒量漂移、理论常值误差、闭轨误差，并用 `sklearn` 得到 `MAE` 与 `R^2`。  
7. 同一函数内通过 `project_to_barycentric` 校验质心和总动量，另用 `torch` 统计能量跨度，形成多工具交叉诊断。  
8. `main` 打印全部指标和 `pandas` 采样表，再由断言门控通过/失败，确保 `uv run python demo.py` 可直接用于验证。  

第三方库没有被当作黑盒算法：`solve_ivp` 只负责通用 ODE 积分，二体动力学、守恒量定义、圆锥拟合与验证逻辑都在源码中逐步展开。
