# 最小作用量原理 (Principle of Least Action)

- UID: `PHYS-0008`
- 学科: `物理`
- 分类: `分析力学`
- 源序号: `8`
- 目标目录: `Algorithms/物理-分析力学-0008-最小作用量原理_(Principle_of_Least_Action)`

## R01

问题定义：给定固定起点和终点，寻找一条使作用量泛函最小（更准确是“驻值”，在本参数下为极小）的轨迹。

本条目使用一维简谐振子边值问题作为最小可运行载体：
- 已知边界 `x(0)=x_start`、`x(T)=x_end`；
- 在离散时间网格上把“路径”视作优化变量；
- 直接最小化离散作用量 `S[x] = ∫(T-V)dt`；
- 与解析解对比，并做扰动路径的作用量对照验证。

## R02

理论背景：最小作用量原理是分析力学统一表述之一。

核心思想：
- 对真实运动轨迹，作用量对一阶变分满足 `delta S = 0`；
- 这与欧拉-拉格朗日方程等价；
- 在满足一定条件（如时间跨度较短、问题凸性较好）时，驻值轨迹常为局部极小。

对数值实现而言，它可转写为“函数空间优化”问题，然后通过离散化变成有限维优化。

## R03

本 MVP 的计算任务：
1. 构建谐振子拉格朗日量 `L = 0.5*m*v^2 - 0.5*k*x^2`；
2. 对固定端点路径做均匀网格离散，得到离散作用量 `S_d`；
3. 使用 `L-BFGS-B` 优化内部节点（端点固定）；
4. 计算离散欧拉-拉格朗日残差（即 `S_d` 对内部节点梯度）；
5. 与解析边值解比较轨迹误差；
6. 生成随机平滑扰动，验证扰动后的作用量增量为正。

## R04

建模假设（MVP 级别）：
- 单自由度线性系统（简谐振子）；
- 无阻尼、无外驱动，保守系统；
- 质量 `m` 与弹簧刚度 `k` 为常数；
- 固定时间端点问题（`t=0` 到 `t=T`）。

该设定旨在突出“作用量优化”本身，不覆盖非线性势场、阻尼耗散、约束多体耦合等扩展情形。

## R05

核心方程（`demo.py` 已实现）：

- 拉格朗日量：`L(x,v)=0.5*m*v^2 - 0.5*k*x^2`
- 作用量：`S[x]=∫_0^T L dt`
- 欧拉-拉格朗日：`d/dt(∂L/∂v)-∂L/∂x=0`
- 对本系统：`m*x_ddot + k*x = 0`

离散后（等间隔 `dt`）脚本使用：
`S_d = Σ_i [0.5*m*(dx_i^2/dt) - 0.25*k*dt*(x_i^2 + x_{i+1}^2)]`
其中 `dx_i = x_{i+1} - x_i`。

## R06

数值算法流程：
1. 时间网格 `t_0...t_N` 固定，端点 `x_0, x_N` 固定；
2. 仅把 `x_1...x_{N-1}` 作为优化变量；
3. 在 `action_and_gradient` 中同时计算 `S_d` 与解析梯度；
4. 调用 `scipy.optimize.minimize(method="L-BFGS-B")` 执行迭代；
5. 收敛后得到最优路径 `x_opt`；
6. 计算梯度范数（离散 E-L 残差）、解析解误差、扰动作用量增量统计。

## R07

复杂度分析（设网格点数为 `N`）：
- 每次函数/梯度评估复杂度：`O(N)`；
- L-BFGS-B 若迭代 `K` 次，总体 `O(KN)`；
- 空间复杂度：`O(N)`（路径、梯度、诊断数组）。

本实测配置 `N=180`，迭代 `nit=573`，运行耗时约秒级。

## R08

数值稳定性策略：
- 显式提供解析梯度，避免纯数值差分带来的噪声；
- 使用平滑扰动而非高频白噪声，减少离散伪振荡误判；
- 同时检查三组指标：
  - `residual_inf / residual_rms`（方程驻值性）；
  - `rmse_vs_exact`（与解析解一致性）；
  - `delta_action_min` 与 `positive_delta_ratio`（局部极小性质）。

## R09

适用场景：
- 分析力学教学中展示“从作用量到轨迹”的数值构造；
- 作为离散变分法/最优控制离散化的最小模板；
- 在已知边值条件下对经典系统做可解释验证。

不适用场景：
- 长时间跨越多个共轭点的情形（驻值不一定是最小值）；
- 强非线性、非光滑势能、碰撞/切换动力学；
- 高维约束系统（需更系统的稀疏结构和约束处理）。

## R10

正确性检查点（脚本中已落地）：
1. 优化器必须 `success=True`；
2. `residual_inf <= 1e-4`；
3. `rmse_vs_exact <= 5e-4`；
4. `positive_delta_ratio >= 99%`；
5. `delta_action_min > 0`。

这些检查保证“收敛 + 方程一致 + 物理/变分意义成立”。

## R13

理论与验证边界：
- 原理层面是“驻值原理”，并非所有系统/时间区间都严格“最小”；
- 本示例在当前参数下通过扰动实验观察到极小性质；
- 该验证是数值证据，不是一般性数学证明。

因此本条目结论应理解为：在给定离散模型和参数范围内，数值结果与最小作用量预期一致。

## R14

常见失败模式与修复：
- 失败：`sin(omega*T)` 接近 0，解析边值解公式数值不稳定。
  - 修复：调整 `t_end` 或 `k/m`，避开退化点。
- 失败：网格过粗导致离散误差偏大。
  - 修复：增加 `num_points`。
- 失败：优化收敛慢或早停。
  - 修复：提高 `maxiter`，适当收紧 `ftol/gtol`。
- 失败：扰动尺度过大，跳出局部邻域导致“并非全局最小”误解。
  - 修复：降低 `perturb_scale` 并使用平滑扰动。

## R15

工程实践建议：
- 对变分问题应优先实现“目标+梯度”成对接口，避免黑盒数值梯度；
- 验证指标应覆盖“方程残差 + 基准解误差 + 扰动鲁棒性”；
- 当扩展到多自由度时，可先保持同一离散模板，再引入稀疏 Jacobian/Hessian 优化。

## R16

方法脉络与扩展方向：
- 将 1D 谐振子扩展到多维耦合二次型系统；
- 引入非线性势能，观察局部极小/鞍点转变；
- 引入约束后可转为拉格朗日乘子或增广拉格朗日框架；
- 对长时保结构需求，可进一步采用离散变分积分器（variational integrator）。

## R18

`demo.py` 源码级算法流拆解（8 步，非黑盒）：
1. `ActionParams` 固化物理参数、网格规模和优化配置，保证实验可复现。
2. `build_path` 把“固定端点 + 可变内部节点”拼接为完整离散路径。
3. `action_and_gradient` 逐区间计算离散作用量 `S_d`，并显式组装每个节点对 `S_d` 的梯度贡献。
4. `optimize_path` 用线性插值初始化内部路径，向 `minimize(..., method="L-BFGS-B")` 提供目标函数和解析梯度。
5. L-BFGS-B 在每次迭代中基于历史 `(s,y)` 向量构造近似逆 Hessian 乘子，得到下降方向并做线搜索更新。
6. 收敛后读取最优内部节点，重建 `x_opt`，并以梯度范数作为离散欧拉-拉格朗日残差指标。
7. `analytic_harmonic_path` 计算同边界条件下的解析解 `x_exact`，输出 `rmse/max_abs_err` 进行外部基准校验。
8. 对 `x_opt` 施加多次平滑随机扰动并重算作用量，统计 `delta_action` 分布，验证局部邻域内“作用量上升”性质。

该实现没有调用任何“现成最小作用量黑盒求解器”；作用量离散、梯度构造、优化与验证链路都可逐行追踪。
