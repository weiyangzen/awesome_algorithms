# Romberg积分

- UID: `MATH-0131`
- 学科: `数学`
- 分类: `数值分析`
- 源序号: `131`
- 目标目录: `Algorithms/数学-数值分析-0131-Romberg积分`

## R01

本条目实现 Romberg 积分（Romberg Integration）的最小可运行版本：
- 基于复化梯形积分逐层细分网格；
- 用 Richardson 外推构造高阶近似；
- 在 `demo.py` 中输出 Romberg 三角表、误差与收敛轨迹。

MVP 目标是把“从低阶数值积分到高阶外推”的过程写清楚并可复现实验结果。

## R02

问题定义（定积分数值近似）：
- 输入：
  - 被积函数 `f(x)`；
  - 积分区间 `[a, b]`；
  - 最大层数 `max_level`（细分到 `2^max_level` 子区间）；
  - 收敛阈值 `tol`。
- 输出：
  - 积分近似值 `I ~= \int_a^b f(x) dx`；
  - Romberg 三角表 `R[k][j]`；
  - 终止层号和函数评估次数。

## R03

核心公式：

1) 复化梯形递推（第 `k` 层）
- 步长 `h_k = (b-a)/2^k`。
- `R[k][0] = 0.5 * R[k-1][0] + h_k * sum_{i=1}^{2^{k-1}} f(a + (2i-1)h_k)`。

2) Richardson 外推（消去偶次误差项）
- `R[k][j] = R[k][j-1] + (R[k][j-1] - R[k-1][j-1]) / (4^j - 1)`，`j=1..k`。

3) 对角线项 `R[k][k]`
- 一般比单纯梯形法精度高，是 Romberg 方法的最终估计序列。

## R04

算法流程：
1. 校验输入参数合法性（区间有限、`max_level >= 1`、`tol > 0`）。
2. 计算 `R[0][0]`（单区间梯形）。
3. 对 `k = 1..max_level`：
   - 只新增奇数节点函数值，更新 `R[k][0]`；
   - 沿当前行执行 Richardson 外推，得到 `R[k][1..k]`。
4. 每层比较对角线差值 `|R[k][k] - R[k-1][k-1]|` 作为收敛指标。
5. 达到阈值则提前停止，返回当前对角线近似和完整表格。

## R05

核心数据结构：
- `RombergResult`（`dataclass`）：
  - `estimate: float` 最终近似；
  - `table: list[list[float]]` 三角表；
  - `levels_used: int` 实际使用层数；
  - `eval_count: int` 函数评估次数。
- `cases: list[tuple[str, callable, float, float, float]]`：
  - 演示用函数名、函数对象、区间与解析真值。

该结构简单透明，便于后续替换成向量化或并行版本。

## R06

正确性要点：
- `R[k][0]` 来自复化梯形递推，等价于在每层把步长减半；
- `R[k][j]` 的构造严格遵循 Richardson 外推，递推系数 `4^j-1` 对应误差阶消去；
- 若被积函数在区间内足够光滑，`R[k][k]` 通常快速收敛到真积分；
- `demo.py` 用多个有解析解的样例验证误差数量级。

## R07

复杂度分析（`L = levels_used`）：
- 时间复杂度：
  - 函数评估点总数约为 `1 + 2^L`，故总体 `O(2^L)`；
  - 外推表构造额外 `O(L^2)`，通常远小于函数评估成本。
- 空间复杂度：
  - 存完整 Romberg 表为 `O(L^2)`；
  - 若只保留两行可降为 `O(L)`（本 MVP 保留全表以便展示）。

## R08

边界与异常处理：
- `a == b`：直接返回积分 `0.0`（零长度区间）；
- 非有限区间端点（`nan/inf`）抛出 `ValueError`；
- `max_level < 1` 或 `tol <= 0` 抛出 `ValueError`；
- 若函数值在迭代中出现非有限数，抛出 `RuntimeError`。

## R09

MVP 取舍：
- 采用 Python 标准库实现，不依赖外部黑盒积分器；
- 保留完整 Romberg 三角表，优先可解释性而非极限性能；
- 示例选择“可解析真值”函数，避免把误差验证也交给外部数值库。

## R10

`demo.py` 主要函数职责：
- `safe_eval`：统一函数值有限性检查；
- `romberg_integral`：构造梯形递推 + Richardson 外推并返回结果对象；
- `relative_error`：统一相对误差计算；
- `print_romberg_table`：按三角形式打印表格；
- `run_case`：执行单个样例并打印误差/收敛信息；
- `main`：批量运行多个样例，无交互输入。

## R11

运行方式：

```bash
cd Algorithms/数学-数值分析-0131-Romberg积分
python3 demo.py
```

脚本会自动执行全部样例，输出每个样例的 Romberg 表和误差统计。

## R12

输出字段解读：
- `estimate`：Romberg 最终估计 `R[L][L]`；
- `exact`：解析真值；
- `abs_error`：绝对误差；
- `rel_error`：相对误差；
- `levels_used`：实际迭代层数；
- `eval_count`：累计函数评估次数；
- `diagonal`：`R[0][0], R[1][1], ..., R[L][L]` 收敛轨迹。

## R13

建议最小测试集（已在 demo 中覆盖）：
- `sin(x)` on `[0, pi]`，真值 `2`；
- `exp(x)` on `[0, 1]`，真值 `e-1`；
- `1/(1+x^2)` on `[0, 1]`，真值 `pi/4`；
- `sqrt(x)` on `[0, 1]`，真值 `2/3`（端点导数奇异，检验鲁棒性）；
- 常数函数与 `a == b` 的退化区间测试。

## R14

可调参数：
- `max_level`：最大细分层数，默认示例设为 `7`；
- `tol`：对角线收敛阈值，默认 `1e-12`；
- `max_rows`：表格展示时最大打印行数。

工程建议：
- 先用较大 `max_level` 观察收敛，再结合目标精度下调参数；
- 对尖峰/振荡函数可适当增大 `max_level`，并监控函数评估成本。

## R15

方法对比：
- 相比单次梯形/辛普森：
  - Romberg 通过外推系统性提升阶数；
  - 在光滑函数上常用更少网格点达到高精度。
- 相比自适应高斯积分：
  - Romberg 实现更直观，便于教学与调试；
  - 对局部奇异和高振荡函数通常不如专门自适应策略稳健。

## R16

应用场景：
- 科学计算课程中的积分误差阶演示；
- 需要可解释、高精度标量积分的工程模块；
- 作为更复杂积分器（自适应分段、并行积分）的基础内核。

## R18

源码级算法流（对应 `demo.py`，8 步）：
1. `main` 定义若干“有解析解”的积分测试用例和统一参数。  
2. 每个用例进入 `run_case`，调用 `romberg_integral` 计算近似积分。  
3. `romberg_integral` 先通过 `safe_eval` 计算端点函数值，构造 `R[0][0]`。  
4. 对每层 `k`，只在新出现的奇数网格点做函数评估并累加，更新 `R[k][0]`。  
5. 在同一层内按 `j=1..k` 执行 Richardson 外推，得到 `R[k][j]`。  
6. 记录对角线误差指标 `|R[k][k]-R[k-1][k-1]|`，满足阈值即提前停止。  
7. 返回 `RombergResult`（最终估计、三角表、层数、评估次数）。  
8. `run_case` 计算绝对/相对误差，打印对角线收敛序列和三角表。
