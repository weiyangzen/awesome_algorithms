## R01

中微子振荡描述的是“味本征态”和“质量本征态”不一致导致的传播相位干涉现象。
在两味近似下，`ν_e` 与 `ν_μ` 的转化概率可写成：

`P(ν_e -> ν_μ) = sin^2(2θ) * sin^2(1.267 * Δm^2 * L / E)`

其中：

- `θ`：混合角
- `Δm^2`：质量平方差（`eV^2`）
- `L`：基线长度（`km`）
- `E`：中微子能量（`GeV`）

本条目实现一个可运行 MVP：

1. 计算两味真空振荡概率；
2. 生成带噪观测；
3. 用二维网格搜索反演 `θ` 与 `Δm^2`。

## R02

典型应用场景：

- 长基线实验的振荡谱快速估算与教学验证；
- 在复杂三味/物质效应模型前，先做两味近似 sanity check；
- 用合成数据演示参数反演与 `χ²` 拟合流程；
- 为后续 PMNS 全模型、系统误差建模提供最小基线。

## R03

`demo.py` 采用的数学模型：

- 外观概率：
  `P_app(E) = sin^2(2θ) * sin^2(1.267 * Δm^2 * L / E)`
- 存活概率：
  `P_surv(E) = 1 - P_app(E)`
- 观测模型（高斯噪声）：
  `y_obs(E_i) = P_app(E_i; θ_true, Δm^2_true) + ε_i`
- 拟合目标：
  `χ²(θ, Δm^2) = Σ_i ((y_obs,i - P_app(E_i; θ, Δm^2)) / σ)^2`

通过离散网格最小化 `χ²` 获得参数估计。

## R04

直观理解：

- 中微子产生时按“味”被探测，但传播按“质量”累积相位；
- 不同质量态相位差随 `L/E` 增长，导致转化概率振荡；
- `θ` 决定振荡振幅（振幅上限由 `sin^2(2θ)` 决定）；
- `Δm^2` 决定振荡频率（`L/E` 轴上波峰波谷疏密）。

## R05

正确性要点：

1. 概率边界：`0 <= P_app <= 1`，`0 <= P_surv <= 1`。
2. 两味幺正约束：`P_surv + P_app = 1`。
3. `L=0` 时无传播相位，必须有 `P_app=0, P_surv=1`。
4. 当 `θ=0` 时无混合，任何 `L/E` 下 `P_app=0`。
5. 反演结果应接近真值，且 `χ²` 在最优点附近形成稳定谷底。

`demo.py` 会对以上性质做程序化检查，失败即抛出异常。

## R06

复杂度分析（`Nθ` 个角度网格点，`NΔ` 个质量差网格点，`NE` 个能量点）：

- 参数网格预测计算：`O(Nθ * NΔ * NE)`
- `χ²` 归约：`O(Nθ * NΔ * NE)`
- 存储 `χ²` 面：`O(Nθ * NΔ)`

MVP 选择 `Nθ=241`、`NΔ=241`、`NE=14`，总计算量约 `8e5` 量级，CPU 可快速完成。

## R07

标准实现步骤：

1. 构建能量采样网格 `E` 与实验基线 `L`。
2. 用真值参数生成无噪声 `P_app(E)`。
3. 注入高斯噪声并裁剪到 `[0,1]` 形成观测。
4. 构造 `θ` 与 `Δm^2` 的二维离散候选网格。
5. 对所有候选参数并行计算预测概率张量。
6. 计算每个候选点的 `χ²`。
7. 取 `argmin(χ²)` 得到最优估计。
8. 输出拟合质量、关键能点对比与物理一致性指标。

## R08

`demo.py` 的 MVP 设计：

- 依赖：`numpy`（向量化计算 + 随机噪声 + 线性代数基本操作）；
- 不依赖黑箱优化器：参数反演使用显式网格搜索；
- 输入：无命令行参数，无交互输入；
- 输出：
  - 真值/估计参数对比；
  - `χ²` 与 RMSE；
  - 代表能量点处的观测与预测表。

运行方式：`uv run python demo.py`。

## R10

测试策略：

- 物理约束测试：概率边界、幺正和 `L=0` 极限；
- 数值稳定性测试：`E` 不可非正，`sigma` 不可非正；
- 反演正确性测试：估计值与真值误差应在合理范围；
- 拟合质量测试：最优 `χ² / dof` 接近 `1` 量级（合成数据条件下）。

## R11

边界条件与异常处理：

- `e_gev` 含非正值：抛 `ValueError`；
- `sigma <= 0`：抛 `ValueError`；
- `theta_grid_rad` 或 `delta_m2_grid` 为空：抛 `ValueError`；
- 网格形状不匹配或广播失败：由显式 shape 检查提前报错。

物理适用边界：本 MVP 只覆盖两味、真空、相对论近似，不含三味 CP 相位与物质效应（MSW）。

## R12

与相关模型关系：

- 两味模型是三味 PMNS 振荡在特定能区/通道下的近似截断；
- 物质中的传播需要在有效哈密顿量中加入势项（MSW）；
- 实验分析常在本模型基础上叠加通量、截面、探测效率和系统误差；
- 本条目聚焦“传播核 + 参数反演”的最小闭环。

## R14

工程实现注意点：

- `L/E` 项在 `E` 较小时变化快，需保持 `E>0` 并进行输入校验；
- 网格搜索采用三维广播，避免 Python 双重循环的性能损失；
- 噪声后裁剪 `[0,1]` 可避免非物理观测概率；
- 使用固定随机种子保证结果可复现。

## R15

最小示例解释：

- 输出中的 `best_theta_deg`、`best_delta_m2` 为反演参数；
- `chi2_min / dof` 用于判断拟合与噪声规模是否一致；
- 表格中 `p_true / p_obs / p_fit` 分别代表真值、观测、拟合预测；
- 若模型与反演流程正确，`p_fit` 会沿能谱追随 `p_obs`，并接近 `p_true` 的振荡形状。

## R18

源码级算法流程拆解（对应 `demo.py`，共 8 步）：

1. **构建振荡核函数**  
   `two_flavor_probabilities` 直接实现两味真空公式，输出 `P_surv` 与 `P_app`。

2. **做输入与物理约束校验**  
   `validate_physical_constraints` 检查 `E>0`、概率边界、幺正关系和 `L=0` 极限。

3. **生成合成观测数据**  
   `build_synthetic_dataset` 先算 `p_true`，再加入 `N(0, sigma)` 噪声并裁剪到 `[0,1]`。

4. **离散化参数空间**  
   在 `main` 中构建 `theta_grid_rad` 与 `delta_m2_grid`，形成二维候选集合。

5. **广播计算整张预测概率张量**  
   在 `chi2_grid_search` 中把 `θ`、`Δm²`、`E` 扩维后一次性得到 `P_app(θ,Δm²,E)`。

6. **逐候选点评估代价函数**  
   用 `(obs - pred)/sigma` 计算残差并在能量轴求和，得到 `χ²(θ,Δm²)` 面。

7. **定位最优参数**  
   对 `χ²` 面做 `argmin`，回读网格坐标作为最优 `θ` 和 `Δm²`。

8. **汇总可审计输出**  
   打印真值与估计、`χ²/dof`、RMSE 及代表能点对比表，形成端到端最小证据链。
