# 光电效应 (Photoelectric Effect)

## R01

光电效应的核心关系是爱因斯坦方程：

`e * V_s = h * f - phi`

其中 `V_s` 为遏止电压、`f` 为入射光频率、`phi` 为材料逸出功。该关系直接体现“单光子能量量子化”，并能用直线斜率测得普朗克常数 `h`。

本条目实现一个可运行 MVP：自动生成合成实验数据、估计 `h` 与 `phi`、验证阈频和拟合误差，最终给出 `PASS/FAIL`。

## R02

任务目标（算法化）：
1. 构造频率扫描实验：`f in [380, 1300] THz`；
2. 根据真实逸出功 `phi_true` 生成理想遏止电压曲线；
3. 添加测量噪声得到观测数据；
4. 用两种方法反推参数：
   - 发射区线性回归 `V_s = a f + b`；
   - 全区间分段物理模型 `V_s = max(0, a(f-f0))`；
5. 检查 `h, phi, f0` 相对误差与拟合质量。

## R03

MVP 中采用的物理量与单位：
- 频率 `f`：Hz（输出中同时显示 THz）
- 光子能量 `E_gamma = h f`：eV
- 遏止电压 `V_s`：V
- 逸出功 `phi`：eV

由于 `1 eV = e * 1 V`，在数值上可写：

`V_s = max(0, h[eV*s] * f[Hz] - phi[eV])`

这让公式实现更直接，同时保留可追溯物理意义。

## R04

数据生成策略：
- 真实常数使用 SI 精确值 `h=6.62607015e-34 J*s`；
- 设定真实逸出功 `phi_true=2.28 eV`；
- 由公式生成理想 `V_s_true(f)`；
- 叠加高斯噪声 `N(0, sigma^2)`（默认 `sigma=0.03 V`）并裁剪负值为 0。

这相当于“带探测噪声的课堂实验数据”，用于验证识别流程。

## R05

算法方法 A（发射区线性回归）：
- 对观测数据用阈值 `V_s > 0.04 V` 判定“发生发射”；
- 在发射子集上拟合 `V_s = slope * f + intercept`；
- 参数映射：
  - `h_est = slope * e`
  - `phi_est = -intercept`（单位 eV）
  - `f0_est = -intercept / slope`

该方法模拟传统实验中“取线性区作图求斜率”的流程。

## R06

算法方法 B（全区间分段物理拟合）：

`V_s(f) = max(0, slope * (f - f0))`

特点：
- 显式包含阈频 `f0`，自然处理“低频无发射”；
- 对全部数据点拟合（不只高频发射区）；
- 通过 `scipy.optimize.curve_fit` 估计 `slope, f0`；
- 再映射得到：
  - `h_est = slope * e`
  - `phi_est = slope * f0`（数值单位 eV）

## R07

`demo.py` 输出包含三类信息：
1. 参数估计摘要：线性法与分段法的 `h, phi, f0`；
2. 验证指标：相对误差、RMSE、发射分类准确率、单调性/有限性检查；
3. 抽样数据表：`freq_thz, photon_energy_ev, v_stop_true_v, v_stop_obs_v` 及两种预测列。

脚本无需任何输入，直接运行即可复现结果。

## R08

复杂度分析（`N` 为频率采样点数）：
- 理想曲线生成：`O(N)`
- 噪声模拟：`O(N)`
- 线性回归：`O(N)`（单特征）
- 分段模型拟合：约 `O(I*N)`，`I` 为迭代步数
- 验证与表格构建：`O(N)`

空间复杂度主要是多个长度 `N` 的数组，约 `O(N)`。

## R09

数值与物理稳定性处理：
- 所有电压结果裁剪到 `>=0`，符合“无发射时遏止电压为 0”的实验语义；
- 线性拟合若出现 `slope <= 0` 直接报错（非物理）；
- 分段拟合设置有界参数区间，避免无意义极值；
- 验证阶段统一做 `finite` 检查，防止 NaN/Inf 漏出。

关键输出字段说明：

- `photon_energy_ev`：`h f` 对应光子能量（eV）
- `v_stop_true_v`：无噪声理论遏止电压

可直接用这些列核查“阈值以下近零、阈值以上近线性”的结构。

## R13

内置验证规则：
1. `h` 估计相对误差（线性/分段）不超过阈值；
2. 分段法 `phi` 与 `f0` 相对误差不超过阈值；
3. 分段拟合 `RMSE` 不超过阈值；
4. 发射分类准确率 `>= 0.90`；
5. 预测曲线在阈频以上单调不减；
6. 全部关键数组/参数有限。

全部通过才输出 `PASS`，否则进程返回非零退出码。

## R14

当前 MVP 边界：
- 仅考虑单一金属、单一逸出功；
- 噪声为独立高斯噪声，未建模系统误差（暗电流、接触电势等）；
- 只拟合遏止电压，不建模光电流-电压完整曲线；
- 未引入强光多光子效应和材料能带细节。

## R15

常见失败模式：
- 噪声过大导致发射区识别点太少，线性拟合不稳；
- 阈值设置不当（`emission_detect_v` 太高/太低）造成偏差；
- 采样频段过窄，`f0` 与斜率难以同时识别；
- 若 `curve_fit` 初值太差，可能收敛到次优局部解。

当前实现通过合理默认参数和边界约束降低这些风险。

## R16

可扩展方向：
1. 从“遏止电压”扩展到“完整 I-V 曲线”并联合拟合；
2. 增加多材料联合估计，比较不同逸出功；
3. 用贝叶斯方法输出 `h, phi, f0` 的置信区间；
4. 引入真实实验 CSV 数据接口，替代合成数据；
5. 增加鲁棒损失函数（Huber/Student-t）处理离群点。

## R17

方法对比与选型：
- 只做线性回归：实现最简单，但依赖“先验发射区筛选”；
- 只做分段拟合：物理一致性更好，但对初值和噪声更敏感；
- 本 MVP 同时提供两者：线性法用于快速可解释估计，分段法用于全数据一致性校准，并通过统一验证指标交叉检查。

## R18

`demo.py` 源码级流程拆解（8 步）：
1. `PhotoelectricConfig` 定义真实逸出功、频率范围、噪声与容差。  
2. `simulate_dataset` 生成频率网格，按 `V_s=max(0,hf/e-phi/e)` 计算 `v_stop_true`，叠加噪声得到 `v_stop_obs`。  
3. `fit_linear_emission_region` 在 `v_stop_obs > emission_detect_v` 子集上做 `LinearRegression`，得到 `slope/intercept`。  
4. 在线性参数中显式映射 `h_est=slope*e`、`phi_est=-intercept`、`f0_est=-intercept/slope`。  
5. `fit_piecewise_all_points` 用 `curve_fit` 拟合 `piecewise_photoelectric_model(f)=max(0,slope*(f-f0))`，直接估计 `slope,f0`。  
6. 由分段参数映射 `h_est=slope*e`、`phi_est=slope*f0`，并计算全体点 RMSE。  
7. `validate` 汇总误差、分类准确率、单调性和有限性检查，生成 `passed + metrics`。  
8. `main` 打印参数摘要、验证指标和抽样表格，最后输出 `Validation: PASS/FAIL` 并在失败时退出非零。  

第三方库仅用于数值求解与回归，物理模型、参数映射和校验逻辑均在源码中可追踪展开。
