# 预测-校正方法

- UID: `MATH-0152`
- 学科: `数学`
- 分类: `数值分析`
- 源序号: `152`
- 目标目录: `Algorithms/数学-数值分析-0152-预测-校正方法`

## R01

本条目实现一个可运行的二阶预测-校正（Predictor-Corrector）ODE 求解 MVP。  
实现形态为 `Euler Predictor + Trapezoidal Corrector`（常记为 PE(CE)^m 结构）。

目标：
- 给定常微分方程初值问题 `y' = f(t,y), y(t0)=y0`，在固定步长网格上计算数值解；
- 显式输出“预测值”和“校正值”，避免把方法藏在黑盒库里；
- 用有解析解案例验证二阶精度收敛趋势。

## R02

问题定义（离散时间步版本）：
- 输入：
  - 右端函数 `f(t, y)`；
  - 初值 `(t0, y0)`；
  - 终点 `t_end` 与步长 `h`；
  - 每步校正次数 `max_corrections >= 1`；
- 输出：
  - 时间网格 `t_n`；
  - 校正后解 `y_n`；
  - 每步预测值 `y_{n+1}^{(0)}`；
  - 每步实际校正迭代次数与误差统计（示例输出中给出）。

## R03

核心公式（本实现采用二阶公式）：

1. 预测（显式欧拉）：
- `y_{n+1}^{(0)} = y_n + h f(t_n, y_n)`

2. 校正（梯形公式固定点迭代）：
- `y_{n+1}^{(k+1)} = y_n + (h/2) [ f(t_n, y_n) + f(t_{n+1}, y_{n+1}^{(k)}) ]`

3. 数值精度：
- 当校正步至少执行一次时，该格式整体是二阶方法；
- 局部截断误差 `O(h^3)`，全局误差 `O(h^2)`（在标准光滑性条件下）。

## R04

算法总览（单步）：
1. 已知 `t_n, y_n`，计算 `f_n = f(t_n, y_n)`。
2. 用显式欧拉得到预测值 `y_hat`。
3. 以 `y_hat` 为初值，迭代执行梯形校正公式。
4. 达到 `max_corrections` 或校正增量范数小于 `tol` 时停止。
5. 记最终 `y_{n+1}` 为下一步解，进入下一网格点。

全局上对全部步重复该流程，得到完整轨线。

## R05

核心数据结构（`demo.py`）：
- `SolverResult`（`dataclass`）：
  - `t`: 时间网格，形状 `(N+1,)`；
  - `y`: 校正后解，形状 `(N+1, d)`；
  - `y_pred`: 预测值轨线，形状 `(N+1, d)`；
  - `correction_iters`: 每步实际校正迭代次数，形状 `(N,)`；
  - `step_size`, `max_corrections`: 方法参数快照。
- 统一把状态表示为 `numpy` 一维向量，兼容标量与向量系统。

## R06

正确性要点：
- 预测阶段给出 `t_{n+1}` 处的显式初猜，使隐式校正方程有可计算起点；
- 校正阶段等价于在离散层面逼近梯形隐式方法；
- 梯形法本身是二阶一致方法，固定点迭代只要收敛即可逼近其离散解；
- 在 `f` 满足局部 Lipschitz 且 `h` 足够小时，迭代校正通常可稳定收敛。

## R07

复杂度分析：
- 设状态维度为 `d`，步数为 `N`，每步校正迭代上限为 `m`：
  - 时间复杂度：`O(N * m * C_f)`，其中 `C_f` 是一次 RHS 计算代价（常见约为 `O(d)`）；
  - 空间复杂度：`O(N * d)`（保存全部时间点与轨线）。
- 本 MVP 以可解释性为主，不做稀疏存储或并行加速。

## R08

边界与异常处理：
- `h <= 0`、`t_end <= t0`、非有限数：抛出 `ValueError`；
- `(t_end - t0)/h` 非整数（在容差外）：抛出 `ValueError`；
- `y0` 不是标量/一维向量或含 `nan/inf`：抛出 `ValueError`；
- RHS 输出形状不匹配或出现 `nan/inf`：抛出 `ValueError`；
- `max_corrections < 1`、`tol <= 0`：抛出 `ValueError`。

## R09

MVP 取舍说明：
- 仅使用 `numpy + Python 标准库`，便于在最小环境运行；
- 不调用 `scipy.integrate.solve_ivp` 作为主求解流程，避免黑盒；
- 选择“二阶预测-校正”而非更高阶 Adams 系列，优先保证方法透明和代码短小；
- 输出中同时展示预测值、校正值和误差，便于教学和验算。

## R10

`demo.py` 函数职责：
- `_as_vector`：把初值规范化为一维向量；
- `_validate_grid`：检查时间网格和步数可整除性；
- `predictor_corrector_heun`：核心求解器（预测 + 迭代校正）；
- `exact_scalar_solution` / `scalar_rhs`：标量测试问题与解析解；
- `oscillator_rhs`：二维简谐振子系统 RHS；
- `run_scalar_convergence_demo`：打印步长收敛阶观测；
- `run_vector_demo`：打印向量系统终值误差；
- `main`：无交互入口。

## R12

输出字段说明（运行后可见）：
- `h` / `steps`：步长与离散步数；
- `max_error`：标量案例在整个网格上的最大绝对误差；
- `error_ratio`：相邻步长误差比（用于观察约 4 倍关系）；
- `observed_order`：由误差比换算得到的经验收敛阶；
- 详细步表中的 `predictor y_hat` 与 `corrected y_{n+1}`：展示 PECE 过程；
- 向量案例中的 `endpoint_inf_error`：终点无穷范数误差。

## R13

建议最小测试集：
- 标量解析解测试：`y' = y - t^2 + 1, y(0)=0.5`，验证误差随 `h` 减半近似降为 `1/4`；
- 向量系统测试：简谐振子 `x'=v, v'=-x`，比较 `2π` 终点误差；
- 参数非法测试：`h<=0`、`max_corrections=0`、`(t_end-t0)/h` 非整数；
- RHS 鲁棒性测试：故意返回错误形状或 `nan`，应触发异常。

## R14

可调参数：
- `h`：步长，越小通常越准但成本越高；
- `max_corrections`：每步校正次数，`1` 常已满足二阶，增大可更接近隐式梯形固定点；
- `tol`：校正迭代提前停止阈值；
- 示例区间与方程：可替换为其他初值问题做快速实验。

实践建议：先固定 `max_corrections=1` 看二阶收敛，再增大到 `2~3` 对比每步修正收益。

## R15

方法对比：
- 对比显式欧拉：预测-校正在同步长下通常显著更准（欧拉是一阶）；
- 对比 RK4：RK4 通常更高精度但每步函数评估更多；
- 对比全黑盒 ODE 求解器：黑盒上手快，但不利于学习“预测-校正”内部机制。

本实现适合教学、验证与小规模工程原型，不替代高阶自适应工业求解器。

## R16

应用场景：
- 常微分方程课程中“显式-隐式折中”方法教学；
- 物理仿真中的小型时间推进原型；
- 控制与信号系统的离散时间预测修正实验；
- 需要输出中间预测值以做解释或诊断的数值流程。

## R18

`demo.py` 源码级算法流（8 步）：
1. `main()` 依次调用 `run_scalar_convergence_demo()` 和 `run_vector_demo()`，组织两个无交互实验。
2. 每次求解先进入 `predictor_corrector_heun()`，通过 `_validate_grid()` 检查 `t0,t_end,h` 并确定离散步数 `N`。
3. `_as_vector()` 把初值 `y0` 统一成一维向量，分配 `t/y/y_pred/correction_iters` 数组。
4. 对每个时间步先算 `f_n = f(t_n, y_n)`，用显式欧拉得到预测值 `y_hat = y_n + h f_n`，写入 `y_pred[n+1]`。
5. 以 `y_hat` 为初值执行梯形校正固定点迭代：`y_new = y_n + 0.5*h*(f_n + f(t_{n+1}, y_corr))`。
6. 若 `||y_new - y_corr||_inf < tol` 则提前停止，否则最多迭代到 `max_corrections`，并记录该步迭代次数。
7. 把最终 `y_corr` 作为 `y_{n+1}` 进入下一步，直至完成整段时间推进，返回 `SolverResult`。
8. 示例层计算解析解误差、误差比与经验阶，并打印前几步预测/校正细节和向量系统终点误差。
