# 泡利不相容原理 (Pauli Exclusion Principle)

- UID: `PHYS-0025`

- 分类: `量子力学`

## R01

泡利不相容原理指出：对同一种费米子（半整数自旋粒子），多体波函数必须在交换任意两粒子后变号，因此任意两个粒子不能占据完全相同的单粒子量子态。数学上，费米子态可由 Slater 行列式表示；一旦两行（或两列）相同，行列式为 0，即“同态占据振幅为零”。

## R02

该原理是原子壳层结构、周期表、金属电子结构、白矮星简并压等现象的核心约束。工程上，它也是电子结构计算（Hartree-Fock/DFT 等）中“占据数上限为 1（或每空间轨道每自旋 1）”的基础规则。

## R03

本目录 MVP 的目标是构建一个最小可验证闭环，而不是完整量子化学软件：

- 用一维无限深势阱轨道构造 `N` 个自旋无关费米子的 Slater 行列式；
- 数值验证交换两粒子坐标后 `psi(x1,x2,...) = -psi(x2,x1,...)`；
- 强制两粒子坐标相同、或强制两粒子占据同一轨道时，验证振幅塌缩为 0；
- 在固定粒子数下求解费米-狄拉克分布化学势，验证每能级占据 `0 <= f_i <= 1`。

## R04

核心公式：

1. 费米子反对称性：`Psi(...,xi,...,xj,...) = -Psi(...,xj,...,xi,...)`。
2. Slater 行列式：
   `Psi(x1..xN)=det(A)/sqrt(N!)`，其中 `A_ij = phi_{n_i}(x_j)`。
3. 无限深势阱轨道：
   `phi_n(x)=sqrt(2/L) sin(n*pi*x/L)`，`E_n ~ n^2`（本脚本采用无量纲缩放 `E_n=n^2`）。
4. 费米-狄拉克占据：
   `f_i = 1/(exp((E_i-mu)/T)+1)`，化学势满足 `sum_i f_i = N`。

## R05

算法流程（高层）：

1. 生成离散单粒子能级 `E_n` 与低阶轨道索引；
2. 选取一组粒子坐标，计算参考 Slater 振幅 `psi_ref`；
3. 交换两粒子坐标计算 `psi_swap`，评估 `|psi_ref + psi_swap|`；
4. 构造“相同坐标”与“重复轨道”两种场景，验证振幅接近 0；
5. 通过 `brentq` 解 `sum_i f_i(mu)-N=0` 得 `mu`，计算低温和较高温占据；
6. 汇总占据表、误差指标和断言门控，形成非交互可回归脚本。

## R06

记粒子数为 `N`、离散能级数为 `M`：

- Slater 矩阵构造为 `O(N^2)`，行列式计算为 `O(N^3)`；
- 每次化学势求值成本 `O(M)`，一维根求解总成本约 `O(M * K)`（`K` 为迭代步数）；
- 表格与统计均为 `O(M)`；
- 空间复杂度约 `O(max(N^2, M))`。

在本 MVP 的默认规模（`N=4, M=12`）下计算量很小，适合快速验证。

## R07

正确性直觉：

- 交换两粒子等价于交换 Slater 矩阵两列，行列式符号翻转；
- 若两粒子坐标相同，则矩阵两列相同，行列式为 0；
- 若两粒子被强制到同一单粒子态，则矩阵两行相同，行列式为 0；
- 费米-狄拉克函数值域天然在 `(0,1)`，因此单能级占据不会超过 1。

## R08

数值稳定策略：

- 对 `exp((E-mu)/T)` 的指数输入做裁剪（`[-80,80]`）避免溢出；
- 化学势求根使用有符号包根区间 + `brentq`，防止非夹逼求解发散；
- 断言同时检查“反对称误差、占据上限、粒子数守恒”，避免单指标误判；
- 使用固定、彼此分离的采样坐标，避免参考态本身偶然接近 0。

## R11

输出关键指标：

- `antisymmetry_error`：`|psi_ref + psi_swapped|`；
- `equal_position_amplitude`：两粒子同坐标时振幅绝对值；
- `duplicate_orbital_amplitude`：重复轨道占据时振幅绝对值；
- `mu_low`、`mu_high`：低温/高温化学势；
- `lowT_MAE_vs_T0step`、`lowT_R2_vs_T0step`：低温占据与 `T=0` 阶跃填充差异；
- `torch_cap_violation`：`max(f_i-1,0)`；
- `torch_particle_error`：`|sum_i f_i - N|`。

## R14

常见失效模式：

1. 把 Slater 矩阵行列定义写反后又误解交换关系，导致符号验证失败；
2. 直接用未包根的一般优化求 `mu`，在极端温度下收敛不稳；
3. 忽略指数裁剪，出现 `exp` 上溢导致 `nan`；
4. 只检查占据上限，不检查粒子数守恒；
5. 把“重复空间轨道但不同自旋可共存”与“完全相同单粒子态禁止”混为一谈。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `PauliConfig` 定义粒子数、能级数、势阱长度和两组温度参数，固定可复现实验条件。  
2. `level_energies` 与 `orbital_value` 显式给出无量纲能级 `E_n=n^2` 和势阱轨道 `phi_n(x)`。  
3. `slater_matrix` 按 `A_ij=phi_{n_i}(x_j)` 构造矩阵，`slater_amplitude` 调 `det(A)/sqrt(N!)` 计算多体振幅。  
4. `run_pauli_demo` 先算参考振幅，再交换两粒子坐标计算 `psi_swapped`，用 `|psi_ref+psi_swapped|` 评估反对称性。  
5. 同一函数中再构造“坐标重合”和“重复轨道”两种禁态场景，验证振幅塌缩到 0。  
6. `solve_chemical_potential` 把粒子数约束写成一元方程 `sum_i f_i(mu)-N=0`，用 `brentq` 求化学势。  
7. 计算低温/高温占据后，用 `sklearn` 得到 `MAE/R^2`，并用 `torch` 检查占据上限违反量与粒子数误差。  
8. `main` 打印关键标量与 `pandas` 占据表，再通过断言门控给出通过/失败。  

第三方库未被当作黑盒物理算法：`scipy` 仅提供通用行列式与求根器，核心物理建模（反对称性、禁态构造、费米占据约束）均在源码中逐步实现。
