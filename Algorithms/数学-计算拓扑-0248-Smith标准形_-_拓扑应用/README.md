## R01

Smith 标准形（Smith Normal Form, SNF）把整数矩阵通过可逆整系数行列变换化为对角形式：

- `U A V = diag(d1, d2, ..., dr, 0, ..., 0)`
- 其中 `U, V` 是 unimodular 矩阵（行列式为 `±1`）
- 不变量满足 `d1 | d2 | ... | dr`

在计算拓扑中，边界算子矩阵是整数矩阵，SNF 可直接给出同调群的自由部分和挠子部分。

## R02

历史定位（简要）：

- H. J. S. Smith 在 19 世纪建立了整数矩阵的标准分解理论；
- 代数拓扑把链复形边界写成整数矩阵后，SNF 成为“可计算同调”的核心工具；
- 现代计算拓扑软件（含 simplicial/CW 复形场景）仍以该思想作为整数同调的基础支柱。

## R03

本条目 MVP 解决的问题：

- 输入：若干小型 CW 复形示例的 `d2: C2 -> C1` 整数矩阵；
- 输出：
  - `d2` 的非零 Smith 不变量因子；
  - `H0, H1, H2` 的群分解字符串；
  - 与已知拓扑结果的一致性断言。

MVP 选取一类可直接演示 SNF 价值的链复形：单顶点曲面模型（`d1 = 0`）。

## R04

时间复杂度（本实现路径）：

- 秩计算：有理数高斯消元，约 `O(m n min(m,n))`；
- `k` 阶行列式子式枚举：`C(m,k) * C(n,k)` 个子式；
- 每个子式行列式用 Bareiss 算法，单次约 `O(k^3)`；
- 总体对小矩阵可接受，但不适合大规模工业求解。

这是“教学型、可审计”的复杂度取舍。

## R05

空间复杂度：

- 存储矩阵本体 `O(mn)`；
- 消元与子式构造额外开销与子矩阵大小相关，单次约 `O(k^2)`；
- 本 MVP 不做缓存与并行，优先保证可读性与可验证性。

## R06

微型例子（`RP^2`）：

- `d2 = [2]`；
- SNF 非零对角因子为 `[2]`；
- `H1 = coker(d2) = Z/2Z`；
- `H0 = Z`，`H2 = 0`。

该例子直观展示了“挠子由 SNF 对角因子直接读出”。

## R07

算法意义（拓扑视角）：

- 把“抽象同调群”降为“整数矩阵分解”；
- 同时得到自由部分（Betti 数）与有限挠子信息；
- 对区分空间类型很关键，例如可区分 `T^2`、`RP^2`、Klein 瓶的 `H1` 结构。

## R08

核心数学关系：

1. 设 `A` 为整数矩阵，`Δk` 为全部 `k×k` 子式行列式的 gcd（行列式因子）。
2. 若 `rank(A)=r`，则 `Δ0=1`，并有 `d_k = Δk / Δ{k-1}`。
3. 这些 `d_k` 正是 SNF 非零不变量因子，满足 `d1 | d2 | ... | dr`。
4. 对单顶点曲面示例（`d1=0`）：
   - `H1 = ker(d1) / im(d2) = C1 / im(d2) = coker(d2)`；
   - `H2 = ker(d2)`（因 `C3=0`）；
   - `H0 = Z`。

## R09

适用条件与局限：

适用：

- 小到中等尺寸整数矩阵；
- 教学验证、原型分析、结果可审计优先的任务；
- 重点在 SNF 与同调结构关系说明。

局限：

- 子式枚举法在大矩阵上组合爆炸；
- 本 MVP 只覆盖 `d1=0` 的示例链复形，不是通用全维同调引擎；
- 未实现完整 `U, V` 变换矩阵，仅输出不变量因子。

## R10

正确性要点（实现层）：

1. `rank_over_rationals` 用 `Fraction` 做精确有理消元，避免浮点误判秩。
2. `det_bareiss` 用无分式消元精确计算整数行列式。
3. `determinantal_divisor` 对所有 `k×k` 子式取 gcd，得到 `Δk`。
4. 由 `d_k = Δk / Δ{k-1}` 得到 SNF 不变量因子，并检查整除链。
5. `run_self_checks` 将输出与已知同调结果比对，不一致即抛异常。

## R11

数值稳定性与整数正确性：

- 全流程不使用浮点做核心代数，避免舍入误差；
- 行列式用 Bareiss，保持整数域精确性；
- 仅在打印时使用 `numpy` 做矩阵展示，不参与核心证明逻辑。

## R12

性能视角：

- 该实现优先“透明可追踪”，非性能最优；
- 当矩阵增大时，瓶颈在子式组合枚举；
- 若要规模化，应改为更高效的整系数矩阵约化算法（直接构造 SNF 的行列操作流程）。

## R13

本目录 MVP 的可验证保证：

- 内置 3 个经典曲面：`T^2`、`RP^2`、Klein 瓶；
- 自动输出每个 `d2` 的 SNF 不变量因子与 `H0/H1/H2`；
- 自动断言：
  - `T^2: H1 = Z^2`
  - `RP^2: H1 = Z/2Z`
  - `Klein: H1 = Z ⊕ Z/2Z`
- 全部通过后打印 `All checks passed.`。

## R14

常见失效模式：

- 输入矩阵非矩形（行长度不一致）；
- 错误地把非整数数据传入核心函数；
- 在更复杂链复形中直接套用本示例公式（`d1=0` 假设失效）；
- 把该原型用于大矩阵而忽视组合复杂度。

## R15

实现设计（`demo.py`）：

- `rank_over_rationals`：有理数精确秩；
- `det_bareiss`：精确行列式；
- `determinantal_divisor`：`Δk` 计算；
- `smith_invariant_factors`：由 `Δk` 推出 `d_k`；
- `cokernel_decomposition`：从 SNF 读出自由部分/挠子；
- `analyze_surface`：组装 `H0/H1/H2`；
- `run_self_checks`：与已知拓扑结论做强校验。

## R16

相关算法链路：

- 计算拓扑：整数同调、边界矩阵约化、持久同调中的域系数对比；
- 抽象代数：有限生成阿贝尔群结构定理；
- 数值/符号计算：Hermite 标准形、模 `p` 约化、整系数消元。

## R18

`demo.py` 的源码级算法流程（8 步，非黑盒）如下：

1. `main` 构造 3 个曲面示例的 `d2` 矩阵，并逐个调用 `analyze_surface`。
2. `analyze_surface` 先把输入转换为合法整数矩阵，随后调用 `cokernel_decomposition`。
3. `cokernel_decomposition` 调用 `smith_invariant_factors` 计算 SNF 非零对角因子。
4. `smith_invariant_factors` 先经 `rank_over_rationals` 得秩 `r`，再对 `k=1..r` 调 `determinantal_divisor` 计算 `Δk`。
5. `determinantal_divisor` 枚举全部 `k×k` 子式，逐个调用 `det_bareiss` 得行列式并做 gcd 累积。
6. `det_bareiss` 使用无分式 Bareiss 消元，精确返回整数行列式值。
7. 回到 `smith_invariant_factors`，按 `d_k = Δk / Δ{k-1}` 生成不变量因子并校验整除链。
8. `analyze_surface` 把因子转为 `H1` 的自由部分与挠子分解，`run_self_checks` 做已知结论断言，通过后打印 `All checks passed.`。

说明：`numpy` 只用于输出展示，核心同调与 SNF 逻辑由源码逐步实现，可逐函数审计。
