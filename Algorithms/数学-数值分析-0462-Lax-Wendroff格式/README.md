# Lax-Wendroff格式

- UID: `MATH-0462`

- 源序号: `462`
- 目标目录: `Algorithms/数学-数值分析-0462-Lax-Wendroff格式`

## R01

Lax-Wendroff 格式是求解双曲守恒律和线性平流方程的经典二阶显式差分方法。它在保持计算简单的同时，比一阶迎风格式显著降低数值耗散，常用于“对平滑解追求更高精度”的基线数值实验。

本文用一维线性平流方程构建最小可运行 MVP：
\[
u_t + a u_x = 0,
\]
其中 \(a\) 为常数传播速度。

## R02

问题设定：
- 空间区间：\(x\in[0,1)\)
- 时间区间：\(t\in[0,T]\)
- 初值：\(u(x,0)=u_0(x)\)
- 边界条件：周期边界 \(u(0,t)=u(1,t)\)

该模型有解析解：
\[
u(x,t)=u_0(x-a t),
\]
在周期区间上可写为 \(u_0((x-a t)\bmod 1)\)。因此它非常适合做数值格式正确性与收敛阶验证。

## R03

离散网格定义：
- \(x_j=j\Delta x,\ j=0,\dots,N-1,\ \Delta x=1/N\)
- \(t^n=n\Delta t\)
- 网格近似 \(u_j^n\approx u(x_j,t^n)\)

定义 CFL 数：
\[
\nu = \frac{a\Delta t}{\Delta x}.
\]
对显式双曲格式，\(|\nu|\) 是稳定性和误差行为的核心参数。

## R04

Lax-Wendroff 更新公式（线性平流）为：
\[
u_j^{n+1} = u_j^n
- \frac{\nu}{2}(u_{j+1}^n-u_{j-1}^n)
+ \frac{\nu^2}{2}(u_{j+1}^n-2u_j^n+u_{j-1}^n).
\]

其中：
- 第一项是中心差分的对流项（主导相位传播）；
- 第二项是二阶时间修正（来自 Taylor 展开 + PDE 代换）。

该格式对时间和空间都是二阶精度（针对足够光滑解）。

## R05

稳定性要点（线性常系数情形）：
- Von Neumann 分析可得稳定条件 \(|\nu|\le 1\)
- 当 \(|\nu|\) 接近 1 时，色散误差通常更明显
- 若超过 1，显式格式会迅速放大高频误差并发散

在工程实践中常取 `0.6~0.9` 之间的 CFL 作为效率与稳定性的折中。

## R06

与一阶迎风法对比：
- 精度：Lax-Wendroff 二阶，迎风一阶
- 数值耗散：Lax-Wendroff 更小
- 色散振荡：Lax-Wendroff 在梯度大区域更容易出现轻微振荡
- 鲁棒性：迎风法更“钝”但更稳健

因此 Lax-Wendroff 非常适合作为“高于一阶、又足够轻量”的教学和原型方案。

## R07

边界条件处理：
- 本项目使用周期边界
- 代码中通过 `np.roll(u, -1)` 与 `np.roll(u, 1)` 获取 \(u_{j+1}\)、\(u_{j-1}\)
- 这种实现天然满足首尾相连，避免手写 ghost cell

若改为非周期问题，可扩展为固定边值、外推边值或特征边界。

## R08

复杂度分析：
- 单步只做常数次向量运算，复杂度 \(O(N)\)
- 总复杂度 \(O(NN_t)\)
- 内存复杂度 \(O(N)\)（只保留当前时间层）

该复杂度与一阶格式同阶，但精度更高，是典型“同量级开销换更低误差”。

## R09

伪代码：

```text
input: nx, a, t_end, cfl_target
x <- uniform grid on [0,1)
u <- initial_condition(x)
dx <- 1/nx
dt0 <- cfl_target * dx / |a|
n_steps <- round(t_end / dt0), then dt <- t_end / n_steps
nu <- a * dt / dx
assert |nu| <= 1
repeat n_steps:
    u_right <- roll(u, -1)
    u_left  <- roll(u, 1)
    u <- u - 0.5*nu*(u_right-u_left) + 0.5*nu^2*(u_right-2u+u_left)
u_exact <- u0((x-a*t_end) mod 1)
report L1/L2/Linf and mass error
```

## R10

MVP 参数建议：
- `nx`: 100、200、400（便于看收敛趋势）
- `a`: 1.0（正向平移）
- `t_end`: 0.4（观察明显平移但不过长）
- `cfl_target`: 0.8（稳定且高效）

注意：实际 `cfl` 会因步数取整而略微偏离 `cfl_target`，代码会打印真实值。

## R11

`demo.py` 默认实验输出：
- 每个网格的 `n_steps` 与 `actual_cfl`
- 三种误差范数：`L1/L2/Linf`
- 离散质量误差 `mass_error`
- 相邻网格之间基于 `L2` 的实验收敛阶估计

对平滑初值，收敛阶通常接近 2（受时间步耦合与有限网格影响会有偏差）。

## R12

代码模块结构：
- `initial_condition(x)`: 构造平滑周期初值
- `lax_wendroff_step(u, cfl)`: 单步更新核
- `solve_advection_lax_wendroff(...)`: 时间推进与 CFL 检查
- `exact_solution_periodic(...)`: 构造解析参考解
- `compute_error_metrics(...)`: 计算误差范数
- `run_case(...)`: 单组参数实验
- `main()`: 批量网格实验并打印结果

## R14

常见实现错误：
- 误把 \(\nu\) 当作 `cfl_target` 直接使用，未按实际 `dt` 重算
- 周期边界索引方向写反（`j+1` / `j-1`）
- 忽略稳定条件 \(|\nu|\le 1\)
- 用非周期精确解比较周期数值解，导致误差评估失真

当前实现已逐项规避。

## R17

适用性边界：
- 适用：平滑解、需要二阶精度且实现成本低的场景
- 谨慎：存在激波或强间断时，可能出现 Gibbs 型振荡
- 不适合：要求严格单调性/无振荡的强间断计算（应考虑 TVD/WENO）

Lax-Wendroff 的工程价值在于：简洁、透明、可解释，是高分辨率方法前的关键中间层。

## R18

`demo.py` 的源级算法流程（非黑盒，8 步）：
1. 在 `[0,1)` 上生成均匀网格 `x_j`，通过 `initial_condition` 构造离散初值 `u^0`。
2. 根据 `dx`、`a` 和 `cfl_target` 先估算 `dt0`，再按目标终止时间反推整数步数 `n_steps`，得到修正后的 `dt`。
3. 计算真实 CFL `nu = a*dt/dx`，若 `|nu|>1` 直接抛错，阻止不稳定推进。
4. 每个时间步内调用 `np.roll` 生成左右邻点 `u_{j+1}^n` 与 `u_{j-1}^n`。
5. 按 Lax-Wendroff 离散公式逐点更新，得到下一层 `u^{n+1}`，并覆盖当前状态。
6. 推进结束后用 `x_shift=(x-a*t_end)%1.0` 构造周期解析解 `u_exact`，保持与离散边界条件一致。
7. 计算误差向量 `e=u_num-u_exact`，输出 `L1/L2/Linf` 三个范数量化整体与局部误差。
8. 对多组网格结果计算 `p=log(e_h/e_{h/2})/log(2)`，给出实验收敛阶，验证二阶实现是否达标。
