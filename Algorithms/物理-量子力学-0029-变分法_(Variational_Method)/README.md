# 变分法 (Variational Method)

- UID: `PHYS-0029`
- 学科: `物理`
- 分类: `量子力学`
- 源序号: `29`
- 目标目录: `Algorithms/物理-量子力学-0029-变分法_(Variational_Method)`

## R01

量子力学中的变分法用于近似求解基态能量：

对任意归一化试探波函数 `psi(theta)`，其期望能量
`E(theta)=<psi|H|psi>` 总满足 `E(theta) >= E0`（`E0` 为真实基态能量）。

因此，把“解偏微分本征值问题”转成“参数优化问题”，即可得到可计算的上界估计。

## R02

本 MVP 选择一维非谐振子哈密顿量：

`H = -1/2 * d^2/dx^2 + 1/2 * x^2 + g * x^4`，其中 `g >= 0`。

试探波函数采用单参数高斯族：

`psi_alpha(x) = (alpha/pi)^(1/4) * exp(-alpha*x^2/2)`，`alpha > 0`。

该设置足够小，但覆盖了“建模-变分最小化-数值验证”完整链路。

## R03

在自然单位 `hbar = m = 1` 下，上述试探函数对应的期望能量可写为闭式：

`E(alpha) = alpha/4 + 1/(4*alpha) + 3*g/(4*alpha^2)`。

其中三项分别来自：
- 动能项 `T`。
- 谐振势能项 `x^2/2`。
- 非谐势能项 `g*x^4`。

最优参数满足 `dE/dalpha = 0`。

## R04

驻点方程可化为三次多项式：

`alpha^3 - alpha - 6g = 0`。

本实现同时使用两种一致性检查：
- 数值优化器求得 `alpha*`。
- 将 `alpha*` 代回上式，检查多项式残差是否接近 0。

这能确认“最小化结果确实对应变分驻点”，而非优化器偶然输出。

## R05

适用前提：
- 关注基态（或低能态）近似，而非完整谱。
- 试探函数族能覆盖真实基态的主要形状特征。
- 参数优化可稳定收敛。
- `g >= 0` 保证势能束缚（有离散基态）。

当势能高度复杂、多阱、强非高斯结构时，需要更丰富的试探基函数。

## R06

`demo.py` 的输入输出约定：
- 输入：脚本内置耦合常数数组 `g_list = [0.0, 0.1, 0.5, 1.0, 2.0]`。
- 输出：每个 `g` 的
1. 最优 `alpha*`
2. 变分能量 `E_var`
3. 有限差分参考基态能 `E_ref`
4. 能隙 `gap = E_var - E_ref`
5. 导数与驻点残差
6. 变分上界检查结果。

脚本无交互输入，直接运行即可。

## R07

算法流程（高层）：
1. 定义 `E(alpha)` 与 `dE/dalpha`。
2. 在给定区间上做一维有界最小化，得到 `alpha*`。
3. 计算 `E_var = E(alpha*)`。
4. 用有限差分离散薛定谔方程，求参考基态能 `E_ref`。
5. 计算 `gap`、梯度、驻点多项式残差。
6. 汇总为表格并执行阈值检查，给出 `PASS/FAIL`。

## R08

复杂度分析（设有限差分内部网格点数为 `N`）：
- 单次变分优化：迭代次数记为 `K`，复杂度近似 `O(K)`（每步常数代价）。
- 单次有限差分基态求解：三对角本征值计算，约 `O(N^2)`。
- 总复杂度：`O(M*(K + N^2))`，`M` 为 `g` 的测试个数。

本 MVP 目标是可读和可验证，不追求超大规模性能。

## R09

数值稳定性策略：
- 显式约束 `alpha > 0`，避免能量表达式奇异。
- `minimize_scalar(method="bounded")` 限定搜索区间，降低发散风险。
- 有限差分采用较大 `xmax` 与较密网格，减小边界截断误差。
- 上界检查使用小容差，避免参考解离散误差导致误判。

## R10

MVP 技术栈：
- Python 3
- `numpy`：向量化、基础数值计算
- `scipy.optimize`：一维有界最小化
- `scipy.linalg`：三对角基态本征值计算
- `pandas`：结果表格化输出

核心变分公式与验证逻辑在源码中显式实现，不依赖黑箱“端到端解题器”。

## R12

输出字段说明：
- `g`: 非谐耦合常数。
- `alpha_opt`: 变分最优参数。
- `E_var`: 变分基态能上界。
- `E_ref(fd)`: 有限差分参考基态能。
- `gap(E_var-E_ref)`: 与参考值差距。
- `dE/dalpha@opt`: 在最优点的导数（应接近 0）。
- `stationarity_residual`: `alpha^3-alpha-6g` 残差。
- `norm_error`: 试探波函数数值归一化误差。
- `bound_ok`: 是否满足 `E_var >= E_ref - tol`。

## R13

正确性检查（demo 内置）：
1. `alpha_opt` 必须为正。
2. 导数 `|dE/dalpha|` 小于阈值。
3. 驻点多项式残差足够小。
4. 试探波函数数值归一化误差足够小。
5. 所有样例都满足变分上界检查。

全部通过时输出 `Validation: PASS`。

## R14

当前实现局限：
- 仅使用单高斯试探函数，表达能力有限。
- 仅演示基态近似，未做激发态正交约束变分。
- 参考解来自有限差分离散，不是解析真值。
- 仅覆盖一维势场，未扩展到多体电子结构问题。

## R17

与其他方法对比：
- 直接网格本征值法：更通用，但需要更细网格和更高计算代价。
- 微扰法：要求可分解为“可解 + 小扰动”，适用面更窄。
- 纯黑箱优化器：开发快，但算法可解释性和审计性弱。

本方案用“显式公式 + 轻量优化 + 参考解校验”实现了小而完整的变分法 MVP。

## R18

`demo.py` 源码级流程拆解（9 步）：
1. `gaussian_energy` 按解析公式计算 `E(alpha)`，并检查 `alpha>0`。
2. `gaussian_energy_grad` 计算一阶导数，用于最优性诊断。
3. `stationarity_polynomial_residual` 计算 `alpha^3-alpha-6g`，验证驻点方程。
4. `optimize_variational_parameter` 在有界区间调用 `minimize_scalar`，得到 `alpha*` 与 `E_var`。
5. `gaussian_wavefunction` 与 `normalization_error` 在离散网格上计算归一化误差，确认试探函数实现正确。
6. `finite_difference_ground_energy` 构造三对角哈密顿矩阵（动能二阶差分 + 势能对角项），求最小本征值得到 `E_ref`。
7. `analyze_single_coupling` 对单个 `g` 汇总 `alpha*`、`E_var`、`E_ref`、残差和上界检查。
8. `main` 遍历多个 `g` 形成 `pandas` 表格，打印结构化结果。
9. `main` 执行统一阈值判断，输出 `Validation: PASS/FAIL` 并在失败时返回非零退出码。

可以看到，第三方库只负责“基础数值原语”（一维最小化、三对角本征值）；建模、变分公式、约束检查和验证逻辑都在源码中展开。
