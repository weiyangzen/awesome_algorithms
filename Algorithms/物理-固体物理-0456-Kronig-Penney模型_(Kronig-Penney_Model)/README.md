## R01

Kronig-Penney 模型是固体物理中描述“一维周期势场电子能带结构”的经典模型。

它把晶体势能近似为周期重复的势阱/势垒（或等效的 delta 势阵列），再结合 Bloch 定理得到允许能带与禁带。该模型是从“自由电子”走向“能带理论”的最小可计算桥梁。

## R02

本目录采用 **delta 势垒版 Kronig-Penney** 作为 MVP（比有限宽方势垒更短小，仍保留能带/禁带核心机制）：

- 周期长度记为 `a`（代码中取 `a=1` 的无量纲化单位）。
- 定义 `x = alpha * a`，其中 `alpha = sqrt(2mE)/hbar`。
- 势垒强度参数 `P` 为无量纲常数，控制带隙开口程度。
- 电子色散关系写成
  `cos(k a) = F(x) = cos(x) + P * sin(x)/x`。

允许态由 `|F(x)| <= 1` 判定；`|F(x)| > 1` 对应禁带。

## R03

数学对象与物理量：

- `k`：Bloch 波矢（第一布里渊区可取 `k in [0, pi/a]`）。
- `x`：与能量单调对应的无量纲变量，`E~ = x^2`（在 `hbar^2/(2m)=1` 约化单位下）。
- `F(x)`：色散方程右侧函数。

模型目标是扫描 `x`（等价扫描能量），找出所有满足 `|F(x)|<=1` 的连续区间，即 allowed bands。

## R04

从 Bloch 条件 `psi(x+a)=exp(i k a) psi(x)` 与单胞传输关系可得：

`cos(k a) = Tr(M_cell)/2 = F(x)`。

对 delta 势垒单胞，`Tr(M_cell)/2` 可化简为

`F(x) = cos(x) + P * sin(x)/x`。

因此：
- 若 `|F(x)| <= 1`，可取 `k = arccos(F)/a`，存在传播型 Bloch 态；
- 若 `|F(x)| > 1`，`k` 变为复数，对应衰减态（禁带）。

## R05

适用前提与假设：

- 一维周期体系；
- 单电子近似（忽略电子-电子强关联）；
- 势场静态且严格周期；
- 采用理想化 delta 势垒，目的是保留带隙机制而减少参数维度；
- demo 使用无量纲约化单位，强调结构而非材料常数拟合。

## R06

`demo.py` 输入输出约定（无交互）：

- 输入（脚本内固定）：
1. 势垒参数 `P=6.0`；
2. 扫描范围 `x in [1e-4, 12*pi]`；
3. 网格数 `n_grid=80000`。
- 输出：
1. 允许能带表（`x_left/x_right`, `E_left/E_right`, 宽度等）；
2. 禁带表（相邻能带之间的 gap 宽度）；
3. 误差/阈值检查；
4. `Validation: PASS/FAIL`。

## R07

算法主流程（高层）：

1. 在高分辨率网格上计算 `F(x)`。
2. 用 `|F(x)|<=1` 生成 allowed mask。
3. 提取 allowed mask 的连续区段，得到粗略能带区间。
4. 对每个区段边界，在邻接网格中对 `|F(x)|-1=0` 做一维根求解（Brent）细化边界。
5. 将 `x` 区间映射为 `E~ = x^2`，并在带内采样计算 `k=arccos(F)`。
6. 输出能带/禁带表并进行一致性校验。

## R08

复杂度（`N` 为扫描网格数，`B` 为检测到的能带数）：

- 扫描计算 `F(x)`：`O(N)`；
- 连续区段提取：`O(N)`；
- 边界细化：每个带最多两个边界，Brent 迭代次数近似常数，约 `O(B)`；
- 总体复杂度：`O(N) + O(B)`，主导项为 `O(N)`；
- 空间复杂度：主要存储 `x/F/mask`，为 `O(N)`。

## R09

数值稳定性要点：

- `sin(x)/x` 在 `x≈0` 处使用 `np.sinc(x/pi)` 计算，避免 `0/0`；
- 允许条件使用小容差 `|F|<=1+tol`，抵抗浮点边界噪声；
- 带边界通过根求解细化，而不是直接用网格点截断；
- `arccos` 前对 `F` 做 `clip[-1,1]` 防止极小越界导致 NaN。

## R10

MVP 技术栈：

- Python 3
- `numpy`（向量化扫描、数值函数）
- `pandas`（表格化输出）
- `scipy.optimize.brentq`（一维边界根细化）

核心物理判据与流程在源码中显式实现，不是“调用黑箱 band solver 一步完成”。

## R12

输出字段说明：

- `band`: 能带编号；
- `x_left/x_right`: 能带在 `x` 轴上的边界；
- `E_left/E_right`: 对应约化能量边界（`E~=x^2`）；
- `E_width`: 能带宽度；
- `k_min/k_max`: 带内采样得到的 Bloch 波矢范围；
- `edge_residual`: 边界点对方程 `||F|-1|` 的残差；
- `allowed_violation`: 带内采样点违反 `|F|<=1` 的最大越界量；
- `gap` 与 `E_start/E_end/E_width`: 相邻能带之间禁带信息；
- `midpoint_forbidden_margin`: 禁带中点处 `|F|-1` 的余量（应为正）。

## R13

demo 内置正确性检查：

1. 至少检测到 4 条能带（在设定扫描范围内）；
2. 所有能带宽度应为正；
3. 带边界残差应小于阈值；
4. 带内不应出现显著 `|F|>1` 违规；
5. 至少存在一个禁带且禁带宽度为正；
6. 禁带中点应满足 `|F|>1`（有正 margin）。

全部通过输出 `Validation: PASS`，否则 `FAIL` 并返回非零退出码。

## R14

当前实现局限：

- 使用 delta 势垒近似，不含有限宽势阱/势垒的额外几何参数；
- 只做静态能带扫描，不含态密度、波函数形状和输运系数；
- 仅限一维周期模型，不涉及二维/三维晶格与多原子基元；
- 未包含实验拟合流程（如从 ARPES 反演有效参数）。

## R17

方案对比：

- 直接用现成第一性原理软件（DFT）精度更高，但成本大且黑箱重；
- 紧束缚模型适合离散格点和局域轨道视角；
- 近自由电子模型更偏弱势场微扰；
- Kronig-Penney 在“可解析结构 + 可计算实现 + 清晰带隙机制”之间平衡最好，适合作为 MVP。

## R18

`demo.py` 源码级流程拆解（8 步）：

1. `kronig_penney_rhs` 实现 `F(x)=cos(x)+P*sin(x)/x`，并用 `np.sinc` 处理 `x≈0` 稳定性。  
2. `allowed_mask` 依据 `|F|<=1+tol` 标记 allowed energy points。  
3. `find_true_segments` 在线性扫描中提取 mask 连续真区段，得到能带粗边界索引。  
4. `refine_edge` 在每个粗边界邻域对 `|F|-1=0` 使用 `scipy.optimize.brentq` 细化到高精度边界。  
5. `analyze_bands` 把细化后的 `x` 边界映射为 `E~=x^2`，并在带内采样 `k=arccos(F)`。  
6. 同函数内构造相邻带之间的 gap，并用 gap 中点计算 `|F|-1` 作为禁带裕量。  
7. `bands_to_dataframe` / `gaps_to_dataframe` 将结构化结果整理为可读表格。  
8. `main` 汇总阈值检查（带数、边界残差、带内违规、gap 有效性）并输出 `PASS/FAIL`。  

说明：第三方库只提供基础数值算子（数组运算和一维根求解）；“物理判据、区段提取、边界构造、验证逻辑”都在源码中显式展开。
