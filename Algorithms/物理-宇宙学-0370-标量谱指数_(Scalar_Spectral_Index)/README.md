# 标量谱指数 (Scalar Spectral Index)

- UID: `PHYS-0352`
- 学科: `物理`
- 分类: `宇宙学`
- 源序号: `370`
- 目标目录: `Algorithms/物理-宇宙学-0370-标量谱指数_(Scalar_Spectral_Index)`

## R01

标量谱指数 `n_s` 描述原初曲率扰动功率谱随尺度 `k` 的倾斜程度，是检验暴胀模型的核心参数之一。若 `n_s=1` 为严格尺度不变谱；观测上通常得到 `n_s<1`（红倾斜）。本条目的 MVP 目标是：在幂律近似下，从一组 `P_s(k)` 数据稳定估计 `n_s` 与 `A_s`，并给出可复现的数值流程。

## R02

实现范围限定为“原初标量功率谱的单参数倾斜估计”最小闭环：

1. 模型：`P_s(k)=A_s (k/k0)^(n_s-1)`（不含 running）
2. 数据：脚本内生成带噪声的合成观测样本（无交互输入）
3. 估计器：
   - 加权线性回归（`scikit-learn`）
   - 非线性最小二乘（`scipy.optimize.curve_fit`）
   - 梯度下降拟合（`PyTorch`）
4. 输出：参数估计值、误差量纲、`chi2/dof` 与方法间一致性检查

## R03

`demo.py` 的输入/输出约定：

- 输入（脚本内部固定）：
1. `k` 范围：`[1e-3, 3e-1] Mpc^-1`
2. 基准点：`k0=0.05 Mpc^-1`
3. 真值：`A_s=2.1e-9`, `n_s=0.965`
4. 噪声：对数空间高斯噪声（异方差）

- 输出：
1. 数据样本表：`k, P_true, P_obs, sigma_lnP`
2. 三种方法的 `A_s`、`n_s`、`chi2/dof`
3. `n_s` 偏差（相对真值）和跨方法离散度

## R04

MVP 使用的核心公式：

1. 原初标量功率谱幂律参数化：
`P_s(k)=A_s * (k/k0)^(n_s-1)`

2. 对数线性化：
`y = ln P_s, x = ln(k/k0)`，则
`y = ln A_s + (n_s-1) x`

3. 加权最小二乘目标（对数空间）：
`min sum_i w_i * (y_i - yhat_i)^2`, 其中 `w_i = 1/sigma_{lnP,i}^2`

4. 曲线拟合卡方：
`chi2 = sum_i ((P_obs,i - P_model,i)/sigma_{P,i})^2`

## R05

算法主流程：

1. 构造 `k` 网格并生成理论谱 `P_true(k)`
2. 在对数空间注入异方差噪声，得到 `P_obs(k)`
3. 用加权线性回归估计 `ln A_s` 与斜率 `n_s-1`
4. 用 `curve_fit` 在原始尺度上做加权非线性拟合
5. 用 PyTorch 对同一目标函数做梯度下降拟合
6. 统一计算 `chi2/dof` 与参数偏差
7. 以 `pandas.DataFrame` 输出对比表，便于验证脚本解析

## R06

正确性依据：

1. `n_s` 在该近似下就是 `ln P_s` 对 `ln k` 的斜率加 1，物理含义直接映射到回归系数
2. 使用 `k0` 作为 pivot，减弱 `A_s` 与 `n_s` 的相关性，提高数值稳定性
3. 三种独立求解器对同一参数化的结果应一致，构成交叉验证
4. 用 `chi2/dof` 判断拟合是否与设定噪声模型同量级匹配

## R07

复杂度分析（`N` 为数据点数，`T` 为 PyTorch 迭代步数）：

1. 数据生成：`O(N)`
2. 线性回归：`O(N)`（特征维度固定为 1）
3. `curve_fit`：约 `O(N * I)`，`I` 为 LM/TRF 迭代次数
4. PyTorch：`O(N * T)`

在默认 `N=80, T=1200` 时，CPU 上通常可在秒级完成。

## R08

数值稳定性设计：

1. 所有功率谱值强制正值下界，避免 `log(0)`
2. 在 `k0=0.05 Mpc^-1` 处展开，缓解截距与斜率共线性
3. 使用异方差权重 `1/sigma^2`，避免高噪声点主导结果
4. 统一采用 `float64`（含 PyTorch）减少小幅参数拟合的舍入误差
5. 固定随机种子，保证回归结果可复现实验

## R09

参数与单位约定：

1. 波数 `k` 单位：`Mpc^-1`
2. 基准波数 `k0=0.05 Mpc^-1`
3. `A_s` 视作在 `k0` 的无量纲谱振幅
4. `n_s` 为无量纲谱指数

该单位体系与常见宇宙学文献参数化兼容，便于后续替换成真实观测链路。

## R11

输出字段解释：

1. `P_true`: 无噪声理论功率谱
2. `P_obs`: 加噪声后的“观测”样本
3. `sigma_lnP`: 对数空间标准差（每个点可不同）
4. `A_s_hat`, `n_s_hat`: 各估计器得到的参数
5. `chi2/dof`: 拟合残差与噪声模型的一致性指标
6. `n_s_bias`: `n_s_hat - n_s_true`

## R12

预期结果特征（与 R04 一致）：

1. 当真值设为 `n_s=0.965` 时，应恢复“红倾斜” (`n_s<1`)
2. 三种方法的 `n_s` 应彼此接近，偏差通常在噪声容忍范围内
3. 若噪声模型与拟合模型匹配，`chi2/dof` 应在 1 附近数量级
4. 拟合曲线在 `log k - log P` 空间近似直线，对应 R04 的线性化关系

## R13

模型边界与简化假设：

1. 仅拟合原初幂律谱，不含 running `alpha_s = dn_s/dlnk`
2. 未引入转移函数、再组合物理、窗口函数卷积等观测链路
3. 数据为合成样本，不替代 CMB/LSS 真正参数估计管线
4. 参数只含 `A_s, n_s` 两个自由度，适合教学与流程验证

## R14

主要失败模式：

1. `k` 动态范围过窄时，`A_s` 与 `n_s` 退化增强
2. 噪声过大或明显非高斯时，`chi2/dof` 失真且估计偏移
3. 若真实谱存在弯曲（running）而仍用单幂律，`n_s` 会被系统偏置
4. 误差估计若严重错误，权重会误导回归结果

## R15

最小测试建议：

1. 正值测试：`P_true`、`P_obs` 均应严格大于 0
2. 恢复测试：默认噪声下 `|n_s_hat - n_s_true| < 0.02`（经验阈值）
3. 一致性测试：三种估计器的 `n_s` 最大差值应较小（如 `<0.01`）
4. 统计测试：`chi2/dof` 不应出现离谱值（如 `>>10`）

## R16

扩展方向（与 R08 稳定性策略相衔接）：

1. 扩展到 `P_s(k)=A_s (k/k0)^(n_s-1 + 0.5*alpha_s*ln(k/k0))` 估计 running
2. 用 MCMC/贝叶斯采样替代点估计，同时沿用 `float64` 与权重归一化策略
3. 以真实观测协方差矩阵替代独立同分布噪声，保留稳定的线性代数求解
4. 与 Boltzmann 求解器输出联动时，继续保留 pivot 选取与参数重标度以降低相关性

## R17

应用场景：

1. 暴胀模型的快速可行性筛查（先验 `n_s` 区间）
2. 宇宙学教学中“谱斜率参数估计”演示
3. 作为更大参数管线（如 `A_s, n_s, tau, Omega_b...`）的单元测试模块
4. 生成可控模拟数据，用于检验统计推断工具链

## R18

`demo.py` 的源码级算法流（拆解第三方库调用）：

1. `generate_synthetic_data` 先显式计算 `P_true=A_s*(k/k0)^(n_s-1)`，再在 `ln P` 空间叠加高斯噪声并指数映射回 `P_obs`。
2. `fit_with_sklearn` 先构造 `x=ln(k/k0)` 与 `y=ln(P_obs)`，调用 `LinearRegression.fit(..., sample_weight=...)` 完成加权线性求解；随后用正规方程 `X^T W X` 手工计算参数协方差与不确定度。
3. `fit_with_scipy` 调用 `scipy.optimize.curve_fit` 对原始非线性模型 `P_s(k)` 直接拟合，`sigma` 参数提供每点误差并返回协方差矩阵。
4. `fit_with_torch` 把 `lnA` 和斜率设为可训练张量，使用 `torch.optim.Adam` 迭代最小化加权残差平方和，得到独立的梯度法解。
5. `evaluate_fit` 统一计算残差与 `chi2/dof`，确保三种方法可在同一指标下比较。
6. `pandas.DataFrame` 汇总方法级参数结果与偏差，并打印固定格式表格，保证输出稳定、可被自动验证。
