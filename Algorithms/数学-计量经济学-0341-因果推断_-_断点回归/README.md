## R01

断点回归（Regression Discontinuity Design, RDD）用于识别“阈值规则”下的局部因果效应。
当处理分配由运行变量 `x` 是否跨过阈值 `c` 决定时，若个体无法精确操控到阈值两侧，
则在 `x=c` 附近，阈值左右样本可近似看作随机分配。

本条目实现的是**锐断点回归（Sharp RD）**最小 MVP：
- 使用局部线性回归估计阈值处跳跃 `tau`
- 用三角核加权强调近阈值样本
- 对带宽做显式网格选择
- 输出稳健标准误与安慰剂断点检验

## R02

识别目标：

`tau = lim_{x -> c+} E[Y|X=x] - lim_{x -> c-} E[Y|X=x]`

关键识别前提（局部）：
- 潜在结果在 `c` 处连续（若无处理，均值函数不跳）
- 处理规则在 `c` 处发生离散跳变（锐断点：`D=1[x>=c]`）
- 个体无法完美操控运行变量跨阈值

在这些条件下，均值跳跃可解释为阈值处局部平均处理效应。

## R03

MVP 输入输出约定：
- 输入：`demo.py` 内部合成数据（无交互输入）
- 输出：
1. 最优带宽和带宽搜索表
2. `tau`、稳健标准误、t 值、p 值、95% CI
3. 窗口内左右侧样本均值差（朴素对照）
4. 安慰剂断点（非真实阈值）估计结果
5. `PASS/FAIL` 形式的最小验证结论

## R04

估计模型（局部线性、允许两侧斜率不同）：

`Y_i = beta0 + tau*D_i + beta1*(X_i-c) + beta2*D_i*(X_i-c) + u_i`

其中：
- `D_i = 1[X_i >= c]`
- 仅使用 `|X_i-c| <= h` 的局部窗口样本
- 核权重 `w_i = K((X_i-c)/h)`，本实现用三角核 `K(u)=max(1-|u|,0)`

则 `tau` 直接对应阈值处左右极限差。

## R05

加权最小二乘（WLS）矩阵形式：

- `beta_hat = (X'WX)^(-1) X'Wy`
- `tau_hat = beta_hat[1]`

实现中不依赖黑箱 RD 包，直接构造设计矩阵、权重矩阵并显式求解线性代数。
若矩阵病态则退化到 `pinv`，保证 MVP 在小样本窗口下仍能运行。

## R06

带宽 `h` 决定偏差-方差权衡：
- `h` 太小：方差大（有效样本少）
- `h` 太大：偏差大（远离阈值的函数形状污染局部估计）

本 MVP 使用**候选网格搜索**：
1. 设定一组候选带宽
2. 对每个 `h` 运行局部线性 WLS
3. 计算核加权 MSE
4. 选取 MSE 最小的 `h`

这不是最复杂的 IK/CCT 自动带宽，但流程透明、可审计、便于教学复现。

## R07

不确定性估计采用异方差稳健 sandwich 形式（HC 类）：

`Var(beta_hat) = B * M * B`

- `B = (X'WX)^(-1)`
- `M = X' diag(w_i^2 * e_i^2) X`

并做有限样本修正 `n/(n-p)`。
最终报告：
- `se(tau_hat)`
- t 统计量
- 双侧 p 值（Student-t 近似）
- 95% 置信区间

## R08

算法主流程：
1. 生成运行变量 `x` 与结果 `y`（含真实阈值跳跃）
2. 构造候选带宽集合
3. 逐个带宽执行局部线性 WLS，并记录加权 MSE
4. 选取最优带宽
5. 在最优带宽下输出 `tau` 与稳健推断统计
6. 计算窗口内左右样本均值差作直观对照
7. 在安慰剂阈值重复同样估计，检查“伪跳跃”
8. 打印验证结论

## R09

复杂度（设样本量 `n`，候选带宽个数 `G`）：
- 单次 WLS：构建设计矩阵与法方程约 `O(n)`，矩阵求逆维度固定（4x4）可视为常数
- 带宽网格：`O(G*n)`
- 安慰剂检验（`m` 个断点）：`O(m*G*n)`

对于教学规模 `n<=1e5` 的简化场景，Python 实现通常可接受。

## R10

`demo.py` 的数据生成策略：
- `x ~ Uniform[-1, 1]`
- 基线函数 `mu(x)` 平滑连续（多项式 + 正弦项）
- 真实处理效应 `tau_true=1.5` 在 `x>=0` 时加到结果上
- 噪声方差随 `|x|` 变化（异方差），用于验证稳健标准误必要性

这使示例更接近实证环境，而非理想同方差教科书数据。

## R12

输出字段解释：
- `Selected bandwidth`: 最优带宽
- `tau_hat`: 断点处处理效应估计
- `robust_se`: 异方差稳健标准误
- `t_stat / p_value`: 显著性检验结果
- `ci95_low / ci95_high`: 95% 置信区间
- `naive_mean_gap`: 同窗口内处理组与对照组均值差
- `placebo` 表：伪断点估计，应围绕 0 附近波动

## R13

鲁棒性与边界处理：
- 若某候选带宽导致单侧样本不足，自动跳过
- 若加权法方程接近奇异，使用 `pinv` 代替直接逆
- 若稳健标准误非正或不可用，返回 NaN 并标记失败
- 全流程固定随机种子，保证可复现

## R16

最小测试建议：
1. 真实效应测试：`tau_true` 设为正值，估计应显著为正
2. 零效应测试：把 `tau_true=0`，估计应接近 0
3. 样本量敏感性：增加 `n` 后估计方差应下降
4. 带宽敏感性：手动固定多个 `h`，观察偏差-方差变化
5. 安慰剂阈值：非真实断点估计应无系统显著跳跃

## R17

方法比较（实现视角）：
- 黑箱库一键 RD：
  - 优点：快
  - 缺点：不透明，难定位估计差异来源
- 直接全局多项式回归：
  - 优点：实现简单
  - 缺点：对函数形式敏感，局部识别解释较弱
- 本实现（局部线性 + 核加权 + 稳健 SE）：
  - 优点：贴近经典 RD 实证主线，细节可审计
  - 缺点：还未加入高级偏差校正与完整诊断

## R18

`demo.py` 源码级算法流（8 步，非黑箱）：
1. `simulate_sharp_rd_data` 生成 `x,y`：先构造连续基线 `mu(x)`，再在 `x>=cutoff` 加入真实跳跃 `tau_true`，噪声由 `torch.randn` 产生并转为 `numpy`。
2. `triangular_kernel` 计算核权重 `w_i=max(1-|u_i|,0)`，只保留局部窗口信息。
3. `fit_local_linear_rd` 按 `|x-c|<=h` 筛样本，构造显式设计矩阵 `[1, D, x-c, D*(x-c)]`。
4. 在同一函数内用矩阵公式 `beta_hat=(X'WX)^(-1)X'Wy` 求 WLS 系数，`tau_hat` 直接取第二个系数。
5. 继续基于残差 `e_i` 计算 sandwich 稳健协方差 `B*(X'diag(w^2 e^2)X)*B`，得到 `robust_se`、t 值、p 值和置信区间。
6. `select_bandwidth` 对候选 `h` 循环调用第 3-5 步，并用 `sklearn.metrics.mean_squared_error(..., sample_weight=w)` 记录核加权 MSE，选最小者。
7. `run_placebo_tests` 在多个伪断点重复第 6 步流程，检查估计是否在非真实阈值处出现系统性跳跃。
8. `main` 汇总主断点结果、朴素均值差、带宽搜索表和安慰剂表，用 `pandas.DataFrame` 格式化打印并给出 `PASS/FAIL` 结论。
