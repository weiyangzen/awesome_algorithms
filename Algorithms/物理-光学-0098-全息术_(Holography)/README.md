# 全息术 (Holography)

## R01

全息术的核心是记录并重建**光场的复振幅（幅度+相位）**，而不是只记录强度。

对物光 `O(x,y)` 与参考光 `R(x,y)`，全息图记录的是干涉强度：

`I(x,y) = |O + R|^2 = |O|^2 + |R|^2 + O R* + O* R`

其中交叉项 `O R*`、`O* R` 携带相位信息，是可重建三维信息的关键。

## R02

本条目选择数字离轴全息（off-axis digital holography）的最小可验证闭环：

1. 生成一个二维复物体光场；
2. 数值传播到传感器面；
3. 与倾斜参考波干涉得到全息图强度；
4. 解调与频域滤波提取一阶项；
5. 反向传播恢复物体平面并量化误差。

目标是把“干涉记录 -> 数字重建 -> 指标验收”完整打通。

## R03

MVP 边界与假设：

- 标量衍射近似，忽略偏振矢量效应；
- 采用角谱法（Angular Spectrum）做自由空间传播；
- 参考光为单一倾斜平面波；
- 解调后使用圆形低通窗口提取带限物光；
- 不建模传感器噪声、像素非均匀、镜头像差等工程误差。

## R04

`demo.py` 使用的核心方程：

1. 角谱传播
`U_z = F^{-1}{ F{U_0} * H(fx,fy,z) }`

2. 传播传递函数
`H = exp(i z kz),  kz = sqrt(k^2 - kx^2 - ky^2),  k = 2pi/lambda`

3. 离轴参考波
`R(x,y) = A_ref * exp(i 2pi(fx_ref x + fy_ref y))`

4. 全息记录
`I = |U_obj_sensor + R|^2`

5. 解调
`I_demod = I * exp(i 2pi(fx_ref x + fy_ref y))`

6. 低通重建传感器复场
`U_hat_sensor = LPF{I_demod} / A_ref`

7. 反向传播重建物面
`U_hat_obj = Propagate(U_hat_sensor, -z)`

## R05

设采样网格为 `N x N`：

- 单次 `FFT2/IFFT2` 复杂度 `O(N^2 log N)`；
- 全流程包含多次 2D FFT，主导复杂度仍为 `O(N^2 log N)`；
- 其余逐点运算（干涉、解调、掩膜）为 `O(N^2)`；
- 空间复杂度 `O(N^2)`（保存复场、频谱与中间结果）。

## R06

`demo.py` 输出三类可验收结果：

- **重建指标表**：幅度 RMSE、幅度相关系数、相位 MAE、复场相对误差；
- **谱分离/干涉质量指标**：载频侧带与零级直流功率比、条纹对比度；
- **中心线样本表**：目标与重建幅度在若干空间点的对照。

并通过阈值检查给出 `Validation: PASS/FAIL`。

## R10

正确性直觉：

1. 物光与参考光干涉后，交叉项携带物光相位；
2. 参考光倾角引入载频，使交叉项与零级项在频域可分离；
3. 解调将目标交叉项搬到基带，低通去除高频干扰项；
4. 若传播模型与反传播距离一致，重建应回到原物面；
5. 去除全局复增益（振幅比例+全局相位）后，重建误差应显著下降。

## R11

数值稳定与精度策略：

- `kz = sqrt(k^2-kx^2-ky^2+0j)` 允许复数分支，避免非法开方；
- 所有误差指标都做归一化或相对化，降低量纲干扰；
- 相位误差仅在有效振幅区域统计（阈值掩膜），避免低信噪比点主导；
- 频域采用圆形低通，减少方窗引入的方向性伪影；
- 验证指标多维度组合（幅度、相位、复场、频谱质量）而非单指标。

## R12

默认参数（见 `HolographyConfig`）：

- 波长 `lambda = 633 nm`
- 像素间距 `pixel_pitch = 8 um`
- 网格 `320 x 320`
- 传播距离 `z = 0.08 m`
- 参考波载频 `(fx_ref, fy_ref) = (18000, 12000) cycles/m`
- 边带低通截止 `7500 cycles/m`

调参建议：

- 若边带混叠，增大参考载频或降低物体带宽；
- 若重建细节损失，适当增大 `sideband_cutoff`；
- 若频谱折叠，需减小像素间距或缩小目标空间频率内容。

## R14

常见失效模式：

1. 参考载频过低，交叉项与零级项在频域重叠；
2. 低通截止过小，目标带宽被截断导致模糊；
3. 低通截止过大，引入孪生像和零级泄漏；
4. 传播距离或像素参数与记录模型不一致，导致失焦或畸变；
5. 物体振幅过弱时，相位评估受噪声主导。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `make_grid` 生成空间采样网格 `x,y,xx,yy`，确定离散光场坐标系统。  
2. `build_object_field` 构造二维复物体 `U0(x,y)`，把幅度纹理和相位结构显式编码。  
3. `angular_spectrum_propagate(..., +z)` 计算物光从物面到传感器面的传播，得到 `U_obj_sensor`。  
4. `build_reference_wave` 生成倾斜参考平面波 `R`，`record_hologram_intensity` 形成干涉全息图 `I=|U_obj_sensor+R|^2`。  
5. `estimate_carrier_to_dc_ratio` 在全息图频谱中统计侧带/零级功率比，检查离轴分离质量。  
6. `demodulate_and_lowpass` 对 `I` 乘载频相因子做解调，再在频域施加圆形低通，提取近似 `O R*` 项并除以 `A_ref`。  
7. `angular_spectrum_propagate(..., -z)` 反向传播得到重建物面复场；`align_global_complex_scale` 去除全局复增益歧义。  
8. `compute_metrics` 与 `main` 汇总幅度/相位/复场误差与阈值断言，输出 `Validation: PASS/FAIL`。  

第三方包职责拆解：

- `numpy`：复数数组、点运算、误差统计；
- `scipy.fft`：`fft2/ifft2/fftfreq` 频域变换与坐标；
- `pandas`：结构化打印结果表。

算法主体（传播方程、干涉记录、解调滤波、对齐与验收指标）均在源码中显式展开，不是黑盒调用。
