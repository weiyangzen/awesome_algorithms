## R01

暗能量问题的核心是解释宇宙加速膨胀。观测上常使用 Ia 型超新星（标准烛光）距离模数 `mu(z)` 约束宇宙学参数。

本目录给出一个最小可运行 MVP：
- 采用平坦 `wCDM` 模型（`Omega_m + Omega_de = 1`，且暗能量状态方程参数 `w` 为常数）；
- 使用可复现实验的合成超新星数据；
- 用三阶段粗到细网格搜索拟合 `Omega_m` 与 `w`；
- 输出 `chi^2/dof`、误差指标与近似 1-sigma 参数区间。

## R02

问题定义（本目录实现）：
- 输入：
  - 一组红移 `z_i`；
  - 观测距离模数 `mu_obs,i`；
  - 观测误差 `sigma_mu,i`；
  - 固定 `H0`（本 MVP 固定为 `70 km/s/Mpc`）。
- 参数：
  - `Omega_m`：当前宇宙物质密度参数；
  - `w`：暗能量状态方程参数（常数）。
- 目标：
  - 最小化
    `chi^2(Omega_m, w) = sum_i ((mu_obs,i - mu_model(z_i; Omega_m, w))/sigma_mu,i)^2`。
- 输出：
  - 最优参数估计 `(Omega_m_hat, w_hat)`；
  - 拟合优度 `chi^2`、`chi^2/dof`；
  - 对观测的回归指标 `MAE/RMSE/R^2`；
  - 近似 1-sigma 区间盒。

## R03

数学模型（平坦 `wCDM`）：

1. 归一化哈勃参数
   `E(z) = H(z)/H0 = sqrt(Omega_m*(1+z)^3 + (1-Omega_m)*(1+z)^(3*(1+w)))`。
2. 视线方向无量纲共动距离
   `chi(z) = int_0^z dz'/E(z')`。
3. 光度距离（Mpc）
   `d_L(z) = (1+z) * (c/H0) * chi(z)`。
4. 距离模数
   `mu(z) = 5*log10(d_L/Mpc) + 25`。
5. 目标函数
   `chi^2 = r^T r`，其中 `r_i = (mu_obs,i - mu_model,i)/sigma_mu,i`。

## R04

算法流程（高层）：
1. 生成固定随机种子的合成超新星样本，形成 `DataFrame`。
2. 校验数据列、有限性、`z>0` 与 `sigma_mu>0`。
3. 在大范围参数盒做第一阶段网格扫描，得到粗最优点。
4. 以阶段一最优点为中心缩小参数范围，执行第二阶段扫描。
5. 再次缩小范围，执行第三阶段高分辨率局部扫描。
6. 取第三阶段最优点作为最终估计。
7. 依据 `Delta chi^2 = 2.30`（两参数近似 1-sigma）从最终网格提取参数区间。
8. 输出拟合指标与最终阶段 top-k 候选点。

## R05

核心数据结构：
- `pandas.DataFrame`（观测表）：
  - `z`：红移；
  - `mu_obs`：带噪声观测距离模数；
  - `sigma_mu`：测量误差；
  - `mu_true`：无噪声参考真值（用于合成数据审计）。
- `FitResult`：
  - `omega_m`；
  - `w`；
  - `chi2`。
- 扫描结果表（`DataFrame`）：
  - 每行一个参数组合 `(omega_m, w, chi2)`，按 `chi2` 升序排序。

## R06

正确性要点：
- 物理可行性：限制 `0 < Omega_m < 1`，保证 `E(z)` 与暗能量密度项可解释。
- 数值稳定性：
  - 先按 `z` 排序后积分，避免积分网格乱序；
  - 使用梯形积分 `cumulative_trapezoid` 构造 `chi(z)`。
- 统计一致性：
  - 采用加权残差定义 `chi^2`，误差条 `sigma_mu` 被显式使用。
- 可验证性：
  - 数据生成与拟合全流程固定随机种子，可重复复现实验结果。

## R07

复杂度分析：
- 设样本数为 `N`，总扫描参数点数为 `K`（三阶段总和）。
- 单个参数点评估：
  - 构建 `E(z)` 与一次累积积分，复杂度 `O(N)`。
- 总时间复杂度：`O(K*N)`。
- 空间复杂度：
  - 数据与临时数组 `O(N)`；
  - 保存扫描表 `O(K)`。

在当前默认设置下，`K` 约为 `61*81 + 91*91 + 101*101`，规模可控。

## R08

边界与异常处理：
- 输入校验：
  - 缺失列时抛 `ValueError`；
  - `z<=0`、`sigma_mu<=0` 或存在 `nan/inf` 时抛 `ValueError`。
- 参数校验：
  - `h0<=0`、`omega_m` 不在 `(0,1)` 时抛异常。
- 数值异常：
  - `E(z)` 非有限或非正、`d_L` 非有限或非正时抛 `RuntimeError`。
- 区间异常：
  - 局部扫描裁剪后区间无效时抛 `RuntimeError`。

## R09

MVP 取舍说明：
- 只做常数 `w` 的平坦 `wCDM`，不扩展到 `w0-wa` 或曲率 `Omega_k`。
- 用合成数据演示算法链路，不依赖外部数据下载。
- 优先透明可审计：采用显式网格搜索替代黑盒优化器。
- 不做系统误差、协方差矩阵与绝对星等零点边缘化，仅保留最小可运行主干。

## R10

`demo.py` 主要函数职责：
- `validate_dataframe`：输入数据完整性与合法性检查。
- `e_of_z`：计算平坦 `wCDM` 的 `E(z)`。
- `distance_modulus_flat_wcdm`：由 `E(z)` 数值积分得到 `mu(z)`。
- `make_synthetic_snia`：生成带噪声可复现超新星样本。
- `chi2_stat`：计算给定参数的 `chi^2`。
- `grid_scan`：在参数网格上枚举并返回最优点与排序表。
- `clip_interval`：围绕当前最优点构建下一阶段局部搜索区间。
- `fit_dark_energy_wcdm`：执行三阶段扫描。
- `estimate_1sigma_box`：按 `Delta chi^2` 提取近似区间。
- `main`：组织实验、输出指标和 top-5 候选。

输出字段说明：

- 若运行慢：降低网格分辨率或减少阶段数。

## R15

方法对比：
- 对比固定 `LambdaCDM`（`w=-1`）：
  - 固定 `w` 可减少参数退化，但无法检验是否偏离宇宙常数。
- 对比 MCMC：
  - MCMC 可给完整后验分布；
  - 本 MVP 网格扫描更直观、可控、实现更短。
- 对比黑盒最优化（如一键最小化）：
  - 黑盒迭代效率可能更高；
  - 本实现每个候选点评估路径完全显式，更适合教学与审计。

## R18

`demo.py` 源码级算法流程（8 步）：
1. `main` 固定 `H0` 与真值参数，调用 `make_synthetic_snia` 生成 `z`、`mu_obs`、`sigma_mu`、`mu_true`。 
2. `validate_dataframe` 检查列完整性、有限性、`z>0` 与 `sigma_mu>0`，提前阻断非法输入。 
3. `fit_dark_energy_wcdm` 启动第一阶段 `grid_scan`，在大参数盒中逐点枚举 `(Omega_m, w)`。 
4. 对每个候选点，`chi2_stat` 调用 `distance_modulus_flat_wcdm`：
   - 先按红移排序；
   - 用 `e_of_z` 计算 `E(z)`；
   - 用 `cumulative_trapezoid` 计算 `chi(z)`；
   - 得到 `d_L` 与 `mu_model`。 
5. `chi2_stat` 将残差按 `sigma_mu` 加权并累加得到 `chi^2`，`grid_scan` 记录到表并维护当前最优点。 
6. 以阶段一最优点为中心，`clip_interval` 构建更小搜索盒，重复第二、第三阶段扫描实现粗到细定位。 
7. 最终最优点用于生成 `mu_fit`，并计算 `chi^2/dof`、`MAE`、`RMSE`、`R2`；`estimate_1sigma_box` 从末阶段网格中按 `Delta chi^2=2.30` 提取近似区间。 
8. `main` 打印真值、拟合值、区间估计及最终局部网格 top-5，完成一次可复现实验闭环。
