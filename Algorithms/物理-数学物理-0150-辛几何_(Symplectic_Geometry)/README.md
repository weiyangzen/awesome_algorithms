# 辛几何 (Symplectic Geometry)

- UID: `PHYS-0149`
- 学科: `物理`
- 分类: `数学物理`
- 源序号: `150`
- 目标目录: `Algorithms/物理-数学物理-0150-辛几何_(Symplectic_Geometry)`

## R01

辛几何研究“相空间上的面积/体积结构如何在动力学下保持不变”。在最常见的正则坐标 `(q, p)` 中，辛形式写作：

`omega = dq ∧ dp`。

若映射 `Phi` 满足 `D Phi^T J D Phi = J`（`J=[[0,1],[-1,0]]`），则称其为辛映射。哈密顿系统的真实时间演化天然是辛映射，这也是长期数值模拟中“几何结构保持”方法的核心依据。

## R02

历史定位（简要）：

- 从哈密顿力学出发，辛结构成为经典力学相空间的标准语言；
- 庞加莱等人的工作揭示了长期动力学中结构保持的重要性；
- 现代几何数值积分（Geometric Integration）把“离散更新也保持辛结构”作为关键设计目标；
- 本目录聚焦可运行 MVP：用最小代码展示“辛积分器 vs 非辛积分器”的结构差异。

## R03

MVP 输入输出约定（无交互输入）：

- 输入（脚本内部固定）：
1. 一维谐振子哈密顿量 `H(q,p)=0.5*(p^2+omega^2 q^2)`；
2. 初值 `(q0,p0)=(1,0)`；
3. `omega=1.0`、步长 `h=0.1`、步数 `n_steps=800`。
- 输出：
1. 显式欧拉与辛欧拉的最大能量漂移；
2. 两种一步映射的辛性误差 `||A^T J A - J||_F`；
3. 两种一步映射行列式；
4. 终态对比与 `All checks passed.` 校验结论。

## R04

典型应用：

- 天体力学长时间轨道积分；
- 分子动力学中的哈密顿系统离散化；
- Hamiltonian Monte Carlo (HMC) 中的相空间推进；
- 约束系统与刚体系统的结构保持仿真。

## R05

数学要点：

- 哈密顿方程：
  `q_dot = dH/dp`, `p_dot = -dH/dq`；
- 对谐振子 `H=0.5*(p^2+omega^2 q^2)` 有：
  `q_dot=p`, `p_dot=-omega^2 q`；
- 辛条件（离散映射雅可比 `A`）：
  `A^T J A = J`；
- 在二维正则相空间中，辛映射还满足 `det(A)=1`（必要且与辛条件一致于此线性场景）。

## R06

本实现核心流程：

1. 定义谐振子哈密顿量与两种一步更新（显式欧拉、辛欧拉）；
2. 在相同初值和步长下分别积分 `800` 步；
3. 记录全轨迹与离散能量序列；
4. 构造两种一步映射的线性雅可比矩阵；
5. 计算辛性误差 `||A^T J A - J||_F` 与 `det(A)`；
6. 用断言检查：辛欧拉应显著更保结构、非辛欧拉应显著漂移。

## R07

工程判据（MVP）：

- 辛欧拉辛性误差阈值：`<=1e-12`；
- 显式欧拉辛性误差：`>=1e-6`（本参数下约 `1e-2`）；
- 辛欧拉 `det(A)` 要接近 `1`；显式欧拉在本设置下 `det(A)>1`；
- 辛欧拉最大能量漂移要求 `<0.05`；显式欧拉要求 `>1.0`；
- 校验不通过立即抛 `AssertionError`，避免“能跑但结论错误”。

## R08

时间复杂度：

- 两种方法每步都是常数复杂度 `O(1)`；
- 共 `n_steps` 步，单方法 `O(n_steps)`；
- 本 demo 同时运行两种方法，整体仍为 `O(n_steps)`（常数因子约 2）。

## R09

空间复杂度：

- 保存轨迹 `q[0..N], p[0..N], energy[0..N]`，单方法 `O(n_steps)`；
- 两种方法并行保存，整体 `O(n_steps)`；
- 若只关心终值与最大漂移，可进一步降到 `O(1)` 流式内存。

## R12

输出字段说明：

- `explicit_max_energy_drift`：显式欧拉 `max |H_n-H_0|`；
- `symplectic_max_energy_drift`：辛欧拉 `max |H_n-H_0|`；
- `explicit_final_state` / `symplectic_final_state`：两种方法的终态；
- `explicit_symplecticity_error`：显式欧拉的 `||A^T J A - J||_F`；
- `symplectic_symplecticity_error`：辛欧拉的对应误差；
- `det_explicit` / `det_symplectic`：两种一步映射雅可比行列式；
- `All checks passed.`：全部结构与数值判据通过。

## R13

鲁棒性与边界处理：

- 通过解析雅可比而非数值差分计算辛性误差，避免差分噪声；
- 断言同时覆盖结构性质（辛条件、行列式）和动力学表现（能量漂移）；
- 固定参数与初值，保证可重复输出；
- 本 MVP 为教学可审计代码，优先透明与可验证，不做过度工程封装。

## R16

最小测试建议：

- 结构性：验证辛欧拉 `A^T J A ≈ J`；
- 区分性：验证显式欧拉辛误差显著非零；
- 能量性：辛欧拉漂移有界，显式欧拉出现明显漂移；
- 一致性：固定随机性/参数后结果可复现；
- 退化检查：改小步长时，两者局部误差都减小但结构差异仍存在。

## R17

方案对比：

- 显式欧拉：实现最简单，但不保辛，长期行为常失真；
- 辛欧拉：同为一阶，但保辛结构，长期定性行为更可信；
- RK4：短期高精度但通常不保辛，长期可能出现系统性漂移；
- 更高阶辛方法（如 Verlet、分裂法）：在保持结构前提下提升精度，是工程常用扩展。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `hamiltonian()` 定义谐振子能量 `H(q,p)`，作为守恒量监控基准。  
2. `explicit_euler_step()` 与 `symplectic_euler_step()` 分别给出非辛/辛的一步离散更新。  
3. `integrate()` 用统一框架迭代 `n_steps`，记录整段 `(q,p)` 轨迹及 `energy` 序列。  
4. `jacobian_explicit_euler()` 和 `jacobian_symplectic_euler()` 显式写出两种映射的雅可比矩阵。  
5. `symplecticity_error()` 计算 `||A^T J A - J||_F`，直接衡量“是否保持辛形式”。  
6. `run_checks()` 先验证辛欧拉结构保持（小误差、`det=1`），再验证显式欧拉结构破坏（误差明显、`det>1`）。  
7. 同一函数中再比较两者能量漂移阈值，确保“几何差异”在动力学指标上可观测。  
8. `main()` 汇总打印关键数值并输出 `All checks passed.`，形成可直接复现实验结论的最小闭环。  

说明：本实现没有把辛几何问题交给现成黑箱求解器；离散映射、雅可比、辛条件检验与误差判据都在源码中逐步展开。
