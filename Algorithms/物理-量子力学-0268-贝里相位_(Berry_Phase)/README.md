# 贝里相位 (Berry Phase)

- UID: `PHYS-0265`
- 学科: `物理`
- 分类: `量子力学`
- 源序号: `268`
- 目标目录: `Algorithms/物理-量子力学-0268-贝里相位_(Berry_Phase)`

## R01

贝里相位是量子系统在参数空间中做绝热闭合演化时获得的几何相位，与动力学相位无关。对给定参数回路 `C` 和瞬时本征态 `|u(\lambda)\rangle`，几何相位写为：

`γ_B = i ∮_C <u(\lambda)|∇_\lambda u(\lambda)> · d\lambda`

本任务的算法目标是：给定一条离散化闭合路径，数值计算 `γ_B`，并验证其规范不变性（gauge invariance）。

## R02

物理模型采用最小可验证系统：自旋 `1/2` 在方向变化的单位磁场中。

- 哈密顿量：`H(θ, φ) = - n(θ, φ) · σ`
- `n(θ, φ) = (sinθ cosφ, sinθ sinφ, cosθ)`
- `σ = (σ_x, σ_y, σ_z)` 为泡利矩阵

沿固定 `θ`，`φ: 0 → 2π` 的闭合回路，基态的理论贝里相位是：

`γ_B = -Ω/2 = -π(1 - cosθ)`

其中 `Ω` 是 Bloch 球面上回路包围的立体角。

## R03

数值任务输入输出定义：

- 输入：
1. 极角 `θ`
2. 路径离散点数 `N`
3. 参数化路径 `φ_j = 2πj/N`

- 输出：
1. 数值贝里相位 `γ_num`
2. 解析相位 `γ_ana`
3. 误差 `Δγ = wrap(γ_num - γ_ana)`
4. 规范变换前后差值 `Δgauge`

## R04

离散闭合链路上的规范不变公式（Wilson-loop 形式）：

`γ_num = -arg Π_{j=0}^{N-1} <u_j | u_{j+1}>`，其中 `u_N = u_0`。

该表达式直接避免了对 `∇u` 的数值微分，且对局域 U(1) 变换
`|u_j> -> e^{iχ_j}|u_j>` 不变。因为相位因子在乘积中逐项相消。

## R05

MVP 的算法步骤：

1. 用 `(θ, φ_j)` 构造每个点的 `2x2` 厄米哈密顿量 `H_j`。
2. 对每个 `H_j` 做本征分解，取最小本征值对应的本征矢 `|u_j>` 作为基态。
3. 计算邻点重叠 `s_j = <u_j|u_{j+1}>`（末点回到起点）。
4. 对每个 `s_j` 取单位相位因子 `s_j / |s_j|`（加入极小值保护）。
5. 取所有单位相位因子乘积并求辐角，得到 `γ_num`。
6. 用解析式 `γ_ana = -π(1-cosθ)` 做误差对比。
7. 对每个 `|u_j>` 乘随机 `e^{iχ_j}` 再算一次，验证 `Δgauge≈0`。

## R06

正确性核心理由：

1. 绝热近似下，几何相位由本征态纤维丛的平行移动决定，与动力学相位可分离。
2. 离散重叠乘积是路径有序指数的离散近似，`N` 增大时收敛到连续线积分。
3. 对局域 U(1) 规范变换，离散公式的链式相位完全望文生义地“望远镜消去”，只留闭环净相位。
4. 在该模型上有已知解析解，可构成强可验证基准。

## R07

复杂度（单次 `N` 点计算）：

- 时间复杂度：`O(N)`
- 空间复杂度：`O(N)`（存整条路径态矢）；若流式处理可降为 `O(1)`

原因：每个点只做常数规模（`2x2`）厄米对角化和常数次复数运算。

## R08

数值稳定性与处理：

1. 相位包裹：最终角度用 `wrap_to_pi` 映射到 `(-π, π]`。
2. 近零重叠：当离散过粗或跨越近简并区时，`|<u_j|u_{j+1}>|` 可能很小。实现中用 `max(|s_j|, 1e-15)` 防止除零。
3. 分支切换：使用整条乘积后再 `arg`，减小逐段取角度造成的分支跳跃误差。
4. 模型层面避免简并：`|n|=1` 固定，能隙恒定（`2`），便于演示绝热几何相位。

## R09

规范不变性检查策略：

- 原始态：`|u_j>`
- 扰动态：`|u'_j> = e^{iχ_j}|u_j>`，`χ_j` 为独立随机相位
- 指标：`Δgauge = wrap(γ_num - γ'_num)`

预期 `Δgauge` 在浮点误差量级（通常 `1e-12 ~ 1e-15`）。这比“看图像”更直接，是算法层面的硬验证。

## R10

实现选择与取舍：

1. 仅依赖 `numpy`，避免额外框架开销。
2. 选择自旋 `1/2` 模型，因为解析式清晰、矩阵小、运行快。
3. 不做交互输入，参数在 `main()` 固定，保证 `uv run python demo.py` 可直接执行。
4. 输出表格包含 `N`、数值相位、解析误差、规范差，便于自动校验。

## R12

输出字段解释：

1. `gamma_numeric(rad)`：离散 Wilson-loop 算得的贝里相位。
2. `error(rad)`：与解析值的差（已包裹到 `(-π, π]`）。
3. `gauge_diff(rad)`：随机规范变换前后的相位差。

随着 `N` 增大，`error` 应收敛到 `0`；`gauge_diff` 应接近机器精度。

## R13

可预期的数值行为：

1. 小 `N` 时离散化误差较大，但趋势正确。
2. `N` 翻倍后，误差通常显著下降。
3. `max |gauge_diff|` 极小，说明算法确实在算几何量而非规范依赖量。

## R14

边界情况与失败模式：

1. `N` 太小（如 `N=3`）会导致路径离散过粗，误差明显。
2. 若参数路径穿过简并点（能隙闭合），绝热假设失效，贝里相位定义需更谨慎。
3. 若相邻态几乎正交，重叠相位数值不稳定，需要加密网格或改用并行输运规范。

## R15

最小测试建议：

1. 基础回归：固定 `θ=60°`，检查 `N=512` 时 `|error| < 1e-3`。
2. 规范回归：随机规范下 `max |gauge_diff| < 1e-10`。
3. 物理极限：`θ→0` 时 `γ→0`；`θ→π` 时 `γ→-2π`（包裹后等价于 `0`）。

## R18

`demo.py` 的源码级算法流（避免把 `numpy.linalg.eigh` 当黑箱）：

1. 生成参数网格 `φ_j`，构建每个点的 `H_j = -n_j·σ`（显式泡利矩阵线性组合）。
2. 调用 `numpy.linalg.eigh(H_j)`；该接口在底层走 LAPACK 的厄米本征求解例程（通常 `heevd` 分治路线）。
3. LAPACK 先把厄米矩阵用酉相似变换约化到实对称三对角形式（保持本征谱不变）。
4. 对三对角问题做分治/QR 型特征求解，得到本征值与三对角空间本征向量。
5. 把三对角空间向量回代到原始基，得到 `H_j` 的本征向量矩阵；MVP 取最小本征值对应列作为 `|u_j>`。
6. 对整条回路计算相邻内积 `s_j=<u_j|u_{j+1}>` 并首尾闭合。
7. 把 `s_j` 归一化为纯相位因子后连乘，取 `-arg(...)` 得到离散几何相位 `γ_num`。
8. 施加随机局域 U(1) 规范后重复第 6-7 步，比较 `Δgauge` 验证规范不变性。
