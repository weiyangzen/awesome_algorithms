# 霍尔效应 (Hall Effect)

- UID: `PHYS-0193`
- 学科: `物理`
- 分类: `电磁学`
- 源序号: `194`
- 目标目录: `Algorithms/物理-电磁学-0194-霍尔效应_(Hall_Effect)`

## R01

霍尔效应描述了载流导体/半导体在外加磁场中产生横向电压的现象。

在样品厚度为 `t`、电流 `I` 沿 `x` 方向、磁场 `B` 沿 `z` 方向时，横向霍尔电压满足：

`V_H = R_H * I * B / t`

其中 `R_H` 为霍尔系数：
- `R_H < 0` 常对应电子主导导电（n 型）；
- `R_H > 0` 常对应空穴主导导电（p 型）。

## R02

本目录 MVP 解决的问题是“由测量电压反推霍尔参数”：

- 输入：
  - 多组电流 `I_k`；
  - 多组磁场幅值 `B_k`（对应 `+B/-B` 成对测量）；
  - 观测电压 `V(+B)_k` 与 `V(-B)_k`；
  - 样品厚度 `t`。
- 输出：
  - 霍尔系数估计 `R_H`；
  - 拟合质量指标（`R^2`、截距、误差）；
  - 载流子类型（n/p）与载流子浓度估计。

脚本使用内置合成数据，`uv run python demo.py` 可直接运行，无需交互输入。

## R03

核心物理与建模公式：

1. 霍尔电压关系：`V_H = R_H * I * B / t`。  
2. 通过反向磁场消除偶次寄生项：  
   `V_H = (V(+B) - V(-B)) / 2`。  
3. 线性回归变量定义：  
   `y = V_H`, `x = I*B/t`, 拟合 `y = R_H * x + c`。  
4. 单载流子近似下浓度关系：  
   `n = 1 / (|R_H| * q)`，其中 `q = e = 1.602176634e-19 C`。

## R04

算法高层流程：

1. 构造或读取 `I/B/V(+B)/V(-B)/t` 数据。  
2. 校验数据形状、有限性与厚度合法性。  
3. 计算反对称电压 `V_H = (V(+B)-V(-B))/2`。  
4. 构造特征 `x = I*B/t`。  
5. 用最小二乘显式求解 `y = R_H*x + c`。  
6. 计算 `R^2`、残差、截距等质量指标。  
7. 根据 `R_H` 的符号判定载流子类型。  
8. 根据 `|R_H|` 计算载流子浓度并输出结果。

## R05

核心数据结构：

- `HallCaseConfig`（`dataclass`）：
  - `true_r_h`：真实霍尔系数（用于合成数据对照）；
  - `thickness`：样品厚度；
  - `offset_uV`：电压偏置项；
  - `even_coeff_uV_per_T2`：偶次寄生项（`B^2`）；
  - `noise_uV`：噪声量级。
- 数组数据：
  - `current`, `b_field`, `v_plus`, `v_minus` 均为 `numpy.ndarray` 1D。
- 字典结果：
  - 拟合输出（`r_h_hat`, `intercept`, `r2` 等）；
  - 物性推断输出（`carrier_type`, `carrier_density_m^-3`）。

## R06

正确性要点：

- 物理关系正确：使用标准霍尔关系 `V_H = R_H I B / t`。  
- 偏置抑制正确：`(+B,-B)` 反对称化去除常量偏置与偶次项。  
- 估计透明：`np.linalg.lstsq` 只做线性代数求解，未调用黑盒物理库。  
- 结果可解释：`R_H` 符号直接映射 n/p 类型，量纲与浓度公式一致。  
- 数值可验证：脚本内置两组不同极性样本，并设置误差门限断言。

## R07

复杂度分析（样本点数为 `N`）：

- 数据预处理与反对称化：`O(N)`；
- 构造设计矩阵并最小二乘：此处是 2 参数线性回归，计算量 `O(N)`；
- 汇总与诊断指标：`O(N)`。

总时间复杂度 `O(N)`，空间复杂度 `O(N)`（保留所有观测向量）。

## R08

边界与异常处理：

- 任一输入不是一维向量：抛 `ValueError`。  
- 输入长度不一致：抛 `ValueError`。  
- 样本数少于 3：抛 `ValueError`。  
- 厚度 `t <= 0` 或非有限：抛 `ValueError`。  
- 回归矩阵退化（秩不足）：抛 `RuntimeError`。  
- `R_H = 0` 或非有限时做载流子推断：抛 `ValueError`。

## R09

MVP 取舍说明：

- 只覆盖最常见单载流子近似，不实现多载流子霍尔模型。  
- 只做线性估计，不引入温度扫场、几何修正和接触电阻复杂建模。  
- 通过合成数据演示完整流程，避免对外部实验文件格式的依赖。  
- 优先保证“可运行 + 可审计 + 物理含义清晰”，而非大而全框架。

## R10

`demo.py` 主要函数职责：

- `check_inputs`：数据形状、长度、数值合法性检查。  
- `antisymmetrize_hall_voltage`：计算 `V_H = (V+ - V-)/2`。  
- `estimate_hall_coefficient`：最小二乘求 `R_H` 与截距并给出 `R^2`。  
- `infer_carrier_properties`：由 `R_H` 推断载流子类型和浓度。  
- `make_synthetic_case`：构建带偏置、偶次项和噪声的合成测量。  
- `run_case`：执行单个案例、统计误差指标。  
- `print_report`：格式化输出单案例结果。  
- `main`：组织多个案例并执行整体门限校验。

## R12

输出字段说明：

- `R_H true / estimated`：真实与估计霍尔系数（`m^3/C`）。
- `R_H abs error / rel error`：霍尔系数估计误差。  
- `fit intercept`：回归截距，应接近 0（理想反对称后）。  
- `fit R^2`：线性拟合优度，越接近 1 越好。  
- `carrier type`：由 `R_H` 符号得到的载流子类型。  
- `density true / est`：真实与估计载流子浓度（`1/m^3`）。  
- `Summary`：跨案例最大相对误差、最小 `R^2` 和总通过标志。

## R13

建议最小测试集（脚本已内置）：

- `N-type semiconductor`：`R_H < 0`，应判定为电子主导。  
- `P-type semiconductor`：`R_H > 0`，应判定为空穴主导。  

建议补充异常测试：

- 厚度设为 `0`（应报错）；
- `v_plus` 和 `v_minus` 长度不一致（应报错）；
- 全部 `I*B/t` 相同导致秩不足（应报错）。

## R14

可调参数：

- `n_points`：样本点数量；
- `noise_uV`：高斯噪声水平；
- `offset_uV`：常量偏置电压；
- `even_coeff_uV_per_T2`：偶次寄生项大小；
- `thickness`：样品厚度。

调参建议：

- 若 `R^2` 偏低，增大 `n_points` 或减小噪声；
- 若 `intercept` 偏大，检查是否存在残留奇次系统误差；
- 若浓度误差大，优先保证 `I/B` 覆盖范围足够宽。

## R15

方法对比：

- 对比“只测单一 `+B`”方案：
  - 本方法通过 `(+B,-B)` 反对称化，抗偏置能力更强。  
- 对比黑盒统计接口：
  - 本实现显式给出 `x/y` 构造与最小二乘流程，便于审计。  
- 对比复杂多载流子模型：
  - 本实现参数少、鲁棒性高，适合作为实验数据首轮基线。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 定义两组可复现的 Hall 合成实验配置（n 型、p 型）。  
2. `make_synthetic_case` 依据 `V_H = R_H I B / t` 构造理想霍尔项，并叠加偏置、`B^2` 偶次项与噪声生成 `V(+B), V(-B)`。  
3. `run_case` 调用 `estimate_hall_coefficient` 进入参数估计流程。  
4. `check_inputs` 对电流、磁场、电压数组及厚度执行维度和有限性校验。  
5. `antisymmetrize_hall_voltage` 计算 `V_H = (V(+B)-V(-B))/2`，显式去除大部分偶次寄生项。  
6. `estimate_hall_coefficient` 构造 `x = I*B/t` 与 `y = V_H`，用 `np.linalg.lstsq` 求解线性模型 `y = R_H*x + c`，并计算 `R^2`。  
7. `infer_carrier_properties` 根据 `R_H` 的符号和大小计算载流子类型与浓度 `n = 1/(|R_H|q)`。  
8. `main` 汇总各案例误差、`R^2`、截距并执行断言门限，确保脚本一次运行即可完成有效性验证。

第三方库没有被当成物理黑盒：`numpy` 仅负责数组运算和线性代数，霍尔效应的建模、去偏置和参数反演逻辑全部在源码中显式展开。
