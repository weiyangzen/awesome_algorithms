# 钱德拉塞卡极限 (Chandrasekhar Limit)

- UID: `PHYS-0059`
- 学科: `物理`
- 分类: `天体物理`
- 源序号: `59`
- 目标目录: `Algorithms/物理-天体物理-0059-钱德拉塞卡极限_(Chandrasekhar_Limit)`

## R01

钱德拉塞卡极限是冷白矮星在电子简并压支撑下可稳定存在的最大质量尺度。  
当白矮星质量接近该上限时，电子趋于超相对论，压强随密度增长的“刚度”不再足以抵抗引力，系统会失稳并向中子星或爆发演化（取决于具体天体环境）。

## R02

本条目的目标是给出一个可运行、可审查的最小数值实现：

- 用 `n=3` Lane-Emden 方程得到无量纲常数 `xi_1` 与 `omega_3`；
- 用相对论电子简并多项式常数计算解析型 `M_ch`；
- 用完整零温简并电子 EOS 扫描白矮星结构，观察质量向极限收敛；
- 通过双积分器交叉和尾部拟合，给出一致的极限质量估计。

## R03

MVP 建模假设：

- 球对称、静态、牛顿引力静力平衡；
- 温度近似为 0（完全简并电子气）；
- 组成由平均每电子重子数 `mu_e` 表征，默认 `mu_e=2`（典型 C/O 白矮星量级）；
- 不考虑自转、磁场、广义相对论修正、库仑修正和有限温度效应。

该设定的核心目的不是高保真天体建模，而是“从方程到极限质量”的算法闭环。

## R04

核心方程：

1. 费米动量参数

`x = p_F / (m_e c)`

2. 电子数密度与质量密度

`n_e = (m_e c)^3 x^3 / (3 pi^2 hbar^3)`  
`rho = mu_e m_u n_e`

3. 零温简并压强

`P(x) = (m_e^4 c^5 / (24 pi^2 hbar^3)) * [x(2x^2-3)sqrt(1+x^2) + 3 asinh(x)]`

4. 压强导数

`dP/dx = (m_e^4 c^5 / (3 pi^2 hbar^3)) * x^4 / sqrt(1+x^2)`

5. 白矮星结构方程（牛顿静力平衡）

`dm/dr = 4 pi r^2 rho`  
`dP/dr = -G m rho / r^2`  
`dx/dr = (dP/dr) / (dP/dx)`

6. Lane-Emden (`n=3`) 与质量表达

`(1/xi^2) d/dxi (xi^2 dtheta/dxi) = -theta^3`  
`M_ch = 4 pi (K/(pi G))^(3/2) * omega_3`，其中 `omega_3 = -xi_1^2 theta'(xi_1)`。

## R05

算法流程（高层）：

1. 设定常数、`mu_e`、中心 `x_c` 扫描区间、积分容差；
2. 解 `n=3` Lane-Emden，得到 `xi_1`、`omega_3`；
3. 由相对论多项式常数 `K` 计算解析型钱德拉塞卡质量；
4. 对每个中心 `x_c` 数值积分白矮星结构方程到表面事件 `x=x_surface`；
5. 形成 `x_c -> (rho_c, M, R)` 数据表；
6. 用高密尾部 `M` 对 `1/x_c^2` 线性拟合并外推 `x_c -> inf` 得 `M_limit`；
7. 用第二积分器重复扫描并计算 `MAE/R^2` 一致性指标；
8. 输出表格与断言，验证极限估计可信。

## R06

复杂度（`N` 为扫描模型数，`S` 为单模型积分步数）：

- 单模型结构积分约 `O(S)`；
- 全扫描约 `O(N*S)`；
- 双积分器交叉约 `O(2*N*S)`；
- Lane-Emden 开销很小，相比扫描可忽略；
- 空间复杂度约 `O(N)`（保存汇总表）。

默认 `N=24`，脚本在普通开发机上可秒级运行。

## R07

正确性直觉：

- 当电子超相对论时，EOS 逼近 `P ~ rho^(4/3)`，对应 `n=3` 多项式；
- `n=3` 多项式质量与中心密度解耦，天然给出“极限质量”；
- 完整 EOS 在高密区应趋近该超相对论极限，因此扫描质量曲线应逼近同一上限；
- 解析 (`Lane-Emden`) 与数值 (`EOS 扫描`) 两路收敛到相近结果时，极限估计具有较强可解释性。

## R08

数值稳定策略：

- 从 `r0=1 cm` 起步，避免 `r=0` 奇点；
- Lane-Emden 在 `xi0` 处用级数初值（避免中心奇异）；
- 白矮星积分用事件函数 `x -> x_surface` 终止，避免外层刚性区步长异常；
- 采用 `DOP853` 与 `RK45` 双解算器交叉，降低单方法偶然偏差；
- 对结果加物理区间与单调性断言，出现异常时快速失败。

## R09

`demo.py` 的最小工具栈：

## R11

关键输出指标：

- `xi1`, `omega3`：`n=3` Lane-Emden 关键无量纲常数；
- `M_ch_lane`：由 Lane-Emden 常数计算的解析极限质量；
- `m_limit_msun`：由白矮星高密序列外推的极限质量；
- `tail_last_mass_msun`：最高中心密度模型质量；
- `mass_mae_msun`, `mass_r2`：两求解器质量曲线一致性；
- `radius_mae_km`, `radius_r2`：两求解器半径曲线一致性；
- `n_positive_dm`, `n_negative_dm`：质量差分符号统计；
- `n_negative_dr`：半径随中心密度增大是否整体收缩。

## R12

默认参数（`ChandraConfig`）：

- 组成参数：`mu_e=2.0`；
- 中心扫描：`x_c in [0.3, 100]`，`n_models=24`；
- 表面事件：`x_surface=1e-3`；
- 积分范围：`r in [1 cm, 3e9 cm]`；
- 数值精度：`rtol=2e-6`, `atol=1e-8`, `max_step=1e7 cm`；
- Lane-Emden：`xi in [1e-8, 20]`；
- 求解器：主 `DOP853`，次 `RK45`。

在这组参数下，典型输出会给出约 `1.45 M_sun` 的钱德拉塞卡极限估计。

## R14

常见失效模式：

1. 常数系数写错（尤其 `h` 与 `hbar` 系数换算）会导致极限质量成倍偏离；
2. `x_c` 扫描上界太小，无法进入渐近区，外推不稳定；
3. `x_surface` 过大导致半径、质量被系统性低估；
4. `r_max` 太小导致事件未触发，模型积分失败；
5. 将牛顿模型结果直接当作 Ia 超新星精确爆发阈值。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `ChandraConfig` 定义 `mu_e`、扫描区间、求解器、容差和事件阈值，统一实验配置。  
2. `pressure_from_x`、`dpressure_dx`、`density_from_x` 明确实现 EOS，不把第三方库当黑盒。  
3. `solve_lane_emden_n3` 用 `solve_ivp + event(theta=0)` 求 `xi_1` 与 `omega_3`，再由 `chandrasekhar_mass_from_lane_emden` 得解析极限质量。  
4. `white_dwarf_rhs` 按 `dm/dr` 与 `dx/dr` 公式显式构造恒星结构微分方程；`make_x_surface_event` 定义表面终止条件。  
5. `solve_single_white_dwarf` 对单个中心 `x_c` 积分并提取 `M`、`R`；`build_white_dwarf_sequence` 批量扫描形成 `DataFrame`。  
6. `estimate_limit_with_sklearn` 对高密尾部执行 `M` 对 `1/x_c^2` 线性拟合，外推 `x_c -> inf` 得 `M_limit`。  
7. `compare_sequences` 用 `MAE/R^2` 比较 `DOP853` 与 `RK45` 结果；`torch_monotonicity_diagnostics` 用张量差分检查收敛形态。  
8. `main` 输出关键表格与指标，并执行物理区间、一致性和单调性断言；全部通过即报告钱德拉塞卡极限估计成立。

第三方库在本实现中仅提供“通用数值工具”（ODE、回归、指标、张量运算），极限质量的物理建模、方程构造与判据均在源码中逐步展开。
