## R01

德拜模型（Debye Model）是描述晶体晶格振动热容的经典模型。核心思想是：

- 把晶体振动模态视为连续频谱声子；
- 引入截止频率（德拜频率）或等价的德拜温度 `Theta_D`；
- 用玻色分布统计占据，得到热容的温度依赖。

最常用表达是定容摩尔热容：

`C_V(T) = 9R (T/Theta_D)^3 * integral_0^{Theta_D/T} [x^4 e^x / (e^x - 1)^2] dx`

其中 `R` 是气体常数。

## R02

德拜模型在固体物理中的价值主要体现在：

- 它统一解释了低温与高温两端的热容规律；
- 在低温给出 `C_V ~ T^3`，明显优于爱因斯坦模型的指数衰减结论；
- 在高温恢复 Dulong-Petit 定律 `C_V -> 3R`（每摩尔原子晶体）。

因此它是“从微观振动模态到宏观热容曲线”的最小可计算桥梁。

## R03

本目录 MVP 目标：

- 输入：`Theta_D` 与温度网格 `T`；
- 输出：`C_V(T)`、`C_V/(3R)`、`C_V/T^3`；
- 验证：
  1. 低温点与 `T^3` 渐近式一致；
  2. 高温点接近 `3R`；
  3. 计算区间内 `C_V > 0`。

实现策略是显式数值积分，不调用黑盒热容库。

## R04

本实现使用的关键公式：

1. 德拜热容积分公式：
   `C_V(T) = 9R (T/Theta_D)^3 * I(T)`
2. 其中
   `I(T) = integral_0^{Theta_D/T} f(x) dx`
3. 被积函数
   `f(x) = x^4 e^x / (e^x - 1)^2`
4. 数值稳定重写：
   `f(x) = x^4 e^{-x} / (1 - e^{-x})^2`

近零点使用 `f(x) ~ x^2` 级数近似，避免 `0/0` 数值不稳定。

## R05

复杂度（设温度点数 `N_T`，辛普森区间数 `N_I`）：

- 每个温度一次积分，复杂度约 `O(N_I)`；
- 全温区复杂度约 `O(N_T * N_I)`；
- 空间复杂度约 `O(N_I)`（单次积分网格）和 `O(N_T)`（结果表）。

默认配置 `N_T=120`、`N_I=4000`，可在普通 CPU 上快速完成。

## R10

正确性直觉：

1. 德拜模型把模态密度离散和为积分，得到标准 `C_V` 公式；
2. 若数值积分正确，`C_V` 应在低温满足 `~T^3`；
3. 当 `T >> Theta_D`，积分上限很小，模型应回到 `3R`；
4. 热容在物理温区应保持非负。

`demo.py` 的三组断言正对应这三条可执行物理检验。

## R11

数值稳定性处理：

- 被积函数改写为 `exp(-x)` 形式，避免大 `x` 时 `exp(x)` 上溢；
- 在 `x < 1e-6` 使用 `x^2` 局部近似，避免除零放大误差；
- 积分上限设 `integration_cutoff=80`，对高上限尾部采取可忽略截断；
- 要求辛普森区间数为偶数，确保算法定义成立。

这些策略能覆盖从几 K 到数千 K 的稳定计算。

## R12

关键参数（`DebyeConfig`）：

- `theta_debye`：德拜温度，决定热容曲线横向尺度；
- `t_min`, `t_max`, `n_temps`：温度网格范围和分辨率；
- `simpson_intervals`：积分精度-耗时平衡旋钮；
- `integration_cutoff`：高上限积分截断阈值。

调参建议：

- 若需更高精度，优先增大 `simpson_intervals`；
- 若关注超低温，可增大 `integration_cutoff` 并缩小 `t_min`。

## R14

常见失效模式：

1. 把 `Theta_D` 单位写错（K 以外单位）；
2. 温度包含 `T<=0` 导致数学定义失效；
3. 辛普森区间设为奇数；
4. 直接用 `exp(x)` 版本造成大 `x` 溢出；
5. 低温采样太少，误判 `T^3` 关系。

脚本已对 2 和 3 做显式异常检查。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 创建 `DebyeConfig`，调用 `build_result_table` 计算全温区热容。  
2. `build_result_table` 生成对数温度网格，并调用 `debye_heat_capacity_molar`。  
3. `debye_heat_capacity_molar` 对每个温度构造积分上限 `Theta_D/T`（并应用上限截断）。  
4. 同函数中用 `np.linspace` 构造均匀积分网格，调用 `debye_cv_integrand` 计算被积函数数组。  
5. `debye_cv_integrand` 在常规区使用 `x^4 exp(-x)/(1-exp(-x))^2`，在 `x≈0` 区使用 `x^2` 近似保证稳定。  
6. `simpson_integral` 对数组执行复合辛普森求积，返回无量纲积分值，再按公式得到 `C_V(T)`。  
7. `run_sanity_checks` 单独计算低温点与高温点，并分别与 `T^3` 渐近式、`3R` 极限比较，执行断言。  
8. `main` 最后打印采样表和误差指标，形成可复现、可审查的最小交付。

第三方库角色说明：`numpy` 仅用于数组与数值计算，`pandas` 仅用于结果表展示；德拜模型核心流程（网格、被积函数、辛普森积分、物理断言）均在源码中显式实现，不依赖黑盒算法库。
