## R01

哈密顿蒙特卡洛（HMC）是一种利用哈密顿动力学进行高效采样的马尔可夫链蒙特卡洛方法。它通过在参数空间中引入辅助动量变量，把“随机游走”改造成“有惯性的长距离提议”，通常能显著降低样本自相关。

在计算物理中，HMC 常用于玻尔兹曼分布采样、路径积分、贝叶斯反演等高维积分问题。核心目标是把

`E[f(q)] = ∫ f(q) pi(q) dq`

转化为样本均值

`E[f(q)] ≈ (1/N) Σ f(q_i)`。

## R02

本条目使用一个可解析基准的最小物理模型：二维耦合谐振子。

- 势能：`U(q) = 0.5 * q^T K q`，其中 `q=[q1,q2]^T`，`K` 为正定刚度矩阵。
- 目标分布：`pi(q) ∝ exp(-U(q))`（等价于二维高斯分布）。
- 任务：用 HMC 估计均值与协方差，并与解析真值（`mean=0`, `cov=K^{-1}`）比较。

选择该模型的原因是：既保留了 HMC 的完整机制（动量刷新、辛积分、Metropolis 校正），又能提供严格可验证参考。

## R03

HMC 将目标分布扩展到相空间 `(q,p)`：

- 位置势能：`U(q) = -log pi(q) + const`
- 动能：`K(p) = 0.5 * p^T M^{-1} p`（`M` 是质量矩阵）
- 哈密顿量：`H(q,p) = U(q) + K(p)`

联合分布为

`pi(q,p) ∝ exp(-H(q,p)) = exp(-U(q)) * exp(-K(p))`。

每次迭代先采样 `p ~ N(0, M)`，再通过离散哈密顿动力学提出新状态。

## R04

离散动力学采用 leapfrog（蛙跳）积分器：

1. `p_{t+1/2} = p_t - (eps/2) * grad U(q_t)`
2. `q_{t+1} = q_t + eps * M^{-1} p_{t+1/2}`
3. `p_{t+1} = p_{t+1/2} - (eps/2) * grad U(q_{t+1})`

重复 `L` 步得到提议 `(q', p')`，并以

`alpha = min(1, exp(-(H(q',p') - H(q,p))))`

执行 Metropolis 接受拒绝，确保目标分布不因数值积分误差而偏移。

## R05

本 MVP 的关键假设：

- 目标分布的 `U(q)` 可微，且 `grad U(q)` 可稳定计算。
- 质量矩阵取对角形式，便于实现 `M^{-1} p`。
- 通过固定 `step_size` 与 `n_leapfrog` 能获得可接受的能量误差与接受率。
- 单链结果可用于教学级诊断（接受率、ESS、协方差误差），不替代生产级多链收敛分析。

## R06

`demo.py` 输入输出约定：

- 输入：无交互输入；脚本内置 `n_steps`、`burn_in`、`step_size`、`n_leapfrog`、`seed`。
- 输出：
1. `mean_q1`、`mean_q2`、`cov_11`、`cov_12`、`cov_22` 的估计值与解析参考值对照表。
2. 诊断指标：接受率、平均 `|delta_H|`、均值向量范数、协方差相对误差、各维 ESS。
3. 阈值检查与最终 `Validation: PASS/FAIL`。

## R07

算法主流程：

1. 定义 `K`、`U(q)`、`grad U(q)`、`K(p)` 与 `H(q,p)`。
2. 初始化 `q`，每步刷新动量 `p ~ N(0,M)`。
3. 用 leapfrog 积分器推进 `L` 个离散步，得到提议状态。
4. 用哈密顿量差计算接受概率并执行接受拒绝。
5. 记录链并丢弃 burn-in。
6. 计算样本均值、样本协方差、ESS、能量误差。
7. 与解析高斯真值比较并输出验证结论。

## R08

复杂度分析（`N` 为链长，`L` 为每次轨迹步数，`d` 为维度）：

- 采样主循环：`O(N * L * d^2)`（通用矩阵梯度形式；若 `K` 对角可降到 `O(N*L*d)`）。
- 存储链：`O(N * d)`。
- 协方差估计：`O(N * d^2)`。
- ESS（自相关截断）：每维 `O(N * lag_max)`。

在本 MVP 中 `d=2`，整体开销很小，适合快速演示和回归测试。

## R09

数值稳定与实现细节：

- 接受判断在对数域进行：`log(u) < min(0, -delta_H)`，避免指数下溢。
- 使用 leapfrog（时间可逆 + 近辛结构）控制长期能量漂移。
- 提议末尾执行动量翻转，保证提议映射严格可逆。
- 固定随机种子，确保结果可复现。
- IACT 使用“正序列截断”（相关系数首次非正即停止）以抑制高延迟噪声。

## R12

输出字段说明：

- `metric`：统计量名称（均值与协方差元素）。
- `estimate`：HMC 样本估计值。
- `reference`：解析参考值。

## R13

## R14

- 轨迹长度固定，未实现 NUTS 的自动终止。

## R16

典型应用场景：

- 统计物理中平衡分布的高维采样。
- 计算化学中的势能面与自由能估计。
- 贝叶斯反演中的后验采样（参数识别、不确定性量化）。
- 复杂系统中难解析积分的数值估计。

## R17

与常见替代方案对比：

- 随机游走 MH：实现简单，但高维下混合慢、相关强。
- MALA：利用梯度但步长仍受限，轨迹长度短。
- HMC：一次提议可跨越更远距离，通常 ESS/计算量更优；代价是需要梯度与更复杂的调参。

因此该 MVP 适合展示“物理动力学 + MCMC 校正”的核心思想与工程落地路径。

## R18

`demo.py` 源码级流程拆解（9 步）：

1. `HMCConfig` 定义采样步数、burn-in、步长、leapfrog 步数、初值和随机种子。
2. `stiffness_matrix`、`potential_energy`、`grad_potential` 构造目标势能与梯度。
3. `kinetic_energy` 与 `hamiltonian` 组合出 `H(q,p)=U+K`。
4. `sample_momentum` 按 `N(0,M)` 刷新辅助动量。
5. `leapfrog` 执行半步动量-整步位置-半步动量更新，并在末尾翻转动量保证可逆性。
6. `hmc_sample` 中每轮先生成提议轨迹，再以 `delta_H` 做 Metropolis 接受拒绝，得到马尔可夫链。
7. 丢弃 burn-in 后，`effective_sample_size_per_dim` 与 `integrated_autocorrelation_time_1d` 计算各维 ESS。
8. `build_report` 计算样本均值/协方差，与解析真值 `cov=K^{-1}` 对比，并执行阈值校验。
9. `main` 打印统计表、诊断指标和 `Validation: PASS/FAIL`，保证一次命令即可复现实验。

可以看到，本实现将 HMC 的关键机制全部显式展开，不依赖第三方采样黑箱。
