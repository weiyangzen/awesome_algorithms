## R01

范德瓦尔斯方程是在理想气体方程基础上加入“分子体积排斥 + 分子间吸引”后的状态方程：

`P = nRT/(V - nb) - a(n/V)^2`

其中 `a` 表示吸引修正，`b` 表示排斥修正。该模型能描述临界点附近的非理想行为，是热力学中最经典的真实气体近似模型之一。

## R02

本条目目标是把范德瓦尔斯方程做成可运行、可校验的最小 MVP，而不是只给公式。

核心验证点：

- 正向计算：已知 `T, V` 计算 `P` 与压缩因子 `Z`；
- 反向计算：已知 `T, P` 通过三次方程求体积根；
- 临界一致性：验证 `Tc, Pc, Vc` 以及临界点导数条件。

## R03

`demo.py` 的输入输出（脚本内置参数，无交互）：

- 输入：
1. 气体参数 `a, b, n`（示例使用 CO2 风格 SI 参数）；
2. 等温线采样温度与体积网格；
3. 反解案例 `(T, P)` 列表（含亚临界/近临界/超临界）。
- 输出：
1. 临界常数表 `T_c, P_c, V_c`；
2. 等温线样本表（`P_vdw, P_ideal, Z_vdw`）；
3. 体积反解结果表（实根个数、每个根的重构压强误差）；
4. 断言通过后打印 `All checks passed.`。

## R04

MVP 使用的核心关系：

1. 状态方程：
`P = nRT/(V-nb) - a(n/V)^2`。

2. 临界常数：
`T_c = 8a/(27Rb)`，`P_c = a/(27b^2)`，`V_c = 3nb`。

3. 临界导数条件：
`(dP/dV)_T = 0` 且 `(d2P/dV2)_T = 0` 在 `(T_c, V_c)` 处成立。

4. 体积反解三次方程（固定 `P,T`）：
`P V^3 - (Pnb + nRT)V^2 + a n^2 V - a n^3 b = 0`。

## R05

复杂度（单次案例）：

- 正向计算 `P(V,T)`：`O(1)`；
- 三次方程求根（`numpy.roots`）：`O(1)`（固定 3 次多项式）；
- 超临界 Brent 根求解：迭代记为 `K`，复杂度约 `O(K)`；
- 空间复杂度：`O(1)`（不计输出表格）。

## R06

`demo.py` 的最小闭环：

- 闭环 A（正向）：`build_isotherm_table` 逐点计算 `P_vdw`、`P_ideal`、`Z_vdw`；
- 闭环 B（反向）：`solve_volume_roots_from_cubic` 从 `(P,T)` 反解体积实根；
- 闭环 C（一致性）：临界点导数断言 + 压力重构残差断言 + 多实根案例断言。

## R10

正确性直觉：

1. 若 EOS 正确编码，则由 `V` 算出的 `P` 代回应满足方程本身；
2. 若反解三次方程正确，得到的每个物理解根带回 `P(V,T)` 应还原目标压强；
3. 若临界公式与导数推导一致，则 `(T_c, V_c)` 处一阶、二阶导数应同时接近 0；
4. 亚临界区出现 3 个实根，体现了范德瓦尔斯模型的非理想多分支结构。

## R11

数值稳定性策略：

- 对 `a,b,n,T,V,P` 做正值与定义域检查，严格要求 `V > nb`；
- 三次根只保留“虚部足够小且 `V > nb`”的物理解；
- 压力重构误差采用绝对残差做统一检查；
- 超临界时使用 `brentq` 作为独立一维根解法进行交叉验证。

## R12

关键参数：

- `a`（`Pa*m^6/mol^2`）：吸引项强度；
- `b`（`m^3/mol`）：排斥体积修正；
- `n_mol`（`mol`）：物质的量；
- `T_K`、`V_m3`、`P_target_Pa`：状态变量。

调参建议：

- 若要稳定观察三实根现象，选择 `T < T_c` 且 `P` 在中间区间；
- 若要单根行为，选择 `T > T_c`；
- 初学时优先使用 `n=1`，便于和文献中的摩尔形式对照。

## R14

常见失效模式：

1. 单位不一致（例如 `a,b` 仍是 `L,bar` 单位却按 SI 计算）；
2. 体积接近或小于 `nb` 导致奇异或非物理值；
3. 错把全部复根当物理根使用；
4. 在亚临界多根情况下未进行相态判别就直接选根；
5. 用范德瓦尔斯参数外推到超出适用区间的高精度任务。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `VanDerWaalsParams.validate` 先检查 `a,b,n` 全为正，确保模型参数合法。  
2. `vdw_pressure` 在 `V > nb` 条件下按 `P = nRT/(V-nb) - a(n/V)^2` 直接计算压强。  
3. `critical_constants` 计算 `Tc,Pc,Vc`，再由 `dP_dV`、`d2P_dV2` 在临界点做导数自检。  
4. `cubic_coefficients_for_volume` 把 EOS 代数化成三次多项式系数 `[c3,c2,c1,c0]`。  
5. `solve_volume_roots_from_cubic` 调用 `numpy.roots` 求全部复根，再手动筛选“近实数且 `V>nb`”物理解。  
6. `build_inverse_check_table` 将每个体积根代回 `vdw_pressure`，显式计算压强残差，验证反解正确性。  
7. 对 `T>Tc` 案例，`solve_supercritical_volume_brentq` 用 `scipy.optimize.brentq` 做独立一维求根，形成交叉验证。  
8. `main` 汇总临界表、等温线表、反解表，并断言导数条件/残差阈值/多实根案例后输出通过信息。  

补充：`numpy.roots` 与 `brentq` 仅负责数值求根；模型构造、系数推导、物理解筛选和正确性判定都在源码中显式实现，不是黑盒一键调用。
