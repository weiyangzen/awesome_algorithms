# 自旋-轨道耦合 (Spin-Orbit Coupling)

## R01

自旋-轨道耦合（Spin-Orbit Coupling）描述电子轨道角动量 `L` 与自旋角动量 `S` 的相互作用。  
在氢样原子低阶近似中，它可写为：

`H_so = xi(r) * (L·S),  xi(r) = [1 / (2 c^2)] * [1/r * dV/dr]`

对库仑势 `V(r) = -Z/r`（原子单位）得到：

`xi(r) = Z / (2 c^2 r^3)`

因此能量修正可写成 `DeltaE_so = <H_so>`，并依赖量子数 `(n,l,j)`。

## R02

该条目的核心意义：

- 解释原子精细结构（fine structure）里 `j=l±1/2` 双线分裂的来源；
- 将“角动量代数”与“可观测谱线位移”直接连接；
- 提供从算符 `L·S` 到数值光谱表的最小实现模板；
- 是学习相对论修正、精细结构与量子数耦合规则的基础案例。

## R03

本 MVP 的输入输出约定（无交互）：

- 输入（脚本内固定）：
1. 原子核电荷扫描 `Z = 1..6`；
2. 主量子数集合 `n in {2,3,4}`；
3. 目标校验态 `n=2, l=1, j=3/2`；
4. 原子单位常数（`c`、`Hartree->eV`）。
- 输出：
1. `L·S` 在 `l=1` 未耦合基的本征值与误差检查；
2. 多组 `(Z,n,l,j)` 的自旋-轨道能量修正表（`pandas`）；
3. `j` 双线分裂表；
4. `Z^4` 标度的三实现拟合结果（`sklearn`、`scipy`、可选 `torch`）；
5. 断言通过后输出 `All checks passed.`。

## R04

MVP 使用的关键公式：

1. 角动量代数：
`<L·S> = [j(j+1)-l(l+1)-s(s+1)]/2`, `s=1/2`。

2. 氢样原子（`l>0`）径向期望：
`<r^-3> = Z^3 / [n^3 l (l+1/2) (l+1)]`。

3. 自旋-轨道修正（原子单位）：
`DeltaE_so = [Z / (2 c^2)] * <r^-3> * <L·S>`。

4. 能级总能量（本 MVP）：
`E_total = E0 + DeltaE_so`, `E0 = -Z^2/(2n^2)`。

5. 标度关系：
固定 `(n,l,j)` 时 `DeltaE_so ∝ Z^4`，可用线性模型 `DeltaE_so = kappa * Z^4` 校验。

## R05

设核电荷样本数为 `Nz`，量子态数为 `Ns`（约为 `sum_n sum_l 2`），`L·S` 矩阵维度为 `d=(2l+1)*2`：

- 构造与对角化 `L·S` 的复杂度约 `O(d^3)`，本例 `l=1` 时 `d=6` 为常数级；
- 谱表构造复杂度约 `O(Ns)`（每个态仅做常数次代数运算）；
- `Z^4` 回归复杂度约 `O(Nz)`；
- 空间复杂度约 `O(Ns)`（主要由 `pandas` 结果表占用）。

## R06

`demo.py` 的最小闭环分为四段：

- **闭环 A（代数算符）**：源码显式构建 `Lx,Ly,Lz,Sx,Sy,Sz`，再组装 `L·S` 并对角化；
- **闭环 B（解析修正）**：按 `(n,l,j,Z)` 计算 `DeltaE_so` 与 `E_total`；
- **闭环 C（标度验证）**：对固定态做 `DeltaE_so ~ Z^4` 拟合并比较不同求解器；
- **闭环 D（物理断言）**：检查 `l=0` 为零修正、`j=3/2` 与 `j=1/2` 符号分裂、退简并计数正确。

## R10

正确性直觉：

1. `l=0` 态应无自旋-轨道劈裂（`L=0`）；
2. 对 `l>0`，`j=l+1/2` 与 `j=l-1/2` 应出现相反符号修正；
3. 固定 `(n,l,j)` 时，`|DeltaE_so|` 随 `Z` 呈 `Z^4` 增长；
4. `L·S` 矩阵本征值应与角动量解析本征值及其退简并度一致。

## R11

数值稳定策略：

- 所有核心计算在原子单位中完成，避免单位混乱；
- 对量子数做前置校验，阻断非法 `(n,l,j)`；
- `L·S` 用 Hermitian 对角化 `eigvalsh`，保证本征值实数稳定；
- `torch` 路径设为可选，避免环境缺失导致主流程失败。

## R14

常见失效模式：

1. 将 `j` 取值写错（遗漏 `j=l±1/2` 规则）；
2. 把 `l=0` 代入 `<r^-3>` 封闭公式引发奇异项；
3. 混用 SI 与原子单位，导致系数量级错误；
4. 把 `Z^4` 标度误写为 `Z^2` 或 `Z^3`；
5. 仅看数值输出、不做代数本征值对照，难以及时发现逻辑错误。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `allowed_j_values` 与 `validate_n_l_j` 先约束量子数域，确保后续计算只在物理允许态上执行。  
2. `ls_expectation` 实现 `[(j(j+1)-l(l+1)-s(s+1))/2]`，作为解析路线的核心角动量因子。  
3. `radial_inverse_r3_expectation_hydrogenic` 计算 `<r^-3>`（`l>0`）并与 `z`、`n`、`l` 组合成径向系数。  
4. `spin_orbit_shift_au` 把 `Z/(2c^2)`、`<r^-3>`、`<L·S>` 相乘得到 `DeltaE_so`；对 `l=0` 显式返回 0。  
5. `angular_momentum_matrices` 通过 `J+`/`J-` 构造 `Jx,Jy,Jz`，`l_dot_s_matrix_uncoupled` 再在未耦合基中拼出 `L·S` 矩阵并对角化。  
6. `build_spectrum_table` 枚举 `(Z,n,l,j)` 生成 `E0`、`DeltaE_so`、`E_total` 与 meV 单位列；`build_doublet_splitting_table` 汇总双线分裂。  
7. `fit_scaling_sklearn`、`fit_scaling_curve_fit`、`fit_scaling_torch_optional` 三路拟合 `DeltaE_so = kappa * Z^4`（其中 torch 路径为无截距闭式最小二乘），比较系数一致性。  
8. `main` 统一打印结果并断言：`l=0` 零修正、`L·S` 退简并正确、`2p` 分裂符号正确、`Z^4` 拟合一致，最终输出 `All checks passed.`。  

说明：第三方库只提供线性回归、曲线拟合与基础线代；物理公式、矩阵构造和校验逻辑都在源码中显式展开，不是黑盒调用。
