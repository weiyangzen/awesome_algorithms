# 布洛赫定理 (Bloch's Theorem)

- UID: `PHYS-0071`
- 学科: `物理`
- 分类: `固体物理`
- 源序号: `71`

## R01

布洛赫定理描述了周期势场中的单电子本征态结构：

`psi_{n,k}(x) = exp(i k x) u_{n,k}(x)`，且 `u_{n,k}(x+a)=u_{n,k}(x)`。

其中 `a` 是晶格常数，`k` 是准动量，`n` 是能带指标。该定理把“平面波相位”与“晶格周期部分”解耦，是能带理论最核心的结构性结论之一。

## R02

本条目目标是做一个最小可运行闭环，展示布洛赫定理如何落地为可计算算法：

- 用 1D 周期势 `V(x)=V0 cos(Gx)` 建立平面波基哈密顿量；
- 在第一布里渊区采样 `k` 并求能带；
- 从本征矢重构 `u_{n,k}(x)`，数值验证 `u(x+a)=u(x)`；
- 输出带隙、对称性误差和近 `k=0` 曲率拟合结果。

## R03

`demo.py` 输入输出约定（无交互）：

- 输入（脚本内参数）：
1. 晶格常数 `a`、势强 `v0`；
2. 平面波截断 `m_max`、`k` 采样数；
3. 取前 `num_bands` 条能带；
4. 近 `k=0` 有效质量拟合窗口。

- 输出：
1. 若干关键 `k` 点的能带表；
2. 相邻带隙统计表；
3. Bloch 周期性误差、`E(k)=E(-k)` 对称误差；
4. `sklearn` / `scipy` / `torch` 的曲率拟合对照与断言结果。

## R04

MVP 使用的核心模型与公式：

1. 周期势：`V(x)=V0 cos(Gx)`，`G=2pi/a`；
2. 平面波基：`|k+mG>`，`m in [-M,M]`；
3. 哈密顿矩阵元（单位取 `hbar^2/(2m_e)=1`）：
`H_{mn}(k) = (k+mG)^2 delta_{mn} + (V0/2)(delta_{m,n+1}+delta_{m,n-1})`；
4. 对角化得到 `E_n(k)` 与系数 `c_m^{(n,k)}`；
5. 周期函数重构：
`u_{n,k}(x)=sum_m c_m^{(n,k)} exp(i m G x)`，应满足 `u(x+a)=u(x)`。

## R05

复杂度（`N_k` 为 `k` 点数，`D=2M+1` 为基维度）：

- 单个 `k` 的本征分解约 `O(D^3)`；
- 全 `k` 网格约 `O(N_k * D^3)`；
- 周期性误差重构约 `O(N_x * D)`。

空间复杂度约 `O(N_k * D * num_bands)`（保存低能本征矢）以及 `O(N_k * num_bands)`（能带）。

## R06

最小闭环由 4 个模块组成：

- **模块 A（建模）**：生成平面波基哈密顿量；
- **模块 B（谱求解）**：对每个 `k` 求本征值/本征矢并装配能带；
- **模块 C（定理验证）**：重构 `u(x)` 并检查晶格平移周期性；
- **模块 D（互校）**：用 `sklearn`、`scipy`、`torch` 对同一低能带曲率拟合并比较一致性。

## R10

正确性直觉：

1. 周期势只耦合相差倒格矢的平面波，因此可在 `k` 分块求解；
2. 若 Bloch 定理成立，从本征矢重构的 `u(x)` 应满足严格周期性；
3. 实势应保证时间反演对称，故 `E_n(k)=E_n(-k)`；
4. `V0>0` 时布里渊区边界附近会打开能隙。

## R11

数值稳定策略：

- 用实对称厄米矩阵并调用 `scipy.linalg.eigh`，避免非厄米噪声；
- 采用对称 `k` 网格，便于直接评估 `E(k)-E(-k)`；
- Bloch 周期误差用 `max|u(x+a)-u(x)|`，指标清晰可比较；
- 曲率拟合只取小 `|k|` 窗口，减少高阶项污染。

## R12

关键参数与调参建议：

- `v0`：越大带隙通常越明显；
- `m_max`：越大基更完备，但计算代价升高；
- `n_k`：越大能带曲线更平滑，带隙估计更稳；
- `fit_k_window`：过大将偏离抛物近似，过小会增噪。

建议优先收敛 `m_max`，再提高 `n_k`。

## R13

可验证保证（本 MVP）：

- 确定性：固定参数和随机种子，输出可复现；
- 物理一致性：
1. Bloch 周期误差低于阈值；
2. `E(k)=E(-k)` 误差低于阈值；
3. 第一带隙为正；
4. 三种拟合曲率结果一致到容差。

说明：本任务不是近似优化问题，不涉及近似比或随机成功率保证。

## R14

常见失效模式：

1. `m_max` 太小导致能带弯曲与带隙失真；
2. `n_k` 太小导致带边定位偏差；
3. 拟合窗口过宽使二次近似失效；
4. 若势强接近 0，带隙可能接近数值分辨率下限。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `BlochConfig` 固定晶格常数、势强、基截断、`k` 网格和断言容差。  
2. `plane_wave_indices` 生成 `m=-M...M`，`build_hamiltonian` 按 `H_{mn}(k)` 显式装配厄米矩阵。  
3. `solve_bands` 对每个 `k` 调用 `scipy.linalg.eigh`，提取前 `num_bands` 本征值和对应本征矢。  
4. `compute_band_gaps` 在离散 `k` 网格上计算相邻带的 `min(upper)-max(lower)`，形成带隙表。  
5. `bloch_periodicity_error` 由系数 `c_m` 重构 `u_{n,k}(x)=sum c_m exp(i m G x)`，计算 `max|u(x+a)-u(x)|`。  
6. `fit_curvature_sklearn`、`fit_curvature_scipy`、`fit_curvature_torch` 在小 `|k|` 窗口对 `E(k)` 做二次模型拟合，得到曲率 `alpha`。  
7. `symmetry_error_k_to_minus_k` 评估实势应满足的 `E_n(k)=E_n(-k)` 最大误差，作为额外结构校验。  
8. `main` 汇总打印能带样本、带隙、拟合对照与误差，并执行断言；全部通过后打印 `All checks passed.`。

说明：`scipy`、`sklearn`、`torch` 在本脚本里只承担通用线代/拟合求解角色；Bloch 定理相关的建模、重构与物理校验流程均在源码中逐步展开，不是黑箱一键调用。
