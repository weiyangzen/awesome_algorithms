# 传输线理论 (Transmission Line Theory)

- UID: `PHYS-0180`
- 学科: `物理`
- 分类: `电磁学`
- 源序号: `181`
- 目标目录: `Algorithms/物理-电磁学-0181-传输线理论_(Transmission_Line_Theory)`

## R01

传输线理论研究沿导体分布传播的电压与电流波，而不是把导线当作“无延迟集中参数”。

在每单位长度参数 `R, L, G, C` 下，时谐稳态的核心量为：

- 传播常数 `gamma = sqrt((R + jωL)(G + jωC)) = alpha + j beta`；
- 特性阻抗 `Z0 = sqrt((R + jωL)/(G + jωC))`；
- 由此可得到输入阻抗、反射系数、驻波比、沿线分布电压电流。

## R02

该理论是射频、微波、高速互连、同轴电缆和 PCB 信号完整性的基础。

当信号上升沿足够快或线长不可忽略时，集中电路模型会失效，必须考虑：

- 传播延迟与相位变化；
- 负载不匹配导致的反射；
- 线路损耗导致的幅度衰减。

## R03

本条目 MVP 要解决的问题：

- 输入：一条均匀 RLGC 传输线参数、频率点、线长、终端负载 `ZL`；
- 输出：`gamma(f)`、`Z0(f)`、输入阻抗 `Zin(f)`、输入端反射与 VSWR、沿线 `V(x), I(x)` 分布样例。

目标是提供可运行、可校验、源码可追踪的最小实现，而不是覆盖多导体耦合线或全波电磁求解。

## R04

MVP 使用频域闭式公式，避免黑盒库：

1. 用 RLGC 定义计算 `gamma(f)` 与 `Z0(f)`；
2. 用有限长度传输线公式计算 `Zin`：
   `Zin = Z0 * (ZL + Z0*tanh(gamma*l)) / (Z0 + ZL*tanh(gamma*l))`；
3. 用 `Gamma = (ZL - Z0)/(ZL + Z0)` 得到负载与输入端反射；
4. 用双曲函数表达式恢复 `V(x), I(x)` 剖面。

## R05

设频率采样点数为 `Nf`，沿线剖面采样点数为 `Nx`：

- 频率扫描主要是向量化复数运算，复杂度 `O(Nf)`；
- 单次剖面计算复杂度 `O(Nx)`；
- 空间复杂度主要来自结果数组，约 `O(Nf + Nx)`。

## R06

`demo.py` 设计了 4 个固定场景：

- 场景 A：20 m 有损线（`R=0.08, L=250nH, G=2e-6, C=100pF`）在 `1 MHz~1 GHz` 扫频；
- 场景 B：匹配负载自检（`ZL=Z0`）验证反射约为 0；
- 场景 C：四分之一波长变换器自检，验证 `Zin ~= Z0^2 / ZL`；
- 场景 D：输出某频率下沿线 `|V(x)|, |I(x)|` 采样表。

## R10

正确性直觉：

1. 电报方程在均匀线假设下有解析解；
2. `gamma` 和 `Z0` 由 RLGC 唯一确定；
3. `Zin` 是解析解在 `x=l` 的边界映射；
4. 匹配条件下解析上应有 `Gamma=0`；
5. 无损四分之一波长线应满足阻抗变换公式。

因此，若代码严格实现这些公式，并通过场景 B/C 断言，MVP 结果具备基础可信度。

## R11

数值稳定性策略：

- 对复平方根结果做分支修正，保证 `Re(gamma)>=0`、`Re(Z0)>=0`；
- VSWR 计算时分母使用 `max(1-|Gamma|, EPS)` 防止除零；
- 断言使用相对误差而不是绝对误差，适配数值尺度。

潜在风险：`tanh(gamma*l)` 在超大实部时可能溢出，工程上可通过缩放或分段公式增强稳定性。

## R12

关键参数与调节建议：

- `R, G` 增大时衰减 `alpha` 增大；
- `L/C` 比值决定低损耗近似下的 `|Z0|`；
- `length` 增大时反射在输入端衰减更明显（有损线）；
- 扫频范围建议覆盖目标系统基频到主要谐波频段。

## R14

常见失效模式：

1. 单位错误（`nH` 当成 `H`，`MHz` 当成 `Hz`）；
2. 负载阻抗填写错误导致误判匹配状态；
3. 极端参数导致复数双曲函数数值不稳定；
4. 把本模型用于强频散或非均匀线而得到偏差结论。

排查顺序建议：先检查单位，再检查边界条件，再检查参数量级。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `RLGCLine._series_impedance` 与 `_shunt_admittance` 先构建 `R+jωL`、`G+jωC` 两个基础复量。  
2. `RLGCLine.propagation_constant` 计算 `gamma = sqrt(Z'Y')`，并做分支修正使衰减常数非负。  
3. `RLGCLine.characteristic_impedance` 计算 `Z0 = sqrt(Z'/Y')`，并统一阻抗主值分支。  
4. `input_impedance` 代入有限长传输线解析式，用 `tanh(gamma*l)` 映射负载到输入端。  
5. `reflection_coefficient` 计算负载与输入端反射系数，再由此得到 VSWR。  
6. `run_frequency_sweep_demo` 对一组频率向量化执行步骤 2~5，并打印关键指标表。  
7. `validate_matched_load_case` 与 `validate_quarter_wave_transformer` 分别断言两条物理可校验关系。  
8. `run_profile_snapshot` 用双曲函数表达式给出 `V(x), I(x)` 剖面采样，`main` 汇总执行并收尾。

这里没有把第三方包当黑盒算法：`numpy` 仅负责复数与向量化算术，核心传输线流程都由源码显式实现。
