## R01

本条目给出一个可运行的 β 衰变理论最小实现（MVP），覆盖两个核心对象：
1. 放射性衰变时间分布（指数律与半衰期估计）。
2. β 粒子动能谱（允许跃迁近似下的费米谱形）。

目标是用少量代码复现可解释的理论-数值闭环，而不是构建完整核数据库或高精度核结构求解器。

## R02

β⁻ 衰变可表示为：
\[
 n \rightarrow p + e^- + \bar\nu_e
\]
其中电子与反中微子连续分配可用衰变能量，因此电子动能呈连续谱，而非离散线谱。

在弱相互作用的费米理论近似下，若忽略高阶辐射修正与复杂禁戒跃迁矩阵元，谱形由相空间因子与库仑修正主导。

## R03

MVP 使用的主方程：

1. 核数随时间衰减：
\[
N(t)=N_0 e^{-\lambda t}, \quad T_{1/2}=\frac{\ln 2}{\lambda}
\]

2. 允许 β 衰变电子动能谱（未归一化）：
\[
S(T) \propto F(Z,T)\,p\,W\,(Q-T)^2
\]
其中
\[
W=T+m_e,\quad p=\sqrt{W^2-m_e^2}
\]
`F(Z,T)` 为费米函数近似（库仑修正），`Q` 为衰变 Q 值。

## R04

`demo.py` 固定参数运行（无交互输入）：
- 示例核反应：中子 β⁻ 衰变。
- `Q = 0.782 MeV`
- 真值半衰期：`610 s`（用于验证估计器，不代表最新实验共识值拟合流程）
- 输出：
  - 半衰期估计表（真值、MLE、线性回归、PyTorch 优化）
  - 拟合诊断（`R^2`、`N0`）
  - 能谱诊断（均值、标准差、KS 检验）

## R05

实现涉及的变量与单位：
- `T`：电子动能，单位 `MeV`
- `W`：电子总能量，单位 `MeV`
- `p`：电子动量（采用 `c=1` 的自然单位）
- `Q`：衰变释放能，单位 `MeV`
- `lambda`：衰变常数，单位 `s^-1`
- `T_half`：半衰期，单位 `s`

代码中显式使用 `M_E_MEV = 0.511` 与 `ALPHA = 1/137.035999084`。

## R06

时间分布算法（指数衰变）采用：
- 采样：`rng.exponential(scale=1/lambda_true, size=N)`
- MLE 估计：`lambda_hat = 1 / mean(t_i)`
- 半衰期换算：`T_half = ln(2)/lambda_hat`

该部分直接对应指数分布的统计最优估计，适合作为 MVP 基线。

## R07

能谱采样算法（拒绝采样）采用：
1. 在 `[0, Q]` 网格上计算未归一化谱形 `S(T)`。
2. 用数值积分归一化并构建 CDF（用于验证）。
3. 在矩形包络 `[0,Q] × [0,S_max]` 内提案采样。
4. 满足 `y < S(T)` 的样本接受，得到目标分布样本。

该方案避免对复杂 CDF 反函数求解，结构清晰且稳定。

## R08

半衰期二次估计方式：
- `scikit-learn` 线性回归拟合
  \[
  \log N(t)=\log N_0-\lambda t
  \]
- `PyTorch` 最小化同一关系的均方误差，参数化 `lambda=softplus(raw_lambda)` 保证正值。

这两种方法用于展示“统计回归视角”与“可微优化视角”的一致性。

## R09

复杂度（设 `N` 为衰变时间样本数，`M` 为能谱样本数）：
- 指数采样与 MLE：`O(N)`
- 生存曲线构造（排序+searchsorted）：`O(N log N)`
- 能谱拒绝采样：期望 `O(M / acc_rate)`
- 网格积分与 CDF 构造：`O(G)`，`G` 为网格点数（默认 4096）

在当前参数规模下可秒级运行。

## R10

数值稳定性处理：
- 对费米函数分母 `1-exp(-x)` 使用 `-expm1(-x)`，避免小 `x` 消减误差。
- 对动量计算使用 `clip` 防止浮点负零导致 `sqrt` 域错误。
- 对 `p` 使用最小安全下界 `1e-12`，避免除零。
- 拟合中仅对 `survivors > 0` 的点取对数。

## R11

正确性检查策略：
- 半衰期估计值应接近设定真值 `610 s`。
- 生存曲线线性化拟合应有较高 `R^2`。
- 采样能谱与理论 CDF 通过 KS 检验（通常表现为较小统计量，p 值不极端）。

这保证“理论公式、采样程序、统计估计”三者一致。

## R18

`demo.py` 的源码级算法流程可拆为 8 步：

1. `main()` 设定物理常量与实验参数（`Q`、`T_half_true`、样本量、随机种子）。
2. 使用指数分布生成衰变时间样本 `decay_times_s`，得到模拟“事件时间”。
3. 调用 `estimate_lambda_mle()` 计算 `lambda = 1/mean(t)`，换算得到第一版半衰期估计。
4. 调用 `survivor_counts()` 在时间网格上构造生存核数 `N(t)`。
5. 调用 `estimate_lambda_linear_regression()` 在 `log N(t)` 空间线性拟合斜率，得到第二版 `lambda`。
6. 调用 `estimate_lambda_torch()` 用可微优化最小化 `log N(t)` 残差，得到第三版 `lambda`。
7. 调用 `build_normalized_spectrum()` 构建 β 谱 PDF/CDF，再用 `sample_beta_energies()` 进行拒绝采样获得电子动能样本。
8. 使用 `scipy.stats.kstest` 比较样本分布与理论 CDF，一并汇总 `pandas` 表格输出，形成完整验证闭环。
