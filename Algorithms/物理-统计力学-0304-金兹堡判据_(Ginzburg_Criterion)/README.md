# 金兹堡判据 (Ginzburg Criterion)

- UID: `PHYS-0301`
- 学科: `物理`
- 分类: `统计力学`
- 源序号: `304`
- 目标目录: `Algorithms/物理-统计力学-0304-金兹堡判据_(Ginzburg_Criterion)`

## R01

金兹堡判据用于判断**平均场理论何时失效**。在临界点附近，序参量涨落会急剧增强；当涨落强度与平均场序参量本身同量级时，Landau 平均场近似不再可靠。

它回答的问题是：给定约化温度 `tau = |T-Tc|/Tc`，当前是否仍处于“平均场可用区间”。

## R02

本条目选用最小可复现模型：三维 `phi^4` Landau-Ginzburg 自由能近临界展开。

- 系数 `a=a0*tau` 控制离临界点远近。
- `b` 控制四次非线性强度。
- `c` 控制梯度项（空间相关）强度。
- `kBT` 给出热噪声强度。

我们通过数值积分直接计算“相关长度尺度内的涨落方差”，再与平均场序参量平方比较，得到金兹堡比值 `R(tau)`。

## R03

MVP 使用的核心定义：

1. 相关长度（平均场尺度）：`xi(tau)=sqrt(c/(a0*tau))`。
2. 平均场序参量平方（`T<Tc` 侧）：`phi0^2(tau)=a0*tau/b`。
3. 截断到 `k<=1/xi` 的涨落方差（3D 各向同性）：

`<delta_phi^2>_xi = (kBT/(2*pi^2)) * int_0^{1/xi} k^2/(a0*tau + c*k^2) dk`

4. 金兹堡比值：`R(tau)=<delta_phi^2>_xi / phi0^2`。

判据：`R << 1` 为平均场有效；`R ~ 1` 为金兹堡边界；`R > 1` 为涨落主导区。

## R04

在该 3D 设定下，`R(tau)` 随 `tau` 增大单调下降，并满足近临界标度 `R ~ tau^(-1/2)`。因此可以定义金兹堡温区宽度 `tau_G` 使得：

`R(tau_G) = 1`

`demo.py` 用二分法求解该方程，给出数值 `tau_G`，并与同一截断假设下的闭式近似 `tau_G ~ C^2` 进行交叉检查。

## R05

适用假设：

- 近临界、弱到中等耦合的连续相变场景；
- 各向同性、均匀介质近似；
- 以 Landau 展开系数 `(a0,b,c)` 描述粗粒化自由能；
- 仅做 MVP 级标度与边界估计，不做材料常数高精度拟合。

不适用于强各向异性、强无序、低维强涨落精确临界指数拟合等场景。

## R06

`demo.py` 输入输出约定：

- 输入：脚本内置参数 `LandauParams(a0,b,c,kbt)` 和 `tau` 网格，无交互输入。
- 输出：
1. `tau` 扫描表（`xi`、`phi0^2`、`<delta_phi^2>_xi`、`R`、区间标签）；
2. 数值二分得到的 `tau_G`；
3. 闭式近似 `tau_G` 及相对误差；
4. 一组通过/失败断言，最终打印 `Validation: PASS/FAIL`。

## R07

算法主流程（高层）：

1. 用 `a0,b,c,kBT` 定义 Landau 参数。
2. 对 `tau` 网格逐点计算 `xi`、`phi0^2` 和涨落积分。
3. 形成金兹堡比值 `R(tau)` 并做物理区间分类。
4. 用二分法求 `R(tau)=1`，得到 `tau_G`。
5. 用闭式近似估计 `tau_G`，与数值解比较。
6. 输出表格与断言结果，构成可回归的最小验证闭环。

## R08

设 `N_tau` 为温度采样点数，`N_k` 为每个积分的波数离散点数：

- 单点评估 `R(tau)`：`O(N_k)`；
- 网格扫描：`O(N_tau * N_k)`；
- 二分法：`O(log((tau_high-tau_low)/tol) * N_k)`；
- 总体空间复杂度：`O(N_tau + N_k)`（主要用于表格和积分网格）。

当前 MVP 默认规模很小（毫秒到秒级）。

## R09

数值稳定策略：

- 对 `tau` 和参数做正值检查，避免非物理输入；
- 积分使用 `numpy.trapezoid`，在固定 `k` 网格上稳定可复现；
- 二分法仅在有效括区（低端 `R>1`，高端 `R<1`）下执行，避免无根误判；
- 通过“`R(tau)` 单调性”与“`R(tau_G)` 接近 1”双重断言验证求根质量。

## R10

MVP 技术栈：

- Python 3
- `numpy`：数值数组、积分、求根辅助计算
- `pandas`：输出诊断表
- 标准库 `dataclasses`

未使用黑箱临界现象库；关键公式、积分、判据与求根均在源码中显式实现。

## R12

输出字段说明：

- `tau=|T-Tc|/Tc`：离临界点的约化温距；
- `xi`：平均场相关长度；
- `phi0^2`：平均场序参量平方；
- `<delta_phi^2>_xi`：相关长度尺度内涨落方差；
- `ginzburg_ratio`：`R(tau)`；
- `regime`：按 `R` 分类的物理区间（`Mean-field valid / Crossover / Fluctuation-dominated`）；
- `tau_G (numerical)`：二分求解得到的金兹堡边界；
- `tau_G (asymptotic)`：同截断下的闭式近似。

## R13

本问题不是近似优化问题，因此没有“近似比”定义；它是确定性数值判据计算。

可验证保证由断言给出：

1. `R(tau)` 随 `tau` 单调下降；
2. 二分所得 `tau_G` 满足 `R(tau_G)` 近 1；
3. `tau_G` 落在合理温区；
4. 数值 `tau_G` 与闭式近似误差受控。

这些检查通过时，视为 MVP 实现正确。

## R14

常见失效模式：

1. 参数量纲/归一化不一致，导致 `tau_G` 数值失真；
2. 括区选择错误（例如两端都 `R>1`），二分法无法收敛；
3. `N_k` 过小导致积分误差放大；
4. 在远离临界区使用该模型，物理意义下降。

- 课程教学中展示“从自由能到判据”的可计算闭环。

## R17

方案对比：

- 纯解析缩放估计速度快，但常数项与截断假设易被忽略；
- 全格点蒙特卡洛最真实，但开发和算力成本高；
- 本 MVP 取中间路线：保留 Landau-Ginzburg 物理结构，使用透明数值积分与求根，低成本且可审计。

因此它适合作为“临界区有效性判定”的基线实现。

## R18

`demo.py` 源码级算法流程拆解（8 步）：

1. `validate_params` 检查 `a0,b,c,kbt` 全为正，阻断非物理输入。  
2. `correlation_length` 与 `mean_field_order_parameter_sq` 分别计算 `xi(tau)` 和 `phi0^2(tau)`。  
3. `fluctuation_variance_within_xi` 构建 `k in [0,1/xi]` 网格，并用 `numpy.trapezoid` 计算 3D 涨落积分。  
4. `ginzburg_ratio` 用 `R=<delta_phi^2>_xi/phi0^2` 形成判据核心标量。  
5. `build_report_table` 在 `tau` 对数网格上逐点评估并生成 `pandas` 诊断表。  
6. `estimate_ginzburg_tau` 对方程 `R(tau)-1=0` 做二分迭代，得到数值 `tau_G`。  
7. `asymptotic_tau_g` 计算同一截断假设下的闭式近似 `tau_G`，用于独立交叉验证。  
8. `main` 汇总打印表格与关键指标，并执行 4 条断言，最后输出 `Validation: PASS/FAIL`。  

第三方库只承担基础数值运算；判据构造、积分边界、求根逻辑和验证标准都在源码中显式展开。
