# 逆幂法 (Inverse Iteration)

- UID: `MATH-0058`
- 学科: `数学`
- 分类: `线性代数`
- 源序号: `58`
- 目标目录: `Algorithms/数学-线性代数-0058-逆幂法_(Inverse_Iteration)`

## R01

逆幂法（Inverse Iteration）用于求矩阵特征值问题中“某个目标谱附近”的特征向量。

给定方阵 `A` 与位移 `sigma`，迭代核心是：

- 解线性方程：`(A - sigma I) y_{k+1} = x_k`
- 归一化：`x_{k+1} = y_{k+1} / ||y_{k+1}||`

当 `sigma=0` 时是经典逆幂法，趋向最小模特征值方向；当 `sigma!=0` 时是位移逆幂法，趋向最接近 `sigma` 的特征值对应方向。

## R02

历史定位（简要）：

- 逆幂法是幂法的直接“逆算子”扩展；
- 它把“放大最大模特征值方向”转成“放大最小模（或位移后最小模）方向”；
- 在现代稀疏特征值算法中，shift-invert 思想仍是核心工具（如 Arnoldi/Lanczos 的 shift-invert 变体）。

## R03

本条目 MVP 问题定义：

- 输入：
  - 实方阵 `A in R^{n x n}`；
  - 位移 `sigma`；
  - 最大迭代次数 `max_iters`；
  - 收敛阈值 `tol`；
  - 随机种子 `seed`（用于可复现初值）。
- 输出：
  - 目标特征值近似 `lambda_hat`（Rayleigh 商）；
  - 对应单位特征向量近似 `v_hat`；
  - 残差 `||A v_hat - lambda_hat v_hat||`；
  - 收敛标志、迭代次数与残差历史。

## R04

时间复杂度：

- 本 MVP 每轮用 `numpy.linalg.solve` 解一次稠密线性系统，单轮约 `O(n^3)`；
- 总迭代 `k` 轮，整体约 `O(k n^3)`；
- 若工程实现预先对 `A-sigma I` 做一次 LU 分解，则可降为“预处理 `O(n^3)` + 每轮回代 `O(n^2)`”。

## R05

空间复杂度：

- 存储矩阵 `A`：`O(n^2)`；
- 位移矩阵与迭代向量（`shifted, x, y`）：`O(n^2) + O(n)`；
- 残差历史：`O(k)`。

若仅看迭代额外向量开销，则为 `O(n + k)`；主存储仍由矩阵主导。

## R06

3x3 微型示例（解释“为何会收敛到目标谱附近”）：

令 `A = diag(1, 3, 5)`，取位移 `sigma = 2.8`。则
`(A-sigma I)^(-1) = diag(-0.555..., 5.0, 0.4545...)`。

对任意含三方向分量的初始向量 `x0`，每次乘以 `(A-sigma I)^(-1)` 后：

- 第二维（对应特征值 3）的放大因子最大（绝对值 5.0）；
- 其余方向放大较小；
- 归一化后方向逐步向 `e2`（特征值 3 的特征向量）靠拢。

这就是位移逆幂法锁定“离 `sigma` 最近特征值”的直观机制。

## R07

算法意义：

- 当只关心某个局部谱（不是最大特征值）时，比直接幂法更有针对性；
- 是理解 shift-invert 光谱变换的最小可运行模型；
- 在结构振动、稳定性分析、PDE 离散特征问题中常用；
- 也是更高级 Krylov 局部特征值算法的基础部件。

## R08

核心理论：

设 `A` 可对角化，`A v_i = lambda_i v_i`。对位移矩阵 `B = A - sigma I`，有

`B^{-1} v_i = 1/(lambda_i - sigma) * v_i`。

因此对 `B^{-1}` 做幂迭代时，主导因子变为 `|1/(lambda_i-sigma)|`。主导方向对应最小 `|lambda_i-sigma|`，即最接近 `sigma` 的特征值。迭代后再用 Rayleigh 商
`rho(x)=x^T A x`
估计目标特征值。

## R09

适用条件与局限：

适用：

- 需要某个指定谱区间附近的特征对；
- 可高效解 `(A-sigma I) y = x`；
- 初值在目标特征向量方向分量非零（随机初始化通常满足）。

局限：

- 若 `A-sigma I` 奇异则迭代无法进行；
- `sigma` 过于贴近特征值会导致线性系统病态，数值不稳定风险上升；
- 对大规模问题，若不利用稀疏结构与分解复用，成本会偏高。

## R10

正确性说明（工程可验证版）：

1. 迭代等价于对 `B^{-1}`（`B=A-sigma I`）执行幂法。  
2. 在目标特征值最近且初值非正交条件下，方向收敛到目标特征向量。  
3. 用 `lambda_hat = x^T A x` 得到特征值近似。  
4. 若已接近特征对，则残差 `||A x - lambda_hat x||` 应接近 0。  
5. `demo.py` 使用残差阈值停止，并与 `numpy.linalg.eigh` 的最近位移特征对交叉校验。

## R11

误差与稳定性：

- 迭代截断误差由 `tol` 与 `max_iters` 控制；
- 浮点误差主要来自反复线性求解和归一化；
- 当 `A-sigma I` 条件数较大时，解方程误差会放大；
- 因此工程上需在“收敛速度（sigma 贴近目标）”与“线性系统稳定性”之间折中。

## R12

性能视角：

- 理论上逆幂法每步比普通幂法贵，因为要解线性系统而非仅 matvec；
- 但若目标是内部特征值，它常比全特征分解更省，且定位能力更强；
- 关键性能杠杆是“分解复用 + 稀疏求解器”，而不是迭代框架本身；
- 本 MVP 选择可审计优先，直接 `solve` 保持代码透明。

## R13

本目录 MVP 可验证保证：

- `demo.py` 固定随机种子，结果可复现；
- 输出 `lambda_hat`、`residual_norm`、迭代次数与残差序列；
- 自动断言：
  - 已收敛；
  - 残差足够小；
  - 与参考目标特征值误差足够小；
  - 与参考特征向量方向高度一致（符号无关）。

## R14

鲁棒性与常见失效模式：

- 输入非法（非方阵、含 `nan/inf`、非法参数）会提前报错；
- `A-sigma I` 奇异时直接拒绝执行；
- 若达到 `max_iters` 仍未达阈值，返回 `converged=False`；
- 若迭代中出现零范数或非有限量，立即抛出异常防止静默错误。

## R15

实现设计（对应 `demo.py`）：

- `InverseIterationResult`：结构化保存结果。  
- `validate_inputs`：统一参数与矩阵合法性校验。  
- `inverse_iteration`：逆幂法主循环，显式执行“解线性系统 + 归一化 + 残差评估”。  
- `build_demo_matrix`：生成对称测试矩阵。  
- `reference_eigenpair_near_shift`：仅做最终对照，不参与主算法。  
- `run_checks`：集中断言收敛性与精度。  
- `main`：组织实验并打印摘要。

## R16

相关算法链路：

- 前驱：幂法（主特征值方向）；
- 直接变体：位移逆幂法（本条目）、Rayleigh 商迭代；
- 子空间方法：Lanczos / Arnoldi（可一次逼近多个局部特征值）；
- 全谱方法：QR / 分治特征分解（成本更高但得到完整谱）。

## R18

`demo.py` 源码级算法流程拆解（8 步）：

1. `main` 调用 `build_demo_matrix` 构造对称矩阵 `A`，并设定位移 `sigma`。  
2. `inverse_iteration` 先在 `validate_inputs` 中校验方阵、有限值和参数合法性。  
3. 构造 `shifted = A - sigma I` 并检查其非奇异性。  
4. 用固定 `seed` 生成随机初始向量 `x`，归一化到单位范数。  
5. 每轮迭代执行 `y = solve(shifted, x)`，即显式求解 `(A-sigma I) y = x`。  
6. 将 `y` 归一化为 `x_next`，再计算 `lambda_hat = x_next^T A x_next` 与残差 `||A x_next - lambda_hat x_next||`。  
7. 若残差小于 `tol` 则提前返回，否则继续下一轮直到 `max_iters`。  
8. 迭代结束后，`main` 用 `reference_eigenpair_near_shift`（`eigh`）做验算，并由 `run_checks` 统一断言精度与方向一致性。

说明：MVP 的核心逆幂迭代并未调用现成“特征值黑盒”直接求解，`eigh` 仅用于最终对照验证。
