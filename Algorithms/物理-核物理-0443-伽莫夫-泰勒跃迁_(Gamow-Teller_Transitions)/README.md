# 伽莫夫-泰勒跃迁 (Gamow-Teller Transitions)

- UID: `PHYS-0423`
- 学科: `物理`
- 分类: `核物理`
- 源序号: `443`
- 目标目录: `Algorithms/物理-核物理-0443-伽莫夫-泰勒跃迁_(Gamow-Teller_Transitions)`

## R01

伽莫夫-泰勒（GT）跃迁是 β 衰变中由轴矢量弱流主导的允许跃迁类型，核心特征是“自旋翻转 + 同宇称”。可写为核矩阵元算符 `sigma * tau_±` 的跃迁强度计算问题。

本目录 MVP 的算法化对象是“给定候选核跃迁，估计其 GT 允许性与半衰期量级”，流程为：
- 输入: `Z_daughter, Q, J_i, J_f, parity_i, parity_f, M_GT, mode(beta-/beta+)`
- 步骤1: 按 GT 选择定则判断是否允许（`ΔJ=0,1` 且 `0->0` 禁止，宇称不变）
- 步骤2: 计算约化强度 `B(GT) = |M_GT|^2 / (2J_i+1)`
- 步骤3: 数值积分相空间因子 `f(Z,Q)=∫F(Z,W)pW(W0-W)^2 dW`
- 步骤4: 用 `ft = K / ((g_A/g_V)^2 B(GT))` 与 `t1/2 = ft / f` 估计半衰期
- 输出: 每条跃迁的 `allowed_gt, B_GT, f, ft_s, t1_2_s`

## R02

关键时间点：
- 1934: Fermi 给出 β 衰变理论（矢量流框架）
- 1936: Gamow 与 Teller 引入自旋依赖项（轴矢量贡献），建立 Fermi/GT 两类允许跃迁分类

因此 GT 跃迁通常以 1936 年作为提出年代锚点。

## R03

提出背景是早期 Fermi 理论对实验选择定则解释不完整：
- 仅有纯矢量项时，许多 `ΔJ=1` 且宇称不变的实验跃迁难以统一解释
- 核 β 衰变观测显示存在明显自旋结构效应

Gamow-Teller 的贡献是把核自旋算符纳入弱跃迁矩阵元，使“费米跃迁（自旋不变）”与“GT 跃迁（自旋翻转）”成为统一框架下的两条允许通道。

## R04

若一次评估 `N` 个候选跃迁，且每次数值积分平均需 `M` 次被积函数评估，则：
- 时间复杂度（典型）: `O(N*M)`
- 若使用近似闭式相空间公式（本实现未用）可退化到近似 `O(N)`

本 MVP 的主要耗时来自 `scipy.integrate.quad` 对每个跃迁做一维自适应积分。

## R05

空间复杂度：
- 计算每条跃迁只维护常数级中间量，单条为 `O(1)`
- 汇总 `N` 条结果到 `pandas.DataFrame`，总体为 `O(N)`

## R06

以 `6He -> 6Li`（示例参数）为例：
- 输入: `Q=3.508 MeV, J_i=0, J_f=1, parity +->+, M_GT=2.0, beta-`
- 选择定则: `ΔJ=1`、宇称不变，判定为允许 GT
- 强度: `B(GT)=|2.0|^2/(2*0+1)=4.0`
- 数值积分得到相空间因子 `f`（程序实时计算）
- 得到 `ft` 和 `t1/2` 的量级估计

`demo.py` 同时给出一个“故意不满足定则”的样例，展示筛除逻辑。

## R07

算法意义：
- 物理层面: GT 跃迁是核 β 衰变分类、核结构信息提取与弱相互作用参数研究的基础对象
- 方法层面: 把“核选择定则 + 矩阵元 + 相空间积分”转化为可重复的计算流水线
- 工程层面: 适合作为更复杂核结构计算（shell model/QRPA）的前置校验模块

## R08

直接依赖的理论与数值组件：
- Fermi 黄金律（跃迁率与矩阵元平方成正比）
- 角动量耦合与选择定则（Wigner-Eckart 思想）
- 允许 β 衰变相空间积分公式
- 库仑修正（Fermi 函数近似）
- 一维自适应数值积分（QUADPACK 路径，经 SciPy `quad`）

## R09

适用前提与边界：
- 仅覆盖“允许”β 衰变中的 GT 分支
- 假设形状因子 `C(W)=1`（未显式建模禁戒跃迁修正）
- `M_GT` 视为外部给定参数，不在本 MVP 内由核结构模型反演
- `Q>0` 才有可用相空间
- 库仑修正采用近似表达，不用于高精度核数据库拟合

## R10

正确性依据来自三层一致性：
- 物理结构正确: 用 `sigma*tau` 对应 GT 通道，选择定则实现与标准定义一致
- 数学形式正确: `B(GT)`、`f(Z,Q)`、`ft`、`t1/2` 的关系按允许跃迁公式连接
- 程序流程正确: 先判定可行域，再计算积分和寿命，避免对无效输入给出伪物理结果

## R11

数值稳定性与误差传播：
- 阈值附近 `Q->0` 时，`W0` 接近积分下限，结果对舍入更敏感
- 被积函数含 `p=sqrt(W^2-1)`，端点附近梯度较大；采用自适应积分与较严 `epsabs/epsrel`
- 库仑因子 `x/(1-exp(-x))` 在 `x~0` 会有消差；代码加入小 `x` 线性近似分支
- 总体上，模型误差（忽略禁戒项、核结构细节）通常大于纯数值积分误差

## R12

真实计算成本（超越大 O）：
- 单条跃迁的积分是一维、低维度，CPU 成本很低
- 常数项主要来自 Python 函数调用与 `quad` 迭代控制
- 表格构建与格式化输出（pandas）在小样本下可忽略
- 若批量扫描上万条跃迁，积分调用次数成为主要瓶颈，可考虑向量化近似或并行

## R13

误差界与保证：
- 本算法是确定性数值算法，不涉及随机采样概率保证
- `quad` 给出可控积分容差（本实现设置 `epsabs=1e-10`, `epsrel=1e-8`）
- `ft/t1/2` 的“物理可信区间”更多受输入 `M_GT` 与模型近似影响，而非积分器本身

## R14

鲁棒性与失效模式：
- `parity` 编码非法（非 ±1）会直接判为无效
- `ΔJ` 不满足 GT 允许条件时，输出 NaN 并给出原因
- `Q<=0` 时无相空间，直接拒绝计算
- 若把本模型用于禁戒跃迁或强核形变体系，结果会系统偏差
- 极大 `Z` 或极端低能区，简化库仑因子近似可能失真

## R15

工程实现要点：
- 先“规则过滤”再“数值积分”，避免无意义积分开销
- 采用 `dataclass` 固化输入字段，降低字段错位风险
- 所有关键常量集中定义，便于做 `g_A` 淬火或常数替换实验
- 输出端统一为 DataFrame，便于后续接 benchmark 或可视化
- 对 NaN 路径显式标注 `reason`，便于批量排查

## R16

前驱/后继与应用位置：
- 前驱: Fermi β 衰变理论、核角动量代数
- 同层配对: Fermi 跃迁（自旋不变）与 GT 跃迁（自旋翻转）共同构成允许 β 跃迁分类
- 后继: 壳模型/QRPA/ab initio 中的 GT 强度分布计算与淬火效应研究
- 应用: 核素衰变评估、天体核合成弱反应率、双 β 衰变核矩阵元分析等

## R17

Python MVP 方案（本目录 `demo.py`）：
- 依赖: `numpy + scipy + pandas`
- 可运行方式:
  - `cd "Algorithms/物理-核物理-0443-伽莫夫-泰勒跃迁_(Gamow-Teller_Transitions)"`
  - `uv run python demo.py`
- 输入: 脚本内置 4 条样例跃迁（3 条允许候选 + 1 条故意禁戒）
- 输出: 表格列出 `allowed_gt, reason, B_GT, f, ft_s, t1_2_s, log10_t1_2_s`
- 实现边界:
  - 提供“可解释的最小计算链路”
  - 不尝试替代高精度核数据库或全尺度核结构求解器

## R18

源码追踪与算法拆解（非黑箱，8 步）：
1. 读取一条跃迁参数，先执行 GT 选择定则检查（角动量与宇称约束）。
2. 若不满足条件，立即返回 `NaN + reason`，保证结果语义正确。
3. 对允许跃迁计算 `B(GT)=|M_GT|^2/(2J_i+1)`。
4. 将 `Q` 转成无量纲端点能量 `W0=1+Q/m_e`，构造被积函数 `F*p*W*(W0-W)^2`。
5. 在被积函数内部计算简化 Fermi 库仑修正 `x/(1-exp(-x))`，并对 `x~0` 做稳定化近似。
6. 调用 `scipy.integrate.quad` 进入 QUADPACK 的自适应积分流程（区间递归细分 + 误差估计控制）。
7. 用积分结果 `f` 与 `B(GT)` 合成 `ft` 和 `t1/2`，形成可比较的寿命量级输出。
8. 将多条结果汇总为 pandas 表，并用 numpy 计算 `log10(t1/2)` 便于跨数量级比较。

其中第 6 步说明：虽然外部接口是 `quad(...)` 一行调用，但核心仍是“自适应求积 + 局部误差驱动细分”的确定性数值算法，而非不可解释黑箱模型。
