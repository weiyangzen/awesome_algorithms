## R01

本条目用一个可执行、可解释的最小模型表达核反应堆物理中的核心问题：
- 中子在堆芯中一边扩散泄漏、一边被吸收、一边通过裂变再生
- 系统是否临界由有效增殖因子 `k_eff` 决定

MVP 采用一维裸堆平板、单群扩散近似：
`-D d^2(phi)/dx^2 + Sigma_a * phi = (1/k_eff) * nuSigma_f * phi`。

## R02

核反应堆物理的工程主线可概括为：
- 1940s: 链式反应与扩散理论建立，临界计算成为堆设计基础
- 1950s-1970s: 多群扩散/输运、均匀化与反应率工程化
- 现代: 高保真蒙特卡罗与确定论并行，但扩散临界模型仍是教学与早期设计的第一层

因此，本实现聚焦“扩散临界基础环节”，强调结构透明而非高阶耦合复杂度。

## R03

数学对象与边界条件：
- 几何: 长度 `L` 的 1D 平板
- 边界: 真空边界，`phi(0)=phi(L)=0`
- 常数: `D, Sigma_a, nuSigma_f` 为空间常数

离散后得到广义本征问题：
`A * phi = (1/k_eff) * F * phi`
- `A`: 泄漏 + 吸收损失算子
- `F`: 裂变产生算子（本 MVP 为对角矩阵）

`k_eff` 是主本征值对应的乘倍因子，主本征向量是基模中子通量分布。

## R04

离散化采用二阶中心差分（`N` 个内部网格）：
- `dx = L/(N+1)`
- 二阶导离散成三对角拉普拉斯
- 形成 `A` 的主对角与次对角项

程序显式构造：
- `A[i,i] = 2D/dx^2 + Sigma_a`
- `A[i,i±1] = -D/dx^2`
- `F[i,i] = nuSigma_f`

这使“物理方程 -> 数值线性系统”路径可直接审计。

## R05

复杂度（以稠密实现计）：
- `A` 构建: `O(N^2)` 存储（物理非零结构本质是三对角 `O(N)`）
- 每轮幂迭代: 一次线性求解 `O(N^3)`（若用通用稠密求解器）
- 总体: `O(I * N^3)`，`I` 为收敛迭代次数

对当前教学规模 `N=120`，运行开销很小，换来更直接的源码可读性。

## R06

`demo.py` 给出三条相互独立的 `k_eff` 计算路径：
1. 主算法: 源迭代/幂迭代（显式 `A^{-1}F` 作用）
2. SciPy 校验: 广义本征值 `A v = lambda F v`，取 `k=1/lambda_min`
3. Torch 校验: 用 PyTorch 再实现一次幂迭代

同时输出通量剖面、峰值位置，以及与解析裸堆公式的差异。

## R07

本算法覆盖了反应堆物理最关键的判据链：
- 由材料参数 (`D, Sigma_a, nuSigma_f`) 与几何尺度 `L` 决定 `k_eff`
- `k_eff < 1` 次临界，`k_eff = 1` 临界，`k_eff > 1` 超临界
- 主本征通量形状反映了“泄漏抑制 + 中心峰化”趋势

因此它可以直接作为更复杂多群/多区模型前的基线模块。

## R08

最小工具栈与职责：
- `numpy`: 矩阵/向量构建与基础线代

## R10

正确性闭环来自四层一致性：
- 物理一致: 符合单群扩散临界方程
- 数学一致: 转换为标准广义本征形式
- 数值一致: 幂迭代与广义特征值结果一致
- 跨后端一致: NumPy 与 Torch 独立实现结果一致

脚本中通过断言固化这些一致性条件。

## R11

数值稳定性要点：
- 幂迭代每步归一化，避免通量向量幅值漂移
- 通过 Rayleigh 商 `k=(phi^T F phi)/(phi^T A phi)` 估计本征值
- 使用双精度 (`float64`) 降低舍入影响
- 迭代终止条件为 `|k_new-k_old| < tol`

在默认参数下收敛稳定，且对初值不敏感（正向量初值即可）。

## R18

`demo.py` 源码级算法流（8 步，非黑箱）：
1. 读取 `Reactor1DConfig` 并执行输入合法性检查。  
2. `build_diffusion_system()` 用二阶差分显式组装 `A`（泄漏+吸收）和 `F`（裂变源）。  
3. `power_iteration_keff()` 执行源迭代：每轮解 `A * phi_new = F * phi_old`，归一化通量并更新 Rayleigh 商 `k`。  
4. 满足 `|k_new-k_old| < tol` 后返回主本征值 `k_power` 与主本征向量（通量）。  
5. `scipy_generalized_keff()` 独立求解广义本征问题 `A v = lambda F v`，由 `k=1/lambda_min` 得到 `k_scipy`。  
6. `torch_power_iteration_keff()` 用 PyTorch 重复同一迭代流程得到 `k_torch`，用于跨后端一致性校验。  
7. `run_length_sweep()` 扫描多组 `L`，得到 `(1/L^2, 1/k)` 数据，再用 `LinearRegression` 拟合并对照理论截距/斜率。  
8. 主程序汇总为 pandas 表，打印关键指标并执行断言（收敛性、一致性、拟合质量）。  

补充：虽然使用了 SciPy / sklearn / torch，但它们仅承担求解器或回归器角色；反应堆物理方程离散、迭代更新、物理量提取与校验逻辑都在源码中逐步展开。
