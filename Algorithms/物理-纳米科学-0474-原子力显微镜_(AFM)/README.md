## R01

本条目把 AFM（Atomic Force Microscopy）的核心成像与反演问题做成可运行 MVP：
- 正向：`真实表面 -> 探针展宽 -> 漂移 + 噪声 -> 测量图`
- 反向：`测量图 -> 漂移估计/去除 -> 形态学反卷积 -> 可微分优化细化`

`demo.py` 中对应到 4 个阶段图：
- `measured_raw`
- `drift_corrected`
- `erosion_recon`
- `torch_refined`

并输出与真值（仿真已知）之间的 `RMSE`、相关系数和前向一致性误差。

## R02

AFM 相关里程碑（面向算法背景）：
- 1981 年：Binnig、Quate、Gerber 发明 AFM 概念并完成早期实验原型
- 1986 年：扫描隧道显微镜相关工作获诺奖，扫描探针显微技术进入主流
- 之后：AFM 从“仅成像”扩展到“成像 + 力谱 + 纳米机械参数反演”

本 MVP 关注其中最常见、最工程化的一段：
- 有限探针半径导致的形貌展宽
- 扫描漂移导致的低频背景形变
- 通过物理先验和数值优化恢复更接近真实表面的高度图

## R03

把 AFM 抽象为一个逆问题：

`y = D(x) + b + n`

- `x`: 真实表面高度图（待恢复）
- `D(·)`: 探针形状诱导的形态学膨胀型成像算子
- `b`: 扫描漂移（逐行偏置/斜率及其低频变化）
- `n`: 噪声

算法目标不是追求“完美物理全模型”，而是用最小代价建立可解释流程：
1. 合成可控真值数据
2. 显式构造失真机制
3. 用可追踪的步骤恢复并量化误差改进

## R04

设图像大小为 `N x N`，探针窗口边长 `K=2r+1`：
- 正向膨胀（`maximum_filter`）：时间复杂度近似 `O(N^2 * K^2)`
- 形态学反演（`grey_erosion`）：同阶 `O(N^2 * K^2)`
- 漂移回归（Ridge）：特征维度固定（6 维），近似 `O(N^2)`
- Torch 细化：每步一次 `max_pool2d` 和 TV 正则，`T` 步时约 `O(T * N^2 * K^2)`

在默认参数下（`N=96, r=3, T=180`）CPU 数秒内完成。

## R05

空间复杂度：
- 主数组（真值、测量、校正、重建）每个 `O(N^2)`
- 回归设计矩阵约 `6 * N^2` 元素，仍为 `O(N^2)`
- Torch 优化保留参数图和梯度图，量级也是 `O(N^2)`

总体空间复杂度为 `O(N^2)`，常数项低，适合单机快速实验。

## R06

默认实验（`demo.py` 内置）：
- 网格：`96 x 96`
- 纳米颗粒：20 个高斯凸起 + 低频基底起伏
- 探针半径：`3 px`
- 观测污染：行漂移 + 高斯平滑噪声

输出包括：
- `ridge_r2_on_anchor_pixels`（漂移拟合诊断）
- `final_loss/final_data_loss/final_tv`（Torch 优化诊断）
- 按 `rmse_to_truth_nm` 排序的阶段对比表

并写出 `metrics.csv` 方便后续自动化对比。

## R07

算法意义：
- 物理层：把 AFM 常见失真（探针卷积、漂移、噪声）显式建模
- 数值层：把“先验 + 优化”串成可落地的逆问题求解流程
- 工程层：提供一个可扩展基线，后续可替换更真实探针模型或材料参数模型

这个 MVP 的价值在于“可解释可运行”，而不是一次性追求复杂设备级全模型。

## R08

直接使用的理论/方法组件：
- 形态学成像近似：AFM 探针接触可近似为局部最大化（膨胀）
- 漂移建模：低频背景可用低阶多项式近似
- 逆问题正则化：
  - 数据一致项 `||D(z)-y_c||^2`
  - 全变分（TV）抑制高频伪纹理
  - 非负约束惩罚 `ReLU(-z)`
- 数值求解：Adam 迭代优化

实现依赖：`numpy + scipy + pandas + scikit-learn + torch`。

## R09

适用边界与前提：
- 当前前向模型把探针简化为平顶窗口最大化，不包含复杂探针几何和材料相互作用
- 漂移仅拟合低频多项式背景，不处理突发跳变/强非平稳漂移
- 示例是“仿真真值可得”的验证框架，真实实验需额外标定步骤
- 目标是纳米形貌重建原型，不覆盖完整 AFM 力谱反演

## R10

正确性链路（代码级）：
- 先构造真值 `true_surface`，确保有可对照基准
- 用显式前向算子生成 `measured`，避免“凭空去噪”
- 回归只拟合低频背景后再做反演，减少结构信号被误删
- 形态学侵蚀对膨胀失真做一阶逆近似
- Torch 阶段通过前向一致性损失约束重建图
- 脚本内断言要求重建阶段优于原始测量阶段

因此是一个闭环验证的、可复现实验流程。

## R11

数值稳定性处理：
- 所有随机过程固定 `seed=20260407`
- 漂移拟合使用 Ridge（`alpha=1`）抑制多项式共线性放大
- Torch 训练中每步 `clamp_(min=0)`，防止物理无意义负高度
- TV 正则抑制过拟合噪声，减少“锯齿型”伪解

稳定性风险主要来自模型失配，而不是浮点下溢/上溢。

## R12

实际耗时来源（超越大 O）：
- 最耗时部分是 Torch 迭代（默认 180 步）
- SciPy 形态学操作开销稳定且可预测
- Ridge 拟合在 6 维特征下成本很小
- `pandas` 仅用于结果汇总，开销可忽略

默认配置在普通 CPU 上约数秒，适合流水线批量验证。

## R13

误差与保证：
- 这是确定性算法（固定种子），同配置应复现同结果
- 误差度量包括：
  - `rmse_to_truth_nm`
  - `corr_to_truth`
  - `forward_rmse_to_corrected_nm`
- 该脚本给出的“保证”是流程一致性保证，不是实验仪器绝对计量保证

在真实数据上应把真值误差替换为外部标定或交叉表征指标。

## R14

鲁棒性与失效模式：
- 若样品几乎全是高频结构，低分位点锚点可能不足，漂移拟合会偏
- 若探针半径设错，侵蚀反演会出现过平滑或过锐化
- 若噪声远高于信号，Torch 可能偏向正则项而损失细节
- 若漂移是突发型（非低频），多项式背景模型会失效

调参顺序建议：先 `tip_radius_px`，再 `tv_weight`，最后 `torch_steps/lr`。

## R18

`demo.py` 源码级算法流（9 步，非黑箱）：

1. `generate_surface()` 逐个叠加高斯纳米颗粒，构造 `true_surface`（单位近似 nm）。
2. `forward_afm_scan()` 用 `scipy.ndimage.maximum_filter` 计算局部窗口最大值，显式实现“探针展宽”前向算子；随后叠加 `simulate_line_drift()` 产生的低频行漂移与噪声，得到 `measured_raw`。
3. `estimate_and_remove_drift()` 构造 `[1, x, y, xy, x^2, y^2]` 六维特征；在低分位像素上拟合 `Ridge`，得到 `drift_hat` 并从测量图中扣除，得到 `drift_corrected`。
4. `morphological_reconstruction()` 调用 `grey_erosion` 对校正图做侵蚀，作为对膨胀失真的一阶逆近似，得到 `erosion_recon`。
5. `torch_refine_inverse()` 以 `erosion_recon` 为初值，把待优化高度图设为参数 `z`。
6. 在每步迭代中，用 `torch.nn.functional.max_pool2d` 计算 `D(z)`（这一步就是前向 AFM 近似，不是神经网络黑箱）。
7. 显式计算损失：`MSE(D(z), corrected) + tv_weight*TV(z) + nonneg_weight*ReLU(-z)`，反向传播后用 Adam 更新，并 `clamp` 保持非负。
8. 用 `rmse/corrcoef/forward_from_surface` 量化各阶段与真值、与前向一致性的差异，汇总成 `pandas.DataFrame` 并落盘 `metrics.csv`。
9. 执行断言：最佳阶段必须来自重建分支，且 Torch 结果前向一致性优于原始测量，最后输出 `All checks passed.`。

补充：本实现虽然调用了 SciPy / sklearn / Torch API，但关键物理流程（前向算子、漂移建模、逆问题目标、约束项）全部在源码中显式展开，便于审查和替换。
