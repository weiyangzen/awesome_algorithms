## R01

多丝正比室（MWPC）是一类气体径迹探测器：带电粒子穿过气体后产生电离电子，电子在电场作用下漂移到阳极丝附近并触发雪崩放大，最终在读出通道形成电荷脉冲。与“只给是否命中”的计数器不同，MWPC 的算法重点在于把多根相邻丝上的电荷分布转成高于丝距的空间定位精度。

## R02

该条目的算法价值是展示一条最短可复现链路：

- 从三丝电荷读出（左/中/右）恢复平面击中位置；
- 将单平面击中串联为多平面直线径迹重建；
- 对比普通加权最小二乘与稳健 Huber 迭代在离群点下的差异；
- 给后续更复杂的粒子重建（多径迹分配、全局拟合）提供基础模块。

## R03

MVP 输入输出（脚本内固定参数，无交互）如下：

- 输入：
1. MWPC 几何参数（平面数、平面间距、丝距）；
2. 电荷响应参数（电荷横向扩展、增益涨落、电子学噪声）；
3. 一条真值直线轨迹 `x(z)=a*z+b`；
4. 一个注入离群平面（模拟串扰/异常放电）。
- 输出：
1. 每个平面的真值位置、测量位置、不确定度、总电荷表；
2. `Weighted-OLS` 与 `Weighted-Huber` 两种拟合指标对比；
3. 自动断言与 `All checks passed.`。

## R04

本 MVP 使用的核心模型与公式：

1. 轨迹模型：`x(z)=a*z+b`。
2. 三丝电荷共享模型（高斯近似）：
   `q_j = G * exp(-0.5 * ((x_true - x_j)/sigma_q)^2) + noise`，`j in {L,C,R}`。
3. 电荷质心定位：
   `x_meas = (q_L*x_L + q_C*x_C + q_R*x_R) / (q_L + q_C + q_R)`。
4. 位置测量方差近似：
   `sigma_x ~ pitch / sqrt(12 * (1 + alpha*SNR))`。
5. 全局拟合目标：
   - 加权 OLS：`min sum w_i * r_i^2`；
   - Huber 迭代：`w_i <- w_i * psi(r_i)`，其中 `psi` 对大残差降权。

## R05

设平面数为 `N`，Huber 迭代轮数为 `I`：

- 事件模拟与质心计算：`O(N)`；
- 一次加权线性拟合：`O(N)`（2 参数，`lstsq` 主成本随样本线性增长）；
- 稳健迭代拟合：`O(I*N)`；
- 空间复杂度：`O(N)`。

在示例参数中 `N=12`，`I<=20`，可用于快速教学和单事件调试。

## R06

`demo.py` 的最小闭环包括：

- 闭环 A（探测器读出模拟）：基于真值轨迹生成每平面三丝电荷；
- 闭环 B（平面位置重建）：用电荷质心把离散丝信号转为连续 `x_meas`；
- 闭环 C（全局径迹重建）：对 `x_meas` 做加权 OLS 与 Huber 迭代；
- 闭环 D（质量验证）：输出指标表并通过阈值断言。

## R07

优点：

- 保留 MWPC 的核心观测机制（电荷共享与质心重建）；
- 在极小代码规模下可直接比较稳健拟合收益；
- 结果可复现（固定随机种子）。

局限：

- 仅做 2D 直线，不含磁场弯曲与多重散射；
- 电荷分布使用高斯近似，未建模真实气体增益细节与电子学整形；
- 仅单条轨迹，不处理多轨道组合冲突。

## R10

正确性直觉：

1. 真值轨迹决定每层与相邻丝的距离关系，电荷分布携带位置信息；
2. 质心公式在噪声适中条件下能恢复亚丝距位置；
3. 全局直线拟合整合多平面信息，降低单平面噪声影响；
4. 注入离群点后，Huber 通过残差降权限制异常点对参数的牵引。

## R11

数值稳定策略：

- 电荷值统一下限裁剪到 `>=1`，避免除零与负电荷；
- 不确定度 `sigma_x` 使用上下限裁剪（`0.015~0.09 cm`）；
- 稳健尺度使用 MAD，避免被单个离群点放大；
- Huber 迭代设置收敛阈值与最大迭代数，防止无限循环。

## R12

关键参数说明：

- `wire_pitch_cm`：几何分辨率上限，越小通常越有利于位置重建；
- `charge_spread_cm`：电荷共享宽度，决定质心可用信息量；
- `electronics_noise`：噪声越高，`sigma_x` 越大，拟合权重越低；
- `outlier_shift_cm`：离群强度，决定稳健拟合优势是否明显；
- `k`（Huber）：残差从二次惩罚过渡到线性惩罚的阈值。

调参建议：

- 若质心偏置明显，可先校准 `charge_spread_cm` 与增益模型；
- 若离群频繁，减小 `k` 可增强异常抑制；
- 若无离群且噪声低，`Weighted-OLS` 速度更快且足够准确。

## R14

常见失效模式：

1. 增益/噪声模型与真实系统偏差过大导致参数不可迁移；
2. 平面几何标定误差（丝距、平面位置）引入系统偏差；
3. 高占有率下电荷叠加导致质心位置偏离真实轨迹；
4. 初始权重估计失真时稳健迭代可能收敛到次优解。

## R18

`demo.py` 源码级算法流程（9 步）：

1. `simulate_event` 生成平面坐标 `z` 与真值轨迹 `x_true=a*z+b`。  
2. 对每个平面定位最近中心丝 `x_center`，并构造左右相邻丝位置 `x_left/x_right`。  
3. 按高斯横向扩散模型生成三丝电荷 `q_left/q_center/q_right`，叠加电子学噪声并裁剪到正值。  
4. 计算总电荷 `q_total` 与电荷质心位置 `x_meas`，完成“丝读出 -> 连续位置”转换。  
5. 根据 `SNR` 估计每平面测量不确定度 `sigma_x`，构造基础权重 `w=1/sigma_x^2`。  
6. 在随机选中的一个平面注入位置偏移，形成离群测量用于稳健性验证。  
7. `weighted_linear_fit` 先给出加权 OLS 基线参数。  
8. `weighted_huber_fit` 进入 IRLS：由残差计算 MAD 尺度，按 Huber 规则更新稳健权重，再做加权线性求解，直到收敛或到达迭代上限。  
9. `evaluate` 计算 `truth_rmse/slope_error/intercept_error` 等指标，`main` 打印表格并执行断言。  

说明：第三方库仅用于数值线性代数和表格展示；探测器读出、质心重建、稳健权重更新和验收指标都在源码中显式实现，不是黑盒调用。
