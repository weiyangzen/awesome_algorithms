# 光线偏折 (Light Deflection)

- UID: `PHYS-0368`
- 学科: `物理`
- 分类: `广义相对论`
- 源序号: `387`
- 目标目录: `Algorithms/物理-广义相对论-0387-光线偏折_(Light_Deflection)`

## R01

问题定义：实现一个可复现、可验证的广义相对论光线偏折最小 MVP。

具体目标：
- 以史瓦西时空中的零测地线方程数值计算偏折角 `alpha_numeric`；
- 与弱场爱因斯坦一阶公式 `alpha_weak = 4GM/(c^2 b)` 做对比；
- 输出误差与守恒诊断，保证不仅“能跑”，而且“物理上自洽”。

## R02

理论背景：广义相对论中，光在弯曲时空中沿零测地线传播，不再是欧式直线。

对静态球对称质量体（史瓦西解）而言，光线会在靠近质量体时发生偏折。历史上日食观测（太阳边缘掠过）得到约 `1.75` 角秒，是广义相对论关键验证之一。

本条目采用赤道平面、无自旋中心体（非 Kerr）设置，聚焦最核心的“引力透镜一阶偏折”计算链条。

## R03

本 MVP 的计算任务：
1. 用无量纲变量 `w=b/r` 与 `epsilon=(GM/c^2)/b` 构建 ODE；
2. 由最近点条件解出 `w0`（`dw/dphi=0`）；
3. 从最近点积分到 `w=0`（对应 `r -> infinity`），得到 `phi_infinity`；
4. 计算数值偏折角 `alpha_numeric = 2*phi_infinity - pi`；
5. 计算弱场解析近似并评估相对误差；
6. 输出不变量漂移和方程残差作为数值质量指标。

## R04

建模假设（MVP 级别）：
- 背景时空为史瓦西度规（球对称、静态、无电荷、无自旋）；
- 仅考虑赤道平面中的光线传播；
- 忽略介质折射、等离子体效应和多体引力扰动；
- 采用散射轨道（`epsilon < 1/(3*sqrt(3))`），不处理光子捕获轨道。

这些假设适合“单透镜、弱场到中弱场”的教学与算法验证，不适用于精密天体测量全流程。

## R05

核心方程（`demo.py` 已实现）：

无量纲零测地线二阶方程：
`d2w/dphi2 + w = 3*epsilon*w^2`

其中：
- `w = b/r`
- `epsilon = (GM/c^2)/b`

最近点条件（由一阶积分常数关系给出）：
`w0^2 - 2*epsilon*w0^3 - 1 = 0`

偏折角定义：
- 数值：`alpha_numeric = 2*phi_infinity - pi`
- 弱场一阶：`alpha_weak = 4*epsilon = 4GM/(c^2 b)`

## R06

数值算法流程：
1. 给定质量 `M` 与冲量参数 `b`，计算 `epsilon`；
2. 用 `brentq` 解最近点方程得到 `w0`；
3. 设初值 `w(0)=w0, w'(0)=0`；
4. 用 `solve_ivp(method="DOP853")` 积分 ODE；
5. 设终止事件 `w=0`，找到 `phi_infinity`；
6. 计算 `alpha_numeric`、`alpha_weak` 与相对误差；
7. 采样轨迹，计算不变量误差与 ODE 残差 RMS。

## R07

复杂度分析（每个 `b` 采样点）：
- 设积分器实际步数为 `N_step`，后处理采样点数为 `N_sample`；
- 时间复杂度约 `O(N_step + N_sample)`；
- 空间复杂度约 `O(N_sample)`（存储采样轨迹与诊断量）。

整表复杂度再乘冲量参数个数 `K`，即 `O(K*(N_step + N_sample))`。

## R08

数值稳定性策略：
- ODE 积分器选 `DOP853`，并使用 `rtol=1e-10, atol=1e-12`；
- 加入事件终止 `w=0`，避免“积分过头”进入非物理区；
- 使用两类诊断双保险：
  - 不变量 `w'^2 + w^2 - 2*epsilon*w^3 = 1` 漂移；
  - 方程残差 `w'' - (3*epsilon*w^2 - w)` 的 RMS。

## R09

适用场景：
- 广义相对论课程中“光偏折”数值演示；
- 引力透镜基础模型的 sanity check；
- 弱场解析式与全方程积分结果对比。

不适用场景：
- 强场透镜成像（需高阶像、时延、放大率矩阵）；
- 自旋黑洞（Kerr）与多透镜系统；
- 精密观测拟合（需真实天体结构与系统误差模型）。

## R10

正确性检查点（脚本已实现阈值）：
1. `alpha > 0`（偏折角必须为正）；
2. `alpha` 随 `b` 增大单调减小；
3. `b >= 2R_sun` 时弱场相对误差 `< 0.2%`；
4. 最大不变量误差 `< 1e-8`；
5. 最大 ODE 残差 RMS `< 3e-5`。

全部通过才判定 `Validation: PASS`。

## R13

理论与验证边界：
- 本实现是“数值物理验证”，不是符号解析证明；
- 对弱场偏折可与一阶解析式严格对照，但高阶项仅通过数值体现；
- 若需毫角秒级甚至更高精度，必须引入更完整天体模型与观测误差传播。

## R14

常见失败模式与修复：
- 失败：`Ray did not reach infinity within phi_max`
  - 修复：增大 `phi_max` 或检查 `epsilon` 是否接近捕获区。
- 失败：最近点方程无法 bracket
  - 修复：确认 `epsilon < 1/(3*sqrt(3))`，并检查输入质量/冲量参数量级。
- 失败：不变量漂移偏大
  - 修复：收紧 `rtol/atol`，减小 `max_step`。
- 失败：弱场误差异常大
  - 修复：核对单位（`m, kg, s`）与角度换算（rad/arcsec）。

## R15

工程实践建议：
- 先做“无量纲化”，再做积分，显著提升数值可控性；
- 每次改 ODE 或事件函数后，先看不变量漂移再看业务指标；
- 输出表格中同时保留 `rad` 与 `arcsec` 量纲，避免沟通歧义。

## R18

`demo.py` 源码级算法流拆解（9 步，避免黑盒）：
1. `gravitational_radius_m` 把质量 `M` 转成几何长度 `GM/c^2`，统一到方程尺度。
2. `solve_w0` 先检查散射条件 `epsilon < 1/(3*sqrt(3))`，再解 `w0^2-2*epsilon*w0^3-1=0`。
3. `brentq` 在 `f(low)<0<f(high)` 区间内迭代收缩，直到得到最近点 `w0`。
4. `rhs` 显式写出系统 `y=[w,w']` 的导数 `[w', 3*epsilon*w^2-w]`，不调用任何 GR 黑盒。
5. `solve_ivp(DOP853)` 对每个候选步执行多 stage Runge-Kutta 估计，并用嵌入误差控制步长。
6. 事件函数 `w=0` 触发后终止积分，对应光线到达无穷远，得到 `phi_infinity`。
7. `alpha_numeric = 2*phi_infinity - pi`，并与 `alpha_weak = 4*epsilon` 对比得到误差。
8. 后处理采样轨迹，检查一阶不变量 `w'^2 + w^2 - 2*epsilon*w^3 = 1` 的最大漂移。
9. 用有限差分近似 `w''` 构造 ODE 残差 RMS，再结合单调性与误差阈值做 PASS/FAIL 判定。

说明：虽然使用了 `scipy` 数值库，但关键物理方程、边界条件、事件定义与验证指标都在源码中逐步展开，可逐行追踪。
