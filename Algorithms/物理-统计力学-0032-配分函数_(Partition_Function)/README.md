# 配分函数 (Partition Function)

## R01

配分函数（Partition Function）是正则系综中把“微观能级”映射到“宏观热力学量”的核心对象。对离散能级系统：

`Z(beta) = sum_i g_i * exp(-beta E_i),  beta = 1/(k_B T)`

其中 `E_i` 是第 `i` 个能级，`g_i` 是简并度。`Z` 本质上是玻尔兹曼权重的归一化常数；知道 `Z` 后可系统性地得到内能、自由能、熵、热容等量。

## R02

为什么它是“算法条目”而不仅是公式：

- 实务里常面对截断能谱、实验拟合能级、数值本征值，`Z` 需要数值稳定地计算；
- 大温区扫描时，直接 `exp(-beta E)` 易下溢或上溢；
- 许多后续量（`U,C,S`）都依赖 `Z` 的高精度，数值误差会被放大。

本目录 MVP 的目标就是实现一个可运行、可验证、可追踪的“从能级到热力学”的最小流程。

## R03

本实现选择 1D 量子简谐振子（QHO）作为演示模型：

- 能级：`E_n = hbar * omega * (n + 1/2)`；
- 简并度：`g_n = 1`；
- 数值方法：截断到 `n=0..n_max` 后做离散求和；
- 验证方式：与解析解逐温度点对照。

这样既保留配分函数算法的通用结构，也有可用作“真值”的解析结果。

## R04

MVP 用到的关键关系（约化单位 `k_B=1`）：

1. `log Z = logsumexp(log(g_i) - beta E_i)`
2. `p_i = g_i exp(-beta E_i) / Z`
3. `U = <E> = sum_i p_i E_i`
4. `F = -T log Z`
5. `S = (U - F)/T`
6. `C = Var(E)/T^2 = (<E^2>-<E>^2)/T^2`

解析对照（QHO）在 `demo.py` 中同时实现：

- `logZ = -x/2 - log(1-exp(-x))`, `x=hbar*omega/T`
- `U = hbar*omega*(1/2 + exp(-x)/(1-exp(-x)))`
- `C = x^2 exp(-x)/(1-exp(-x))^2`

## R05

复杂度（单温度点）：

- 构造权重与概率：`O(M)`，`M=n_max+1`；
- 计算 `U, Var(E)`：`O(M)`；
- 总体：`O(M)` 时间，`O(M)` 空间。

若扫描 `K` 个温度点，总复杂度 `O(KM)`。这也是配分函数方法在离散谱场景下易于工程化的原因。

## R06

`demo.py` 输出内容：

- 数值与解析的 `logZ/U/C` 对照表；
- 每个温度点的相对误差；
- 全温区最大相对误差汇总。

脚本末尾有断言，保证：

- `U(T)` 随温度单调上升；
- 数值解与解析解在预设阈值内一致。

## R10

正确性直觉：

1. 低温 `T -> 0` 时，权重集中在基态，`U` 逼近零点能；
2. 温度升高后，高能级权重增加，`U,S` 上升；
3. 热容 `C` 反映能量涨落，来自 `Var(E)`；
4. 若 `Z` 计算正确，则这些趋势会与解析公式一致。

`demo.py` 的断言和误差统计就是对以上 1-4 的最小可执行检验。

## R11

数值稳定策略：

- 用 `scipy.special.logsumexp` 计算 `logZ`，避免直接指数求和下溢；
- 解析式使用 `log1p` 和 `exp(-x)` 形式，减少极端温区损失精度；
- 相对误差分母设置下限 `1e-12`，避免接近零时误差比值爆炸。

## R12

关键参数（`PartitionConfig`）：

- `omega`：能级间距尺度，控制量子效应强弱；
- `hbar`：量子尺度因子；
- `n_max`：谱截断上限，过小会在高温造成系统性偏差；
- `temperatures`：温区采样网格。

调参经验：若高温点误差偏大，优先增大 `n_max`。

## R14

常见失效模式：

1. 忘记简并度 `g_i`，导致 `Z` 和 `U` 偏差；
2. 直接计算 `Z=sum(exp(-beta E_i))` 在低温下数值下溢；
3. 谱截断太短，高温尾部被截断，热容误差明显；
4. 公式混用单位（`k_B, hbar, omega`）导致量纲错误。

本实现用对数域求和和解析交叉校验规避上述问题。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 创建 `PartitionConfig`（`omega,hbar,n_max,temperatures`）并调用 `run_partition_function_demo`。  
2. `build_qho_spectrum` 生成离散能级数组 `E_n` 与简并度数组 `g_n`。  
3. 对每个温度 `T`，`thermodynamics_from_spectrum` 计算 `beta=1/T`，并构造 `log_weights = log(g)-beta*E`。  
4. 使用 `logsumexp(log_weights)` 得到 `logZ`，再恢复 `Z`；由 `exp(log_weights-logZ)` 得到规范化概率 `p_i`。  
5. 基于 `p_i` 计算 `U=<E>`、`Var(E)`、`C=Var(E)/T^2`，并计算 `F=-T logZ`、`S=(U-F)/T`。  
6. `analytic_qho_thermodynamics` 在同一温度上给出 QHO 解析 `logZ/U/C/S` 作为对照真值。  
7. `run_partition_function_demo` 汇总数值/解析结果到 `DataFrame`，逐列计算相对误差。  
8. `main` 打印结果表与最大误差，并用断言检查单调性和误差阈值，确保 MVP 可执行且物理趋势正确。

第三方库拆解说明：`numpy` 负责数组代数，`scipy.special.logsumexp` 仅用于稳定求 `logZ`，`pandas` 只做结果表组织与打印；配分函数主逻辑（权重、归一化、热力学量推导、校验）均在源码中显式实现。
