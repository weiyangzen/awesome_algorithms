## R01

引力红移描述的是：光子从较深引力势阱（小半径）传播到较浅引力势阱（大半径）时，被远处静止观测者测得频率降低、波长变长。  
在 Schwarzschild 度规下，静止观测者之间的频率比为：

`nu_obs / nu_emit = sqrt((1 - r_s/r_emit) / (1 - r_s/r_obs))`

其中 `r_s = 2GM/c^2` 是 Schwarzschild 半径，红移量定义为 `z = nu_emit/nu_obs - 1`。

## R02

该算法是广义相对论中最直接可计算、且有实验意义的预测之一：

- 可用于恒星表面谱线红移估计；
- 可用于理解地面与卫星钟频率差（GPS 频率校准背景之一）；
- 可作为从弱场近似过渡到强场相对论修正的教学示例。

本目录 MVP 的目标是把“精确公式 vs 弱场近似”的差异跑通并量化出来。

## R03

实现策略采用最小可运行流程：

- 输入一组天体参数与发射/观测半径；
- 用 Schwarzschild 静态解计算精确红移 `z_exact`；
- 用弱场近似 `z_weak ≈ GM/c^2 * (1/r_emit - 1/r_obs)` 对照；
- 输出 `pandas.DataFrame` 表格，给出误差与频率比；
- 通过断言验证核心物理趋势（越靠近天体表面，红移越大）。

## R04

核心公式：

1. Schwarzschild 半径：
   - `r_s = 2GM / c^2`
2. 精确红移（静止发射源与静止观测者）：
   - `z_exact = sqrt((1 - r_s/r_obs) / (1 - r_s/r_emit)) - 1`
3. 弱场近似（`|Phi| << c^2`）：
   - `z_weak ≈ (Phi_obs - Phi_emit)/c^2 = GM/c^2 * (1/r_emit - 1/r_obs)`
4. 频率比与波长比：
   - `nu_obs / nu_emit = 1/(1+z)`
   - `lambda_obs / lambda_emit = 1+z`

当 `r_obs -> inf` 时，上式自动退化到常见的“从天体表面到无穷远”的红移表达式。

## R09

## R10

正确性直觉：

1. 光子沿测地线传播时，静态时空的 Killing 能量守恒；
2. 局域静止观测者的固有时间尺度由 `sqrt(1-r_s/r)` 决定；
3. 深势阱（小 `r`）处同一坐标时间对应更少固有时间，导致局域频率标定更高；
4. 光子爬升到大 `r` 后被测得频率降低，因此 `z > 0`。

脚本用多组场景和半径剖面断言来最小化验证上述趋势。

## R11

数值稳定性与实现细节：

- 在计算前检查 `r_emit > r_s` 且 `r_obs > r_s`，避免视界内无效输入；
- 使用 `np.isinf` 处理 `r_obs = inf`，避免 `1/inf` 误判；
- 红移公式分步计算，减少表达式拼接错误；
- 对弱场误差使用相对误差定义并处理 `z=0` 边界。

## R12

关键参数说明：

- `mass_kg`：中心天体质量，直接决定 `r_s`；
- `r_emit_m`：发射位置半径；
- `r_obs_m`：观测位置半径，可设为 `inf`；
- `lambda_emit_nm`：发射谱线波长（用于输出直观波长变化比例）；
- `n_profile_points`：半径剖面采样点数，用于单调性检验。

参数影响：质量越大或发射半径越小，`z_exact` 越大，弱场近似误差通常也越大。

## R14

常见失效模式：

1. 把 `z` 和 `nu_obs/nu_emit` 混淆，符号写反；
2. 在强场区误用弱场近似并当作精确结果；
3. 输入半径小于 `r_s` 导致表达式进入非实数域；
4. 把 `r` 写成高度而非“距中心半径”，单位错配；
5. 忘记把 `r_obs = inf` 作为极限处理，造成数值分支错误。

本实现通过半径有效性检查和多场景断言降低这些风险。

## R18

`demo.py` 源码级算法流程（7 步）：

1. 在 `main` 中定义多个 `RedshiftScenario`（地球、太阳、白矮星、中子星），并设置发射谱线波长。  
2. `build_scenario_table` 逐场景调用 `schwarzschild_radius` 计算 `r_s`，先做半径有效性检查（必须在视界外）。  
3. 对每个场景调用 `redshift_exact` 计算 GR 精确红移 `z_exact`，再调用 `redshift_weak_field` 得到 `z_weak`。  
4. 同步计算 `nu_obs_over_nu_emit = 1/(1+z_exact)` 与 `lambda_obs_nm = lambda_emit_nm*(1+z_exact)`，并求近似相对误差。  
5. 将所有标量组织为 `pandas.DataFrame`，打印结构化结果表。  
6. `build_earth_profile` 构造不同发射半径的 Earth 剖面，验证 `r_emit` 增大时 `z_exact` 单调减小。  
7. `run_consistency_checks` 执行断言（`z>0`、`nu_obs/nu_emit<1`、弱场误差在强场显著增大）并输出通过信息。

第三方库说明：`numpy` 仅用于数组/数值计算，`pandas` 仅用于表格展示，`scipy.constants` 仅提供常数；引力红移核心公式与流程全部在源码显式实现，无黑盒求解器。
