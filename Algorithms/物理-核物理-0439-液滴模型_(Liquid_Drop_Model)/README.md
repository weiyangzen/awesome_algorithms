# 液滴模型 (Liquid Drop Model)

- UID: `PHYS-0419`
- 学科: `物理`
- 分类: `核物理`
- 源序号: `439`
- 目标目录: `Algorithms/物理-核物理-0439-液滴模型_(Liquid_Drop_Model)`

## R01

液滴模型把原子核视为“不可压缩、近似均匀带电液滴”，以少量宏观项刻画结合能与裂变趋势。该条目的 MVP 关注三件事：

1. 用液滴模型主方程计算结合能 `B(A,Z)`；
2. 用小样本拟合系数并做误差校验；
3. 用易裂参数（fissility）与形变势能判断核裂变稳定性。

## R02

模型核心思想是“竞争机制”：

- 核力带来体积束缚（稳定）；
- 表面效应减少束缚（不稳定）；
- 库仑排斥削弱束缚（不稳定）；
- 中子质子不平衡带来对称能惩罚；
- 奇偶配对修正局部稳定性。

在重核区，库仑项与表面项的竞争直接决定裂变势垒是否存在。

## R03

本目录 `demo.py` 的目标输出：

1. `LDM` 系数的 `SciPy` 拟合值与线性回归交叉拟合值；
2. 一组代表核素的预测结合能误差（`MAE/RMSE/R^2`）；
3. 易裂参数 `x`、液滴裂变势垒估计 `B_f`、球形曲率系数；
4. 形变能曲线最小值位置，对比 `x<1` 与 `x>1` 两种情形。

## R04

MVP 使用的主公式：

1. 结合能（半经验液滴型）

`B(A,Z)=a_v A-a_s A^(2/3)-a_c Z(Z-1)/A^(1/3)-a_a(A-2Z)^2/A+delta(A,Z)`

其中 `delta(A,Z)=a_p * pair_sign(A,Z)/sqrt(A)`，`pair_sign` 为偶偶 `+1`、奇奇 `-1`、奇 `A` 为 `0`。

2. 易裂参数（球形参考）

`x = E_c0 / (2E_s0)`，
`E_s0=a_s A^(2/3)`，`E_c0=a_c Z(Z-1)/A^(1/3)`。

3. 势垒经验估计（`x<1`）

`B_f ≈ 0.36 * a_s * A^(2/3) * (1-x)^2`，若 `x>=1` 则在该近似下视为无势垒。

4. 形变能曲线（Landau 型最小实现）

`DeltaE(alpha)=E_s0[(1-x)alpha^2+k4 alpha^4]`。

## R05

输入与输出定义：

- 输入：`(A, Z)`、系数 `a_v,a_s,a_c,a_a,a_p`、形变参数 `alpha`。
- 输出：
  - `B(A,Z)` 总结合能（MeV）；
  - `B/A`；
  - `x`、`B_f`、球形曲率；
  - 指定核素的 `DeltaE(alpha)` 曲线与极小点。

单位约定：能量均为 `MeV`，`A/Z` 无量纲。

## R06

复杂度（样本数 `N`，质量数扫描长度 `M`，形变网格 `K`）：

- 单核素结合能计算：`O(1)`；
- 样本预测与误差统计：`O(N)`；
- 小参数最小二乘拟合：近似 `O(N * iter)`；
- `A` 轴搜索最大 `B/A`：`O(M)`（每个 `A` 只在小窗口搜索 `Z`）；
- 单核形变曲线采样：`O(K)`。

当前默认规模下运行通常 < 1 秒。

## R09

适用场景：

- 核物理教学中的宏观模型讲解；
- 快速估计核素结合能趋势；
- 裂变稳定性的一级筛查（`x` 与 `B_f`）。

不适用场景：

- 需要高精度核质量表级别预测；
- 强壳效应、魔数附近精细结构研究；
- 需要完整多维形变势能面和动力学路径的任务。

## R10

正确性校验策略：

1. 数值拟合质量：`MAE`、`R^2` 达到合理阈值；
2. 双求解器一致性：`SciPy` 与线性回归系数差值受控；
3. 物理趋势检查：`B/A` 峰值落在铁峰附近（`A~56`）；
4. 裂变常识检查：`U-238` 的 `x` 在典型范围且势垒为正；
5. `x>1` 的假想重核应出现“无势垒 + 形变更低能”特征。

## R11

数值注意点：

- 系数存在相关性，小样本下不可辨识性会增大；
- `x` 与势垒公式是宏观近似，不能直接替代实验势垒；
- 使用 `float64` 以降低拟合和差分误差；
- 对奇偶项做显式离散处理，避免把配对项错误连续化。

## R12

关键参数：

- `a_v,a_s,a_c,a_a,a_p`：液滴模型主系数；
- `k4`：形变曲线四次稳定项（默认 `0.18`）；
- `alpha_max,num_points`：形变采样范围与分辨率；
- `A` 扫描区间：决定 `B/A` 峰值搜索范围。

调参建议：先用教科书系数验证流程，再切换到拟合系数做分析。

## R13

理论保证说明：

- 该条目不属于经典离散优化近似算法，故无“近似比”可给；
- 也非随机算法，故无“成功概率”指标；
- 提供的是“可解释的物理近似模型”，精度由模型假设与样本质量共同决定。

## R14

常见失效模式：

1. 错把 `Z^2` 与 `Z(Z-1)` 混用导致系统偏差；
2. 遗漏配对项符号导致奇偶趋势错误；
3. 只看拟合误差，不检查物理趋势（铁峰、裂变阈值）；
4. 将液滴模型外推到极端滴线区域并过度解读；
5. 把 `x>=1` 解释成“实验中必然瞬时裂变”（这是模型层面的势垒结论，不是完整动力学结论）。

## R18

`demo.py` 源码级算法流程（9 步）：

1. `build_reference_dataset` 构造核素样本表 `(A,Z,B_exp)`。  
2. `pairing_sign` 与 `design_matrix` 显式生成五项液滴特征（体积、表面、库仑、对称、配对）。  
3. `fit_params_scipy` 对 `Xc-B_exp` 做有界最小二乘，得到主拟合系数。  
4. `fit_params_linear` 用线性回归/最小二乘做独立拟合，形成交叉校验。  
5. `evaluate_dataset` 计算逐核素预测、误差列与 `MAE/RMSE/R^2`。  
6. `fissility_parameter`、`surface_curvature_at_sphere`、`fission_barrier_mev` 计算裂变稳定性指标。  
7. `deformation_energy_curve` 在 `alpha` 网格上计算 `DeltaE(alpha)`，识别球形核与超易裂核的极小点差异。  
8. `scan_binding_peak` 在质量数区间搜索最大 `B/A`，验证铁峰位置。  
9. `main` 打印系数表、误差表、裂变诊断表并执行断言；若断言全部通过则输出 `All checks passed.`。

说明：实现没有把第三方库当黑箱。物理项定义、特征构造、目标函数、裂变判据、断言条件都在源码中逐步展开并可追溯。
