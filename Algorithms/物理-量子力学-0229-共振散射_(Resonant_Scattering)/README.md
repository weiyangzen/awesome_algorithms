# 共振散射 (Resonant Scattering)

- UID: `PHYS-0228`
- 学科: `物理`
- 分类: `量子力学`
- 源序号: `229`
- 目标目录: `Algorithms/物理-量子力学-0229-共振散射_(Resonant_Scattering)`

## R01

共振散射描述的是粒子散射截面随能量变化时，在某个特征能量附近出现尖峰增强的现象。量子力学中常用 Breit-Wigner（洛伦兹）线型近似单一窄共振：

`σ(E) = σ_bg + A * (Γ/2)^2 / ((E - E0)^2 + (Γ/2)^2)`

其中 `E0` 是共振能量，`Γ` 是全宽半高（FWHM），`A` 是共振贡献振幅，`σ_bg` 是非共振背景。

本目录 MVP 的目标是：
- 生成可复现实验风格的共振散射数据；
- 用可审计的手写 LM（Levenberg-Marquardt）最小二乘拟合 `E0, Γ, A, σ_bg`；
- 输出参数估计、95% 区间、RMSE/R²、Q 因子等关键物理量。

## R02

待解决的计算问题：

给定一组测量点 `(E_i, σ_i)`，假设单峰共振模型成立，求解参数向量：

`theta = [E0, Γ, A, σ_bg]`

使得残差平方和最小：

`min_theta Σ_i (σ_i - σ_model(E_i; theta))^2`

并进一步给出：
- `Γ` 作为共振宽度（寿命相关量）；
- `Q = E0 / Γ` 作为品质因子；
- 参数不确定性（标准误差与 95% 置信区间）。

## R03

选择该问题作为 MVP 的原因：
- 共振散射是量子散射章节中的核心形状识别问题；
- 单峰 Breit-Wigner 模型参数少、解释清晰，适合最小实现；
- 用手写迭代求解可显式展示“模型-雅可比-更新-收敛”闭环，不依赖黑盒拟合。

## R04

模型与导数定义：

1. 记 `w = Γ/2`, `d = E - E0`, `q = d^2 + w^2`。
2. 线型函数：`g(E)=w^2/q`，截面 `σ = σ_bg + A*g`。
3. 目标函数：`S(theta) = Σ (σ_obs - σ_model)^2`。
4. LM 近似线性系统：
   `(J^T J + λ * diag(J^T J)) * Δ = J^T r`
   其中 `r = σ_obs - σ_model`。
5. 参数更新：`theta_new = theta + Δ`，若 `S` 下降则接受并减小阻尼 `λ`，否则增大 `λ`。

本实现用解析雅可比：
- `∂σ/∂E0 = A * 2*d*w^2 / q^2`
- `∂σ/∂Γ = A * w*d^2 / q^2`
- `∂σ/∂A = g`
- `∂σ/∂σ_bg = 1`

## R05

算法流程（MVP）：

1. 生成带噪声的单峰共振合成数据，保证可复现随机种子。
2. 对输入做合法性检查并按能量排序。
3. 根据峰值和半高宽构造初值 `theta0`。
4. 进入 LM 迭代：计算模型、残差、雅可比与增量 `Δ`。
5. 试探更新参数并根据 SSE 改善与否调节阻尼系数。
6. 触发步长收敛阈值或达到最大迭代后停止。
7. 基于最终雅可比近似协方差，输出标准误差和 95% 区间。
8. 计算 RMSE、R²、Q 因子并打印结果表。

## R06

正确性要点：
- 模型是标准 Breit-Wigner 单峰形式，参数具有明确物理含义；
- LM 在局部线性化下结合阻尼，兼具 Gauss-Newton 与梯度下降稳定性；
- 接受/拒绝更新用 SSE 单调性判定，避免发散步；
- 最终协方差 `cov ≈ s^2 (J^T J)^(-1)` 给出局部不确定性近似；
- `Γ > 0`、`A >= 0` 的投影约束保持参数物理可解释。

## R07

复杂度分析：

记：
- `n` 为采样点数；
- `p=4` 为参数维度；
- `T` 为迭代轮数。

每轮主要成本：
- 计算模型与雅可比 `O(n*p)`；
- 组装 `J^T J` 与 `J^T r` 约 `O(n*p^2)`；
- 求解 `p x p` 线性系统 `O(p^3)`（此处可视作常数级）。

总复杂度约 `O(T * n * p^2)`，在 `p=4` 下近似线性随 `T*n` 增长。空间复杂度 `O(n*p)`。

## R08

边界与异常处理：
- 输入必须为等长 1D 数组，且无 NaN/Inf；
- 至少需要 8 个不同能量点；
- 若数据未按能量递增，内部自动排序；
- 若 `σ` 没有动态范围（常数序列）直接报错；
- 每次更新后强制 `Γ >= min_gamma`，`A >= 0`；
- 线性系统病态时回退到最小二乘解（`lstsq`）。

## R09

MVP 取舍：
- 只实现单峰、常背景模型，不覆盖多重共振与干涉；
- 不做外部文件 I/O，直接内置合成数据以便一键验证；
- 使用 `numpy + scipy + pandas` 的轻量栈，不引入大型拟合框架；
- 以可读性优先，不做高度工程化封装。

## R10

`demo.py` 关键函数：
- `breit_wigner_cross_section`：计算共振截面模型。
- `breit_wigner_jacobian`：输出解析雅可比矩阵。
- `estimate_initial_guess`：由峰值/半高宽给初值。
- `fit_resonance_lm`：手写 LM 主循环，执行参数更新与阻尼调节。
- `validate_inputs`：输入合法性与排序。
- `make_synthetic_dataset`：生成测试数据与真值参数。
- `main`：串联流程并打印可读结果表。

## R12

输出字段说明：
- `converged`：是否达到步长收敛条件；
- `iterations`：实际迭代次数；
- `accepted_steps`：被接受的参数更新步数；
- `RMSE`、`R2`、`SSE`：拟合误差指标；
- `E0`、`Gamma(FWHM)`：共振中心和宽度；
- `Q=E0/Gamma`：品质因子；
- 参数表中的 `std_err`、`ci95_low/high`：局部线性近似不确定性。

## R13

最小测试覆盖（脚本自带）：
- 固定随机种子确保结果可复现；
- 在含噪声数据上拟合并回收真值附近参数；
- 打印前 10 行观测/拟合对比便于人工检查残差；
- 输出每个参数的估计值、误差和置信区间。

## R14

关键调参建议：
- `max_iter`：不收敛时增大；
- `damping_init`：初期震荡可适当调大；
- `damping_up/down`：决定阻尼调整快慢；
- `tol_step`：收敛阈值，过小会迭代更久；
- 初值质量很关键，可通过更平滑的预处理提升稳定性。

## R15

与相关方法对比：
- 对比黑盒 `curve_fit`：调用更短，但算法细节不可见；本实现可审计每一步。
- 对比直接数值解径向薛定谔方程：物理更完整，但实现成本更高。
- 对比 Fano 线型：Fano 能描述非对称共振；当前 Breit-Wigner 主要针对近似对称单峰。

## R16

典型应用场景：

- 原子/分子散射实验的线宽与寿命关联分析；
- 教学实验中演示“共振峰 -> 参数反演 -> 物理量解释”的完整流程。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `main` 调用 `make_synthetic_dataset` 生成能量网格、观测截面和真值参数。
2. `validate_inputs` 对输入做维度、有限性、长度和动态范围检查，并按能量排序。
3. `estimate_initial_guess` 从峰值位置、背景分位数和半高宽估计 `theta0=[E0, Γ, A, σ_bg]`。
4. `fit_resonance_lm` 每轮先用 `breit_wigner_cross_section` 计算模型值，再构造残差 `r=σ_obs-σ_model`。
5. 同轮调用 `breit_wigner_jacobian` 构造解析雅可比 `J`，形成 `(J^T J + λD)Δ = J^T r` 并解出增量 `Δ`。
6. 对试探参数执行物理约束（`Γ>0`, `A>=0`），若 `trial_sse` 下降则接受并减小 `λ`，否则拒绝并增大 `λ`。
7. 迭代结束后基于最终 `J` 计算协方差、标准误差与 95% 置信区间，同时计算 RMSE、R²、Q 因子。
8. `main` 用 `pandas` 输出参数汇总表和观测/拟合前 10 行预览，完成最小可验证闭环。
