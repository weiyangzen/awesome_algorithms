# 锁模 (Mode Locking)

- UID: `PHYS-0472`
- 学科: `物理`
- 分类: `激光物理`
- 源序号: `495`
- 目标目录: `Algorithms/物理-激光物理-0495-锁模_(Mode_Locking)`

## R01

锁模（Mode Locking）是把激光谐振腔内多个纵模的相位关系固定下来，使它们在时域发生相干叠加，从连续或准连续输出转变为周期性超短脉冲列的技术。

## R02

本条目聚焦一个最小可复现问题：
给定同一组纵模频率与幅度分布，比较“随机相位（未锁模）”和“相位一致（理想锁模）”两种情况下的时域强度，验证锁模会显著提高峰值功率并形成窄脉冲。

## R03

MVP 输入（代码内固定参数）:
- 腔长 `L`、折射率 `n`
- 纵模数量 `N_modes`
- 频率间隔 `Δf = c / (2nL)`
- 模幅度包络（高斯）
- 相位配置（随机 or 锁定）

MVP 输出:
- 两种工况的时域强度数组
- 峰值、平均值、峰均比（PAR）
- 估计脉宽（FWHM）
- 导出的 `mode_locking_timeseries.csv`

## R04

核心物理关系：

1. 纵模等间隔:
`f_m = f_0 + mΔf`

2. 场叠加（包络表示）:
`E(t) = Σ A_m exp(i(2π mΔf t + φ_m))`

3. 强度:
`I(t) = |E(t)|^2`

当 `φ_m` 彼此随机时，干涉项在时间上平均化，输出较平坦；当 `φ_m` 近似线性且固定（理想示例用同相），会在 `T_rep = 1/Δf` 周期上形成强脉冲。

## R05

算法思想是“同谱不同相位”的对照实验：
- 频谱（模频率和模幅）保持一致
- 仅改变相位统计特性
- 通过数值叠加把频域差异映射到时域脉冲结构差异

这样可以直接把“锁模本质是相位工程”这个概念量化出来。

## R06

实现流程：
1. 设定腔参数并计算 `Δf` 与 `T_rep`。
2. 构造离散模索引 `m` 与高斯幅度 `A_m`。
3. 在多个往返周期上建立时间采样网格 `t`。
4. 生成随机相位 `φ_random` 与锁模相位 `φ_locked`。
5. 分别计算复包络 `E_random(t)`、`E_locked(t)`。
6. 取强度并归一化到相同平均功率。
7. 估算 FWHM、PAR 等指标并打印对比。
8. 导出时序 CSV 便于后处理/绘图。

## R07

时间复杂度：`O(N_modes * N_time)`，因为每个时间点都要累加所有模。
空间复杂度：`O(N_modes * N_time)`（当前实现用矩阵相位一次性向量化计算），对于 MVP 规模（几十到几百模、几千时间点）开销很小。

## R08

数值注意点：
- 直接使用光学载频 `f0 ~ 10^14 Hz` 会导致采样压力巨大，MVP 使用包络模型（仅保留 `mΔf`）。
- FWHM 由离散采样估计，会受时间步长影响；可通过增大 `samples_per_period` 改善。
- 随机相位工况有统计波动，代码固定随机种子保证可复现。

## R09

MVP 范围（刻意简化）:
- 不包含增益饱和、色散、Kerr 非线性、SESAM 动力学
- 不做主动/被动锁模器件级建模
- 只做“多模相干叠加”的第一性演示

这满足 Stage0 的“最小但诚实”要求，并可作为后续复杂模型的基线。

## R10

`demo.py` 默认参数（可直接运行）:
- `L = 1.2 m`
- `n = 1.5`
- `N_modes = 81`
- 观察窗口 `6 * T_rep`
- 每个往返周期采样 `1200` 点

该配置下可稳定看到：锁模工况脉冲尖锐、峰均比较高；未锁模工况较噪声化/平坦。

## R11

运行方式（无交互输入）:

```bash
uv run python demo.py
```

运行后会在当前目录打印指标并写出 `mode_locking_timeseries.csv`。

## R12

结果解读要点：
- `PAR_locked >> PAR_random`：说明能量被压缩到窄时间窗。
- `FWHM_locked << FWHM_random`（若随机工况可定义）：说明脉宽缩短。
- 脉冲间隔接近 `T_rep`：符合腔模间隔决定重复频率的理论。

## R13

最小正确性检查：
- 两工况平均强度都应接近 1（归一化后）。
- 锁模工况应出现明显局部峰，且峰值显著高于随机相位工况。
- CSV 三列长度一致，时间单调递增。

## R14

边界与失效情形：
- 模数太少（例如 `< 5`）时，脉冲压缩效果有限。
- 时间采样过粗时，FWHM 估计会失真。
- 若幅度包络极窄（几乎单模），锁模与否差异会减弱。

## R15

可扩展方向：
- 引入二阶/三阶色散相位，观察啁啾与脉冲展宽。
- 引入非线性相位项（SPM）做简化 Haus master equation 近似。
- 加入主动调制（AM/FM）项，区分主动锁模与被动锁模。
- 用 FFT 在频域和时域间双向验证。

## R16

工程关联：
- 飞秒激光加工（高峰值、低热影响）
- 光频梳与精密测量
- 超快泵浦-探测实验
- 非线性光学过程（SHG、OPA、SCG）中的高峰值驱动

## R17

与真实系统差距：
- 真实锁模是“增益-损耗-色散-非线性”共同平衡，本 MVP 只保留相干叠加机制。
- 实验中还需考虑噪声启动、腔漂移、热效应与器件响应时间。

但该最小模型能正确回答“为什么锁模能把多模光变成短脉冲”。

## R18

`demo.py` 的源代码级算法流（8 步）：

1. `cavity_mode_spacing(L, n)` 计算 `Δf = c/(2nL)`，并由此得到 `T_rep = 1/Δf`。
2. `build_modes(num_modes, sigma_ratio)` 生成对称模索引 `m`，并构造高斯幅度向量 `A_m`，再归一化能量。
3. 构造时间网格 `t`，覆盖多个 `T_rep`，保证脉冲结构可见。
4. 用固定随机种子生成 `φ_random`，并设置 `φ_locked = 0`（理想同相锁模）。
5. `synthesize_envelope(...)` 通过向量化相位矩阵 `2π mΔf t + φ_m` 计算复包络 `E(t)`。
6. `intensity_from_field(...)` 取 `|E|^2` 并按平均值归一化，得到可比的强度曲线。
7. `estimate_fwhm(...)` 在峰值附近找半高交点，线性插值估计脉宽；`summarize_case(...)` 汇总 `peak/mean/PAR/FWHM`。
8. 主程序打印对照指标并把 `t, I_random, I_locked` 写入 CSV，形成可复查结果。
