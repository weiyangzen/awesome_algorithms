# 卢瑟福散射公式 (Rutherford Scattering Formula)

## R01

本条目实现经典力学中的卢瑟福散射微分截面公式，用于计算库仑中心力下带电粒子的角分布强度，并通过最小化残差拟合反推靶核有效电荷数。MVP 目标是：
- 给定入射粒子电荷数 `Z1`、靶核电荷数 `Z2`、入射动能 `E`、散射角 `θ`，计算 `dσ/dΩ`。
- 构造合成观测数据（含噪声），再拟合 `Z2`，验证公式可用于参数反演。

## R02

核心公式（SI 单位）：

\[
\frac{d\sigma}{d\Omega}
=
\left(
\frac{Z_1 Z_2 e^2}{16\pi \varepsilon_0 E}
\right)^2
\csc^4\left(\frac{\theta}{2}\right)
\]

其中 `dσ/dΩ` 单位为 `m^2/sr`。该式反映了小角散射强增强（`θ -> 0` 时发散）的经典特性。

## R03

符号与单位约定：
- `Z1, Z2`：无量纲电荷数。
- `e`：元电荷（C）。
- `ε0`：真空介电常数（F/m）。
- `E`：入射粒子动能（J），代码中由 `MeV -> J` 转换。
- `θ`：散射角（rad），展示时可转成度（deg）。
- `dσ/dΩ`：微分截面（`m^2/sr`）。

## R04

物理前提与适用范围：
- 只考虑静电库仑势，忽略强相互作用与屏蔽修正。
- 采用非相对论经典轨道近似（卢瑟福原始模型）。
- 靶核视作固定散射中心，不处理靶核反冲细节。
- 角度区间需避开 `θ=0` 附近的数学发散点。

## R05

本目录 MVP 的输入/输出：
- 输入（脚本内置示例）：`Z1=2`（α 粒子）、`Z2=79`（金核）、`E=5 MeV`、`θ∈[20°,160°]`。
- 中间量：无噪声理论 `dσ/dΩ` 与含噪“观测” `dσ/dΩ_obs`。
- 输出：拟合得到的 `Z2_fit`、协方差估计标准差、相对 RMSE、以及样例表格行。

## R06

算法流程（MVP）：
1. 用常量与单位换算函数将 `E(MeV)` 转为 `E(J)`。
2. 在角度网格上计算理论微分截面。
3. 叠加乘性高斯噪声生成可重复观测（固定随机种子）。
4. 用 `scipy.optimize.curve_fit` 拟合 `Z2`。
5. 计算拟合曲线与观测的相对误差指标并输出表格。

## R07

正确性依据：
- 公式实现与理论表达一一对应：`prefactor * sin(θ/2)^(-4)`。
- 量纲自洽：`e^2/(4π ε0)` 具有 `J·m` 量纲，结合 `E` 后平方得到面积量纲。
- 拟合变量仅 `Z2`，其余条件固定，便于验证“参数可辨识性”。
- 设置随机种子保证每次运行输出一致，可重复验证。

## R08

数值稳定性与工程处理：
- `θ` 不取 0，避免 `sin(θ/2)=0` 导致无限大。
- 噪声后对截面下界裁剪（`clip`）避免出现非物理负值。
- 拟合加入边界 `Z2∈[1,120]`，防止搜索空间无界。
- 使用适度角度范围（20°~160°）平衡动态范围与可拟合性。

## R09

复杂度分析（设角度采样点数为 `N`）：
- 单次公式计算：`O(N)`。
- 观测构造与误差计算：`O(N)`。
- 非线性拟合：`O(I·N)`，`I` 为迭代次数（通常远小于大规模仿真场景）。
- 空间复杂度：`O(N)`。

## R10

示例数据策略：
- 使用理论曲线作为“真值”。
- 注入 8% 相对噪声模拟实验测量误差。
- 该策略可在无外部数据文件的情况下完成端到端验证，保证目录自包含。

## R11

参数估计目标：
- 已知：`Z1`、`E`、角度采样 `θ_i`。
- 观测：`y_i = (dσ/dΩ)_obs(θ_i)`。
- 未知：`Z2`。
- 通过最小二乘拟合使 `Σ_i (f(θ_i; Z2)-y_i)^2` 最小，得到 `Z2_fit` 及协方差估计。

## R12

结果解释方式：
- `Z2_fit` 接近真实值 79，说明模型和实现一致性良好。
- `±std`（由协方差矩阵对角线开方获得）反映局部线性近似下的不确定度。
- `relative RMSE` 越小，说明拟合曲线与观测一致性越高。

## R13

局限与潜在失效场景：
- 极小角散射理论发散，真实实验通常受探测器开角和屏蔽效应约束。
- 高能区可能需要相对论修正，经典公式会偏离。
- 实验若存在系统误差（几何因子、背景扣除偏差），单参数模型会失真。
- 强核力主导区（超近距离）不适用纯库仑卢瑟福模型。

## R18

源码级算法流追踪（含第三方库拆解，不把 `curve_fit` 当黑箱）：
1. `main()` 先定义实验条件（`Z1, Z2_true, E`）并调用 `mev_to_joule()` 完成单位换算。
2. 通过 `np.linspace` 构造角度网格，转为弧度数组 `theta_rad`。
3. `rutherford_dsigma_domega()` 内部按公式拆成两部分：`prefactor` 与 `sin(θ/2)^(-4)`，最终返回向量化截面数组。
4. 用固定随机种子生成高斯噪声并构造 `dsigma_obs`，随后 `clip` 保证观测非负。
5. 定义包装函数 `fit_fn(ang, zt)`，把 `z_projectile` 与 `E` 固定，只暴露 `Z2` 作为拟合变量。
6. `curve_fit` 调用链本质上是最小化残差平方和：反复试探 `Z2`，每次都回调 `fit_fn -> rutherford_dsigma_domega` 计算模型值，再与 `dsigma_obs` 比较残差；在有边界时采用受约束最小二乘路径。
7. `curve_fit` 返回最优参数 `popt` 与参数协方差 `pcov`；代码取 `sqrt(diag(pcov))` 作为 `Z2` 的一阶不确定度估计。
8. 最后计算拟合相对 RMSE，组装 `pandas.DataFrame`，输出关键统计量与前几行样例，形成可验证闭环。
