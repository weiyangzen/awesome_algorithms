# 二次谐波产生 (Second Harmonic Generation)

- UID: `PHYS-0474`
- 学科: `物理`
- 分类: `非线性光学`
- 源序号: `497`
- 目标目录: `Algorithms/物理-非线性光学-0497-二次谐波产生_(Second_Harmonic_Generation)`

## R01

本题实现一个“可运行、可解释、可验证”的二次谐波产生（SHG）最小 MVP：

- 输入：基频场初值、相位失配 `Δk`、非线性耦合强度 `κ`、传播长度 `L`；
- 输出：传播过程中基频强度 `I_ω(z)`、二次谐波强度 `I_2ω(z)`、转换效率 `η(z)`；
- 目标：观察相位匹配条件对能量转换效率的影响。

## R02

SHG 是二阶非线性极化 `P^(2)` 驱动的频率转换过程：两个频率 `ω` 的光子耦合到一个 `2ω` 光子通道。工程上，核心不是“有没有 SHG”，而是“在给定晶体长度下效率有多高”，这直接由相位失配 `Δk = k_{2ω} - 2k_ω` 控制。

## R03

`demo.py` 使用复包络耦合波方程（慢变包络近似）：

- `dA_ω/dz = -i κ A_ω^* A_2ω exp(-iΔkz) - (α_ω/2)A_ω`
- `dA_2ω/dz = -i κ A_ω^2 exp(+iΔkz) - (α_2ω/2)A_2ω`

其中 `A_ω, A_2ω` 为复振幅，`α_ω, α_2ω` 为可选损耗系数（默认 0）。

## R04

变量与默认参数：

- `A_ω(0)=1+0j`：归一化基频输入；
- `A_2ω(0)=0`：初始无二次谐波种子；
- `κ=1.0`：归一化耦合强度；
- `L=10.0`：归一化传播长度；
- `Δk` 扫描：`[0.0, 1.5, 4.0, 8.0]`；
- 采样点数：`n_points=1000`。

## R05

MVP 物理假设：

- 1D 传播，不考虑横向衍射与空间走离；
- 单色/窄带近似，不建模群速度失配；
- 晶体参数沿 `z` 均匀；
- 通过调节 `Δk` 表征相位匹配好坏。

这些假设保证模型小而诚实，适合教学和快速数值实验。

## R06

数值方法：

- 将复数状态拆成 4 维实向量 `[Re(A_ω), Im(A_ω), Re(A_2ω), Im(A_2ω)]`；
- 使用 `scipy.integrate.solve_ivp(method="RK45")` 自适应步长积分；
- 设置 `rtol=1e-8, atol=1e-10`；
- 在均匀 `z` 网格上输出轨迹用于后处理。

## R07

`demo.py` 的函数结构：

- `SHGParams`：参数容器；
- `_pack_complex / _unpack_complex`：复数状态与实向量互转；
- `shg_rhs`：耦合方程右端；
- `simulate_shg`：单次仿真并返回轨迹；
- `summarize_case`：提取峰值效率、终值效率等指标；
- `build_trace_table`：输出人工检查用采样表；
- `main`：执行 `Δk` 扫描并保存 CSV。

## R08

算法流程（实现视角）：

1. 构造默认参数 `SHGParams`。
2. 对每个 `Δk` 调用 ODE 积分器得到 `A_ω(z), A_2ω(z)`。
3. 计算 `I_ω=|A_ω|^2`、`I_2ω=|A_2ω|^2`。
4. 计算 `η=I_2ω/(I_ω+I_2ω)` 与守恒漂移指标。
5. 汇总为 `results_summary.csv`。
6. 对相位匹配案例（`Δk=0`）抽样输出 `phase_matched_trace.csv`。

## R09

复杂度分析（每个 `Δk` 案例）：

- 状态维度固定为 4，右端函数代价 `O(1)`；
- 总成本约为 `O(N_eval)`，`N_eval` 为自适应积分器内部评估次数；
- 扫描 `m` 个 `Δk` 时总体为 `O(m * N_eval)`；
- 内存开销约 `O(n_points)`（保存轨迹）。

## R10

数值稳定与误差控制：

- 采用自适应 RK45 处理不同 `Δk` 下的振荡尺度；
- 较严格容差降低相位误差积累；
- 通过 `power_drift = max| (I_ω+I_2ω) - (I_ω+I_2ω)_0 |` 监控积分一致性；
- 对效率分母加 `1e-12` 防止除零。

## R12

输出指标解释：

- `peak_i2`：传播区间内 `I_2ω` 峰值；
- `z_at_peak`：达到峰值的传播位置；
- `final_i2`：出口端二次谐波强度；
- `final_conversion`：出口端转换效率；
- `power_drift`：总强度漂移（积分误差/模型不守恒程度指示）。

## R13

期望物理趋势：

- `Δk=0`（理想相位匹配）通常给出更高 `I_2ω`；
- `|Δk|` 增大时，相位滑移导致有效累积减弱，效率下降；
- 在无损、强耦合下可能出现能量往返交换（非单调增长）。

## R14

正确性与门禁建议：

1. 脚本必须无交互直接运行成功；
2. 所有 `Δk` 案例积分器 `success=True`；
3. `I_ω, I_2ω` 不得为负（由模方定义保证）；
4. `Δk=0` 的 `peak_i2` 应不低于大失配案例（如 `Δk=8`）；
5. `power_drift` 应明显小于主量级（默认通常远小于 1）。

## R15

常见失效模式：

- 容差过松：出现虚假的效率震荡或漂移放大；
- 传播长度过短：不同 `Δk` 案例差异不明显；
- `κ` 过大且容差未调：局部高频振荡难以稳定解析；
- 误把系统 Python 当运行环境：可能触发 `numpy/scipy` ABI 不匹配。

MVP 边界说明：

## R18

第三方求解器在本实现中的源码级算法链路（8 步）：

1. `main` 组装参数后调用 `simulate_shg`，构造初值向量和 `z` 采样网格。
2. `simulate_shg` 把 `shg_rhs` 作为回调传给 `solve_ivp(..., method="RK45")`。
3. `shg_rhs` 每次被调用时完成“实向量 -> 复振幅”解包，并计算相位因子 `exp(±iΔkz)`。
4. `shg_rhs` 计算两个复导数 `dA_ω/dz, dA_2ω/dz`，再打包回 4 维实导数返回给求解器。
5. SciPy 的 RK45（Dormand-Prince 5(4)）在内部做多阶段斜率评估（`k1...k7`）并给出 5 阶主解与 4 阶误差估计。
6. 求解器根据局部误差与 `rtol/atol` 比较，自适应调整下一步步长（接受或拒绝当前步）。
7. 当步进覆盖整个区间后，SciPy 按 `t_eval` 返回插值/采样后的状态序列 `sol.y` 与坐标 `sol.t`。
8. 本脚本将 `sol.y` 还原为 `A_ω(z), A_2ω(z)`，计算强度、效率、峰值位置与漂移指标并导出 CSV。

这 8 步展示了从“物理方程”到“数值步进”再到“可解释指标”的完整闭环。
