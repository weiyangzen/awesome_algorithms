## R01

有限元方法（Finite Element Method, FEM）是一类把连续偏微分方程离散到有限维函数空间的数值方法。其核心思路是：

1. 把计算域划分为许多小单元（mesh / elements）；
2. 在每个单元上选取低阶基函数（如线性形函数）；
3. 将 PDE 的强形式转为弱形式（积分方程）；
4. 通过组装单元贡献得到全局线性系统并求解。

本条目采用最小可验证场景：一维 Poisson 方程的线性有限元实现。

## R02

MVP 问题定义：在区间 `x∈[0,1]` 上求解

`-u''(x) = f(x),  u(0)=0, u(1)=0`。

我们选择制造解（manufactured solution）

`u_exact(x)=sin(pi x)`，

于是右端项为

`f(x)=pi^2 sin(pi x)`。

这样可直接量化数值误差并验证收敛阶。

## R03

弱形式：取测试函数 `v` 且满足齐次边界条件，

`∫_0^1 u'(x) v'(x) dx = ∫_0^1 f(x) v(x) dx`。

把近似解写成基函数展开 `u_h = Σ U_i φ_i`，得到离散线性系统

`K U = F`，其中

`K_ij = ∫_0^1 φ_i'(x) φ_j'(x) dx`,

`F_i = ∫_0^1 f(x) φ_i(x) dx`。

## R04

离散选择：

- 网格：均匀划分为 `N` 个单元，节点数 `N+1`；
- 单元类型：2 节点线性单元；
- 形函数：参考区间 `[-1,1]` 上 `N1=(1-ξ)/2`, `N2=(1+ξ)/2`；
- 数值积分：2 点 Gauss 积分计算单元载荷向量。

对线性单元，单元刚度矩阵有解析形式。

## R05

对单元 `e=[x_i,x_{i+1}]`，长度 `h=x_{i+1}-x_i`：

- 单元刚度矩阵

`K_e = (1/h) * [[1,-1],[-1,1]]`。

- 单元载荷向量

`F_e[a] = ∫_{x_i}^{x_{i+1}} f(x) φ_a(x) dx`，

在 `demo.py` 中通过 2 点 Gauss 显式计算：

`F_e[a] ≈ Σ_q w_q f(x_q) N_a(ξ_q) J`，`J=h/2`。

## R06

全局组装流程：

1. 遍历每个单元，计算 `K_e, F_e`；
2. 按单元自由度映射把局部量累加到全局 `K, F`；
3. 对 Dirichlet 边界自由度进行处理（本题边界值为 0）；
4. 仅提取内部自由度子系统 `K_ii U_i = F_i` 求解；
5. 回填边界值，构造完整数值解向量 `U`。

脚本中 `scipy.sparse` 仅用于稀疏矩阵存储与线性求解，装配逻辑由源码逐行实现。

## R07

复杂度（1D 均匀网格，`N` 个单元）：

- 装配：每单元常数规模运算，时间 `O(N)`；
- 稀疏求解：1D 三对角结构理论上近 `O(N)`（通用稀疏解法器也通常近线性）；
- 空间：稀疏矩阵与向量存储 `O(N)`。

因此该 MVP 能在普通 CPU 上快速完成多层网格收敛实验。

## R08

数值性质（本模型）：

- 线性元对足够光滑解满足：
  - `H1` 半范数误差通常 `O(h)`；
  - `L2` 误差通常 `O(h^2)`；
- 选 `sin(pi x)` 作为解析解可稳定观察到上述趋势；
- 2 点 Gauss 对当前平滑右端项与线性基函数组合足够准确。

## R09

`demo.py` 的验证策略分两层：

1. **主算例**：`N=80` 时输出误差、残差和边界值检查；
2. **收敛性实验**：`N={10,20,40,80}`，统计误差随网格细化变化并估计观测阶。

如果实现正确，`L2` 误差应随 `h` 大约按二阶衰减。

## R10

MVP 技术栈：

- `numpy`：网格、局部矩阵、误差计算；
- `scipy.sparse` + `scipy.sparse.linalg.spsolve`：稀疏系统求解；
- `pandas`：输出收敛表与摘要表。

未调用专门 FEM 黑盒框架（如 FEniCS、deal.II Python 封装），核心算法步骤均在源码中可追踪。

## R12

输出指标说明：

- `l2_error`：积分意义下的 `L2` 误差 `||u_h-u||_{L2}`（3 点 Gauss 计算）；
- `h1_semi_error`：基于单元导数的 `H1` 半范数误差近似；
- `residual_inf`：内部自由度线性系统残差无穷范数；
- `observed_order_l2`：相邻两层网格的 `L2` 观测收敛阶；
- `max_abs_error`：节点最大绝对误差。

这些指标能同时覆盖“方程解对不对”和“收敛趋势对不对”。

## R13

内置验收阈值（`demo.py` 中 assert）：

1. `residual_inf < 1e-10`（线性系统解精度）；
2. `l2_error < 2e-4`（`N=80` 主算例精度）；
3. `observed_order_l2 > 1.8`（接近线性元理论二阶）。

全部满足后输出 `Validation: PASS`。

## R17

常见失效模式与排查：

1. **边界条件处理错误**：表现为边界值漂移或残差异常；
2. **局部到全局映射错位**：会导致矩阵非对称/奇异；
3. **载荷积分实现错误**：解形状可能“看起来平滑”但误差阶异常；
4. **网格太粗**：主误差偏大但残差仍可能很小；
5. **仅看残差不看解析对照**：可能掩盖建模或组装错误。

本 MVP 同时报告残差、绝对误差、收敛阶，避免单指标误判。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `FEMConfig` / `FEMResult` 定义参数与结果容器，`validate()` 保证网格层级合法。  
2. `generate_uniform_mesh` 构建节点坐标与两节点单元连接表。  
3. `local_stiffness` 与 `local_load_vector` 分别计算单元刚度矩阵和 2 点 Gauss 单元载荷。  
4. `assemble_system` 遍历单元，把局部 `K_e/F_e` 显式累加到全局稀疏矩阵 `K` 和向量 `F`，并处理 Dirichlet 右端修正。  
5. `solve_poisson_1d_fem` 提取内部自由度子系统 `K_ii U_i = F_i`，用 `spsolve` 求解，再回填边界自由度得到全局解。  
6. 在同一函数内计算三类质量指标：节点 `L2` 误差、`H1` 半范数误差、内部残差 `inf` 范数。  
7. `run_convergence_study` 在 `N={10,20,40,80}` 上重复求解，构建 `pandas` 表并估计 `L2` 观测收敛阶。  
8. `main` 打印收敛表和摘要，执行阈值断言（残差、误差、收敛阶）并输出 `Validation: PASS`。  

说明：第三方库只承担基础数值线代功能；有限元核心步骤（弱式离散、单元积分、组装、边界处理、误差评估）均在源码中显式实现，不是黑盒调用。
