# 达朗贝尔原理 (D'Alembert's Principle)

- UID: `PHYS-0109`
- 学科: `物理`
- 分类: `分析力学`
- 源序号: `109`
- 目标目录: `Algorithms/物理-分析力学-0109-达朗贝尔原理_(D'Alembert's_Principle)`

## R01

达朗贝尔原理把动力学问题改写为“虚功平衡”问题：对一切满足约束的虚位移 `δr`，有

`(F - m a) · δr = 0`

其中 `F` 是真实受力，`-m a` 是惯性力。该写法的核心价值是：理想约束力在可行虚位移上不做功，因此可以在不显式消元全部约束反力分量的前提下，直接得到广义坐标方程。

## R02

本目录 MVP 选取的具体问题：
- 系统：长度 `l`、质量 `m` 的平面单摆，受重力和外加切向激励力 `F_t(t)`。
- 约束：摆球只能在圆周上运动，`|r| = l`。
- 目标：
  1. 用达朗贝尔原理推导并数值积分摆角动力学；
  2. 在整条轨迹上验证虚功分解 `W_g + W_app + W_c + W_i ≈ 0`；
  3. 验证理想约束虚功 `W_c ≈ 0`。

## R03

符号与坐标约定：
- 广义坐标：`θ`，`θ=0` 表示竖直向下。
- 位置：`r(θ) = [l sinθ, -l cosθ]`。
- 切向单位向量：`t̂ = [cosθ, sinθ]`。
- 向内径向单位向量：`n̂ = [-sinθ, cosθ]`。
- 速度与加速度：
  - `v = l θ̇ t̂`
  - `a = l θ¨ t̂ + l θ̇² n̂`
- 外力：`F_g = [0, -m g]`，`F_app = F_t(t) t̂`，约束力 `F_c = T n̂`。

## R04

达朗贝尔原理在本问题中的写法：

`(F_g + F_app + F_c - m a) · δr = 0`

可行虚位移满足 `δr = (∂r/∂θ) δθ = l t̂ δθ`，因此
- `F_c · δr = T n̂ · (l t̂ δθ) = 0`（理想约束不做虚功）；
- 仅保留切向平衡：`(F_g + F_app - m a) · l t̂ = 0`。

## R05

代入切向分量得到动力学方程：

`m l θ¨ = F_t(t) - m g sinθ`

即

`θ¨ = F_t(t)/(m l) - (g/l) sinθ`

同时，径向方向可用于恢复约束反力：

`T = m l θ̇² + m g cosθ`

`demo.py` 同时计算 `T`，用于验证 `W_c = F_c · δr` 的数值零性。

## R06

MVP 数值算法：
1. 定义激励 `F_t(t) = A cos(ω t)`。
2. 定义状态 `y=[θ, θ̇]` 与右端函数 `f(t,y)=[θ̇, θ¨]`。
3. 用 `scipy.integrate.solve_ivp(method="RK45")` 在 `[t0, t1]` 上积分轨迹。
4. 对每个采样点计算 `F_g, F_app, F_c, a, δr`。
5. 计算虚功分量 `W_g, W_app, W_c, W_i` 与总和 `W_total`。
6. 统计 `max|W_total|`、`rms|W_total|`、`max|W_c|` 作为一致性指标。

## R07

正确性依据：
- 公式层面：`θ¨` 方程直接来自达朗贝尔原理在切向方向的投影。
- 几何层面：`n̂ ⟂ t̂`，保证理想约束力不做虚功。
- 数值层面：若积分精度足够，轨迹上应满足
  - `W_total` 接近 0；
  - `W_constraint` 接近 0；
  - 广义残差 `(F_g+F_app-m a)·(∂r/∂θ)` 接近 0。

## R08

复杂度分析：
- 设采样点数为 `N`。
- ODE 积分主成本与步数线性相关，记为 `O(N)`（RK45 每步常数次函数评估）。
- 虚功后处理对每点 `O(1)`，总计 `O(N)`。
- 额外存储数组与轨迹，空间复杂度 `O(N)`。

## R09

数值稳定性与工程处理：
- `solve_ivp` 使用较严格 `rtol=1e-9, atol=1e-11`，降低残差污染。
- `delta_theta` 取较小固定值（`1e-5`）构造虚位移，兼顾量纲可读性与浮点稳定。
- 报告同时给出 `max` 与 `rms` 指标，避免单点尖峰掩盖整体质量。

## R10

`demo.py` 的模块划分：
- `PendulumParams`：集中管理物理参数。
- `basis_vectors`：几何基底 `t̂, n̂`。
- `dynamics`：达朗贝尔方程右端。
- `acceleration_vector` / `tension_force`：恢复笛卡尔加速度与约束力。
- `virtual_work_terms`：单时刻虚功分解与广义残差。
- `run_demo`：仿真、统计、打印结果。

## R11

最小依赖栈：
- `numpy`：向量与统计计算。
- `scipy.integrate.solve_ivp`：常微分方程积分。

未引入额外框架，保持实现短小且可审计。

## R13

输出字段说明：
- `max |W_total|`：轨迹上达朗贝尔总虚功绝对值最大值，应很小。
- `rms |W_total|`：总虚功均方根，反映整体误差水平。
- `max |W_constraint|`：约束力虚功最大值，应接近机器误差量级。
- `max |generalized residual|`：切向广义残差最大值。
- `pass(...)`：对预设阈值的布尔检查结果。
- `Probe samples`：起点、中点、终点三个时刻的局部观测。

## R14

自检清单：
- 修改 `force_amp=0` 后，模型应退化为无驱动单摆，依然满足 `W_total≈0`。
- 适度缩小积分精度（如 `rtol=1e-6`）时，残差应变大，验证指标敏感性。
- 改变初始角速度符号后，`W_constraint` 仍应接近 0（与运动方向无关）。

## R15

边界与局限：
- 当前为单自由度、理想完整约束系统。
- 未建模空气阻力、枢轴摩擦、杆质量分布等非理想因素。
- 约束被假设严格满足；若约束柔化（弹簧杆等），需改写模型。

## R16

可扩展方向：
- 多摆/多体系统：推广到 `Σ_i (F_i - m_i a_i)·δr_i = 0`。
- 与拉格朗日乘子法联立：同时求广义运动与约束反力。
- 引入非完整约束或速度约束，转向更一般的分析力学框架。

## R17

与相邻方法的关系：
- 牛顿法：直接对每个方向列 `F=ma`，常需显式处理约束反力。
- 达朗贝尔法：把惯性项移到“力”侧并做虚功投影，天然忽略理想约束在可行位移上的功。
- 拉格朗日方程：可看作达朗贝尔原理在广义坐标下的系统化表达。

## R18

本实现的源码级算法流（9 步）：
1. `run_demo` 构造 `PendulumParams`、时间网格和初始状态 `y0=[θ0, ω0]`。
2. `solve_ivp` 调用 `dynamics(t,y,params)` 作为 RHS，`dynamics` 按 `θ¨ = F_t/(ml) - (g/l)sinθ` 返回 `[ω, α]`。
3. `solve_ivp` 的 `RK45` 在每个自适应步内部执行多次 RHS 评估，组合出 4/5 阶估计并控制局部截断误差。
4. 步进完成后，解被插值/采样到 `t_eval`，形成离散轨迹 `θ(t_k), ω(t_k)`。
5. 对每个 `t_k`，再次调用 `dynamics` 得到 `α_k`，并在 `basis_vectors` 中构造 `t̂_k, n̂_k`。
6. `virtual_work_terms` 用 `δr = l t̂_k δθ` 计算 `W_g, W_app, W_c, W_i`，其中 `W_c` 来自 `T n̂_k · δr`。
7. 同函数内还计算广义残差 `(F_g+F_app-m a)·(∂r/∂θ)`，用于直接对应达朗贝尔投影方程。
8. `run_demo` 聚合所有采样点统计量：`max/rms |W_total|`、`max |W_constraint|`、`max |generalized residual|`。
9. 打印阈值判定与 3 个探针时刻，形成可复现、可核验的最小验证闭环。
