## R01

多重网格法（Multigrid, MG）是求解椭圆型偏微分方程离散线性系统的高效算法。核心思想是：
- 细网格上平滑器快速消除高频误差；
- 粗网格上低频误差变成相对高频并被高效修正；
- 在多层网格间循环（V-cycle）实现近似线性复杂度收敛。

本条目实现一个最小可运行 MVP：
- 问题：1D Poisson 方程 `-u'' = f`，齐次 Dirichlet 边界；
- 离散：线性有限元（FEM）显式组装刚度矩阵和载荷；
- 求解：几何多重网格 V-cycle；
- 验证：残差收敛、相对 `L2` 误差、与直接解对比。

## R02

MVP 的连续模型：

- 方程：`-u''(x) = pi^2 sin(pi x), x in (0,1)`
- 边界：`u(0)=u(1)=0`
- 解析解：`u*(x)=sin(pi x)`

选择制造解的目的：
- 可以定量验证离散误差是否合理；
- 便于区分“线性系统未收敛”与“离散本身误差”。

## R03

有限元离散方式（1D 线性单元）：

1. 将 `[0,1]` 划分为 `N` 个等长单元，步长 `h=1/N`；
2. 每个单元使用线性基函数 `N1,N2`；
3. 局部刚度矩阵：`k_e = (1/h) * [[1,-1],[-1,1]]`；
4. 局部载荷向量：`b_e[i] = integral f(x) N_i(x) dx`；
5. 使用 2 点高斯积分计算载荷项；
6. 装配到全局系统后施加 Dirichlet 边界，仅保留内部自由度。

## R04

离散后得到线性系统：

`A u = b`

其中：
- `A` 对称正定（三对角稀疏结构，本实现用 `numpy` 致密数组表达）；
- `u` 为内部节点未知量；
- `b` 是体力项与基函数积分后的载荷。

对于 `N=128` 个单元：
- 内部未知量个数 `n=127`；
- 这是多重网格的 finest level。

## R05

多重网格在本题中的误差校正逻辑：

- 预平滑：快速衰减高频误差；
- 计算细网格残差 `r = b - A u`；
- 将 `r` 限制到粗网格，求解误差方程；
- 将粗网格误差延拓回细网格并修正解；
- 后平滑：消除插值带来的高频噪声。

这种“平滑 + 粗网格校正”的组合是 MG 高效率的关键。

## R06

网格层级构造（几何多重网格）：

- 细网格单元数设为 `2^k`，便于逐层二分；
- 每降一层单元数减半；
- 每层都重新按 FEM 规则组装该层 `A_l`；
- 当最粗层未知量很小（本实现 `n<=3`）时用直接法精确解。

示例层级：`127 -> 63 -> 31 -> 15 -> 7 -> 3`（内部自由度）。

## R07

跨层算子采用显式公式实现：

1. 限制（Full Weighting）：
`r_c[j] = 0.5*r_f[2j-1] + 1.0*r_f[2j] + 0.5*r_f[2j+1]`
2. 延拓（Linear Interpolation）：
- 粗网格节点注入到对应细网格偶点；
- 细网格奇点取相邻粗网格值平均。

说明：上式等价于 `P^T`（也是 `2 x` 经典 full-weighting），与本 FEM 残差定义匹配。代码中没有调用外部黑盒 MG 包，算子构造完全可审查。

## R08

平滑器选择：加权 Jacobi。

- 迭代式：`x <- x + omega * D^{-1}(b-Ax)`；
- 默认 `omega=2/3`，`nu_pre=2`，`nu_post=2`；
- 原因：实现简单、向量化友好，足够展示 MG 机制。

对于 1D Poisson，这个设置通常可获得稳定残差下降。

## R09

V-cycle 递归框架：

1. 若到最粗层，直接解 `A_c e_c = r_c`；
2. 否则执行：预平滑 -> 限制残差 -> 递归求粗网格误差 -> 延拓修正 -> 后平滑。

外层循环重复调用 V-cycle，直到：
- 达到最大循环次数，或
- 残差低于阈值（默认 `1e-10`）。

## R10

复杂度（针对该 MVP）：

- 单次 V-cycle 的工作量近似 `O(n)`（每层都是稀疏近邻操作，层级几何级数收敛）；
- 内存使用约 `O(n)`（按层存储矩阵与向量，本实现教学化使用小规模致密矩阵）；
- 对比直接解 `O(n^3)`（致密口径），MG 在更大规模时优势明显。

## R11

`demo.py` 主要函数职责：

- `assemble_fem_system`：组装 1D 线性 FEM 系统；
- `build_hierarchy`：构建从细到粗的层级矩阵；
- `restrict_full_weighting`：全加权限制；
- `prolong_linear`：线性插值延拓；
- `weighted_jacobi`：平滑迭代；
- `v_cycle`：递归 V-cycle；
- `run_multigrid`：多次 V-cycle 并记录残差历史；
- `main`：执行实验并输出收敛与误差指标。

## R13

输出中关键指标解释：

- `Residual history`: 每次 V-cycle 后 `||b-Au||_2`；
- `factor`: 相邻两次残差比，越小表示收敛越快；
- `MG relative L2 error`: 与解析解的离散相对 `L2` 误差；
- `||u_mg - u_direct||_inf`: MG 与直接解的一致性检查。

若实现正确，残差应单调或总体下降，且 MG 解接近直接解。

## R15

常见错误与排查建议：

1. 层级尺寸不满足 `n_f = 2*n_c + 1`，导致限制/延拓越界；
2. 只看误差不看残差，可能掩盖线性系统未收敛问题；
3. 边界处理不一致（包含边界自由度但未消元）；
4. 平滑参数不当（`omega` 过大可能导致震荡）；
5. 粗网格方程求解不准确，会明显拖慢整体收敛。

## R16

与相关方法比较：

- 与单层迭代（Jacobi/GS/CG）相比：
  MG 通过粗网格纠正低频误差，通常显著减少迭代数。
- 与 AMG 相比：
  本实现是几何多重网格（GMG），依赖网格层级结构；AMG 更通用但实现复杂。
- 与纯直接法相比：
  直接法实现简单但扩展性差；MG 更适合大规模稀疏椭圆问题。

## R18

`demo.py` 源码级算法流程（8 步，非黑盒）：

1. `main` 设定 finest 网格（默认 `N=128`），调用 `build_hierarchy` 逐层组装 FEM 矩阵，得到多层 `A_l`。  
2. `assemble_fem_system` 在每个单元上用线性基函数和 2 点高斯积分计算 `k_e,b_e`，再装配成全局 `A,b` 并剔除 Dirichlet 边界自由度。  
3. `run_multigrid` 以零初值开始，循环执行 V-cycle，并记录每轮残差 `||b-Au||_2` 形成收敛历史。  
4. `v_cycle` 先在当前层执行 `weighted_jacobi` 预平滑，去除高频误差成分。  
5. 计算残差 `r=b-Au`，通过 `restrict_full_weighting` 用 `0.5-1.0-0.5`（`P^T`）模板映射到粗网格。  
6. 在粗层递归求解误差方程；到最粗层时用 `numpy.linalg.solve` 直接求解。  
7. 将粗网格误差经 `prolong_linear` 线性插值回细网格并修正解，然后再做后平滑。  
8. 结束后计算解析解相对 `L2` 误差、最大节点误差及与直接解差异，验证 FEM+MG 实现的正确性与收敛性。
