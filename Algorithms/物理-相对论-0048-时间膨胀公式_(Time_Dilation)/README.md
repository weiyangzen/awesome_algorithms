## R01

时间膨胀公式来自狭义相对论，描述不同参考系对同一物理过程计时差异：

- `Δt = γ Δτ`
- `γ = 1 / sqrt(1 - v^2/c^2) = 1 / sqrt(1 - β^2)`，其中 `β = v/c`

含义：

- `Δτ`：固有时（事件在同一惯性系同地点发生时的时间间隔）；
- `Δt`：在相对运动观察者坐标系里测到的时间间隔；
- `γ >= 1`，因此 `Δt >= Δτ`。

## R02

该公式是狭义相对论中最常用的定量关系之一，直接连接：

- 高能粒子寿命延长（如大气 μ 子）；
- 粒子束流飞行时间估算；
- GPS 等系统中的相对论时间修正（工程系统通常还叠加广义相对论项）。

它是“速度接近光速时，牛顿绝对时间失效”的最直观量化版本。

## R04

核心方程：

1. 洛伦兹因子  
   `γ = 1 / sqrt(1 - β^2)`，`β = v/c`
2. 时间膨胀  
   `Δt = γΔτ`
3. 反解速度  
   `β = sqrt(1 - 1/γ^2)`
4. 一维共线运动下的间隔不变量  
   `c^2Δt^2 - Δx^2 = c^2Δτ^2`，且 `Δx = vΔt`
5. 衰变存活率（泊松衰变模型）  
   `P = exp(-t/τ)`；相对论情形取 `t = Δτ_muon = Δt_lab/γ`

## R10

正确性直觉：

1. `β` 越接近 1，`1-β^2` 越小，`γ` 越大，时间膨胀越显著。  
2. 当 `β=0` 时，`γ=1`，退化回经典同一时间尺度。  
3. 若 `Δx=vΔt` 代入 `c^2Δt^2-Δx^2`，可恢复 `c^2Δτ^2`，说明固有时定义一致。  
4. μ 子案例中，实验室系飞行时间固定，但粒子自身固有时被 `γ` 缩小，因此衰变更慢、存活率更高。

## R11

数值稳定与精度策略：

- 全程 `float64`；
- 限制 `0 <= β < 1`，避免 `sqrt(负数)`；
- 对 `γ < 1` 的反解输入直接抛错；
- 间隔不变量采用相对误差断言（`<1e-12`）；
- 对核心已知点做显式断言（如 `β=0.8 -> γ=5/3`）。

## R14

常见失效模式：

1. 把 `β` 当成 `v`（单位错）；
2. 输入 `β>=1` 导致数学奇异；
3. 将 `Δt` 与 `Δτ` 语义混淆，导致重复乘/除 `γ`；
4. 用经典寿命直接解释高能粒子存活率，低估地面通量；
5. 在 `β≈1` 极限做低精度计算，误差被放大。

本实现通过输入校验和断言覆盖上述高频错误。

## R18

`demo.py` 源码级算法流程（7 步）：

1. `run_assertions` 先验证已知解析点：`γ(0)=1`、`γ(0.8)=5/3`、`β(γ(β))` 可逆。  
2. `lorentz_gamma` 对输入 `β` 做区间检查后，按 `1/sqrt(1-β^2)` 向量化计算。  
3. `time_dilation` 直接执行 `Δt=γΔτ`，并由 `make_time_dilation_table` 组装为 `pandas.DataFrame`。  
4. `verify_interval_invariance` 由 `dt=γτ` 和 `dx=vdt` 构造两侧表达式，比较 `c^2dt^2-dx^2` 与 `c^2dτ^2` 相对误差并断言。  
5. `muon_survival_report` 先算实验室系飞行时间 `t_lab = L/v`。  
6. 同函数内分别计算经典存活率 `exp(-t_lab/τ0)` 与相对论存活率 `exp(-(t_lab/γ)/τ0)`，得到存活增益因子。  
7. `main` 按顺序打印三段报告，确保一次无交互运行即可完成公式计算与物理一致性校验。  

说明：`scipy` 仅用于常数 `c`，核心算法步骤全部在源码显式展开，没有将时间膨胀逻辑委托给第三方黑盒函数。
