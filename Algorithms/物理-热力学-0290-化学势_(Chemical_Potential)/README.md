## R01

化学势（Chemical Potential）是多组分热力学系统中“增加一个组分粒子时自由能变化率”的状态量。对组分 `i`，定义为：

`μ_i = (∂G/∂n_i)_{T,P,n_{j≠i}}`

其中 `G` 为吉布斯自由能，`n_i` 为组分摩尔数。对于理想混合模型，常写成：

`μ_i = μ_i*(T,P) + RT ln x_i`

其中 `x_i` 为摩尔分数，`μ_i*(T,P)` 为标准态项。该条目使用可运行 MVP 同时验证“导数定义”和“相平衡判据 `μ_i^α = μ_i^β`”。

## R02

工程和科研价值：

1. 化学势是扩散、渗透、相平衡、化学反应平衡的统一驱动力表达；
2. 在模型实现里，它把“热力学理论”转成可计算的一阶导数量；
3. 对状态方程/自由能函数的正确性，可通过化学势一致性快速做数值体检。

本 MVP 目标是“小而透明”：不用黑盒热力学包，直接从 `G(n_A,n_B)` 出发做差分与平衡求解。

## R04

MVP 采用的数学模型：

1. 标准态项（示意模型）：
`μ_i*(T,P) = μ_ref_i + RT ln(P/P0)`。

2. 二元理想混合总 Gibbs 自由能：
`G = n_A μ_A* + n_B μ_B* + RT(n_A ln x_A + n_B ln x_B)`。

3. 解析化学势：
`μ_A = μ_A* + RT ln x_A`，`μ_B = μ_B* + RT ln x_B`。

4. 数值定义验证：
`μ_A ≈ [G(n_A+h,n_B)-G(n_A-h,n_B)]/(2h)`，
`μ_B ≈ [G(n_A,n_B+h)-G(n_A,n_B-h)]/(2h)`。

5. 相平衡条件（组分 A）：
求解 `μ_A^α(n_A^α) - μ_A^β(n_A^α) = 0`，并满足 `n_A^β = n_A,total - n_A^α`。

## R05

## R09

## R10

正确性直觉：

1. 若 `G(n_A,n_B)` 写对，则其偏导就是化学势定义本身；
2. 理想混合下 `μ_i = μ_i* + RT ln x_i` 有解析表达，可作为“真值”；
3. 中心差分对光滑函数是二阶精度，`h` 足够小时误差应很小；
4. 两相平衡本质是“同组分在两相中的边际自由能相等”，即 `μ^α=μ^β`；
5. 因此“误差小 + μ_gap≈0”是该 MVP 正确性的直接证据。

## R11

稳定性处理：

- 采样区间保证 `n_A,n_B > 0`，避免 `ln(0)`；
- 中心差分优于前向差分，截断误差更低；
- 使用 `float64` 统一计算；
- 二分法要求端点异号，保证求根鲁棒收敛；
- 通过断言限制误差上界，防止“静默失败”。

## R12

关键参数：

- `fd_step`：差分步长，影响截断误差和舍入误差平衡；
- `tolerance`：`μ` 验证的误差阈值；
- `grid_points`：采样密度；
- `root_tol`、`max_iter`：二分法收敛控制；
- `mu_shift_alpha/beta`：两相偏置项，决定平衡分配位置。

调参建议：

- 差分误差偏大：先试 `fd_step` 在 `1e-5 ~ 1e-7` 间扫描；
- 想更严：增大 `grid_points` 并降低 `tolerance`；
- 二分法若无根：检查端点异号及参数是否物理可行。

## R13

理论保证类型：

- 近似比保证：N/A（非组合优化问题）；

## R14

常见失效模式：

1. 组分摩尔数接近 0 导致对数奇异；
2. 差分步长设置不当（过大截断误差，过小舍入噪声）；
3. 把固定条件写错（例如导数时误改了另一个组分）；
4. 相平衡求根区间未覆盖根，二分法无法收敛；
5. 单位混用（Pa/bar, J/kJ）导致数量级异常。

## R18

`demo.py` 源码级算法流程（8 步）：

1. 读取 `ChemicalPotentialConfig` 与 `PartitionConfig`，固定热力学参数、差分步长、阈值和求根精度。  
2. 定义 `standard_chemical_potential`，计算 `μ_i*(T,P)=μ_ref_i+RT ln(P/P0)`。  
3. 定义 `ideal_mixture_gibbs`，构造二元理想混合的总 Gibbs 自由能 `G(n_A,n_B)`。  
4. 写出解析化学势 `analytic_mu_a/analytic_mu_b`，作为数值校验基准。  
5. 用 `finite_difference_mu_a/finite_difference_mu_b` 对 `G` 做中心差分，得到数值化学势。  
6. `build_verification_table` 遍历 `n_A,n_B` 网格，记录解析值、数值值和绝对误差，并生成 DataFrame。  
7. `solve_partition_equilibrium` 构造两相 `μ_gap` 函数，调用 `bisection_root` 求 `μ_A^α=μ_A^β` 的平衡分配点。  
8. `main` 打印汇总、执行断言（误差阈值与 `mu_gap`），全部通过后输出 `All checks passed.`。  

补充说明：本实现未把第三方库当黑盒直接“返回化学势结论”，核心链路是显式的“自由能建模 -> 数值导数 -> 误差验证 -> 平衡求根”。
