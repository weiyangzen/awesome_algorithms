# 欧拉方程（刚体） (Euler's Equations for Rigid Bodies)

- UID: `PHYS-0105`
- 学科: `物理`
- 分类: `经典力学`
- 源序号: `105`
- 目标目录: `Algorithms/物理-经典力学-0105-欧拉方程（刚体）_(Euler's_Equations_for_Rigid_Bodies)`

## R01

刚体欧拉方程描述的是“在刚体自身主轴坐标系中，角速度如何随时间演化”。
当刚体绕固定点运动时，若惯量主轴已知，可把三维转动动力学写成 3 个一阶常微分方程，比直接处理旋转矩阵/姿态参数更紧凑，是经典力学、航天姿态动力学与机器人学中的核心模型。

## R02

核心思想是将角动量定律写到随体坐标系（body frame）中：

`dL/dt + ω × L = τ`

在主惯量轴下，`L = Iω` 且惯量张量对角化，得到分量形式：

- `I1 * dω1/dt = (I2 - I3) * ω2 * ω3 + τ1`
- `I2 * dω2/dt = (I3 - I1) * ω3 * ω1 + τ2`
- `I3 * dω3/dt = (I1 - I2) * ω1 * ω2 + τ3`

## R03

本题使用“无外力矩”最小场景（`τ=0`）验证算法闭环。

状态变量：`x = [ω1, ω2, ω3]^T`。

参数：主惯量 `I=(I1,I2,I3)`，且示例取 `I1<I2<I3` 来展示中间主轴不稳定现象。

方程组：

`dω/dt = f(ω; I)`，其中每个分量都由另外两个角速度分量的乘积耦合得到，是典型非线性 ODE 系统。

## R04

推导要点（简版）：

1. 从角动量定律 `dL/dt_inertial = τ` 出发。  
2. 用随体导数关系：`dL/dt_inertial = dL/dt_body + ω×L`。  
3. 代入 `L = Iω`。在主轴系中 `I` 为常量对角矩阵，因此 `dL/dt_body = I dω/dt`。  
4. 将叉乘 `ω×(Iω)` 写为分量形式，移项即可得到上面的三条欧拉方程。  

这一步把“几何向量方程”转成“可数值积分的状态空间方程”。

## R05

`demo.py` 的实验设置：

- 惯量：`I=(1.0, 2.0, 3.0)`
- 初始角速度：`ω0=(0.02, 2.0, 0.02)`（接近中间主轴）
- 外力矩：`τ(t)=0`
- 积分区间：`t ∈ [0,40]`
- 数值方法：`scipy.integrate.solve_ivp(method="DOP853")`

输出同时给出：

- `ω1, ω2, ω3` 轨迹
- 动能 `E = 1/2 * Σ Ii*ωi^2`
- 角动量模长 `|L| = ||(I1ω1, I2ω2, I3ω3)||`
- 守恒量相对漂移与 `ω2` 符号切换次数

## R06

MVP 代码结构（`demo.py`）：

- `SimulationConfig`：统一维护惯量、初值、积分精度
- `euler_rhs()`：实现欧拉方程右端
- `simulate()`：调用 `solve_ivp` 并构造 `pandas.DataFrame`
- `summarize_diagnostics()`：计算能量/角动量漂移与运动统计
- `main()`：执行仿真并打印头尾轨迹

该结构强调“最小可复现”，避免引入大型框架。

## R07

伪代码：

```text
input: I1,I2,I3, omega0, [t0,tf]
def rhs(omega):
  w1,w2,w3 = omega
  dw1 = ((I2-I3)*w2*w3 + tau1)/I1
  dw2 = ((I3-I1)*w3*w1 + tau2)/I2
  dw3 = ((I1-I2)*w1*w2 + tau3)/I3
  return [dw1,dw2,dw3]

omega(t) = solve_ivp(rhs, omega0)
for each time step:
  E = 0.5 * sum(Ii*wi^2)
  L = norm([I1*w1, I2*w2, I3*w3])
report conservation drift and trajectory table
```

## R08

复杂度分析（设输出采样点数为 `N`）：

- 每次 RHS 评估是常数开销 `O(1)`（仅若干乘加）
- 总开销由积分器调用 RHS 次数决定，近似与 `N` 同阶：`O(N)`
- 存储轨迹（`t,ω,E,|L|`）需要 `O(N)` 内存

因此该 MVP 在 CPU 上非常轻量，适合快速验证动力学模型。

## R09

数值稳定性处理：

- 选用高阶自适应方法 `DOP853`
- 设置较严格容差：`rtol=1e-9, atol=1e-11`
- 使用主轴坐标系输入惯量，避免非对角惯量引入额外数值误差源
- 通过守恒量（`E`、`|L|`）漂移反查积分精度是否足够

若漂移过大，优先缩小容差或缩短步长区间。

## R10

与其他转动建模方式对比：

- 欧拉方程（本题）：直接建模角速度动力学，状态少，适合动力学求解
- 姿态参数法（四元数/旋转矩阵）：更适合同时跟踪空间姿态，但需额外运动学方程
- 拉格朗日法：适合广义坐标建模复杂约束系统，但在纯刚体主轴问题上不如欧拉方程直接

本任务目标是验证“刚体角速度演化”，所以优先选欧拉方程最小实现。

## R11

关键参数建议：

- 惯量排序：使用 `I1<I2<I3` 便于观察中间主轴不稳定
- 初值：若希望明显翻转，给中间轴分量较大、其余分量小扰动
- 积分精度：默认 `rtol=1e-9`；若只看趋势可放宽到 `1e-6`
- 采样点：`num_points=1000~5000` 对可视检查通常足够

## R12

实现说明（`demo.py`）：

- `euler_rhs()` 把公式逐分量实现，保持可读与可审计
- `simulate()` 使用 `solve_ivp` 返回 `ω(t)`，并在同一处计算 `E(t)` 与 `|L|(t)`
- `summarize_diagnostics()` 计算：
  - `max_rel_energy_drift`
  - `max_rel_L_drift`
  - `w2_sign_changes`
- `main()` 打印配置、守恒性指标、轨迹头尾，保证无交互可运行

## R14

结果解读建议：

- `max_rel_energy_drift`、`max_rel_L_drift` 越小越好（通常应远小于 `1e-6`）
- `w2_sign_changes > 0` 说明中间轴分量发生翻转，符合“网球拍定理”现象
- `trajectory_head/tail` 可快速查看 `ω` 是否保持有界且演化平滑

守恒量稳定而角速度分量出现周期性交换，是该系统的典型特征。

## R15

常见问题排查：

- 报 `ODE integration failed`：检查惯量是否为正、初值是否包含 `NaN`
- 守恒量漂移偏大：减小 `rtol/atol`，或缩短 `t_end`
- 没观察到翻转：增大中间轴初值或延长仿真时间
- 角速度爆炸：通常是参数单位/量级不一致导致

## R16

可扩展方向：

- 加入时变外力矩 `τ(t,ω)`（推进器、阻尼、控制力矩）
- 联立姿态运动学（四元数）得到完整 6D 姿态动力学仿真
- 用辛积分器对比长期守恒性
- 叠加参数辨识或控制器（如 LQR / MPC）形成闭环实验

## R17

边界与适用性：

- 本 MVP 假设刚体、主惯量常量、无结构变形
- 未建模空气阻力、关节摩擦、柔性振动等非理想因素
- 结果主要用于算法验证与教学，不直接替代高保真工程仿真
- 若用于航天/飞控等高风险场景，需要更严格验证与硬件在环测试

## R18

`demo.py` 的源码级流程拆解（8 步）：

1. `main()` 创建 `SimulationConfig`，固定惯量、初始角速度、时间区间和积分容差。  
2. `main()` 调用 `simulate(cfg)`，进入数值积分主流程。  
3. `simulate()` 先构造 `t_eval`（均匀采样时刻），再把 `euler_rhs` 通过 `lambda` 传给 `solve_ivp`。  
4. `solve_ivp` 在内部做自适应步长推进；每个内部步都会回调 `euler_rhs(t, omega, inertia, torque_fn)`。  
5. `euler_rhs()` 按欧拉方程分量计算 `dω/dt`：使用 `(I2-I3)ω2ω3` 等耦合项和 `τ` 项，返回三维导数向量。  
6. `solve_ivp` 完成后，`simulate()` 从 `solution.y` 取出 `ω(t)`，并逐时刻计算 `energy` 与 `L_norm`，组装成 `DataFrame`。  
7. `main()` 调用 `summarize_diagnostics()`，对 `energy/L_norm` 相对初值求最大漂移，并统计 `ω2` 符号变化次数。  
8. `main()` 打印诊断指标与轨迹头尾，实现“公式 -> 积分 -> 守恒验证 -> 结果输出”的最小闭环。
