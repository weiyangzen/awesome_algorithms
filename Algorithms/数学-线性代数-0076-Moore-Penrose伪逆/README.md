# Moore-Penrose伪逆

## R01

Moore-Penrose 伪逆（记作 `A^+`）是普通逆在任意矩阵（含非方阵、奇异矩阵）上的推广。

对任意 `A in R^{m x n}`，`A^+` 是唯一满足以下四个 Penrose 条件的矩阵：

1. `A A^+ A = A`
2. `A^+ A A^+ = A^+`
3. `(A A^+)^T = A A^+`
4. `(A^+ A)^T = A^+ A`

当 `A` 可逆且方阵时，`A^+ = A^{-1}`。

## R02

历史定位（简要）：

- E. H. Moore 在 1920 年代提出广义逆思想；
- R. Penrose 在 1955 年给出上述四条件并证明唯一性；
- 现代数值线性代数中，伪逆通常通过 SVD 稳定计算，是最小二乘、降维、信号处理、机器学习中的基础算子。

## R03

本条目 MVP 解决的问题：

- 输入：
  - 一个实矩阵 `A`（本 MVP 用确定性 5x3 秩亏矩阵示例）；
  - 奇异值阈值参数 `rcond`（可选，默认使用 `eps * max(m,n)`）；
- 输出：
  - 由 SVD 显式构造出的 `A^+`；
  - 有效秩估计；
  - 四个 Penrose 条件残差；
  - 与 `numpy.linalg.pinv` 的差异；
  - 最小二乘相关诊断（`||A^T(b-Ax)||`、与 `lstsq` 解差异）。

## R04

时间复杂度（SVD 主导）：

- 对 `m x n` 矩阵，完整数值代价近似为 `O(min(m n^2, m^2 n))`；
- 阈值处理与 `A^+ = V Sigma^+ U^T` 回乘约 `O(mn*min(m,n))`；
- 因而总复杂度仍以 SVD 为主。

## R05

空间复杂度：

- 存储 `A`：`O(mn)`；
- 存储 `U, s, V^T`：`O(mn + min(m,n))` 量级；
- 存储 `A^+`：`O(nm)`。

总体额外内存通常为 `O(mn)` 级别。

## R06

直觉示例（列相关矩阵）：

令

`A = [[1,2,3],[2,4,6],[-1,1,0],[0,1,1],[3,6,9]]`。

这里第 3 列等于前两列之和，因此 `rank(A)=2<3`，普通逆不存在。
伪逆通过“只反转非零奇异值”获得稳定解：

- 对足够大的奇异值取倒数；
- 对接近 0 的奇异值置 0；
- 从而避免对病态方向做无穷放大。

## R07

算法意义：

- 统一处理可逆、不可逆、长方矩阵三类情形；
- 给出最小二乘问题的最小范数解：`x* = A^+ b`；
- 是岭回归、主成分分析、线性模型诊断与欠定系统求解的核心工具。

## R08

核心公式（SVD 路线）：

1. 做奇异值分解 `A = U Sigma V^T`；
2. 设阈值 `tau = rcond * sigma_max`；
3. 构造 `Sigma^+`：
   - 若 `sigma_i > tau`，令 `sigma_i^+ = 1/sigma_i`；
   - 否则令 `sigma_i^+ = 0`；
4. 得到 `A^+ = V Sigma^+ U^T`。

该构造天然满足 Penrose 四条件，并且对数值噪声更稳健。

## R09

适用条件与局限：

适用：

- 任何实/复矩阵；
- 需要稳健最小二乘解或最小范数解；
- 存在秩亏、近奇异、欠定/超定系统。

局限：

- SVD 代价较高，不适合超大规模稠密矩阵的高频在线计算；
- 阈值 `rcond` 选择会影响有效秩判断；
- 极端病态问题仍可能产生较大误差放大，需要正则化策略配合。

## R10

正确性要点（工程版）：

1. `A = U Sigma V^T` 的左右奇异向量提供正交基；
2. 在非零奇异值子空间上取倒数，在零奇异值子空间上置零，得到 `Sigma^+`；
3. 由正交矩阵性质可推得 `A A^+ A = A` 与 `A^+ A A^+ = A^+`；
4. `A A^+` 与 `A^+ A` 都是正交投影矩阵，因此对称，满足 Penrose 3/4；
5. 四条件联合给出唯一性，因此该 `A^+` 即 Moore-Penrose 伪逆。

## R11

数值稳定性：

- 直接基于正规方程 `(A^T A)^{-1}A^T` 在病态场景下会放大条件数；
- SVD 通过奇异值阈值截断小尺度方向，稳定性更好；
- 本 MVP 使用 `rcond = eps * max(m,n)` 默认阈值，并输出 cutoff 与有效秩，便于审计。

## R12

性能视角：

- 对中小规模问题，SVD 方案实现简洁且可靠；
- 若只需近似伪逆或在线更新，可考虑截断 SVD、随机 SVD、迭代法；
- 本条目优先选择“可验证正确性”的小型实现，而非复杂优化框架。

## R13

本目录 MVP 的可验证保证：

- 固定矩阵输入，可复现运行；
- 自动检查四个 Penrose 残差；
- 自动比较 `A^+` 与 `numpy.linalg.pinv` 的 Frobenius 误差；
- 验证最小二乘正交性：`||A^T(b-Ax)||` 足够小；
- 验证与 `numpy.linalg.lstsq` 解一致；
- 全部通过后打印 `All checks passed.`。

## R14

常见失效模式：

- 输入不是二维矩阵；
- 矩阵含 `NaN/Inf`；
- 阈值 `rcond` 设为负数；
- 过小阈值导致噪声奇异值被误反转，结果抖动；
- 过大阈值导致有效秩被低估，解偏差增大。

- `moore_penrose_pinv_svd`：显式执行 SVD、阈值反转、重组 `A^+`；
- `run_checks`：集中断言 Penrose 条件与最小二乘性质；
- `main`：执行、打印报告并给出可读输出。

## R16

相关算法链路：

- 正规方程法：`x = (A^T A)^{-1}A^T b`（仅满列秩稳定时建议）；
- Tikhonov/Ridge：通过正则项抑制病态方向；
- 截断 SVD（TSVD）：仅保留前 `k` 个奇异值；
- 随机 SVD：大规模近似低秩场景常用；
- 迭代 Krylov 法（LSQR/CGNR）：超大稀疏问题常见替代。

## R18

`demo.py` 的源码级算法流程（8 步，非黑盒）如下：

1. `main` 调用 `build_rank_deficient_matrix` 构造固定 5x3 秩亏矩阵。  
2. `moore_penrose_pinv_svd` 先做输入校验，再执行 `np.linalg.svd(A)` 得到 `U,s,V^T`。  
3. 用 `rcond * max(s)` 计算截断阈值 `cutoff`，决定哪些奇异值可反转。  
4. 显式构造 `s_inv`：`s_i > cutoff` 时取 `1/s_i`，否则置 0，得到离散版 `Sigma^+`。  
5. 按公式 `A^+ = (V^T)^T * s_inv @ U^T` 重组伪逆矩阵。  
6. 逐项计算 Penrose 四个残差：`AA^+A-A`、`A^+AA^+-A^+`、两投影对称性误差。  
7. 用 `b` 计算 `x=A^+b`，并检查 `||A^T(b-Ax)||` 与 `lstsq` 解差异，验证最小二乘/最小范数性质。  
8. `run_checks` 统一断言阈值，全部通过后打印矩阵形状、奇异值、残差与 `A^+`。
