## R01

有限体积法（Finite Volume Method, FVM）是一类基于守恒律的离散方法。它先对控制体积积分，再把体积分量转为控制体表面的数值通量平衡。相比“直接离散微分算子”的方式，FVM 在质量、动量、能量等守恒型问题里更自然，工程 CFD 与传热计算中使用广泛。

## R02

本目录 MVP 求解一维稳态扩散边值问题：

- 方程：`-d/dx(k du/dx) = q(x), x in [0,1]`
- 系数：`k=1`
- 边界：`u(0)=0, u(1)=0`
- 验证真解：`u*(x)=sin(pi x)`
- 由真解反推源项：`q(x)=pi^2 sin(pi x)`

我们在单元中心上离散并比较数值解与解析解。

## R03

控制体积分思想：

对第 `i` 个体积 `V_i=[x_{i-1/2}, x_{i+1/2}]` 积分：

`∫_{V_i} -d/dx(k du/dx) dx = ∫_{V_i} q(x) dx`

得到离散守恒式：

`F_{i+1/2} - F_{i-1/2} = Q_i`

其中：
- `F = -k du/dx` 为扩散通量
- `Q_i ≈ q(x_i) * dx` 为体源积分

这正是 FVM 的核心：面通量差等于体源。

## R04

离散通量近似（均匀网格、单元中心）采用二点差分：

- 内部面：`F_{i+1/2} ≈ -k (u_{i+1}-u_i)/dx`
- 左边界面：`F_{1/2} ≈ -k (u_0-u_L)/(dx/2)`
- 右边界面：`F_{N+1/2} ≈ -k (u_R-u_{N-1})/(dx/2)`

边界面用半个网格距离处理 Dirichlet 条件，这也是单元中心 FVM 的常见做法。

## R05

由上式可组装线性系统 `A u = b`：

- 内部单元系数：
  - `a_W = k/dx`
  - `a_E = k/dx`
  - `a_P = a_W + a_E`
- 边界单元额外出现 `2k/dx` 的边界导热系数，并并入 `a_P` 与 `b`

MVP 直接手工填充稠密矩阵，强调算法透明性而非大规模性能。

## R06

`demo.py` 中实现的关键函数：

- `source_term` / `exact_solution`：定义验证问题
- `build_fvm_system`：按控制体通量平衡组装 `A,b`
- `compute_face_fluxes`：计算各面的离散通量
- `cell_balance_residuals`：计算每个控制体的守恒残差
- `run_case`：执行一次网格求解并汇总误差指标
- `main`：做多网格收敛演示并输出误差阶

## R07

正确性检查分两层：

- 解析误差：`||u_h-u*||_2 / ||u*||_2`
- 守恒误差：每个控制体 `F_e - F_w - Q_i` 的最大绝对值

前者验证逼近精度，后者验证“守恒离散”是否实现正确。

## R08

复杂度：

- 当前实现手工组装稠密 `N x N` 矩阵并用 `numpy.linalg.solve`
- 时间复杂度约 `O(N^3)`，空间 `O(N^2)`
- 这对教学级小规模 MVP 足够；工程场景通常改为稀疏矩阵与迭代法，复杂度可显著下降

## R09

数值性质：

- 该扩散问题在均匀网格 + 中心近似下通常具备二阶空间精度
- 随网格加密（`N` 增大）相对误差应下降
- 若边界半网格处理写错，收敛阶会明显退化，守恒残差也会变差

## R10

默认演示参数：

- 网格数：`N in [10, 20, 40, 80, 160]`
- 系数：`k=1`
- 边界：`u_L=u_R=0`
- 输出：每个 `N` 的相对 `L2` 误差、最大守恒残差、以及相邻网格误差阶

无需任何交互输入。

## R11

MVP 的取舍：

- 仅依赖 `numpy`，保持最小依赖栈
- 不使用黑盒 PDE 库，保留完整 FVM 组装过程
- 使用已知真解问题，确保结果可直接量化评估

## R12

## R14

预期输出现象：

- `relative_l2_error` 随 `N` 增大持续下降
- `max_balance_residual` 接近机器精度量级（线性方程被精确解后）
- `observed_order` 接近 2（有限网格下会有轻微波动）

## R15

常见实现错误：

- 把守恒式写成 `F_w - F_e = Q`（符号反）
- 边界面距离误用 `dx` 而非 `dx/2`
- 体源项忘记乘控制体长度 `dx`
- 求解后未做守恒残差核对，导致“看似收敛但不守恒”

## R16

可扩展方向：

- 扩展到对流-扩散方程并引入迎风格式
- 使用非均匀网格和空间变系数 `k(x)`
- 从 1D 推广到 2D/3D 非结构网格 FVM
- 用 `scipy.sparse` + Krylov 方法替代稠密直接解

## R17

与有限差分法（FDM）对比：

- 在规则网格和简单扩散方程上，两者离散形式可非常接近
- FVM 的优势在于“先积分后离散”，守恒含义更直接
- 对复杂几何和通量边界条件，FVM 往往更容易保持物理一致性

## R18

`demo.py` 的源码级算法流程（8 步，非黑盒）：

1. `main` 定义一组网格数 `N=[10,20,40,80,160]`，逐个调用 `run_case`。  
2. `run_case` 在单元中心生成网格点 `x_i=(i+0.5)dx`，并用 `build_fvm_system` 组装 `A,b`。  
3. `build_fvm_system` 对每个控制体写入通量平衡方程：内部面用 `k/dx`，边界面用 `2k/dx` 并把 Dirichlet 值并入右端。  
4. 调用 `numpy.linalg.solve(A,b)` 得到单元中心数值解 `u_h`。  
5. `run_case` 用 `exact_solution(x)` 计算解析真解并得到相对 `L2` 误差。  
6. `cell_balance_residuals` 调用 `compute_face_fluxes` 计算每个面通量，再逐单元计算 `F_e-F_w-Q_i`。  
7. 统计该网格下 `max_balance_residual` 与误差并返回结果。  
8. `main` 汇总所有网格结果，计算相邻网格的观测收敛阶 `log2(e_h/e_{h/2})` 并打印最终报告。  

这 8 步完整覆盖了“控制体积分 -> 面通量离散 -> 线性系统 -> 守恒校验 -> 收敛评估”的 FVM 最小闭环。
