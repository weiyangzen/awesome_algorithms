# Richardson外推

## R01

Richardson外推是一类“误差消元”技术：当某个数值近似 \(A(h)\) 的截断误差可写成幂级数（如 \(A(h)=A^\*+c_1h^p+c_2h^{p+s}+\cdots\)）时，通过组合不同步长的结果来消掉主导误差项，显著提升精度。

## R02

典型目标问题：
- 数值求导：由差分公式得到 \(f'(x)\) 的更高精度近似。
- 数值积分：由梯形/辛普森序列进一步提升收敛阶（如 Romberg 积分本质上就是 Richardson 外推表）。
- 一般离散化问题：只要已知误差主阶 \(p\)，都可套用。

## R03

核心公式（步长缩放比 \(r>1\)，常用 \(r=2\)）：

\[
A(h)=A^\*+c_1h^p+c_2h^{p+s}+\cdots
\]
\[
A(h/r)=A^\*+c_1(h/r)^p+c_2(h/r)^{p+s}+\cdots
\]

消去 \(c_1h^p\) 后得到一阶外推：
\[
R_1=\frac{r^pA(h/r)-A(h)}{r^p-1}
\]

若误差幂次按步长 \(s\) 递增（例如中心差分和梯形积分常见 \(s=2\)），则可构造三角外推表：
\[
R_{i,0}=A(h_i),\quad h_i=h_0/r^i
\]
\[
R_{i,j}=R_{i,j-1}+\frac{R_{i,j-1}-R_{i-1,j-1}}{r^{p+s(j-1)}-1},\quad j=1,\dots,i
\]

## R04

适用前提：
- 已知或可估计基方法主误差阶 \(p\)。
- 近似值序列处于渐近区（步长不能过大）。
- 浮点舍入误差未主导（步长也不能过小）。
- 目标函数在邻域内足够光滑（保证误差展开成立）。

## R05

算法直觉：
1. 先用多个步长算出一列“粗到细”的近似值。
2. 每次用“粗-细配对”消掉一个最低阶误差项。
3. 沿三角表对角线向右下推进，理论阶数逐级提高。
4. 取最后一个对角元素作为最终估计。

## R06

正确性要点（非严格证明）：
- 若 \(A(h)\) 误差主项为 \(c_1h^p\)，则 \(A(h)\) 与 \(A(h/r)\) 的线性组合可令该项系数为 0。
- 消元后新误差主项变为更高次幂（如 \(h^{p+s}\)），因此收敛速度加快。
- 重复此过程即可逐层抬高精度阶。

## R07

复杂度（使用 \(m\) 个步长层级）：
- 生成基础近似：\(m\) 次基方法调用。
- 构造三角表：\(\sum_{i=1}^{m-1} i = O(m^2)\) 次常数运算。
- 总时间：\(O(m\cdot C_{\text{base}} + m^2)\)，其中 \(C_{\text{base}}\) 为单次基础近似成本。
- 空间：完整表 \(O(m^2)\)；若仅保留相邻列可降到 \(O(m)\)。

## R08

数值稳定性与误差平衡：
- 步长过大：截断误差主导，外推收益有限。
- 步长过小：差分相减导致灾难性消去，舍入误差放大。
- 实务上常用 4-8 个层级，且观察“对角线误差先降后升”的拐点，选最优层级。

## R09

伪代码：

```text
input: base_estimator(h), h0, levels m, order p, ratio r, stride s
for i in 0..m-1:
    h[i] = h0 / r^i
    R[i,0] = base_estimator(h[i])
for i in 1..m-1:
    for j in 1..i:
        factor = r^(p + s*(j-1))
        R[i,j] = R[i,j-1] + (R[i,j-1] - R[i-1,j-1]) / (factor - 1)
return R[m-1,m-1], R
```

## R13

结果解释指南：
- `raw` 列是基础方法值，通常随步长减小而收敛。
- `diag` 列是同层对角外推值，通常比 `raw` 精度更高。
- 若后几层对角误差不再下降，往往是浮点噪声开始影响。

## R14

常见实现错误：
- 把分母写成 `r^(p+j)` 而忽略误差幂步进 `s`。
- 外推时误用同一行而非上一行的数据。
- 步长更新错误（应按几何缩放）。
- 忽视基础方法误差阶与 `order` 的一致性。

## R15

可扩展方向：
- 自适应终止：根据相邻对角线差值自动停止。
- 与高精度浮点（`decimal`/`mpmath`）结合，减轻舍入误差。
- 用于 ODE/PDE 网格收敛率验证（Grid Convergence Study）。

## R16

与相关方法对比：
- 仅减小步长：精度提升慢，成本上升快。
- 提高基格式阶数：需要推导新公式，改动大。
- Richardson 外推：可复用现有低阶格式，通过后处理显著提精度，工程上性价比较高。

## R17

最小测试建议：
1. 求导测试：\(f(x)=e^x,\ x=1\)，真值 \(e\)。
2. 积分测试：\(\int_0^1 e^x dx=e-1\)。
3. 参数健壮性：`levels` 取 3/5/7 比较误差趋势。
4. 退化检查：`levels=1` 时应等于基础近似值。

## R18

`demo.py` 源码级算法流程（8 步）：
1. 定义基础近似器：中心差分 `centered_difference` 与复化梯形 `trapezoidal_rule`。
2. 通过 `build_raw_sequence_*` 生成步长按 \(h_0/r^i\)（或分段数按 \(n_0r^i\)）变化的基础近似序列。
3. 在 `richardson_table` 中创建 \(m\times m\) 表，第一列填入基础序列。
4. 对每个 `i=1..m-1`、`j=1..i` 计算放大因子 `factor = ratio ** (order + error_stride*(j-1))`。
5. 按递推式更新 `table[i,j] = table[i,j-1] + (table[i,j-1]-table[i-1,j-1])/(factor-1)`，逐层消去误差项。
6. 提取每一层 `raw=table[i,0]` 与 `diag=table[i,i]`，和解析真值比较绝对误差。
7. 打印结构化文本表，展示“基础近似 vs 外推对角”的误差下降趋势。
8. 返回最终对角值作为 Richardson 结果，完成一次可复现实验闭环。
