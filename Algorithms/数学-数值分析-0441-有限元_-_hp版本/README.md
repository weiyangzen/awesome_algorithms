## R01

`hp-FEM`（有限元 `hp` 版本）把两类离散手段结合在一起：

- `h`-refinement：把网格单元切小；
- `p`-refinement：在单元内提升多项式阶数。

相比只做 `h` 或只做 `p`，`hp` 方法能在“局部不光滑区域”倾向细网格、在“光滑区域”倾向高阶基函数，通常能以更少自由度得到更高精度。

本条目给出 1D Poisson 方程的可运行 MVP，用残量 + 局部平滑度做 `h/p` 决策。

## R02

本条目离散问题定义：

- 连续问题：
  - `-u''(x)=f(x), x in (0,1)`
  - `u(0)=u(1)=0`
- 数值输入（由脚本内参数给定）：
  - 初始网格（6 个单元，初始阶数交替 `p=1/2`）；
  - 最大多项式阶 `p_max=6`；
  - 最小单元尺度 `h_min=1/256`；
  - 自适应轮数 `n_cycles=5`。
- 数值输出：
  - 每轮 `elements`、`dofs`、`L2` 误差、`H1` 半范误差；
  - 每轮 `h-ref` / `p-ref` 次数；
  - 最终一致性检查（误差下降且同时发生过 `h` 与 `p` 加密）。

## R03

核心离散模型：

1. 弱形式：求 `u_h in V_h^p`，使
   `a(u_h,v_h)=l(v_h)`，其中
   `a(u,v)=∫_0^1 u'v' dx`，`l(v)=∫_0^1 f v dx`。
2. 单元映射：每个单元 `K=[xL,xR]` 由参考区间 `[-1,1]` 仿射映射到物理区间。
3. 基函数：单元内使用高阶 Lagrange 节点基（Chebyshev-Lobatto 节点）。
4. 积分：使用 Gauss-Legendre 求积，装配局部刚度矩阵与载荷向量。
5. 自适应指标：
   - 残量指示子 `eta_K ~ h_K * ||f + u_h''||_{L2(K)}`；
   - 平滑度指标 `sigma_K`（局部 `p` 阶解与 `p-1` 阶拟合差比值）。
6. 决策：`sigma_K` 小则优先 `p`-refine，反之优先 `h`-refine。

## R04

时间复杂度（单轮）可按三部分看：

- 装配：`O(sum_K (p_K+1)^2 * nq_K)`；
- 线性求解：使用 `scipy.sparse.linalg.spsolve`（稀疏直接法，实际复杂度依稀疏结构与填充而变）；
- 误差与指标评估：`O(sum_K (p_K+1) * nq_K)`。

在该 1D MVP 中，自由度规模较小，装配与评估都很快，瓶颈通常在稀疏求解。

## R05

空间复杂度：

- 解向量与载荷向量：`O(N)`；
- 稀疏刚度矩阵：`O(nnz)`，在 1D 高阶单元下可近似看作 `O(sum_K (p_K+1)^2)`；
- 求积与基函数缓存：`O(max_p * max_nq)` 量级的小常数缓存。

总空间规模通常由稀疏矩阵主导。

## R06

一步局部装配微例（线性单元）：

若单元长度 `h=0.5`，线性基函数在物理坐标上的导数是常数：

- `dphi_1/dx = -1/h = -2`
- `dphi_2/dx =  1/h =  2`

局部刚度矩阵
`K_e(i,j)=∫_K dphi_i/dx * dphi_j/dx dx`
得到

`K_e = [[ 2, -2],
        [-2,  2]]`。

这说明单元越小（`h` 越小），局部刚度贡献越大，反映了离散导数惩罚增强。

## R07

`hp-FEM` 的工程意义：

- 对局部层结构、边界层、尖峰等问题更高效；
- 允许把计算资源集中在“困难区域”；
- 在 PDE 数值解、计算力学、波动问题中广泛应用；
- 适合做从教学到工程原型的渐进扩展（从 1D 到高维、从标量到多物理）。

## R08

理论基础要点：

1. 有限元把无限维问题投影到有限维子空间；
2. `h` 收敛依赖网格尺寸，`p` 收敛依赖解的光滑性；
3. 对解析/高光滑解，`p`-refinement 常有快速收敛；
4. 对非光滑区域，盲目升阶收益差，`h`-refinement 更稳健；
5. `hp` 策略本质是“局部正则性驱动的离散自由度分配”。

## R09

适用条件与局限：

- 适用：
  - 椭圆型边值问题；
  - 解具有混合正则性（局部粗糙 + 全局平滑）；
  - 需要在精度与自由度之间做平衡。
- 局限：
  - 自适应判据设计会直接影响效果；
  - 高维问题实现复杂度显著上升；
  - 本 MVP 仅是 1D Poisson，不含复杂边界条件与并行优化。

## R10

正确性框架（本实现中的可验证链路）：

1. 使用制造解 `u_exact` 明确给出 `f=-u_exact''`；
2. 离散方程按弱形式装配到全局线性系统；
3. 施加齐次 Dirichlet 边界后求解自由自由度；
4. 用高阶求积评估 `L2` 与 `H1` 误差；
5. 多轮自适应后检查 `final L2 < initial L2`；
6. 若出现非有限值、网格非法、收敛异常，脚本直接抛错。

## R11

误差与稳定性说明：

- 本问题是对称正定椭圆方程，有限元离散后系统在边界约束下可稳定求解；
- 积分精度由 Gauss-Legendre 阶数控制，脚本按 `p` 自适应增加 `nq`；
- `h` 与 `p` 同时可调，理论上能覆盖“局部细化”与“全局高阶”两种误差下降路径；
- 自适应是启发式策略，不保证最优，但对演示目标（误差下降 + hp 行为）足够稳健。

## R14

鲁棒性与常见失效模式：

- 网格不连续、单元长度非正、`p<1`：在 `validate_mesh` 阶段拦截；
- 线性系统异常导致 `nan/inf`：在求解后立即检查；
- 自适应过度细化：`h_min` 和 `p_max` 限制增长；
- 指标误判：保留回退逻辑（在光滑区强制一次 `p` 候选）以避免退化成纯 `h`。

## R18

源码级算法流程拆解（`run_hp_fem_demo` 主线，8 步）：

1. 构造初始网格（区间 `[0,1]`，混合初始阶数 `p=1/2`）。
2. 在 `solve_fem` 中为每个单元构建高阶 Lagrange 基与导数表，并用 Gauss 求积装配局部 `K_e/F_e`。
3. 将局部矩阵向量汇总到全局稀疏系统，施加两端 Dirichlet 边界后调用 `spsolve` 求解。
4. 在 `compute_errors_and_indicators` 中逐单元重建 `u_h`、`u_h'`、`u_h''`，计算 `L2/H1` 误差。
5. 同时计算残量型指示子 `eta_K` 与平滑度 `sigma_K`，作为自适应输入。
6. 在 `hp_adapt` 中按 `eta_K` 标记单元，再用 `sigma_K` 决策：平滑则 `p+1`，否则二分单元做 `h` 细化。
7. 迭代上述流程多轮，持续记录 `dofs`、误差、`h/p` 细化计数。
8. `run_checks` 做终态验收（误差下降且出现过 `h` 与 `p` 两类动作），最后输出成功信息。

说明：本实现只把 `scipy` 用作稀疏线性代数工具，有限元离散与 `hp` 决策均在源码中显式展开，不是黑箱 PDE 调用。
