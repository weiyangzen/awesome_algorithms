## R01

路径积分表述把量子传播子写成“对所有路径求和”的形式。对一维粒子，实时间传播子为：

`K(x_f, t_f; x_i, t_i) = ∫ D[x(t)] exp(i S[x] / hbar)`

其中 `S[x] = ∫ L(x, x_dot) dt` 是作用量。该方法与薛定谔表述等价，但更强调“振幅来自全部历史路径的干涉叠加”。

## R02

典型应用场景：

- 从拉格朗日量直接构造传播子，而不先写哈密顿算符本征问题
- 在量子场论中通过泛函积分计算关联函数
- 在统计物理中经 Wick 旋转转为欧氏时间，计算配分函数或基态投影
- 在半经典近似中围绕经典路径做涨落展开

## R03

本 MVP 采用欧氏时间 `tau = i t` 的离散化形式，目标是 1D 谐振子传播子 `K_E(x_f, beta; x_i, 0)`。

离散网格：`beta = N * delta_tau`，固定端点 `x_0 = x_i`, `x_N = x_f`，对中间点 `x_1...x_{N-1}` 积分。

离散欧氏作用量（对势能做对称拆分）：

`S_E ≈ Σ_{k=0}^{N-1} [ m/(2*delta_tau) * (x_{k+1}-x_k)^2 + (delta_tau/2) * (V(x_k)+V(x_{k+1})) ]`

对谐振子 `V(x)=0.5*m*omega^2*x^2`，该离散作用量对积分变量是严格二次型，可化为高维高斯积分。

## R04

直观理解：

- 在实时间里，不同路径贡献复相位并发生干涉
- 在欧氏时间里，权重变为 `exp(-S_E/hbar)`，作用量越小权重越大
- 对二次作用量系统（如谐振子），路径积分可退化为“线性代数问题”
- 离散路径积分可看作把无穷维积分截断为有限维积分，再让 `N` 增大逼近连续极限

## R05

正确性要点：

1. 对谐振子，离散后积分核是高斯型，解析积分步骤严格可复核。
2. 本实现将 `S_E` 写成 `0.5 * x^T A x - h^T x + c`，并通过 Cholesky 解 `A^{-1} h` 与 `det(A)`。
3. 当 `m>0`, `beta>0` 且 `omega^2>=0` 时，矩阵 `A` 对称正定，保证高斯积分收敛。
4. 随 `N` 增大，离散传播子应收敛到已知解析解，本 demo 用相对误差下降验证这一点。

## R06

复杂度分析（`N` 为时间切片数，积分维度 `d=N-1`）：

- 构造离散系统矩阵 `A`：`O(d)`（三对角结构）
- 当前 MVP 以稠密矩阵交给 Cholesky：时间 `O(d^3)`，空间 `O(d^2)`
- 若后续改用专用三对角算法，可降到 `O(d)`~`O(d^2)` 级别

本条目优先“可读、可核验”的最小实现，因此保留稠密求解路径。

## R07

标准实现步骤：

1. 设定 `hbar, m, omega, beta, x_i, x_f` 与切片数 `N`。
2. 由离散作用量构造二次型参数 `A, h, c`。
3. 计算短时核归一化项 `(m/(2*pi*hbar*delta_tau))^(N/2)`。
4. 用 Cholesky 分解得到 `log(det(A))` 与 `A^{-1}h`。
5. 按高斯积分公式合成离散传播子 `K_E^(N)`。
6. 与谐振子欧氏传播子解析解 `K_E^exact` 比较，输出相对误差。
7. 扫描不同 `N`，观察收敛趋势。

## R11

边界条件与异常处理：

- `n_slices < 2` 时抛出 `ValueError`
- `beta <= 0`, `mass <= 0`, `hbar <= 0` 时抛出 `ValueError`
- `omega ≈ 0` 时走自由粒子解析解分支，避免除以 `sinh(omega*beta)`
- 若矩阵不是正定（极端参数导致），Cholesky 会失败并显式暴露问题

## R12

与其他量子力学表述的关系：

- 与薛定谔方程等价：都给出相同传播子
- 与海森堡表述等价：只是动力学信息组织方式不同
- 欧氏路径积分与统计物理联系紧密：`exp(-beta H)` 形式直接对应热权重
- 在场论中，路径积分推广为对场配置的泛函积分

## R14

工程实现注意点：

- 直接算行列式可能数值不稳，MVP 使用 `log(det(A))` 路线避免下溢/上溢
- 指数项统一在对数域组装后再 `exp`，减少浮点误差
- 当前实现把三对角矩阵显式展开为稠密矩阵，重在可读性；高性能场景应改用带状矩阵算法

## R18

源码级算法流程拆解（对应 `demo.py`，共 8 步）：

1. **离散化与变量分离**
   `build_euclidean_system` 固定端点 `x_0, x_N`，把中间路径点整理成向量变量，得到有限维积分问题。

2. **从作用量展开二次型**
   将离散 `S_E` 展开为 `0.5*x^T A x - h^T x + c`，其中 `A` 是对称三对角矩阵，`h` 只在首末分量受边界驱动。

3. **构造短时核归一化对数项**
   在 `discrete_euclidean_propagator` 中先计算 `log_prefactor`，避免直接连乘导致数值不稳。

4. **调用 `scipy.linalg.cho_factor`（LAPACK `dpotrf`）**
   该步骤把 `A = L L^T`（或 `U^T U`）分解，既用于后续线性求解，也用于提取 `log(det(A))`。

5. **调用 `scipy.linalg.cho_solve`（LAPACK `dpotrs`）**
   通过三角回代得到 `A^{-1} h`，避免显式矩阵求逆。

6. **组装高斯积分闭式项**
   计算 `0.5 * h^T A^{-1} h` 与 `-0.5 * log(det(A))`，并与常数项 `c` 合并为总对数权重。

7. **指数回到传播子数值**
   将 `log_prefactor + log_gaussian + exponent` 求 `exp`，得到离散路径积分近似 `K_E^(N)`。

8. **解析解对照与收敛输出**
   `run_convergence_experiment` 逐个 `N` 计算 `rel_error`，检验离散路径积分是否向解析核收敛。
