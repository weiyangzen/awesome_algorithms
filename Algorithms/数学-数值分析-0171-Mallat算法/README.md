## R01

Mallat算法是离散小波变换（DWT）的快速分解与重构框架，本质上是“二通道滤波器组 + 下采样/上采样”的多分辨率实现。它把原始信号逐层拆成：
- 低频近似系数（approximation，保留整体趋势）
- 高频细节系数（detail，保留突变和边缘）

在数值分析里，Mallat算法常用于去噪、压缩、特征提取和多尺度分析。

## R02

核心历史脉络：
- 1980s：多分辨率分析（MRA）与正交小波理论逐步完善。
- 1989：Stéphane Mallat提出快速小波分解/重构算法，将理论转为可高效计算的工程流程。
- 之后：Mallat框架成为小波应用的标准计算骨架（图像压缩、信号去噪、时频分析等）。

## R03

数学基础（以一维离散信号为例）：
- 设尺度函数对应低通滤波器为 `h[k]`，小波函数对应高通滤波器为 `g[k]`。
- 第 `j` 层近似系数为 `a_j[n]`，细节系数为 `d_j[n]`。

分解公式（analysis）：
- `a_{j+1}[n] = Σ_k h[k - 2n] * a_j[k]`
- `d_{j+1}[n] = Σ_k g[k - 2n] * a_j[k]`

重构公式（synthesis）：
- `a_j[n] = Σ_k h_tilde[n - 2k] * a_{j+1}[k] + Σ_k g_tilde[n - 2k] * d_{j+1}[k]`

当滤波器满足完美重构条件时，可无损恢复原信号（浮点误差范围内）。

## R04

输入与输出定义：
- 输入：长度为 `N` 的离散信号 `x`，分解层数 `L`（通常要求 `N` 可被 `2^L` 整除，或采用边界延拓）。
- 输出：
  - 最粗层近似系数 `a_L`
  - 每层细节系数集合 `{d_1, d_2, ..., d_L}`

本文MVP采用 Haar 小波（Daubechies-1）实现 Mallat 流程，便于把算法机制写清楚而不是黑箱调用。

## R05

核心思想可概括为三点：
- 分而治之：每层把信号分成“慢变化”和“快变化”。
- 尺度递进：只对低频近似继续分解，形成金字塔结构。
- 局部性表达：突变信息集中到少量高频系数，天然利于阈值去噪与压缩。

## R06

标准流程：
1. 设 `a_0 = x`。
2. 对 `a_j` 做低通/高通卷积。
3. 对两路结果分别下采样（隔点取样）。
4. 得到 `a_{j+1}` 与 `d_{j+1}`。
5. 重复直到达到层数 `L`。
6. 任务相关处理（如对 `d_j` 做阈值收缩去噪）。
7. 用逆过程逐层上采样、滤波并叠加重构。

## R07

伪代码（Haar版本）：

```text
function mallat_decompose(x, L):
    a = x
    details = []
    for j in 1..L:
        low  = (a[0::2] + a[1::2]) / sqrt(2)
        high = (a[0::2] - a[1::2]) / sqrt(2)
        details.append(high)
        a = low
    return a, details

function mallat_reconstruct(a, details):
    cur = a
    for d in reverse(details):
        y[0::2] = (cur + d) / sqrt(2)
        y[1::2] = (cur - d) / sqrt(2)
        cur = y
    return cur
```

## R08

正确性直觉：
- Haar情形下，每对样本 `(x0, x1)` 被正交变换为 `(low, high)`：
  - `low = (x0 + x1)/sqrt(2)`
  - `high = (x0 - x1)/sqrt(2)`
- 逆变换恰为：
  - `x0 = (low + high)/sqrt(2)`
  - `x1 = (low - high)/sqrt(2)`

因此每层都是可逆的；多层串联后仍可逆，得到完美重构（数值上仅有浮点舍入误差）。

## R09

复杂度分析：
- 时间复杂度：`O(N)`（每层处理样本数减半，总和 `N + N/2 + ... < 2N`）。
- 空间复杂度：`O(N)`（存放各层系数）。

相较直接卷积型连续小波变换，Mallat离散方案在工程上更高效。

## R10

数值稳定性与边界问题：
- 稳定性：正交小波（如Haar）能量守恒较好，数值行为稳定。
- 浮点误差：层数很深时误差累积会增大，但通常可接受。
- 边界处理：真实应用中需选择延拓策略（零填充、对称延拓、周期延拓等），会影响边缘系数质量。

本MVP使用长度为 `2^k` 的信号以避免边界分支复杂化。

## R11

典型应用场景：
- 一维时序去噪（传感器信号、生理信号）
- 图像压缩前的多尺度变换（二维可分离扩展）
- 异常点/瞬态检测
- 多尺度特征工程（机器学习前处理）

## R12

不太适合的场景：
- 信号极短，分层意义不足。
- 需要严格平移不变性的任务（DWT受下采样影响，平移敏感）。
- 频率分辨率需求非常精细且固定（可考虑STFT/小波包等替代）。

## R13

关键参数与经验：
- 小波基：Haar简单快速，Daubechies/Symlets更平滑。
- 分解层数 `L`：通常 `L <= log2(N)`，过深可能放大边界效应。
- 阈值策略（去噪）：
  - 通用阈值 `T = σ * sqrt(2 log N)`
  - `σ` 可由最高频细节MAD估计：`σ ≈ median(|d1|)/0.6745`
- 阈值函数：软阈值通常比硬阈值更平滑。

## R14

与相近方法对比：
- 对比傅里叶变换：
  - 傅里叶基是全局正弦波，时间局部性差。
  - Mallat小波在时频两域更均衡，适合非平稳信号。
- 对比STFT：
  - STFT窗长固定；Mallat是多尺度窗，低频看全局、高频看局部。
- 对比直接调用黑箱库：
  - 库函数效率高，但不利于理解细节。
  - 本项目MVP显式实现核心步骤，便于教学与审计。

## R17

结果解释与最小验证标准：
- 重构一致性：未阈值处理时，`reconstruct(decompose(x))` 应近似 `x`。
- 去噪有效性：相对 noisy 信号，denoised 的 MSE 应下降、SNR应上升。
- 可复现性：固定随机种子后，每次运行输出稳定。

## R18

`demo.py` 的源码级算法流（非黑箱）分为 8 步：
1. 构造测试信号：叠加正弦、阶跃和窄脉冲，得到 `clean`；再加高斯噪声得到 `noisy`。
2. 执行 `mallat_haar_decompose(noisy, levels)`：逐层做相邻点线性变换，产出 `a_L` 和 `d_1...d_L`。
3. 从第一层细节 `d_1` 用 MAD 估计噪声标准差 `sigma`。
4. 计算通用阈值 `threshold = sigma * sqrt(2*log(N))`。
5. 对每一层细节系数执行 `soft_threshold`，得到收缩后的 `d'_j`。
6. 执行 `mallat_haar_reconstruct(a_L, d'_1...d'_L)`，逐层逆变换回时域信号。
7. 计算指标：`MSE(clean,noisy)`、`MSE(clean,denoised)`、`SNR` 与 `SNR提升`。
8. 打印系数维度、统计指标和前10个样本对比，完成最小可验证闭环。
