# 量子谐振子 (Quantum Harmonic Oscillator)

- UID: `PHYS-0026`
- 学科: `物理`
- 分类: `量子力学`

## R01

量子谐振子是量子力学中最基础、最核心的可解析模型之一：
它描述势能为 `V(x) = (1/2) m omega^2 x^2` 的束缚粒子，具有离散能级与正交本征态。

该模型是许多更复杂体系（晶格振动、量子场模态、近似势阱展开）的局部基元，因此具有很强的方法论价值。

## R02

本条目要解决的问题是：
在给定 `m, omega, hbar` 时，数值求解一维定态薛定谔方程

`H psi_n(x) = E_n psi_n(x)`，`H = -(hbar^2/(2m)) d^2/dx^2 + (1/2) m omega^2 x^2`

并验证：
1. 数值能级是否接近解析解 `E_n = hbar*omega*(n+1/2)`。
2. 数值本征函数是否满足离散正交归一。
3. 数值波函数是否接近解析 Hermite-Gaussian 形式。

## R03

理论基线：
- 哈密顿量由动能项和二次势能项组成。
- 能谱离散且等间隔：`Delta E = hbar*omega`。
- 第 `n` 态解析波函数可写为 Hermite 多项式与高斯包络的乘积。

本 MVP 采用此理论作为“数值实现 + 误差验证”的金标准。

## R04

离散化策略：
- 在区间 `[-x_max, x_max]` 上取均匀网格，边界施加 `psi=0`（Dirichlet）。
- 用中心差分近似二阶导数：
`psi''(x_i) ≈ (psi_{i+1} - 2psi_i + psi_{i-1}) / dx^2`。
- 得到一个对称三对角离散哈密顿矩阵 `H`。

这样，连续本征值问题转化为标准的对称矩阵本征问题，适合稳定求解前若干低能态。

## R05

模型假设与适用范围：
- 仅一维、单粒子、非相对论框架。
- 势场固定且严格二次型。
- 使用有限区间近似无限域，要求 `x_max` 足够大使边界截断误差很小。
- 只求低能若干本征态（MVP 默认前 6 态）。

若考虑非谐势、时变势、散射态或高维耦合，需要更一般的数值框架。

## R06

`demo.py` 输入输出约定：
- 输入：脚本内置参数 `m, omega, hbar, x_max, n_points, n_states`，无交互输入。
- 输出：
1. 每个能级的 `E_num / E_exact / 误差`。
2. 本征方程残差 `||H psi - E psi||`。
3. 波函数 `L2` 误差（对解析解）。
4. `<x^2>` 的数值-解析对比。
5. 正交归一指标与总体验证 `PASS/FAIL`。

## R07

算法主流程（高层）：
1. 生成对称均匀网格与内部节点。
2. 组装三对角离散哈密顿量（主对角 + 副对角）。
3. 求解最低 `n_states` 个本征对。
4. 按离散积分 `sum |psi|^2 dx = 1` 重新归一化本征函数。
5. 计算本征残差和 Gram 矩阵，验证正交性。
6. 计算解析能级与解析波函数并进行相位对齐。
7. 统计能量误差、波函数 `L2` 误差、`<x^2>` 误差并给出阈值判定。

## R08

复杂度分析（设内部网格点为 `N`，目标本征态数为 `k`）：
- 组装三对角矩阵：`O(N)`。
- 三对角本征求解（前 `k` 态）：常见实现约 `O(N*k)` 到 `O(N^2)`，通常显著优于稠密 `O(N^3)`。
- 误差与积分统计：`O(N*k)`。

在本 MVP 的中等网格规模（约千点）下可快速完成。

## R09

数值稳定性与误差控制：
- 使用对称三对角结构，避免显式构建病态逆矩阵。
- 波函数在离散内积下再次归一化，减少缩放误差。
- 对比不仅看能量，还看残差、正交性和观测量 `<x^2>`，防止“只对一个指标拟合好”。
- 通过增大 `x_max` 与 `n_points` 可系统降低截断误差与离散误差。

## R10

MVP 技术栈：
- Python 3
- `numpy`：网格、向量化、离散积分与线性代数基础操作
- `scipy`：`eigh_tridiagonal`（三对角本征求解）、`eval_hermite`（解析波函数）
- `pandas`：结果表格化输出

未使用黑箱“量子求解器”；核心离散建模与验证逻辑均在源码中显式实现。

## R12

输出字段说明：
- `n`: 量子数（从 0 开始）。
- `E_num`: 数值本征能量。
- `E_exact`: 解析能量 `hbar*omega*(n+1/2)`。
- `abs_err_E` / `rel_err_E`: 能级绝对与相对误差。
- `residual_L2`: `||H psi_n - E_n psi_n||_2`。
- `psi_L2_error`: 数值波函数与解析波函数的离散 `L2` 差。
- `<x^2>_num` / `<x^2>_exact`: 二阶矩观测量数值值与解析值。
- `abs_err_<x^2>`: 二阶矩绝对误差。

## R13

正确性验证（脚本内置阈值检查）：
1. 最大相对能量误差应小于阈值。
2. 最大本征残差应足够小。
3. 波函数 Gram 矩阵非对角元素应接近 0。
4. Gram 对角应接近 1（归一化成立）。
5. 波函数 `L2` 误差与 `<x^2>` 误差应在可接受范围内。

全部通过时输出 `Validation: PASS`，否则返回非零退出码。

## R17

方案对比：
- 解析法：理论最干净，但仅适用于少数可解势。
- 基函数展开法（如 Hermite 基）：收敛快但需要基截断与矩阵元推导。
- 通用黑箱 PDE 求解器：开发快但算法细节不透明。
- 本方案（有限差分 + 三对角本征）：实现简洁、结构清楚、验证链完整，适合作为最小可审计 MVP。

## R18

`demo.py` 的源码级算法流可拆为 9 步：
1. `build_uniform_grid` 构造 `[-x_max, x_max]` 均匀网格并取内部点，边界默认 `psi=0`。
2. `build_hamiltonian_tridiagonal` 按中心差分把动能项离散成三对角形式，并叠加势能主对角，得到 `(main_diag, off_diag)`。
3. `solve_lowest_states` 调用 `scipy.linalg.eigh_tridiagonal` 只提取最低 `k` 个本征值/向量，而非求全谱。
4. `normalize_on_grid` 在离散积分度量下对每个本征函数归一化，确保 `sum |psi|^2 dx = 1`。
5. `residual_norms` 通过 `apply_tridiagonal` 显式计算 `H psi - E psi`，得到每个能级的残差范数。
6. `extend_with_boundaries` 把内部本征函数补齐到全网格（两端填零），用于和解析波函数逐点比较。
7. `analytic_energy` 与 `analytic_wavefunction` 生成解析基准；`align_phase` 处理本征向量符号不唯一性。
8. `expectation_x2` 与 `gram_metrics` 计算观测量误差与正交归一误差，形成多视角验证。
9. `main` 汇总为 `pandas` 表格并执行阈值检查，最终输出 `Validation: PASS/FAIL`。

虽然使用了线性代数库函数求本征值，但离散建模、误差定义、物理量验证和判定逻辑都在源码层明确展开，不是黑箱调用。
