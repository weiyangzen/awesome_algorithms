## R01

等离子体频率（通常指电子等离子体频率）是冷等离子体中电子在离子背景上发生小振幅集体振荡的本征频率。  
基本定义：

`omega_pe = sqrt(n_e * e^2 / (epsilon_0 * m_e))`

其中 `n_e` 为电子数密度，`e` 为元电荷，`m_e` 为电子质量，`epsilon_0` 为真空介电常数。对应普通频率 `f_pe = omega_pe / (2*pi)`。

## R02

物理意义上，`omega_pe` 给出了“电荷分离后恢复力驱动的最快典型时间尺度”。  
在无磁、冷、均匀近似下，电磁波在等离子体中的传播满足：

`omega^2 = omega_pe^2 + c^2 k^2`

这意味着当外加波频率 `omega < omega_pe` 时，波数变为虚数，波在介质中表现为倏逝而非传播（截止现象）。

## R03

本条目 MVP 聚焦 3 个最小闭环目标：

- 从输入密度 `n_e` 直接计算 `omega_pe`、`f_pe`、周期 `T=2*pi/omega_pe`；
- 数值模拟冷电子振荡方程 `x'' + omega_pe^2 x = 0`，并反推频率，验证与解析值一致；
- 给定探测频率 `f0`，判断不同密度下电磁波“可传播 / 被截止”并估算倏逝深度。

## R04

`demo.py` 使用的核心方程：

1. 电子等离子体角频率：
   `omega_pe = sqrt(n_e e^2 / (epsilon_0 m_e))`
2. 振荡动力学（线性冷等离子体近似）：
   `d^2x/dt^2 + omega_pe^2 x = 0`
3. 数值推进（半隐式 Euler / symplectic Euler）：
   `v_{n+1} = v_n - omega_pe^2 x_n dt`  
   `x_{n+1} = x_n + v_{n+1} dt`
4. 冷无磁色散关系：
   `k^2 = (omega^2 - omega_pe^2)/c^2`
5. 截止区 (`omega < omega_pe`) 的倏逝深度：
   `delta = 1/kappa,  kappa = sqrt(omega_pe^2 - omega^2)/c`

## R05

设时间步数为 `N_t`，密度扫描点数为 `N_d`：

- 频率公式计算：`O(N_d)`；
- 振荡数值推进：`O(N_t)`；
- FFT 频率估计：`O(N_t log N_t)`；
- 截止表评估：`O(N_d)`。

总体时间复杂度由 FFT 主导，为 `O(N_t log N_t)`；空间复杂度为 `O(N_t + N_d)`。

## R10

正确性直觉：

1. `n_e` 越大，恢复电场越强，振荡应更快，因此 `omega_pe` 应随 `sqrt(n_e)` 增大；
2. `x'' + omega^2 x = 0` 的本征频率应与 `omega_pe` 一致；
3. 若探测 `omega` 低于介质本征 `omega_pe`，传播常数应变虚，出现倏逝；
4. 仿真中频率误差与能量漂移足够小，可视为离散实现与理论一致。

## R11

数值稳定与精度策略：

- 时间步按 `dt = dt_factor / omega_pe` 自适应，保证振荡器离散稳定；
- 采用 symplectic Euler，相比显式 Euler 更能控制长时能量漂移；
- FFT 估频只在后段样本上进行，减小初始瞬态影响；
- 峰值点使用局部二次插值，提高频率估计分辨率。

## R12

关键参数（`demo.py` 默认值）：

- `n_e = 1e16 m^-3`：演示密度；
- `x0 = 1e-5 m`、`v0 = 0`：初始微扰；
- `dt_factor = 0.02`：无量纲步长系数，越小越精确；
- `steps = 40000`：总步数，影响 FFT 分辨率；
- `probe_frequency_hz = 1e9`：传播截止判别频率。

调参建议：

- 高频密度场景可适当减小 `dt_factor`；
- 估频分辨率不足时优先增加 `steps`；
- 若关心传播边界，密度扫描应围绕临界密度 `n_crit` 加密。

## R14

常见失效模式：

1. `dt_factor` 过大导致离散相位误差变大甚至不稳定；
2. `steps` 过小使 FFT 分辨率不足，频率估计偏粗；
3. `n_e <= 0` 导致物理参数非法；
4. 在截止附近直接用单点判据，可能对边界条件过于敏感。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `PlasmaConfig` 固化密度、步长系数、总步数和探测频率等输入参数。  
2. `plasma_angular_frequency` / `plasma_frequency_hz` 由常量与 `n_e` 直接计算解析 `omega_pe` 和 `f_pe`。  
3. `simulate_cold_plasma_oscillation` 依据 `x'' + omega_pe^2 x = 0` 用 symplectic Euler 逐步推进 `x,v`，并记录归一化能量序列。  
4. `estimate_angular_frequency_fft` 对位移序列做 `rFFT`，选取主峰后用局部抛物线插值给出 `omega` 估计。  
5. `build_density_scan_table` 对一组 `n_e` 生成 `omega_pe`、`f_pe`、周期表，验证 `sqrt(n_e)` 标度。  
6. `cutoff_analysis_table` 根据 `omega^2 = omega_pe^2 + c^2 k^2` 判断每个密度点是传播还是截止，并在截止区计算 `delta=1/kappa`。  
7. `critical_density_for_frequency` 计算指定探测频率的临界密度 `n_crit`，用于解释扫描结果中的传播边界。  
8. `main` 汇总打印关键指标并执行断言（频率误差、能量漂移、单调性与截止一致性），最后输出 `Validation: PASS/FAIL`。  

第三方库没有被当黑盒求解器直接替代算法：`numpy` 负责数组与 FFT，`pandas` 仅负责表格展示，核心物理方程、离散推进与验证逻辑均在源码中显式实现。
