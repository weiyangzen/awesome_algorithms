# 矩阵平方根计算

- UID: `MATH-0101`
- 学科: `数学`
- 分类: `线性代数`
- 源序号: `101`
- 目标目录: `Algorithms/数学-线性代数-0101-矩阵平方根计算`

## R01

矩阵平方根问题是：给定矩阵 `A`，求矩阵 `X` 使得

`X^2 = X @ X = A`。

对实对称正定矩阵（SPD）而言，存在唯一的实对称正定主平方根 `A^{1/2}`。本条目 MVP 聚焦这一最常见、最稳定的工程场景。

## R02

历史定位（简要）：

- 19 世纪后期起，矩阵函数理论逐步形成，平方根是最早被系统研究的矩阵函数之一。
- 20 世纪数值线性代数成熟后，特征分解、Schur 分解、迭代法（如 Denman-Beavers/Newton-Schulz）成为主流计算路径。
- 现代科学计算中，矩阵平方根用于协方差白化、最优传输、控制与滤波、量子信息等任务。

## R03

本条目解决的问题（MVP 版本）：

- 输入：实对称正定矩阵 `A`。
- 输出：一个实对称矩阵 `X`，满足 `X @ X ≈ A`。
- 方法：使用特征分解 `A = QΛQ^T`，构造 `X = Q sqrt(Λ) Q^T`。
- 验证：
  - 重构残差 `||X^2 - A||_F / ||A||_F`；
  - 对称性误差 `||X - X^T||_F`；
  - 可选对比 `scipy.linalg.sqrtm(A)`。

## R04

时间复杂度：

- 对 `n x n` 稠密 SPD 矩阵，核心是特征分解（`eigh`）；
- 渐近复杂度为 `O(n^3)`；
- 重构与误差计算（矩阵乘法、范数）同样为 `O(n^3)`，但常数较小。

## R05

空间复杂度：

- 存储输入矩阵、特征向量矩阵与输出矩阵，主量级为 `O(n^2)`；
- 若保留中间量（如 `X @ X`、`sqrtm` 参考值），额外仍是 `O(n^2)` 级别。

## R06

微型示例（对角情形最直观）：

设
`A = diag(4, 9)`，
则可取
`X = diag(2, 3)`，满足 `X^2 = A`。

一般 SPD 情形不是对角矩阵，但可通过正交变换转到特征空间，逐特征值开根号后再变换回原空间。

## R07

算法意义：

- 把标量开方推广到矩阵函数，为更复杂函数（`exp(A)`、`log(A)`、`A^alpha`）打基础；
- 是协方差矩阵处理（白化、着色、距离度量）中的核心算子；
- 在最优传输与高斯分布运算中，经常出现 `Sigma^{1/2}` 结构；
- 相比“逐元素开方”，矩阵平方根保留了线性变换结构信息。

## R08

核心理论与性质（SPD 场景）：

- 谱分解：`A = QΛQ^T`，其中 `Q` 正交、`Λ` 为正特征值对角阵；
- 主平方根：`A^{1/2} = Q sqrt(Λ) Q^T`；
- 唯一性：当 `A` 为 SPD，主平方根唯一且仍为 SPD；
- 特征值映射：`eig(A^{1/2}) = sqrt(eig(A))`（按主值取正）。

## R09

适用条件与局限：

适用：

- 输入可保证为实对称正定；
- 需要稳定、可解释且易验证的 MVP；
- 中小规模稠密矩阵计算。

局限：

- 非对称或含负特征值时，实主平方根未必存在；
- 特征值非常接近 0 时，数值敏感性会上升；
- 大规模稀疏问题更适合迭代/近似方法而非显式稠密分解。

## R10

正确性说明（工程版）：

1. SPD 矩阵可正交对角化：`A = QΛQ^T`，且 `Λ` 对角元全正。  
2. 定义 `S = sqrt(Λ)`（逐元素开方），构造 `X = QSQ^T`。  
3. 利用正交性 `Q^TQ = I`：  
   `X^2 = (QSQ^T)(QSQ^T) = QS(Q^TQ)SQ^T = QS^2Q^T = QΛQ^T = A`。  
4. 因而该构造在精确算术下严格正确，在浮点算术下通过残差检验确认数值正确性。

## R11

数值误差与稳定性：

- `numpy.linalg.eigh` 对实对称矩阵通常具有良好数值稳定性；
- 浮点舍入导致 `X` 可能有极小非对称噪声，代码中通过 `0.5*(X + X.T)` 做对称化；
- 条件数较大的 `A` 会放大误差，建议重点观察相对残差；
- 若使用 `sqrtm` 参考值，可能出现极小虚部（数值误差），应检测其范数而非直接报错。

## R12

性能视角：

- 特征分解法在 `n` 较小时实现最简洁、结果可解释；
- 与通用 `sqrtm` 相比，针对 SPD 的专用路径通常更快、更稳定；
- 当只需要 `A^{1/2}v` 而非完整 `A^{1/2}` 时，可进一步采用算子级方法降低开销。

## R13

可验证保证（本目录 MVP）：

- `demo.py` 固定随机种子，运行结果可复现；
- 自动验证两个样例矩阵（一个手工 SPD、一个随机 SPD）；
- 每个样例都断言：
  - 相对残差低于阈值；
  - 对称误差低于阈值；
  - 若 SciPy 可用，则与 `sqrtm` 结果一致到数值误差范围。

## R14

鲁棒性与常见失效模式：

- 非二维或非方阵输入：抛 `ValueError`；
- 非对称输入：抛 `ValueError`；
- 非有限值（`nan/inf`）：抛 `ValueError`；
- 存在非正特征值：抛 `ValueError`，提示输入不满足 SPD 前提。

## R15

实现设计（对应 `demo.py`）：

- `SqrtReport`：统一承载每个样例的误差指标；
- `validate_spd_matrix`：输入检查（维度、有限性、对称性、正定性）；
- `matrix_sqrt_spd_eigh`：核心算法，按谱分解构造平方根；
- `analyze_case`：计算残差、对称误差，并可选与 `sqrtm` 对比；
- `run_checks`：统一阈值断言；
- `main`：构造样例、执行计算、打印报告。

## R16

相关算法链路：

- Schur 分解法：适用于更一般矩阵的平方根；
- Denman-Beavers / Newton-Schulz：迭代式平方根（及逆平方根）算法；
- 矩阵幂函数 `A^alpha`：平方根是 `alpha=1/2` 的特例；
- 特征值分解、SVD 与矩阵函数近似（Padé/有理逼近）共同组成矩阵函数工具链。

## R18

源码级流程拆解（`demo.py` 主路径，8 步）：

1. `main` 生成两个 SPD 测试矩阵（一个手工、一个固定种子的随机构造）。  
2. 对每个样例调用 `analyze_case`，进入统一分析流程。  
3. `analyze_case` 先执行 `validate_spd_matrix`，确保输入满足 SPD 前提。  
4. 调用 `matrix_sqrt_spd_eigh`：先 `eigh(A)` 得到 `Q, Λ`，再构造 `Q sqrt(Λ) Q^T`。  
5. 对结果做对称化，计算 `X @ X` 与原矩阵的 Frobenius 相对残差。  
6. 计算 `X` 的对称误差，并记录输入矩阵最小/最大特征值。  
7. 若本机安装 SciPy，则额外调用 `sqrtm(A)` 作为参考，统计与主实现差异和虚部范数。  
8. `run_checks` 对关键指标进行阈值断言，全部通过后打印 `All checks passed.`。

说明：第三方 `sqrtm` 只用于验算，主算法不是黑盒调用，而是显式谱分解构造。
