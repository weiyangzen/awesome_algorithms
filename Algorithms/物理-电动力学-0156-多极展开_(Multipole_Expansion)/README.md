# 多极展开 (Multipole Expansion)

- UID: `PHYS-0155`
- 学科: `物理`
- 分类: `电动力学`

## R01

多极展开用于近似计算局域电荷分布在远场处产生的电势。  
对观测点 `r`（相对展开中心 `r0`）有：

`phi(r) = (1 / (4*pi*epsilon0)) * [ Q/R + (p·n)/R^2 + (1/2)*(n^T Q2 n)/R^3 + ... ]`

其中：

- `R = |r-r0|`，`n = (r-r0)/R`；
- `Q` 是总电荷（单极矩）；
- `p` 是偶极矩；
- `Q2` 是无迹四极矩张量。

## R02

该方法的核心价值是把 `N` 个源点的直接求和，转换为“少量矩 + 方向投影”的近似计算：

- 远场求值更快（特别是多观测点、固定源分布场景）；
- 物理解释更清晰（单极、偶极、四极分别代表不同对称性）；
- 是快速多极子法（FMM）、辐射理论、原子分子电势近似的重要基础。

## R03

本条目 MVP 目标：

- 构造一个离散点电荷分布；
- 用库仑定律计算“精确电势”作为参考；
- 分别计算单极、单极+偶极、单极+偶极+四极近似；
- 验证截断阶数提高时误差单调下降；
- 验证在更远球壳上四极截断误差继续减小。

## R04

`demo.py` 使用的公式：

1. 精确电势
   `phi_exact(r) = k * sum_a q_a / |r-r_a|`, `k = 1/(4*pi*epsilon0)`
2. 单极矩
   `Q = sum_a q_a`
3. 偶极矩（相对原点 `r0`）
   `p = sum_a q_a * x_a`, `x_a = r_a-r0`
4. 无迹四极矩
   `Q2_ij = sum_a q_a * (3*x_ai*x_aj - |x_a|^2*delta_ij)`
5. 截断到二阶（四极）势
   `phi_mp(r) = k * [Q/R + (p·n)/R^2 + (1/2)*(n^T Q2 n)/R^3]`

代码里 `max_order in {0,1,2}` 分别对应单极、偶极、四极截断。

## R05

设源点个数为 `N`，观测点个数为 `M`：

- 精确势计算：`O(M*N)`；
- 多极矩预处理：`O(N)`；
- 单次多极近似评估：`O(M)`。

在“同一源分布 + 大量观测点”下，多极近似可显著降低计算量。  
`demo.py` 同时计算精确解和近似解用于对比，因此总体仍为 `O(M*N)`。

## R06

脚本输出关键指标：

- `rel_l2_monopole / rel_l2_dipole / rel_l2_quadrupole`；
- `rel_max_monopole / rel_max_dipole / rel_max_quadrupole`；
- `rel_l2_quadrupole_shell_i`（不同半径球壳上的四极误差）；
- `min_far_field_ratio_r_over_a`（最小观测半径与源尺度比）。

并通过断言保证：

- 误差满足 `monopole > dipole > quadrupole`；
- 四极误差低于预设阈值；
- 观测半径增大时四极误差继续下降。

## R09

适用场景：

- 分子静电势远场近似；
- N 体静电计算中的远近分离；
- 课堂中展示“物理对称性 -> 矩展开”的计算意义。

不适用场景：

- 观测点靠近电荷源；
- 需要严格近场精度；
- 高频时变电磁辐射的完整求解。

## R10

正确性直觉：

1. 当 `R >> a`（`a` 为源尺度）时，`1/|r-r_a|` 可按 `a/R` 展开；
2. 低阶项先捕捉总量与主导方向性（`Q`、`p`）；
3. 四极项修正二阶角向结构；
4. 因此阶数越高，远场误差应越小，且半径越大误差越小。

## R11

数值策略：

- 固定随机种子在多个球壳上采样观测点，保证可复现；
- 先算精确解，再算不同截断阶近似；
- 用全局 `L2` 相对误差和最大相对误差双指标评估；
- 再做“分球壳”误差，验证随半径增大而收敛。

稳健性处理：

- 显式检查观测点与源点/展开中心的最小距离，避免除零；
- 所有输入转换为 `float64`，减少舍入误差。

## R12

关键参数与影响：

- `charges, positions`：决定矩大小和对称性；
- `origin`：展开中心；选得不好会放大高阶项；
- `shell_radii`：越大越满足远场条件；
- `points_per_shell`：越大误差统计越稳定；
- `max_order`：截断阶数，越高精度通常越好但代价增加。

调参建议：

- 若误差不降，优先检查 `min_far_field_ratio_r_over_a` 是否过小；
- 把展开中心放在电荷分布几何中心附近，通常可减小高阶项。

## R14

常见失效模式：

1. 把四极张量公式写错（例如漏掉无迹修正）；
2. 近场直接用低阶截断，导致误差异常大；
3. 观测点与电荷点太近触发数值发散；
4. 展开中心选择不当，使低阶展开“看起来不收敛”；
5. 用点对点相对误差除以接近零的参考值导致假性爆炸。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `compute_multipole_moments` 读取离散电荷 `q_a, r_a`，相对展开中心计算 `Q`、`p`、`Q2`。  
2. `compute_multipole_moments` 同时计算 `source_radius`，用于衡量远场条件 `R/a`。  
3. `sample_observation_points` 在多个球壳半径上生成确定性随机方向样本。  
4. `exact_potential` 直接执行 `sum_a q_a/|r-r_a|` 得到参考真值。  
5. `multipole_potential(max_order=0/1/2)` 按解析公式显式拼接单极、偶极、四极项。  
6. `error_metrics` 分别计算 `L2` 相对误差和最大相对误差，量化截断质量。  
7. `run_demo` 汇总全局误差与分球壳误差，形成“随阶数/半径变化”的证据链。  
8. `main` 执行断言（阶数越高误差越小、半径越大误差越小），确保脚本可非交互自动验证。

说明：`numpy` 仅提供数组和线性代数基础算子；多极矩定义、势函数拼装、误差判据均在源码中逐步展开，没有把核心算法交给黑盒库。
