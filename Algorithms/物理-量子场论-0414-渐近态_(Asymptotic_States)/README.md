## R01

渐近态（Asymptotic States）指散射过程在 `t -> -∞`（入射）与 `t -> +∞`（出射）时可用“自由粒子态”描述的极限态。量子场论里它是 S 矩阵理论的基础：

- 入态 `|alpha, in>`：远过去看起来像自由多粒子态；
- 出态 `|beta, out>`：远未来看起来像自由多粒子态；
- 相互作用只在有限时空区域显著，远离后态重新近似自由演化。

## R02

本条目目标：给出一个最小可运行数值 MVP，把“渐近态”从抽象定义变成可计算、可检查的流程。

做法：

1. 用 1D 波包散射作为 QFT 渐近态的教学类比；
2. 哈密顿量取 `H = H0 + V(x)`，其中 `V(x)` 局域在原点附近；
3. 直接演化波函数并检查：
   - 早期/晚期在相互作用区概率很小（近似自由）；
   - 晚期动量分布几乎不再变化（“出射渐近态”稳定）。

## R03

要解决的问题：如何判断一个数值散射态已进入“渐近区”。

`demo.py` 的输入输出（无交互）如下：

- 输入（脚本内固定参数）：
1. 空间网格与时间步长（`n_grid=2048`, `dt=0.02`, `total_time=120`）；
2. 初始高斯入射波包（`x0=-100`, `sigma=8`, `k0=1.5`）；
3. 局域高斯势（`V0=0.8`, `width=3`）；
4. 采样时刻 `t in {0, 40, 70, 90, 100, 110, 120}`。
- 输出：
1. 每个采样时刻的 `norm`、左右/相互作用区概率；
2. 相对最终态的动量分布总变差距离 `TV(momentum, final)`；
3. 最终 `k>0` 与 `k<0` 的动量通道概率；
4. 断言检查通过后输出 `All checks passed.`。

## R04

MVP 的数学模型：

1. 薛定谔演化
`i d/dt psi(x,t) = [ -1/(2m) d^2/dx^2 + V(x) ] psi(x,t)`。

2. 分裂算符（Strang splitting）
`U(dt) ~= exp(-iV dt/2) exp(-iT dt) exp(-iV dt/2)`，
其中 `T = p^2/(2m)` 在动量空间对角。

3. 渐近态判据（数值版）
- 空间判据：`P_int(t) = ∫_{|x|<=R} |psi|^2 dx` 在早期和晚期都很小；
- 动量判据：晚期 `n(k,t)=|phi(k,t)|^2` 几乎不再随时间变化。

这对应 QFT 语境下“远离相互作用区后，态可由自由哈密顿量描述”。

## R05

复杂度（`N` 为空间网格点数，`M` 为时间步数）：

- 每步一次 FFT + IFFT：`O(N log N)`；
- 总时间复杂度：`O(M N log N)`；
- 空间复杂度：`O(N)`（波函数、势能、相位因子等）。

在本配置下 `N=2048`, `M=6000`，可在普通 CPU 上快速完成。

## R06

算法流程（高层）：

1. 建立均匀空间网格与对应 FFT 动量网格；
2. 构造归一化入射高斯波包；
3. 构造局域高斯势 `V(x)`；
4. 预计算 `exp(-iV dt/2)` 与 `exp(-i k^2 dt/(2m))`；
5. 用 split-operator 推进到终时并保存若干快照；
6. 计算每个快照的空间通道概率与动量分布；
7. 用晚期动量分布漂移与相互作用区概率判断是否达到渐近态；
8. 输出表格并做断言验证。

## R10

正确性直觉：

1. split-operator 在小 `dt` 下近似幺正，范数应接近守恒；
2. 当 `|x|` 远离局域势时，`V(x) ~ 0`，演化近似自由；
3. 因此晚期动量分布应趋于稳定（仅剩整体相位转动）；
4. 若 `P_int(0)` 与 `P_int(T)` 都很小，而中间显著增大，说明发生了局域相互作用并最终离开。

## R11

数值稳定性策略：

- 使用 Strang splitting（`V/2 -> T -> V/2`）降低时间离散误差；
- 显式归一化初态并监控每个采样时刻 `norm`；
- 动量密度按 `dk` 重新归一，减小离散 FFT 常数偏差；
- 检查快照索引完整性，防止因为舍入漏采样。

## R12

关键参数与影响：

- `dt`：过大将增大 Trotter 误差；
- `n_grid` 与区间长度：决定分辨率及是否出现边界回卷；
- `packet_k0`、`V0`、`potential_width`：决定反射/透射比例；
- `total_time`：必须足够长，才能观测到晚期稳定动量分布；
- `interaction_radius`：用于定义“是否在相互作用区内”。

调参经验：先固定边界安全裕量，再调 `dt` 与 `total_time`，最后调势能参数以得到可见散射信号。

## R14

常见失效模式：

1. `dt` 过大：范数漂移或相位误差明显；
2. 空间区间太短：波包触边后因 FFT 周期性回卷，污染结果；
3. `total_time` 太短：还没达到渐近区就提前读出；
4. `interaction_radius` 选取不合理：误判是否“远离相互作用区”；
5. 势能过强或过弱：导致演示过于极端或几乎无散射特征。

## R18

`demo.py` 源码级算法流程（8 步）：

1. `Config` 固定网格、时间步、初态和势能参数，形成可复现实验设置。 
2. `make_grid` 生成 `x` 与 FFT `k` 网格；`gaussian_wavepacket` 构造并归一化入射波包。 
3. `localized_potential` 构造局域高斯势 `V(x)`，保证相互作用只在原点附近显著。 
4. `evolve_split_operator` 预计算 `exp(-iVdt/2)` 与 `exp(-ik^2dt/(2m))`，在循环中按 `V/2 -> T -> V/2` 推进，并在目标步保存快照。 
5. `momentum_density` 把每个快照做 FFT，得到 `n(k)=|phi(k)|^2`；`channel_probabilities` 计算左/相互作用区/右通道概率。 
6. 主流程中以终时快照作为参考，计算各时刻相对终时的动量总变差 `TV(momentum, final)`。 
7. 输出 `pandas` 表格，报告 `norm`、空间通道概率和动量稳定性，并额外打印最终 `k>0` 与 `k<0` 通道占比。 
8. 断言验证三件事：范数守恒、初末相互作用区概率很小、晚期动量分布稳定（且中期明显不同于最终态），据此确认“渐近态”数值闭环成立。 

说明：核心算法（分裂算符传播、通道分解、稳定性判据）都在源码中逐步展开，没有将关键物理流程外包给第三方黑盒。
