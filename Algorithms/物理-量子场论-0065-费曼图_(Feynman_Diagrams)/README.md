## R01

费曼图是量子场论微扰展开的可计算记账系统：
把相互作用项按耦合常数展开后，每一阶贡献都可以映射成图，并通过“顶点因子 + 传播子 + 动量守恒”得到散射振幅项。

它不是单一公式，而是一套从拉氏量到可计算振幅的算法流程。本条目提供最小可运行 MVP，聚焦 `phi^3` 理论下 `2->2` 树图（`s/t/u` 三道）生成与数值求值。

## R02

本条目要解决的问题：

- 如何把“费曼图规则”从概念落到可执行代码，而不是停留在符号推导；
- 如何显式枚举 `2->2` 树级 `phi^3` 图拓扑（`s/t/u`），避免黑盒；
- 如何在固定运动学点上计算各图贡献并合成为总振幅。

MVP 目标：

- 构造一组中心质心系四动量并计算 Mandelstam 变量 `s,t,u`；
- 显式列出三张树图及其内部线动量平方 `q^2`；
- 依据费曼规则计算每道贡献 `M_c ~ g^2/(q^2-m^2+i*eps)`；
- 输出 `pandas` 表格并做基础一致性断言。

## R03

`demo.py` 的输入输出约定（无交互）：

- 输入（脚本内固定参数）：
1. 质量 `m=1.0`；
2. 耦合 `g=0.8`；
3. 质心系单粒子能量 `E=2.0`；
4. 散射角 `theta=60 deg`；
5. 传播子极点位移 `eps=1e-9`。
- 输出：
1. 运动学表（`s,t,u,s+t+u,4m^2,identity_error`）；
2. 图级贡献表（通道、外腿分组、`q^2`、传播子与振幅实虚部）；
3. 汇总表（约化总振幅与完整总振幅）；
4. 断言全部通过后打印 `All checks passed.`。

## R04

MVP 使用的核心公式：

1. 交互项（约定）
`L_int = -(g/3!) phi^3`。

2. 费曼规则（标量场）
- 顶点因子：`-i g`；
- 内线传播子：`i/(q^2 - m^2 + i*eps)`。

3. `2->2` 树图有三道：`s,t,u`。

4. 对应约化振幅（去掉公共 `-i` 因子）
`M_tilde = sum_{c in {s,t,u}} g^2/(c - m^2 + i*eps)`。

5. 完整振幅
`M = -i * M_tilde`。

`demo.py` 逐图实现上述规则，不调用专用 QFT 黑盒库。

## R05

复杂度分析（外腿数固定为 4）：

- 图枚举：常数规模（3 张树图），`O(1)`；
- 每图传播子与振幅计算：`O(1)`；
- 总时间复杂度：`O(1)`；
- 空间复杂度：`O(1)`（仅少量向量和表格）。

若推广到更高点函数，树图数量随外腿数呈组合增长，主成本来自拓扑枚举与代数化简。

## R06

MVP 算法闭环：

1. 在质心系构造四个外线四动量 `p1..p4`；
2. 计算 Mandelstam 变量 `s,t,u`；
3. 枚举 `phi^3` 的 `s/t/u` 三个树图拓扑；
4. 给每个通道绑定内部线 `q^2`；
5. 用标量传播子计算每图约化振幅；
6. 聚合得到约化总振幅与完整总振幅；
7. 输出运动学表与图级贡献表；
8. 断言关键物理恒等式与数值一致性。

## R10

正确性直觉：

1. `phi^3` 的 `2->2` 树级连通图在拓扑上仅有 `s/t/u` 三道；
2. 每张图都由两个顶点和一条内部线组成，耦合阶均为 `g^2`；
3. 每道只需替换内部线动量平方 `q^2 in {s,t,u}` 即可得到对应项；
4. 总振幅即三道代数和，因此“枚举正确 + 规则正确”即可得到正确树级结果。

## R11

数值稳定策略：

- 传播子分母加入 `i*eps`，避免极点处分母为零；
- 统一使用 `float64/complex128`，减少舍入误差；
- 对 `s+t+u=4m^2` 做阈值断言，验证运动学构造无误；
- 对分道和总和做一致性断言，防止表格拼接或索引错误。

## R12

关键参数与影响：

- `m`：决定内部线极点位置 `q^2=m^2`；
- `g`：振幅整体缩放约为 `g^2`；
- `E`：控制 `s` 的能区与传播子远离/接近极点程度；
- `theta`：主要影响 `t,u`，改变角分布结构；
- `eps`：影响极点附近的虚部正则化强度。

调参建议：

- 教学展示可先用远离极点参数，避免巨大数值；
- 若想观察共振附近行为，可让某道 `q^2` 接近 `m^2` 并适当增大 `eps`；
- 若只关心通道相对权重，可固定 `g=1` 便于阅读。

## R14

常见失效模式：

1. 度规符号写错（把 `(+,-,-,-)` 写成全正）导致 `s,t,u` 全部错误；
2. `p3,p4` 构造不满足动量守恒，引发恒等式断言失败；
3. 漏掉某个通道（只算 `s,t`）导致总振幅偏差；
4. 传播子误写为 `1/(q^2+m^2)` 改变极点结构；
5. 忘记复数 `i*eps` 使接近极点时数值不稳定。

## R18

`demo.py` 的源码级算法流程（9 步）：

1. `build_com_momenta` 在给定 `E,m,theta` 下构造 `p1..p4`，满足动量守恒。  
2. `minkowski_dot` 与 `mandelstam_variables` 用 `(+,-,-,-)` 度规计算 `s,t,u`。  
3. `enumerate_phi3_2to2_diagrams` 显式返回三张树图拓扑及其外腿分组。  
4. `channel_q2` 将每张图映射到内部线不变量 `q^2`（`s/t/u`）。  
5. `scalar_propagator` 计算 `1/(q^2-m^2+i*eps)` 的复传播子。  
6. `reduced_diagram_amplitude` 生成每图约化振幅 `g^2 * propagator`。  
7. `evaluate_diagrams` 逐图汇总到 `pandas` DataFrame，并累加约化总振幅。  
8. `main` 组装运动学表、图级表和总振幅表，打印关键结果。  
9. `main` 最后执行三类断言（通道完整性、`s+t+u` 恒等式、分道求和一致性），通过后输出 `All checks passed.`。

说明：本实现没有把核心图枚举与振幅计算交给第三方 QFT 黑盒；每个计算步骤都在源码中可追踪。
