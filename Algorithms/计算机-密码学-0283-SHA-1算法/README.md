## R01

`SHA-1`（Secure Hash Algorithm 1）是一种经典密码学哈希函数，输入任意长度字节串，输出固定 160-bit（20 字节）摘要。  
它采用 Merkle-Damgard 结构，以 512-bit 分组迭代压缩，历史上广泛用于签名、完整性校验和版本控制系统。

## R02

本条目目标是实现一个“可运行、可审阅、非黑盒”的最小 SHA-1 MVP：
- 在 `demo.py` 中从源码实现填充、消息扩展、80 轮压缩；
- 给出标准测试向量与 `hashlib.sha1` 的交叉验证；
- 保证 `uv run python demo.py` 无交互直接运行。

## R03

核心思想：
- 先把消息补齐到 512-bit 分组边界，并附加原始比特长度；
- 每个分组扩展为 `80` 个 32-bit 字后进入压缩函数；
- 压缩函数维护 5 个 32-bit 工作寄存器 `(a,b,c,d,e)`，迭代 80 轮；
- 最终输出 `H0..H4` 级联后的 160-bit 摘要。

## R04

算法关键参数：
- 分组长度：`512 bit`（64 字节）；
- 输出长度：`160 bit`（20 字节）；
- 初始向量：
  - `H0=0x67452301`
  - `H1=0xEFCDAB89`
  - `H2=0x98BADCFE`
  - `H3=0x10325476`
  - `H4=0xC3D2E1F0`
- 轮常量：
  - `0..19: 0x5A827999`
  - `20..39: 0x6ED9EBA1`
  - `40..59: 0x8F1BBCDC`
  - `60..79: 0xCA62C1D6`

## R05

输入输出定义（对应 `demo.py`）：
- `sha1_digest(data: bytes) -> bytes`：返回 20 字节摘要；
- `sha1_hexdigest(data: bytes) -> str`：返回 40 字符十六进制摘要；
- `main()`：执行自检并打印样例摘要。

## R06

实现约束与数据表示：
- 全部使用 Python 标准库与原生整数，避免第三方黑盒；
- 32-bit 运算通过 `& 0xFFFFFFFF` 显式截断；
- 字节序使用 big-endian（符合 SHA-1 规范）；
- 补位时长度字段采用 64-bit 无符号整数（模 `2^64` 语义）。

- `_self_test_known_vectors`：标准向量校验；

## R08

复杂度分析：
- 设消息长度为 `n` 字节；
- 填充后分组数约为 `ceil((n+9)/64)`；
- 每个分组固定 80 轮，因此总时间复杂度为 `O(n)`；
- 额外工作内存主要是 `80` 个 32-bit 字与少量寄存器，空间复杂度为 `O(1)`（不计输入输出）。

## R09

正确性要点（直观）：
- 填充规则保证“不同长度消息”的块序列可区分（尾部编码原始长度）；
- 每轮使用布尔函数、常量与旋转操作推进状态，确保每组消息字影响扩散到全部状态字；
- 分组压缩后做链式累加（feed-forward），使最终摘要同时依赖历史所有分组。

## R13

安全注意事项：
- SHA-1 已被实用碰撞攻击击破，不应再用于新系统中的抗碰撞场景；
- 生产环境应优先选择 `SHA-256` / `SHA-3` / `BLAKE2` 等更安全方案；
- 本条目仅用于算法教学与历史理解，不作为安全设计推荐。

## R14

与相近算法比较（简述）：
- 对比 SHA-256：SHA-256 输出更长（256 bit）且安全余量显著高于 SHA-1；
- 对比 MD5：SHA-1 曾被视为强于 MD5，但二者现均不满足现代抗碰撞需求；
- 对比 SHA-3：SHA-3 采用海绵结构，与 SHA-1 的 Merkle-Damgard 路径不同。

## R18

源码级算法流程拆解（对应 `demo.py`，无第三方黑盒）：
1. 读取输入消息 `data`，计算原始比特长度 `len(data) * 8`。  
2. 执行填充：追加 `0x80`，再追加若干 `0x00` 使长度满足 `mod 64 == 56`，最后追加 8 字节长度字段。  
3. 将填充后的字节流按 64 字节分块；每块解析成 16 个 big-endian 的 32-bit 字。  
4. 通过 `W[t] = rol1(W[t-3] XOR W[t-8] XOR W[t-14] XOR W[t-16])` 把 16 个字扩展到 80 个消息字。  
5. 初始化工作寄存器 `a,b,c,d,e` 为当前链值 `H0..H4`。  
6. 进入 80 轮压缩：按轮区间选择布尔函数 `f` 与常量 `k`，计算 `temp = rol5(a) + f + e + k + W[t] (mod 2^32)` 并更新寄存器。  
7. 块结束后把寄存器回加到链值：`H0+=a ... H4+=e (mod 2^32)`，进入下一块。  
8. 所有块处理完后，将 `H0..H4` 以 big-endian 拼接成 20 字节摘要，并提供 hex 输出与自检比对。
