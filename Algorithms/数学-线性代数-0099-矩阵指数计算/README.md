## R01

矩阵指数定义为

`exp(A) = e^A = sum_{k=0}^{\infty} A^k / k!`。

它把标量指数函数推广到方阵，是线性常微分方程 `x'(t)=Ax(t)` 的闭式解核心：`x(t)=exp(tA)x(0)`。

## R02

简要背景：

- 矩阵函数理论在 20 世纪逐步系统化；
- `exp(A)` 是最常用且最“工程化”的矩阵函数之一；
- 数值实现从幂级数直接截断发展到更稳定高效的“缩放-平方 + 有理逼近（Padé）”路线；
- SciPy、MATLAB 等科学计算库都以该类方法为主。

## R03

本条目 MVP 的任务定义：

- 输入：一个有限值的实/复方阵 `A`；
- 输出：矩阵 `E`，近似 `exp(A)`；
- 主算法：`Scaling and Squaring + Padé(13)`（自实现，不把第三方当黑盒）；
- 验证：
  - 与已知闭式样例对比（零矩阵、对角矩阵、二维旋转生成元、幂零矩阵）；
  - 检查 `exp(A)exp(-A)≈I`；
  - 检查 `det(exp(A))≈exp(trace(A))`；
  - 若 SciPy 可用，额外对比 `scipy.linalg.expm`。

## R04

算法概览（Padé(13) 路线）：

1. 计算 `||A||_1`；
2. 依据阈值 `theta_13` 选择缩放次数 `s`，令 `B = A / 2^s`；
3. 对 `B` 构造 `[13/13]` Padé 有理逼近 `R(B) = (V-U)^{-1}(V+U)`；
4. 通过重复平方恢复尺度：`E = R(B)` 连续平方 `s` 次得到 `exp(A)`。

## R05

复杂度分析（稠密 `n x n`）：

- 时间复杂度：`O(n^3)`，主耗时来自矩阵乘法和一次线性方程求解；
- 空间复杂度：`O(n^2)`，主要存储若干中间矩阵（`A2/A4/A6/U/V` 等）。

## R06

微型示例（幂零矩阵可手算）：

令

`N = [[0,1],[0,0]]`，有 `N^2=0`。

则

`exp(N) = I + N`（因为二次及以上幂全为零）。

该例在 `demo.py` 中作为可精确校验的基准之一。

## R07

算法意义：

- 连续时间线性系统离散化：`x_{k+1}=exp(hA)x_k`；
- 马尔可夫过程、控制、量子演化、图扩散等领域的核心算子；
- 是其他矩阵函数（`log(A)`、`sin(A)`、`cos(A)`）数值工具链的基础节点。

## R08

关键理论性质：

- 幂级数绝对收敛，`exp(A)` 对任意方阵都存在；
- 若 `A` 与 `B` 可交换（`AB=BA`），则 `exp(A+B)=exp(A)exp(B)`；
- `exp(A)` 总可逆，且逆为 `exp(-A)`；
- 恒等式：`det(exp(A)) = exp(trace(A))`；
- 相似变换协变：`exp(P^{-1}AP)=P^{-1}exp(A)P`。

## R09

适用边界与局限：

适用：

- 中小规模稠密矩阵；
- 需要稳定、可解释、无需外部交互的 MVP；
- 需要可复现实验和多种性质校验。

局限：

- 大规模稀疏场景更适合 Krylov/`expm_multiply` 一类“算子作用”方法；
- 极端病态矩阵上误差仍可能被放大；
- 本 MVP 聚焦通用正确性，不追求库级极限性能优化。

## R10

正确性说明（工程版）：

1. 对缩放矩阵 `B=A/2^s`，Padé(13) 构造 `R(B)` 逼近 `exp(B)`；
2. 当 `||B||` 足够小，Padé 截断误差很低（高阶有理逼近）；
3. 由指数半群性质：`exp(A)=exp(B)^{2^s}`；
4. 代码用“重复平方”把 `R(B)` 提升回 `A` 的尺度；
5. 通过多个可解析样例和结构恒等式（逆、行列式-迹关系）进行数值验算，形成闭环。

## R11

数值稳定性要点：

- 先缩放再逼近，避免在大范数矩阵上直接逼近导致误差放大；
- 用 `numpy.linalg.solve(V-U, V+U)` 代替显式求逆；
- 对阈值附近样例，`s` 自动增大，平方阶段承担放大；
- 结果质量由多指标联合评估，而非单一误差。

## R12

性能视角：

- 对 `n<=几百` 的稠密矩阵，Padé(13) 是成熟稳健的默认选择；
- 与“高阶 Taylor 直接累加”相比，Padé 在同精度下通常需要更少项；
- 与黑盒 `expm` 比较时，本实现强调“源码可解释”和“最小依赖可运行”。

## R14

鲁棒性与异常处理：

- 非二维、非方阵、含 `nan/inf`：抛 `ValueError`；
- 线性方程求解若遇极端数值问题，会抛 `LinAlgError`；
- 校验阈值集中定义，便于后续按设备精度统一调整。

## R15

实现结构（`demo.py` 对应关系）：

- `validate_square_matrix`：输入合法性检查；
- `pade13_approximant`：构造 `U/V` 并求解 Padé 有理近似；
- `matrix_exponential_pade13`：缩放-平方主流程；
- `analyze_case`：收集每个样例误差与结构指标；
- `run_checks`：统一阈值断言；
- `main`：构造样例并串行执行、打印报告。

## R16

相关算法链路：

- Schur-Parlett：一般矩阵函数的通用途径；
- Krylov/Arnoldi `expm_multiply`：大规模稀疏 `exp(A)v`；
- Chebyshev/有理近似：特定谱区间上的加速方法；
- 对角化法：仅在可稳定对角化时简洁，但通用性弱于 Padé 路线。

## R18

源码级流程拆解（`demo.py` 主路径，8 步）：

1. `main` 构造 5 个固定测试矩阵，并逐个调用 `analyze_case`。  
2. `analyze_case` 先执行 `validate_square_matrix`，拒绝非方阵或非有限值输入。  
3. 调用 `matrix_exponential_pade13`：先计算 `||A||_1`，依据 `theta_13` 决定缩放次数 `s`，得到 `B=A/2^s`。  
4. `matrix_exponential_pade13` 内部调用 `pade13_approximant(B)`，构造 `A2/A4/A6` 与 Padé 系数多项式组合，形成 `U`、`V`。  
5. 通过求解线性系统 `(V-U)X=(V+U)` 得到 `X≈exp(B)`，避免显式逆矩阵。  
6. 对 `X` 连续做 `s` 次平方，恢复到 `exp(A)` 的尺度，返回结果与缩放信息。  
7. `analyze_case` 计算闭式对比误差（若有）、`exp(A)exp(-A)-I` 误差、`det(exp(A))` 与 `exp(trace(A))` 相对误差，以及可选 SciPy 差异。  
8. `run_checks` 对上述指标做阈值断言，全部样例通过后打印 `All checks passed.`。  

说明：`scipy.linalg.expm` 仅作“对照验算”，主计算路径完全由本地源码实现。
