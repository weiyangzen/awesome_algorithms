# 微扰理论 (Perturbation Theory)

## R01

微扰理论用于求解“可解主问题 + 小扰动”结构的量子系统。将哈密顿量写为：

`H = H0 + lambda * V`

- `H0`：可解析求本征值/本征态的未扰动哈密顿量
- `V`：扰动项
- `lambda`：无量纲小参数（`|lambda| << 1`）

非简并定态微扰理论中，能量按幂级数展开：

`En = En^(0) + lambda En^(1) + lambda^2 En^(2) + ...`

## R02

典型应用场景：

- 外电场/磁场较弱时原子能级修正（Stark/Zeeman 的弱场近似）
- 势阱中小势能修饰导致的本征值偏移
- 量子化学中把复杂相互作用视作参考哈密顿量的校正
- 数值计算中作为“精确对角化前的快速估计”与误差基线

## R03

非简并、时间无关微扰的标准公式（`|n^(0)>` 为 `H0` 本征态）：

- 一阶能量修正：
  `En^(1) = <n^(0)|V|n^(0)>`
- 二阶能量修正：
  `En^(2) = sum_{m!=n} |<m^(0)|V|n^(0)>|^2 / (En^(0) - Em^(0))`
- 一阶态修正：
  `|n^(1)> = sum_{m!=n} <m^(0)|V|n^(0)> / (En^(0) - Em^(0)) * |m^(0)>`

本条目 `demo.py` 选择有限维 Hermitian 矩阵模型，直接实现上述公式并与精确对角化比较。

## R04

直观理解：

- `H0` 给出“主导结构”，`V` 只做小幅扭曲
- 一阶修正由“本态对自身的平均扰动”决定（对角元）
- 二阶修正来自“与其他能级的虚跃迁耦合”（非对角元 + 能隙分母）
- 能隙越小，二阶项可能越大，说明级数更容易失效

## R05

正确性要点：

1. 公式来源于将本征方程 `(H0 + lambda V)|n> = En|n>` 做同阶收集。
2. 一阶能量只取 `V` 在未扰动本征基下的对角元。
3. 二阶能量的分母 `En^(0)-Em^(0)` 要求非简并，否则需改用简并微扰理论。
4. 当 `lambda` 足够小时，二阶近似应比一阶近似更接近精确谱。
5. `demo.py` 用 `numpy.linalg.eigh` 计算“精确值”作为数值对照，验证收敛趋势。

## R06

复杂度（系统维度 `N`）：

- 基变换 `V0 = U0^dagger V U0`：`O(N^3)`
- 一阶能量：读取对角元 `O(N)`
- 二阶能量：双重循环求和 `O(N^2)`
- 精确对角化（Hermitian）：`O(N^3)`

因此主开销通常在对角化步骤，微扰修正本身较轻量。

## R07

标准算法流程：

1. 输入 Hermitian 的 `H0` 与 `V`。
2. 求 `H0` 的本征对 `(E0, U0)` 并检查非简并性（最小能隙阈值）。
3. 将 `V` 变换到未扰动本征基：`V0 = U0^dagger V U0`。
4. 计算 `dE1 = diag(V0)`。
5. 按二阶公式累加得到 `dE2`。
6. 对每个 `lambda` 形成 `E^(1)`、`E^(2)` 近似。
7. 精确对角化 `H(lambda)=H0+lambda V` 得到 `E_exact`。
8. 评估 `max|E_approx - E_exact|` 与态保真度。

## R08

`demo.py` 的 MVP 设计：

- 依赖：`numpy`
- 模型：4 维非简并实对称（Hermitian）矩阵
- 输出：
  - 多个 `lambda` 下一阶/二阶能量近似误差
  - 指定 `lambda` 下的一阶态近似与精确本征态保真度
- 无交互输入，运行即打印结果

## R09

`demo.py` 主要函数：

- `build_problem()`: 构造 `H0` 与 `V`
- `assert_hermitian(matrix)`: 检查 Hermitian 约束
- `unperturbed_spectrum(H0)`: 求 `E0, U0` 与最小能隙
- `first_order_energy_shift(V0)`: 一阶能量系数
- `second_order_energy_shift(E0, V0)`: 二阶能量系数
- `perturbative_energies(E0, dE1, dE2, lambda_, order)`: 生成近似能级
- `exact_eigensystem(H0, V, lambda_)`: 精确对角化
- `first_order_state_vector(n, E0, V0, lambda_)`: 一阶态近似
- `state_fidelity(a, b)`: 态保真度

## R10

测试策略：

- 结构测试：`H0`、`V` 必须是 Hermitian
- 理论前提测试：`H0` 的最小能隙需大于阈值，避免误用非简并公式
- 数值正确性测试：检查二阶近似误差通常不大于一阶近似误差
- 态近似测试：一阶态与精确态应有较高保真度（在小 `lambda` 下）

## R11

边界条件与异常处理：

- 若输入矩阵非 Hermitian，立即 `ValueError`
- 若最小能隙过小（近简并），立即 `ValueError`
- 若二阶公式分母过小（`|En^(0)-Em^(0)| < atol`），立即 `ValueError`
- 若 `order` 不是 1 或 2，立即 `ValueError`

## R12

与相关方法关系：

- 与精确对角化：微扰理论是近似法，精确对角化是基准法
- 与简并微扰理论：本条目只覆盖非简并，简并情形需先在简并子空间对角化扰动
- 与时间依赖微扰理论：本条目是定态（时间无关）版本，不涉及跃迁概率的时域积分公式

## R13

本示例参数选择：

- 未扰动能级：`E0 = [0.0, 1.0, 2.3, 3.7]`
- 扰动矩阵 `V`：含对角与非对角项，体现一阶与二阶共同贡献
- 扫描参数：`lambda in [0.00, 0.02, 0.05, 0.08, 0.12]`

该参数组保证非简并且扰动适中，能清楚展示“二阶优于一阶”的趋势。

## R14

工程实现注意点：

- 二阶项对小能隙敏感，需明确阈值防止数值爆炸
- 本征向量存在整体相位不定性，比较态时应使用保真度而非逐分量差
- `eigh` 返回按升序排列的本征值，本例在小 `lambda` 区间内按索引对齐可行

## R15

最小示例解读：

- 一阶能量近似主要由 `V` 的对角元驱动
- 二阶能量近似引入“耦合项平方 / 能隙”后，通常显著降低误差
- 一阶态近似在 `lambda=0.08` 下应与精确态保持高重叠（保真度接近 1）

## R18

源码级算法流程拆解（对应 `demo.py`，共 8 步）：

1. **构造模型并做物理约束检查**  
   `build_problem` 生成 `H0` 与 `V`，`assert_hermitian` 保证两者可用于量子哈密顿量。

2. **求未扰动本征基并校验非简并前提**  
   `unperturbed_spectrum` 调用 `numpy.linalg.eigh(H0)` 得到 `E0, U0`，并检查最小能隙。

3. **将扰动映射到未扰动本征基**  
   计算 `V0 = U0^dagger V U0`，之后所有微扰公式都在该基下执行。

4. **按显式公式计算一阶/二阶能量系数**  
   `first_order_energy_shift` 取 `diag(V0)`；`second_order_energy_shift` 双循环累加 `|V0[m,n]|^2/(E0[n]-E0[m])`。

5. **形成有限阶近似能级并进行参数扫描**  
   `perturbative_energies` 生成一阶和二阶近似，对 `lambda` 网格逐点计算。

6. **精确谱基线：`eigh` 的源级流程（LAPACK 路线）**  
   对 `H(lambda)` 调用 `numpy.linalg.eigh`，底层通常走 Hermitian 特化例程：
   - 先用 Householder 变换把矩阵约化为三对角形式  
   - 再用分治/QR 等特征值算法求三对角谱  
   - 回代特征向量并做正交归一化  
   - 最终返回升序本征值与对应本征向量

7. **误差度量输出**  
   对每个 `lambda` 计算 `max|E^(1)-E_exact|` 与 `max|E^(2)-E_exact|`，验证二阶改进效果。

8. **态近似质量验证**  
   `first_order_state_vector` 构造 `|n^(0)> + lambda|n^(1)>` 并归一化，再与精确态计算保真度 `|<psi_approx|psi_exact>|^2`。
