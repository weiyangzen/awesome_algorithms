## R01

功率谱 `P(k)` 用于刻画随机密度场在不同空间尺度（波数 `k`）上的方差分布，是连接“初始涨落”与“大尺度结构统计”的核心对象。  
对零均值密度对比度场 `delta(x)`，在统计各向同性条件下可写为：

`<delta_k delta_k'* > = (2pi)^3 delta_D(k-k') P(k)`

本条目的 MVP 聚焦于“从三维离散密度场估计各向同性 `P(k)` 并回归谱斜率”。

## R02

实现目标是构建一个可重复、可审计、无交互的最小流程：

1. 在周期盒中生成三维高斯随机密度场；
2. 在傅里叶空间注入目标谱形 `P_in(k)`；
3. 用壳平均估计离散 `P_hat(k)`；
4. 在大尺度区间拟合 `log P - log k` 斜率；
5. 通过断言验证恢复谱斜率与输入一致（允许统计噪声）。

## R03

MVP 的物理与数值假设：

- 统计均匀、统计各向同性；
- 周期边界立方盒；
- 只关注标量谱 `P(k)`（不含各向异性多极展开）；
- 使用高斯随机场示例，不包含非线性引力塌缩耦合；
- 输入谱用简化模型 `A*(k/k0)^n*exp(-(k/kc)^2)`，用于教学与验证，不替代 CLASS/CAMB 级精密谱。

## R04

关键公式（与 `demo.py` 一一对应）：

1. 波数网格：
`k_i = 2pi * fftfreq(N, d=L/N)`

2. 输入谱模型：
`P_in(k) = A * (k/k0)^n * exp(-(k/kc)^2), k>0`

3. 颜色滤波：
`delta_k^colored = delta_k^white * sqrt(P_in(k))`

4. FFT 归一化下的谱估计：
`P_hat(k_grid) = (L^3 / N^6) * |FFT[delta(x)]|^2`

5. 壳平均：把 `|k|` 落入同一 bin 的模式取均值，得到一维 `P_hat(k)`。

6. 斜率拟合：
`log P_hat = c + n_fit * log k`（大尺度区间线性回归）。

## R05

`demo.py` 的算法设计：

1. 用 `numpy.fft.fftfreq` + `meshgrid` 构建三维 `|k|` 网格；
2. 生成实空间白噪声场 `delta_white`；
3. 对其 FFT 后乘 `sqrt(P_in(k))` 注入目标谱形；
4. `ifftn` 回到实空间得到合成密度场，并做零均值、单位方差标准化；
5. 再次 FFT 并计算 `P_hat(k_grid)`；
6. 对 `k` 做几何分箱并在每个球壳上求平均，得各向同性 `P_hat(k)`；
7. 在低 `k` 区间拟合 `n_fit`，并计算 `R^2`；
8. 对比输入模型形状（只差一个全局振幅），输出 `log-RMSE` 与分位比值。

## R06

脚本输出包含三部分：

1. 全局摘要：网格、盒尺度、输入谱参数；
2. 诊断指标：
- `n_fit` 与输入 `n_in`；
- 线性拟合 `R^2`；
- 谱形匹配 `alpha`、`log-RMSE`、`ratio[p10,p50,p90]`；
3. 采样表：若干 `k` 节点上的 `Pk`、`P_target(k)` 与比值。

并在运行时执行断言，失败会直接报错，防止静默错误。

## R10

正确性直觉：

1. 若颜色滤波正确，`P_hat(k)` 应与输入 `P_in(k)` 在形状上同型；
2. 大尺度区间的 `n_fit` 应接近输入 `n`；
3. 谱形差异可由单一振幅 `alpha` 吸收后显著下降；
4. 固定随机种子后指标应稳定复现。

这些都由断言或诊断统计量覆盖。

## R11

数值稳定性策略：

- `k=0` 模式单独置零，避免幂律奇异；
- 对谱估计仅保留 `k>0` 且 `P>0` 的模式；
- 分箱时要求最小模式数（`count>=20`），降低小样本噪声；
- 拟合区间避免极低 `k`（模式太少）和高 `k`（截止项主导）区域。

## R12

关键参数（`PowerSpectrumConfig`）：

- `n_grid`：网格分辨率（默认 `64`）；
- `box_size_mpc`：盒尺度 `L`；
- `spectral_index`：输入谱斜率 `n`；
- `k_cut_mpc_inv`：高 `k` 截止尺度；
- `n_k_bins`：一维壳平均分箱数量；
- `seed`：随机种子。

调参建议：

- 增大 `n_grid` 提升 `k` 分辨率；
- 增大 `box_size_mpc` 改善低 `k` 采样；
- 若拟合波动较大，可增加 `n_k_bins` 或提高最小模式数阈值。

## R14

常见失败模式：

1. 忘记 FFT 归一化因子，导致量纲和幅度错误；
2. 把 `k=0` 纳入拟合，导致对数拟合失真；
3. 分箱过细导致每壳模式太少，谱估计抖动大；
4. 拟合区间进入高 `k` 截止段，斜率偏离输入幂律；
5. 随机种子漂移导致回归测试不可复验。

当前实现分别通过显式归一化、`k>0` 过滤、最小模式数、拟合窗口限制、固定 seed 控制。

## R18

`demo.py` 的源码级算法流程（8 步）：

1. `main` 创建 `PowerSpectrumConfig`，定义网格、盒尺度、输入谱参数与随机种子。  
2. `synthesize_density_field` 先用 `numpy.random.default_rng` 生成白噪声实空间场，再 `fftn` 进入傅里叶空间。  
3. `target_power_spectrum` 在每个 `|k|` 上显式计算 `A*(k/k0)^n*exp(-(k/kc)^2)`，并构造滤波因子 `sqrt(P_in)`。  
4. 将白噪声谱乘以滤波因子后 `ifftn` 回实空间，得到合成密度场；随后做去均值和方差归一化。  
5. `estimate_isotropic_power_spectrum` 再次 `fftn`，按 `P_hat = (L^3/N^6)|FFT|^2` 计算三维离散谱，并在几何 `k` 壳上求平均得到一维 `P_hat(k)`。  
6. `fit_large_scale_power_law` 在低 `k` 拟合窗上做 `log P` 对 `log k` 的线性回归，输出 `n_fit`、截距和 `R^2`。  
7. `evaluate_model_match` 不把第三方库当黑箱：先显式构建模型 `P_in(k)`，再求最优单参数振幅 `alpha=<P_hat P_in>/<P_in^2>`，计算 `log-RMSE` 与比值分位数。  
8. `main` 执行断言（斜率误差、拟合质量、形状残差）并打印表格，形成可重复的最小验证闭环。  

第三方库边界：`numpy` 负责随机数、FFT 与向量化；`pandas` 负责表格化输出；`scipy` 仅用于回归工具。谱模型、归一化、分箱与判据均在源码中显式实现。
