# 史瓦西解 (Schwarzschild Solution)

- UID: `PHYS-0047`
- 学科: `物理`
- 分类: `广义相对论`
- 源序号: `47`
- 目标目录: `Algorithms/物理-广义相对论-0047-史瓦西解_(Schwarzschild_Solution)`

## R01

史瓦西解是爱因斯坦场方程在“真空 + 球对称 + 静态”条件下的精确解，描述了无电荷、无自转天体外部时空几何。

本条目给出一个最小可运行 MVP，覆盖三件核心事情：
1. 明确写出并计算史瓦西度规分量；
2. 计算事件视界、引力红移、Kretschmann 标量等关键物理量；
3. 用测地线方程积分一个径向自由落体轨迹，验证史瓦西解的动力学后果。

## R02

MVP 选择“几何单位制 `G=c=1` + 径向类时测地线（`L=0`）”作为演示对象，原因是：

1. 公式紧凑，便于把重点放在时空结构而不是单位换算；
2. 径向自由落体同时体现“固有时有限、坐标时拉伸”的史瓦西关键现象；
3. 可以拿到解析固有时公式，便于和数值积分做强校验。

## R03

史瓦西度规（坐标顺序 `t,r,theta,phi`）写为：

`ds^2 = -(1-2M/r)dt^2 + (1-2M/r)^(-1)dr^2 + r^2 dtheta^2 + r^2 sin^2(theta) dphi^2`

定义 `f(r)=1-2M/r`，则：
- `g_tt = -f(r)`
- `g_rr = 1/f(r)`
- `g_thetatheta = r^2`
- `g_phiphi = r^2 sin^2(theta)`

MVP 只在 `r>2M` 的外部区域计算这些量。

## R04

本条目使用的关键派生量：

1. 事件视界半径：`r_s = 2M`；
2. 光子球半径：`r = 3M`；
3. 类时稳定圆轨道下界（ISCO）：`r = 6M`；
4. Kretschmann 标量：`K = 48 M^2 / r^6`（坐标不变量，用于识别真实奇点）；
5. 静止观察者发射到无穷远的红移：`z = 1/sqrt(1-2M/r) - 1`。

## R05

前提与假设：

1. 中心体不自转、无电荷（否则应使用 Kerr 或 Reissner-Nordstrom 解）；
2. 真空外部区域（忽略物质分布细节）；
3. 径向类时轨道取 `L=0`，并默认 `E=1`（从无穷远静止落下）；
4. 数值积分在 `r_end > 2M` 停止，避免坐标奇点处 `dt/dtau` 发散导致数值不稳定。

## R06

`demo.py` 输入输出约定：

- 输入：脚本内置参数（`M=1`, `E=1`, `r0=20`, `r_end=2.05`, `h=1e-2`）。
- 输出：
1. 度规/不变量表（`f(r), g_tt, g_rr, z, K`）；
2. 径向自由落体积分结果（数值固有时、解析固有时、相对误差、坐标时增量）；
3. 一组阈值检查与最终 `Validation: PASS/FAIL`。

脚本无交互输入，可直接运行。

## R07

算法主流程（高层）：

1. 初始化史瓦西参数和采样半径；
2. 计算视界、光子球、ISCO 与度规表；
3. 用守恒量写出径向测地线常微分方程；
4. 用显式 RK4 在固有时上积分 `r(tau), t(tau)`；
5. 用解析固有时公式对数值结果做误差校验；
6. 输出检查项并给出 PASS/FAIL。

## R08

复杂度分析（设积分步数为 `N`）：

1. 度规表计算：`O(K)`，`K` 为采样半径点数（本例常数级）；
2. RK4 轨道积分：每步常数成本，整体 `O(N)`；
3. 存储轨迹：`O(N)`。

该 MVP 是轻量线性复杂度，适合教学与验证。

## R09

数值稳定与误差控制策略：

1. 在 `r_end > 2M` 截止，避免 `f(r)->0` 时 `dt/dtau=E/f` 过度发散；
2. 对 `sqrt(E^2-f)` 的被开方项做 `max(...,0)`，抑制浮点负零噪声；
3. 使用 RK4（而非欧拉）提升轨道积分精度；
4. 用解析固有时 `tau_exact` 做强对照，阈值设为相对误差 `<5e-4`。

## R10

技术栈（最小化）：

- Python 3
- `numpy`：向量化计算、RK4 数值积分状态更新
- `pandas`：结果表格化输出
- 标准库 `dataclasses`

未调用黑箱 GR 求解器；测地线方程、RK4 推进和验证逻辑全部在源码中显式实现。

## R12

输出字段说明：

1. `f(r)=1-2M/r`：史瓦西 lapse 函数；
2. `g_tt`, `g_rr`：度规关键分量；
3. `z_to_infinity`：静止观察者到无穷远的引力红移；
4. `K=48M^2/r^6`：曲率不变量；
5. `tau_num`, `tau_exact`, `relative tau error`：径向自由落体固有时数值/解析对比；
6. `coordinate time delta t`：同过程中的史瓦西坐标时间增长量。

## R13

正确性验证（demo 内置检查）：

1. `lapse(2M)=0`（视界位置）；
2. 度规张量对称性 `g = g^T`；
3. `r(tau)` 严格单调下降且全程 `r>2M`；
4. 数值固有时与解析固有时相对误差小于阈值；
5. `dt/dtau` 在近视界区域显著增大（坐标时间拉伸）；
6. Kretschmann 标量向内增大；
7. 轨迹数组无 `NaN/Inf`。

## R14

当前实现局限：

1. 仅覆盖 Schwarzschild 外部真空解，不含内部匹配；
2. 仅实现径向轨道（`L=0`），未展示轨道进动；
3. 只处理无自转、无电荷中心体；
4. 数值积分使用固定步长，未做自适应步长控制。

## R17

方法对比：

1. 纯公式推导：可解释性高，但缺少可执行验证链路；
2. 直接调用大型 GR 库：开发快，但流程黑箱程度高；
3. 本方案：保留解析结构 + 手写 RK4 + 明确断言，兼顾透明性与可复现。

因此该实现适合作为“最小但诚实”的史瓦西解工程化入口。

## R18

`demo.py` 源码级流程拆解（9 步）：

1. `SchwarzschildConfig` 固定 `M,E,r0,r_end,h`，定义实验边界。
2. `schwarzschild_radius` 与 `lapse` 计算 `r_s=2M` 和 `f(r)=1-2M/r`。
3. `metric_tensor` 在给定 `(r,theta)` 组装对角度规张量分量。
4. `kretschmann_scalar` 与 `redshift_to_infinity` 计算曲率不变量和引力红移。
5. `radial_rhs` 写出径向类时测地线守恒方程：`dr/dtau=-sqrt(E^2-f)`, `dt/dtau=E/f`。
6. `rk4_step` 用四阶 Runge-Kutta 对 `[r,t]` 做一步推进。
7. `integrate_radial_infall` 循环推进直到 `r<=r_end`，记录 `tau,r,t,dr/dtau,dt/dtau` 全轨迹。
8. `analytic_tau_for_e1` 给出 `E=1` 情况下从 `r0` 到 `r` 的解析固有时，并与数值结果计算相对误差。
9. `main` 汇总度规表、轨道结果和 8 项检查，全部通过时输出 `Validation: PASS`。

说明：第三方库仅用于数值数组与表格展示，史瓦西几何、测地线方程与积分流程均在源码中逐步展开，不是黑箱封装。
