# 哈密顿力学 (Hamiltonian Mechanics)

- UID: `PHYS-0007`
- 学科: `物理`
- 分类: `分析力学`
- 源序号: `7`
- 目标目录: `Algorithms/物理-分析力学-0007-哈密顿力学_(Hamiltonian_Mechanics)`

## R01

问题定义：把哈密顿力学从公式落到可运行代码，验证“正则方程驱动的相空间演化”在数值上可复现。

本条目选择 1 自由度单摆（非线性）作为最小模型，状态采用正则变量：
- 广义坐标 `q`（摆角）
- 共轭动量 `p`

MVP 目标：
- 用 `dq/dt=∂H/∂p, dp/dt=-∂H/∂q` 直接求解轨迹；
- 对比保结构积分器（辛欧拉）与高精度参考积分；
- 检查能量漂移、振动周期和哈密顿流散度。

## R02

哈密顿力学核心思想：
- 不直接写二阶牛顿方程，而是把系统写成一阶正则系统；
- 动力信息收敛到哈密顿量 `H(q,p)`；
- 相空间演化满足李乌维尔性质（哈密顿流局部体积保持）。

对于单摆：
- 势能非线性（`1-cos q`），比小角线性振子更能体现哈密顿框架价值；
- 仍然足够小，便于完整展示算法与诊断。

## R03

本 MVP 的计算任务：
1. 给定物理参数 `m, l, g` 与初值 `q0, p0`。
2. 构造哈密顿量并实现正则方程右端项。
3. 用辛欧拉积分得到主轨迹。
4. 用 `scipy.solve_ivp(DOP853)` 作为高精度参考轨迹。
5. 输出以下诊断：
   - 辛欧拉与参考解的能量相对漂移；
   - `q/p` 最大轨迹差；
   - 周期估计及相对误差；
   - 相空间散度有限差分估计值。

## R04

建模假设（最小可运行版本）：
- 单摆杆长固定、摆球集中质量、无阻尼、无外力驱动；
- 地球重力 `g` 视为常量；
- 仅处理 1 自由度平面摆，不考虑三维姿态耦合；
- 数值对照以“高精度常微分积分”为工程参考，不视为形式化真值证明。

## R05

核心方程（`demo.py` 直接实现）：

哈密顿量：
`H(q,p) = p^2/(2 m l^2) + m g l (1 - cos q)`

正则方程：
- `dq/dt = ∂H/∂p = p/(m l^2)`
- `dp/dt = -∂H/∂q = -m g l sin(q)`

离散积分（kick-drift 辛欧拉）：
- `p_{n+1} = p_n - dt * m g l sin(q_n)`
- `q_{n+1} = q_n + dt * p_{n+1}/(m l^2)`

这两条更新式保证了“先更新动量再更新位置”的辛结构。

## R06

数值算法流程：
1. 参数校验（正值、有限值）。
2. 按 `t_end/dt` 构造时间网格。
3. 使用辛欧拉逐步推进 `(q,p)`，并记录哈密顿量序列。
4. 在同一 `t_eval` 上调用 `solve_ivp(DOP853)` 得到参考轨迹。
5. 计算两条轨迹的能量漂移和状态差值。
6. 通过“上穿零点”法估计振动周期。
7. 在若干采样点上用有限差分估计哈密顿流散度。
8. 汇总为 `pandas` 表并打印。

## R07

复杂度（设时间步数 `N`）：
- 辛欧拉主流程时间复杂度：`O(N)`；
- 轨迹存储空间复杂度：`O(N)`；
- 参考积分器开销仍近似随采样点线性增长，常数项更大。

该问题的代价主要由时间步密度和仿真时长决定，而非大型线性代数。

## R08

数值稳定性策略：
- 使用辛积分器作为主解，长期能量行为更稳定；
- 使用严格容差的 `DOP853` 生成参考基线；
- 采用“相对漂移”而非绝对漂移，避免基线尺度敏感；
- 周期估计使用线性插值上穿零点，减少采样格点误差；
- 额外检查散度接近零，确认正则场实现未破坏结构。

## R09

适用场景：
- 分析力学课程里“从拉格朗日到哈密顿”的代码化演示；
- 数值方法课中“辛积分 vs 常规积分”对比实验；
- 后续扩展到多自由度哈密顿系统的最小模板。

不适合：
- 含阻尼、驱动或碰撞切换的复杂真实机械系统；
- 长时极高精度天体力学任务（需更高阶辛算法与误差控制）。

## R10

正确性检查点（脚本已实现）：
1. 参数非法时显式抛错（`ValueError`）。
2. `solve_ivp` 失败则抛 `RuntimeError`。
3. `symplectic_energy_rel_drift <= 5e-2`（MVP 阈值）。
4. `scipy_energy_rel_drift <= 1e-6`（参考轨迹应极小漂移）。
5. 周期估计必须可计算（非 `nan`）。
6. 散度数值接近 0，验证哈密顿流结构。

## R13

理论保证（本条目范围内）：
- 连续系统满足哈密顿正则结构，理论上能量守恒；
- 辛欧拉是结构保持法，长期通常优于非辛显式方法的能量行为；
- `DOP853` 参考解并非“解析真值”，但在本参数下可作为高精度对照；
- 因此本 MVP 属于“结构一致 + 数值诊断一致”的工程级验证。

## R14

常见失败模式与修复：
- 失败：`dt` 过大导致辛欧拉轨迹偏差明显。
  - 处理：减小 `dt` 或改用更高阶辛格式。
- 失败：初值过大接近旋转态，周期估计不稳定。
  - 处理：切换到峰值检测或频谱估计。
- 失败：参数含 `nan/inf`。
  - 处理：入口参数统一校验并早失败。
- 失败：把非正则变量误当正则变量使用。
  - 处理：先明确 `p = ∂L/∂q_dot` 的定义再离散化。

## R15

工程实践建议：
- 不要只看轨迹图，必须同时看能量漂移曲线；
- 对结构化动力系统，优先考虑辛积分器；
- 对照解应和主解共享 `t_eval`，减少采样引入的误判；
- 把“可解释诊断”（周期、散度、漂移）做成固定输出，便于批量回归。

## R18

`demo.py` 源码级算法流拆解（8 步，非黑盒）：
1. `HamiltonianParams` 固定物理常数、初始条件和离散时间参数。  
2. `check_params` 对质量、杆长、重力、步长和初值做输入合法性检查。  
3. `hamiltonian` 实现 `H(q,p)`，`canonical_rhs` 实现 `dq/dt, dp/dt`。  
4. `symplectic_euler` 按 kick-drift 规则逐步更新 `p -> q`，生成主轨迹与能量序列。  
5. `scipy_reference` 在同一时间网格上用 `solve_ivp(DOP853)` 计算高精度参考轨迹。  
6. `estimate_period_from_crossings` 用上穿零点插值法估计两个轨迹周期，并计算相对误差。  
7. `divergence_numeric` 在多个采样点上有限差分估计散度，验证哈密顿流近似无散。  
8. `build_report` 聚合指标成 `DataFrame`，`main` 打印并执行漂移阈值断言。  

第三方库仅承担通用数值算子与 ODE 积分器角色，哈密顿建模、离散更新和诊断逻辑均在源码中显式实现。
